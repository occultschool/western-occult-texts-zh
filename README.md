## 繁體中文

### 簡介

本倉庫是「神秘學講堂」（OccultSchool）發布的**西方神秘學典籍中文譯庫**，收錄西方密傳哲學、神智學、卡巴拉、赫密斯主義、玫瑰十字會、占星學及相關靈性傳統的中文翻譯與評注文章，提供**繁體中文**與**簡體中文**兩種版本。

### 內容

本倉庫分為兩個主目錄：

| 目錄 | 說明 |
|------|------|
| `zh-Hant/` | 繁體中文版本 |
| `zh-Hans/` | 簡體中文版本 |

每個主目錄下包含：
- `books/` — 完整書籍翻譯（共 20 部），涵蓋：
  - 布拉瓦茨基著作（《秘密教義》、《寂靜之聲》等）
  - 弗朗茨·哈特曼《與開悟者同行》
  - 《以諾書》《薄伽梵歌》《啟示錄》注釋
  - 及其他西方密傳經典
- `docs/` — 原創評注文章（共 18 個主題分類，1,000+ 篇），包含：
  - 冥想與實修、占星與星座、魔法與靈視
  - 赫爾墨斯主義、生死奧秘
  - 布拉瓦茨基著作導讀、神秘學十二堂課 等

### 文章格式

所有文章均使用 **UTF-8 編碼的 Markdown（.md）**格式，附帶 YAML 前置元數據（標題、描述、難度標籤等）。

## English

### About

This repository is the **Western Occult Texts — Chinese Translation Archive** published by **OccultSchool** (神秘學講堂). It contains Chinese translations and original commentary articles on Western esoteric philosophy, Theosophy, Kabbalah, Hermeticism, Rosicrucianism, Astrology, and related spiritual traditions.

All content is provided in both **Traditional Chinese (zh-Hant)** and **Simplified Chinese (zh-Hans)**.

### Repository Structure

```
western-occult-texts-zh/
├── zh-Hant/                  # Traditional Chinese
│   ├── books/                # 20 fully translated books
│   │   ├── 秘密教義第一卷/    # The Secret Doctrine, Vol. I (Blavatsky)
│   │   ├── 與開悟者同行/      # Among the Adepts (Hartmann)
│   │   ├── 以諾書/            # The Book of Enoch
│   │   └── ... (17 more)
│   └── docs/                 # 1,000+ original articles (18 categories)
│       ├── 冥想與實修/        # Meditation & Practice
│       ├── 布拉瓦茨基著作/    # Blavatsky Studies
│       ├── 神秘學的十二堂課/  # 12 Lessons in Occultism
│       └── ... (15 more)
└── zh-Hans/                  # Simplified Chinese (mirror content)
    ├── books/
    └── docs/
```

### Format & Longevity

All files are **UTF-8 encoded Markdown** (`.md`) — one of the most future-proof text formats available, human-readable without any software, and trivially convertible to any other format. YAML front-matter encodes structured metadata per document (title, description, keywords, difficulty level).
