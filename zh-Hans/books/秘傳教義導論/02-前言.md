#  前言

对于学习神秘学的学生而言，要弥合基础知识和进阶知识之间的差距，是非常困难的。这两种极端的神秘学文献很多，但很少能引导学生跨越两者之间的鸿沟。本书试图弥合此鸿沟。我们假定读者已经阅读或学习过一些既有的神秘学文献，并且尝试阅读了一些更为艰深的著作，也许自认成效不彰。此学习指南旨在提供基础知识，使学生准备好接触更进阶的内容，勇敢面对其中令人困惑的问题，并能有条理地形成自己的见解。书中除了探讨棘手的哲学难题和教义中的难点外，也提到一些较为浅显的内容。

我坦率诚实地指出，晚期神智学作者在某些观点上开始存在分歧，并且已偏离布拉瓦茨基及早期「智慧大师」所传授的教义。别人可能指责我将布拉瓦茨基奉为权威，我只能坦然承认，并请求谅解。

我也请求读者原谅，因自身理解有限，某些内容难免与原教义有所出入。任何人只要自以为可以论述神智学，却非担任大师的书记员  ( 如布拉瓦茨基 ) ，必然承受此「诅咒」。

每当这本学习手册再版时，我都想根据自身多年来不断改进的理解，对内容做出各种修改。但大幅度的修订将会极大地改变和扩展原本内容，导致无法实现最初的写作目的。因此，我尽量只做最少量的修改，希望本指南对某些学生群体仍然有益。这次我特别感谢大卫布鲁斯和尼古拉维基审阅手稿并提出改进建议。
