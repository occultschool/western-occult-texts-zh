# VIII. 生命、力或重力

*不可量测的*流体已经过时了；「机械力」很少被提及；在过去的 25 年里，科学呈现出一种新的面貌；但是万有引力仍然存在，在几乎被旧理论杀死后，它靠新的理论组合得以延续生命。它可能很好地回答了科学假设，但问题是它是否也回答了真理，并代表大自然中的事实。引力本身并*不足以*解释行星的运动；它怎么能假定能解释无限**空间**中的旋转运动呢? 光靠引力是无法填补所有的空缺的，除非每个恒星都有一种特殊的冲力，而且表明每个行星及其卫星的旋转，都是由某种*同一起因*结合了引力所造成的。即便如此，一位天文学家说(《自然哲学》，艺术，142)，科学将不得不命名其起因。

神秘主义已经给它取名很久了，且所有的古代哲学家都是如此；但后来，所有这些信仰都被宣布为被驳倒的迷信。「超宇宙」的神已经消灭了对于宇宙*内*智性力量的信仰可能性，然而是谁或是什么*推动*这一运动的原动力 ? 『当我们了解了推动的独特起因，我们将准备把它与那吸引的东西结合起来。』 **弗朗索瓦** (Francoeur) 如此说到 (《天文学》第 342 页)。再一次—『天体之间的吸引力只是斥力 : 是***太阳****驱使它们不停地前进*；否则，它们的运动就会停止。』

如果能接受**太阳力**是地球上所有生命和天上运动的原始起因的理论，且如果**赫谢尔** (Herschell)的另一个大胆得多的理论被接受 (关于太阳中的某些有机体)，甚至被接受为临时假设，那么，我们的教导将被证明是正确的，且密传寓言或许在数百万年就预示了现代科学，因为这些是古老的教导。**太阳** (Marttanda ) 守望著和威胁著他的七个兄弟，即行星们；而太阳没有离弃他**母亲阿底提** (Aditi) 将他置于的中心位置 ; 『他追著他们，自己慢慢转身 ... 并从远处跟著它们，朝同一个方向移动，沿著那环绕它们宫位的道路』 - 或称轨道。(见第四节注释，第一卷) 正是太阳的流体或**散发物**赋予了太阳系中所有运动和唤醒了所有生命。它是引力和斥力，但不像现代物理学和引力定律所理解的那样；而是与***显化期****黎明的运动法则*相和谐，即太阳系重建和更高*变革*的**黎明**。这些法则是不可改变的；但是所有物体的运动是不同的，且随每一个*较小的***劫而**改变— 这是由***移动者***所调节的，即**宇宙灵魂**中的智性体。我们相信这一切真的大错特错吗 ? 好吧，这里有一个现代和伟大的科学家谈到生命电，他使用的语言更接近神秘主义，而非现代唯物主义的思想。我们建议持怀疑态度的读者参阅**罗伯特亨特** (Robert Hunt F.R.S.) 的一篇关于《太阳的热源》的文章，(在《大众科学评论》第四卷第 148 页) 他谈到太阳发光的外壳和它「独特的卷曲外观」时，说道 : —

『**阿拉戈** (Arago) 提议把这个外层称为**光球层**，现在被广泛采用。老**赫歇尔**把这光球层的表面比作珍珠母 ... 它就像宁静夏日的**海洋**，海面在微风的吹拂下微微起皱 ... **内史密斯** (Nasmyth) 先生发现了一种比先前任何猜测都更为惊人的情况 ... 具有特殊透镜形状的物体... 如同「柳叶」... 有不同大小 ... 没有按任何顺序排列 ... 在各个方向相互交叉 ... 它们之间有不规则的运动 ... 可以看到它们彼此靠近和后退，有时还会出现新的角度位置，所以其外表 ... 被比作一群密集的鱼群，的确，它们在形状上很像... 这些物体的大小，让人们对于太阳物理运作的巨大规模有了一个宏大的概念。它们的长度不少于1,000 英里，宽度不能少于 200 到 300 英里。在那些叶状或透镜状物体的猜想上，*最有可能的是光球层\**是一个巨大的气态物质之海洋 (什么样的「物质」?) ... 处于强烈 (表面上的) 白热状态，且它们是火焰片的透视投影 .... 』

【\*而且还会发现*中央物质*，或者说是映像的中心。】

神秘主义说道，通过望远镜所看到的太阳「火焰」是*映像*。但请看神秘主义者在第一册中对此的观点。

『不管它们 (那些火焰片) 是什么，*很明显它们是太阳光和热的直接来源*。这里围绕著发光性物质的包层，\*摆动著强大的能量，并传递它的运动到恒星空间的以太性媒介，从而在遥远的世界产生热和光。我们已经说过，那些形式被比作某些生物，且**赫谢尔**说道，「虽然说这样的组织*拥有生命* (为什么不呢?) 未免太过大胆，†然而，我们不知道那活跃行为能发展热、光和电。」... *这个美好的想法里是否存在真理？难道在我们太阳系的中心太阳里，其中活跃物质的脉动是所有生命的源泉吗 ? 这些生命聚集在地球上，并且毫无疑问蔓延到其他行星上，而太阳是这些行星威武的的****部长*** *?* 』...

【\*这种「物质」就「发光性」*灯芯*在镜子里的火焰映像。】

【†见《神智学的五年》，第 258 页—回答赫谢尔的推测。】

神秘主义肯定地回答了这些疑问；且总有一天，科学会发现事实如此。

同样，在第 156 页，**亨特**先生写道:——

『但若将**生命**—**生命力**—视为一种远比光、热或电更为崇高的力量，并且*确实能够对它们全部施加一种控制的力量*』(这绝对是神秘的) ... 『我们确实倾向于欣然接受这种推测，即光球层是*生命力量的主要来源*，并以*诗意的愉悦来看待把太阳能比作****生命****的假设*。』

因此，我们有了一个重要的科学确证来阐述我们的基本信条之一，即 (a) 太阳是**生命力**的储藏室，而**生命力**是**电**的***本体***; (b) 生命之流正是从它那神秘的、永远无法探测的深处流过**太空**，也流过地球上一切生物的有机体。我们看看另一位著名的医生是怎么说的，他称这种 (我们所谓的生命流体 ) 称为「神经以太」。 将下面摘录的文章改动几个句子后，你就有另一篇关于**生命力**的*准神秘的论文*。这里又再一次是**理查森**博士在《大众科学评论》卷 X第 380 -3 页中，针对「神经以太」发表了他的观点，如同他对「太阳力」和「地球力」的看法一样：—

『这个理论试图传达的观点是，在固体或液体的分子之间，在神经组织、在构成身体的所有器官部分之间，存在一种精细的、微妙的介质、蒸气或气体，它使分子处于一种状态可以相互运动，也可以排列和重新排列形体；所有的运动都通过这种媒介来传递；身体的一个器官或部分正是经由它来与其他部分交流，且外在生命世界是通过它与活人交流 : 这种媒介的存在，使生命现象得以显现，而当它的普遍离去时，将使身体实际上死亡 ... 』

且整个太阳系都落入了***沈睡期***—作者可能如此补充道。但让我们进一步阅读:

... 『我使用***以太***这个词的一般意义上，是指一种非常轻的、蒸汽或气态的物质; 简而言之，我使用它的方式就如同天文学家在谈论太空中的**以太**时一样，指的是一种微妙但物质性的媒介。... 当我说到*神经***以太**时，我并不是说以太只存在于神经结构中 : 我确实相信它是神经组织的一个特殊部分 ; 但是，正如神经进入所有具有运动和感知能力的结构，神经以太也进入所有这些部分; 且根据我的观点，神经以太是血液的直接产物，所以我们可以把它看作血液氛围的一部分。

... 有令人信服的证据表明，存在一种遍及神经物质的弹性介质，这种介质能够受简单压力的影响。...在神经结构中，毫无疑问，存在一种真正的神经流体， *正如我们的前辈*所教的那样。\*这种液体的确切化学 (？) †成分尚不为人所知；它的物理特性很少被研究。它是否随流而动，我们不知道；它是否循环，我们不知道；它是否在中枢形成后进入神经、还是形成于血液进入神经的各个地方，我们不得而知。因此，我们不知道这种液体的确切用途。然而我意识到，真正的神经物质流体本身不足以充当微妙媒介，来连接人类和动物的外部宇宙和内部宇宙的。我认为—这是我对旧理论的修正—生命中一定存在另一种形式的物质；一种以蒸汽或气体的形式存在的物质，弥漫在整个神经有机体中，如同一层*包围著神经结构分子的氛围*‡一样，是所有运动的媒介，与神经中枢相互传递。... 一旦意识到在活著时，*在动物体内存在著一种精细弥漫的物质型态*，一种充满每一个部分的蒸汽— 甚至储存在某些部分；由生命化学不断更新的物质；一种像呼吸一样容易处置的物质，它在完成了使命之后—一股新的光之洪流照耀那智性上。』...

【\*比如**帕拉塞尔苏斯** (Paracelsus)，他称它为*生命液体* (liquor vitae, and Archaeus)】

【†或者说是*炼金术上*的「组成」。】

【‡『这个生命力量......在人的周围像一个发光的球体辐射』...**帕拉塞尔苏斯**在《帕拉格拉嫩》("Paragranum") 中如此说。】

一股新的光之洪流当然是照耀在古代和中世纪神秘主义和它信徒的智慧上。因为在三百多年前，即在十六世纪时，**帕拉塞尔苏斯** (Paracelsus) 曾写过同样的东西，如下 : ——

『整个**微观宇宙**都潜在的包含在*生命液体* (Liquor Vitae) 里，是一种神经液体 ... 其中包含著存在物的本性、性质、特征和本质 。 』...(《论人类的生成》 "De Generatione Hominis") ... 『***生命液体*** (Archæus 或 Liquor Vitae) 是一种平均分布在人体的所有部分的要素 ... ***生命之气*** (Spiritus vitæ) 源自***世界之气*** (Spiritus Mundi)。它作为*后者的一种散发物*，包含了所有宇宙影响的元素，因此可以解释为是星星们 (宇宙力量)对于人的不可见身体 (生命星光体) 作用的起因。』 (《论肢体力量》 ("De Viribus Membrorum”)见《帕拉塞尔苏斯的一生》，**弗朗茨·哈特曼**M.D., F.T.S.著)

如果**理查森**博士研究过**帕拉塞尔苏斯**所有的秘密著作，他不必那么频繁地忏悔—『我们不知道』... 『这是我们还不知道的』... 等等,等等。他也不会说出下面这句话，反悔他独自*重新发现*的最好部分，其中说到 (第 384 页):——

『有人或许会认为，在这种思考方式就是以太存在的理论 ... 据说它遍布整个空间 ... 可以说，这种普遍的以太从外部渗透到所有动物的器官中，且是每一个组织的一部分。*如果这一观点是正确的*，那么它就是在物理上发现的泛神论 (!!)。它不可能是真的，因为它会摧毁每一种独立感知的个体性 ...

我们看不到它，但*我们知道*它并非如此。**泛神论***可能在*『物理上被*重新发现*』。所有的古人都知道、看到和感觉到它。泛神论表现在浩瀚的星空中，表现在海洋的呼吸中，且表现在最小草叶的生命颤动中。哲学拒绝一个*有限*和*不完美*的上帝存在于宇宙中，如一神论追随者所描绘的拟人化神。它以其名称「斐洛-神-智慧」(Philo-Theo-Sophia) 否定了荒谬的想法，此想法认为无限的、***绝对的*神**应该，或者更确切地说，*能*与物质的有限虚幻进化有直接或间接的联系，因此，不能想象在该**神***之外*还有宇宙，或想象后者在活跃或不活跃物质中不存在它。\*为什么**空间**的**以太**、或者「神经**以太**」会『摧毁每一种感知的个体性』，这对于一个熟悉「神经以太」真实本质的人来说，似乎是不可理解的，他熟悉其梵语或者说是密传和卡巴拉的名字。**理查森**博士同意——

【\*这并不是说每一丛灌木、每一棵树或每一块石头都是**神**或*一个*神灵；而是宇宙显化的每一个微粒都属于「神」的，并且是它的实质，无论它在**万古**永远进化的循环中降至何等低处；而且无论每一个单独的微尘，还是整体的宇宙， 都是那***遍一灵魂***的方面和提醒—哲学拒绝称它为上帝，因为这会限制了永恒和永远存在的根和本质。】

『如果我们没有独自制造与外部世界之间的沟通媒介，如果它是由外部世界产生的， *只适应一种振动*，那么我们需要的感官就比我们拥有的要少 : 因为，只提两个例子— 光的以太不适于声音，因而我们既能看见也能听见；空气作为声音运动的媒介， 并不是光的媒介，因而我们能看见和听到。』

事实并非如此。 这位博学博士的观点 :『**泛神论***不成立*，因为它会消灭每一种独立感知的个体性』，表明他所有结论都是建立在现代物理理论的基础上的，尽管他很乐意对它们进行改革。但他会发现这不可能达成，除非他允许精神感官的存在以取代逐渐萎缩的肉体感官。『我们所看到和听到的，』(当然在**理查森**博士看来) 对应于唯物主义科学对视觉和听觉现象的解释，后者也假定我们不能以其他方式看到和听到东西。神秘学者和神秘主义者知道得更多。吠陀雅利安人熟悉声音和颜色的奥秘，就像我们的生理学家熟悉物理层面上的事物，但前者掌握了两种层面上的秘密，是唯物主义者无法企及的。他们知道一种双重的感知；精神上和物质上。一个人失去了一种或多种感官后，其余的感官会变得更加发达 : 即，盲人可以通过触觉、听觉等感官恢复视力，而聋人可以通过*视觉来听*，即透过有声的看见说话者嘴唇和嘴型。但这些仍然是属于物质世界的情况。至于精神感知、那些在更高层面意识上运作的感官，被生理学的前提拒绝了，因为后者对神圣科学一无所知。它将以太的运作限制为只是振动， 并把它与空气区分—尽管空气单纯是*分化*和复合的以太—使它假定的功能能适应生理学家的特殊理论。但是如果能正确地理解《奥义书》的教导，其中包含了更多真正的科学，是那些完全不理解它们的东方学家未准备好承认的。*七种感官在****心智****与物质上的关联* (七个在物质层面， 七在心智层面) 《吠陀经》 中得到了清楚的解释和定义， 尤其是在称为《随歌》 (Anugita) 的《奥义书》中 : 『不会毁坏的和会毁坏的，这些就是**本体**的双重显化。在这些事物中，不会毁坏的是存在者 (真实本质或**本体**的本性，是潜在的原则)。会毁坏的是作为个体 (或实体) 的显化。』**苦行僧**在《随歌》中如此说到 ; 而且 : 『两次诞生 (启蒙) 的人都知道这是古人的教诲 ... **空间**是第一个实体 ... 而现在**空间** (***阿卡莎***，或**以太**的本体) 有一个性质 ... 就是声音而已 ... 且声音的性质是**沙伽** (Shadga) 、**利沙婆** (Rishabha) 、**犍陀罗** (Gandhara) 、**中令** (Madhyama) 、**盘查婆** (Panchama) ，除此之外，还有五种**尼沙婆** (Nishada) 和**达瓦塔** (Dhaivata) 』; (印度全音阶)。音阶中的这七个音符是声音的原则。(参阅《随歌》第三十六章) 每一种**元素**性质都是七重的，如同每一种敢之一样，若根据它们在物质上或客观层面上的显化 (同样在其本身中是七重的) 进行评判和教条化是相当武断的。因为只有当**本体**从这 (七种) 幻像的起因中解放出来，我们才能在感官对象性质的知识 (秘密智慧) ，位于双重显化层面—可见的和不可见的。因此有人说 : —

『陈述这个美妙的奥秘 ... 仔细聆听起因的分配。鼻子、舌头、眼睛、皮肤，而耳朵作为第五 (感官) **心灵**和**觉悟**，\*这七 (感知) 应被理解为 (认识它们) 性质的起因。嗅觉、味觉、色彩、声音和第五的触觉，是心智运作的对象，也是**觉悟** (最高精神感知或知觉) 的对象，*这七个是行为的起因*。那会闻、会吃、会看、会说、会听、会想、会理解的，这七者应该被看作是*媒介的起因*。†这些(媒介)拥有性质( *悦性*(sattwa)、*激性* (rajas)、*惰性* (tamas) )，享受他们自己的性质，不论愉快和不愉快。』( 《随歌》)

神智论者们充分意识到，**里斯·大卫斯** (Rhys Davids) 先生同样对他们信仰表达了自己的看法。他说，《密传佛教》作者所提出的理论『不是佛教，也不是**密传**。』会有这个评论是因为 (a) 不幸的错将「智慧」(Budhism) 写成「佛教」(Buddhism)，如此一来与**乔达摩**的宗教相联系，而非**克里希那**、**商羯罗**和其他许多人(也包括**佛陀**)传授的**秘密智慧**；(b) **里斯·大卫斯**先生不可能知道任何真正的密传教导。但无论如何，他都是当时最伟大的巴利和佛教学者，无论他说什么，都有资格受到尊敬的聆听。但是，如果一个人对于以科学和唯物主义观点解释的外传佛教一无所知，也对于神秘哲学一无所知，却怀著恶意诽谤他人，并且在神智论者面前摆出一副高深学者的架子，我们只能微笑并尽情地嘲笑他。】

当然，神秘主义者完全意识到，**活力论者**的「谬误」曾被**沃格特**和**赫胥黎**所嘲笑的，然而，此理论在非常高等的科学领域仍然得到支持，因此，他们将高兴地感到他们不是孤独的。因此，**德·卡特法吉斯** (Quatrefages) 教授写道：——

『确实，我们不知道生命*是什么*；我们也不知道是*什么*力量使星星运 ... 生物有重量，因此受地心引力的影响；它们是众多不可或缺的物理化学现象发生的地方，而且必与以太动力学(电、热等等)的作用有关。但是这些现象是*在另外一种力的影响下*展现出来的 ... 生命与无生命力量并不是对立的，而是以自身的法则掌管和支配著它们行动。』\*

【\*《人类物种》，第 11 页。】
