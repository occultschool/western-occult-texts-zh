# VIII. 莲花，作为一种普遍的象征

没有一种古老的符号是不具有深刻的哲学意义的；它们的重要性和意义随著它们的古老性而增加。「莲花」正是如此。它是代表大自然和其众神的花，且代表抽象和**具体的宇宙**，象征著精神和物质性质的生产力量。自古以来，雅利安印度教徒、埃及人和他们之后的佛教徒都将它奉为神圣；它在中国和日本都受到尊崇，并被希腊和拉丁教会作为基督教的象征，如同现在的基督徒一样把它当作信使，而用睡莲来代替它。\*它过去所拥有的神秘含义在地球上的每一个国家都是一样的，现在仍然如此。我们建议读者参考**威廉·琼斯**爵士的著作†。对于印度教徒来说，莲花通过火和水 (精神和物质) 的作用，是大自然生产力量的象征。《薄伽梵歌》中的一节说 : 『永恒 ! 我看见创造者**梵天**坐在你的莲花之上 ! 』；**琼斯**爵士指出，莲花的种子甚至在发芽之前，就含有完美形成的叶子，这缩小版终有一天会变成完美植物，正如诗节中所提到的。在印度，莲花是大地多产的象征，更重要的象征著**须弥山**。**天上层面**四部份里的四位天使或神灵 (「大君主」(Maharajahs)，见诗节) 各自站在莲花上。 莲花是**神圣**的双性同体与人类两性同体的双重类型。

【\*基督教的**大天使加百列**拿著一束睡莲，在每一幅《天使报喜》中都向**圣母玛利亚**显现。这种花象征著火和水，或者关于创造和生成的理型，这与菩萨手中*的莲花象征著完全相同的想法*；这位菩萨向**乔达摩**的母亲**摩诃玛雅** (Maha-Maya) 宣布世界**救世主佛陀**的诞生。埃及人也经常如此把**奥西里斯**和**荷鲁斯**与莲花联系在一起，他们两个是**太阳神灵**或**火** (**圣灵**仍然以「火舌」为代表)(使徒行传)】

【†参见**威廉·琼斯**爵士的《关于亚洲的论文》。】

**火** (或**热**) 的精神 (从其理想的原型) 能激发、使之结果实，并发展为具体形式的一切；而这一切诞生于「水」或称原始**土**，进化出印度教徒的**梵天**。莲花被描绘成从**毗瑟奴**的肚脐上长出来的—此**神**在空间之水和他**无限之蛇**上休息— 这是有史以来最生动的寓言：宇宙是从中央**太阳**演化而来的，即那个「点」，这是永远隐藏的胚芽。**拉克什米** (Lakshmi) 是**毗瑟奴**的女性方面\*，他也被称为「莲花」(Padma)，在「创造」时漂浮在莲花上，且在空间「海洋的搅动」中，从「牛奶的海洋」中升起，就像**维纳斯**从泡沫中升起一样。

『... 然后坐在莲花上

美丽的光明女神，无与伦比的**室利** (Sri)

从海浪中升出来 ... 』

一位英国东方学家和诗人 (莫尼尔 · 威廉姆斯爵士，Monier Williams) 唱道。

【\***拉克希米**是「维纳斯—阿芙罗狄蒂」 (Venus - Aphrodite，而她和后者一样，手里拿著一朵莲花从海洋的泡沫中跃出。在《罗摩衍那》她被称为「莲花」(Padma)。】

这个符号背后的思想是非常美丽的，而且它还显示了它在所有宗教体系中的同源性。无论是莲花还是睡莲的形式，它都代表著同一种哲学理型—即客体存在从主体存在散发出来、或神圣**理型**从抽象过渡到具体或可见的形式。因为一旦黑暗— 或者更确切地说，无知的「黑暗」— 在它自己永恒**光明**的领域里消失时，只留下它那神圣显化的**理型**，创造性的**逻各斯**打开了他们的理解，他们在理型世界中(迄今为止隐藏在神圣思想中)看到了万物的原型形式，并在这些模型的基础上继续复制、建造或塑造那些瞬息即逝的超然形式。

在这个行动阶段，这个**造物者\***还不是**建筑师**。他出生在行动的黎明，还没有首先察觉到这个计划、还没有意识到埋藏在**永恒理型**的怀抱里的理型形态，就像未来的荷叶、无暇的花瓣，隐藏在那棵植物的种子里 ...

【\*在神秘哲学中，**造物者**或***逻各斯***被认为是「创造者」，而这只是一个像「军队」依样抽象的术语与概念。 正如后者是对一整团活跃力量或运作单位—士兵们—的总称一样，**造物者**也是众多「创造者们」或「建造者们」性质上的合成。伟大的东方学家**比尔努夫** (Burnouf) 完美地理解了这个观点，他说**梵天**并*不*创造地球，也不创造宇宙的其他部分一样。『他从世界的灵魂中进化而来，一旦脱离了第一因，他随著自己蒸发、并从自己身上散发出所有大自然。他没有立于它之上，而是和它混在一起；**梵天**和宇宙形成一个**存在**，其每一个粒子本质上都是**梵天**自己，它从自己诞生。』】

在***仪式***(《死者之书》) lxxxi 章名为《转化为莲花》的章节中，有一个头从莲花中伸出来，该神灵喊道 : 『我是纯洁的莲花，从**光辉者**显现出来 ... 我带著**荷鲁斯**的信息。我是纯洁的莲花，来自**太阳之地** ... 』

莲花的思想甚至可以追溯到**埃洛希姆**的章节，《创世纪》第一章，如《揭开伊希斯的面纱》所述。

正是在这种思想中，我们必须寻找犹太宇宙生成论中下面这一节的起源和解释 : 『神说 : 地要长出 ... 结果子的树木，各从其类，果子都包著核。』在所有的原始宗教中，「父亲之子」是创造性**神灵** — 即**他的**思想变得可见；且在基督教时代之前，从印度教徒的**三相神**到三个卡巴拉主义经文的开头，如犹太人所解释的那样，每个民族的三位一体的神在其寓言中都得到了充分的定义和证实。

这就是这个伟大的象征对于东方民族的宇宙和理型上意义。但是，当莲花被应用于实际的和外传的崇拜时—这也有它神秘的象征意义— 它及时地成为更加尘世想法的载体和容器。没有任何教条主义的宗教能逃脱其中的性因素；直到今天，它还玷污著根本想法的道德之美。下面这段话摘自已提及的同一卡巴拉主义手稿：—

『尼罗河水里生长的莲花也有类似的含义。它的生长方式特别适合作为生殖活动的象征。荷花是孕育种子来繁殖的载体，在其成熟后像胎盘一样的相连附著大地母亲，或伊西斯的子宫；长长的绳状茎是脐带，而子宫里的水是尼罗河，没有什么比这个符号更清楚的了，为了使它的本意更加完美，一个孩子有时被描绘成坐在花里或从花里发出。\*因此，**奥西里斯**和**伊希斯**，作为**克罗诺斯** (Chronos) 或称无尽时间的孩子们，在他们的自然力量的发展中，在这幅画中成为了人类的父母，此人类名字叫**荷鲁斯** ... 』(见 IX. 《月亮神》 )

【\*在印度《往世书》中，**毗瑟奴**是第一个逻各斯，而**梵天**是第二个逻各斯，或分别代表理型和实际创造者们，一个使莲花显现，另一个是从莲花中发出。】

『 我们不能过分强调使用这种生成功能作为象征性语言和科学艺术演讲的基础。对于此想法的思考，立即导致对于创造原因这一主题的反思。在它的运作过程中，大自然被观察到塑造了一个奇妙的活跃机制，由一个额外活跃灵魂所控制；灵魂的生命发展和历史，包括它的起源、现在和去向，都超过了人类智力的一切努力。\*新的诞生是一个不断重现的奇迹，作为证据展示在子宫的工作坊里，一种智性创造力介入以将一个活跃灵魂固定在一台物质机器上。这一事实的奇妙之处在于，它将所有与生殖器官有关的事物赋予一种神圣性，并作为神灵明显建设性介入的居所和场所。』

【\*除了东方形上学的**启蒙者**外，他们拥有训练有素的心灵感应能力，能探究了大自然的创造性奥秘。正是过去时代的世俗者把宇宙创造的纯粹理型，贬低为仅代表人类生殖和性功能 : 若要重新救赎和使原始概念再次变得高尚，要靠神秘的教导，以即**未来**的启蒙者，这是他们现在与未来的使命；因为此原始概念被神学和教会的宗教界人士可悲地亵渎了，他们粗暴和粗糙地应用于外传的教条和人物。对抽象或*本体***大自然**的无声崇拜是唯一的神圣的表现， 是人类崇高的宗教。】

这是对史前时代的古代哲学家的古老、纯粹的泛神论观念、*非人格*的和虔诚的基本观念的正确诠释。然而，若应用于罪恶的人性、依附于人格的粗鄙观念时就不是这样了。因此，每个泛神论哲学家都能找到上述且代表了犹太符号学拟人论的评论，这除了对真正宗教的神圣性有危险， 而且只适合我们这个唯物主义的时代，这正是这种拟人化特征的直接后果和结果。因为这是《旧约》全部精神和精髓的关键所在。手稿继续针对《圣经》艺术演讲的象征意义说到：——

『*子宫*的所在地将被视为**最神圣的地方**、**至圣所**、以及**活跃神**的*真正***圣殿**。\*对于男人来说，拥有女人一直被认为是他自身的重要部分，以使二合一，并被小心翼翼地守护且视为神圣。甚至连普通的房屋或家庭中奉献给妻子居住的部分也被称为「密室」(penetralia)，秘密或神圣的，因此隐喻为神圣建筑中**至圣所**，来自于繁殖器官的神圣性。若以隐喻来极致的描述†，，屋的这一部分在**圣书**中被描述为「房屋的两大腿之间」，且有时这个想法被建设性地贯彻到教堂的大门上，置于两侧的扶壁之间内部。』

【\*当然，这些话源自于基督教*原始*密仪的古老**启蒙者** : 『你们不知道你们是**神**的**殿**』(《哥林多前书 1》，iii，16) 在*这个*意义上不是能应用于人吗 ? 毫无疑问，在《旧约》的希伯来语编纂者的心目中，这个意思可能曾经是如此。这就是《新约》的象征者主义和犹太教正典之间的鸿沟。 如果不是基督教—尤其是最引人注目的拉丁教会—在这条鸿沟上架起了一座桥梁，这条鸿沟或许就会一直存在下去，而且还会不断扩大？ 现代罗马天主教现在已经完全持续了它，借由它两个「无沾成胎」的教条、以及它赋予**上帝之母**的拟人化和偶像崇拜的性格。】

【†它*只*在希伯来圣经和它的奴性模仿者 (基督教神学) 中被如此延伸。】

在古老的原始雅利安人中，从未存在过这些「延伸至极端」的思想。事实证明在吠陀时期，他们的妇女并没有在「密室」(闺房) 里被置于男性之外。她们的隔离始于**回教教徒**—是希伯来象征主义的下一个继承者，继基督教教会主义之后—征服了这片土地，并逐渐将他们的生活方式和习俗强加给印度教徒。吠陀之前和之后的女人和男人一样自由；早期雅利安人的宗教符号学中从未掺杂过任何不纯洁的尘世思想。这种思想和应用纯粹是**闪族人**的。这一点得到了先前提到的博学和卡巴拉主义启示的作家本人证实，他在结束上述段落时补充道:——

『 如果这些象征创造性宇宙媒介的器官，可以附加度量的起源以及时间周期的概念 ，那么确实， 在神庙的建筑中，它作为**神**或**耶和华**的**居所**，被指定为**至圣所**，应该是从公认的生殖器官神圣性中借用其名称，被视为量测的象征以及创造性的原因。对于古代的**智者**来说，**第一因***没有名字、没有想法、也没有象征* 。』 ...

明显的不是 。宁愿像早期的**泛神论**家那样，永远不去想它，且让它保持*无名字*，也不愿拖垮它的象征使之成为拟人化的形式，从而贬低那***众理型之理型***的神圣性 ! 在这里，我们再一次看到雅利安人和闪组人宗教思想之间的巨大鸿沟 : 两个对立的两极—**真诚**和**隐蔽**。婆罗门从来没有把「原罪」元素投入到人类的自然生育功能中去，对他们来说生子是一种*宗教责任*。在过去的日子里，一个婆罗门完成了他作为人类创造者的使命后，退隐到丛林中，并在宗教冥想中度过余生。他作为凡人及大自然的同事，已经完成了他对大自然的责任，因而从此，他把他所有的思想都集中在他自己精神上不朽的部分上，把尘世仅仅看作一种幻觉，一种瞬息即逝的梦—确实如此。而闪族人则不同。他虚构了伊甸园中肉体的诱惑；显示他的**神** ( 在密传上是**诱惑者**和**大自然**的**统治者** ) *永远***诅咒**一个行为，这是在大自然的逻辑程序中。\*所有这一切都是外传的解读，就像在《创世纪》和其他的*掩盖*和字面意思一样；然而同时在*密传上*， 他认为所谓的*罪恶*和「堕落」是如此神圣的行为，以至于选择了器官这个*原罪*的犯罪者，来作为最合适和最神圣的象征来代表神， 其发挥的作用被显示为不服从和永恒的**罪**！

【\*同样的想法在埃及的事件中得到了外传体现。**主上帝**极力试探**法老**，并「用大灾难来折磨他」，以免国王逃脱惩罚，从而没有任何借口让他的「选民」再次获得胜利。】

谁能理解到闪族人思想矛盾的深处呢 ? 且这个矛盾的因素在*除去*它最内在的意义后，现在已经完全变成基督教神学和教义了!

早期教会的神父是否知道希伯来 (旧约) 《圣经》的密传含义，或者是否只有少数人知道、而其他人仍然不知道这个秘密，这要留给子孙后代来决定。至少有一件事是肯定的。因为《新约》中的神秘主义与**希伯来摩西书**中的完全一致；且因为与此同时，一些纯粹的埃及符号和异教教义—例如**三位一体**—已经被《对观福音书》和**圣约翰**所模仿并融入其中，很明显，这些符号的身份是《新约》的作者所知道的，无论他们是谁。他们一定也意识到埃及神秘主义的优先地位，因为他们采用了几个代表了纯粹的埃及概念和信仰的符号— 包含其外在和内在的意义—而这些在**犹太教正典**中是找不到的。其中之一是**大天使**手中的睡莲，在早期描绘他对著**圣母玛利亚**时；这些象征性的图像至今仍保存在希腊和罗马教堂的图像中。因此，水、火、**十字**、**鸽子**、**羔羊**和其他神圣的动物，连同它们的所有组合，在神秘意义上产生了相同的含义，并且必须被接受为对犹太教纯粹和简单的改进。

因为**莲花**和**水**是最古老的符号之一，它们的起源纯粹是雅利安人，尽管它们在第五种族分支后成为共同的财产。让我们举个例子。字母和数字一样都是神秘的，不管是组合在一起还是分开来看。最神圣的是字母 M。它既有女性和男性的特征，或称雌雄同体，其原始象征为水，即伟大的深渊中。它在所有的语言中都是神秘的，无论是东方还是西方，且作为波浪的符号，如下：。在雅利安人的神秘主义中这个字母一直代表水，正如在闪族中一样 ; 即，在梵语中**摩伽罗** (makara) —十二星座的第十个 — 意思是鳄鱼，或者更确切地说，是一种与水有关的水生怪物。字母 MA 相当于且对应于数字 5 — 由二和三组成，前者象征两性分离，后者象征第三个生命，是二的后代。这再一次经常由一个五角形所象征，后者是一个神圣的符号，一个神圣的**组合图案**。**弥勒**是***第五位*佛**的秘密名字，而婆罗门的***迦乐季*化身**是最后一位将在**大周期**顶点到来**弥赛亚**。它也是**希腊墨提斯** (Metis，***神圣智慧***)的首字母；是***米姆拉*** (Mimra，话语或***逻各斯***) 的首字母；是***密特拉*** (Mihr)、***单体*** (Monad)、***奥秘*** (Mystery) 的首字母。这些人都生于伟大**深渊**，且都是母亲**摩耶** (Maya) 之子们; 在埃及则有***嘴***，希腊有***密涅瓦*** ( Minerva，**神圣智慧**)、**马丽** (Mary)、或***米利亚姆*** (Miriam)，***没药*** (Myrrha)等；基督教**逻各斯之母**，以即作为佛陀之**母**的**摩耶 (**Maya**)**。玛达瓦 (Madhava) 和**玛达薇** (Madhavi) 是印度教万神体系中最重要的神灵和女神的名字。最后，**曼荼罗** (Mandala) 在梵文中是「一个圆」，或一个球体(《梨俱吠陀》的十部分)。在印度，最神圣的名字通常以这首字母开头，包括最初显化的**智性** (Mahat)，以及**须弥山**(Mandara)，是众神用来搅动***海洋***的大山，一直到天上的**恒河** (Mandakin)、**摩奴**(Manu) 等等。

这算是巧合吗 ? 那这会是奇怪的巧合，因为们发现甚至是摩西—在尼罗河水里发现— 在他的名字里也有象征意义的辅音。法老的女儿『给他起名叫摩西 ... 因为，』她说 : 『*我把他从***水里***拉出来了*。』(Exod，II，10)\* 除此之外，在希伯来语中神的圣名应用于字母 M ，是「神圣的」或「有福的」(Meborach)，***洪水***的名字是 M 'bul。最后的例子还包括**十字架**的「*三个***玛丽**」，以及它们与***大海***(Mar)或***水***的联系。这就是为什么在犹太教和基督教中，***弥赛亚*** (Messiah) 总是关联于**水**、**洗礼**、***鱼*** (十二星座，在梵语中称为 Meenam)，甚至与**鱼化身** (Matsya) 和做为子宫象征的**莲花**或睡莲，两者相同。

【\* 甚至在**米甸**(Midian)祭司七个女儿的情况来说，她们来汲取*水*时，让**摩西**喂她们的羊群喝*水*，为此**米甸**祭司许配**西坡拉** ( Zipporah = *闪亮的*波浪) 作为摩西的妻子 (《出埃及记 ii.》) 。这一切都有同样的秘密含义。 。】

在古埃及的遗迹中，当挖掘出来的物品的祈愿符号和象征越古老，就越常见到与**太阳神灵**有关的莲花和水。***库努牡神*** (Khnoom)—是湿的力量—水，正如**泰勒斯** (Thales) 所教导的那样，是万物的法则，坐在莲花中供奉的宝座上 (**塞易斯时代**Saitic epoch ，塞拉潘神殿 Serapeum)。**贝斯神** (Bes) 站在一朵莲花上，准备吞噬他的后代。(同上, 阿拜多斯Abydos.) **托特** (Thot) 是奥秘和**智慧**之神，是**阿门提** (Amenti) 的神圣**抄写员**，他戴著**太阳圆盘**作为头饰，他有著公牛的头 ( **门德斯**(Mendes) 的圣牛是**托特**的一种形式)和一个人的身体坐著，在一个完全绽放的莲花上。(第四王朝) 最后是女神***赫克特*** (Hiquet)，她在青蛙的形状下于莲花上休息，从而显示了她与水的联系。而这个青蛙的象征不可否认是埃及最古老的神灵，埃及古物学家们一直徒劳的试图从青蛙无诗意的外形中发现其奥秘和功能。早期基督徒在教会中采用它，表明他们比我们现代东方学家更了解它。「青蛙或蟾蜍女神」是与创造有关的主要宇宙神灵之一，是鉴于其两栖的天性；且主要是因为她在古老的墙壁、岩石等中度过了漫长的孤独生活之后，显然复活了。她不仅和***库努牡***一起组织了世界，但也与*复活的教条*有关。\*这个象征一定具有某种深刻而神圣的意义，因为尽管有可能被指控为一种令人厌恶的动物崇拜， 早期埃及的基督徒还是在他们的教堂里采用了这个符号。一只青蛙或蟾刻在莲花上、或干脆不刻莲花，来*作为教堂灯*的图案，灯上刻著「我是复活者」(ego eimi anastasis)†这些青蛙女神也在所有的木乃伊身上看到。

【\*对埃及人来说，这是 3,000 年净化后的重生，要么在 「天界」(Devachan) 或「极乐之地」。】
