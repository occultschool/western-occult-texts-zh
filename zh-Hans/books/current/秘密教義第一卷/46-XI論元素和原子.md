# XI. 论元素和原子

从科学和神秘学的立场来看。

当神秘学家谈论「元素」和在那些地质年代中生活的人类，根据英国最好的地质学家之一的观点\*，无法确定其持续时间与物质的本质，这是因为他知道自己在谈论什么。当他说「人」和**元素**时，他既不是指目前的生理学和人类学形式的「人」，也不是指那些假设性概念的基本原子，即存在于科学头脑中极其稀薄物质的实体抽象 ; 也不是古代的复合**元素**。在神秘学中，***元素***无论如何都有「初始」的意思。当我们说「元素的人」时，我们指的是人类的起始和初步轮廓，处于未完成和未发展的状态，因此，它以潜伏的形式存在于人类的身体之中，在他们的一生中只在某些特定的条件下偶尔形成形体; 或者幸存于死去肉体一段时间的形体，也就是所谓的「亡灵」。†至于「元素」若用于形而上学，它指的是最初的*神圣的*人，而不是凡人； 并且，当此术语用于物质时，是处于最初未分化状态的初生物质，或者处于基质永恒*一般的*状态，及同质状态的，只会定期地分化，并且在这种分化过程中处于一种*异常*状态 — 换句话说，这是感官的暂时错觉。

【\*那位杰出的地质学家在回答一位朋友时写道 : ... 『在答复您的信时，我只能说，目前，甚至永远都**不可能**将地质时间大约所小到年分，或甚至千年。』 (签名威廉·彭格里，F.R.S.) 】

【†**柏拉图**说非理性、狂暴的**元素**是『由火、气、水和土组成』，这指的是**亡灵恶魔**。 (见《蒂迈欧篇》) 】

至于所谓的「元素原子」，神秘学者在称呼它时所指的含义，类似于印度教称**梵天**为「原子」 (Anu)。 不止一个化学家沿著炼金术士所出的道路，去寻找的每一个元素*原子*，这些在他们坚定的信念中 ( 当不是*认知*中时 )，都是一个**灵魂** ; 不一定是一个脱离肉体的灵魂，而是印度教所说的*单体* (jiva)，一个**潜在生命力**的中心，里面有潜在的智性，且在复合**灵魂**的情况下——是一种智性活跃的存在，从最高到最低的阶层，是有著或多或少分化的形体。只有一个形而上学者——一个东方形而上学家——才能理解我们的意义。所有这些「原子-**灵魂**」都是源自「一」的分化，其关系就像*神圣****灵魂***——**菩提**——与它活化和不可分割的**精神** (**阿特曼**) 之间一样。

现代物理学在借鉴古人的原子理论的同时，忘记了最重要的一点学说；因此，它们只能吃到果皮，永远也吃不到果仁。他们在采用物理原子的过程中，留下了一个发人深省的事实，即从**安纳萨哥拉斯** (Anaxagoras) 到**伊壁鸠鲁** (Epicurus)，到罗马的**卢克莱修** (Lucretius)，最后到**伽利略** (Galileo)，所有的哲学家都或多或少地相信**有生命**的原子，而不是所谓的「野蛮」物质的不可见微粒。在他们看来，旋转运动是由更大的原子 (也就是更神圣和纯粹的) 强迫其他原子向下而产生的；较轻的原子被同时向上推。它的密传含义是，分化的元素在通过存在的周期间阶段时，是不断地向下和向上的环形曲线，直到每一个元素再次到达其起点或诞生地。这种思想是形而上学的，也是物质性的；隐藏的解释采纳了「众神」或灵魂以原子的样貌，作为产生所*有效应*的*起因*，是由神圣天体在尘世的*分泌物*造成的。\*古代哲学家，甚至犹太的卡巴拉主义者，从未将**精神**与物质分离，反之亦然。一切都起源于**一**，又从一出发，最终又回到一。『光变成热，并凝结成炽热的粒子们 ; 它们被点燃后，变成又冷又硬的颗粒，又圆又光滑。而这称为***灵魂***，被禁锢在物质的外衣里；」†***原子们***和***灵魂们***在启蒙者的语言中一直是同义的。 「旋转的灵魂们」 (Gilgoolem) 是许多博学的犹太人都相信的教义(参见麦肯齐的皇家共济会百科全书)，密传上没有其他含义。博学的犹太受启蒙并非只是指「应许之地」**巴勒斯坦**，而是指***涅槃***，如博学的佛教徒和婆罗门所做的那样——**永恒者**的胸怀，以**亚伯拉罕**的胸怀为象征，以尘世的**巴勒斯坦**为代表。‡「灵魂-原子」『穿过七个行星房间』具有同样的形而上学和物质上意义。后者的含义是它*溶解成****以太****时* (见《揭开伊西斯的面纱》，第一卷，第 297 页)。即使是*典型的无神论者*和唯物主义者**伊壁鸠鲁**，也深知并相信古老的**智慧**，他教导**灵魂**( 与不朽的**精神**完全不同，前者*潜藏*在其中，就像在每一个原子微粒中一样 ) 是由*最光滑、最圆、最精细的原子*所形成的一种细腻、柔软的物质所组成的。

这表明古代的**启蒙者**的思想，受到了凡俗古代人或多或少的追随，用「原子」一词指一个**灵魂**、**神灵**或**天使**，是永远隐藏的众因**之**因的最出诞生者；从这个意义上讲，他们的教导变得可以理解了。他们和他们的后继者一样宣称，**众神**与**神灵**、天使或「恶魔」的存在并非独立于**普遍充实**，或在**普遍充实**之外，而是在其内部。在整个生命周期中，只有这个**充实**是无限的。他们承认并教授了许多现代科学现在告诉我们的东西——也就是说，存在著一种原始的「世界填充物或称宇宙基质」， 从而产生众世界；此基质永远都是同质的，除非在它的周期性存在期间，它在无限空间中的普遍弥漫中进行了分化；并逐渐从其形成恒星体。他们教导**诸天**的旋转、地球的自转、日心系统、 和**原子漩涡**——**原子们**——事实上**灵魂**和智性体们。但那些「原子论者」是精神上的、最先验的和哲学上的泛神论者。他们不曾设想或梦想过那种畸形对立的后代，这是我们现代文明种族的噩梦 ; 即认为——在一方面，是*无生命*的物质，自我引导的原子们，而在另一方面，是一个超宇宙的神。

【‡当然，*受过教育的*犹太人不会相信这个寓言的*字面意思*——即，在埋葬于外国的犹太人遗体内，包含著一种无法安息的**灵魂**原则，直到通过一种叫做「**灵魂**旋转」的过程，不朽的粒子再次触及「应许之地」的神圣**土壤**。这意义对神秘学家来说是显而易见的。这个过程应该通过一种轮回来完成， 这种心灵感应火花传递通过鸟、兽、鱼和最微小的昆虫。(参见《皇家共济会》 "Cyclo. Mackenzie.")。这个寓言与*身体的原子有关*，每个原子在到达*最终*状态之前，都要经历各种形体。此最终状态是每个原子出发的第一个起点——它的原始同质**状态**。但 「灵魂的旋转」(Gilgoolem) 的原始含义是**灵魂们**或***自我们***的转世。『所有的**灵魂**进入「旋转」(gilgoolah)』 ，进入一个循环或旋转的过程，即他们都在转世的循环道路上前行。有些卡巴拉学派将这个教义解释为只是一个邪恶灵魂的炼狱。但实际并非如此。】

这里展示一下古代**启蒙者**的教导中，**单体**是什么以及它的起源可能会有帮助。

现代精确科学一旦开始走出青少年期，就会意识到一个伟大而迄今*密传的*公理，即在精神性、心灵感应性或物质存在的领域中，任何事物都不能从虚无中产生。在显化的宇宙中，任何起因都有相应的效应，无论是在空间上还是在时间上； 也不可能有任何结果没有原始起因，而此起因又要归因于一个更高的起因——那最终的、绝对的起因对于人类将仍是一个不可理解的**无因之因**。但即使这也不能算是一个解决方案，如果有必要讨论，也必须从最高的哲学和形而上学的角度去看待，否则最好不要接触这个问题。它是一种抽象，人类的理性——无论受过怎样形而上的训练——一到它的边缘就会颤抖，有崩溃的危险。这可以向任何欧洲人展示，只要他打算通过真正的**吠檀多**信仰来解决关于存在的问题。让他阅读和研究商羯罗 (《智慧冠宝》"Viveka Chudâmani")\* 中关于**灵魂**和**精神**的崇高教义，读者就会明白他在说的是什么。

【\*由神智学者，莫希尼查特吉翻译为《智慧冠宝》，1886 年。(见《神智学者》，七月和八月期)。】

而基督徒教导人类灵魂是**神**的气息——被他创造而半永久存在，即，有开始，*但没有结束* (因此不能称为永恒) ——神秘学教义说 : 『*没有什么是创造出来的，只有转化*。在这个宇宙中，任何东西若不是早已在宇宙中——从一个地球到一个模糊的、快速的想法——否则不可能在此宇宙中显化自身；主体层面上的一切都是永恒如是；如同客体层面上的一切都是*不断变化*的——因为是暂时的。』

根据**古德**所定义的，单体即「不可分割的事物」，与我们给出的含义不同；在这里，单体被解释为与**菩提**和高等**心灵**相结合的**阿特曼** (Atma)。这三元组是一体和永恒的，在结束所有有限和*虚幻的*生命时，后者被吸收在前者中。只有从显化宇宙的初始阶段开始，才可以通过单体的朝圣过程和它暂时载体的变化来追踪。在**沈睡期 (**或称两个显化期之间的中间时期)时，它失去了它的名字，就像当人的真正自我在高级**三摩地** (***超觉***状态) 或最终**涅槃**中融入***梵天***时，就会失去名子。 用**商羯罗**的话来说 : 『当第子获得了那原初意识， 绝对的幸福，其中的本质就是真理，没有形体和行动，就像一个演员 (脱下) 服装一样，放弃了这个由*阿特曼*假定的虚幻身体。』因为**菩提** (极乐鞘)只是一*映像著*绝对极乐的镜子；而且，该*映像*本身也尚未脱离无知，也*不是***至高精神**，因为它受制于有限，是**原质**的一种精神性更改，是一种效果； 只有***阿特曼***是万物的遍一真实永恒基质——本质和绝对知识 (Kshetragna)\*。在神秘哲学中，它被称为「遍一见证者」， 而当它停留在天界时，被称为「业力的三个见证者」。

【\*现在福音书的修订版已经出版，旧版本中最明显的误译也得到了纠正，人们将更好地理解**圣约翰**五、六和七的文字: 『并且有**精神**作*见证*，因为**精神**就是真理。』 在错误翻译版本中接下来提到「三个见证人」，这以前一直被认为代表「圣父、话语和圣灵」，非常清楚显示了这位作者 (圣约翰) 的真正意思，并更加有力地将他的教义与**商羯罗**的教义联系一起。如果它们与伟大吠檀多教师更具哲学性的陈述无关或无联系，那么句子「有三个作证：**圣灵**、***水***和***血***」意味著什么？**吠檀多**教导谈到*鞘* (人内在的*原则*) 如生命 (Jiva)、 智力鞘 (Vignanamaya) 等等，在物质显化中是「水和血」或称生命；并且补充道，*阿特曼* (精神) 是唯一在*除去*鞘后剩下的东西，且它是**唯一**的*见证者*，或综合的统一性。而不那么精神性和哲学的学派，仅仅著眼于三位一体从「一」中产生了三个见证人，因此这与尘世的关联比与天上层面的关联更多。】

既然***阿特曼*** (我们的第七个原则) 等同于普遍**精神**，而人的本质又与它等同，那么**单体**是什么呢? 它是从原初「七者」所放射出数以百万计光线的同质火花——又再更进一步七重划分。它是从**未被创造*光线*** (一个奥秘) 所**散发出**的*火花*。在密传佛教中，甚至是北方外传佛教中，**本初佛** (Chogi dangpoi sangye) 是**遍一**未知的，没有开始或结束，相同于**梵**和**无限** (Ain-Soph)，从它的黑暗中发出明亮的光线。

这是***逻各斯*** (最初)，或**金刚总持** (Vajradhara)，**最高佛** (也称 Dorjechang)。他作为**所有奥秘的主**，不能显化，但会把他的心发送到显化的世界——「钻石心」，**金刚萨埵**( Dorjesempa)。这是创造的第二*逻各斯*，从它散发出七**禅那佛** (在外传的掩盖是五个) ，称为「无父母」(Anupadaka)。这些佛是来自*非实体存在*的世界 (***无形体***界) 的原始单体， 其中的**智性体** (仅在该层面上) 在外传系统中既没有形状也没有名称， 但在神秘哲学中它们有各自不同的七个名称。这些**禅那佛**凭借**禅那**的力量，而从自身散发出或创造出天上的**自我**——超人类**菩萨**。在尘世上每个人类周期的开始时，他们会根据个人的功德，偶尔会成为**人类之子**的**菩萨**，之后他们可能会以**人类佛** (Manushi Buddhas) 的身份再次出现。因此，「无父母」(或**禅那佛**) 等同于婆罗门的「心智所生之子们」——无论源自**梵天**的或三相神其他两者的，因此也等同于**圣人们** (Rishis) 和**生主们**(Prajapatis) 相同。因此，在《随歌》 (Anugita) 有一段话，若以密传的方式阅读，明显展示了相同的观念和体系，尽管使用了另一种意象。它说 : 『无论此世界上有什么实体，可动或不可动，它们都是最先被消融的(在*沈睡期*) ; 接下来消融的元素产生的发展 (塑造可见宇宙)；并且，在这些发展 (进化的实体) 之后，是所有的元素。这就是实体向上分级。**众神**、**人**、**干闼婆**、**毕舍遮** 、**阿修罗**、**罗刹**，所有的一切都是由**自性** (原质，或可塑性) 创造的， 不是由行为，也不是由起因』——即不是由任何物质起因造成的。

『这些**婆罗门** ( **圣人生主?**) 是世界的创造者，一次又一次地在这里 (尘世上) 诞生。它们所产生的一切，在适当的时候，都溶解在那五大元素中 (五**禅那佛**，或者更确切地说是七**禅那佛**，也被称为人类的**元素们** )，就像溶解于大海中的波涛。这些五大元素在任何方面都超越了构成世界的元素 (粗大元素)。而从这五元素 (精微物质)\*中解脱出来的人就达到了最高的目标。「**生主** **(梵天**) 仅凭心智创造了这一切，』即，通过***禅那***，或称抽象冥想和像**佛那佛**般的神秘力量 (见上) 。显然，这些「婆罗门」等同于天上**禅那佛**的 (尘世) **菩萨**。两者作为原始的、智性的「元素们」，*创造*或*散发出*单体们，而这些单体在那个周期注定成为人类；在此之后，他们进化自己，或者说，扩展自身，成为天上和尘世的**菩萨**或**婆罗门**，最终成为简单的人——『世界的创造者们*一次又一次在此尘世上诞生*』——确实。在北方的佛教体系，或流行的外传宗教中，它教导说每一个佛在尘世宣讲善法的同时，也同时显化在三个世界中： 在无形体世界中是**禅那佛**，在有形体世界中是**菩萨**，在最低欲望世界(或我们的世界)中，是人类。密传教导与此不同 : 神圣的，纯粹的**本初佛***单体*显化为普遍**菩提** (大菩提Maha-buddhi*印度哲学中的「伟大」 (Mahat) )，是神圣智性的根源，是精神性的、无所不知、和无所不能的，是最高的*世界之魂*或**逻各斯**。它的下降『像从永恒之**火**中蔓延出来的火焰，此火不动、不增不减，始终如一地直到存在周期的结束』，并在**世俗层面**上成为普遍生命。从这个*有意识*的**生命层面**射出**光之子们** (**生命**的*逻各斯们*) 像七只火舌；然后是沈思的**禅那佛** : 他们无形体**父亲**的具体形式—— **光之七子们**，*仍然是它们本身*，能对其应用婆罗门神秘语 : 『你是「那个」——梵。』 正是从这些**禅那佛**中散发出他们的**影子们** (chhayas) ，即天上层面的**菩萨**，是超尘世**菩萨**的原型，是尘世**佛**的原型，最后是人的原型。「光之子七子」也被称为「星星们」。

【\*「精细物质」(Tanmatras) 字面上是无性质元素的类型或雏形 ; 但在密传教导上，它们是进化成为宇宙元素的原始*本体*，这是在古代所赋予的意义上，而不是物理学的意义上。它们是逻各斯们，是逻各斯的七散发物或光线。】

根据神秘学的教义，一个人类**实体**诞生时所属的星星，在一个**显化期**内的所有转世中都将永远是它的星星。但*这不是他占星学上的星*。后者与*人格*有关，前者与**个体性**有关。那颗星的「天使」，或称**禅那佛**，将在该单体的每个新转世中，作为引导或仅仅是主持的「天使」，*单体是他自己本质的一部分*，尽管此单体的的载体 -- 人，可能永远对这个事实一无所知。开悟者们都有每一个自己的**禅那佛**， 他们年长的「双生**灵魂**」，他们知道这一点，称其为「父亲灵魂」和「父亲-火」。 然而，只有在最后和最高的启蒙时，当他们与明亮的「形象」面对面时才得知它。在**布尔沃·林顿** (Bulwer Lytton) 用他最具灵感的情绪来描绘**扎诺尼** (Zanoni) 与其**光辉体** (Augoeides) 面对面时，他对这个神秘事实了解多少呢？

***逻各斯***是未显化和显化的**话语**，被印度教徒称为「自在主」(Iswara)，意为「主」，尽管神秘学者给它另一个名字。吠檀多学者说，**自在主**是大自然的最高意识。『这最高的意识，』 神秘学者回答 : 『只是显化**逻各斯**世界中 (或在*幻象层面*) 的一个*合成性单元*; 因为它是**禅那主***意识*的总和。』 **商羯罗**说到 :『哦，智者，请消除「非精神即精神」的观念。』在最后的**梵**状态中，**阿特曼**是***非精神***，而***自在主***或***逻各斯***是**精神**；或者，就像神秘主义解释的那样，它是显化活跃**神灵**的复合统一体，是所有尘世单体的父母源和苗圃，*再加上*它们的*神圣*映像，从**逻各斯**发散出来且，各自再其时间顶点返回到**逻各斯**。有七个主要的这样 **禅那主群体**，这些群体将在每一种宗教中被发现和承认， 因为它们是**原初七光线**。神秘主义教导我们，人类被划分为七个独特的群体及其分支，包括心智上、精神上和肉体上。\*单体，若视为**遍一**，则位于 (宇宙和人) 的第七原则之上；若视为三元组，则是*复合性***单元**的直接辐射后代，而不是「神」的气息 (和从*虚空*中诞生的特殊*创造*) ，那单元如此被称为「神」; 因为这种思想是很不符合哲学的， 而且贬低了**神**，把它拖到一个有限的、具属性的限制。正如《智慧冠宝》的译者所表达的那样——尽管在*沈睡期*的最深处和*显化期*最活跃时，***自在主***是不改变的「神」』... 仍然『在 (他) *之上*的是「阿特曼」， 它的帐篷周围是永恒**幻觉**的黑暗。』 \*这些「三元组们」出生在同一个**父母行星**下， 或者更确切地说，是同一个**行星神灵** (**禅那佛**) 的辐射物，且在他们所有尘世生命和转世中，是姐妹或「双生灵魂」。†

【\*因此，有七个主要的行星，是七个神灵所居住的*层面*，在各自之下都有一个受其引导和影响的人类群体诞生。总共只有 7 个行星 (*特别*与地球相连 )和 12 个宫，但它们有无数相位的可能组合。由于每个行星都可以在 12 个不同的相位与其他行星相结合，因此它们的组合几乎是无限的；事实上，就像人类属种中有无数种的精神性、心灵感应性、智力性和身体能力一样，每个种类诞生于七大行星和无数行星组合之一的影响下。见《神智学者》，1886 年 8 月。】

【†现在有个普遍误解，即认为古人未提及其他行星，所以只知道七颗行星，这完全是基于对他们神秘学说的一般无知。问题不在于他们是否意识到后来发现的行星的存在；而是他们对四位外传和三位密传的伟大神灵——「星-天使们」——的崇敬，有其特殊原因。作者大胆地说有这样一个原因，即此原因。如果他们知道的行星和我们现在知道的一样多 (而且这个问题目前何种方式都很难解答 )，他们与他们的宗教崇拜关联在一起的仍然只有七个，因为这七个直接而特别地与我们的地球联系在一起，或者用密传术语来说，与我们的星球的七重环。 (见上述) 】

【†这与基督教的三位一体的概念是相同的，只不过是更形而上学的——「三合一」——即，**普遍的**「超灵」，显化在两个高等层面，**菩提**和**心灵**层面; 这就是三位一体，形而上的，*而不是人格化的*。】

这是每个时代和每个国家的每一个高等**启蒙者**所知道的: 「我与我**父**是一体的。」 **耶稣**说如此说 (《约翰福音》 X.30) 。\*当他在其他地方说 (xx。17) : 『我升到*我的***父**和你的**父**那里。』意思与刚才所说的相同。这只是为了表明，被**他**所吸引的弟子和追随者群属于同一个**禅那佛**、「星星」或「父亲」，属于同一个行星领域和划分。对于这种神秘教义的*知识*，能在《白莲田园诗》("The Idyll of the White Lotus") 的评论中得到了体现，**舒巴罗**先生写道 : 『每一个佛在他最后的启蒙中，会遇到先前时代所有成佛的大开物者 ... 每一类的开悟者都有自己的精神交流纽带把他们编织在一起。... 要进入这样的同胞关系唯一可能和有效的方式 ... 是通过将自己置身于**精神性**光芒的影响之中，这是*由自己的****逻各斯***所散发出的。我可以在此进一步指出... 若这些人要发生这样的交流，*他们的灵魂需从同一种神圣****光线****中获得生命和养分*，且就像「核心精神太阳」辐射出七光线一样，*所有的开悟者和****禅那主****都可分为七类*，每一类都由神圣智慧的*七形体*或显化中的一种来引导、控制和映照。』 (《神智学者》1886 年 8 月)

【\*下面的句子显示了***天使*单体**和***人类*单体**的同一性，同时也显示了他们的幻觉区别 : 『我**父***比我伟大*。』(《约翰福音》XIV.26)；『你们要荣耀*你们在天上*的**父**』(《马太福音》V.16)；『义人要在**他们父**的国里*发光*』(不是*我们的***父**) (《马太福音》 XIII.43) 『岂不知你们是神的*一个殿*，***神的灵****住在*你们里头吗？』(《哥林多前书1》 iii. 16);『我*升到*我的**父**』等等。】

这就是「光之七子」—— 以他们的行星命名，(一般人) 经常将它们等同——即土星、木星、水星、火星、金星和——*想必是*现代批评家的观点，他们只触及古宗教的表面\*——太阳和月亮，根据神秘的教义，它们综合上是我们的天上**父母**，或「父亲」。因此，如上所述，就事实和性质而言，多神论确实比拟人的一神论更为哲学和正确。土星、木星、水星和金星，这四颗外传的行星，以及另外三颗必须保持匿名的行星，这些天体与地球进行直接星光性和心灵感应性的交流，是它的**指导者**和**守望者**—— 在道德上和肉体上；可见的星球用它的外在和内在的特性来装饰我们的人性，而它们的「统治者们」或***教长们***赋予我们**单体**和精神能力。为了避免产生新的误解， 让我们声明，在这三个*秘密的*星球 (或「星-天使」)中，不包括天王星和海王星 ; 不仅因为在古代圣贤没有以这些名称称呼它们，而且因为它们和其他所有行星一样，无论有多少，都是我们太阳系中其他行星七重链的*神灵*和守护者。

【\* 采用这些行星仅是为了决疑占星术的目的。占星神谱的划分与此不同。太阳是一颗中央*恒星*而不是行星，它与*我们*星球的七行星之间存在著更为神秘和隐秘的关系。因此，太阳被认为是所有七「父亲们」的伟大**父亲**，这解释了在迦勒底和其他国家中*七位*和*八位*伟大神灵之间的差异。地球与月球——它的卫星——都不是星星，因为另一个原因——*只是密传目的的替代品*。然而，即使在计算中排除了太阳和月亮，古人似乎已经知道了七大行星。到目前为止，如果我们排除地球和月球，我们还知道多少 ? ***七颗***，仅此而已 : 七颗主要行星，其余的都是*小行星*而不是行星。】

最后发现的两颗大行星也不像其他行星那样完全依赖太阳。否则，如何解释海王星比地球接收到的光少 900 倍，天王星少 390 倍，且它们的卫星显示出逆旋转特性，是太阳系中其他行星所没有的。无论如何，我们这些话也适用于天王星，尽管最近此事实再次开始有争议。

当然，对于那些把存在物的普遍阶层与自己的分类体系混为一谈的人来说，这个主题仅仅被认为是异想天开。然而，在这里，神秘的教导陈述了简单的事实， 要么接受或拒绝，视情况而定。有些细节，由于其伟大的形而上学的抽象性，是*不能*触及的。因此，我们只是说，只有七个行星与我们的地球有如此密切的联系，就像太阳之于他体系中所有服从他的天体一样。事实上在这些天体中， 天文学只知为数不多的*主行星*和*副行星*，看上去已经够可怜的了。\*因此，可以推断出还有许多尚未被发现的行星，大小不一，但古代天文学家—— 他们所有受启蒙的开悟者——肯定已经意识到它们的存在。但是，由于它们与众神的关系是神圣的，所以必须保持神秘，其他各种行星和恒星的名称也是如此。

【\*当我们想起著名的天文学家**赫谢尔**爵士 (Sir W. Herschell) 用他的高倍望远镜，仅仅在天球的赤道面上测定，在大约以地球为中心的地方，每小时通过的星星数量达到16,000颗；把这个计算应用到「银河系」的整体上，他发现其中不少于 一千八百万个太阳——难怪**拉普拉斯**在和**拿破仑一世**谈话时，把**上帝**称为一种**假设**了——对于*精确的*物质科学来说，这是毫无用处的推测。神秘形而上学和超验哲学将单独能够在这个方向，揭开不可穿透面纱的最小的角落。】

除此之外，甚至罗马天主教神学也提到『掌管著这个地球各国命运的*七十颗*行星』；除了错误的应用以外，这种传统比现代精确天文学更有道理。七十个行星关联于色列人的*七十个*长老 (Numb.11,16) 因为这里意指行星的*统治者*，而不是行星本身; 而 70 这个词是一个*掩饰*，按指 7 × 7 的细分。如前所述，每个民族和国家都有自己的*直接***守望者**、**守护者**和**天上的父亲**——一个**行星神灵**。我们愿意将以色列自己的国家神**耶和华**留给以色列后裔，是**土星** (Sabaoth or Saturn) 的崇拜者；因为神的选民的*单体们*乃是他自己的，《圣经》也从来没有把这事当作秘密。只有英文 (新教)《圣经》的文本，一如既往地的《七十士译本》和《武加大译本》不一致。因此，在前者读到的是 ( 《申命记》. xxxii., 8 and 9) : 『当**至高者** (不是**耶和华**) 将他们的遗产分给列国的时候...他照以色列人的数目定百姓的界限，』《七十士译本》中则是『根据***天使****的数目*』(**行星天使**)，这更符合真理和事实。此外，所有文本都同意『**主**(**耶和华**) 的*部分*是他的子民; **雅各**是*他的*遗产的份。』 ( 《申命记》. xxxii. 9)；这就解决了问题。「主」**耶和华**把**以色列**作为*他的部份*——其他国家和这个特定的*国家***神**有什么关系 ? 那么，让「天使加百列」来守护伊朗，「米克尔-耶和华」来守护希伯来人。这些不是其他国家的神灵，很难理解基督徒为什么要选择一个神然后反抗他的诫命， 而耶稣是第一个起义反抗的。

**单体** (**灵魂**) 的**行星**起源和它的能力被**诺斯替派**所教导。在它到达尘世的路上，就像从尘世回来的路上一样，每一个在「无限之光」\*中或从它出生的灵魂，都必须通过七个行星区域。随著时间的推移，最古老的宗教的纯粹**天上人**和**天神**，已经成为**琐罗亚斯德教**的**七天使**，**阿里曼** (Ahriman) 的大臣们，『个自栓于他的行星』 (见奥利金的图表备份 )；对于婆罗门来说，则是**阿修罗**和一些**圣人**——有善的、恶的、漠然的；在埃及的诺斯替派中，***托特***或(**赫尔墨斯**) 是七者之首，**奥利金**给他们起的名字是**阿多乃** (Adonai)，太阳的神灵；**道** (Tao)，是月亮的神灵；**爱洛伊** (Eloi)，木星的神灵；**萨堡** (Sabao)，火星的神灵 ; **奥莱** (Orai)，金星的神灵；**阿斯塔法** (Astaphai)，水星的神灵 ; **伊尔达博**(Ildabaoth，**耶和华**)，土星的神灵。最后，《直觉智慧》 ("Pistis-Sophia") 被现代*外传***斯诺替**信仰最伟大的权威，已故的**金** (C. W. King) 先生，称为「诺斯底主义的珍贵纪念碑」 - 这份古老的文件虽然因宗派目的被歪曲，却回响了那些古老时代的信仰。**天体** (**行星**) 的**星光体统治者**从他们自己的基质中 (『从他们眼中的泪水和痛苦的汗水中』) 创造了单体们 (**灵魂们**)，赋予单体一丝**神圣之光**，这是它们的基质。这将在第二卷中展示，为什么这些「黄道和天体之主们」被教派神学转变成基督徒反叛的天使；他们从**祆教**的**七天使**借来的，但不理解这个寓言的意义。(见第二部分《七灵魂》和此部分《众神、单体和原子》 第十五节)

【\* **金** (C. W. King)把它等同于『东方所向往的*至善*，即佛教的**涅槃**」，是完美的休息，伊壁鸠鲁派的「无痛苦」(Indolentia)，这在表达上尽管看起来很轻率，但也不是完全不真实。】

一如既往，那些从一开始*就是*或*曾是*神圣、纯净和精神性的东西，由于人类观念的扭曲棱镜所产生的分化，而变得人性化和不纯净，如同反映了人的罪恶本性。因此，随著时间的推移，其他「神灵」的崇拜者开始厌恶土星这颗行星。**耶和华**之前被**奥菲特派** (Ophites) 和《雅舍记》 ("Book of Jasher”) 中视为土星之子或**伊尔达-宝特** (Ilda-Baoth) ，而在土星座下诞生的国家，例如犹太人，使它成为耶和华，且他们与在木星、水星或其他行星 (除了「土星-耶和华」) 下诞生的人永远在战斗；尽管有宗谱和预言，*启蒙者***耶稣** (或**约书亚**) ——「历史上」**耶稣**参考的原型——没有纯正的犹太血统，因此他不承认耶和华 ; 他也不崇拜任何行星神灵，除了他自己的「父」，他知道他的「父」，他像每一个高等启蒙者一样和他交流，『**精神**对**精神**，**灵魂**对**灵魂**。』这几乎无法予以异议，除非评论家能令人满意地解释，第**四福音书**的作者为什么让**耶稣**在他与**法利赛人**争论的时候，说出奇怪句子(第八章)。

『我知道你们是**亚伯拉罕**的后裔\* ... 我所说的，是在*我的***父**那里所看见的。你们听见从*你们***父**那里来的，就去行 ... 你们做*你们***父**的事 ... 你们是属于*你们*自己的**父**，**魔鬼** ... 他从起初是杀人的，不守真理，因他心里没有真理。当人说谎， 是出于自己。因为他父也是说谎的，他父也在其中。』等等。

【\***亚伯拉罕**和土星在星象符号学上是相同的，他是**耶和华犹太人**的祖先。】

**法利赛人**的「父」是**耶和华**，因为他等同于**该隐**、**土星**、**祝融星** (Vulcan) 等一样——是他们生于其下的星球，也是他们所敬拜的神灵。显然，这些话和训诫中一定有某种神秘的含义， 尽管翻译错了，因为他曾威胁任何说自己兄弟愚蠢 (raca) 的人，都将受到地狱之火的惩罚 (《马太福音》 V.22)。显然，再一次地，行星并非仅是在太空中闪烁的球体，或没有任何目的地发光，而是不同存在的领域，这些领域的存在对于世俗的人来说是未知的；然而，它们与人类和行星有一种神秘的、不间断的、强大的联系。每一个天体都是*一个*神灵的殿，而这些神灵是**神** ( 未知的「非精神」)的各个殿。 宇宙中没有世俗的东西。整个大自然都是神圣的地方，正如**杨**所说:——

『每一颗**星星**都是一座宗教殿堂。』...

因此，所有外传的宗教都可以被证明是密传教义的伪副本。现代人对于唯物主义的反动，应该归咎于祭司阶层。最新的外传宗教通过崇拜、并强制大众崇拜异教理型的外壳，将西方国土变成了一个地狱，其中上层阶级崇拜金牛犊，下层无知的群众被迫用泥脚崇拜偶像。
