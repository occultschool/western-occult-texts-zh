# XV. 众神、单体和原子

几年前，我们说到\*『**密传教义**可以被称为「串联教义」，因为它像吠檀多哲学里的「串-灵魂」 (Sutratman)†， 它通过和串在起的所有古代哲学宗教系统，调和并解释他们一切。』我们说现在它做得更多了。它不仅调和了各种各样且明显相互冲突的系统，而且它检验了现代精确科学的发现，并表明其中一些必然是正确的，因为它们在古代记录中发现得到了确证。毫无疑问，所有这些都将被认为是极其鲁莽和无礼的，是真正的亵渎科学的罪行；然而，这是事实。

【\*《七重原则》文章，在《神智学的五年》第 197 页。】

【† **阿特曼** (Atman) 或称**精神** (**精神性本体**) 像一条线穿过五个精微身体(或原则、鞘Koshas) 被称为吠檀多哲学中的「串-灵魂」( Sutratman )。】

不可否认，科学在我们这个时代是超唯物主义的 ; 但从某种意义上说，它找到了正当的理由。大自然行为实际上常常很深奥，正如卡巴拉学派所说的，她处于隐藏状态，只能通过她的表象来判断，而这种表象在物质层面上总是具有欺骗性。另一方面， 自然主义者拒绝将物理与形而上学混合，或将身体与其传递信息的灵魂和精神混合在一起， 他们宁愿忽略后者。这是某些人自己的选择，然而仍有少数人非常明智地努力踏进某些唯物主义者所不愿涉及的形而上学领域，来拓宽物质科学的范畴。这些科学家是他们那一代的智者。除非他们揭开物质的面纱，努力*超越*眼前所见，否则他们所有精彩的发现都将毫无意义，永远只是*无头*躯壳。现在他们已经研究了大自然的长度、宽度和厚度，是时候将骨架转移到第二层面，并在未知的深处寻找生命和真实的实体，寻找它的**基质**——瞬息即逝物质的本体。

只有按照这样的思路行动，才能发现现在被称为「爆炸迷信」的一些真相，它其实是事实和古代知识和智慧的遗迹。

在否认一切的怀疑论者看来，这种「有辱人格」的信念之一，体现在认为宇宙 (Kosmos) 不仅有客体的行星居民，其他世界有其人类居住外，还充满了不可见、智性的***存在***。西方所谓的**大天使**、**天使**和**神灵**，是复制了东方的原型， 及**禅那主** (Dhyan-Chohans)、**天神** (Devas) 和**祖灵** (Pitris)，都不是真实**存在**而是虚构的。在这一点上，唯物主义科学是不可逃避的。为了支持自己的立场，它打破了自身的公理法则，关于自然法则的一贯性、连续性以及进化过程中的逻辑类比序列。世俗的大众被要求相信，历史积累的证据表明，即使是古代的**无神论者**——如**伊壁鸠鲁**和**德谟克利特**——也相信*神灵*，这是错误的; 且像**苏格拉底**和**柏拉图**这样的哲学家，断言他们的存在，是错误的狂热者和傻瓜。如果我们仅仅基于历史背景、众多最著名的**圣贤**，**新柏拉图主义者**，各个时代的**神秘主义者** (从**毕达哥拉**斯到当今杰出的科学家和教授) 的权威来坚持我们的观点，如果他们拒绝「神灵」，相信「精灵」，我们是否应该把这样的权威视为弱智和愚昧，就像任何信仰并向**圣米迦勒** (曾经的人类**圣人**，或**大天使**) 祈祷的罗马天主教农民一样? 但是农民的信仰和西方**蔷薇十字会**和中世纪**炼金术士**继承者的信仰之间没有区别吗？从**罗杰·培根** (Roger Bacon) 到**圣哲曼** (St. Germain)，**凡·海尔蒙特 (**Van Helmonts)、**昆拉斯**(Khunraths)、**帕拉塞尔斯** (Paracelsuses) 和**阿格里帕** (Agrippas) ，难道他们都是盲目的狂热者、歇斯底里者或骗子，还是少数现代的怀疑论者—— 「想领袖」——谁被否定的盲目打击 ? 我们认为是后者。如果少数的否定者成为*真理*的唯一守护者，而千百万的相信神灵、天使和灵体的人——仅在欧洲和美国——也就是希腊和拉丁的基督教徒、神智学者、通灵论者、神秘主义者，只是被欺骗的狂热者和幻觉灵媒，不比骗子和欺诈者的受害者更高的话，那将是一个*奇迹*，在概率和逻辑的领域里反常的事实。无论它们的外在表现和教条如何不同，在信仰不同等级的不可见**智性体群众**上，都有相同的基础。真理和错误混杂在一起。大自然奥秘的确切范围、深度、广度和长度只能在东方神秘科学中找到。这些是如此巨大和深刻， 几乎只有很少，很少的最高**启蒙者**——*他们的存在只有少数****开悟者****知道—*—能够吸收这些知识。然而，一切都在那里，而大自然工坊中的事实和过程被允许逐渐进入准确科学中，且会有神秘的帮助给非常稀少的人来解开它的奥秘。在与种族发展有关的大**周期**的结束时，这样的事件通常会发生。我们正处在现在的**雅利安人迦梨时代**五千年循环的最后阶段；在这段时间和1897年之间，**大自然的面纱**将有一个巨大的裂缝，唯物主义科学将受到致命一击。

在不诋毁历史悠久信仰的前提下，无论在哪个方向，我们都被迫出一条明显的界线，区分由神学演变而来的盲目信仰，以及好几世代开悟者独立研究而来的知识 ; 简而言之， 在信仰和哲学之间。无论在哪个时代，都有不可否认的博学和善良的人，他们在宗派信仰中成长，固守著他们的坚定信念而死去。对新教徒来说，伊甸园是人类历史剧中的原初起点，而基督受难山充满庄严的悲剧，是那被期待的千禧年的序幕。对罗马天主教徒来说，撒旦是宇宙的根基，基督在其中心，反对基督者在其顶端。对于两者来说， **存在的阶层**开始和结束在他们各自的神学的狭窄框架内：一个自我创造的*人格***神**，和一个**最高天**回响著*被创造的*天使的**哈利路雅**；其余的是*虚假的*神， 撒旦和恶魔。

神圣哲学在更广泛的领域进行。从**万古**之始——在时间和空间中，在我们的**轮次**和**地球**上——**大自然的奥秘** (无论如何，是那些我们种族能合法了解的奥秘)以几何图形和符号的方式，被那些现在不可见的「天上人」的弟子记录了下来。钥匙由一代「智者」传给另一代。一些符号是**毕达哥拉斯**从东方带来西方的， 他不是他著名的「三角形」的发明者。后者这个图形，连同平面立方体和圆形，在描述宇宙进化顺序上，包括精神性和心灵感应性、以及物质性，比大量描述宇宙生成和揭示的《创世纪》书籍是更为优美和科学性。在「毕达哥拉斯*三角形*」内镌刻的*十个点*，价值同等于一切从神学思维中发出的神谱和天使学理论。因为阐释它们的人——在它们的表面上， 按照给出的顺序——将在这十七个点 (隐藏了七个**数学点**) 中发现从最初***天上****人到尘世*人不间断的系谱系列。而且，在给出了**存在**的阶层同时，也揭示了进化出宇宙、我们的地球、以及产生后者的原始元素的顺序。他在不可见的***深处***诞生，在同一个「母亲」的子宫里，就像它的同类一样——他将掌握我们地球的奥秘，也将掌握所有其他的奥秘。

无论无知、傲慢还是狂热主义怎样辩驳，密传宇宙起源论可以被证明与哲学和现代科学不可分割的。古代的诸神，单体们——从**毕达哥拉斯**到**莱布尼茨**——以及现在唯物主义学派的原子们 (他们从古老希腊原子论者的理论中借用的)，只是一个复合的单元， 或一个像人体一样的渐进的统一体，开始于身体，而以精神告终。在神秘科学中， 它们可以被单独研究，但除非在它们的生命周期中观察它们之间的相互关联，且在***沈睡期***时将其视为**普遍统一体**，否则永远无法被掌握。

**拉普拉修** (La Pluche) 表现出了真诚，但他在谈到对于**单体**或**数学点**的个人观点时，却给人留下了对他哲学能力较差的印象。『一个点，』他说 : 『就足以让世界上所有的学派陷入火海。但是，既然创造如此微小存在超出了人类的能力范围，人类又有什么必要知道这个点呢? 更不用说，当哲学从那一个吸收和困惑她所有沈思的点出发，企图过渡到世界的生成过程时，就违背了概率。』

然而，如果哲学没有**圆**中的**数学点**作为基础，它就永远不能形成一个逻辑的、普遍的和绝对**神**的概念。在**圆圈**的无限性和*不可认识性*中，只有显化的**点**在生成前的出现消失于我们感官时，才使哲学和神学得以调和——前提是后者应放弃其粗略的唯物主义教条。正是因为它如此不明智地拒绝了**毕达哥拉斯的单体**和几何图形，基督教神学才发展出它自己创造的人的和有人格的**神**，从这个怪物的**头**产生了**救赎**和**诅咒**两大信条。这是如此真实，甚至那些将是哲学家、以及过去是共济会员的神职人员，在他们的主观解释中，将奇怪的观念归因于古代圣贤，认为『**单体**代表 (和他们一起)**全能神**的*王座*，位于**最高天**的中心，以表示 T.G.A.O.T.U.\*——即「宇宙的伟大建筑师」。这是一个奇特的解释，更加象**共济会**而非严格**毕达哥拉斯派**的解释。

【\*《数字的科学》，作者是Rev. G. Oliver (第 36 页)。】

『**圆形**或等边**三角形**内的象形文字』也从未意味著『神圣**本质**一体性的例证』；因为这是以无限**圆**的平面为例证。它真正意味著最初分化**基质**的三元共等**性质**，或者 (显化的) **精**神、物质和宇宙的*同性质*——他们的「儿子」，其从**点** (真正的，密传的**逻各斯**) 或**毕达哥拉斯单体**诞生。因为希腊语**单体** (Monas) 在最初的意义上意味著「统一性」。 那些不能弄清楚单体——**普遍单元**——和***众单体*** ( 显化的**统一性**) 之间区别的人，或分不清「始终隐藏者」与揭示的**逻各斯**或***话语***的人，就不应当插手哲学，更不用说神秘科学了。毋庸置疑，受过良好教育的读者对于**康德**的***论文***中的第二个***二律背反***并不陌生。\*那些读过并理解它的人，将清楚地看到我们划出的界线，用来区分*绝对****理型***的宇宙和已显现但不可见的宇宙。我们的众神和**单体**不是*延展*本身的**元素**，而是那些不可见现实的元素，它是显化宇宙的基础。无论是神秘哲学、**康德**、还是**莱布尼茨**， 都不会承认延展可以由简单的或非延展的部分组成。但是「神学家-哲学家们」不会理解这一点。**圆**和**点**，后者在发散出前三个点并以线将它们连接起来之后，退入前者并与前者合并，因此，形成**显化世界**中**第二个三角形**的最初*本体*基础，过去一直是神学探索进入教条性**最高天**时的无法逾越的障碍。在这个**古老符号**的权威下，一个男性的、人格的神，一切的***创造者***和***父亲***，变成了一个三等的发散物，质点下降中排*第四*，且在**无限** (En-Soph) 的左手(参见《卡巴拉主义生命之树》)。因此， **单体**被降级为一个**载体**——一个「王座」!

【\*参见康德的《纯粹理性批判》(巴尔尼的翻译，第二卷，第54 页)。】

**单体**——仅仅是现象**世界**中**点** (**逻各斯**) 的发散物和映像——作为显化的等边三角形的*顶点*，成为「父」。 左边或线是***二元***，「母亲」，被认为是邪恶的，是对抗的原则 (**普鲁塔克**，"De Placitis Placitorum")；右边代表**儿子** (在*每一个*宇宙进化论中都是「他母亲的丈夫」，等同于*顶点*) ; 而底线是生产性大自然的**普遍**层面，统一于现象层面上的「父-母-子」，如同在超感官的**世界**里统一于*顶点*。\*通过神秘的转变，他们成为**四元**——三角形成为「十点三角形数」(Tetraktis)。

【\*在希腊和拉丁的教会里——他们把婚姻视为圣礼之一——在婚礼仪式上，主持的牧师代表*三角形*的顶点；新娘站在女性的左边，新郎站在右边，而横线则由一排见证人、伴娘和男傧来象征。但在牧师的后面是祭坛，有著有神秘的容器和象征意义，只有神圣的牧师才能进去。在早期的基督教，结婚仪式是一个神秘和真实的象征。然而，现在连教会也失去了这种象征主义的真正意义。】

这种将几何学超验的应用到宇宙和神圣神谱——神秘思想的最初(Alpha)和最末(Omega)——在**毕达哥拉斯**之后被**亚里士多德**所矮化。他略去**点**和**圆，**不考虑顶点，从而降低了理型的形而上学价值，且如此将量度学说限定为一个简单的**三元组**——*线*、*面*和*体积*。他的现代继承者们热衷于**唯心主义**，把这三个几何图形解释为**空间**、**力量**和**物质**——「一个相互作用**统一体**的力量们」。\*唯物主义科学感知的只是*显化*「三角」的底线——物质层面——实际上把它翻译为**(父)-物质、(母)-物质、(子)-物质**，而在理论上则是**物质**、**力**和**相互关系**。

【\*参见冯·哈特曼和赫伯特·斯宾塞的作品。】

但是对于一般的物理学家来说，正如一位卡巴拉主义者所说，『**空间、力、物质**，只是常规符号， 就像数学家的代数符号一样；』或者『**力**作为力， 和**物质**作为物质，都是绝对不可知的，假定为空的空间也是如此，它们在其中相互作用。』作为象征抽象事物的符号，『物理学家基于对事物起源的推理假设 ... 在他所谓的创造中，看到了三种需要 : (a) 用以创造的地方 ; (b)用来创造的媒介 ; (c)用于创造的材料。而且，通过用空间、力量、物质等术语来给这个假设以逻辑表达，他相信他已经证明了每个术语所代表的存在，*正如他所想像的那样。*」\*

【\*亨利·普拉特的《生命的新方面》】

一些物理学家认为**空间**仅仅是我们心智的表现，或与其中事物无关的延伸性，**洛克**定义空间既不能抵抗也不能运动 ; 矛盾的唯物主义者视之为一个*虚空*，见不到任何物质，他会拒绝并且蔑视有关『**空间**是一个实质尽管 (表面上) 是绝对无法被认知的**活跃实体**』的主张。(《新方面》第 9 页) 然而，这就是卡巴拉主义教导， 这是古老的哲学。空间是真实的世界，而我们的世界是人造的。它是贯穿它的无限的**统一体**，在它无底的深渊里，也在它虚幻的表面上 ; 表面布满无数现象性宇宙、太阳系和海市蜃楼般的世界。然而，对于根本上是客体**唯心主义者**的东方神秘主义者来说，在*现实*世界中，即**力量的统一性**里，『充实物 (plenum) 中存在著的一切物质的联系』，正如**莱布尼兹**所说的。这以**毕达哥拉斯三角形**作为象征。

它由*十个点*以金字塔状排列 (从第一个到最后一个四个) 刻在三条线内，以**毕达哥拉斯**著名的**十元组**来象征宇宙。上面的单一点是**单体**，代表一个「单元-点」，这个**统一体**诞生一切事物，且都与它一样的本质。三角形内的十个点代表现象界，而正三角形有三个边包围著这些点的金字塔，这些是*本体***物质**或**基质**的屏障，将它与**思想界**隔开。『**毕达哥拉斯**认为一个*点*成比例的对应于统一性；一条*线*对应于 2；一个*表面*对应于 3；一个*立体*对应于 4；且他把一个点定义为一个有位置的**单体**，以及一切的开始；一条线被认为对应于二元性，因为它是由不可分割本性的最初运动产生的，并且形成了两点的连接。一个表面被比作数字 3，因为它是数字中的最初起因 ; 就一个圆形，它是所有圆图形的原型，是由一个三元组组成，包括中心、空间和周长。但一个三角形是所有直线形中的的一个，包含在一个三元里，它根据三而得到了它的形体；这被**毕达哥拉斯学派**认为是月下事物的创造者。毕达哥拉斯三角形底部的四个点，对应著一个立体或立方体，它结合了长、宽、厚的原则，因为任何立体能有少于四个极限边界点。(毕达哥拉斯， 三角形，19 页。)

有人认为，『人类的心智无法理解的不可分割的单元，除非将其主体的想法湮灭。』这是一个错误，**毕达哥拉斯学派**和更之前的许多**灵视者**已经证明了这一点，尽管这需要经过专门训练，且世俗的头脑很难理解这一点。但是有所谓的*元数学*和*元几何学*。就算是纯粹的数学也是从**普遍性**到特殊性，从数学上*不可分割的***点**到实体图形。这种教导起源于印度，并由**毕达哥拉斯**在欧洲教导，他在**圆**和**点**上蒙了一层面纱——活人只能把它们定义为无法理解的抽象——将分化宇宙物质的起源置于**三角形**的底线或水平线上。因此，后者成为最早的几何图形。《生命的新方面》和《卡巴拉奥秘》的作者——反对将**毕达哥拉斯**的概念和使用等边三角形的方式客体化，并称之为一个*不当名称*。他的论点是，一个实质的等边体——『它的底和每边构成相等的三角形——必须有四个面，而一个三角平面必须有五个』， 相反地，这证明了这一概念的伟大之处，在于它对*创世前*和宇宙生成的密传应用。诚然，一个用数学、想象的线描绘的理想三角形『可能根本没有面，只是一个*心智的幻影* (如果要把面归于此，它们必须视所建设性地代表物体的面)。』但在这种情况下，大多数的科学假设也如同「心灵的幻影」；它们除了透过推理之外是无法证实的，并且仅仅被用来回应科学的需要。此外，理型的三角形——『作为一个三角形形体的抽象概念，因此， 作为一个抽象概念的类型』——完成并实现了预期的双重象征。它作为一个适用于客体想法的象征，这个简单的三角形变成了一个立体。当用石头在四个基点上重复时，它呈现出**金字塔**的形状，且在四个三角形的顶端，象征著现象性宇宙融合进入思想的本体性宇宙；而且，作为一个『由三条数学线构成的想象图形』，它象征著主体的层面——这些线『包围著一个数学空间 ——就等于无物包围著无物。 』因为，对于世俗和科学家的感知和未经训练的意识来说，任何超越分化物质界线的事物——即，甚至在最精神性*基质*的领域之外——必须永远*等于无*。这是「无-物」(Ain-Soph)。

然而，相对于一般在进化和物质发展的抽象概念——例如，**重力**、**物质**、**力**等——精确科学的基础，这些「心灵的幻影」实际上并不更抽象。我们最杰出的化学家和物理学家都在孜孜不旦地追求著，能最终找到*原质*的藏身之处，即毕达哥拉斯三角的底线，这并非是毫无希望的尝试。如前所述，后者是可想象到的最宏伟构想，因为它象征著理型的宇宙和可见的宇宙。\*因为如果『可能的单元只是一种大自然实在的可能性，是任何种类的个体，』并且由于每一个单独的自然对象都能够分裂，而分裂就失去了它的统一性，*或者不再是一个单元*†，这只会发生在精确科学的领域里，因为它是如此欺骗性且虚幻的世界。在神秘科学领域里，这个被无限分割的单元不但没有失去它的统一性，而且每一次的分割都接近唯一永恒**现实**的层面。**灵视者**的眼睛可以跟踪并看到它所有生成前的荣耀。关于主体宇宙的实在性和客体宇宙的虚幻性，同样的概念存在于**毕达哥拉斯**和**柏拉图**的学说的根本——只给***受拣选***的人；因为**波菲利**在谈到 《单体和二元》时，说只有前者被认为是实质性和真实的，『最简单的存在，所有统一性的起因，和一切事物的度量。』

因此，当**克鲁克斯**教授宣称『如果我们能够展示所谓的化学元素是如何产生的， 我们将能够填补我们对宇宙知识的一个巨大空白，...』答案已经准备好了。理论的知识包含在《往世书》中每一个印度教宇宙起源的密传含义中；这种实际的证明，掌握在那些在*本世纪*中将不被认可的人手中，除了极少数人会认可外。各种发现的科学可能性，必将无情地引领准确科学接受东方神秘观点，并包含填补这些「空白」所需的全部材料，但目前仍取决于现代唯物主义。只有沿著**克鲁克斯**教授采取的方向努力，才有希望认识到一些迄今为止还神秘的真理。

与此同时，如果有人渴望一瞥原初质发展的实际图表，即在循环法则的推动下分离和分化，并分为七阶的**基质** (从一般的观点来看)，那么最好的办法就是看看**克鲁克斯**演讲《元素的起源》所附的图 ，并仔细思考文本中的一些段落。在一个地方 (第 11 页) 他说：——

『... 我们对化学元素的概念已经扩展了。迄今为止，分子一直被认为是两个或两个以上原子的聚集，而没有考虑到这些原子结合在一起的结构设计。我们可以考虑到一个化学元素的结构比以前所认为的更为复杂。在我们习惯处理的化学反应中的分子和最初创造的原子之间，存在著更小的分子或物理原子的聚集体；然后，亚分子根据它们在镱结构中所占的位置而有所不同。』

『如果我们把钇想象成一块五先令的硬币，也许能简化这个假说。通过化学分馏，我把它分成了五个单独的先令，发现这些先令并不是对应体，而是像苯环中的碳原子一样，刻有它们的位置，1、2、3、4、5，印在上面 ... 如果我把我的先令扔进熔炉里，或者用化学方法把它们溶解，这些铸币印都消失，然后都变成了银。』...

这将是所有的原子和分子将面临的情况，即从他们的化合物形式和身体分离——当*沈睡期*开始时。把情况颠倒过来，想象一下新的显化期的曙光。物质被吸收后的纯「银」会再次分离成**基质**，并将产生「神圣本质」，其各个「原则」\*是主要元素们，子元素，是物质能量和主体和客体的物质；或者，这些都是代表——**众神**、**单体们**、和**原子们**。如果暂时撇开形而上学或先验方面的问题，——将卡巴拉和基督教所信仰的超感知和智性存在，排除在当前考虑之外——我们转向进化论的原子理论，神秘教义仍然在确切科学及其自述中得到证实，至少就假定的「简单」元素而言，现在突然沦为可怜的远亲——甚至不是后者的表亲。因为**克鲁克斯**教授告诉我们:

在《揭开伊希斯的面纱》第一卷 429 页，我们说 : 『最初创造的神秘一直是科学的绝望，是深不可测的，除非他们 (科学家) 接受赫尔墨斯的教义。*他们将不得不跟随赫尔墨斯主义者的脚步*。』我们的预言开始应验了。

但在**赫尔墨斯**和**赫胥黎**之间有一条中间路线和点。让科学家们把桥搁在一边，认真地想一想**莱布尼茨**的理论吧。我们已经展示了*我们*关于原子进化的理论——它们最后形成复合化学分子的过程，是在地球大气层的尘世工坊中产生的，而不是在其他地方——如此奇怪的一致于**克鲁克斯**所展示的图。在这本书中已经好几次提到，**太阳** (Mârttânda) 和他七个**弟弟**从他**母亲**(**阿底提**) 的胸怀里进化和聚集，那胸怀是**原初质** (prima mater-ia)——讲师的原初*原质*。密传教义教导存在『一种能量前身形式，具有周期性的退潮和涨潮、休息和活动状态。』(第 21 页)——看哪，一位伟大的科学家现在要求全世界接受这个作为假设之一。我们已经展示了火热的「母亲」逐渐变冷和发光，且同一位科学家提出的第二个假设，这是*科学的必然性*，似乎是——『一种类似冷却的内部操作，在原质中缓慢运行。』神秘科学教导说，「母亲」无边际的伸展开来 (在*沈睡期*)，作为大**深渊**，且根据《教义问答》里的奇怪的表达，它是「*干的*空间之水」，且只有在分离后和**那罗延** (Narayana) 在其表面移动之后，它才会变成*湿的*。**那罗延**是『**精神**，是不可见的**火焰**，它从不燃烧，却能点燃它所接触到的一切，并赋予它生命和生成。』\*现在科学告诉我们，『最初诞生的元素 ... 最接近于原质』... 会是『氢 ... 在有段时间内，它将是宇宙中唯一存在的物质形态。』***古老的***科学怎么说 ? 它回答说 : 正是这样；但我们会把创造前、甚至是地质时代前的氢和氧 (通过孵化向「母体」注入生命之火 ) 称为***精神***，这个*本体*后来在尘世上发展成最粗大的氧、氢和氮——氮没有神圣起源，它只是一种生于尘世的粘合剂，用来将其他气体和流体结合在一起，就像一块海绵，携带著**生命**的气息——纯净的空气。†在这些*气体*和流体成为*我们*大气中的样子之前，它们是星际以太；在更早且*更深*的层面上——还有别的东西，同理无穷无尽。这位杰出博学的绅士必须原谅一位神秘学者如此长篇大论地引用他的言论；但是，这是这位皇家学会成员所付出的代价，他如此接近神秘的**神圣内殿**，实际上超越了禁止的边界。

【\*『**主**是烈*火*。』... 『在他里面是*生命*，这生命就是人的光。』】

【\*如\*果用**炼金术**将其分离，就会产生**生命的精神**和其**灵药**。】

但是，但现在是时候离开现代*物质*科学，转向这个问题的心理学和形而上学方面了。我们只会提到这位杰出演讲者所要求的「两个非常合理的假设」，『为了一窥深藏在**未知**之门后面的一些秘密』——应该增加第三个加设\*——唯恐对它的攻击无用；**莱布尼茨**的推测是建立在事实和真理的坚实基础上。**约翰 · 西奥多·梅尔兹** (John Theodore Merz) 在他的《莱布尼茨》一书中，对这些推测给出了令人钦佩和深思熟虑的摘要，显示出他几乎接触到了《单子论》中密传**神谱**的隐藏秘密。然而，那位哲学家在他的推测中，几乎还没有超越第一个层面，即**宇宙大躯体**的低等原则。他的理论并不比那些*显化的*生命、自我意识和智性体的理论更崇高， 且未触及早期起源后的奥秘领域，因为他的空灵流体行星后的。

【\*最重要的是，假设大自然中没有*无机*物质或物体。石头、矿物、岩石，甚至化学「原子」都只是深沈休眠中的有机单元。它们的昏迷有终结，它们的惰性变成了活动。】

但是，这第三个假设很难为现代科学家所接受；而且，像**笛卡尔**一样，他们宁愿保留外在事物的属性，如延展性，这是无法解释运动现象的， 而应接受后者是一种独立的**力**。在这一代人中，他们永远不会变成**反笛卡尔主义者**；他们也不承认『惯性的这种特性不是一个纯粹的几何特性，它指出了在外部实体中存在的某些东西，而不仅仅是延伸性。』这是**莱布尼茨**的想法，由**梅尔茨**进行了分析，他补充说，**莱布尼茨**将这称为**力量**，并认为外部事物具有**力，**且为了要使它们具有这种力，它们就必须有一种实体，因为它们不是无生命和惰性的物体，而是形体的中心和承载者，这纯粹是一个密传的主张，因为*力*在**莱布尼茨**看来是一种*活跃的*原则，所以心智和物质的划分消失于这个结论里。但是——

『**莱布尼茨**的数学和动力学探究，在一个纯粹科学家的思维中可能不会导致相同的结果。但**莱布尼茨**并不是现代意义上的科学家。如果他是的话，他可能已经发展出了能量的概念，数学上定义了力和机械功的概念，并得出的结论是，即使是出于纯粹的科学目的，我们也应该把力看作是一种从其他值衍生出来的量，而不是一种基本的量。』

但是，幸运的是——

『莱布尼茨是位哲学家；因此，他有一些基本的原则，这使他倾向于某些结论，且他发现外部事物是具有力量的实质，便立即被用来应用这些原则。其中一项原则是连续性法则，即相信整个世界是相连的，没有无法连接的间隙和裂缝。对他来说，延展的思维物质的对比对他来说是无法忍受的。延展物质的定义已经站不住脚 : 很自然地，人们也对心灵-- 思维物质--的定义也进行了类似的究。』

**莱布尼茨**所作的划分，无论从神秘主义的立场来看，是多么不完整和错误，却显示出一种形而上学的直觉精神，是没有任何一个科学家，包括**笛卡尔**，甚至**康德**曾达到过的。对他来说，思想有无限个层次。他说，我们思想的内容中，只有一小部分上升到统觉的清晰性，『进入完美意识的光中。』许多人仍然处于一种困惑或模糊的状态，处于一种「感知」的状态；但它们确实存在 ; ... **笛卡尔**否认动物有灵魂，而**莱布尼茨**却像神秘主义者那样，认为『整个创造物都有心智生命，按照他的说法，这种存在能够有无限的层次。』正如**梅尔茨**公正地观察到的，这『立刻拓宽了心智生命的领域，摧毁了*有生命和无生命物质*的对比；它还做了更多——它反作用于物质的概念与延展的基质。因为很明显，外部或物质事物只能通过我们的感官呈现出延展性质，而不是透过我们的思维能力。这位数学家为了计算几何图形，不得不把它们分成无数个无限小的部分，而物理学家则认为物质可以分解成原子是无限的。外部事物似乎填充了空间的体积，这种属性要通过我们粗大感官来获得。... **莱布尼茨**在一定程度上遵循了这些论点，但他不能满足于假定物质是由有限数量的非常小的部分组成的。他的数学头脑迫使他无限地进行论证。那么原子变成了什么呢 ? 他们失去了延展性， 只保留了抵抗性 ; 它们是力量的核心。它们被简化为数学点 ... 但是，如果他们在空间中的延伸是无物的，*他们的内在生命要更加的充实*。假设那个内在存在，比如人的心灵，是一个新的维度，不是几何维度而是形而上维度 ... 在把原子的几何延展缩小到无之后，**莱布尼茨**在其形而上学维度方向上赋予了无限延展。在空间世界里看不到它们之后，心灵就如同潜入一个形而上学的世界，以寻找和掌握在空间中仅仅作为数学点的事物的真正本质 ... 就像一个圆锥体立于其顶点上，或一条垂直线与一个水平面交会只产生一个数学点， 但在高度和深度上可以无限延伸，所以*真实事物*的本质在这个物质空间世界中只有瞬时存在；但在思想的形而上学世界中拥有内在生命的无限深度 ... 』 (114 页)

这就是精神，神秘教义和思想的根本。「精神-物质」和「物质-精神」在*深度上*无限延伸，像**莱布尼茨**的「事物的本质」一样，我们*真实事物的本质在第七个深度*；科学和外部世界的*不真实*和粗大东西位于我们感知的最低端。神秘主义者知道后者有无价值。

在**单体**的问题上，我们现在必须向学生展示**莱布尼茨\***体系和神秘哲学体系之间的根本区别，而这可以通过摆在我们面前的《单子论》来完成。可以正确地说，如果能够调和**莱布尼茨**和**斯宾诺莎**的体系，那么神秘哲学的本质和**精神**将会展现出来。从这两种 (与笛卡尔系统相对立的) 冲击中，才显露出古老教义的真理。他们都反对**笛卡尔**的形而上学。他对两种基质 (**延展性**和**思维**) 之间对比的想法，过于武断和不符合哲学原则，如此彼此彻底不同且互不可约。因此，**莱布尼茨**把这两种**笛卡尔式**的基质视为普遍统一体的两种属性，在这个统一体中他看到了神。**斯宾诺莎**只承认遍一不可分割的基质和绝对的**一切**，就像**梵**一样。相反的，**莱布尼茨**却觉察到基质的多样性。对**斯宾诺莎**来说只有「一」; 对**莱布尼茨**来说，有无限多的存在，*从*「一」而来也在「一」*里面*。因此，虽然两人都只承认*遍一真实*的**实体**，**斯宾诺莎**视它为非人格的和不可分割的，而**莱布尼茨**却把他*人格***神**分为许多神圣和半神圣的**存在**。**斯宾诺莎**是一个*主体性*的泛神论者， 而**莱布尼茨**是一个*客体性的*泛神论者，但两人在直觉感知方面都是伟大的哲学家。

【\*莱布尼茨名字的真正拼写是Leibniz - 他自己这么拼的。他虽然出生在德国，但却是斯拉夫后裔。】

现在，如果将两种学说混合在一起并互相纠正，——最重要的是，**遍一现实**剔除了它的人格化——他们整体上仍保留著一种真正神密哲学的精神; 非人格的、无属性的、绝对神圣的本质，它*不是*「存在物」，而是一切存在存在的根源。在你的思维中，划定一个深不可测的界线，来区分那不可认识的本质，以及不可见但可理解的**临在** (原初质) 或称**圣在**，从这*之上或通过它*来震动***话语***的**声音**，并进化出无数智性***自我们***的阶层，有著意识或半意识的、*有知觉*或*统觉*的**存在们**，它们的本质是精神性**力量**，它的**基质**是**元素们**，它的**身体** (当有需要时) 是*原子们* —— 这就是我们的教义。因为，正如**莱布尼茨**所说，『一切物质体的原始**元素**都是**力**，**力**不具有 (客体的) 物质的任何特征——它可以被构想，但永远不可能成为任何想象表达的对象。』对他而言，每个身体和物体的原初和终极元素，并不是或多或少延展的物质原子或分子，如**伊壁鸠鲁**和**加森迪** (Gassendi) 的观点，而是正如**默茨**所展示的，非物质和形而上学的原子，「数学的点」；或*真正的灵魂* ——正如他的法国传记作家**亨利 · 拉希勒** (Henri Lachelier，Professeur agrégé de Philosophie)所解释的那样。『那些**灵魂**以绝对方式存在于我们之外，其本质就是力量。』(《单子论》, 引言.)

因此，在显化世界中的*现实*是由一个个*单元*组成的*统一体*，可以说，这些单元是非物质的 (从我们的立场来看) 且无限的。**莱布尼茨**称之为「单子」，东方哲学称「单体」(Jivas)——神秘主义、卡巴拉主义者和所有的基督徒，给了它各种各样的名字。他们对我们来说，如同**莱布尼茨**认为——是「宇宙的表达」\*，而每一个物质点都只是本体的、形而上学的点的现象表现。他对*知觉*和*统觉*的区分，是密传教导的哲学上表述，尽管稍显模糊。他的「简化的宇宙」，「数量等同于**单子**的数量」—— 是我们的**七重系统**及其划分和子划分的混乱表示。

【\*『**莱布尼茨**的动力学，』**拉希勒**教授说 : 『如果一个人能将**单子**视为一个*盲目力量*的简单原子，那就没有什么困难。但是 ... 』人们完全理解现代唯物主义的复杂性!】

关于他的**单体**是如何关联于我们的**禅那主、宇宙神灵**、**天神**和**元素精灵**，我们可以简单地重述一位博学且有思想的神智学者，**比耶勒高** (H. A . Bjerregaard) 先生在这个问题上的观点。**比耶勒高**在「纽约雅利安神智学会」 (见 1887 年 1 月和 2 月的第 10 号和第 11 号) 阅读一篇《论元素精灵以及它们与人的关系》的优秀论文时，他明确地陈述了他的观点 ... 『对**斯宾诺莎**来说，物质是死的且不活跃的，但对**莱布尼茨**洞察的心智来说，一切都是有生命的活动和活跃能量。在持这种观点方面，*他比他那个时代的任何思想家、或他之后的任何思想家，都更接近东方教导*。他发现*一个活跃的能量构成了****基质****的本质*，这个原则*将他与东方的****灵视者们****直接联系在一起*。』

讲师接著展示**莱布尼茨**的原子和元素是*力量的核心*，或者更确切地说，是『精神性存在，其本质是行动』，因为基本粒子不是机械地行动，而是来自*内在的*原则。它们是无形体的精神性单元 (「实质的」，然而，在我们的意义上不是*非物质的*)，不受外部变化影响，不可被任何外力摧毁。讲师补充说，**莱布尼茨**的单子『在以下几个重要细节上，元素与原子有所不同，我们必须记住这些细节，否则我们将无法看到元素与纯粹物质之间的区别。』... 『原子之间是没有区别的，它们在性质上是相似的；但是一个单子彼此之间性质各不相同；每个都是自己一个独特的世界。原子则不是这样；它们在量和质上完全相同，没有自己的个体性。\*再一次，唯物主义哲学的原子 (确切地说是分子) 可以被认为是延展的和可分割的， 而单体仅仅是数学上的点和不可分割的。最后，这是**莱布尼茨**的单体与神秘哲学的元素精灵非常相似的地方——这些单子是代表性的**存在**。每一个单子反映各个其他。每个单子在自己的层面上，都是宇宙的一面活生生的镜子。请注意这一点， 因为这取决于这些单子所拥有的力量，也取决于他们能为我们做的工作 ; 在反映世界时，单子不仅是被动的映像媒介，而且是*自发的自我活化*；它们自发地产生图像，就像灵魂做梦一样。因此，开悟者能在每个单子中阅读一切，甚至未来。每个单子或***元素精灵***是一个能说话的镜子...』

【\***莱布尼茨**是一个*绝对的***唯心主义者**，他坚持认为『物质原子与理性相悖。』 ("Système nouveau"，厄德曼，第 126 页第 2 行) 。对他来说，*物质*是单子的简单表现，无论是人类还是原子的单体。他认为 (和我们一样) 单子无处不在。因此，人的灵魂是一个单子，人体内的每一个细胞都有它的单子，在动物、植物、甚至 (所谓的) *无机*物体内的每一个细胞都是如此。他的*原子*是现代科学的分子，他的单子是物主义科学不加怀疑地接受的*简单原子*，虽然它永远不会成功地*审视*他们——除了在想象中。但**莱布尼茨**对**单子**的看法是相当矛盾的。当他谈到他的***形而上学的点***和***形体原子***，在某个时候它作为现实占据著空间；在另一时后则是纯粹的**精神性**理型；又在另一时候，在它们的相互关系中赋予客体性、聚合性和位置。】

就在这一点上，**莱布尼茨**的哲学崩溃了。在「元素精灵」单体和高等**行星神灵**的单体之间，甚至包括人类单体 (**灵魂**) 之间，都没有设定任何规定或区别。他甚至有时过度怀疑『神是否只创造了**单体**或没有延伸的物质，而没创造其它东西。』(Examen des Principes du P. Malebranche.) 他在**单子**和**原子**之间画下区别，\*因为正如他反复说的，『物体的所有性质只是现象性，就像彩虹。 』 (给狄斯波斯神父的信，书信，第十八封信)——但不久之后，他在众要的通信中找到了这样的规定，是单子之间某种形而上学的联系 (vinculum substantiale)。神密哲学教导一种*客体性*的**唯心主义**——尽管它认为客体宇宙和其中的一切都是短暂的**幻象** (Maya) ——在纯粹形而上学的角度上，区分了**集体幻象** (Mahamaya) ，以及当此幻象持续时，其中各种有意识的***自我们***之间的客体关系。因此，开悟者可以在一个**元素精灵单体**读到未来，但他大量吸引它们来达到此目的， 因为每个单体只代表它所属的界的一部分。『**单体**的限制并不在于对象本身，而在于对对象的认识的修正。它们都混乱地走向无限，走向全体，但它们都受限且区别于不同的感知程度。』 (§60,《单子论》)\*。正如**莱布尼茨**解释的那样，『宇宙的所有部分都清楚地表现在**单子们**上，*但有些反映在一个单子上，有的则在另一个单子上*；』 但是许多的单子可以同时代表二百万巴黎居民的想法。

【\*事实上，莱布尼茨的*原子*只不过是希腊唯物主义者的原子，甚至是现代科学的*分子*。他称它们为*形体*原子，并将它们比喻为**亚里士多德**的实质形体。(见《新体系》，3)】

【†**莱布尼茨**和**亚里士多德**一样，将那些被创造的或*散发出*的单子 (从**宇宙神灵**或**众神**发出的**元素精灵** )— 称为隐得莱希 (Entelechies) —和非实质*自动机*。(§ 18, \*Monadologie.\*)】

但是神秘科学对此有什么解释，且他们补充了什么呢 ?

他们说，被**莱布尼茨**集体上称为***单子***的东西——目前粗略地看，且不计算各种细分，\*——可以分成三个不同的**群体**，从最高的层面开始，首先是「众神」，或有意识的精神性***自我***；是智性的建筑师们，按照***神圣心灵***的计划工作。然后是**元素精灵**，或***单体们***，它们集体无意识地形成了**普遍大镜子**，关联于它们各自领域的一切事物。最后是原子们，或物质分子，依次由他们的*统觉性*单体来传讯，就像一个人体中每个细胞是如此*活化的*。(见第一卷结束页。) 有一群如此*活化*的原子，轮到它们使分子活化 ; 有无限多的单体，或称**元素精灵**，也有无数的精神**力量**——***无单体***的，因为它们是纯粹的非实体\*，除了在一定的法则下，他们会呈现某种形体时——*不一定是人类*。它们穿上的那些物质——围绕它们中心进化而来的表象有机体——是从哪里来的 ? ***无形体的*** (Arupa) **辐射物**，存在于**普遍意志**的和谐之中，是我们所谓宇宙意志的集体或集合，位于主体宇宙层面上，它将无限多的单体结合在一起——各自都是自己宇宙的镜子——如此暂时个体化成为独立的心灵，无所不知和普遍的 ; 且通过同样的磁聚集过程，它们从星际原子中为自己创造出客体可见的身体。对于原子和**单体**，无论它们是关联还是解离，是简单还是复杂，从最初分化的那一刻起，它们只不过是「众神」的*原则*，物质上的、心灵感应上的和**精神上**的——这些**众神**是是原初性质的**辐射物**。因此，在**灵视者**的眼中，高等**行星力量**表现在两个方面 : 主体上——作为*影响*，而客体上——作为神秘的**形体**，并在业力法则之下成为一种***临在***，**精神**和**物质**是**一体的**，正如反复陈述的那样。**精神**是*第七层面*的物质 ; 物质就是精神——在其循环活动的最低点 ; 而两者——都是**幻觉**。

【\*这三个「粗略的划分」对应于人类组成中的*精神*、*心灵*(或灵魂) 和*身体*。】

【†**比耶勒高** (C. H. A. Bjerregaard) 兄弟在他的演讲中 (已经提到)，警告他的听众不要把「圣在」( Sephiroth) 视为*个体性*，但同时要避免将它们视为*抽象*。『我们将永远无法达到真理，』他说，『更不用说与那些*天上存在交流*的能力，除非我们回归到原始时代的简单和无畏之中，在那个时代，人们与众神自由交流，神灵也降临人间，并以真理和圣洁引导他们。』( No.10， 《道路》) ... 『《圣经》中有好几个「天使」的名称，清楚地显示一些存在如同卡巴拉的**元素精灵**和**莱布尼茨**的单子，必须用这些术语来解，而不是往常的理解方式。他们被称为「晨星们」，「燃烧的火焰们」，「大能者」，且**圣保罗**在他的宇宙起源异象中把这些人视为「**权天使**和**力天使**」。 诸如这样的名称在排除了人格化的概念后，我们发现自己被迫将其视为非人格化的**存在** .. .一种*影响力*、一种精神性基质或*有意识的****力量***。』 (《道路》,No. 11, 322 页)】

原子在神秘学中被称为「振动」;也称为「声音」——集体上地。这并不冲突于**廷德尔**先生的科学发现。他在单孢体生物的底层追踪*大气振动*的整个过程——这构成了大自然过程的*客体*部分。他追踪并记录了它们运动和传播的速度；它们的冲击力 ; 它们在鼓室中引起振动并将此传递到耳石，等等，直到听觉神经开始振动——现在出现了一种新的现象 : 这个过程的*主体方面*，或者说***声音****的感知*。他感知到或看到它了吗 ? 没有 ; 因为他的专长是发现物质的行为。但难道一个通灵者、一个已开启内在之**眼**的精神性灵视者，不能看穿物质的面纱吗？科学的波和波动的产生，源自于原子*从内部*推动其分子成为活跃。原子充满了无限的**空间**， 且通过它们不断的振动，才使**生命**的轮子永远**运转**。正是这种内在的运作产生了被称为**力**的相互关系的自然现象。只有在每一种这样的「力量」的起源处，才存在著*有意识*的指导本体——**天使**或**神**，**神灵**或**恶魔**——统治的力量，但都是一样的。

正如**灵视者们**所描述的那样——它们能看到星际一群存在的运动，并以灵视力追踪它们的进化过程——它们闪耀得如同灿烂阳光下的初雪微光。它们的速度比思想还要快，比任何凡人的肉眼所能跟随的快，而且，从它们运动的巨大速度可以判断，其运动是圆形 ... 站在开阔的平原上，尤其是山顶上，凝视著上方广袤天穹和周围无限的空间，整个大气仿佛都点燃了，空气中弥漫著这些耀眼的闪烁。有时，它们强烈的运动会产生像北极光一样的闪光。这景象如此奇妙，当**灵视者**凝视这内在世界，感受这闪烁的点在身边穿行时，他对隐藏在这辉煌海洋中、和超越其中的更大奥秘感到敬畏 ...

无论这个关于「众神、单体和原子」的解释多么不完美、不完整，我们希望一些学生和神智学者至少能感觉到，在唯物主义科学和神秘主义之间，可能确实存在著一种密切的联系，后者是前者的补充和缺失的灵魂。
