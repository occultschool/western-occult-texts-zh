# VII. 一位科学家对力学科学理论的攻击

现在我们必须引用几位英国科学家的智言来帮助我们。由于「原则问题」，他们被少数人排斥，但受到许多人的默许。他们当中有一个所宣扬的近乎是神秘教义，在某些方面相当于我们的「***宇宙电*** (Fohat) 和他的七个**儿子们**」(《吠陀经》的神秘***干闼婆***)，有时甚至相当于公开承认它；这点每一个神秘者译者都会看出来，甚至一些世俗读者也能够。

如果后者打开《大众科学评论》第五卷 (第 329-334 页)，他们会发现其中有一篇**理查森** (B. W. Richardson) 博士，F.R.S，写的关于《太阳力和地球力》的文章，其内容如下: —

『仅仅运动是各种力量起源的这个理论，已再次成为普遍的思想；因而若想重新开启这个看似普遍关闭的辩论的话，会显得是异端邪说；但我愿意冒这个险，因此我要说明一位不朽的异教徒 (**塞缪尔·梅特卡夫**，Samuel Metcalfe)对**太阳力**的看法，他的名字我已经悄悄告诉过读者。我们开始于几乎所有物理学家都同意的论点，即大自然存在两个媒介，一种可测量的、可见的、有形的物质，以及一种不可测量的、不可见的，而只能通过其对物质的影响而感知的东西—**梅特卡夫**将他称为「热量」，这种不可量测而活跃的媒介*并非只是一种运动形式*、并非可测量物质粒子的振动，*而是本身是一种物质，从****太阳***流过**空间**，\*并填满固体粒子之间的空隙，并通过感官传递一种叫做热的特性。他在以下方面主张热质或**太阳力**的性质 : —

【\*不仅「穿越空间」，而且充满了我们太阳系的每一个点，因为它是**以太**的物质残留物，也就是它在我们层面上的*衬里*；**以太**除了作为传送光的「媒介」外，还必须服务于其他宇宙和地球的目的。它是星光界流质或者是卡巴拉主义者的「光」，也是「太阳-毗湿奴」的「七光线」。】

『(i) 它可以被加入到其他物体，或从中提取出来，并用精确数学测量。

『(ii) 它增加了物体的体积，且能又因抽出它而体积缩小。

『(iii) 它改变所有其他物体的形式、性质和条件。

『(iv) 它能*借由辐射来通过最完美的真空*\*，它对温度计产生的影响与在大气中一样。

【\*那么，如果*这种*物质可以通过真空，那么还需要以太波来传输光、热等等吗?】

『(v) 它能产生任何东西都无法制止的机械力和化学力，例如在火山中、火药的爆炸和其他雷暴化合物的爆炸。

『(vi) 它以一种可感知的方式作用于神经系统，产生强烈的疼痛；若过量则会导致组织紊乱。

『**梅特卡夫**反对振动理论 ，进一步提出如果热量*仅仅是一种属性或性质*，它就不能增加其他物体的体积 ; 为了达到这个目的，它本身必须有体积、必须占据空间，因此它必须是一种物质媒介。如果热量仅仅是可量测物质粒子之间*振动运动的结果*，那么它在*从热的物体辐射出来*时，振动粒子必须同步转变; 但明显事实是，热量可以从可测量物质辐射出来、而此物质重量却不减少 ... 有了这种关于热量或太阳力的物质性质观点; 他脑海中牢牢地印刻著一个印象，即『大自然的一切都是由两种物质构成的，一种本质上是活跃、空灵的，另一种本质上是被动、静止的。』\***梅特卡夫**假设**太阳力**或称热量，是一个**自我活跃**的原则。他认为，它对于自己的粒子有排斥力；而对于所有可量测物质粒子具有亲和力；它吸引可量测物质的粒子的力以与距离平方成反比。它如此*通过*可量测物质来运作。如果宇宙空间只充满了热量、太阳力 (没有可量测物质)，那么热量将是不活跃、并将构成一个的无力量或静止的以太无际**海洋**，因为如此它就没有可以作用的东西了，而那些可测量的物质无论它自身多么不活跃，都具有「某些特性」，可以改变和控制热量的运作，两者都是由不变法则所支配的，这些法律源于彼此间的相互关系和各自特有的属性。

【\*怎么可能是其他情况呢 ? 粗大*可测量*的物质是身体，是物质或**基质**的**外壳**，是女性被动原则；而这种***宇宙电*** (Fohatic) 力就是第二个原则，*生命能量* (prana)—男性和活跃的原则 ? 在地球上，这种**基质**是七重***元素***的第二原则 - **土**；在大气中，*空气*是宇宙的粗大身体；在太阳中它成为***太阳体***和**七光线**的*身体*；在恒星空间中，它与另一个原则相对应，以此类推。整体是一个同质的**统一体**，各部分都是分化的。】

『他制定了一条他相信的绝对法则，如此表述 : —

『 「热量通过吸引可量测的物质来连结和维持所有物体；而通过自身的排斥力分离和膨胀一切物体。」』

当然，这几乎是对凝聚力的神秘解释。**理查森**博士继续说 : —

『正如我已经说过的，*现代教导的倾向是建立在假设之上* ... 假设*热就是运动*，或者也许更确切地说，是一种特殊的力或运动形式。\*

【\*或者是回音，对于*声音*而言，那是更高层面上***基质***永恒运动在*我们层面上*的回响。我们的世界和感官不断地成为***幻觉***的受害者。】

『但是这个假设虽然很受欢迎，不应该排除更简单的观点，例如太阳力的物质性质，以及太阳力改变物质状态的影响。我们还知道不够多而不能教条化。』 ‡

【†一个诚实的承认】

... 『**梅特卡夫**关于太阳力和地球力的假设不仅非常简单，而且是最令人著迷的 ... 宇宙中有两种元素，一种是可量测的物质 ... 第二种元素是无处不在的**以太**，是太阳之火。它*没有重量、实质、形体或颜色；它是无限可分的物质*，且它的粒子相互排斥；它如此稀薄，以至于我们除了以太\*以外没有别的词能表达它。它弥漫并充满了空间，但单独的它也是静止的—死的。†我们把这两种元素结合在一起，即惰性物质和自我排斥的**以太**(?)，于是死的(?) 可量测物质就活跃了起来。』 \[*可量测物质可能是惰性的，但永远不会是死的—这是神秘法则*。—布拉瓦茨基夫人\] ... 『以太\[*以太的第二原则*。—布拉瓦茨基夫人\] 渗透可量测物质的粒子，且渗透到它与可量测粒子结合在一起，并维持它们成块、使它们结合在一起；它们溶解在以太中。』

【\*然而它不是***以太***，而是**以太**的原则之一，后者本身就是***阿卡莎*** (Akasa) *的原则之一*。】

【†因此*生命能量* (prana 或 Jiva)也遍及整个人的身体；但如果没有一个可作用的原子，它就会*静止不动*—死的；即会在同质状态 (laya) 中，或者用**克鲁克斯** (Crookes) 的话说，『锁在*原质*中。』 『***宇宙电***对化合物的作用、甚至是对一个简单物体，就产生了生命。当一个身体死亡时，它变成与它男性能量相同的即性，因此排斥其活跃媒介，失去了对*整体*的控制，而依附于部分或分子，这种作用被称为化学作用。**毗湿奴**这个**维持者**将自己变成**毁灭者**「楼陀罗-湿婆」(Rudra-Siva)—一种在科学上似乎是未知的关联。】

『根据我们面前的理论，固体可测量物质的分布通过以太延伸到目前存在的一切事物。以太无处不在。人体本身充满以太\[例如星光界流质——布拉瓦茨基夫人\]；其微小的粒子被它维持在一起；植物处于相同的状况；最坚硬的泥土、岩石、铁石、水晶、金属，都是一样的。但是不同种类的可测量物质在接受太阳力的能力是不同的， 且这取决于物质的各种变化情况；固态，液态和气态。固体比流体吸引到更多的热量， 因此它们具有牢固的*凝聚力*；当一部分的熔化的锌被倒在一盘固体锌上时，融化的锌会变成固体，因为有一股热量从液体涌向固体，且在平衡过程中，以前是松散的或液体的粒子，现在更紧密地聚集在一起 ... **梅特卡夫**本人详细论述了上述现象，并用作用原理的统一性来解释它们，他已经解释过了，他在有关各种物体密度的评论中，用非常清晰的语言总结了他的论点。「坚硬性和柔软性」(他说)、「坚固性和流动性，并不是物体的基本状态，而是取决于其组成中以太和可量测物质的相对比例。最富有弹性的气体可以通过抽出热量而变成液体，然后再变成坚固的固体，而这些粒子则会黏结在一起，其力正比于对热量的加强亲合力。另一方面，如果在最致密的金属中加入足够数量的相同原则，当它们膨胀成气态时，它们对金属的吸引力就会减弱，且凝聚力就会破坏。」』

在此详细地引用了大「异端」的非正统观点，只要在术语上略作改动就可以了；这位杰出的科学家不可否认的是一位富有独创性和自由的思想家，继续总结这些观点，并继续说到：——

『我不打算详细讨论这个理论所暗示的太阳力和地球的一体性。但我还可以补充说， 从这一点、或从假设力仅仅是运动、是没有实体的性质时，我们可以得出以下推论，来作为对这个最复杂、最深刻问题的最接近真理的结论 : ——

『 (a) 太空、星际间、行星间、物质间、有机物间并不是真空，而是充满了一种微妙的液体或气体，由于没有一个更好的词\*，我们仍可以像古人那样叫它**太阳之火** (Aith-ur )，即**以太** (AEther)。这种液体其成分不可改变、不可摧毁、不可见†，遍及万物和一切 \[*可量测的*—布拉瓦茨基夫人\] 物质‡，包括流水淙淙的小溪里的卵石、悬垂的树、在一旁观看的人，都以不同程度地充满了以太；卵石比树少，树比人少。这颗行星上的一切都以同样的方式如此充满它 ! 一个世界在空灵流体中建立起来，并在它的海洋中移动。

【\*的确，除非采用卡巴拉主义者的神秘术语 !】

【†只有在**显化期**时才是「不可改变的」，之后它再次融合进**原初质** (Mulaprakriti) ；其本质而永远「不可见」，但可见于它的反射光芒，被现代的卡巴拉主义者称为「星光界流质」(Astral light)。然而，有意识而伟大的**存在们**裹著同样**本质**在其中移动。】

【‡我们必须加上(可测量的)，以便把它与***以太***区分开来；后者虽然是一种基质，但仍然是物质。】

『(b)**以太**，无论它的性质是什么，都是来自太阳和各个太阳\*，太阳们是它的产生者、它的储存者、它的散播者。†

【\*神秘科学颠倒了这一说法，说它是太阳，且所有的太阳都来自它，它在**显化期**黎明时从***核心太阳***散发出来。】

【†在这一点上，我们坚决不同意这位博学先生的看法。让我们记住，这个**以太**不管是***阿卡莎***、还是它的更低等原则，**以太**都是七重的。***阿卡莎***在寓言中是**阿底提** (Aditi)，是太阳 (Marttanda) 的母亲，是「众神之母」 (Deva-matri)。在太阳系中，太阳是她的**菩提**和**载体** (Vahan)，因此是第六原则；在宇宙中，所有的太阳都是 **阿卡莎**的**欲体** (Kama rupa)，我们的太阳也是。只有当**苏利耶** (太阳) 被视为他自己**王国**中的独立**实体**时，它才是此大*物质*体的第七原则。】

『(c) 没有以太就没有运动；没有它的话，可量测物质的粒子就不能相互滑动；没有它的话，就不会有激发这些粒子行动的冲动。

『(d) 以太决定物体的组成。如果没有以太，物质的结构就不会发生变化；例如， 水只能作为一种紧实的、不可溶解的物质存在，超出了我们所能形成它的任何概念。它不可能是冰、不可能是燧石，也不可能是蒸汽，除了以太。

『(e) 以太连接了太阳与行星、行星与行星、人与行星、人与人。若没有以太的话，就没有宇宙中的交流；没有光、没有热、没有运动现象。』

因此，在所谓的宇宙的*机械*概念中，我们发现**以太**和*弹性*原子是宇宙的**精神**和**灵魂**，而该理论无论用任何方式伪装—总是把一个更广泛的问题留给科学家去思考，并超出了现代唯物主义所划定的界限—或者更准确地说，它是*不可知论*\*—比大多数更有帮助。关于**原子**、**以太**、或者两者而言，现代的推测都无法摆脱古代思想的圈子;而后者则浸透了古老的神秘主义。波动或微粒理论—都是相同的。它是从现象的各个方面进行推测，而不是对*起因*和*众起因*的本质的认识。当现代科学向其观众解释**本生** (avails) 和**基尔霍夫**(Kirchoff) 最近的成果时，在屏幕上显示了七种颜色，这些是光线按固定顺序分解的「原色」，并分别描述了光波的波长，但这证明了什么? 它甚至通过测量波长来证明它在数学成果上的*精确性*—『在光谱的红色一端大约是750奈米，在紫色一端大约是393奈米。』但是，当证实这种计算的准确性与光波的*效应*有关时， 科学被迫承认，这种*力* (其*假定的起因* ) *被认为*会在*某种*媒介中产生『无法觉察的微小波动』—『*通常被认为与以太媒介相同*』†—而这种媒介本身仍然只是—一个『假想的媒介! 』

【\*在我们的时代，残酷但坦率的唯物主义比双面的不可知论更诚实。***一元论***是现代哲学的伪君子，它把伪善的面孔面向了心理学和唯心主义，而把它本来罗马占卜者的面孔，用舌头鼓起他的脸颊 - 面向**唯物主义**。**一元论者**比**唯物主义者**更糟糕 ; 因为，当它同样从负面的角度看待宇宙和「心灵感应-精神性」的人时， 后者提出的论据要比怀疑**廷得耳**(Tyndall)先生、甚至是**赫胥黎** (Huxley) 先生的人更不具说服力。对于普世真理来说，**赫伯特·斯宾塞** (Herbert Spencer)、**贝恩** (Bain) 和**刘易斯(Lewes)**比**布赫纳** (Buchner) 更危险。】

【† 温切尔 (A. Winchell) 教授的《地质学》。】

**奥古斯特·孔德** (Auguste Comte) 对于未来不可能知道太阳的化学组成的悲观主义，并没有像声称的那样在三十年后被**基尔霍夫 (Kirchoff)** 推翻。光谱仪帮助我们认识到，现代化学家所熟悉的元素*很可能*存在于太阳的外袍—*而不是在太阳之中*；而物理学家们将这些「袍子」(太阳的宇宙面纱)视为太阳本身， 宣称其亮度是由*燃烧*和*火焰*引起的，并把那发光体的生命原则误认为是纯*物质性*的东西，称它为「色球层」。 \*到目前为止，我们只有假设和理论，无论如何都没有定律。

【‡参见 《神智学的五年》——文章 : 《开悟者们否认星云理论吗?》和《太阳仅仅是一个冷却物质吗 ?》—以得到真正的神秘教学。】
