# XII. 穿上新衣的古代思想

**现代科学是扭曲的古代思想**，仅此而已。然而，我们已经看到了**具有直觉的科学家在想什么**，在忙碌于什么；现在读者可以得到更多的证据，证明不止一个皇家科学学会院士 (F.R.S.) 正在无意识地接近被嘲笑的**秘密科学**。

关于宇宙进化论和原初物质问题，现代的推测不可否认是古代思想的延续，而通过最近的相互矛盾的理论*得到改进*。但整个基础归于希腊和印度的古代天文学和物理学，在当时通常被称为哲学。在所有雅利安和希腊的思辨中，可见到一种变即一切、无组织的、同质的物质或***混沌***的概念，现代科学家将其重新命名为「世界填充物的星云状态」。 **阿纳萨哥拉斯** (Anaxagoras) 在他的**同质论** (Homoiomeria) 中称之为「混沌」的东西，现在被 W.**汤姆森** (Thomson) 爵士称为「原初流体」。 印度和希腊的**原子论者**——**卡那达** (Kanada)、**卢克利普斯** (Leucippus)、**德谟克利特** (Democritus)、**伊壁鸠鲁** (Epicurus)、**卢克莱修** (Lucretius) 等等，现在都像一面清晰的镜子一样，反映在我们现代原子理论的支持者们的身上，从**莱布尼茨**的***单子***开始， 到 **汤姆逊**爵士的「漩涡原子」结束。\*诚然，古老的微粒理论被否定了，取而代之的是波动理论。但问题是，但问题是，*是否后者能够如同前者一样牢固地确立，而不会轻易被废黜呢？*《揭开伊西斯的面纱》充分讨论光的形而上学方面 : ——

【\*由*心灵*引发的**元素漩涡**并没有在现代转化后改进。】

『**光**是首先诞生的，是**至高者**的最初散发物，且**光**就是**生命**，福音传道者和卡巴拉主义者如此说。两者都是电——生命原则，是遍及宇宙的*世界之魂，*是万物的电活化者。光是伟大的变幻术师，在建筑师的神圣意志下\* (准确地说是*建筑师们*，被统称为「建造者」，且集体上是「一」) ，它无所不能的多样波动创造了一切形式和所有的生物。从它膨胀的电力胸怀中，涌出*物质*与*精神*。在它的光线中存在著所有物理和化学作用的开端，以及所有宇宙和精神现象的开端；它能活化也能导致混乱；它赋予生命，也产生死亡，从它的原始点逐渐出现了无数的世界、可见的和不可见的天体。根据**柏拉图**的理论，正是在***最初***母亲 (三合一) 的光线下，那「神」点燃了我们现在称之为太阳的火。』†它*不是*光或热的起因，而只是一个焦点，或者我们可以说，透镜，原始光的光线通过它得以物质化，且聚焦在我们的太阳系上，并产生所有相互关联的力。』

【\*我经常因为在《揭开伊西斯的面纱》中传达出*人格化*和拟人化的神的信仰而受到责备。这*不是*我的主意。从卡巴拉主义的角度来说，「建筑师」是***质点***的通称，即宇宙的**建造者们**，如同「普遍心灵」代表了即体上的**禅那主心灵**。】

【†《蒂迈欧篇》】

这就是***以太***，正如刚才**梅特卡夫** (Metcalfe) 的观点中所解释的那样，再次由**理查森** (Richardson) 博士复述，只是前者在现代波动理论的某些细节上有所保留。我们并不是说我们否定这个理论，而是断言它需要完善和重新整理。但在这方面，神秘教徒绝不是唯一的*异端*； 皇家科学学会院士**罗伯特·亨特** (Robert Hunt) 先生，在他《对光的化学关系研究》中说：——

......『波动理论不能解释他的实验结果。**大卫·布鲁斯特** (David Brewster) 爵士在他的《光学论文》中指出，「植物生命的颜色产生于 ... 这些物体粒子所受到不同颜色光线的特殊吸引，』且「正是由于阳光的作用，植物的彩色汁液得以生成，身体的颜色得以改变，等等...」他指出很难接受「这样的效果仅仅是由以太媒介的振动产生的」这一说法。他说，『在这类事实的驱使下，他不得不把光当作*物质* (？) 来进行推理。』哈佛大学教授**库克** (Josiah P. Cooke) 说，他『不能同意 ... 和那些把光的波动理论当作已确立的科学原理的人。』\***赫歇尔** (Herschell) 的理论认为，光的强度在每次波动的作用下，『与与发光体距离的平方成反比。』如果这一理论是正确的，那么它就会对波动理论造成很大的损害，甚至使之无法成立。他是对的，这被光度计的实验反复证明；虽然它一开始受到很多质疑，但波动理论仍然存在。』 (《揭开伊西斯的面纱》)

【\*《现代化学》】

对于**布鲁斯特**爵士的这句话——「不得不把光当作物质来进行推理」——有很多可以回答之处。光，在某种意义上，肯定具有和电本身一样的物质性。如果电*不是*物质，如果它只是「一种运动模式」，它怎么能被*储存*在**福尔** (Faure) 的蓄电池里呢? 赫**姆霍尔兹** (Helmholtz) 说，电必定有著和物质一样的原子性 ; 而**克鲁克斯**先生在他担任英国协会化学分会主席的演讲 (在伯明翰，1886 年) 中支持了这一观点。这就是**赫姆霍尔兹** (在他的《法拉第讲座》，1881 年) 说的：——

『如果我们接受基本物质是由原子组成的假设，我们就不能避免得出这样的结论 : 电不管是正是负，也被划分为明确的基本部分，它们的行为就像电的原子一样。』

在这里，我们必须重复在第九节中已经说过的话 : 从今以后，只有一门科学可以引领现代研究走向发现这全部、前所未知的真理的道路，而它就是最年轻的科学——如今改革后的*化学*。没有任何其他学科，包括天文学在内，能像化学一样能够如此准确地引导科学直觉。在科学界有两种证据可以证明这一点——两名伟大的化学家，他们各自都是自己国家最伟大的化学家，即**克鲁克斯**先生和已故的**巴特尔** (Butlerof) 教授: 一个是彻底相信异常现象的人；另一位在自然科学中表现出色，也是一位狂热的通灵主义者。很明显，当思考物质的最终可分性， 以及迄今为止*徒劳*追求负原子量元素时，那受过科学训练的化学家的思维，肯定会不自觉地被那些常常被蒙上神秘面纱的世界所吸引，朝著神秘的未知追寻，在那里浩瀚无垠的深处，似乎正在抵挡著过于物质主义的手的接近，它们试图揭开它的面纱。『它是未知和*永远不可知的*，』**一元论的不可知论者**警告说。不是这样 ； 坚持不懈的化学家回答说 :——『我们正在前进，丝毫不畏惧，因为*无知标榜著未知*。』\*

【\* **克鲁克斯**先生在**伯明翰**的《主席致词》中说 : 『只有一个未知的东西——***精神****的最终基础* (**空间**)。正由于其差异性而不属于***绝对者****和****遍一***，不管离物质感官多远，但始终可被人类精神性心智所接触，而这也是不可区分的整体的光辉。』——(神秘学实践课程)】

在他关于《元素起源》的演讲的最后几句话，这位杰出的科学家展示了他正在通往最伟大发现的皇家之路上。一段时间以来，他一直掩盖了「原始的*原质*」，并得出结论：『掌握了这个**关键**的人，将被允许解开一些关于创造之最深奥秘的谜题。』正如那位伟大的化学家所解释的 :——

『 ... *原质*是一个类似于原生质的词，表达的是化学元素演化之前存在的原初物质。我为此冒险使用复合词「早于」 (pro)和「制造事物的材料」( hyle)。这个词不是一个新创的词，因为 600 年前**罗杰·培根** (Roger Bacon) 在他的作品 《炼金术技艺》(Arte Chymiae) 中写道 : 『元素由「制造事物的材料」( hyle) 组成，每一种元素都转换成另一种元素的性质。』

**罗杰·培根**这位了不起的老魔法师\*的***知识***并非来自于的灵感，而是因为他研究古代的魔法和炼金术，掌握了解读真正含义的钥匙。且看看**克鲁克斯**先生是怎么说*原质*的，这接近于神秘学者无意识的***原初质*** (Mulaprakriti) ：——

【\*因此，此作品的作者在十年前《揭开伊西斯的面纱》(第一卷) 中说的似乎是有预见性的。其中有这样一句话 : 『许多这些神秘主义者，通过遵循一些论述教导的方式，秘密地将其从一代传承给下一代，*取得了在我们的现代精确科学时代也不会被轻视的发现。***罗杰·培根**，那个修士，曾被嘲笑为骗子，现在也被普遍归入秘术的「伪装者」之列；但他的发现却被人们所接受，且被那些最嘲笑他的人所使用。**罗杰·培根**在权利上和事实上都属于那个研究神秘科学的**兄弟会**。他生活在十三世纪，几乎与**大阿尔伯特** (Albertus Magnus) 和**托马斯·阿奎**那 (Thomas Aquinas) 同时代， 他的发明——例如火药和光学眼镜，以及他在机械方面的成就——被每个人都视为奇迹。他被指控与**恶魔**订立了契约。』】

......『让我们从最初元素出现的那一刻开始。在此之前，我们所知道的物质是不存在的。我们不可能设想没有能量的物质，也不可能设想没有物质的能量；从一个角度来看，两者都是可转换术语。在原子诞生之前，那些在物质相互作用时显现出来的能量形式是不存在的\*——它们仅仅作为潜在潜能被锁在*原质*中。在原子产生的同时，所有这些属性和特性，构成了区分各种化学元素和另一种化学元素的手段，开始存在并充满能量。』 (主席致词，第 16 页)

【\*正是如此 ; 在化学家和物理学家的实验室里『这些能量形式 ... *变得很明显* ... 』；但是，*还有其他形式的能量*与*其他形式*的物质结合在一起，这是*超感官的*，但为开悟者所知。】

神秘学者尊敬这位学识渊博的演讲者，但仍会用另一种方式来解释。他们会说，没有一个原子是「被创造出来的」，因为原子永恒存在于***遍一原子***的胸怀中——「原子的原子」——在显化期被视为「世界的*物质*起源子宫」(Jagad- Yoni)。**原初质** (Pradhana，未更动物质)，是***物质*** (Prakriti) 的最初形式，或可见物质性*与不可见*性质，这与**精神** (Purusha) 永远是一体的；它们只在沈睡期期间、以及超越任何意识存在层面时，是没有附加的性质或特征 (Nirupadhi)。正如现代科学所知，原子与精神是分不开的，但现在在科学上被称为「能量」。 *原质*原子没有被粉碎或细化 : 它只是进入了那个层面，那不是层面，而是超越幻觉层面的一切事物的永恒状态。**原质**和**精神**在永恒中都是不可改变 (Aparinamin) 和不可消耗的 (Avyaya)，或者 ; 在幻觉时期，两者都可以被称为可以扩展、逝去 (Vyaya )和消失，是「可改变的」 (Parinamin) 。在我们的概念中，在这个意义上，必须清楚地将**精神**与**梵**区分开来。然而，被科学称为「能量」或「力量」、且并被**梅特卡夫**解释为*双重***力量**的东西，实际上从来不是纯粹的*能量*，也不能只是能量 ; 因为它是世界的基质，是其灵魂，是*渗透一切的* (Sarvaga)，与时间 (Kala) 结合在一起。在**显化期**间，这三个是三合一，是潜在一切的**统一性**，在幻觉 (Maya) 层面上作用而成为三个不同的事物。在**奥菲斯派**哲学中，他们被称为**法涅斯** (Phanes)、**混沌**和**克罗诺斯** (Chronos)——是那个时期神秘哲学家的三位一体。

且看**克鲁克斯**先生与「不可知者」有多接近，以及在他的发现中，有多少接受神秘真理的「潜力」。他继续谈到原子的进化 : ——

『 ... 让我们在第一次完整振动结束时停下来，检查结果。我们已经发现了水、氨、碳酸、大气、植物和动物生命的元素，磷是大脑的元素，盐是海洋的元素，粘土是固体土壤的元素 ... 磷酸盐和硅酸盐足以满足一个世界和居民的需要，与我们今天所享用的并没有太大的不同。确实，人类居民必须过著比田园牧歌更为简朴的生活，而缺少磷酸盐则会对骨骼构成困扰\*... 在曲线的低端 ... 我们看到了一个巨大的间断 ... 这片绿洲，以及它前面和后面的空白，很有可能与我们的地球发展成为太阳系一员的特定方式有关。如果是这样的话，可能只有在我们地球上才会发生这种情况，而不是普遍存在于整个宇宙中。』

【\*神秘主义者声称，这只是存在于其他意识层面的世界。秘密科学教导说，原始种族是*没有骨头的*。(见第二卷) ; 此外，有 (对我们来说) 不可见的世界，与我们自己的世界一样有居民，也有**禅那主们**的*居民*。】

这证明了神秘学作品中的几个断言是正确的。

*首先*，『太阳和星星都不能说是由化学家熟悉的地球元素所构成，尽管它们存在于太阳的外衣中，还有更多迄今为止科学未知的元素。』

*第二*，我们的地球在其大气层的边缘拥有自己特殊的实验室，穿越这个实验室，每个原子和分子都会从它们的原初性质中发生变化和分化。

*第三*，虽然地球上没有任何一个元素不存在于太阳中，但有很多其他元素要么还没有到达，要么还没在地球上被发现。『有些可能在某些恒星和天体形成的过程中消失了 ; 或者，这些元素虽然存在，但由于它们目前的状态，可能还不能对通常的科学测试产生反应。』\***克鲁克斯**先生谈到了一种原子量比氢还要小的元素， 这种*元素*对于我们的地球来说*纯粹是一种假设* ... 虽然这大量存在于太阳色球层——*氦*。神秘科学补充说，化学所认为的*元素*中没有一种真的配得上这个名字。

【\*《神智学的五年》，第 258 页等。】

我们再次发现**克鲁克斯**先生赞扬『**卡内利** (Carnelly) 博士有力地论证了*所谓元素的复合性*，并从它们与复合基团的类比来支持这一观点。』迄今，在历史时期以及在所谓的文明国家里，只有炼金术成功获得一种真正的*元素*，或一种均匀物质的微粒，**即帕拉塞尔苏斯** (Paracelsus) 的**大奥秘物质** (Mysterium Magnum)。这是在**培根**勋爵的时代之前。\*

【\***克鲁克斯**先生在同一处说 : 『我们在化学中遇到的第一个谜是 : 「元素是什么?」 迄今为止，人们试图定义或解释某种元素，但没有一种能满足人类智力的要求。教科书告诉我们，元素是「没有被分解的物体」，它是「一种我们可以添加、但不能取走任何东西的东西」，或「一种随著化学变化而增加重量的物体」。这样的定义是令人双倍不满意 : 它们是临时的定义，明天可能就不适用于特定情况了。他 们接受他们的的立场，不是基于任何所定义事物的属性，而是基于人类力量的局限性 : 它们是智力上无能的自白。』】

『 ... 现在让我们看一下这一体系的上部分。以氢原子质量为1来说，几乎没有空间容纳其他元素，或许除了假设的***氦***元素外。但是如果我们「进入视镜中」，跨越零线寻找新的原则 - 我们在零的另一边会找到什么呢？**卡内利**博士要求一个负原子量的元素；这里有足够的空间和范围来容纳这样一系列飘渺非实质事物。**赫姆霍兹**说，电可能和物质一样是原子性的；难道电是负元素之一，而发光以太是佞一个吗 ? 正如我们现在所知，物质在这并不存在；目前在物质运动中所表现出来的能量形式，仅仅是潜在的可能性。*负重量的物质不是不可想象的。\**但我们对于一个物体与其他物体以负性质比例进行的结合，能否形成一个明确的概念呢？』 †

【\*讲师引用了**乔治·艾里** (George Airy) 爵士的话(在法拉第的生活和书信中，第二卷，354 页)， 『我可以很容易地设想，在我们周围有许多物体不受这种相互作用的支配，*因此也不受万有引力定律的支配*。』】

【†吠檀多哲学就有这样的想法；但它不是物理学，而是形而上学，**廷德尔**称之为「诗歌」和「小说」。】

『所描述的元素起源不限于我们的小太阳系，每个现在可见星星的能量中心，可能都会按照相同的一般事件序列进行。』

『在原子相互引力产生之前，无法施加任何压力；而在火雾球的外围 (其内皆是原质)，那些参与化学元素诞生的强大力量，会在这个壳层上全面发挥作用，剧烈的热量将伴随著足够的引力，阻止新诞生的元素飞逝至空间中。随著温度的升高，增加了膨胀和分子运动，分子倾向于飞散，且化学亲和力减弱 ; 但是，由于原子物质的质量所带来的巨大引力压力，超过了我简称为出生壳的范围，这会抵消热量的作用。』

『在出生壳以外的地方，由于温度高于所谓的化合物离解点，因而不会发生任何化学反应。在这个空间里，狮子和羊羔躺在一起；磷和氧不会结合而混合；氢和氯的化学键没有拉近的趋势；甚至连氟这种在过去一两个月里才被化学家分离出来的高能气体，也会自由漂浮而不结合。』

「在这个自由原子物质空间的外部，将有另一个壳层，在那里形成的化学元素会冷却到结合点，且会发生**马提乌·威廉姆斯** (Mattieu Williams) 在《太阳的燃料》一书中生动地描述的一系列事件 : 最终形成固体地球，且地质时代开始了。』 (19 页)

这里用了*严谨科学性*但又极其美妙的语言，描述了分化宇宙在神秘教导中的演化过程。这位博学的绅士在总结讲演时，使用的每一个句子都如同一道闪光，来自物质世界黑暗帷幕之外，照耀在精确科学领域，使之向了神秘学的至圣所迈进一步。 (见第十五章《众神、单体原子》) 他这样说:——

『我们已经看到了定义元素的困难；我们还注意到，许多领先的物理学家和化学家反对元素一词的普通含义；我们权衡了它们不太可能永恒存在\**，也不可能只是偶然起源*。作为另一种选择，我们认为它们的起源是一个进化过程， 如**拉普拉斯**(Laplace) 的天体进化，**拉马克** (Lamarck)、**达尔文** (Darwin) 和**华莱士** (Wallace) 的地球上植物和动物的进化。†在我们所知的元素的一般排列中，我们看到了与有机世界极为接近的情况。‡在缺乏直接证据显示任何元素分解的情况下，我们一直在寻找并找到了间接证据 ... 下面我们浏览了元素起源的观点；最后，我们回顾了**雷诺德** (Reynold) 教授提出的用他的周期分类法来阐述起源的方案。§ ... 综上所述，我们的确不能大胆地断言，*我们所谓的元素是由一种原初质演化而来的；但是我们可以争辩说，我认为，证据的天秤相当有利于这种猜测。*』

【§这种描述元素分类中周期定律的方法，是由都柏林大学的**埃默森·雷诺兹** (Emerson Reynolds) 教授提出的，正如**克鲁克斯**先生所说，他 ... 『指出在每一个周期中，元素的一般性质都各不相同，大致具有规律性，直到我们看到*第七个成员*，它与同一周期的第一元素或与下一周期的第一元素或多或少形成鲜明的对比。因此，氯作为**门德列夫**第三周期中的第七成员，与同一周期中的第一元素钠和下一周期中的第一元素钾形成了鲜明的对比；而另一方面，钠和钾是非常相似的。在原子量介于钠和钾之间的六种元素，它们的性质逐步发生变化，一直到氯，与钠形成对比。但是从氯到钾 (钠的类似物)，在性质都有飞跃变化 ... 如果我们因此认识到每个周期的第一个和最后一个成员在性质上——或多或少是确定的——的差异，我们不得不承认在每个系统中存在一个平均变化点。一般来说，每个周期的*第四个*元素，具有我们期望过渡元素会展示的特性 ... 因此，为了以图形化方式翻译，**雷诺**教授认为周期表的第四个成员，比如硅，可以放置在对称曲线的顶点位置。它代表那个特定周期，其元素性质变化方向是随原子量的增加。

现在，本卷作者谦虚地承认对现代化学及其奥秘的完全无知。但是她对于大自然中*类型和反类型的相应关系*非常了解，且神秘主义的基本定律是完美的类比。因此，她大胆地说出了一句令每一个神秘主义者震惊的话，尽管这句话可能会遭到正统科学的嘲笑。这种解释元素行为周期定律的方法，不管在化学中是否仍然是一种假设， 在*神秘科学中是一种法则*。每一个博学的神秘学家都知道*第七*和*第四*成员——无论是在世界的七重链中，天使的七阶层中，还是在人、动物、植物或矿物原子的结构中——我们会说，在几何和数学上均匀的不变自然法则的运作中，*第七*和*第四*成员总是在七重体系中起著独特而特殊的作用。从天上闪烁的星星，到野蛮人在森林里燃起大火迸出的火花；从**禅那主**的阶层和基本构成来看——组织成更神圣的理解和更高的感知范围所，是西方心理学家未曾梦想的，甚至到大自然最卑微昆虫物种的*分类*；最后，从世界到原子，宇宙中的一切，从大到小，都在循环地、七重地进行著精神和物质的进化，它的第七和第四 (后者是转折点) 表现出与原子周期定律相同的行为。大自然从不飞跃行事。因此，当**克鲁克斯**先生对此评论说，他『不希望将**门德列夫**的表和其图示(原子演变的图表)中的间隙，推断为必然有实际存在的元素来填补间隙；这些间隙可能只是意味著在元素诞生的时候，有一个潜力易于形成适合此处的元素』——神秘学家会恭敬地对他说，后一种假设只有在原子的七重排列不受干扰的情况下才成立。*这是遍一的法则*，也是一种绝对正确的方法，遵循它的人一定总是导向成功。】

因此，归纳性的科学在其天文学、物理学和化学分支中，在踌躇前进、探索大自然在我们尘世上最终作用上的秘密时，退回到**安纳萨哥拉** (Anaxagoras) 和迦勒底人的时代，以发现 (a) 我们的现象世界的起源，以及 (b) 构成宇宙物体的形成方式。他们不得不从早期哲学家的信仰和体系中，来寻求宇宙起源的假设——这些体系都是建立在关于原初质的性质、功能和法则的普遍秘密教义之上——我们难道没有权利希望在不久的将来，科学将比迄今为止更好地欣赏古人的智慧吗 ?

毫无疑问，神秘哲学可以从精确的现代科学中学到很多东西；但另一方面，后者可以从古代的学问中得到多方面的好处，主要是在宇宙进化论方面。例如，在星际空间充满了许多*不可测*物质，它们的在神秘上、炼金术和超验上的意义，它们相互渗透，在低层面作为自然现象的直接起因，通过 (所谓的) *振动*显化。简而言之，对于**以太**或***阿卡莎****真实*性质的认识 (而非假想) ，以及对于其他奥秘的理解，才能真正导向对**力量**的认识。正是这种基质引起了物理学唯物主义派的强烈反抗，尤其是在法国，\*而且精切科学必须进行辩护。他们要想除掉它，就得冒著像现代**参孙**一样的风险，拆掉了**科学神殿**的柱子后被埋在它的屋顶下。

【\* 一群电工刚刚对波恩大学著名教授**克劳修斯** (Clausius) 的新理论提出抗议。抗议的性质表现在签名上，写著『**朱尔斯·布尔**丹 (Jules Bourdin) 代表电工团体，该团体曾于1881年有荣幸与**克劳修斯**教授会面，而他们的战斗口号是「推倒以太」。』他们想要**普遍的*虚无***，你看到了吧！】

那些认为**力量**并非独立于*纯粹简单****物质***、或独立存在于*其外*的理论，都已被证明是错误的。它们无法涵盖全部情况，且许多科学数据被证明是*不科学的*。《往世书》中说「以太产生的声音」，而这种说法被嘲笑。我们被纠正「声音是*空气中*的振动」。什么是空气 ? 若**空间**中没有以太媒介来支撑它的分子，它会存在吗 ? 事情就是这样。唯物主义无法承认物质*之外*有任何东西的存在，因为一旦接受了一种不可量测的***力量***——所有物理**力量**的来源和源头——其他*智性***力量**就不得不被承认，而这将引领科学到很远。因为它将不得不承认，人类身上存在著一种更为精神性的力量——这完全独立于物理学家所知的任何一种物质。因此，除了**空间**的假想以太和粗大物质体外，在唯物主义者看来， 整个**恒星空间**和未见**空间**本质是一个无边无际的*虚空*——盲目、无智性、无用的。

现在的下一个问题是 : 这种**宇宙基质**是什么 ?一个人能多大的程度推测其性质，挖出多少秘密，从而觉得有理由给它取个**名字**? 特别是，现代科学在揭开这些秘密方面已经走了多远，它正在做些什么来解决这些秘密。科学的最新爱好「星云理论」，可能给我们这个问题提供一些答案。让我们接著检查**星云理论**的证据。
