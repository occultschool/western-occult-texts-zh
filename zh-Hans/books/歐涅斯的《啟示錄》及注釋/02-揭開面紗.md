# 揭开面纱

以至圣者之名。<br/>
我的灵立于上帝宝座之前，<br/>
在至高者前的光中移动；<br/>
祂的眉宇隐没于无垠之中，<br/>
祂的双足安息于太阳之上。 <br/>
诸天在祂面前如地毯铺展； <br/>
祂身披闪电之焰； <br/>
黑暗与神秘的寂静 <br/>
在主面前环绕、笼罩著。 <br/>
祂独自处于光中—— <br/>
独自于遥远的天域。 <br/>
独自——独一，<br/>
无受造物与至高者共享黑暗的奥秘。 <br/>
所有的力量，所有的王子，<br/>
宇宙所有蒙福的灵， <br/>
所有伟大的能量与灵质， <br/>
皆立于上帝宝座之前。 <br/>
他们以群星的节奏向祂颂歌， <br/>
在元素之球永动的光辉中， <br/>
欢欣于祂本质的荣光， <br/>
在主流溢的美丽中喜乐。 <br/>
此时声音自宝座传来， <br/>
深沈如雷、如恒久旋律； <br/>
明亮如闪电，柔美如长笛轻吹。 <br/>
那是上帝的声音； <br/>
我认得它——我灵在灵中认得它： <br/>
那是父与王的永恒声音， <br/>
祂一开口，宇宙便静止。 <br/>
声音说道：「人子啊！」 <br/>
我回答道：「看哪，我在这里。」 <br/>
声音又道：「我是主，你的上帝，是我创造了你， <br/>
并将我的火焰吹入你的胸膛。 <br/>
我差遣你到世上，向世人宣讲 <br/>
真理、光明与智慧的话语； <br/>
去复兴古老的圣洁； <br/>
去揭示我所显明之事：<br/>
在晨曦荣耀行走之处，<br/>
我以太阳的闪电托举你； <br/>
在黄昏时分群星沈落之地， <br/>
我以智天使的翅膀载你； <br/>
我揭开奥秘的帷幕， <br/>
向你的眼睛开启诸多秘密； <br/>
将我的雷霆置于你火焰之手， <br/>
去击碎迷信的神殿。 <br/>
如今，回答我， <br/>
你是否已完成庄严的使命？ <br/>
你是否已从宇宙阴暗的结构中 <br/>
投下一块石头？」 <br/>
声音止息，寂静降临； <br/>
那沉默刺透我的灵魂； <br/>
一种可怖的静默笼罩我的灵； <br/>
我惶恐、羞愧、低头不敢仰视。 <br/>
不久，声音再次响起； <br/>
对我说：「人子啊！」 <br/>
我回答说：「看哪，我在这里。」 <br/>
声音说：「回答我。」 <br/>
我俯伏在宝座之前； <br/>
死亡的寒意贯穿我全身； <br/>
那神秘寂静的威严 <br/>
冻结了我灵魂深处的灵。 <br/>
「至圣者啊，我比虫还卑微； <br/>
我是满载罪孽的卑贱罪人； <br/>
我曾迷失于黑暗的云雾中； <br/>
我曾行走在虚妄与死亡之路； <br/>
我偏离了应与祢同行的正道； <br/>
我由尘土、云雾与灰烬所造， <br/>
追逐著尘土、灰烬与腐朽。 <br/>
主啊，尘世与尘世之物， <br/>
使我的灵远离了祢； <br/>
上帝的声音在我不安的灵魂中低语，<br/>
我却充耳不闻。 <br/>
我将心思系于必朽之物，<br/> 
系于终将消逝的事物； <br/>
却鲜少将漂泊的思绪安放于<br/>
永恒不灭的王。 <br/>
然而如今，主啊，如今我将觉醒， <br/>
如今我将诉说； <br/>
愚昧啊，离开我吧！ <br/>
从今以后，我是至高者的仆人。」<br/> 
我说话时仍俯伏在地， <br/>
因为上帝的奥秘如山压在我身上；<br/> 
一种深沈、无可抗拒的敬畏； <br/>
如死亡般昏厥之影。 <br/>
此时诸天界传来乐音， <br/>
柔和而神圣的和声； <br/>
远处的力量 <br/>
以金色乐器甜美颂唱： <br/>
一道如紫红宝石的光芒， <br/>
缓缓降临在我头顶； <br/>
如同荣耀的冠冕将我覆盖， <br/>
宛如彩虹横跨大海。 <br/>
当我伏地时，听见号角响起， <br/>
孤独而狂野的爆响； <br/>
使我的灵震颤， <br/>
仿佛被雷霆劈为两半。 <br/>
一道光的漩涡将我向上卷起； <br/>
是一道闪耀、颤动的火焰旋涡； <br/>
宇宙间所有的光辉 <br/>
以斑斓色彩将我环绕。 <br/>
上帝是无限、全能者， <br/>
将我升至群星的火焰基质之中； <br/>
我见到了太阳的壮丽； <br/>
行星合唱团环绕著舞蹈。 <br/>
多彩美丽的晨曦 <br/>
照亮金色的拱门； <br/>
天界七条纯净的河流 <br/>
在阳光下流淌闪耀。 <br/>
自然的创造者映入眼帘， <br/>
那神圣辉煌的天界之神； <br/>
其临在如春夏般温暖； <br/>
是乐园中生长的生命之树。 <br/>
如同月亮著伴随旅人， <br/>
在午夜孤行之时； <br/>
她以柔光安慰他， <br/>
仿佛在他身旁美丽地滑行： <br/>
然而她依旧静止遥远， <br/>
高坐于苍穹深处； <br/>
上帝也如此引领我， <br/>
向上，进入广阔的天界。 <br/>
我未见祂，却知祂在； <br/>
我未见祂威严的形象： <br/>
祂远在自己的领域； <br/>
是神圣临在的中心荣光。 <br/>
其面容不断展现新辉， <br/>
如同天际变幻的美丽； <br/>
紫色、白色、玫瑰色、银色； <br/>
以及夜空星火的深邃浓烈； <br/>
上帝永恒闪耀的面容也是如此， <br/>
在我眼前一掠而过； <br/>
却如转瞬即逝的梦境， <br/>
祂在其中闪耀，却又自隐其身。 <br/>
我被火焰托举前行， <br/>
见到一位太阳的辉煌天使； <br/>
他高声宣告， <br/>
主旨如下： <br/>
「向你致敬，你是持斧罗摩， <br/>
在美丽的金色原野中； <br/>
是闪电与真理所拣选的孩子；<br/> 
我久已盼望你的到来。 <br/>
向你致敬，你是伊玛目·马赫迪， <br/>
被天言烈焰所环绕： <br/>
我久自群山遥望， <br/>
只为见证你降临的阳光。<br/> 
如今你升至上帝支撑宇宙的柱石，<br/> 
包裹在黑暗闪电中， <br/>
在烈焰迅疾翻滚的雷霆中。<br/> 
这些高耸柱石<br/>
从寂静之水的深渊中升起 ；<br/> 
黑暗笼罩其可怖的胸膛； <br/>
无数伟大的灵引导其广阔的力量；<br/>
进入山岳， <br/>
深处涌出这些海洋之水； <br/>
进入风的怀抱中安息，<br/>
比阳光中的尘埃还要渺小。 <br/>
进入那静止而庄严的旋风，<br/>
其微弱的气息诞生这些风； <br/>
旋风远在彼方， <br/>
安居于全能上帝的怀抱。 <br/>
这些柱石被称为真实； <br/>
这些水被称为古老； <br/>
这些山被称为坚强； <br/>
这些风被称为永恒； <br/>
在其中，你将成为真理的见证，<br/> 
见证神圣之光——上帝的气息； <br/>
在你狂喜的目光前<br/>
将揭开宇宙的奥秘 。」 <br/>
随即闪电划破长空， <br/>
雷声回应了闪电； <br/>
太阳的天使离去； <br/>
我继续前行飞翔。<br/>
看到世上无人能见的群星， <br/>
在金色光域中闪耀； <br/>
美丽且迅疾舞动； <br/>
不可思议，唯有思想可比拟。<br/> 
黑暗环行的广阔， <br/>
超越人类一切想象； <br/>
太阳光的七芒星球亦在其中，<br/> 
唯有上帝能丈量其伟大轨迹。 <br/>
苍穹在远方延展， <br/>
如同金与绿柱石筑成的墙；<br/> 
在最深的夜、火焰与星辰的掩盖中，<br/> 
我见到上帝荣耀的永恒宝座。 <br/>
天界的诸灵被光芒包裹， <br/>
美如阳光下安歇的花朵； <br/>
那里没有昼夜的更替， <br/>
唯有上帝恒久的荣耀照耀一切。<br/> 
「听我说吧，美丽的星球， <br/>
倾听我，天堂的孩子们； <br/>
还有你，时间，永恒者的象征， <br/>
你，地球，侧耳倾听； <br/>
因为无垠的威严向我开启， <br/>
展现宇宙浩瀚的本质； <br/>
诸天与恒转太阳的炽烈光辉，<br/> 
广阔的大地与无数的海洋星球。 <br/>
在光中——在火中——在雾中——在焰中， <br/>
我如乘日光漂浮， <br/>
屏息于炽天使的狂喜中；<br/> 
晨曦的骏马载我前行。 <br/>
那是什么光芒——什么荣耀在远方绽放？<br/>
是何辉煌在太阳中闪耀？<br/>
<br/>
她起身——她微笑——她站立无遮——我看著——我看到了一切。
