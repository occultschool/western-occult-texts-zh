# 专注本体：与宇宙心智合一的修行之道

## 深入解析专注本体的实践要点与心智合一的高阶意识，说明以空白开始到内在大师形象的实践路径。

![cover](/img/downloaded/focus.png)

K.H.大师论专注于本体并与之结合

去专注在「本体」的概念上，  一开始可以先花个半小时的时间，不允许其它想法进入。渐渐地，便能把意识与本体结合起来。本体总是要在内在寻找的。向外看是一个致命的错误。你要努力达到所能达到的最高境界，并将自己保持在那里。在返回肉体的最后一刻，才会觉察到此更高层次意识，而此时星光体也通常是活跃的，因此经常产生双重意识，这正是错误的来源。欲获得专注力，第一步是保持空白。 然后，逐渐地产生意识，最后，这两种状态之间的转换变得迅速且容易，以至于几乎不会注意到。 在专注的时刻，身体处于出神状态中，并保留了一种梦境的感知（当然这只适用于弟子）。所要克服的巨大困难，是如何在物质层面上记录对于本体的认知。要达成这一点，必须使肉体大脑对于一切完全空白，而只意识到「高层意识」；而星光体必须被麻痹，否则会导致错误和混乱。首先，试著让自己处于某种状态，使你对身体经历的一切都不再有所感觉；事实上是把自己与身体分开。若在此尝试过程中，你感觉到任何东西、任何外来的影响从外面进入，应立即中断专注力。最好的方法是专注在大师上，也就是你内在活跃之人。在你内心形成其形象，成为你专注的焦点，以便在一心一意中失去对身体存在的感知。在专注的过程中，必须尽可能积极对抗幽灵和所有星光界低等影响，而尽可能接收大师的影响。心怀大师最能对抗幽灵。 对于通灵者而言，面对的危险比其他人要多，因为其身体更敏感，吸引更多其他自然力量。幽灵和星光界的影响会通过星光体对肉体起作用。 星光体是肉体的本质；但若无肉体， 星光体也无智能和意识。肉体感官的本质是星光体感官，肉体感觉的本质是星光体感觉。自由感是高等意识的显著特征之一，而使身体安静下来所需的意志努力，与忘记痛苦所需的意志努力基本相同。人人在结合本体的经验上都不相同。真正的本体是《道路上的光》中提到的「勇士」； 本体从不在此层面上行动，「心智」才是此层面活跃的代理人，是真正的参与者。这种与本体的结合是消除「分离感」的最佳手段，因此一个人必须成为本体的奴隷。本体是无形状的、无性别的、无形体的，是一种意识状态，一种气息，而不是一个身体或形式。其最高等的形体是幻身，指的是整个人减去肉体和生命。其形体的样貌如同肉体，除非被意志改变。本体是完美的□，「心智」(幻身)加上「菩提」和「阿特曼」，共同构成一面镜子，反映「梵」，也就是「至一本体」。本体是宇宙灵的一个火花-「阿特曼-菩提」-是普遍的、永恒的、在此层面上无感知的。高等心智是心智，当上升到「阿特曼-菩提」时，就完成了三位一体，这就是至一。心智是自我意识，受限于一个大显现期 。菩提则会从大显现期延续至下一个大显现期。幻身是中间的自我，是开悟者所使用的身体，并以欲体作为载体。这是人类之魂，是情绪、情感以及思想的所在地。当在身体里时，便是心智。这七项原则是同一实体的不同方面，而其形体和特征取决于意志。在19世纪，人类的本能（欲望）在很大程度上被心智或智能的发展所压制。心智是堕落的天使，是「心智之子」所吸入的本质，而「心智之子」共同构成宇宙心智或显现的逻各斯。单体，即「阿特曼 - 菩提」，其实根本不属于这个层面，可以说是在梵层面。因此，心智是人在此层面上最高原则，正是这个心智使人成为神灵或魔鬼，因为神圣的单体通过心智在此层面上发挥作用，或者说，当「心智」在此层面上产生作用时，乃是借由影响「菩提」较低部分中的「神-生命」之力而实现的。大师是那些生来就有应身的人。你们每个人都能为自己创造一个大师，在星光界中赋予他客观存在。 若他是真正的大师，他将发出自身的声音。若他并非真正的大师，那么声音将是本体的声音。每个人都会根据自己的内在功德和发展来接受。
