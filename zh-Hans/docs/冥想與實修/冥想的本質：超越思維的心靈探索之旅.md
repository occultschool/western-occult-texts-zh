# 冥想的本质：超越思维的心智探索之旅
## 探索意识的深层次，从日常生活中提取内在智慧

![cover](/img/downloaded/62f82e278971810226b36ca89b3374f2.png)

一则简要指引

「冥想，是将思绪溶入永恒纯粹的意识，超越表象；是知晓，而非思考，让有限回归无限。」——伏尔泰

有天，侄儿尚向我提问如何开始冥想，想我推荐几本入门书。本想寄书给他，翻了几本，却觉不妥。有的太浅，有的深陷宗教窠臼。于是决定亲自写，简述原理与实效，附上步骤，或许更为实在。冥想种类繁多，我选择专注自己四十多年来反复练习的一种。

以下是我试图解释冥想本质与过程，力求清晰，避开无谓的宗教与神话牵缠。当然，冥想形式各异，有人深信必须奠基于宗教神话，且离不开其文化母体。我不以为然。我相信冥想能惠及众人，不分国族、种族或信仰。

为何？冥想是探索意识的过程，观察如何显现与流变。冥想者只是在体验意识的诸般样态。注意力如一名船长，在广阔意识海域中定舵，引领航者驶向遥远未知的岛屿。意识宇宙里风景无尽，我们却鲜少花时间精力真正探入其中。头脑是模拟器，经漫长演化，人得以藉它设想现实中尚未发生之事。正如纽约市立学院的物理学家加来道雄在其近著《心智的未来》所说：

「人类意识独特，能构建世界模型，并置于时间轴上模拟——亦即透过评断过去来演算未来。这需要调节与评估大量回馈循环，方能形成决策，达成目标。」

杰出的诺奖得主、大脑研究先驱杰拉尔德·埃德尔曼认为，意识有两个基本层次。第一是「当下」意识，即对外界刺激与周遭环境的知觉，几乎所有鸟兽（包括人类）皆有。它引发联想，引动注意。第二层次，可说唯有人类完备体现（部分高等哺乳动物略有雏形），是自我反思与抽离，演变为对过往的回顾、沉思与重构（著重「重」字），以及对未来的设想与规划。若第一层专注当下，第二层便是我们陷入白日梦、幻想与放空之时。总体而言，它像个虚拟模拟器。

如加来道雄所阐明：

「人类是唯一理解『明天』的动物。与他者不同，我们不断自问：几周、数月乃至数年后，会是怎样？因此我认为，第三层意识（埃德尔曼称第二本性）创造了模型，评估自身在世界的位置，并透过粗略预测来模拟未来。」

加来道雄的时空意识论与埃德尔曼的双重本性论，确是理解冥想有效运作的宝贵参照。

冥想实修涵盖多层，其核心在将注意力引回当下，觉知此时此地正发生的一切。为此，需在第二本性与第一本性间搭桥，让我们远离喧嚣，与当下和谐相处。这并不容易，因我们大脑已演化得善于逸出当下，设想种种情境，即便那些情境与此刻无关。

观察自己头脑如何不息运转，便知我们多易随波逐流。无论是关注视觉、听觉与触觉领域的流变之物（如搭机飞行），或是白日梦般设想未来生涯（如坐在教室听无聊演讲），意识总如野兽，不肯久留一处。这颗「猴子心」从未真正歇息（即便睡眠时），持续运转，耗费心神。

令大脑沉静并非易事，但若能达成（哪怕只是片刻），不仅使人神清气爽，更开启一扇门，让我们体验意识诸多迄今被忽视的面向。第一与第二本性，是更深更广的意识波浪。冥想过程，便是寻找意识最初萌生的源头，并亲见它如何超越我们熟悉的清醒与梦境。

凭个人经验，我可以坦率地说，这趟内在探索带来无数美妙收获，最重要的是对心智实际运作有了更深洞见，而随著深入意识本身，幸福感也随之增强。当然，我们必须谨慎解读内在旅程所见。人类似乎有种近乎遗传的倾向，将自己的大脑状态与宇宙的「真实」状态混为一谈，并对自己最新的「顿悟」过早定论。

宗教或其他意识形态，常常试图占据我们的内在体验，将其与自身神学体系交织，如此过早的遮蔽了我们的眼界，无法接纳其他诠释。所有冥想实践的共通点，终究是人类大脑（无论其背景是佛是耶稣或世俗人文主义），幸而，大脑与任何宗教或地域并无必然牵系。

但究竟如何冥想？

在家、办公室或外头找个安静角落，最好是你常待之地。衣著宽松舒适，别太紧绷。备好垫子或椅子，能让你安坐良久。坐直不僵，手臂双手轻放腿上，尽量避免无谓移动。开始前，最好定下固定时长。初学可短，如十分钟。日后渐增。

冥想关键，在于运用技巧让忙乱的大脑慢下来。有人建议观察呼吸，有人主张保持观者立场。我发现持诵咒语、重复短句或名字相当有效，因其有助凝神。选用哪个名字或短语全凭个人，一旦选定，最好坚持。方法之效不在词语本身，而在于它们帮你集中与专注。

闭上眼睛，尽量保持安静，不刻意移动身体任何部位。但也不必强抑不动；可借鉴街头艺人，他们能长时间维持静止。

刚一坐下，你立刻察觉脑海展开连串幻想与喧哗。我将其比作投影信囊，一旦捕获你注意，便如搭上头等舱，瞬间将你带离原点。这般投射无穷无尽。

试试能否避开它们的引力。起初极难，因为大脑的本能便是领你滑入这些想像的漩涡。

若能避开幻想，并将注意轻轻放在眼窝后方（切忌用力），你便开始觉知内在。

这体验可能令人惊异，恍如盲者初见虹霓，或聋者乍闻莫扎特。

随著冥想深入，几种情形或会浮现。其一是入睡，常成潜在问题，尤其当人未获充分休息时。其二是惰怠状态，感觉空无一物，或对整个过程生厌，便起身觅食、开电视，或在社交软体更新状态。其三是进入半睡半醒的有意识状态，类似清明梦，差别在于人能半清醒地进入。这阶段引人入胜，因你可完整看见整个景象，栩栩如生，宛若外境。譬如一个苹果，能清晰浮现眼前，其细微纹理历历在目，事后犹可回忆。这是冥想的一种附加状态，介于清醒与睡眠之间。

这些状态与类似状态皆是过渡，可预期会在冥想过程中出现。但这些并非进展，也未必带来幸福，不过是清醒与做梦的大脑在知觉上的局部变动。

当一个人能保持专注，避开睡意、超现实幻境与无聊懈怠后，某些显著变化便悄然浮现。身体开始感到一种奇异的麻木，迥异于久坐后腿脚的痹感，虽略令人心惊，却也伴随愉悦。当这麻木感笼罩全身，你会感觉仿佛踏入一种高度觉知状态，一个被遗忘的世界正徐徐展开。内在感官变得锐利，黑暗为光明取代。

此时，注意转向内在，开始享受这片提升之境带来的一切。不仅视觉更敏锐，听觉与幸福感亦然。你会看见、听见全新事物，恍如初次踏入陌生国度——这正是冥想与内观的馈赠。新的异象与声响在新的意识层面浮现，这不该被误认为梦境或清醒，因其不属任一。这就像冒险踏入源头之地，所有虚拟模拟皆源于此。或是像进入电影院，不看银幕，而是转身观察放映机本身；不看移动影像，而是凝视影像的源头。

这可能相当令人不安，因我们的大脑惯于投入种种模拟。极少有人能避开这些情境涌现的引力，但这好比鳟鱼需溯流而上，才能追寻显现的源头。

一旦抵达这新的意识水平，此时所经历的事物，似乎比以往任何时刻都更为真实。然而危险在于，处于新状态下，人可能开始为不同的洞见所迷惑，滋生妄自尊大之感。若想再进一步，关键在于保持冷静与超然，避免过分迷恋或执著于这些内在的缤纷。说来容易，行之甚难。

此处的光芒能抚慰观照者，令思绪沉静下来。然而这光，如同物质世界的光，亦可拆解为一列光谱；若不细察，便可能使人引至虚构，陷入封闭难喻的世界。好比沉入梦境的情节，忘了所追寻的，不过是自己投下的影子。

起初，内在听闻的声音或许粗糙，毫不动人；渐渐地，却流露精致的特质，散发难以抗拒的魅力。这股力量推著人往内心深处、往高处去，宛如超音速的航程。声音愈是精微，愈令人沉醉。此时的冥想者，可比作冲浪的人，渴望知道自己所乘浪涛从何生成。顺著光与声的流向追寻，便能寻得其源头（不为途中幻象所惑），更深地融入无客体的意识之中。我们开始清醒地看见：幻象如何如浪起伏，而深海始终平静。意识本身并无形体，是看不见边际、难以定义的主体。

冥想犹如显微镜，让人窥见意识的本质；也像望远镜，瞥见意识如何在时间中展开。然而这两种探看能及多远，端看一个人愿意投注多少光阴向内凝视。

就实际而言，持续冥想——例如每日定时修行——比零星的尝试更有益。因绵延的静观常在心底凿出更深的通道，助人更明白思绪如何模拟运转。

## 结论

总之，冥想三分钟或三小时，皆有益处。正如任何运动总胜过不动，冥想亦然。请记得：你所体验的世界，正是由你的意识所孕育。深入的探问这意识如何生成，是一趟惊奇的历险。至于它终将引我们至何方，以及我们如何诠释这趟旅程——是透过神经科学或神秘学的镜片——都取决于自己。

「内省之奥秘，在于它永无止境。」——菲利浦·金德里德·迪克
