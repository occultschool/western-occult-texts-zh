# 自杀的人会发生什么事？
## 企图脱离生命，却发现生命仍在；苦难未尽，反添新愁。

![cover](/img/downloaded/52089c33266b8aa2a2c11813e8e833ca.jpg)

Photo by [Bianca Berg](https://unsplash.com/@biancaberg?utm_source=medium&utm_medium=referral) on [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral)

『自杀乃诸罪之极，其果可畏。』—— 布拉瓦茨夫人  
『……生者所犯之最大罪愆，莫过于自杀。』—— S.B.大师  
『自杀者并未死去，只是脱离其三层较低的躯体……他们愚妄地想逃离生命，却发现自己依然活著——此后，无尽苦楚候于前路。所谓惩罚，视其受苦深浅而定。』—— K.H.大师

自杀者将受到惩罚，因其死亡非自然，而是强行催逼。纵然生前的遭遇可怜，令人同情，但此举在灵性层面依然悖逆，因为这并非寻常自然的死亡。

自杀者神识将完全清醒，困于「欲界」（梵语 Kama Loka，意即「欲望与情感之境」）。此乃环绕并渗透物质层面的星光界氛围。他们在此处能看见尘世，看见自杀当时相关的人事，且必须一直停留，直至命中注定的阳寿终了。

譬如一人业力注定享寿九十，却在二十岁自尽，他便得在欲界滞留七十年，其间不得寸进。他们无法经历完整的死亡过程，进入「天界」——或称天堂的状态。

对此，K.H.大师解释：「每一波生命进化的浪潮，终须拍上属于他的岸。」他们真真切切地困住了，此种状态带来的痛苦，远甚于他们在世时意图逃避的苦难。

他们必须在欲界度过余命，并非意味著某个更高权威或神圣存在施以形式化的「惩罚」。每人皆由七个部分构成（神秘学称「七大原则」），因此，依据不可变易的自然法则，这七部分须依循正确方式、次序与时机逐渐分离，死亡过程方能自然顺遂。

自然死亡者，各部份依据其命定历程，走完全程，并依其意愿，松脱彼此连结。但亲手终结自己生命者，显然并非如此。

他们会发现自己仍然活著，只是失去了肉体躯壳，处境比从前更加困顿。他们往往满怀悔恨，渴望再度接触尘世生活，因此极易主动联系灵媒，或任灵媒寻至。

这却是他们能做的最糟之事。这些蓄意自杀者，意图重新连结世间生命，在灵性上实属非法。大师们与布拉瓦茨夫人教导：如此行事的自杀者，待其命定寿终之时，往往因此永远丧失灵魂。而那些促成、容许此事发生的灵媒，也将面临黑暗命运，为自身造下可怖业力。

若能让逝去的灵魂不受侵扰，安稳前行于最终的上升之路，该有多好。述尽这些之后，K.H.大师凄然写道：「此刻你或能明白，为何我们如此强烈反对灵媒与通灵。」

十九世纪末，濒死体验尚不为人所知。直至近几十年，此事方获认识，并有严肃的科学研究随之展开。

雷蒙·穆迪博士在名著《死后的世界》中，采访了那些自杀未遂、濒死获救之人。他写道：

「这些经历皆属不愉快。如一位妇人所说：『你若带著一颗受苦的灵魂离开，到了彼处，灵魂依然受苦。』简言之，他们发现自杀企图逃避的困境，死后并未消失，反而更添纠结。失去肉体之后，他们既无力改变自己酿成的困局，亦必须直面行为招致的苦果。一名男子因丧妻之痛举枪自尽，获救后说：『我没去到妻子所在之处，却在可怕的地方……当下便明白自己做错了……真想从未如此。』另一些人亦经历类似不快的『灵薄狱』状态，感觉『仿佛要在那里停留漫长岁月』。这是对『破坏规则』的惩戒——他们企图从应尽的『任务』中提早脱身，未能完成生命该有的章节。」

穆迪在后作《对生命后的思考》中进一步指出：

「这些人都认同一点：自杀解决不了任何问题。他们在彼岸面对的困境，与生前意图逃避的毫无二致。无论是何种艰难，都依然存在，未曾化解。有人提到，自己仿佛『困』在当初引发自杀念头的情境里，不断循环重演。」

西尔维亚·克兰斯顿在《轮回——科学、宗教与社会的新视野》中，记述了精神科医师乔治·里奇濒死经验里的见闻。书中有一段格外沉重的记载，引自里奇本人的笔记：

「据乔治·里奇所言，自杀者最悲惨的命运之一，便是死后得见自己毁灭之举，是如何使亲友悲痛欲绝。里奇在濒死时，被天界的引导者带到一幢屋中。

他看见一名青年尾随老人，从一房跟到另一房。『爸，对不起！』他在老人身后反复说道：『我那时不知这对妈打击这么大。真的不知道。』

里奇能清楚听见青年的声音，老人却毫无所觉。老人端著托盘走进卧房，老妇坐在床上。『对不起，爸。』青年又说：『对不起，妈。』道歉无休无止，向著一双听不见的耳朵。我困惑地转向引导者。尽管能感受祂对屋中景象涌流著怜悯，我仍无法理解。

后来几次，我们停驻在相似的场景前：一个男孩拖著十几岁的女孩穿过学校走廊，『对不起，南希！』；或是一名中年妇女向白发男子乞求宽恕。里奇急切地问引导者：『为何他们总对著听不见的人说话？』我身旁的光传来意念：他们是自杀者，被自己行为的每一道后果捆绑。这令我震慑——我知道此想法来自祂，而非我自己。而后我不再看见类似场景，仿佛祂要传达的真理已经说尽。」

因此，大师们将自杀者称为「地面行者」，实是贴切。

威廉·贾奇是布拉瓦茨基夫人的密切同事，神智学会的共同创立者——在《自杀不是死亡》一文中如此描述：

「一般而言，自杀者的命运极为可怕。他以物理手段断开与肉体的连结，却没伤到真正的『人』，即灵魂。随后他被抛掷至星光界，因他总须有个存身之处。在那里，看似冷酷却实属慈悲的律法，迫使他等待，直至自然死期到来。他必须等候数月乃至数年，半死不活，直到肉体、灵魂与灵依自然律彻底分离，方能被引离该境。他成了幽灵，活在某种炼狱中，也就是神秘学家所称『欲望与情感之境』。他全然地陷在星光界里，被自身的思想吞没，不断在鲜明的心念中重演结束生命的那一幕，眼睁睁看著与自身分离的人与地，却无法与任何一方交流……」

自杀率在二十一世纪急遽攀升，其实自五、六十年代便持续上升。此前数百年间，人们有同样的能力结束性命，却未如此频繁为之。这岂非暗示：在我们引以为傲的现代「文明」里，科学成了人类的神，唯物与感官成了信条，电视是发声的圣经，网络是人类的游乐场？

真正的灵性并非幻象或妄念，真正的灵性，是对事实与现实的揭示。
