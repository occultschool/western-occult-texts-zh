# 去世的人会到哪里去？
## 所有人都将活在自己的行为里，或美梦、或噩梦、或就此消亡

![cover](/img/downloaded/f3b25232ae7f28a823e62b149e168208.jpg)

Photo by [Zac Durant](https://unsplash.com/@zacdurant?utm_source=medium&utm_medium=referral) on [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral)

神秘学学生最常追问的问题之一，莫过于：「去世的人会到哪里去？」

你若问寻常基督徒，他或可引用《圣经》经文答你，但你依旧茫然。新纪元的神秘主义者或各式「异教徒」，则可能依据各自信仰与通灵经历，给你纷纭答案，却同样令人困惑。

答案其实极其简单：**死者，住在自己的行为里。** 请容我再说一遍：**死者，住在自己的行为里。** 你或许要问：这如何可能？「行为」所指为何？

我们所谓的行为，即「契合」。我们与什么契合，便成为什么。残忍之人将堕入下层星光界，那里住满折磨者——许多宗教称之为「地狱」，也为现代濒死经验者如霍华德．斯托姆所印证。

斯托姆曾是公开的无神论者，敌视一切宗教与信徒。他以愤怒控制周遭所有人，在任何事物中都寻不著快乐。他不信物质世界之外有更高层面，认为一切宗教皆是无根幻梦，科学之外别无真实。然而一九八五年六月某日，三十八岁的他经历一场濒死体验，从此彻底翻转人生与视野。你可从他的叙述中读到，手术中他「死亡」时的际遇。

那些彻底自私或怠惰者，将被抹去，或永远沉睡；而善良、慈悲、仁慈之人，那些不计回报为同胞服务者，以及怀抱灵感的美丽梦想家、诗人、音乐家，将升至更高层的星光界。

在那里，恋人们将在玫瑰与紫水晶色的云霞下，于金色之境重逢。银色溪流蜿蜒，永恒花朵绽放各色芬芳，芳香树木结出甜美果实，青翠山丘环抱著晶莹瀑布与闪烁湖泊。一切皆在梦幻与至福的领域中，此为俗世永不可知之境；二人远离窥探的目光，独自沉浸于完美结合的狂喜里。

真正的神秘主义者与真理爱好者，将翱翔至远超人类眼界之域，与同侪相遇，在智慧中生活，于不断扩大的崇高循环里扩展，永与众神同在。

因此，所有人都将活在自己的行为里，或美梦、或噩梦、或就此消亡——若「光明之主」如此定夺。但对某些人而言，这行为或梦境，将成为对更高意识、对宇宙意识的真正觉醒，非任何语言所能描摹；对这些人，再无重返沉睡的可能。
