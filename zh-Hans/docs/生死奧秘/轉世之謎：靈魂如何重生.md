# 转世之谜：灵魂如何重生？
## 灵魂运作与业力法则的转世剖析

![cover](/img/downloaded/40d80fcbf65c0a27f5152138d487d849.png)

若相信转世（一旦看清其逻辑与哲理，便难以不信），又见证据越来越多，人难免想追问：转世究竟如何发生？其方法、过程和阶段为何？

本站[《对转世的正确理解》](/zh-Hans/docs/what-is-reincarnation-correct-understanding)、[《死亡与来世》](/zh-Hans/docs/death-and-afterlife)、[《去世的人会到哪里去？》](/zh-Hans/docs/where-do-the-dead-go-after-death)诸文，已论转世大要，探讨人生尽时之过程，描述灵魂离体后、投生前的各段旅程。然则，重生具体如何发生的？

我们明白一切依业力法则而行，但这非本文焦点。我们意在探讨：灵魂或自我如何自天界状态显现，于物质层面重获生命，并从中窥见几分真知。

神秘学提供若干有趣视角，亦涉及诞生过程的其他关键。对智慧大师而言，这些并非「奥秘」；对其「直接代理人」与「同志」布拉瓦茨基亦然。

然则，仍有诸多事理、真相、细节，是这伟大兄弟会尚不得公诸于世；故在循环法则与业力法则严密框限下，他们仅能在某些领域投以暗示、一瞥、或简短指引。转世的实际过程即是一例。

身为神秘学子，当务之急是深入、精准、且怀抱深意，去理解布拉瓦茨基留予我们的一切教诲与智识。若连此都不愿深究，又何能企盼更多洞见？

无论如何，且让我们把握机会，窥探此中玄机。…

于《秘密教义对谈》中，学生问及此问题，布拉瓦茨基答曰：「生殖细胞仅含种子，亦称星光体。父亲将此种子植于物质土壤。… 单子到了第七月开始笼罩胎儿，待孩童具意识方完全融入。天界实体环绕新身体，点燃其火；然而两者同化过程，要到意识初萌之后才开始，即七八月时。因此它未即刻进入。一开始单子笼罩新身体，业力引之趋近，却无法立时合为一体。」（第559页）

综览此言，并参照布拉瓦茨基及其他大师散见各处之论述，可归结数项事实：

\* 父之种子仅含星光体之种，于母体内开始生长和形成。

\* 星光体之形成早于肉体，其生命亦较肉体绵长；此乃无形、精微之蓝图、模型与框架，肉体依此构筑。

\* 受孕后，新星光体开始形成，迅即为肉体外壳所包覆。

\* 此形体虽未真正「具有灵魂」，然自受孕一刻，便以某种特定方式，与将降临之「自我」相系。

\* 此时之「自我」仍处天界状态，故称「天界实体」。

\* 其天界时光到了尾声，便开始「环绕」、「点亮」、「笼罩」此微小形体；后者将于数月内，成为其新物质载具。

\* 此为神秘学反对堕胎诸多缘由之一，亦是布拉瓦茨基直指堕胎乃「谋杀」、「蓄意毁生」、「扰乱自然秩序」及「逆天之罪」的缘故。

\* 孕程约七月时，关键的「融合过程」启动。新生儿的低等人格部分，此时与新身体结合。即便孩儿出生，灵魂较高部分（即高等心智），通常须待其七岁，方能与新人格建立稳固完整之连结。正因如此，七岁前夭折的孩童，其灵魂往往能迅即或几乎立刻转世重生。

\* 天界实体与新物质载体，日渐紧密联系，这与K.H.大师所述后半段相关：「自我在天界，犹如在尘世，历经心智初动、臻于鼎盛、力渐消散、陷于半昧、遗忘昏沉、乃至全然遗忘……之后并非死亡，乃是诞生：诞生进入另一人格，重启行动，每日种下新的因；此等积聚须待下一段天界时期处置，并再次投生肉体，成为新人。」

天界与尘世的生命，皆由业力注定。

同一位大师曾述及胎儿的发育：

「……母腹之中，胎儿历经矿物、植物、动物诸态，终成人形。以人胎为例，自著床至七月，蜷缩于胞衣，如演化缩影，重复矿物、植物、动物的周期；直至末两月，方始具备未来人形的雏型。其发育至七岁方成。人之本质亘古存在，不增不减；唯在自然之母的子宫里——正如在俗世母亲的怀中——才得以进一步开展。一位博学的哲人，笃信直觉胜于当代科学，主张：『人类在子宫的各阶段，实乃地球历史缺失片段的浓缩。』因此，你必须回溯动植矿物诸态。须将每一显化初萌的实体，视为宇宙原初粒子，在生命初始的震荡中，分化而出。因此，圆满行星神灵的潜能，正潜藏于这些原初粒子之中。」

此外，这位跨喜马拉雅的开悟者，对费解的「二重身」现象，以及陌生人容貌极似的奇事，给出了一番精辟阐释：

「有时，婴孩的相貌会与另一人惊人相似，尽管两人或许远在千里，与婴孩之母从无交集。然而，那人的形象早已烙印在母亲的灵魂深处，或于梦中，或于清醒之时；这印象遂映照于她即将诞生的孩儿面容。」

据信中所指，「漂浮的图象」实为星光界流质中的影像与画面。星光界流质乃一种心灵感应的介质，环绕甚至渗透物质层面。其中，每一念、每一言、每一行皆被纪录，永镌于星光界流质。故如贾奇先生所言：「星光界流质充斥万千图像」，而我们与之接触实属频繁，往往超乎我们的认知。

现在，且探轮回如何发生。

受孕之际，即将到来的「自我」之星光体种子，蕴含在男性生殖细胞中。然而，此「自我」何以抵达？为父者的精子，又如何获此神圣特性与力量？一切都在伟大的业力法则下运转，无论双亲是否知晓、是否信服。

威廉·贾吉于《神智学的海洋》中《论轮回》一章提及：「如何描述『自我』从非物质转为物质状态必经的实际过程」，以及「从不可见层面降临可见层面的途径、方式或方法」。

他继而言道：「此题在神秘学著作中尚未明朗。一则因这需要极细腻处理，再则即便神秘学家自身，也未必全然信服这些细节——虽终有一日它们将被接受。既然此细节并非最关键，眼下暂不深究。但众所周知，人体成形需两性结合，胚种锁于两性之中，且必源自摄入体内的食物；故而，食物实与转世之自我息息相关。」

于是，有人谨慎推想：「轮回之路，或藉特定食物展开。」然此观点，终未见进一步阐明。

贾吉先生另有一处，深刻描述了轮回过程及其初期阶段，令人惊异。晚年时，他构思一本关于前世经历的书，且希望由同僚执笔，不署己名，不落任何个人痕迹。他写此书，只为给细心而直觉敏锐的读者，提供宝贵的灵性教诲与洞见。他从不渴求瞩目，故坚持书中主角的身份必须隐去。

不巧，书未起笔，他便辞世，此作遂成未竟之章。然而，他曾交给朱莉娅·基特利（她在神学界以笔名贾斯帕·尼曼德闻名，并允诺协助完成此书）的若干笔记，后被收入《对我有帮助的信件》作为某种附录出版。其中引人入胜的段落，见于第253至254页：

「当他们抵达南印度，他见到了塔楼里的老者。他向前致意，再次急切询问羊皮纸之事。老人所言如旧，并附加一句：最好能在该处回到此世界。

「黑暗笼罩，万籁俱寂。本是晴朗炎热之日，却无半点雨意。听罢老者话语，他心中默许在该处生活，然倏忽之间，狂风暴雨袭来，雨水倾盆。他感觉自己被拽回尘世，陷入深沉黑暗。一道清晰的声音环绕著他——那是植物生长之声。眼前展开一片稻田，间杂几簇芝麻。水滴落下，触发萌动；他环顾四周，万物皆在运转，生机勃勃。他被困于茫茫稻禾之中，为自身命运感到悲哀。他出生在一个婆罗门家庭。」

注：轮回之谜是否透过云雾、雨水与种子，从人类的种子中投生？

应补充的是，即便此处所述经历属实，也并非所有轮回的灵魂皆能意识到此过程，多数人恐浑然不觉。此特殊个例中，该「自我」已然在灵性上臻于成熟且进化完满；在加上，他前世是因被谋杀而仓促终结——正因这双重缘故，他能有意识地滞留于欲界，并得以游荡四方，「寻觅投生之机会」。

故而，他遇到启蒙者指引他前往南印度，并劝言其当下便「重返人间」时，他能清醒地应允，随即透过「云、雨、种子」之过程，转世进入人的种子，且对此中每一阶段皆了然于心。

远在三千五百至五千年前，古印度的圣者、大师与先知借由神圣《奥义书》传授教诲时，所思所虑或许正在于此。他们反复申明：「雨生食物，食物生性别种子」（《蒙达卡奥义书》），「食物孕种子，种子育众生」（《六问奥义书》），又言「一切身躯皆从食物而来」，并倡言「敬重食物，因其塑成肉身」（《鹧鸪氏奥义书》）。

在印度吠檀多不二论哲学中，肉体亦被称作「食物制成的身鞘」。此说与前述或有关联，或无关涉，端看读者如何解会。

言尽于此，不宜再赘。或许对读者而言，本文所陈资讯与细节，产生的疑惑反多于解答。然而我们希望，这些文字能激起几分兴味与启发，至少为原本幽邃难明的谜题，投下一缕微光。

末了，且让我们沉心默思《他氏奥义书》中的教示。此书视受孕为「初次诞生」，视脱离母体、踏入尘世为「二次诞生」，视舍离肉身为「三次诞生」。一切乃生命，而你，即是生命本身。
