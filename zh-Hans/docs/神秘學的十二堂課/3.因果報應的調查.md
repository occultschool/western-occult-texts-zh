# 因果报应的调查
## 揭开因果报应学说的面纱

![cover](/img/downloaded/3a8ceb7e43c32e30d38d44f55975cdd1.jpg)


在神秘学研究课程的第三部分，我们直面「因果报应」这令人困惑的概念，探究其本质与谬误。

关于因果报应的阐释，众说纷纭，光怪陆离。倘若因果报应是头驴子，恐怕得生就一副宽阔的脊背，才驮得动人们强加于它的种种指责。一位神秘主义者曾说，某人之所以整骨手术失败，乃因前世效力于西班牙宗教法庭，专司酷刑。我们不信这套，深知许多良善之人，亦因医者疏失而受苦；你必然也不会相信。尽管如此，仍有人将此逻辑推演得更远，声称手术失手的医生，前世曾被这些病人伤害。

更有些怪诞的东方教派相信，杀鸡斩牛者，来世必投生为鸡为牛，身受同样命运。但愿这并非事实！我们虽未亲自宰杀牲畜，却可能做过同样残忍的事：上周才喷洒杀虫剂除蛾。我们可不愿来世化作飞蛾，经历同样遭遇。不过，也许飞蛾蛀蚀衣裳在先，因果报应由此介入，驱使我们喷雾惩治——如此说来，一切倒也合理。

我们认为，灵性追求者往往过分忧虑因果报应。因果报应对他们而言已成心病，渗透思绪，干扰行动，滋生恐惧与罪恶感。另一些人则认为因果报应过于复杂难解、晦涩幽微，索性放弃理解。

事实上，正统的因果报应学说，核心不过是「因果」。

多数神秘主义者都能理解并接受此正统学说。权威典籍皆言，因果报应即是「因」与「果」的法则。你若作恶，恶果迟早降临，好教你学会日后行事更善。但此说前提是，你至少不能比驴子笨！驴子不会被同一块石头绊倒两次，牠第一次就学乖了。这是否意味著驴子能在进化阶梯上攀升，来世投胎为骡，甚至成为一匹壮硕的拉车马？我们认为，这得看牠成功避开石头时，那份新获的智慧在心智中刻下多深的印记。

这是理解「因果报应」真义的第一个线索：行为对心智的影响。我们来深入探讨。古印度教义中，「因果报应」或「业力」（Karma, Karman）源于梵文，意为「行为」。此外，在印度哲学与日常语境中，指的是「一个人的行为决定其未来命运」。这不只是模糊的善恶有报，而是指每个行为都必将引发其无可避免的后果，承受其报应，无论过程需要多少年才到来。物质宇宙的每一部分——男女、虫蚁、树木、石头，乃至令我们食坏肚子的小虫——皆是一个永恒之灵的寓所。万物正决定自己的命运：领受过去行为赢得的奖赏或惩罚，同时为未来播下奖惩的种子。

东方视存在为无尽的播种与收获。博学者与无知者皆接受此观点，这是人世不平等的缘由；若非如此，人们或将质疑神之公正。然而，这并非意味印度教徒不相信天堂与地狱。印度教中，二者以各种形式存在，或多或少有些怪诞。据说死后，高我会前往其一，安于应得之位；但这取决于欲望与行为所积累的功德或罪过。若高我仍困于业力之网，便须再度投生尘世，继续修行。印度教的救赎概念，在于从因果报应之力中解脱，各派哲学自有其解脱法门。此学说源头虽不可考，但无疑属后吠陀时代产物。公元前六世纪，佛陀便欣然接纳（尽管作了相当修改），至今仍存在于婆罗门教、佛教及诸多「新纪元」信仰体系。

此教义进一步阐明，所行之事皆有永恒影响，简言之：「种瓜得瓜，种豆得豆。」行动并非单一，而是包含三要素：构思行动的「思想」、寻求实现方法的「意志」，以及二者结合以「付诸行动」。由此可见，思想力量强大，无论善恶。思想如何，行动便如何：满心贪婪的守财奴欲壑难填，念头污秽的浪荡子恶行不断。反之，心怀善念者，行为自然展现美德。然而，因果报应学说不能仅限于一世，必须考量转世。初生的男女带著前世的种子；他们的性格承袭过往，也将延续下去，除非他们决心改变——人人皆有此能力。在每一次接续转世中，性格会朝某个方向愈发固化。若是邪恶，改变便日益艰难。邪恶的诱惑更加强烈，而难以抵抗；然而，真正明了业力之人，纵使最终屈服于命运，也必先经历最顽强的挣扎。请记住，这并非我们的信仰；到目前为止所述，不过是东西方普遍接受的因果报应学说，而这是多么阴郁且令人不安。

## 因果报应的诞生与传播

然而，仅凭上述就能解开因果报应之谜、阐明其运作吗？我们一直强调的是思想的力量，或曰心智之力。只要意志足够强烈，心智所造之物便能化为实体。心理学家称这些心智图像为幻觉，但在创造者眼中，这与可见之物同样真实。意志愈强，专注愈深，便能将这些形体在众人眼前变得客观可见。思想若受强力引导与维持，便能成就这些壮举。数千年前，「因果报应」之说尚不为人知，人们自由快乐地生活，如空中飞鸟、田间百合，不思明日之忧，不恼昨日之逝。然后，有人想出一个主意；我们猜想他是祭司，且必是射手座——这类人总想改变一切，美其名曰「进步」。他将此念传与同僚，告诉他们，只要齐心协力，向世人宣扬这个新思想，祭司的权力便将更加强大；最终借由恐惧，他们将统治世界与一切财富。

于是，因果报应学说（或许称之为教条更贴切）公诸于世，祭司掌权的时代就此开启。人作为神之子女本应享有的自由真知，被抹杀遗忘。强力的仪轨与劝诫、因果、罪罚、对死亡与来世的恐惧，这套新学说被无情灌入人类心智，自此，人便成了它的奴隶。我们因而目睹这可悲景象：牧师们在讲坛上无休止地宣扬毁灭，将对神的恐惧注入人心，而非以神之爱充盈会众的胸怀。

任何人皆可于《西藏度亡经》（Bardo Thodol）中读到，狡黠的西藏喇嘛将地狱描绘最为精致狂喜的残酷刑罚；或者中世纪及后期「神圣」宗教裁判所，对异端、女巫与术士的可怕迫害、火刑与拷打。今日则怀著二十一世纪的沮丧，听见狂热毛拉吹响仇恨的号角，他们要残害、切割、谋杀伊斯兰的敌人，甚或穆斯林同胞——只要对方恰巧属于不同教派；而这一切，皆以真主——那慈悲仁爱之神——的名义进行。最后，我们也见证《和平王子福音》中伪善的女布道者，她诅咒同性恋者与在美国堕胎之人，祈祷他们死亡。

## 何以没有「因果报应」？

若你还未看清因果报应的本质，我们愿直言相告：因果报应是某种意识状态的认知，与现实无关。当我们说「没有因果报应」，但愿这不会令你过分震惊。我们再说一次：因果报应并不存在！我们不信此说，正如我们不信罪与罚的教条。

我们的思想与行为，若是消极与破坏性，便会连结宇宙万物的黑暗面；若是积极建设，则根植于真理与善的光中，不像那些懦夫蜷缩于黑暗角落，与地狱众魔为伍。此间并无中间道路，除非将第三种状态称为漫无目的的漂流，连尝试转向光明或黑暗都不愿。但任何理智之人不会视此为「道路」：我们要么是黑暗的居民，终日憎恨、呻吟、毁灭；便是光明的孩子。中间地带并不存在。

然而，每天都可见焦虑的人们，悲伤地与我们擦肩而过。我们痛苦地意识到，确有太多人于生活中漫无目的地漂泊，内心充斥恐惧与虚妄想像。我们撰写此文，正是为了帮助这些可怜的迷途者，让他们认识到，有个更伟大生活等待著他们，认识那不断召唤游子归家的光明灯塔，以及那份浩瀚的爱、耐心与理解——如此深切渴望再次将游子们拥入庇护的怀抱。一旦你认识那伟大的光，便会觉知它对你完全的护佑，以及闪耀著永恒真理；至此，黑暗、恐惧、疾病、迷信或因果报应这些念头，便再无从想起。

但请勿就此来信，引用各派「神秘学」与其他玄学「权威」，试图证明我们的论点错误，因为我们不会回应。神赋予人人自由意志，接受或拒绝此教义皆由己择。但我们希望，你若已读懂并理解我们对此议题的阐述，便会将「因果报应」一词彻底从你的词汇中抹去。如此，你将不再是「命运」或「宿命」的受害者，而是扮演积极角色，塑造自身未来「转世」，这正是本神秘学课程系列第四篇文章的主题。
