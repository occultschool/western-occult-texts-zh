# 力量之戒的象征含义
## 隐藏在力量之戒里的数字含义及其与配戴者的关联

![cover](/img/downloaded/f35ae1af447c6d6d83d114c5bef57494.jpeg)

力量之戒在《指环王》里举足轻重。精灵凭借索伦传授的知识打造了十九枚戒指；索伦却在厄运山的烈焰中，私自铸造第二十枚——「至尊魔戒」，或称「统治戒」，用以驾驭其余戒指。二十枚戒指如何分配？书中一首短诗已然说尽：

「三枚戒指给天底下的精灵王，
七枚给石厅里的矮人领主，
九枚给注定要死的凡人，
一枚给黑暗宝座上的黑暗主宰
在阴影笼罩的魔多之地。
至尊魔戒统驭众戒，觅得众戒，
将它们尽数牵引，在黑暗中束缚
于阴影所在的魔多之地。」

戒指的象征意义，固然不宜穿凿，却有几处格外引人注目。首先，戒指的数量和分配就算只是巧合，也已经很了不起了。数字十九可拆作一与九，相加为十——此乃毕达哥拉斯学派的神圣数字，更在著名的「十点三角形数」中永存。我们曾在谈论[《数字学》](/zh-Hans/docs/mystic-numerology-intro-1-to-10-meanings)的文章里提过。三枚戒指对精灵意义尤深：三结合了单子（一）与二元（二）而诞生。三是第一个奇数，因一不被视为数字，而是万数之源。毕达哥拉斯学派有言：「三是良策、智慧与知识的缘由，它是音乐的女主人。」这些美德，无一不与托尔金笔下的精灵深切相系。稍后我们将细说三枚精灵戒各自对应的美德。

不少读者知道，七是关乎「生命」的神圣数字，古人视为众数中最玄秘者。托尔金为何将它与矮人相连？我们不得而知。矮人纵有长处，却不像他笔下的精灵那般幽深莫测——故且将保留对七的解释。九是「当下」与「未来」的数字，因此，将九枚戒指赐予「注定要死的凡人」意味深长；人类正是终有一死的「不朽者」，为了进化到更高的领域而化身血肉。

索伦私铸至尊魔戒，将戒指总数从十九增至二十。从数字学看，他在十九上加一，使之化为二（一加九为十，十加一为十一，一加一为二），无异于玷污、贬损了神圣的「十点三角形」标志。二是二元与分裂的象征，滋生混乱、困惑、离散与迷惑，或可称之为「邪恶」。托尔金是否通晓神圣数字科学？这套知识多大程度上影响他的思维？我们无从确知。但考虑到他至少与一位神秘主义者有所往来，他受数字学启发，并非不可能。

众戒大抵具备几样共通属性，唯独三枚精灵戒例外——它们各有独特的优点。所有戒指首要的力量，在于延缓躯体朽坏，拉长生命。它们也能增强佩戴者与生俱来的能力，无论善恶。此外，它们赋予了窥探不可见世界的门径，我们称之「星光界」。但唯有至尊魔戒能将佩戴者完全送入此隐秘维度，令其血肉之躯从凡人眼前消逝。在索伦统御下，那九枚与七枚戒指力量如此强大，足以令佩戴者——尤其是人类——逐渐褪色，化为戒灵，成为拥有超自然力量的半人半魔。

此外，书中亦载，矮人所得的七枚戒指，能增强佩戴者的天赋与占有之欲——凡经他们开采之物，皆因此倍增，铸就了他们的贪婪与巨富。而赐予「凡人」的九枚戒指，可延寿续命，代价却是终将为索伦意志所奴役。至于至尊魔戒，如前引诗所言，拥有统御众戒之力。然从神秘学视角观之，最引人深思的，当属精灵所持的三枚戒指。

据载，索伦之手从未触及此三戒。《精灵宝钻录》中如是描述：「三戒最后铸成，力量亦最为强大。其名乃亚之戒（火）、南雅之戒（水）、与维雅之戒（气），各镶红宝石、钻石与蓝宝石。」若这三戒对应火、气、水三重元素，则可推断（尽管托尔金未曾言明）至尊魔戒象征著土元素。此说合理，因土元素既能巩固三者之力，亦能将它们结合在一起，正呼应了前述诗章的末节。读者中的炼金术士当会记得，三重伟大的赫耳墨斯在《翠玉录》第六节中指出：「其力降临于地，至为完满。」

托尔金未言明火之戒的材质，只道其能鼓舞人心，抵御暴政、统治与绝望，亦能消解岁月积累的倦意。水之戒则以他独创的金属「秘银」打造；此银不染尘晦，永保光洁，其力在于护持、隐匿，令邪恶不侵。第三枚气之戒则以黄金为体，嵌一颗蓝宝石。本书第三部《王者再临》中提到，此戒为三戒中最强，虽其确切威能未曾详述。然我们可合理推测，它亦具备疗愈与守护之力。

镶红宝石的「火之戒」后来赠予甘道夫，助他对抗索伦。而「水之戒」亦称「钻石之戒」，自铸成之日便由加拉德瑞尔佩戴。镶蓝宝石的「气之戒」则为埃尔隆德所持，用以护佑中土世界东部精灵家园瑞文戴尔。正如前篇所见：「火……是生与死，是万般物质与灵之始与终。」值得注意的是，在第二部《双塔》中，甘道夫曾对恶巫萨鲁曼的爪牙巧言喝道：「我穿越了火与死亡，不是为了和一个无知的虫子闲扯！」我们亦不应忘记，甘道夫自称是「秘火」的持者——此点[《魔戒中的善与恶》](/zh-Hans/docs/tolkien-good-evil-in-lotr)已有讨论。由此可见，甘道夫在多方面皆与神秘的火元素及其特质紧密相连。事实上，全书始于庆祝霍比特人比尔博一百一十一岁寿宴，结束于一场由甘道夫筹划施放的绚烂烟火，诚非偶然。

火之戒上那颗红宝石，又有何深意？威廉·T·费尔尼在《贵重宝石的神秘性质与治疗能力》一书中写道：「红宝石被认为能侦毒示警。若有邪祸逼近佩戴者，宝石便会晦暗浑浊；但它亦能驱散悲伤，祛除多种罪孽与恶习。」若说托尔金将「火之戒」、「红宝石」与甘道夫相连仅为巧合，那这定是一桩可喜又恰切的巧合！因他正是书中首位警示中土世界居民，须提防叛徒萨鲁曼那「毒计」之人。亦无任何角色比他更决意驱除「罪孽」与「恶习」。在索伦之战中，甘道夫更驱散了萦绕精灵与人类心头的悲戚与绝望。

读过[《金黄星》](/books/golden-star-dr-michaud)者皆知，该著作者特选钻石为神的象徴。因此，加拉德瑞尔佩戴「钻石之戒」便不意外，因她正是托尔金神话中「诸神」维拉的主要化身。费尔尼告诉我们，钻石象徴纯真、正直、信念、力量与对命运的坦然——这些美德皆体现于精灵一族，尤在加拉德瑞尔身上。而此戒又与特定神秘元素相连，即水。即便是大地之上的寻常之水，亦具备诸多非凡特质：水能愈病，世上多有「圣水」泉井，如法国卢尔德著名的圣地。广义言之，「疗愈」正是加拉德瑞尔在书中所扮演角色的本质。

水是最具穿透力之物。早在十一世纪，一位中国学者便写道：「诸元素中，圣人当以水为师。水柔顺，然能克万物。水可灭火，若遇强敌，则化汽逸走，变形易质。水能蚀软土，逢巨石则绕道另辟。它浸润大气，令风止息。水以谦卑之姿为障碍让路，却无力量能阻其奔赴海洋之途。水以屈为伸；从不强攻，却常赢取最后的战役。」凡此种种品质，皆可在加拉德瑞尔身上觅得。

那么，精灵三戒中最强大的气之戒呢？费尔尼称镶嵌其上的蓝宝石为「圣物」，它能令持有者变得「平和、亲切、虔诚，坚定良善之心；若持有者心术不正，它便黯然无光。仅凭自身纯净的光芒，便能诛灭一切毒害生灵。」这枚珍宝的特质，不正与埃尔隆德的德行遥相呼应？我们难以视为巧合，尤其在戒灵追捕佛罗多一行时，埃尔隆德施展法力召来洪水，淹没的正是一群「毒害之物」。此外，书中再无人物比他更「虔诚」——熟悉故事的读者皆知，对于共抗索伦的盟友而言，他的虔诚有时甚至显得过于执著。

气之戒关联风与大气，其特质多处映照埃尔隆德的性情。若读者中有占星爱好者，定能领会：无论字面或隐喻，风总与「沟通」息息相连。因此，书中最关键的会议，正是于埃尔隆德召开的议事厅中举行；众人商讨魔戒去留，各抒己见。同样，故事一开始，佛罗多在夏尔边境遇上一队精灵，他向领袖吉尔多问道：「我该等待甘道夫，或独自前行？」精灵答道：「甘道夫迟来，恐非吉兆。但俗话说：『莫插手巫师之事，因其既狡猾且容易发怒。』因而，等与不等，皆由你抉择。」佛罗多听罢，语带讥诮：「也有人说：『勿向精灵询问意见，因其既会答否，亦会称是。』」这段对话既显火象星座的易怒特质，亦透出风象的犹豫与摇摆。天秤座的读者读到此处，想必会心一笑——这两样「弱点」，确是此星座常见的特质。

尽管不少占星师与业余爱好者，曾试图将书中要角与十二星座逐一对应，我们在此不欲跟随。专业的占星师皆知，一个人的太阳星座，仅是塑造其性格的众多因素之一。话虽如此，甘道夫确实带有白羊座的勇猛、果敢、领袖气质与前述的急躁；但他亦具摩羯的沉稳、严峻与顽固。佛罗多则颇似双鱼，此星座常与自我牺牲、以及对崇高境界的朦胧向往相系。至于索伦，任何略通占星者皆不会否认，他集齐了天蝎最阴暗的特质；而山姆则体现了金牛最美好的品质，尽管他亦不乏巨蟹的温情。但咕噜却是例外；那些仅读报章星座栏的读者，硬将他与双子座相连，只因双子「有两张脸」、似具双面性格——这恰恰暴露了其对占星之浅见。若真要为这位关键又复杂的角色指配一星座，或可是天蝎。然我们并非专业占星师，此说切勿当真！若你对此主题深感兴趣，我们鼓励你自行深入探索。毕竟，占星之象与书中象征确有牵系，或能助人窥见故事更深层的寓意。
