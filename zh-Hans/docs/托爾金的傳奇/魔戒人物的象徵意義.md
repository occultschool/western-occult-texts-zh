# 魔戒人物的象征意义
## 弗罗多与咕噜：高我与小我

![封面图](/img/downloaded/d3ebdeece9b0d051831407693cff30d5.jpeg)

《魔戒中的善与恶》（/docs/tolkien-good-evil-in-lotr）提过，甘道夫是「秘火之仆」，乃众神维拉直接的使者。托尔金曾在信中写道：「甘道夫并非人类，也非哈比人。现代词汇无法定义他。他是降世的「天使」，或更精准地说，是信使。」希腊文「angelos」即是信使之意。托尔金借此词印证其神圣信使之身分，正如[《金黄星》](/books/golden-star-dr-michaud)里的涅特鲁-赫姆。书中的甘道夫，不似彼得·杰克逊电影所描绘，沉溺于拳脚纷争，更非挥舞魔杖的卡通英雄。冲突当前，甘道夫给予启发、鼓励与睿智建言。即便偶尔对抗敌对势力，他也仅是领袖与发起者，绝非冲锋陷阵的战士。

巫师萨鲁曼同为信使，故事一开始，他力量更在甘道夫之上。然而他堕入「黑暗」，为膨胀私欲，不惜倾其所能摧毁远征队。他善用催眠术，或称「暗示」的魔力，在故事中屡屡奏效。但他终究败了。甘道夫对他说：「萨鲁曼，我太了解你。你的言词作为，我记得一清二楚。你看，我不再是遭你背叛的灰袍甘道夫，我已是死而复生的白袍甘道夫。如今你一身灰蒙，我要将你逐出教团与议会。」

书末，弗罗多与潦倒失势的萨鲁曼有过一番对话。萨鲁曼的仆人称为「巧舌」，名副其实，曾暗藏匕首意图刺杀弗罗多，但未得逞。「十几个哈比人在山姆带领下，呐喊扑来，将这恶棍压倒在地。山姆抽出剑来。『别，山姆！』弗罗多阻止：『现在也别杀他。他并未伤我。何况，我不愿他在这邪恶之境丧命。他昔日何等伟大，曾是我们不敢忤逆的尊者。他虽堕落，治愈之法已超出我们所能；但我仍想饶恕他，盼他能寻得救赎。』

萨鲁曼站起身，直直望向弗罗多，眼神复杂交织著惊诧、敬意与憎恨。『你长大了，半身人，』他说：『你确实长大许多。你既睿智，又残忍。你夺走我甜美的复仇，如今我只能含恨离去，皆因我欠你这份怜悯。我痛恨这怜悯，也痛恨你！』」
骄傲与我执令最强者沉沦；智慧与胸怀却让最卑微者崛起。然而，正如弗罗多所言，纵是最邪恶的生灵或神祇，只要仍渴望「治愈」，救赎犹有可能。这确是耐人寻味。

书中人物，弗罗多背负著的魔戒重担，也历经最剧烈的转变——最终，他更似精灵，而非哈比人。他穿过[《中土世界的生与死》](/zh-Hans/docs/tolkien-mysticism-middle-earth-life-death)中提到的「都林之门」，是他启蒙之路关键一步。这趟觉醒之旅，始于他慨然承接摧毁魔戒的重任，最终将它投入铸造它的烈焰之中。任务始于夏尔，当时甘道夫向他阐明原委。「我恨不得毁了它！」弗罗多喊道，随即补充：「或是让它被毁。但如此险途，我并不合适。」
若将「魔戒」视为小我的某种象征，那诱惑欺瞒人心的力量，弗罗多既想摆脱它、又惧怕涉险的心境，听来何等熟悉且人性化？掌握小我是一回事，真正扛起责任又是另一回事。因此，当弗罗多对甘道夫吐露：「但我自觉渺小，满心绝望。敌人如此强大可怖。」我们毫不意外。在学会驾驭小我之前，谁不是如此？待论及咕噜时，我们将重返这双重自我的主题。

后来在爱隆会议上，佛罗多才真正体会到任务的艰巨。「众人皆垂首默坐，仿佛沉入深思。一股巨大的恐惧攫住了他，像在等候早已预知的判决，又暗暗盼望那判决永不被宣出口。他心底涌起强烈的渴望，只想回到瑞文戴尔，留在比尔博身边，求一份平静。最后，他挣扎著开口，听见自己的声音时吃了一惊——仿佛有别的意志藉著他微弱的嗓音在说话。他说：『我会带上魔戒，虽然我不识路。』」这话多么有力而动人，且蕴藏著深意！教人想起从前听过的另一些话：『那没有看见就信的有福了。』（约翰福音20:29）『若有人要跟从我，就当舍己，背起他的十字架来跟从我。』（马太福音16:24）「十字架」或「戒指」所指原是同一物：即人的肉身与小我。只要活著，这副重担就得背上，是走向荣耀与解脱，或是奴役与黑暗，选择在于我们。

书中有个极神秘的人物，汤姆・庞巴迪。托尔金在一封信里写道：「即便在神话时代，也须留有谜团，因为谜团本是常在的。汤姆・庞巴迪便是一个（有意为之的）谜。」这话何解？回答前，先得分清这角色两个不同的来历：其一所谓「汤姆・庞巴迪」，原只是个憨傻的荷兰玩偶，属托尔金次子麦可所有，还被兄长约翰冲进了马桶（！）；其二才是现于《魔戒》中的神秘人物。幸而玩偶最终获救，未因那不愉快的洗礼而损毁。此事促托尔金写下《汤姆・庞巴迪历险记》，1934年刊于《牛津杂志》。他后曾向出版社提议，将庞巴迪的故事扩写为《哈比人》续篇，出版社却反应冷淡。于是汤姆二度复活，出现在《魔戒》第七章里。托尔金自承，一个孩童玩偶竟在他笔下演变成《魔戒》里深具神秘色彩的角色，连他自己也觉困惑。早先一封信中，他称庞巴迪为「（正在消失的）牛津与柏克郡乡野之灵」，同时坦言无法将他从《魔戒》中抽离，正因他代表了某种更伟大的存在——尽管他不愿细说那是什么。

托尔金这番话，我们并不意外。或许这也是导演彼得・杰克逊将汤姆・庞巴迪从电影中删去的原因之一。汤姆既非精灵、矮人、人类，亦非哈比人，也不是甘道夫或萨鲁曼那类巫师；他到底是什么？《魔戒》第七章里，佛罗多曾向汤姆展示魔戒。「汤姆将戒指套上自己小指末端……片刻间，哈比人未觉异样。接着他们齐抽一口气：汤姆丝毫没有隐形！」佛罗多困惑极了，疑心汤姆戏弄他，便自行戴上戒指，顷刻消失。但汤姆与哈比人不同，他依然清楚看见隐形的佛罗多从身旁悄悄走过。事实上，书中无人能完全不受魔戒影响。连甘道夫亦难豁免其力：本书稍早，佛罗多想将戒指交给他时，他厉声道：「别引诱我！我不愿变得如黑暗魔君一般。」后来到罗斯洛立安，佛罗多想将戒指交给凯兰崔尔，她也拒而不受。然而魔戒到了汤姆・庞巴迪手中，却失却诱惑与腐败之力。这该如何看待？汤姆究竟是谁？

托尔金自己在书中留下了谜题的线索，他写汤姆是「森林、水与山峦的主人」。第七章里「主人」一词反复出现，且多以大写标示，足见其分量。当佛罗多问庞巴迪「你是谁」，他的回答深富启示：「你还不知我的名吗？这便是唯一的答案。告诉我，你又是谁，孤身至此而无名无姓？你年轻，我年老。我是最古老者。在河流与树木出现之前，汤姆已在此地……他知晓星光下的黑暗，那时黑暗尚不可惧；那是在黑暗魔君自外而来之前。」因此可以说，汤姆是一种力量的化身，这力量既在中土之内，亦在中土之外。若他再多说一句，便等于要点明那托尔金不愿「仔细探讨」的「更伟大的存在」为何了。

我们亦知汤姆是个伟大的歌者。他正是通过歌声来发现并解救了困在老森林恶柳中的霍比特人。后来，古冢尸妖囚禁他们时，仍是汤姆的歌声破除了禁锢。英国古时的德鲁伊自称「歌者」，汤姆的歌声令人想起他们如何以话语施法，因而赢得凯撒等人的敬重。据《英国德鲁伊的神话与仪式》记载，大酋长辛维林曾言：「凡我谱曲，若由我歌之，咒语便生。」换言之，倘若他如书中的汤姆‧邦巴迪尔那般吟唱，魔咒便会涌现，令囚禁霍比特人的荒冢石崩岩落。尸妖亦歌，然其声蕴黑暗魔力，能将哈比人牢牢缚住。由此可见，托尔金不自觉间重现了诸多神秘科学真理，并以灵感充盈之笔加以呈现。

## 佛罗多与咕噜：高我与小我

至于咕噜，他许是书中最复杂的人物，在善恶欲望的撕扯间煎熬。在《霍比特人》中，比尔博怀著怜悯与同情取得魔戒；咕噜——原名史麦戈——却是杀害友人狄戈夺得，视之为生日礼物。同类相感，道德层面上，他一旦戴上戒指，邪恶便占了上风，这点在比尔博与佛罗多身上并未发生。于此可见：小我对高我犹有优势；这优势因人而异，或大或小。然而，尽管恶行累累，咕噜却拯救了中土：若非他在霍比特人即将戴戒挑战索伦之际，从佛罗多手中抢走戒指，使命必将失败，中土亦将陷入新的黑暗纪元。

若视佛罗多为求解脱、求启悟的高我，咕噜便是他的小我。但莫忘，每个自我皆有多重面貌。高我非尽善，因受物质沾染；小我非尽恶，因受另一半——即高我——影响。随著故事展开，我们在佛罗多与咕噜的纠葛中窥见此理。托尔金告诉我们，得戒之前的咕噜类似于霍比特人，与佛罗多并无二致。到了故事后段，当咕噜在他较善的部分（史麦戈）与最恶的本能（咕噜）之间自我辩论时，其双重性展露无遗。我们由此更进一步窥见两个自我或两种心智间的复杂关系与相互依存。不确知托尔金是否察觉此中深意，但凡稍解双我及其纠葛者，读《双城奇谋》第四卷第二章中咕噜的自相辩难，皆可深味。

「史麦戈答应了，」第一个念头说。  
「是，是，我的珍宝，」应声道：「我们答应了：保住咱们的珍宝，绝不让他到手。可珍宝会往他那儿去呀，一步近一步。霍比特人拿它想做啥，咱们倒要瞧瞧……」  
「我不知道。我没法子了。主人得了它。史麦戈答应要帮主人的。」  
「是，是，要帮……珍宝的主人。可要是咱们成了主人，那就能帮自己啦，可不是？诺言也守了。」  
「但史麦戈说他会好好待人，非常非常好……」  
「非常非常好，嗯，我的珍宝？那咱们就好好的——只对自己好。」  
「可珍宝会守信哪，」史麦戈抗议。  
「那就拿了它，」另一个声音说：「咱们自己拿了！那咱们就是主人啦……」  
「说不准我能变得好强，比戒灵还强。史麦戈大人？咕噜大帝？咕噜！天天有鱼吃……要新鲜，从海里来的。非得吃不可。咱们要，咱们要，咱们非要不可！」

我们身上的两个自我，便是这般争夺主导。有趣的是，咕噜的欲望不过是天天有鱼吃。现实中，多少人的志向亦不过如此，由此可见他们的命运受哪个自我主宰！多数人身上，时而此我占上风，时而彼我得势，而争斗的最终胜负，或许取决于争斗者自身。德国有句谚语，恰切道出此中艰难：「与己斗最难，胜己者至伟。」智慧的埃及人早知这场内在战争，便以荷鲁斯与赛特之争为喻。神话中，托特拯救了二者。他令「狂暴者」（小我）静定，洗净「赤红者」（高我）。「赤红」一词源自与太阳神拉的关联；拉为荷鲁斯之父，而荷鲁斯在神话的某一阶段正代表高我。托特在此象征神圣灵魂——我们在[神秘学课程](/zh-Hans/docs/soul-essence-origin-and-mysticism)末篇曾加探讨。那便是我们内在深处的「微小声音」，即「寂静之音」；唯倾听它，我们方学会在两个交战的自我间缔造和平，使二者皆得解脱，各获应有之报。
