# 人类灵魂的永恒进化：从矿物到神性
## 灵魂转世的真相：人类灵魂不会退回到较低界别的进化之谜

![cover](/img/downloaded/e4f201e410760fd9f75a6aefd95bff34.png)

[《对转世的正确理解》](/zh-Hans/docs/what-is-reincarnation-correct-understanding)列有十点说明，其第四点阐明：「人类总是转世为人类。」文中指出：「自然界划分不同领域或部门，各自代表演化中清晰独特的阶段。灵魂一旦从某领域进化至下一阶段，便无法退回已完成的周期与阶段。演化虽缓慢渐进，进程却清晰有序。」

「从矿物界进入植物界后，便不能再成为矿物；自植物界跨入动物界后，也无法重返植物界。一旦我们以具自我意识的个体步入人类领域，历经漫长系列生命，就绝无可能退回低层次演化——那已是全然通过的阶段，不再对我们产生影响或作用。

「人类灵魂永不会转世为动物、树木、花草、石块，或任何非人的存在。」

神秘学固然承认，人类灵魂在意识上可能堕落至近乎动物的境地，并在连续人生中逐渐趋近兽性，但它坚信：纵然遥远的过去我们或曾隶属动物界，转世之后，却绝不可能重返动物界或任何更低层的领域。

然而，诸多印度教、佛教或其他哲学体系却主张：因恶业之故，灵魂可能轮回转世为动物乃至更低等的存在。对此，我们应视之为迷信或无知吗？抑或这些教诲背后潜藏著某种真理？

布拉瓦茨基在《生命原子的转世》中写道：「此说立基于一项真理——更准确地说，是一项公理般的真理，但仅牵涉人类的原子及其流溢；这不仅发生在人死后，更贯穿其一生。《摩奴法典》（第十二章第三、五十四及五十五节）记载：『每一行为，无论属思想、言语或身体，皆结善果或恶果（业）；人（非指灵魂）的各种转世，所经历的最高、中间与最低阶段，皆由其行为所生。』又言：『杀婆罗门者将投身为狗、熊、驴、骆驼、山羊、绵羊、鸟禽等。』此处的秘传真义，并非指向人类的自我，而是指其身体的原子、低等三元组，及其流体发散物。」

神秘学指出，每个人在投生时，皆由七个「原则」或组成部分构成。其中三者——阿特曼、菩提与心智——具备灵性且持久不灭；其余四个低等原则，包括欲望、生命能量、星光体与肉体，则属暂时、必死，仅存续于特定生命期内。它们共同构成小我，即当下的人格。此结构在[《人的七重性质》](/zh-Hans/docs/blavatsky-seven-principles)中有更深入的阐述，剖析了这七项原则的本质与功能。

布拉瓦茨基进而解释：「婆罗门为了自身利益，曲解了法则中的真义。前述引文的原意，绝非后世所解读的那般。原本『婆罗门』在寓意上代表人第七原则，即其不朽的单子与自我的本质；婆罗门却自私地指他们自己。凡是在自身之内扼杀或熄灭婆罗门之光者——即将自我与阿特曼切断、从而扼杀未来『天界者』——便成了『杀婆罗门者』。此人不但没有过上德行生活与追求灵性，心智无法与菩提相融，反而以其恶行，任凭欲望创造出磁性亲和力，导致自身每一低等原则的原子，都被吸入低等动物或畜生的形体中。

「这正是『灵魂转生』学说的真义。」她总结道：「这并非惩罚某人，使其粒子与动植物原子混杂。绝非如此。此种混合反映一种因果关系，其影响会显现于未来转世之中，除非当事人的人格已被抹除。从起因到结果，每一个果都将成为新因，沿著轮回循环运行；最初的冲动唯有在休止期的门槛上，方得自行消散。但那终将降临的时刻，必会到来。」

业力的重担，始终压在每个人的肩上。要到何时，我们才肯认真看待自己的每一念、每一言、每一行？终有一日，那朴素而巨大的真理终将显现：世间没有无果之因，也不存在毫无回响的行动。此时此刻，我们必须学著成为自己的主人——《入菩萨行论》中，寂天早已说得明白：「从今而后，我当成为自己的主宰，世界的仆人。」

威廉・贾吉在其短篇《波斯学生的教义》里亦有触及此主题。书中人物如此说道：「那些细微的原子，时时刻刻在我们周身飘移。它们寻觅著相应的中心，投向性质相似的事物……我们从同伴身上，吸纳与自己相近的成分。人在这粗重的世界轮转生灭，他既是自然的统治者，也是关键、是焦点，是天地实验室中最精密的聚合体。他曾施加于禽兽身上的原子，终会在未来的某个生命阶段，重返其身，带来伤痛或哀愁。但他──那永恒的部分──不会堕落；堕落的，是低劣的、个体的、原子般的残渣。他是所有低于他者的兄弟与导师。当心，莫因你道德的残缺，阻滞并拖累整个大自然。」

因此，「每时每刻自我们身上飘散的那些原子」，正凭著磁性的亲和与吸引，寻觅它们应有的归处。布拉瓦茨基曾说：「这不需开悟者的智慧，只需一点清明的灵视，便能看见它们在人与物之间往返穿梭，恍若一簇幽蓝闪烁的火焰。」
