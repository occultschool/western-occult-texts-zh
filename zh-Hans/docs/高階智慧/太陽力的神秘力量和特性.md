# 太阳力的神秘力量和特性
## 这段文字取材自《秘密教义》，内容描述了太阳中神秘力量的性质和特性

![cover](/img/downloaded/9e7a9f03e977bc171bd42a8372696d72.jpg)

Photo by [Rajiv Bajaj](https://unsplash.com/@rajiv63?utm_source=medium&utm_medium=referral) on [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral)

本文取材自布拉瓦茨基的《秘密教义》，阐述太阳中那股神秘力量的性质。欲深入探究，可参阅该书第一卷第五百二十四页起。迄今无人能如她这般详尽精确地描述此力——这在神秘科学中，向来是极度隐秘的一环。

布拉瓦茨基在书中详引了一位几近被遗忘的思想家兼科学家：塞缪尔·莱特勒·梅特卡夫（1798–1856）。这位美国化学家兼内科医生，著有《热质：其在自然现象中的机械、化学与生命作用》一书。维基百科自诩政治正确，却带著几分虚伪地宣称：「热质……是一种过时的科学理论，视热为一种自我排斥的流体，从热物体流向冷物体。」我们将证明，梅特卡夫绝非陈旧理论的作者；他精准描述了布拉瓦茨基所说的「太阳力」之性质，而这知识，仅为最高层次的启蒙者所洞悉。

我们讨论的起点是一个十九世纪的物理学家大多认同的前提（这一点直到21世纪的物理学家们仍未有新的发现）。自然界存在两种媒介：一是可量、可见、可触的「物质」；另一种则不可量、不可见，我们只能透过它对物质的作用感知其存在。梅特卡夫假设，有种活跃而不可量度的媒介，他称之为「热质」，并非单纯的运动形式，亦非可量物质粒子的振动。它是一种源自太阳、流布太空的物质基质，充斥整个太阳系，并透过感官传递一种名为「热」的特性。梅特卡夫列举此力的特性如下：

「它既能注入物体，亦能从中抽出，可用精确数学方法测度其量。

它使物体体积膨胀，一旦移除，体积便相应收缩。

它能改变一切物体的形状、性质与状态。

即便在完美真空中，它亦能辐射传播，且对温度计的影响，在真空或大气中并无二致。

它能产生无可抗拒的机械力与化学力，其强堪比火山爆发、火药爆炸或其他剧烈化合反应。

它以可感的方式作用于神经系统，引发剧痛；若过量，则导致肌体组织紊乱。」

梅特卡夫进而补充：「若太阳力仅是属性，便无法增加他物体积；故其自身必具体积，必占据空间。因此，它必是一种物质媒介。」此说更近神秘学法则，而非物质科学观点。他的假设基于一事实：太阳力或曰热质，是一种自我活跃的原则。它排斥自身粒子，却对一切可量物质粒子怀有亲和力。此力依距离平方反比作用，吸引任何可量物质粒子——这也印证了现代物理的平方反比定律。

梅特卡夫还提出一条他视为绝对的定律：「热质借由吸引可量物质，将万物凝聚；同时，凭其自我排斥之力，它又分离并扩张万物。」这，便是纯粹的神秘科学。

《大众科学评论》中，理查森博士评述梅特卡夫假说时写道：「梅特卡夫关于太阳力与地球力的假说，不仅简明，更别具魅力。宇宙存有二元素，一为可量的物质，另一为无所不在的以太——太阳之火，无重无体，无形无色。它是无限可分的质体，微粒相斥；除『以太』一词形容其稀薄，别无他语可述。它弥漫空间，充满万有，独存时静止，是死的。」

梅特卡夫对太阳能之描述，以及理查森对该研究的评论，大致符合已知的神秘科学原则。然而，正如布拉瓦茨基在脚注所言，他们的推论仍有谬误。「太阳力并非仅在太空流动，而是浸透太阳系每一角落，因它可视为以太的物质残余，是以太在我们层面上的内衬。以太不单是光的传播『媒介』，更肩负其他宇宙与物质世界的功能。它是卡巴拉学者所称的星光界流质，亦是『太阳–毗湿奴』的『七道光线』。」从神秘科学观之，理查森将太阳力称作「以太」并不妥当，因太阳能仅属以太众多原则之一；事实上，以太本身也只是阿卡莎或原初质诸原则的其中一项。理查森另有一误，即视可量物质为「死物」。物质静止时或显惰性，却永不死亡。

理查森继续写道：「以太渗入可量物质的微粒，与其结合，使之凝聚；物质亦溶解于以太之中。依梅特卡夫之说，这坚实可量之物质透过以太分布，延伸至当下一切存在。以太无所不在。人体满载以太；身躯的微渺粒子由它聚拢；植物亦然；就连最坚硬的土地、岩石、磁石、水晶与金属，也不例外。」若在理查森对梅特卡夫理论的总结中，将「以太」替为「星光界流质」，便与神秘科学学说无异。

神秘科学主张，不同物质对太阳力的吸收各异，主要取决于物质自身的多变状态；如固体、液体、气体。固体较流体更易吸收太阳力，因而凝聚力更强。若将部分熔化的锌倾倒于固态锌板，熔锌即转为固体。这是因为太阳力自液态流向固态，在平衡过程中，原本散逸的颗粒或液体变得更紧密结合。梅特卡夫对此现象详加探讨，并将其归因于作用原则的统一性，如前文所述。在一篇关于物质密度的评论中，他清晰总结了自己的观点。

他说：「硬与软，固体与液体，皆非物体本然状态，而取决于其所含以太物质与可量物质的相对比例。最具弹性的气体一旦被抽去热质，或将复为液体，再转为坚实固体。微粒间的黏合力，与其对热能吸收的亲和程度成正比。反之，若向最密实的金属注入足够的同成分，令其膨胀为气态，则其吸引力减弱，内聚力亦随之瓦解。」

梅特卡夫以如下话语总结其卓越理论：「我不拟深究此理论所隐含的太阳力与地球之同一性。但仍须补充，基于此假设——即将纯粹运动视作一种无形质、仅有特性而无实体之力——我们能从所有这些繁复深奥的问题中，推导出最贴近真相的结论：

(a) 宇宙、星际空间、行星之间、物质之间、有机体之间，并非虚空，而是充满微妙流体或气体。虽无更恰当之名，我们仍可如古人般称其为「太阳之火」或「以太」。此种液体成分恒定、不可毁、不可见，却无所不在，渗透所有可测物质。溪中卵石、摇曳树木、凝视之人，皆受以太影响；卵石少于树木，树木少于人。此行星之万物皆以同样方式带有电荷！世界筑于以太流体中，航行于其海洋。

(b) 以太，无论本质为何，皆源自太阳。太阳不仅产生以太，更是其储藏之所与传播媒介。

(c) 若无以太，运动不存；若无以太，物质粒子无法彼此滑动；若无以太，激发粒子运动之力亦将消逝。

(d) 以太是决定物体结构之关键。若无以太，物质的组成变化无从发生；例如，若无以太，水仅能以一种紧密不可溶的形态存在，超乎我们一切认知。它将无法成冰、化液，甚至转为蒸气。

(e) 以太连结太阳与行星、行星之间、人类与行星、乃至人与人。在没有以太的宇宙中，信息无从传递；无光、无热、亦无运动现象。」

布拉瓦茨基夫人所言极是：「以太仅在显现阶段时不可改变，之后便永远复归『不可见』之中。」尚可补充：太阳虽是太阳系中以太的储库，却非此力之源。此力源自核心太阳，即宇宙中那不可见、不可窥的心脏，我们的太阳不过是其映象。

若塞缪尔·梅特卡夫的太阳力量理论获得认同，且他关于太阳内存有生命的大胆推论被接受，这将证明神秘科学的启示正确无误，远早现代科学理论数百万年。
