# 神秘的宇宙时代
## 简介印度教四时代概念，剖析迦梨时代的长度与隐秘数字，论及各年代对人类的寓意与影响。

![封面图片](/img/downloaded/cbfcf74e13f4ef345cb291649d0839be.jpeg)

印度教阐述四个时代，合称「大时代」。这大时代不过是「大显现期」——宇宙生命周期——中极其短暂的一瞬。

四时代名称与年数如下：

1.  萨提亚时代（又称克里塔年代）——黄金时代——1,728,000年
2.  崔塔时代——白银时代——1,296,000年
3.  德瓦帕拉时代——铜时代——864,000年
4.  迦梨时代——铁时代——432,000年

总计：4,320,000年（即一整个大时代）

依印度教传统，我们现正处于迦梨时代。据说此时代始于约五千年前，适逢克里希纳离世之时。克里希纳是伟大的神圣导师与改革者，亦是《博伽梵歌》的核心人物，确有史实可考。

布拉瓦茨基在《秘密教义》中指出：「迦梨时代据说始于公元前3102年2月17日至18日之交。」她更进一步澄清：「确切而言，始于3102年2月16日凌晨02:27:30。」至今已历5116年。

简单推算，迦梨时代尚余426,884年，一段近乎无法想像的漫长岁月。然而，这些数字当真仅作字面解读？威廉·贾吉论「周期」的文章，揭示了时代年表下隐藏的数字密码。

萨提亚时代据称长达1,728,000年。将1、7、2、8相加，得18。
崔塔时代据称长达1,296,000年。将1、2、9、6相加，得18。
德瓦帕拉时代据称长达864,000年。将8、6、4相加，亦得18。
迦梨时代据称长达432,000年。将4、3、2相加，得9。

18+18+18+9，总和为63。将6与3相加，得9。6乘以3，得18。再将1与8相加，复得9。其中显然蕴含深意，然贾吉、布拉瓦茨基乃至其他大师，皆未点破其真正玄机。贾吉在其重要著作《神智学的海洋》中明言（第126页）：「我们正处过渡时期，哲学、宗教与社会皆在流变。于此阶段，关于各周期的完整数字与法则，不会透露予这一代人；现代人将金钱置于一切思维之上，对人与自然所怀的灵性视角嗤之以鼻。」

上述数字虽源于印度外传教义年表，广为流传与引用，连布拉瓦茨基亦称其「几与神秘哲学所授相同」（《秘密教义》卷二，第67页），我们最好避免拘泥字面。此理同样适用于根种族那惊人漫长的存在期；据称自亚特兰提斯主体最终沉没，已逾八十五万年。

何以如此断言？原因有二。首先，除前述《神智学的海洋》之声明，布拉瓦茨基本人在《秘密教义》中亦写道：

「我们自始便声明，且一再强调：任何神秘学研究者，即便是被认可的弟子，亦不应期待有人会对其全面揭示秘密教义（对外行人而言更是如此），除非他们向兄弟会作出不可反悔的承诺，并至少经历一次启蒙。任何秘传数字皆不得泄露于众，因这些数字乃神秘传承的核心。」（卷一，第164页）

「神秘计算的数字不可外流，诸多大师强调，只能传授予立誓的弟子；即便如此，弟子亦须严守此规。」（卷一，第170页）

「我们未被告知大劫的长度，亦不获准公布各小时代的确切持续时间，仅能提供概略范围。」（卷一，第206页）

「我们并未习得那些神秘科学大师永远保密的数字，如《秘传佛教》所言。」（第二卷第251页）

另一缘由在于，布拉瓦茨基曾在一篇鲜为人知的短文中，揭开了真传体系的一角，指出时代实有七个；这真相早已映现在神秘学徒的直觉里——他们深知一切显现皆具七重本性，并懂得运用对应与类比之法，此乃领悟宇宙奥秘的枢纽。

历经黄金、白银、青铜、黑铁诸时代（「那恐怖黝暗的世代」——《秘密教义》第一卷第645页），我们并非重返黄金时代，开启又一轮四重循环；大时代实则沿著返本与演化的弧线行进，一如我们行星链中七颗星球的七轮次。有一道逐渐沉降的弧，始于纯净、崇高却近乎无觉无识的状态，直至坠入谷底。随后曲线徐徐回升，重返起初的高度，但这一次，它携著珍贵的觉知、智慧与灵魂的历练。此即所谓「阴影之弧」与「光明之弧」。

这讯息见于一八八四年一月，布拉瓦茨基发表的一篇短文，题为《过早而异常的增长》。或许无人料想，如此标题的文章竟藏著关键线索。然而，正如本站文章屡屡指出的：正因如此，我们更该细心严谨地对待布拉瓦茨基所写的每一字句，在她浩瀚著述中，从无一句一段是闲笔。

该文收录于《布拉瓦茨基神秘学文集》首卷，其中引述了她所谓「某些亚洲古书的预言」。其内容与风格，颇似数年后《秘密教义》两卷中许多深邃的注解，尤以第二卷《人类起源》为显：

「正如第四（种族）由『红–黄』渐化为『棕–白』（躯体），第五种族亦将逐渐转为『白–棕』（白人渐黑）。第六与第七人类降生即为成人；他们寿命漫长，却不显衰老。正如在萨提亚、崔塔、德瓦帕拉与迦梨诸时代，身心渐次衰颓，于德瓦帕拉、崔塔与克里塔的上升中，卓越亦将逐步复归。人类寿数在萨提亚世代为四百年、崔塔世代三百年、德瓦帕拉世代二百年、当今迦梨世代仅百年；故于将来（第六种族）之自然年寿，将（逐步）递增至二百年、三百年，乃至四百年（于最后两世代）。」

由此可见，依次是萨提亚时代、崔塔时代、德瓦帕拉时代、迦梨时代，继而再一轮德瓦帕拉时代、崔塔时代，终以萨提亚时代完结这七重循环。

显然，大师们如何将七时代与七根种族相系或对应，尚未全然揭示，但此刻我们无意急于探明。诸位自可凭直观与冥想，细究此间奥义。

虽无法确言迦梨时代或任一时代实际长度，神秘学文献中有一点却清晰无疑：当迦梨时代首个五千年周期告终，将是人类演化极其关键而重大的时刻。此说多处可见。贾吉于《曾助我之信》（第97页）中写道：「当前周期终于一八九七年十一月十七日至一八九八年二月十八日间，乃历来最重要之周期之一。」

以下摘录自《古代智慧》一文：

「十九世纪最后二十五年，是人类历史与演化至关紧要的关口。三大周期在此交汇，这般情形极为罕见。迦梨时代首个五千年周期于一八九七年末至一八九八年初始终结。双鱼座时代正急遽褪色，而自一九〇〇年起，新世纪的曙光升起，水瓶座的新时代将徐徐展开其长达二千一百五十五年的历程。

「十四世纪的西藏，宗喀巴——乔达摩佛陀之转世——嘱其秘传兄弟会，此后每世纪末二十五年，须遣一成员西行，以助引动更深层的灵性觉醒与启蒙。众所周知，唯特定之二十五年能积极成就此事，此乃周期与业力法则的真实局限。

「大师们称之为『百年周期』，与另两周期同时终结。必有人现身；必有人被遣往世间；一位具足力量、知识与能耐者，将为全人类带来真切而持久的变革，不独为西方或东方，而是为整体人类而来。

「必须将人心中的『时代积垢』彻底扫除，真理的种子方得撒落，结实累累。大师及其使者深知，前方横亘著巨大的艰困、阻碍与反对——或明或暗——皆将试图拦阻使命的完成。

「于是，一位伟大的『某人』借由俄裔女子布拉瓦茨基之身显现。少数亲近布拉瓦茨基（她宁愿自称HPB）者渐次察觉，她实为一位东方开悟者的男性化身，为最大程度践行大师之命，而采此欧陆女性之人格。大师们指明，世人虽知她为『布拉瓦茨基』，大师们却以另一名字称呼她。他们宣称，她是他们的『兄弟』与『直接代行者』。」

一八七五年，她与数人在美国纽约创立神智学运动。其后迁居印度，最终定居英国伦敦。而今，正是重新揭示古代智慧之时——以「神智学」为名，此词源于希腊文「神圣智慧」。

这仅标示迦梨时代最初五千年的终结，并非整个时代的终点。新纪元运动普遍认为，我们已步入黄金时代，甚或始于二零一二年。此说不仅缺乏哲学根基，更显荒诞。那不过是受蒙蔽的通灵者与「光之工作者」的狂想，毫无实据。正如布拉瓦茨基曾言：「最后的时代——铁时代或迦梨时代——结束了吗？恰恰相反，它的影响方兴未艾；不仅因印度教徒沿用此名，日常经验亦处处可见。……我们难道真处于所谓『黄金时代』？在黄金时代里，蛇毒与草木之毒皆不再致命，人人安居于神选君主的温和统治下？若将此概念套用当今之世，或过往任何一年，怕是荒谬绝伦、最不贴切的描摹了。」

岁月推移，我们正更深陷落迦梨时代，且「在这极度唯物主义的『黑暗时代』中，不会出现新的人类救世主。」（《秘密教义》第一卷第四七〇页）

所幸，神秘学不但示人如何自救，亦揭示：需要拯救的是灵性上的无知黑暗。真正的「救主」，是指引导自我解脱之路者。如佛陀所言：「无人能救赎他人。」「诸佛不过指路标；你得自己前行。」神秘学认为「佛陀是近期最后一位伟大化身，且其所处周期远长于犹太人耶稣的时代。」（威廉·贾吉，《神智学的海洋》第一一九页）周期之中又有周期，我们此刻，正处某些神秘学家所称的「佛陀周期」。布拉瓦茨基的著作与教诲，意在重申乔达摩佛陀的信息——那位被大师们誉为尘世间至伟至圣者。

《秘密教义》第一卷第三七七页，布拉瓦茨基写道：「奇妙的是，《毘湿奴往世书》的作者向弥勒佛预言迦梨时代的黑暗影响与罪恶时，几已说尽一切。」她继而引述这部古印度经典对迦梨时代的描述：

「财富与虔敬逐日稀薄，直至世间全然堕落。唯财富赋予地位；财富成唯一奉献；情欲是两性间仅存的纽带；谎言乃诉讼取胜之途；女子仅被视为感官对象……生命层次的区别，只剩外在形貌；人若富有，便被看作纯洁；虚伪成了普遍生存之道，软弱招致依赖，威胁与傲慢取代学识；慷慨等同奉献，合意即成婚姻，华服便是尊严。最强者为王；百姓不堪重税，遁入山谷栖身……如此，迦梨时代的衰朽将持续，直至人类濒临绝灭（即休止期）……当迦梨时代临终，那神圣存在与灵性本质的一部分……将降临世间……（迦乐季化身）获赋八种超凡之力……他将重树人间正义，并在时代终末唤醒人的心智，使之澄明如水晶。」

《神秘学对话》第一册中，贾吉记录自己与布拉瓦茨基的问答——他以「学生」自居，她则饰「圣人」。该篇题为「迦梨时代：现今时代」。圣人于对话中说道：「当今迦梨时代有一殊胜处，可惠及求道者。相较其他时代，甚或更美好的年代，此时因果循环显现更快。人在迦梨时代历经三次轮回所达成的，远超其他时代所能企及。故此，若能忍受此时诸般艰难，稳步取胜，其目标将更快实现；纵然障碍看似顽强，所需力量亦会更早获得。」

学生追问：「虽说这时代灵性晦暗，但心智日益驾驭物质，科学亦疗治诸多人间疾苦——包括病因、病体、残酷、偏狭、不公之法等——这些难道不算某种补偿吗？」

她答道：「是，那确是黑暗中的一丝微光，如夜里点灯，终究还不了白昼的明朗。这时代科学成就斐然，但多指向罪恶之果，未除罪恶之根。尽管现代人在艺术与疗疾上进步显著，未来随著文明之花盛放，仍将涌现更多源于人心深处的奇症异疾，唯赖灵性生活方能根治。」

这场对话最终提问：「除了推广神秘学，还有何法能扭转当今唯物趋势？」

圣人答道：「传扬业力与轮回的法则，相信万物本属同一灵性源头，便能遏制此势。然此事须循序渐进；所有有益的因素，都将在终了前逐渐发挥作用，虽然效果可能不如在繁荣时期那样显著。每有学子过著更美善的生活，其风范在星界流体铭刻更高愿景，此愿景便于世间运转；故而能助若干高阶灵魂自他界降临。当今世道晦暗太甚，高阶灵魂实难久留。」
