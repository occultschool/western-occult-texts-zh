# 三个逻各斯
## 真理只有一个，圣人却用了许多名字称呼它。

![封面](/img/downloaded/a42ae3844edbec1e5ae684e6f6535e13.jpeg)

**问：** 读完[《逻各斯的哲学基础》](/zh-Hans/docs/logos-and-mysticism-philosophical-foundations)一文，我对「逻各斯」的实质更清晰了——那些内容源于布拉瓦茨基与大师的教诲，并非后世「教师」编造、严重曲解的产物。但布拉瓦茨基常提及「第一逻各斯」、「第二逻各斯」、「第三逻各斯」，这些术语究竟何指？我原以为逻各斯只有一个，即那「至一全灵之光与生命」。

**答：** 逻各斯的确只有一个，那便是宇宙逻各斯。每个大显现期之初，它自「绝对者」辐射而出——无论你称那绝对者为「梵」、「本初佛」、「无限」或其他名号。逻各斯是「绝对者」辐射的生命原则与进化能量，继而化为活跃的宇宙本身。

正如另一篇文章所述，布拉瓦茨基与大师从未使用「行星逻各斯」一词，仅有一次提及「太阳逻各斯」，用法也与伪神秘学家截然不同。真正的神秘学谈论的是宇宙逻各斯，那是「第一因」；而「绝对者」是万有的「无因之因」、「无根之根」。这个宇宙逻各斯，也称作「核心灵性太阳」或「伟大核心太阳」。

若要弄清这三个逻各斯，不妨留意布拉瓦茨基在[《秘密教义阐释》](/books/secret-doctrine-exegesis)中的定义。顺带一提，该书是不可或缺的优秀资源，厘清了《秘密教义》中诸多内容，包括逻各斯、七光线、行星神灵、绝对神性原则、《多基安集》、时间与永恒等，还有一章启发人心的论梦篇章。

书中，她将三个逻各斯称为「进化的三个灵性阶段的拟人化象征」。

因此，它们实际上只是符号，旨在呈现关键概念供人领会。现在，让我们特别关注这三个逻各斯的几个要点：

第一逻各斯总被称为「未显之逻各斯」。其象征是圆心一点。圆圈代表「梵」那无边无际的绝对性，圆心一点则标志著宇宙分化的起点（或者说重新开始），发生于大休止期之后。宇宙大休止期中空无一物——「既无声响，亦无静默」——只有未分化、无制约、无限而绝对的梵本身。

未显的第一逻各斯也称「最高逻各斯」。据说它超越时间与空间，仅是「潜在的潜能」。第一逻各斯是宇宙的「潜能」，第二逻各斯则是宇宙的「效力」。第二由第一发出。由于第一逻各斯只是潜伏、潜在而未实现的状态，它极其接近「绝对者」，是后者的直接辐射。除此之外，无法多言。但在神秘学教义中，我们注意到，作为人类最高构成原则的「阿特曼」，有时对应于未显之逻各斯；尽管多数时候，阿特曼被等同于「绝对者」。这并不矛盾，反而是一种明确的暗示。

第二逻各斯是「显现的逻各斯」。它本身其实只是「半显现」，因为仍需透过第三逻各斯才真正显现。在[《秘密教义阐释》](/books/secret-doctrine-exegesis)中，布拉瓦茨基说它是「普遍而智性的灵魂，是神之意念，蕴含万事万物的理型蓝图与原型，不管是客观或主观的显现世界。」

第一逻各斯被描述为「潜在的潜能」，第二逻各斯则被描述为「分化的意识」。我们得知，当第二逻各斯自第一逻各斯流溢后，「时间」与「空间」便开始了。未显现的是潜能；显现的是效力。

第三逻各斯便是「宇宙心智」，印度教称之为「伟大」。正如印度寓言所示，神之心智产生了「七位心智所生之子」，这些儿子在宇宙中扮演主要而重要的角色。这七者的总和（实际是七道光线、七种力量或七种作用力），便是神秘学所称的「第三逻各斯」。

第三逻各斯又称七光线、七创造力或七逻各斯。如前所述，第二逻各斯本身便涵容第三逻各斯，并借由它而显现。若将第一逻各斯视为潜存的能量，第二逻各斯为分化的意识，则第三逻各斯可谓第二逻各斯最终的展开——它以宇宙力量个体化之姿，从中「发展出无穷的阶序序列」。

这「七者」实为宇宙七种奥秘之力，将宇宙心智中潜伏的主观原型化为现实，使之客观显现。换言之，宇宙建构的蓝图，由它们实现。

问：我想我掌握了这番解释的要义。但这说法是否太过抽象，太形而上学？

答：确实如此。然而，当我们谈论宇宙诞生与演化的课题时，是如此形上且宏伟，又怎能使用其他语汇？任何简化、拟人或人格化的尝试，只会遮蔽它、亵渎它。我们必须开始以形上、抽象、哲学的方式思考，这终将提升我们的意识，抵达更高、更真实的灵性层面。大师们深知这是我们所缺，因此《秘密教义》主要以此风格写成。脱离哲学的灵性，往往只是蒙昧。大师们在信中提到，此时「正是一个更具灵性智慧的周期的黎明」。我们该成长了，该真正运用天赋的思考能力，而非停滞于现今的寻常状态。

问：你提到布拉瓦茨基曾说，三个逻各斯其实是「演化的三个灵性阶段的拟人化象征」；照你所说，是否实际上并无三个不同的逻各斯，而是一个逻各斯经由三个阶段推动宇宙演化？

答：我相信这样理解正确——只要我们始终记得：那「推动宇宙演化的至一逻各斯」并非某种存在、实体或神灵，而是宇宙生命本身的能量；它依循永恒法则的内在脉动而回应、行动。此外，这「三个阶段」划分，也不似我们有限心智所期望的那般独立分明。我们总习惯将事物切分清楚、放入整齐的格子；但稍加思索便知，大自然万物多非如此。

问：这三个逻各斯是否有各自的名称？我确信逻各斯本可无名；但若为说明方便，用什么名称描述它最恰当？

答：你说得对，逻各斯可以无名，正如「绝对者」必然无名。实际上也无须使用具体名称，「第一逻各斯」、「第二逻各斯」、「第三逻各斯」已足够。但我们仍能从各宗教与哲学中见到逻各斯的种种称呼。正如你先前提到的文章所说：

「不同灵性传统所用的诸多名称，或会令人困惑；除非我们牢记，这些名称指涉的都是同一逻各斯。《奥义书》哲学中，梵天即逻各斯，源自绝对之『梵』。卡巴拉中，天人亚当卡蒙是逻各斯，源自『无限』。密传佛教里，观世音菩萨是逻各斯，源自本初佛。有印度教徒将『绝对者』及其逻各斯对应于湿婆与莎克蒂，亦有人以毗湿奴、那罗延或自在主称呼逻各斯；更有人称逻各斯为宇宙神圣母亲。实际上，神秘学指出，逻各斯与所谓『逻各斯之光』不可分离，二者在印度思想中对应于自在主与莎克蒂——宇宙的男性能量与女性能量，后者传递前者的光、生命、能量与力量。」

须谨记，这些名称并非指不同存在或实体，只是描述那赋予宇宙灵魂与生机的「逻各斯原则」。布拉瓦茨基写道：「在秘传哲学中，逻各斯仅是一个抽象术语。」

在秘传学说中，未显之逻各斯常称那罗延。那罗延是毗湿奴的别名，可解释为「神之意念的灵在水面移动」。还记得《圣经·创世记》开篇说，世界未成之时，「神之灵」运行在「深渊之水」上吗？更古的印度经文则载，那罗延在抽象空间的无垠水面移动，这正是「最初显化的颤动」，令宇宙在大休止期后循环重现。

不过，布拉瓦茨基有时称梵天为第一逻各斯，有时又说是第二逻各斯。因此我们最好以她为榜样——不必过于执著名称，而应关注名称背后的思想与概念。

七光线的总和，是为第三逻各斯。这概念在印度教中对应七鸠摩罗，即七位「心智所生、梵天之子」，或曰「宇宙心智之子」；在佛教是七禅那佛，在基督教是七大天使，在犹太教是七埃洛希姆，在卡巴拉则为七个下层质点。名相虽异，所指实同。正如《梨俱吠陀》这部人类最古老的经文所揭示：「真理唯一，圣人却以诸多名号呼之。」

简言之，三逻各斯作为「进化三阶段的灵性拟象」，精确对应于人类物质演化的三阶段：先是受孕，这不可见的神秘过程，是胚胎形成的初阶，尚未显化；继而妊娠，胎儿在子宫内成形生长，始终隐于内里，半显半藏；历经前两阶段后，人——这宏观世界的缩影——终于诞生，客体全然显现。

宇宙逻各斯之说源远流长，自然非自布拉瓦茨基开始。但研习比较古代哲学后便会察觉，她关于逻各斯的教义，清晰而准确地重述了这永恒真理。若欲更深入、更全面地理解与探索，则必须研读《秘密教义》，尤其是其中论及神秘象征语言与「逻各斯」的章节。
