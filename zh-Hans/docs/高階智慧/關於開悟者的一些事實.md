# 开悟者的一些事实
## 大师的内在真相：开悟非无谬，错误亦是灵性旅程

![cover](/img/downloaded/7c3f4c5f6412d40afef13a6533641ad1.png)

下文摘自K.H.大师于1882年末写给A.P.辛尼特的一封信。此信值得细读深究，因其揭示几项重要事实：

一、大师与开悟者，亦可能犯错。
二、达到开悟的，是内在实体、灵魂，而非外在人格；此实体于前世今生，皆是那开悟者。
三、悟者或大师，是那看不见的永恒个体，并非此生可见的外在人格。
四、一位转世为人的开悟者，不会令内在灵性自我时刻全面运作，既不便，亦不必。
五、因此，偶见开悟者与大师犯错，亦不足奇；这些错误不在哲学、意识形态或形而上层面，多是些微末小事。

值得注意的是，真正的K.H.大师谦逊、诚实、直率，绝非所谓「扬升大师库图米」——那只是虚构，从未存在。后者却在通灵传讯中颇为流行，甚至新纪元运动中许多受惑者对他崇拜祈祷；他们对现实一无所知，或毫无兴趣，宁可沉溺幻想。

将此话公诸网络，用意不在鼓励我们批判大师，而是帮助我们更精确地理解大师，理解他们是谁、做些什么、如何运作。事情不全然如我们所想。

这位伟大的跨越喜马拉雅山启蒙者，在1900年《最后大师信》中所言：「关于『大师』的空谈（那些虔敬、神圣、宗教式的言论）必须被默默而坚定地制止。让我们的奉献与服务，只专注于那『至高灵』，人人皆是『至高灵』的一部分。我们默默工作。若不断提及己名，只会引来紊乱的氛围，阻碍我们的工作。」此言当永志不忘。

「在我们现阶段的沟通中，尽管生出误会，甚至你对我有所误解，我诚挚的朋友，或许值得一试，这对双方都有助益：你应当了解某些事实，这些事实与开悟程度息息相关。因此，请记住以下几点：

（一）一位开悟者，无论身分高低，唯有在运用其神秘力量时，才真正显出开悟者的本色。
（二）当需要这些力量时，主导的意志便为内在之人（那开悟者）敞开门户，让他得以自由现身行动。然而，前提是他的狱卒（外在之人）必须处于完全或部分瘫痪状态，情形包括：（甲）心智与身体皆然；（乙）仅心智受限，身体不然；（丙）身体受限，心智未全限；（丁）二者皆未受限，但外在与内在之间隔著一层阿卡莎薄膜。
（三）诚如你现在所见，即便动用微小的神秘力量，也需费力。可比作运动员发力前，肌肉的紧绷。没有运动员会无故令血管膨胀，只为预期举重；同样，在无实际需要时，开悟者也不会令意志持续紧绷，或让内在之灵全面运作。当内在之人歇息，开悟者便复归常人，局限于感官与大脑功能。习惯虽能提升物质大脑的直觉，却无法使之成为超感知。内在的开悟者时刻准备著，时刻警醒，这便足够。歇息时，开悟者的感知也似一同放松。例如，当我坐下用餐，或穿衣、阅读、处理杂务时，甚至完全未察觉周遭有人；又比如，前几夜朱尔·霍尔在黑暗中不慎撞上梁柱，伤了鼻子。那是因为他与远方友人交谈时，不觉让所有外部感官陷入麻木——而我对此浑然不知，平静得仿佛与我无涉。我未想到他，因此也就不知晓此事。」

「综上所述，你当能清楚理解：在日常生活中，开悟者如同普通人，唯一不同，在于他们内在的运作方式。」

「当KH来信时，他尚未开悟。凡人犯错本属常态。因此，KH笔下极易出错：标点一误，句意全非；用词失准，尤在仓促行文时更甚；偶有术语混淆之失，我尚须向你请教——毕竟『轮次』、『环』、『地球环』诸词，本由你创。我想说的是：在细读了所谓『著名矛盾』之处，经M先生审视，也提交给一位高等开悟者核阅，最终得知如此讯息：『一切无误。我明你的意思，对此教义比旁人更熟稔；这些散落片段中，实未见真正冲突。然因句多残缺，主旨芜杂，你这位「门外」弟子能指出谬误，亦不足奇。是的，这些阐述确需更明澈、更清晰。』」  

「此乃开悟者之信念，我当忠实遵循；为你，我必竭力完成此任。」  

\*「……我须向你学的术语——因你是『轮次』-『环』-『地球环』等词之作者。」  
如行星链、星球、轮次、根种族等概念，本属大师教义，然昔时英文并无相应专名。辛尼特于早期与KH大师、M大师书信往还之际，承担此任，创制若干词汇术语，后为大师及其所谓「直接代理人」布拉瓦茨基所纳用。
