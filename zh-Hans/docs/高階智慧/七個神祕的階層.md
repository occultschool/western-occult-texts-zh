#  七个神秘的阶序
## 这套秘传教义，所揭示的尚不及全貌十分之一。

![cover](/img/downloaded/60efdbb188d2d0cc4e7ff68e2bcd6b37.jpeg)

研读《秘密教义》的学生会记得，书中大师与布拉瓦茨基提及十二种伟大的「天界存在」秩序，其中包含七个阶序、群体或秩序。它们有时被称为「天界阶序」、「禅那主阶序」或「天人阶序」。

这些内容大多人难以领会，除了熟读《秘密教义》的学生外。他们对这部书钻研至深，方能看懂看似无意义的章节。《秘密教义》必须视作整体研究，才能得其真义；也应置于布拉瓦茨基思想的完整图景之中——连同《揭开伊西斯的面纱》、《神圣智慧之钥》、《寂静之声》等核心著作，一并考量。

《秘密教义》第一卷《宇宙生成论》第213至233页，稍稍揭开这个深奥主题的面纱。首先，「创造力」划为七个（或说四加三个）神秘阶序，存于十二大秩序内，并记录于黄道十二星座；此外，七个显现阶序又与七颗行星相连。这一切再细分成无数神灵、半神与空灵存在。

因此，共有十二种伟大的灵性存在秩序，某种程度上对应黄道十二星座：白羊、金牛、双子、巨蟹、狮子、处女、天秤、天蝎、射手、摩羯、水瓶、双鱼。

十二大秩序中，存在七个「创造力的阶序」，与七颗神圣行星相连：金星、水星、火星、木星、土星，以及另外两颗据说是太阳与月亮——实则是神秘的祝融星，与另一颗神秘学未具名的行星。但布拉瓦茨基在[《秘密教义阐释》](/books/secret-doctrine-exegesis)中描述后者为「一颗逆行的行星，夜半时分偶尔可见，似在月球附近。其神秘影响经由月亮传递。」

以下是关于七个阶序的记述……

\#1 — 「最高等的群体由神圣『火焰』构成，亦称『火狮』与『生命之狮』，其神秘面向深藏于狮子座中。它是高等神圣世界的核心。它们是无形的『火热气息』，某种意义上等同于卡巴拉『原型世界』的上层三质点。」

\#2 — 「第二阶序的『天界存在』属火与以太，对应灵与灵魂，或称『阿特曼-菩提』。其名目繁多，虽无形体，却更具『实质』……如其名所示，他们是投生单子的原型，由生命的火热之灵构成。光线穿过他们，如纯净阳光，他们输送光线与其未来载体，即神圣灵魂。这群体与我们太阳系的高等世界直接相连。从这个双重单元，流溢出三重单元。」

\#3 — 「第三阶序对应『阿特曼-菩提-心智』，即灵、灵魂与心智，被称为『三元组』。」

\#4 — 「第四阶序是实质性的存在体，为原子形体中最高等的一组。它是人类灵魂之温床，具有意识与灵性。他们被称为『不朽的灵魂』，并透过其下的阶序，构成最初的七重群体——此乃人类意识与智性存在的伟大奥秘。」

由于第四阶序居于七阶序中间，或许正是「禅那主身体的『心』，据说曾投生于人类第三种族，使其臻于完美。」（《秘密教义》第二卷，第91页）

#5 — 第五阶序极为神秘，它与微观世界的五角星相连，而五角星象征人。在印度与埃及，这些天界存在与鳄鱼相关，居所位于摩羯座。印度占星中，这两个词可以互换，因为黄道第十二宫（摩羯座）直译即为「鳄鱼」。……这第五组天界存在被认为具备双重属性：宇宙的灵性面与物质面，可说是「宇宙智性」的两极，也对应人类灵性与肉身的双重本质。因此，其数字五乘以二成为十，正与黄道第十宫「摩羯」相系。……那些执掌摩羯座——在印度与埃及称为「鳄鱼」——的神秘存有，其使命是教导空灵缥缈的动物形体（即地球第四轮次早期的人类），使其进化为理性之人。这是公众鲜少谈论的主题。它确实是个奥秘。有些人只将意识完全局限于人类、视意识仅为「大脑功能」；他们拒绝承认宇宙中存在著具备智性与意识的灵性生命。自人类诞生之初，便有无数灵体投生于肉身，同时，且如以往般独立存续于无限的空间。……更明确地说：这些不可见的实体以肉身居于尘世，同时也不失去其在超感知领域的地位与职能。

# 第六与第七阶序占据四元组的下层性质。他们是有意识、空灵的实体，如以太般不可见……在灵性下行的尺度上，第六组是最后第七组精灵的前一阶，他们逐步形塑、构筑并凝聚自己的躯体。这第六组的演化形成了人类朦胧的形体，如一层薄膜、近乎透明的复制体。……此外，第六组与人几乎不可分割，人汲取了此组全部原则，如除了至高（灵性）与至低（肉体）外；五个居中的人类本质，正是「天界之人」的真髓。神圣之光（阿特曼）直接源自「至一」。

必须牢记这关键一点：「正如教义所示，宇宙中并无享有特权的存在，如西方宗教与犹太教天使那般；无论此存在处于太阳系或他系，在外显世界或内在之境。一位禅那主须经修持方能成为禅那主；他无法在诞生之际、或显现于生命层面时，便立刻成为完全发展的天使。当前显现期中的『天界存在阶序』，将在下一生命周期中迁移至更高、更优越的世界，并为由人类菁英组成的新阶序腾出空间。『存在』是绝对永恒中无尽的循环，其内运行著无数受限且有制约的内在周期。」

《秘密教义》第一卷第292–293页引述舒巴罗《黄道十二星座》，其中列举六种宇宙力量的名称与简述，并由第七种综合代表——即「逻各斯之光」。

「宇宙的七种主要力量整体亦称七道光线，包括：(1) 至上之力 (2) 智识之力 (3) 意志之力 (4) 思维之力 (5) 火焰之力 (6) 咒语之力。」布拉瓦茨基随后指出，这些「皆是秘传教义，所述尚不及可言之十分之一。此处所提六力之名，即那六阶序的禅那主，统合于至高的第七力。」它们是宇宙大自然（神秘意义中的「母亲」）第五原则的化身。仅列举瑜伽力量便需十卷篇幅。每一力量顶端皆有一活跃的意识实体，乃是该力之流溢。

布拉瓦茨基与大师们屡次强调，对应法则与类比法则统御万有；欲清晰准确领悟秘密教义，此二法则是不可或缺的关键。
