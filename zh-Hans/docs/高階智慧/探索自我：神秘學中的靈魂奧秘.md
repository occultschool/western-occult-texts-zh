# 探索自我：神秘学中的灵魂奥秘
## 理解「自我」的真谛：从人格到高等心智的启示之旅

![cover](/img/downloaded/2679e25956111b7a871b282b6e40da47.png)

「有意识的自我（EGO），或称作人类之魂，其中藏著一个巨大的奥秘。」

\- 布拉瓦茨基，《秘密教义》第二卷

现今许多流行的灵性教导，常贬低「自我」一词，令人困惑不解。然而翻阅神秘学的典籍文献，「自我」一词频繁出现，却不带半分负面或批判的意味。

「自我」真是个负面的词吗？我们是否真如许多人强调的，必须「摆脱自我」、「克服自我」，甚至「摧毁自我」？

要回答这些问题，先得厘清两件根本之事：

(1\) 「自我」字面即「我」。一个人活在尘世，若对内在毫无「我」的感知，便不可能过上有意识、有智识的生活——无论此人如何声称已「抛下」自我。我们每个人都是「我」，这事实无从逃避。我们并非拥有自我，我们本身就是一个自我。

(2\) 布拉瓦茨基在《神秘学词汇》（第111页）中阐释：「秘传哲学教导，自我有两个：一个是凡俗的人格自我，另一个则是更高、神圣、非人格的自我。前者称为『人格』，后者称为『个体性』。『自我』意指『个体性』，绝非『人格』；它与『自我主义』或『自私』恰恰相反——自私，才是人格的特征。」

在神秘学的语境里，那随著我们不断转世的部分，常被唤作「自我」。它被视为我们的「永恒个体」，是真正的「我」，是那看不见的思想者——一个存续于生生世世的「心智实体」。真正的思考者、行动者、实践者、体验者，创造并承担业力的，正是它。

它借由「当下的人格」来行动。这人格包括了个性心智（即日常的大脑意识），以及用以运作的诸多低等载具或面向：欲望本质、生命能量、精微的星光体，还有我们最熟悉不过的肉体外壳。

那永存的个体性，即转世的自我，被称为高等心智；而在每一次崭新独特的投生中，当下的人格则被称为低等心智。这人格是我们自身业力的产物，是过去思想、言语、行为的结果。根据神秘学，心智原只是一项原则，但在投身尘世时，它便显出双重性质。

也就是高等自我与低等自我。古希腊哲学将二者分别称为「心智」(Nous) 与「赛琪」(Psyche)，对应于我们存在中的理智元素与心灵感应元素。布拉瓦茨基在《心灵感应与智性行为》一文中，以极其深刻、启人心智的方式探讨了这个议题，那是她最深奥的文章之一。

「我们相信同一个自我在一连串转世中发展，这观念是我们整个思想体系的核心。例如，所谓『史密斯先生』并非一个独立存在的个体，而是无数记忆线索串连起的日常经验集合，这集合构成了史密斯先生所认知的『自我』。然而这些经验并不能真正代表『我』，因为史密斯先生会遗忘大部分日常经历，唯有在回忆时，才感到『自我』存在。因此，我们称这经验集合为虚幻的人格（因其有限且短暂），用以区别那些引发『我是我』感觉的内在元素。我们称那『我是我』为真正的个体性；同时，我们将这个『自我』或个体性比作一位演员，在生命的舞台上扮演种种角色。每一次新生在世间，都像剧院里的一场戏。某个夜晚，这位演员或『自我』以『麦克白』登场；隔天化身『夏洛克』亮相；再下一回，又以『罗密欧』的姿态现身。如此轮转，直至完成整个投生的周期。自我踏上生命旅程，先是精灵，接著是警官、士兵、仆人、合唱团员；然后晋升为『叙事者』、担当领袖，其间也饰演些无足轻重的角色，最终，以魔法师『普洛斯彼罗』的身份，从舞台从容退场。」

— 布拉瓦茨基，《神圣智慧之钥》 p. 33–34

人心中有一种内在元素，催生出「我就是我」的感觉。这「元素」正是我们内在的心智成分，即心智原则。它代表我们的灵魂，是我们永恒的灵性个体。

在布拉瓦茨基《神秘学词汇》第154至155页，「个体性」条目如此定义：「人类高等自我的称呼之一」。永恒的神圣自我，与终将消逝的凡人自我在此区分。肉体死亡后，凡人自我——那人格之我——仅在欲界残存片刻；个体性却长存不灭。

高等自我以「本体性」为特质，低等自我则陷于自私与我执。

自我意识无从根除。「个体性永远存在。」何以如此？这可能吗？这所谓个体性、自我、内在而更高的「我」，究竟是什么？古东方的神秘主义宣称：它是一种纯净神圣的实体，一场「天界的流放」，一位神灵，是「宇宙心智」投射出的个体化光线，亦是「心智之子」。

「本质上，它是『思想』，是个体化的『思想』。其中栖居著所谓『（宇宙）心智之子』。心智既是一项『法则』，也是一种『实体』，更是个体性或自我。它是一位『神灵』，却注定投入无尽的轮回，次次背负责任、次次承受苦楚。这便是转世之法则，或我们所称的神圣之人——贯穿整个生命周期，无可摧毁：作为能思的实体，乃至作为空灵之形，皆属永恒。」布拉瓦茨基于《神圣智慧之钥》第184、183、177页如此写道。

现代求道者或许对此感到不安。他们不仅轻视个体性的概念，甚至企图否定思想、思维与行动的价值。摧毁你的思想，超越心智，放弃思考，活在无心、无念、无思之境——他们如此热切地催促，却浑然未觉：我们正是进化中的心智与思考者，只因「思智与灵魂，本为同一词。」（布拉瓦茨基，《秘密教义评论》，第29页）

然而，我们确实必须摆脱低等心智的思维惯性、习气与自私。这些皆属人格，而非个体性；后者才是高等心智的展现。神秘学称低等自我为「人格」，高等自我为「非人格的个体性」。此事值得深究细品。「真我的普遍意识，较之人格假我的自我意识，广阔何止百万倍。」（[《秘密教义阐释》](/books/secret-doctrine-exegesis)，第74页）

这高等自我居于其独有之境，即高等心智层面；它本具全知，超越物质层界与肉身大脑。透过灵性修持——诸如自制、自律、净化，以及在冥想与日常中锻炼专注——我们能使大脑与人格心智愈加澄澈，更易承接那永恒存有的感通与启发；这自我在教义中有时被唤作我们的「本源自我」或「高等自我」。

「倘若人格（低等心智或肉身心智）能蒙高等自我启迪照亮，世间将再无罪恶。可惜现实并非如此；他们陷落星光界流质之网，渐次与高我分离。」——布拉瓦茨基，[《秘密教义阐释》](/books/secret-doctrine-exegesis)

所幸，高等自我仍会在短暂而稀有的时刻透出微光：例如那些罕见却精准的直觉、预感，以及良知的呼唤。

「安塔卡拉那」一词，「…被神秘主义者解释为连结高等心智与低等心智之径或桥，亦即『神圣自我』与人类『人格』之间的系带。它担任媒介，将个人一切印象与思想，由低等自我传递予高等自我，从而为不朽实体所吸收储存，并与之同臻不朽。这些元素，是短暂人生中唯一能超脱死亡与时间者。」——布拉瓦茨基，《神秘学词汇》，第23页

许多研习神秘学之人，得以目睹并理解「光之存有」此一现象——在众多濒死经验中尤其鲜明——这正与我们高等自我的存在直接相系。以下节录自一九八九年出版的《十九世纪对濒死体验的阐释：布拉瓦茨基神秘学提出的超个人死亡模型》，作者为法国巴黎神智学联合会资深成员让-路易·西蒙斯博士。

「进入光」或「遇见光之存有」，象征著人格自我重新融归其意识本源。

「难怪濒死经验者总觉无法以日常言语描述其经历。犹如柏拉图《理想国》第七卷所喻洞中囚徒：自出生便惯于幻影在壁上的摇曳（印度教谓之幻相），困于二元对立之见（我、他者…）与线性时间之流——以为时间只在过去与未来之间推移。忽然，囚徒获释，被拽出洞穴，来到阳光普照之地，超越一切惯常的局限。此时，他们回归了『本源自我』，虽不自知；其心理运作却将此视为另一人格存有：一道迎接他们的光，一位『光之存有』。用穆迪的话说：

「这存有带著鲜明的特质。对濒死者而言，它所流露的爱与温暖难以言喻，令人感到全然被环抱、充满，在其面前完全自在、被接纳。光的吸引力无可抗拒，他不由自主地向它靠近。」（R. 穆迪，《死亡回忆》，第59页）

许多曾有此经历的人，常用不同的词汇形容那「存在」——或称神、基督、天使、引导者，乃至其他名相。他们对深层心智的学问所知有限，自然难寻恰当术语，也难将自身与各自「自我」的意外相遇，说得明白易解。那存在似乎「知晓他们一切」，予以「全然的爱与接纳」，并与之进行亲密而「具人格」的交流。实则从神秘学观之，这自我对其尘世人格毫不陌生，反而对其命运怀著绵密的「关照」：自出生至死亡，这超越人格的个体始终在反思——或曰「冥想」——它的尘世代表（或称流溢体）。它记取低等自我的作为，并藉直觉、梦境等无声言语，予以启发。

「人生万物皆属无常，唯阿赖耶永恒不改，那纯净明澈的本质。人如水晶折射的光线；内里蕴著一束无瑕辉光，下层表面却为粗砺尘泥所覆。那光即是导引你生命者，是你的真实本体，既是观者，亦属默思者，却常受低等自我所苦。」

——《寂静之声》，布拉瓦茨基译自《金诫书》

人人皆有高等自我。据说「天界」的「众神」，其数不输尘世凡人，更有些正处于两次投生之间、居于天界状态者。

每个男子、女子、孩童，皆是其「自我」的具现化身；那永恒的个体性、真实的内在「我」，显化于此时此际这稍纵即逝的人格、这暂时的角色之中。低等之「我」常是自私欲念所驱，高等之「我」却具非人格、普遍、利他、智慧、慈爱与悲悯的性质本质。一个人若能成功驾驭外在人格，令内在个体性持续焕发光辉、穿越外在躯壳，这样的生命方堪称神圣的投生。此为我们每人皆有的使命与命运。昔有人问布拉瓦茨基：「但高等心智与低等心智岂非一体？」她答：

「是，亦不是。此即伟大奥秘。高等心智本质神圣，因而纯洁；无任何污点能染，无任何刑罚可加；它清白无辜，亦不涉其低等自我的蓄意所为。然则，虽则高等与低等心智呈双重性、在世时彼此有别，『父与子』实为一体——当低等心智重与『本源自我』结合，便将一切恶行善业皆系于后者之上，或烙印其中。两者将共承苦果。高等自我虽无瑕完美，在未来的投生里，却须与低等自我同担其错行之罚。整个救赎之说，正是立基于这古老的神秘信条；因高等自我实为尘世人格的原型。明其内涵者，即知这是毗首羯摩天这个古老的吠陀故事再度活现。毗首羯摩天乃全知之父神，超越凡人所能理解；他最终以神圣之灵布瓦纳之子的身份，牺牲己身以救世界。印度哲学中，『高等自我』的神秘名称是『灵之化身』（Kshetrajna），能使『身体』（kshetra）活跃。若究其词源，可见『最初诞生』（aja）一词，亦含『羔羊』之意。此中启示深长，足以著书无数，探讨『基督—灵之化身』在诞生前后，其类型与原型的开展——他被视为『神—人』，是最初诞生者，以『羔羊』为象征。秘密教义揭示：『心智之子们』，亦称『自我们』，乃知情且自愿担起未来人格一切罪愆之重负。因此不难明白，真正的受难者并非某甲某乙，亦非周期性披上外衣的人格；而是那些牺牲自身的自我，他们才是我们内在无辜的基督。故而，神秘主义的印度教徒认为，永恒的自我（一中含三，三而为一）是『驭车者』，人格仅是短暂乘客，马匹则象征人的动物欲望。总而言之，每当我们对『良心之声』闭目不闻，便是在将内在的基督钉上十字架。」

这确是「伟大的奥秘」，正如布拉瓦茨基所言。在地球第三根种族后期，这些自我周而复始的投生朝圣之旅，更显意味深长；此时期称为雷姆利亚纪元，约在一千八百万年前。

当他们集体转世于无觉知的「动物性人」之际，真正的人类方始诞生。「人」此字源于梵文「心智」（Manas）。此概念在布拉瓦茨基不朽巨著《秘密教义》——尤以第二卷《人类起源》为要——中详述阐释，该卷涵盖人类的起源、诞生与进化历程。

我们的自我，即是我们的个体自我意识；或者说，那具自我意识的个体性。

因此，神秘学研究者最好保持此术语的完整纯粹，一如布拉瓦茨基、其同侪威廉·贾吉，以及他们背后智慧大师所倡导的那般。

在神秘学的典籍里，「自我」一词，无论大小写，皆蕴含著正面光明的意义。布拉瓦茨基的著作卷帙浩繁，她笔下的「自我」，与当今流行的灵性概念并不相同。若要精准阐述其教诲，理当追随她与诸位大师的典范，而非人云亦云。论及低等自我时，她总冠以「低等」、「虚假」之名。更多时候，她仅称之为「人格」，而将「自我」一词，保留给那历劫轮回的永恒个体性。身为神秘学的诚挚学徒，我们亦当如此措辞。

因此，若有人对此心生误解，那是他们对崇高词汇与形上概念的非哲学滥用、草率诋毁所致，并非我们的过失。

文末尚有一点值得提及：在神秘学中，「高等自我」与「本体」并非同义。本文所论，仅涉高等自我，未及本体。或有人驳道：「然而英文里，Ego与Self本是同义！」此言不虚，但在神秘教义之中，二者未必全然等同。

我们真正的本质，非「心智」，亦非「菩提」，而是「阿特曼」。梵文里，「阿特曼」直译即「本体」，然此本体并非某种「自我」。它是众生共通的、纯粹永恒的灵，超越个体性，是无限神圣的本质，是至高无上的无所不在之能量——而非任何自我、个体或个人实体。阿特曼并非实质存在，而是「绝对的非实体」，是无限的神圣原则，是神本身，是终极。

我们每人皆有各自的高等自我，然而「本体」普遍存在，人人皆同。灵魂众多，灵唯至一。正如克里希纳在《博伽梵歌》中告诫阿朱纳：一如同一轮太阳普照世间万千躯体，同一个灵，亦点亮了每一个灵魂。灵、阿特曼、本体，并非个体化的意识，而是纯粹的意识本身。

我们当时刻铭记永恒智慧的教诲：不应仅为一己之人格或个体而活动、行动与生活，更应为「一切生命的本体」奋力前行。
