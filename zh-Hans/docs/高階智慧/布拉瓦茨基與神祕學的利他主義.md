# 布拉瓦茨基与神秘学的利他主义

## 布拉瓦茨基的观点指出，自我与周遭生物界间的分离感只是幻觉，真正的神秘主义在于消解这种错误的观念。

![封面](/img/downloaded/ffccf6cda2e1fdab4c4d384feaf43643.png)布拉瓦茨基 著

神秘学的根本教义是：我们在自我与万物之间划出的那道「界线」，实为幻影。真相是，万物本然一体，这并非情感的喧哗，而是冷静的觉知。东方哲学恒言，宇宙之中唯有那「至一本体」真实存在；我们自认的「自我」，不过是这本体投射于尘世的、一段稍纵即逝的虚幻映像。

真正的神秘主义，便在于拆穿「自我」这道幻幕。心智圆满与智慧的表征，即是将有限的「自我」等同于那宏伟的「一切」。除非人类作为一个整体取得进步，否则想要在灵性层面提升几乎是不可能的。个体永恒的幸福系于社会整体的福祉——个体本就镶嵌于整体之中，不可分割。

因此，神秘学的利他主义与「断除物欲，追寻灵性圆满」的教导，并无冲突。因为灵性的圆满与真知，唯在灵性层面方能抵达；唯有在此境界下，所有隔阂、自私、个人欲念，才会消融于人类共通的、更广大的意识之海。

这也意味著，要求人盲目服从他人指令，毫无意义。每个人都必须亲身跋涉，经历考验，学会辨认何者有益于人类。随著他灵性成长，逐一击溃自私的壁垒，他的心智便会敞开，接纳其内在那神圣单子的指引，即他真实「本体」。于他而言，过去与未来俱皆消隐，唯剩永恒的此刻……

与真实灵性生活的喜悦相比，物质生活的幸福便显得轻浅。人类在尘世逗留数十寒暑，远不及在主体层面度过的百万年之久。

至于能力与天赋的培养，答案极为明澈：应当培育它们，用以服务人类。毕竟我们同属人类社会，应当无条件地为其福祉尽心。
