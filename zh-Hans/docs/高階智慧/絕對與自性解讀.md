# 黑暗与光明：《秘密教义》中的绝对与自性解读
## 揭示绝对光暗之互换与非字面解读，洞察宇宙根源与自性深意

![封面](/img/downloaded/light_absolute.png)

布拉瓦茨基笔下的「黑暗」众说纷纭，常令人费解，《秘密教义》中尤甚。

某些虔诚基督徒惊呼：「她竟称神为黑暗！骇人听闻！此人必是邪恶，崇拜魔鬼！」

他们或许忘了，《圣经》亦有言：「黑暗包围神的亭子，祂在黑暗里藏身。」这些描述，从不意指神性邪恶、阴险或缺乏光明。

布拉瓦茨基在[《秘密教义阐释》](/books/secret-doctrine-exegesis)中解释：「此处『黑暗』指未显现、不可知者，乃显现之反面，只能透过推测。并非无光之黑暗，而是一种不可理解的原初原则，即绝对者本身。对吾人的智识感知而言，它既无形体、颜色、实质，亦无任何言语可以表述。」

《秘密教义》内容多基于神秘主义、象征主义与高等形而上学。布拉瓦茨基屡屡提醒读者：切莫拘泥字面，当以抽象或东方直观领会。

哲学术语中的「绝对」，意指完全超越一切相对性，无限、抽象、无所限制，圆满深奥且不可言说，全然超脱人类思想与智识感知。它与任何可想像、可构思之物毫无相似，亦与一切显现者不同，因其永恒未显、永不可显。

故而，「绝对的光」对我们而言，便是黑暗。「绝对光明」与「绝对黑暗」实可互换，「绝对意识」与「绝对无意识」亦然。然欲深究，仍须亲阅《秘密教义》下文，且当缓读、冥想、反思。这类真理与概念，纵含纯粹逻辑，亦永不能单凭智识推理理解或掌握。形而上之领悟，需形而上之思维。

摘自布拉瓦茨基《秘密教义》

「唯有黑暗充盈无边一切，因父亲、母亲与子再度合而为一，而子未醒，新循环与朝圣之旅尚未开始。」——《德基安集》，偈颂1:5（第一卷，第27页）

「建造者们，那显现期黎明之光辉诸子何在？…他们居于阿希（禅那佛）超涅槃中之未知黑暗；『宇宙空间』与『自性』为世界根源，将生形体与无形体，然此刻安息于非存在之极乐。」

「那七子尚未自光之网诞生。唯有黑暗存在，此即『父亲—母亲』、自性，而自性居于黑暗。」——《德基安集》，偈颂2:1, 5

「…第七个万古的最终振动，于无限中颤鸣。母亲自内而外扩展膨胀，宛如莲苞。」

「此振动掠过，迅捷之翼（同时）触及全宇宙及黑暗中之胚种：黑暗在沉睡水面上呼吸（浮动）。」

「黑暗辐射光，光则将一束孤独光线投于水中、落入母亲深处。」

「哦，弟子，且看！那两者所诞发光之子，辉煌无匹，明亮空间实乃黑暗空间之子，自伟大黑暗之水深处涌现。」

「『父亲―母亲』编织一网，上端系于灵——即至一黑暗中之光明；下端系于物质——乃灵之阴影一端；此网即宇宙，由两种基质合一组成，亦即自性。」——《德基安集》，偈颂 3:1-3, 7, 10

「永恒黑暗之光线重新唤醒空间中之能量（禅那主们），『一』来自卵、六者与五者。继而为三、一、四、一、五——总和为两倍之七。此即本质、火焰、元素、建造者、数字、无形体、有形体、及神圣之人力量——万有之总和。且自神圣之人流溢形体、火花、神圣兽类，及神圣先祖之信使——万物尽在神圣四中。」——《德基安集》，偈颂 4:3（第一卷，第30页）

「有句古谚云：『黑暗乃父母，光明是其子。』光必有源，否则无从想像；此源即光之因。理性与逻辑虽竭力溯源，那原初之光究竟何来，终不可知；若以智性观之，此源头称为『黑暗』。至于借来之光或次级光，无论出处为何，皆属暂现之幻相。故黑暗为永恒基质，光源于其中生灭。在我们物质层面，黑暗不会因添加了什么而成光明，光明亦不因添加了什么而成黑暗。二者实可互换；科学上言，光仅是黑暗的一种模式，反之亦然。这两种现象都具有同样的本体，对科学头脑而言是绝对的黑暗；对于一般神秘主义者的知觉来说，是种灰暗的朦胧；而对于启蒙者的灵性之眼而言，这是绝对的光明。我们在黑暗中辨别光亮的能力取决于视力。对我们而言的光，对某些昆虫来说是黑暗的；而灵视者所见的光，对一般肉眼是黑暗。当整个宇宙沉睡后，一切回到至一原初元素，此时既没有光亮的中心，也没有感知光的眼睛，黑暗必然充满了无边无际的一切。」(第一卷，第40-41页)

「黑暗的本质是绝对的光，休止期状态的宇宙被比喻为黑暗，这对于我们有限的心智而言，是绝对休息或非存在的。」(第一卷，第69页)

「根据玫瑰十字会的信条：『光明和黑暗本身是相同的，受到人类心智的区分。』根据罗伯特·弗劳德 的说法:『黑暗通过光使自己可见。』根据东方神秘主义的信条，黑暗是至一真正的实在，是光的基础和根源，若无黑暗，光就永远无法显现自身，甚至不能存在。光是物质，而黑暗是纯粹的灵。黑暗在根本形而上学的基础上，是主观的、绝对的光；而物质光在其一切看似光辉和荣耀中，只是一团阴影，因为这不可能是永恒的，只是一个幻觉。」

「即使在《创世纪》中，光也是从黑暗中创造出来：『黑暗在深渊的表面上。』（第一章第2节） 『在他（黑暗）之中有生命，而此生命就是人的光。』（《约翰福音》第四章）或许有一天，人们的眼睛会被打开，更能理解《约翰福音》中的这节经文：『光在黑暗中照耀，黑暗却不理解。』此处『黑暗』并非指人的灵视力，而是指实际的『黑暗』，即绝对者，它不理解（不认识）短暂的光，无论此光对人眼来说多么超然。」(第一卷，第70页)

「黑暗产生光。」(第一卷，第110页)

「没有任何影子的光将是绝对的光，换句话说，是绝对的黑暗...」 (第一卷，第201页)

「迦勒底 《数字之书》说道：『至一宇宙之光永远存在，但对人而言是黑暗。』能量周期性地从黑暗产生，并反映在『深渊』或混沌中，混沌是产生未来世界的仓库，一旦被唤醒，就会激起潜在的力量，并使之产生结果；这些力量是永远存在其中的永恒潜力。梵天和诸佛被重新唤醒，皆为永恒力量，于是出现新的宇宙。」(第一卷，第337页)

「绝对者不是神。它是无物，也是黑暗；是无名的，因此被称为无限 。」（法兰克 《卡巴拉》 )

「最高神（未显现的逻各斯）是黑暗之子。」(第一卷，第350页)

「回来谈论卡巴拉的神，这个「隐藏的合一」被视为是无尽、无限、不可知的存在。只要绝对者还处于『奥罗姆』（即无穷无尽、没有界限的时间）之中，『无限』就不能被视为宇宙的创造者，甚至连宇宙的塑形者都不是，也不能被称为『光』。因此，无限也是黑暗。此不变的无限和绝对的无边既无意志、无思考、也无行为。唯有化为有限时，才能做到这些。于是它将光线穿透进入世界蛋 (无限空间)，然后以有限的神明形态显现出来。这一切都取决于那道潜藏于『至一』之中的光。当此时期到来时，绝对者会自然地释放其中的力量，依据其内在终极本质的法则。」(第一卷，第353-354页)

「初级创造被称为光（灵）的创造；而次级创造称为黑暗（物质）的创造。......这不能与宇宙诞生前的『黑暗』相混淆，后者是神圣的一切。」(第一卷，第450页)

「每当宇宙重新诞生、生命初现时，自性是『永恒无意识不变黑暗的可变光芒』，会从不活跃的状态转为极度活跃，并开始分化，然后透过这种分化展开其运作。运作过程即业力。」(第一卷，第634-635页)

「在神秘主义中有三种光，如同卡巴拉所述：(1）抽象绝对的光，也就是黑暗；（2）『显现-未显现』之光，有些人称之为逻各斯：（3）映像到诸禅那主的光，即次级逻各斯（统称埃洛西姆），而后由这些将光照在客观宇宙上。」(第二卷，第37页)

「黑暗是绝对的光明，此事实已被神学忽视、甚至完全遗忘了。」(第二卷，第489页)

「从形而上学而言，父亲和儿子对应于『宇宙心智』和『周期性的宇宙』；对应于『天使』和『人』。儿子和父亲是一体的；在《神圣皮曼德》中，代表活跃的意念和产生的被动思想；这是大自然中的根本基调，产生了七个音符 ，是创造性力量的七级音阶，以及棱镜的七色，而一切源自一个白色光线或光，源自于黑暗。」 (第二卷，第492页）

印度教经文《自在奥义书》中有一段众所周知的段落：人若将外部世界、感官和物质事物视作真实，将「走向黑暗」；若将内在世界和灵性视为唯一真实，则「走向更大的黑暗。」K.H.大师从秘传的角度解释：「圆满的大成就者最终融入了『更大的黑暗』，即绝对的黑暗，也就是绝对的光明。」

最后引用14世纪基督教神秘主义者雷斯博克的话:

「当爱将我们带离尘世、超越万物，进入神圣的黑暗，便能平静地接收那不可思议的光，这环绕并穿透我们。光不就是无限者的沉思，对永恒的直觉吗？我们所见即是我们自身，我们自身也正是我们所见；因为我们的思想、生命与存在都被提升至纯粹，与神圣的真理合而为一，也就是上帝。」
