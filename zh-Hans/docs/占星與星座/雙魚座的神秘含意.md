# 双鱼座的神秘含意
## 我是伟大的鱼，我的无数后代是闪烁的繁星

![cover](/img/downloaded/81e33d17192d33bd6619bcbc5f83c472.jpeg)

现今，双鱼座被视为春分的星座。我们总以为春分点仍落在白羊座，然而岁差推移了天体轨迹，过去两千年间，春分时刻太阳所穿越的，其实是双鱼座。更早的两千多年里，春分点则驻守金牛座，沿著黄道十二宫的序列缓缓迁徙。

古人习惯将太阳与春分当值的星座相连，赋予象征。于是，太阳的形象依序化为公牛、公羊、羔羊、鱼，视哪个星座宣告春天来临而定。微妙之处在于，即便岁差为太阳带来「崭新」的象征，旧有的象征却迟迟不肯退场。正因为如此，基督教的「太阳—神」基督，其形象同时汇聚了白羊座的羔羊与双鱼座的鱼。

真相是否尽然如此，不得而知。但希伯来人确曾将期盼的导师与救世主唤作「鱼」（dag）。后有学者将这「鱼」与美索不达米亚等地渊远流长的「鱼神」（Dagon）相系。然而，如今更倾向认为，「鱼神」之名或许并非源于游鱼，而是出自意为「谷物」或「种子」的古词根。

纵有疑窦，耶稣基督的形象与鱼之间的千丝万缕联系，大多可溯源至埃及。

「春分点已退入双鱼座；先前在白羊座时，荷鲁斯或耶稣尚是『神的羔羊』，或在金牛座为小牛，后来却被『埃及—诺斯替』的艺术家绘成了鱼。」

多年以来，人们总爱将多个星座并置思索，提供更辽阔的视角，或自成体系。例如，黄道上遥遥相对的星座常被一同解读。双鱼座的对宫是处女座，二者时而被视为一对。但双鱼座位居黄道之末，有时也与起点的白羊座相配。由此衍生一种信念：双鱼座既圆满了始于处女座的历程，也终结了发端于白羊座的循环。

类似的思路，也解释了双鱼座自身显而易见的双重性。对某些人而言，两尾鱼可喻指人的两部分；或指灵性与物质；或象征航行的起点与归宿。

双鱼座的拉丁名为Pisces，希腊人则称之Ichthyes。阿拉图斯为这组星宿，描绘了这般形象：

「这两尾鱼，一尾比一尾优雅，  
当它们初次浮现，北风也变得温柔。  
它们的尾鳍向两侧舒展，如一条锁链将彼此牵系，在行进中交缠。  
其间镶著一颗星，既美且亮，人们称它为『系尾之结』。」

星图上，一尾鱼游向天球北极，另一尾则西行。二者总以一缕绳索、丝带或锁链相连。从托勒密为双鱼座群星编目的方式来看，他认定两鱼由两根绳索在中间打结系缚。

古典神话流传诸多故事，皆自称为双鱼座形象的起源。其一讲述阿佛洛狄忒与厄洛斯为躲避百首巨怪提丰的攻击，仓皇逃逸。另一方面，埃拉托色尼则叙说：南鱼座——两尾小鱼（即双鱼座）的母亲——从湖中救起因跌落的女神德赛托；女神为表感激，将三鱼一同送上星空。还有一则关于维纳斯（或称阿佛洛狄忒）诞生的版本，部分解释了她与鱼、鸽子的神圣关联：

「据说一枚巨蛋落入幼发拉底河，鱼将其推至岸边。鸽子伏于蛋上温孵，最终诞出了维纳斯（或称阿斯塔蒂，叙利亚的阿佛洛狄忒），后世尊为叙利亚女神。因她在正义与正直上超越众人，宙斯赐予殊荣，将鱼群置于星宿之间；故而叙利亚人既不食鱼，也不食鸽，奉二者为神灵。」

叙利亚女神阿斯塔特，常与伊什塔尔等同。此神话形貌源起中东，其特质或映现了更古远的星座方位。

虽起源未明，然有现代学者推断，今日双鱼座，实由古巴比伦三座拼合而成：大燕子、鱼绳、天女。后者涵括双鱼北鱼及仙女座部分星辰，名为阿努尼特姆，乃阿卡德女神。她与阿斯塔特一般，亦被视作伊什塔尔或伊纳娜的化身。燕子尾端星宿与飞马座众星，则共构西鱼。因此之故，某些星图所绘双鱼天域，并非两鱼相连，而是一鱼以带索系于燕身。

「『阿努尼特姆』、『燕子』与『田野』诸星图，共叙『叙利亚女神』之神话，亦可谓此季节的主要神话。」

「田野」可能是古代飞马座四边形的起源，丹达腊黄道图示其位处双鱼之间。《死者之书》载有二鱼引领拉神乘舟渡越冥界，恐亦此星座遥远余音。

鱼于印度教洪水神话中，同样担当引导之责，祖先摩奴在此取代了诺亚。某些版本称，鱼（摩蹉）乃梵天化身之一。

「洪水滔天之际，波澜之上空无一物，唯余摩奴与七圣，及牵舟之鱼。

岁月悠悠，鱼不厌其烦引导船乘风破浪，终抵喜马拉雅山巅。

鱼轻轻的微笑，对圣人说：『赶紧将船系于高崖。当知我乃万物主、大造者梵天，力胜一切，无所不能。我化鱼形救你于危厄。一切造物、诸神、阿修罗、世人，皆由鱼摩奴所创；世间动植万象，亦由他出。』」

另有一更古老的说法，视其为印度三相神之第二身——毗湿奴。

主星「双鱼座α」名出阿拉伯语「绳索」，标示绳结所在。有研究者认为这很可能即所谓伯利恒之星。它过去比较明亮，虽然在现代星表仍居α位，绳索中另有一星现已比它更亮。此星乃「双鱼座η」，其名来源虽不详，有人称之为「水之倾注点」。另有一星「双鱼座β」，意即「鱼口」。

噢，愿众同伴皆抵达水之倾注点，于全然觉知中，在亚麻线上系上光辉的结。愿他们于智慧之水中调和这两条鱼。愿他们举手向天，俯身向地，事奉我主。在平和安稳中，愿他们列队迎候欢笑孩童之歌声。

尘世之人浸溺于物质浊流，梦见队伍的尽头；叛逆者则渐次没入深渊暗影。然而，我的孩子，你与我将共浴光中升华，登临天界之水的水晶表面；我们的水之旅程，将从深渊直抵天界高处。

因我乃伟大的鱼，我无数后裔即闪烁群星；我是水域引导者；我引舟至安稳之地；我是悬于枝头的鱼；我是伊希斯之鱼，我的时刻将临，无可置疑。
