# 天秤座的典故
## 平衡、法则、正义、判断与辨识的星空歌咏

![cover](/img/downloaded/f8b7426b03f761654480e19d9b692d57.png)

在某些古文明里，天秤座与太阳有关，奉为真理与正义之神。美索不达米亚人视其为沙玛什（Shamash）的神圣符号，这位神祇掌管度量衡的准绳。埃及则将天秤形象简化为轭或横梁，呼应尼罗计——一种测量洪峰水位的器具。

如今许多人相信，构成天秤座的星域，古时原属天蝎座的螯爪。更有人说，在更早的年代，处女座与天蝎座并非独立星座，而是一体横跨今日处女、天秤、天蝎三星座的大星座。据传，待处女与天蝎分离，才在其间置入天秤座，将星座总数从十调整为十二。无论此说真假，天秤座的引入似乎意在标记秋分——这象征再贴切不过，此时昼夜等长，不偏不倚。

考量天秤座的位置与历史，它与两侧处女、天蝎的渊源仍未断绝。一侧是纯洁的阿斯特拉亚，那位「星之少女」，希腊的纯洁女神，身影常现于处女座。她与执掌正义、手持天秤的女神狄刻关系密切。另一侧则是带来死亡的蝎子，沙漠的住民。构成天秤座的星辰，古典天文学家时而称之为天秤，时而唤作螯爪。至今，天秤座中若干星辰仍以蝎螯为名。居于两星座之间，天秤座的位置自有深意。

「为免这场可怖争斗，永恒者在空中悬起他的黄金天秤，高挂于阿斯特拉亚与天蝎座之间；他先以此称量万物，令悬浮的尘世与苍天保持平衡……」

阿拉伯人的宫位 Al Jubana 囊括了天秤座相关星辰，其名却指蝎螯或其他爪具。印度教中，天秤座四星组成星宿维萨卡，意为分叉、具分支。其象征被描绘为一座华丽的拱门或凯旋门。

在中国星野，这四星亦代表「氐」，此为第三宿之名，与青龙有关。氐字本义为根柢、基础，犹如树根。然而，氐虽常译作「地」，与其互补原则「天」相对，诠释这些术语时却须格外谨慎，因其涵义往往随语境流转，别有层次。

其中一例便是所谓「崆峒的天界」。虽名为天界，实则深藏地下：

「文字流变中，崆峒一词既指庞大地穴，亦指远西高峰。这处神话之地，象征重生、启蒙与顿悟。相传黄帝曾在此地领悟终极实在的启示。」

纵然这些典故如今看来有些幽深难测，但追寻奥秘者当知：阴阳调和，于中国人而言何其要紧。不论显现为月与日、尘世与天穹、底与巅、根与冠、湿润与干涸——创造之力，正源自二者的交会。恰似凤凰栖于树梢，唤醒沉眠的龙；看啊，那位自发的金童伸手采摘熟透的桃实。

然则天秤座亦预示岁时的「临界」将至。那时晨星的光缕被裁断，坠入灼热的西天，踏上漫长孤独的旅途，重新寻觅属于它的高处。因为天秤之歌吟咏平衡、法则、正义、裁断与明辨。他们歌颂真理与正义；歌颂言语、意志、行止的权衡；歌颂从内里光芒升起的良知之曲。

人啊，当你的时刻到来，世界的重担会压垮你吗？抑或你将如飞鸟挣脱罗网，腾入芬芳的空气？

埃及的末日观里，心脏的称量在审判厅进行——有时称为奥西里斯之厅，或「双重真理」之殿。奥西里斯既是天界之主，亦为玛特之主，故被视为亡者面前的公正判官。

「玛特的核心意义是『直』，正如我们所见，与希腊语『正典』（κανών）同源——原指直竿、石匠的矩尺，后引申为规范人与其行为的律则、法典、准绳。埃及人在物质与道德层面皆用此词，意为『正确的、真实的、真理的、真正的、正直的、正义的、公正的』。因而《普塔霍特普箴言》有云：『真理（玛特）伟大而强韧，永不动摇，自奥西里斯时代以来从未被破毁。』普塔霍特普告诫听者：『令真理永存。』公正正直之人即为玛特，是『被断为正义的神』。」

玛特作为阿门提的女主，手中执一根羽毛。她面前，一束金光从绿松石色的天穹垂落，轻触下方绯红的大地。啊，当我们在众神审判前被称量之际，愿我们诚实述说真相。当我们凯旋踏入白昼，愿先祖为我们欢呼。在那里，阳光将在水中起舞，欢笑孩童的歌声会在我们心底回荡。

仇恨终将吞噬仇恨，受卑劣欲望驱使之人，必被命运的暗影吞没。他们将在道上踉跄而行。但亲爱的孩子，你们不会失败。因为善者如舟，虽行于惊涛之上，终将抵达灵魂的港湾——届时，我平和的庇护必将降临。

我乃天平，游移于扩展的光线；我是金色天秤的光缕；我是公正的权杖；我是真理与正义的太阳；我是地平线的主宰；我是交握双手者——我的时刻即将来临，无可抗拒。
