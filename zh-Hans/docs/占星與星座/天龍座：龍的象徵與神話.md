# 天龙座：龙的象征与神话
## 龙或蛇是最古老的象征之一，世界上几乎到处都能见到不同种类的龙

![cover](/img/downloaded/3ea3786c9020342b8ae7864cf6a6d7c4.jpeg)

你，是夜空里那条繁星点缀的巨龙，盘踞在创造的顶点；永恒静止，永不西沉，而少女们围著你欣然起舞。我们该称你为欲望之龙，是一切求道者心智的敌手吗？还是称你为生命之龙，万物缺你不得？抑或你是智慧之龙，带来白昼的甘泉？我们该把你画得如夜一般墨黑，吞噬光芒，双眼如炭火般冷酷而赤红吗？还是在万物向你鞠躬的时刻，为你染上正午烈阳的辉黄？或者你是电光般的湛蓝，像一道慑人的霹雳，疾速划过紫罗兰色的天穹，迅疾如思绪？

龙或蛇是最古老的象征之一，古代神话里，牠们的形态变幻万千，远比其他生物更富特色；至少在某些方面，龙与蛇常被视为等同。古代习俗也惯于将各类复合生物唤作龙。多数情况下，这些神话造物纯粹只是象征或标志。

世界各地几乎都见得到各类龙的身影：伊甸园里的老蛇，《圣经》中飞天的火蛇；希腊与他处的守护龙，波斯的食人龙，威尔斯的红龙，还有吠陀神话里的「覆盖者」巨龙弗栗多。奥菲克蛋周围盘著龙；在巴比伦，一头巨龙是众怪之母；甚至如埃及所描绘，龙即是夜空本身，鳞片便是繁星。牠是中国的国徽，尽管中国的观星者将牠画在天秤座的天空。整个亚洲，复合生物摩伽罗有时也被视为龙。也有无数传说讲述龙被众神、圣人、天使与英雄所杀。这样的例子难以穷尽，各自皆有深意；此处只需留意，星座之中亦有几处龙或蛇的踪迹。

与此同时，舞动的星空下，船长往烟斗里填满烟草；填烟时，他想起了家，山谷外那片草地——清晨，那只眼神温柔的母鹿曾领著幼鹿前去。在他心眼中，牠们再次浮现，那优美身姿令人惊叹；周遭，大地的银色露水清澈纯净，正被朝阳从羁绊中释放。

他就这样在甲板上沉思，船飘浮于汪洋，忆起故乡美景，爱意自心底涌起。但那涌起不像以往——不像杯中斟满酒，不像圣酒洒上洁净桌布，也不像爱的眼泪盈眶而出。不，它是猝然的，犹如巨龙刹那冲出海面，带著燃烧的吐息，将混沌残酷的黑夜一口吞尽。他想：你们寻求善的人啊，去寻求爱吧；你们追寻力量的人啊，去寻求爱吧；你们渴求智慧的人啊，去寻求爱吧！

太阳系最初的创造者，常被塑造为龙或蛇的模样。牠象征永恒与完整，自成一体，别无所缺。

「神的灵在混沌上盘旋，每个民族皆将其化作火蛇之形；祂在原初的水面吐纳火与光，直至蛇身盘蜷成形，衔住自己的尾巴。」

在中国，禁锢于物质之中的火，有时亦被称为龙。据说那是一道神圣的白炽，既为火亦为水，既阳刚亦阴柔：是日与月的拥抱，是阴与阳的交合，显现于存在的每一层阶。

古老的故事里，有的龙长著七颗头，有的三颗或十二颗，还有一百颗头的。从某个角度看，太阳系确实「七呎深」，但龙能去往任何地方——上下、高低、左右、前后；牠环抱著整个世界与其中的一切。传说牠守护山洞、圣园与极乐天堂。

另一方面，传说中那些吞噬光明的暗黑之龙，会在每个黎明被强大的太阳杀死；正如干旱之龙总在雨季来临时殒命。然而，基督教神学家将牠塑造成叛逆、邪恶的天使，一个具象化的魔鬼，是神的对手。

无论何时何地，英雄试图制服并战胜恶龙的历程，总被视为艰险重重，但成功总带来丰厚回报。传说巴哈姆国王独自出猎时，曾被一头野驴领到山洞，洞里住著一头吞食幼驴的恶龙。国王屠了龙，驴为表谢意，将他引向洞中埋藏的一批巨大宝藏。

「一位为野驴伸张正义的君王；一条被囚于墓穴的龙，最终，君王因他的义行得救，且获珍宝。」

据说，若在龙的食物中掺入精心调制的蜜水混合物，或能稍减其凶性。极少数情形下，这种龙甚至可被骑乘。大英博物馆藏有一枚古巴比伦的绿蛇纹石印，其上刻有神灵骑乘驯化之龙的景象。

「可见神灵跨坐怪兽背脊，一名崇拜者立于其后。前方为一公牛，吉尔伽美什正以匕首刺之。上方女神舒展双臂，水流自臂膊泻落地面。吉尔伽美什身后天际，亦有一瓶倾水，淌入尘土。」

自古龙分善恶，有上界之龙，亦有下界之龙。善龙有时象征太阳，太阳又代表神圣之光与生命之火。例如，波菲利曾述及密特拉秘仪：

「……你们称太阳（及其相关本性）为公牛、狮子、龙与鹰。」

白昼时，太阳代表此龙；夜晚则由月亮代表，或深居北极星空的天龙座星群代表。天龙座环绕极星，永不沉落（至少在北半球），恰似那象征光明与生命的善龙，永远守护神话中宝树，植根于天界永恒之水、结满金果。

此即所谓「崇高山巅之树」。

「……乃天极一隅，审判之地，由天蛇——即天龙座——守卫。」

或有一些解释者称树上金苹果喻指星辰。然更具玄思之人，则视金果为智慧宝藏，由永恒警醒的龙所看守。

「于北温带的纬度地区，此星座永不没入地平，故希腊人视之为永恒警惕之象征，喻为园中守护珍果的龙。」

这座园子属于赫斯帕里得斯三姊妹。关于极乐园、龙与果树的希腊神话版本纷纭，以下精要亦不逊于任何一说：

赫斯帕里得斯三姊妹乃黄昏的「女神－仙女」，亦是落日金光。她们是黑夜妮克丝之女，或谓天界泰坦阿特拉斯之女。三姊妹受托照看金苹果树——此树为大地盖亚于婚礼当日赠予天后希拉。她们得到百首的守护龙德拉贡帮助。赫拉克勒斯受命摘取金苹果，作为其十二功业之一；他屠龙后窃得珍果，雅典娜其后将苹果归还三姊妹。

赫斯帕里得斯亦保管众神其他宝物。珀尔修斯诛杀美杜莎所需之神器，便是从她们获得。

三仙女与其辉光流转的金苹果，被认为是日落金霞的来源，此景庆祝著天王宙斯与天后希拉之婚礼。

天龙座起源尚有他说：包括守护金羊毛之龙；或被卡德摩斯所杀之龙；或被密涅瓦（罗马之雅典娜）所诛、掷于天际之龙。

早于希腊人，埃及人已将「逻各斯」表现为火龙。其大祭司有时被称为「龙之子」，龙象征神圣智慧之光与火；而古埃及一切伟大的开悟者与启蒙者，皆与神圣智慧的守藏、隐蔽、有条件启示紧密相系。其最早形态之一，乃以鳄鱼神索贝克为代表，该神于埃及人的日出日落观中占有要位。

在中国，天龙座部分星宿组成与天帝相关之星象，尤构成了紫宫——即紫禁城——的两道垣墙。余星代表诸般皇家侍从与皇帝内寝部分。而同观沙漠苍穹的阿拉伯人，所见非龙：他们将我们视为龙首的星群，看作一群骆驼，正护卫幼驼免遭鬣狗侵袭。

天龙座中值得注目的单星乃右枢（天龙座α）。此星今虽非星座最亮，却被视为至关重要。公元前四千年至二千年间，右枢本是北极星。因岁差之故，极星今为勾陈一（北极星）。有研究者与埃及学者认为，大金字塔之方位特意对准右枢。此星名称源于此星座的阿拉伯语称谓。亦有人主张，巴比伦星表中极古老、被称为「王冠」与「天庭法官」之星，所指正是右枢。

此星另有「拉斯塔班」之名，意为「龙之睛」。顺带一提，天龙座的其他星星如天棓四等，或者龙首龙尾相关之名，不宜与指称月交点的术语混淆。

在平和的天界，于智慧之水中的圣丘之巅，生长著神圣知识的常青树，具有永恒生命。太阳一旦西沉，终会再度登上神的山巅，为这浊世重沐光辉。星夜璀璨，同伴们继续航行，寻觅那珍贵宝石。晨风将至，男孩的笑声便不远了。

凡耽溺低俗欲望者，必遭死亡之龙吞噬，永坠坟冢幽暗，又聋又哑；悖逆之徒行至半途，只得折返，因无法穿过烈焰气息。

但那些追随我、心口如一的人，必在深处听见智慧回响——那是我鲜活的话语。我将赐予他们智慧之爱，一路相随；而我，作为爱的智慧，将安居在他心中。

因我是活水的灵，是光辉之言，是宝藏的守护者；  
我是神圣光明的龙，是神圣生命的龙，是神圣的爱之龙。  
我的时刻迫近，绝不落空。
