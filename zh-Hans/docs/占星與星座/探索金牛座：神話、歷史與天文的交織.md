#  探索金牛座：神话、历史与天文的交织
##  从古代文明到天文观测，揭开金牛座的神秘面纱，探索它与昴宿、毕宿星团的丰富传说。

![cover](/img/downloaded/b20c3b0d315db434cdea1b9cde6f9ef8.png)

金牛座曾标志著西元前四千年至一千七百年间的春分点。阿拉伯人称其为Al Thaur，叙利亚人唤作Taura，波斯语里则有Tora、Ghav或Gau等名。希伯来人称之为Shor或Re’em，意皆为「野牛」，圣经学者多认为那指的是已绝种的「原牛」。希腊语称Tauros，时常联想到朱庇特化身公牛、拐走欧罗巴公主的故事；也有人将朱庇特的偷情对象描绘为伊俄——那被唤作「流浪者」、视为月亮化身的女子。印度教寓言里，则有梵天与其言语女神瓦查，被喻为吠陀经的吟唱之牛。

巴比伦人将金牛座称作「天上之牛」（Gud.Anna或Gu-an-na）。值得一提的是，当英雄吉尔伽美什与恩基杜战胜这头公牛后，曾将牠的心脏携往高天，献祭给太阳神沙马什。

胡神是德鲁伊教里守护神圣牛栏的神祇，尊号「至高拥有者」，以公牛之形显现，「其圣所坐落在潮水环抱的岛屿上。」

金牛座肩上簇著七颗昴宿星，其中六颗可见，一颗据说隐而不显。在印度，这七颗星象征七位圣人的妻子。传说战神卡蒂克雅由她们抚养长大，最终斩杀暴虐的阿修罗，完成使命。印度二十七宿中，昴宿星构成第一宫「克里蒂卡」。

西方有时视昴宿星为鸽群，夜里遭猎户座追赶；在荷马笔下，这些鸽子啣来甘露，哺育婴儿宙斯，因而被赋予不朽的象征。基督徒心中，这关联到圣灵，并不意外。

有人认为「昴宿星团」（Pleiades）之名源于普勒俄涅——七姊妹的母亲、永恒阿特拉斯的妻子。但这名字本身来自希腊语的「航行」（Pleine）。昔时水手相信，昴宿星团隐没的月份不宜出航，故称其为「水手之星」。亦有人追溯至阿拉伯语「丰收」（Al Thurayya）的译名，推测指的是与星团同时出现的雨水，因而联想其对农作的益处。

这些联想，让神秘主义者将昴宿星团诠释为光的高等灵性力量，一种「甜蜜的影响」；有人说，这需要神之诸子与物质生活的情境结合、契合、调和。更有人认为，它们不仅象征求道者为完成修行必须历经的阶段，也代表物质束缚之后的解脱，以及采取这些步骤所需的神圣手段。

「来吧，持杯人！这杯如皓月在我手中，为纪念国王亲吻大地。当我将摇篮曳至昴宿星团之巅，便饮下金杯，为国王祝祷。」

据说贾姆希德国王的珠宝杯曾埋藏城基之下。另有一说未经证实：昴宿星团标志著我们宇宙的中心。

金牛座另有一片美丽星群，同样常与滋润大地的雨水相系。毕宿星团标志著公牛的脸庞，是昴宿星团的姊妹星群，数量说法不一，常见为五或七颗。两团星虽皆与雨水之赐有关，古希腊人却赋予毕宿星团狂暴的声名。

神话里，许阿得斯（毕宿星团）是猎人许阿斯的姊妹。许阿斯之死众说纷纭，或狮或野猪或蛇，端看采信哪个版本。姊妹们悲痛欲绝，因而升天化为星群。

在天上，她们依然徒劳地倾泻液态的光辉，为坠落的兄弟哀悼，吟唱悲歌。

金牛座群星之中，毕宿五最为明亮，位列四大「皇家星」之一。其光辉如炬，点亮巨牛的眼眸。在牡羊座尚未取代金牛成为春分首星座之前，此星曾是众星之主，标志著春分降临。更早的世代，它被唤作「天穹之沟」（阿卡德语 Gisda），或「光之信使」（Dilgan）。亦有人称它为「耀眼的随行者」（Na’ir al Dabaran），因它总似追随著昴宿星团的踪迹。

昴宿六被尊为「中央之星」，是昴宿星团中最醒目、最恢弘的主星。阿拉伯人称之为「核桃」（Al Jauz）、「居中者」（Al Jauzah 或 Al Wasat）与「明耀者」（Al Na’ir）。在巴比伦，它被定为黄道第四星的基石（Temennu）。

故而，让我们挣脱一切卑下形体的桎梏。应如昴宿六那般稳立中心，根基深固。让我们点燃纯净火炬，宛若昴宿星团中的毕宿五。开启通往昴宿丰饶之门；愿那美德之光，如温润雨露，轻洒于正舒展的太阳系。让昴宿星歌唤醒我们，使人自梦中苏醒；愿星光化为甘霖，润泽大地丰收。让遥远星辰的歌吟，与孩童欢笑的歌声交融——唯有他，能将最深沉的长夜，转为蒙福的白昼：

「当时日流转，我将再临，喂养你们的并非苦涩之物，而是善、真与美。我的甘言将触动你灵魂深处，仿佛五月向晚，花香随风拂过繁枝。我的光将成为你的光，如同远星为夜空加冕。我将拥抱你们每一个人，亲爱的孩子，一如我们光辉之母升起，以她璀璨的护佑之光，将人心提升至神前。

纵使尘世中人困于小我牢笼，你们必得自由；这自由，正是我们神圣耕耘的果实。

因我是生死与重生之主脚下的坚石；我是历久不摧的强者；我是田野间耐心的耕者；我是金棚之主；我是「我是」之公牛；我将天界的祝福倾洒予同伴；我是尘世牢狱中熊熊燃烧的火炬；我的时刻即将来临，无人可挡。」
