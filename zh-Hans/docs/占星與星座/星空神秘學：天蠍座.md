# 星空神秘学：天蝎座
## 天蝎座的起源：多种神话解释

![cover](/img/downloaded/23c3f7dffa3a7c3836b3110542553a3e.jpeg)

在希腊神话里，天蝎座杀死了伟大的英雄猎户座。

一说猎户座过于夸口，自认能战胜地上一切野兽，才引来天蝎。有人指牠是大地的使者，另一些人则归因于月神阿耳特弥斯。此后，猎户座沉入地平，天蝎座随之升起。

苏美尔人称其为「蝎子」(GIR-TAB)。于巴比伦，此星座与西方神域相系：

「……冥界的蛇身守护者，引渡亡魂前行，直至他们在星群中的最终安息处。」

蝎人，巴比伦混沌怪兽半蝎人的一种形体，守护著太阳之门。在《吉尔伽美什史诗》中，蝎男与蝎女镇守圣山，那是英雄追索永生必经之路。守卫放此英雄进入地下通道，只因认出他是「太阳-神」的宠儿。

埃及人相传，太阳进入天蝎座之时，便是奥西里斯遭背叛、苏特开始统治之刻。

「基尔切尔的某些希腊-埃及星图里，也将苏特视为杀死奥西里斯的那只蝎子。」

然而，当光明之神与黑暗巨蛇阿帕普搏斗时，蝎子女神塞尔凯特却站在光明一方。因而她据说能开阖喉咙，给予或夺取生息。

波斯人同埃及人相似，视此时节为阉割公牛之时。中国人将天蝎座部分星宿划入青龙；希伯来-基督教传统则视蝎子为邪恶、背叛与冲突的象征。

玛雅祭司也将天秤与天蝎视为同一星座。他们似乎还把这片天域的星辰与死亡、重生相连。一份古老的玛雅黄道手稿中，蝎子伏于祭坛之上，其下暗室内躺著一具人形。

「左侧，一株绿植于夜空下开花生长，暗指玛雅人的『绿树太阳』(Yaxkin)。」

整幅图景明示蝎子、坟墓、死亡与来世之间的关联。此外，他们庆祝羽蛇神库库尔坎之死与升天，便在与天蝎座相关的「终结」(Xul)时期举行。库库尔坎等同于《波波尔·乌》中的古库马茨，也相当于阿兹特克的羽蛇神奎萨尔科特尔，时而被描绘为鹰或美洲虎。

《吠陀经》中，死亡与重生同样联系著天蝎座。此星座主要由两大象征代表：蝎子与老鹰。吠陀传统有时描绘蝎子居于暗坑，坑中藏匿珍宝。据推算，印度教的迦梨时代约始于五千年前，当时天蝎座标记秋分点；每个与天蝎相关的月宫，皆象征如此剧烈的变迁。

另一种与天蝎座相关的生物是巴斯里斯克蛇。传说此物单凭气息与目光便能致人死命，其毒剧烈，

「纵使骑士以长矛将其刺杀，毒素亦会沿矛杆上溯，不仅杀死骑士，亦夺其马性命。」

一些中世纪炼金术士，依据三重伟大赫尔墨斯之权威宣称，巴斯里斯克蛇的灰烬能点银成金。

至于天蝎座的个别星体，主星是那颗超巨型红星「心宿二」。其希腊名有时译作「火星的对手」，尽管现代权威更倾向直译为「火星」。无论何解，名字总归指向它那抹红色。

在古代美索不达米亚，与此星相关的名号有数个：乌尔巴特、种子之主、繁荣创造者、国王、英雄与国王，以及朱雀之星。在波斯，它被称作萨特维斯（Satevis），乃四颗皇家星之一，被视为天界的守望者。其中文名之一为「火」，意为火之星。于印度，它与另两颗恒星共组二十七宿中的「心宿」(Jyeshthā nakshara)。巴比伦人称它「蝎子的乳房」(GABA GIR.TAB)；拉丁语中则名为「蝎子的心」(Cor Scorpii)。古埃及将心宿二奉为蝎子女神塞尔凯特的象征；据巴吉所述，蝎子亦是伊希斯圣物。

天蝎座其余星辰之名：天蝎座β (Acrab)，阿拉伯语意为蝎子，亦称拉丁语之「爪子」(Graffias)。天蝎座δ (Dschubba)，源自阿拉伯语「额头」；天蝎座λ (Shaula)，意为「刺」，悬于蝎尾末端。
