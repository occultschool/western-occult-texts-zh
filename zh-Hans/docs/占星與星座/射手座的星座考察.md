# 射手座的星座考察
## 射手座之歌既咏唱战争，也礼赞战后的和平。

![cover](/img/downloaded/3ed2437d7b3ceebbd33724c6dd9d620d.jpeg)

射手座之歌既咏唱战争，也礼赞战后的和平。描绘了骑在马上的弓箭手，于汹涌风暴中扭转力量的平衡；以无声而致命的劲力，建立「父亲」的法则。他们唱出开始之后的结束，也唱出结束之后的开始。

他们也咏唱和解，咏唱天与地之子的调停。他们如同两个对立阵营的特使，斡旋于星夜女王与金色白昼之王之间。他们将天界的甜美气息带往尘世；他们是黎明时温和芬芳的微风，引领尘世通往天界。他们是天使窒息时的救赎者，也是火海里众生的救星。

巴比伦的帕比萨格，或称「斑纹大腿的公牛」或「多色腿的野牛」，被视为射手座的基石。在古代艺术中，他以各种象征姿态现身。其妻尼尼希纳常以犬的形象出现；描绘帕比萨格时，有时还添上一颗向后看的狗头。某些图像更赋予他双翼与狮头。

据说他曾前往尼布鲁，这是一个与渡河或渡船有关的地方，似乎暗指春分。传说中他的角色，似乎是引导新逝灵魂升入天界的向导。尼布鲁亦是恩利尔的城邑——帕比萨格之父，是风息主宰；这或许对应埃及的舒。值得注意的是，舒与帕比萨格都曾被描绘成头部具前后双重面向，将自身特性与其女性对应者泰芙努特的特质交融。

帕比萨格与尼尼希纳生下达穆，即美索不达米亚的疗愈之神，具备典型神圣孩童的诸多特质。他与植被重生有关，描绘为树液每年上升之际。他隶属于「下降与回归」的神话主题：母亲与妹妹为他哀悼，祈求归来，并下至冥界寻觅。寻找达穆的过程可见一则叙事：妹妹沿一条干涸的河床，随他步入冥界：

「冥界之河水流潺潺，却无以解渴。
冥界之地不生谷物，无从磨成面粉。
冥界的羊不长羊毛，无布可织。」

另一片段则描绘雨季水位高涨，与干涸河床形成鲜明对照：

「……自河中走出的孩童」

此景呼应荷鲁斯洪水的来临。

古老的歌声回荡岁月，穿越肥沃平原，抵达阳光遍洒的山巅，深入树林茂密山谷的幽暗洞穴，甚至掠过风起浪涌的海面。无论身在何方，他们都唱出神圣原则与动物本性的融合。他们歌颂牛人、鸟人、蛇人、狮人，乃至与蝎子合为一体者。然而，历经长久时光，射手座最终普遍凝定为人马合一的形象：手持弓箭的射手。有人说，这位弓箭手正将箭矢瞄向天蝎座。

顺带一提，大英博物馆的西帕尔界石上，刻有一尊蝎子人像，头戴圆柱形冠饰，正拉弓射击。蝎子人上半身是人首、臂膀与躯干，下半身却是蝎身蝎尾，并配备鸟腿与利爪。据我们推测，射手座或曾以蝎子人而非人马的形象呈现，这或许间接解释了，为何埃及的船星座定位于射手座区域。

「……船星座的船头横跨射手座至天秤座，航行于天河（银河）水面。」

此处的船头，可能指埃及星座中的「船头红星」，亦即天蝎座的心宿二。

在埃及智慧的其他面相，仲裁之职时而由托特代表，时而由塞布担当。然而在星象图景中，通常由舒（有时以狮形现身）作为调和的象征，分开天堂与地狱、白昼与黑夜、荷鲁斯与赛特等对立两极。

「在某些埃及黄道十二宫图里，射手座的形象与狮子结合，而非骏马。其前方是舒的人面，后方则是雌狮泰芙努特的面容。」

在此，舒与其雌狮伴侣泰芙努特这对双生子，其描绘方式与前文提及的巴比伦神话颇为相似——帕比萨格及其犬伴尼尼希纳。

在印度教传统中，阿周那拥有十二个名字，被视为主之至爱，据称是最伟大的弓箭手，因而与射手座相连。

当克里希纳向阿周那显现其神性，指引他为更高天命奋斗时，阿周那获得了「持弓者」（Dhanvin）之称。这份觉知的启示，正源自射手座的象征。

众所周知，克里希纳在《摩诃婆罗多》的库鲁克什特拉战场上，为阿周那驾驭战车、指引迷津，二人间的哲学对话遂成《博伽梵歌》。阿周那的弓「甘狄拔」，相传乃梵天以神木所制。

希腊人称此星座为「弓手」（Toxotes），拉丁文转译即成「射手座」。希伯来译法或作「弓箭手」，亦有主张应解为「彩虹」。阿拉伯人则用「Al Kaus」一词，意指箭矢。

射手座的人马形象，常被误认为半人马座的喀戎。然而多数权威视此为谬误。埃拉托色尼主张，射手座应为半人半羊，乃潘神之子、缪斯伴侣克拉托斯。古时亦有论者指出，射手座征战之气与喀戎的睿智温和不符。

此片星空之中，有中国星官「箕宿」，状如簸箕，喻示谷糠分离；另有「天鸡」星群，暗喻长夜将尽、破晓在即。

射手座主要的星包括：「人马座α」（Rukbat），源于阿拉伯语「弓手之膝」；「天渊一」「天渊二」（Arkab与Urkub），取自阿拉伯语脚踝肌腱之名，或喻阿基里斯之腱；「人马座γ」（Alnasl），阿拉伯语意为「箭镞」。巴比伦星名NUN-KI，原指其圣城埃里杜的星群，有时亦代指整座星座。

愿所有伴侣皆敢去爱；愿爱如光，照亮彼此相识者的心灵；愿我们同在名为「生命」的星穹子宫里，行当行之事。让我们携手前往尼布鲁，在渡口扎根。于巍峨与辽阔之间，于手足和睦之中，见水流一侧是先驱者的星辉，另一侧是众生辉耀之父的星辰。

在那里，聆听晨风中灵的回声，融于欢笑孩童的歌咏。

尘世之人，埋于梦之坟茔；长夜里，叛徒徒然闪躲自招的命运之箭。但你们，我的孩子，毫不迟延，当金曙降临，请随我在芳风中向地平线伸展；夜的暗袍将被抛却。

因我是光之勇者；我是提升者；解放者；杀死杀手之人；生命与意识的金色少年；拥有真正目标的弓手；我是双重吐纳；我是中道，凝于一点。我是青金石之桥；通行之星；无声的宣扬希望者；晨风之灵；水中甘甜；我是骑手，我的时刻已近，不可拒。
