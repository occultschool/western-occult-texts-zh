# 水瓶座的象征含意
## 巴比伦视水瓶座为「伟大者」，其形象关联冬季的洪流与雨水。

![cover](/img/downloaded/71215c0ca3042d7c5aebcb80e1cd1514.png)

水瓶座在巴比伦称为「伟大者」。传说他手持花瓶，倾泻出冬日的洪水与雨。这「伟大者」常塑作男子形象，双股水流自瓶口涌出，紧密连结水神与智慧之神，亦即苏美尔的恩基或阿卡德的「泉水之主」埃亚。然而，另有一说指「伟大者」曾名为古拉，是位女神，其名意谓「深渊之水」。

「古拉，即宇宙初始混沌的深渊水域；故与水瓶座恰有关联。」

更泛论之，黎明时分，太阳据说正从黑夜的深水中浮现。

每日晨光，太阳自水中重生，许多人以此比喻人之诞生。

「有人将此描绘为逃离冥界的孩子，或栖于水中的新生太阳；若以人间术语解读，犹如胎儿安居子宫的创造之水。」

埃及尼罗河每年泛滥，河水漫过密布的灌溉渠道，使土地重获丰饶。尼罗河神哈皮的形象，正是手持双罐，倾倒流水——这或许是水瓶座形象极为古老的源头。

每年水的再现与重生，亦关联一位寻找孩子的母亲，其子迷失于黑暗深渊。埃及人相信，水的源头是神圣奶妈梅纳特(Menat)，位于水瓶座；而摩羯座的山羊努姆 (Num) 或卡努姆 (Khnum)，则被尊为「水之主」。

这水纵有不同名号与头衔，埃及人却比谁都清楚：那最初的深渊之水，蕴藏著宇宙创生的永远隐藏起因。

荷鲁斯在另一层面亦是水之子，他前来拯救濒临干涸死亡的埃及。这救赎主题，在世界诸多古老传统中，皆可寻得各式版本。

若是赤诚者，救赎便是众神的恩赐。然而，这间接也是一种神圣介入，开辟通往更高、更善存在层面的道路。

这灵知的救赎根基于超越推理的洞见，亦即启示与启蒙；它揭示了一条「归家」之路，通向「太阳之门」与「彼岸」。

在印度教的黄道十二宫中，水瓶座称为水罐 (kumbha)。大壶节 (Kumbha Mela) 是印度教的朝圣盛事，信众齐聚圣河沐浴，相信能涤净罪愆。大壶节大抵十二年一轮，择地举行；每隔一百四十四年，则有摩诃壶节 (Maha Kumbha Mela)。

而今西方世界，水瓶座的代表人物则是希腊神话的伽尼美德 (Ganymede)。这美少年据说为宙斯所爱，携至天界成为持杯者，为众神斟上美酒与甘露。有说他原是特洛伊王子，亦有指他是牧羊人；关于他如何担任此职，一如往常，众说纷纭。

此星座最亮的星是水瓶座α (Sadalmelik)，名称源于阿拉伯语「国王的幸运儿」(Al Sa’d al Malik)，标示右肩；水瓶座β 则标示左肩。「女宿一」光芒自左手发出，「泣二」标示臀部，「羽林军二十六」则位于小腿。水瓶座多颗星名皆含幸运吉祥之意。

在中世纪伊斯兰传统中，基德尔能在人需要时施予超凡助力，为遇到他的人带来好运。他在传说中的性格和特征，与《古兰经》记载不尽相同：他有时是灵性导师，传授神秘奥义；在《古兰经》里，他是神的仆人，摩西曾受其指引。其他记载中，他时而为抚慰者、安慰者与帮助者，时而疗治人类过错，时而是魔法师、拯救英雄之人，是邪灵之敌、诗人与恋人的灵感之源，是圣地的圣化者与守护者，更是神之友、先知等等。然而对于苏菲派而言，他最鲜明的标志，始终关乎那生命之水的追寻；在这条路上遇见他，便是至要且至幸之事。传说他本是无形的临在，却不时显现于「神之友」眼前，与他们直接交流。

不少苏菲诗人将他与圣水紧密相系，有时，他自身便似那美妙泉眼的延伸。也难怪他总与绿意、春日、海洋相连；诸多传统、故事与传说，皆隐约指向他在宇宙学上的深意。不过，既然基德尔是生命之水的发现者，他最重要的使命，终是指引他人寻获此泉。关于他如何觅得，说法各异，但那水具有惊人神效，世人倒无异议。

「并非每颗宝石皆有那般光辉；人既可称之为水，亦可称之为日。
当基德尔认出这泉源时，他双眼顿时被照亮。
他下车，迅速褪去衣衫，将头与身躯浸入那纯净泉中。他尽情啜饮，以求配得永生。」

基德尔是否确为某一具体人物，抑或是一长链灵性导师的原型，甚或是求道者自我的投射拟现，历来争论不休。真正的持杯者是谁？那盈满圣杯的闪耀液体又是何物？这奥秘始终萦绕不散。无人能向他人展现其奇迹，唯有寻者能向自己揭示；但曾有一位作者，在目睹天界及其圣水时，描摹得比谁都更为接近：

「……世间之水不过影子，而爱我、信我之人，将与我同享天界活水。
须知，一切「如是」者，皆源自天上之井，是我父——人称毗湿奴圣主——的流溢。
一切亦源自我母，即毗湿奴的天后德瓦基，在世间被称为摩耶（少女），她乃至高神的配偶。
正如那水是万物之始，不论存于尘世、诸天界，抑或诸神世界之间的空间；
我亦是那活水，是那神圣双亲最初的流溢。
因我是万物的起始、中程与终结，无论生死；我是那活水，凡渴慕我者皆可自由取饮，无阻无碍。
我即永生之水，而水即是灵；于一切生命，我是其永恒本质；于一切似死非死之物，我亦是其本质。
因它们同样充满生命之水的灵，唯对俗眼隐蔽，故凡人若欲寻觅这初始之水的灵性本质，终将徒然。
它超越一切知识与概念，全然神圣，承继父母所创之性。
它是一种本质、一种水、一种灵、一道金焰，至圣无比，见我者即见本质。
故我告你真相：水乃万物中最先被创造，包含毗湿奴之火与摩耶之湿；凡以灵眼寻觅观照者，必于其本质中见我，即克里希纳——父与母之子。
凡居于其辉光中者，必得永福，因他们知晓，是我将火与湿融合，成为遍在之本质，在我人身与灵性之中。
我是宇宙之水的主，行于其波澜上，吐露温婉言语，如抚慰灵魂的油膏，使软弱者得安慰——皆奉我父我母之名。」

啊，但愿神所立的祭司能向寻求者亲身显现。愿他们能同行，扎营于高原，傍依永生的清溪。愿不落的星在头顶柔柔发亮；愿澄澈的水在心底注入甜润乐音，唤起欢笑少年的歌：

世人在荒芜沙漠度其一生；任性的叛徒，被无羁欲念的诅咒击成聋哑。

但跟从我的人，必在平和园中、天堂泉边安歇；他们将于天使歌吟里，得享喜乐与满足。

因我是那献上言语之杯者；是注满杯盏者；是泉畔取水者；是疗治诸病的药；是使一切圆满者；是善人的命运；是分赐灵感活水者；是圆顶厅堂的导师；是洒水之人；是翠色极光；是红颜黎明；是内在异象的赐予者；是永葆青春的老者；是盛纳「如是」的壶；我是隐藏之神的儿子，我的时刻将至，无可拒却。
