# 天鹅的神话与象征：从古代神话到宇宙的启示
## 探索天鹅在神话、宗教与占星学中的象征意义 —— 从阿波罗的优美歌声到宇宙的起源传说

![cover](/img/downloaded/fcac2ccab1a7bf9415253835d019c761.jpeg)

「……天鹅平日也唱歌，但预知死期将至，反会以最甜美的歌声，欢欣地唱给神听。天鹅本是神的仆役。凡人惧死，这才误解了天鹅，以为牠临终悲鸣，是为自己哀悼……天鹅既然属于阿波罗，便有预见异象的本事；牠们预知彼岸有福，所以死前欢唱。」

卡里马科斯在《德洛斯赞歌》中记载，阿波罗诞生时，鸣唱的天鹅曾绕德洛斯岛盘旋七匝：

「……雪白的天鹅身为阿波罗的美妙歌者，自帕克托卢斯的金色海岸启程，绕岛飞了七圈，歌声温煦如流转的日光；柔美的旋律深得九位缪斯钟爱，以绝妙音色迎接新生。阿波罗是和声之神：七弦琴之主。随后他赐下金竖琴。在第八场温柔合奏开始之前，他降临人世…」

天鹅是候鸟，在秋天如太阳般南飞，春日又如备受期盼的太阳归来。在《帕西法尔》故事里，罗恩格林被一些人视为太阳的象征，乘坐著白色的月天鹅之舟。凯尔特神话中，天鹅少女著名的变形能力，对神秘主义者而言，隐约暗指德鲁伊祭司学院里的神秘转化。德鲁伊祭司仪式所穿的斗篷，便以天鹅皮羽织成。

北半球的民间传说与神话，常将天鹅与转变、蜕变相连。此观念流传甚广，历久不衰，就连丑小鸭化为美天鹅的故事，亦可追溯至这古老的思想宝库。

中世纪炼金术士曾隐晦提及黑乌鸦变为白天鹅，同时暗示这天鹅体现了对立面的结合或连结。宙斯与莱达的结合亦复如是；宙斯化身天鹅诱引了少女。此番结合诞下了双子座的孪生子，以及与之相对应的女性。有人留意到，天鹅的形貌融合了男女特质，既似产卵的鸟，亦如使之受孕的蛇。

在埃及，鹅被描绘为产下太阳金卵的巨禽；牠也是伟大母亲或伟大雌鹅的典型，被视为人类的起源。圣母哈索尔是慈爱的典范，纯洁无瑕，曾以神圣母牛之形，以乳汁哺育荣耀者。然而数世纪后，其形象竟演变为「维纳斯—阿佛洛狄忒」的淫祀，连同其圣鸟白天鹅，也一并成了情欲与感官愉悦的象征。

印度与埃及共有相似的观念，认为宇宙源于绝对者光线所生的「天鹅」，或称万古之鹅。据说此鹅产下宇宙蛋，万物自此诞生。另有传说提到，梵天乘坐七鹅牵引的战车；天鹅亦是萨拉斯瓦蒂的坐骑——萨拉斯瓦蒂乃梵天之伴，是智慧、诗歌与音乐女神。在这片夜空区域的音乐象征里，她与流水息息相关。

「愿水为吾母，涤净吾身；愿那以酥油净化者，以酥油净化吾。因众女神承担污秽，我自她们之中诞生，纯洁而光明。」

古老体系中的天鹅，与纯洁、灵感之水、歌声及音乐紧密相系。阿波罗身为音乐与诗歌的守护神，其战车由白天鹅牵引，传说他有时也会化为天鹅。其他传统中，天鹅与缪斯女神关系密切；许多著名诗人与歌者据说死后皆化为天鹅。

奥菲斯死后化为天鹅，与他珍爱的里拉琴一同安息于天堂。名琴师阿里翁死后亦成天鹅，他曾为德尔菲努斯的海豚所救。荷马是米兰德的天鹅，维吉尔是曼图亚的天鹅，而离我们时代最近的莎士比亚，有时被追随者唤作艾芬的天鹅。

至于天鹅座，阿拉图斯与托勒密皆未称之为「天鹅」星座。对他们而言，这星座仅是一只「鸟」。有些现代学者认为，牠对应远古巴比伦的安祖鸟星座，尽管据说此星座早在约三千年前便已自天际隐去。

「天鹅座展翅翱翔，双足轻曳，与安祖鸟的描绘极为相似，而安祖鸟座原先应当位于这片天域。」

埃拉托色尼与希吉努斯则将此星座与天鹅相连，并讲述类似的故事：宙斯化身天鹅，追求能产卵的涅墨西斯。埃拉托色尼称之为「天鹅」，拉丁文「天鹅座」便由此而来。

一如往常，天鹅座的渊源向来众说纷纭。有人说是宙斯为求莱达，化身天鹅；亦传法厄同驾日车坠落埃里达努斯河，这星座便是为他哀泣的故友或手足。阿拉伯观星者视其为飞鹰，亦有人称作母鸡。因群星勾勒出十字轮廓，它亦被唤作南十字。

座中最亮者当属天鹅座α，其名取自阿拉伯语「尾羽」，正是天鹅尾端那点银芒。此星与天鹰座α、织女一相映，织就夏日大三角。天鹅座β美如珠玉，标记鸟喙；天鹅座δ与ε则素朴无名，悄然点缀双翼。

在中国星图中，天鹅座α偕众星构成「天津」，恰是银河一处津渡。天鹅、天琴两座之间，星光迤逦形成「辇道」，仿佛直抵天宫的御路。另有「羲仲」之星，相传乃发明战车的古人。

哦，让我们如旷野白鹄，航向龙口，静候命定之时——届时此岸与彼岸的帷幔将薄如晨雾。让我们怀著内心的宁静，吟唱重生的天鹅之歌，航行于银河澄澈的水面，驶向蓝天与青梧的丽土；在那儿与灵魂知己共享恩典与欣悦，庆贺安然归返，远离一切宵小之扰。

且乘这叶轻舟，凭纯粹本心引领，载著圣洁之货——圣杯、生命之水与光明之源——穿越长夜，迎向新创造的黎明。当旭日初升，光芒遍洒，我们将听见欢笑孩童的歌声。
