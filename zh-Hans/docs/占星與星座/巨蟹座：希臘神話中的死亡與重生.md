# 巨蟹座：希腊神话中的死亡与重生
## 探寻巨蟹神话与文化意涵，揭开生死缠绕与重生微光

![封面图片](/img/downloaded/f70816a4bd66f406800e147899e042de.png)

那夜，一轮神秘的月亮将它王国的秘语倾泻大地。月色圆满、静谧、高悬，幽邃难测。星光洒落的松林间，旋律流淌，仙女与林间精灵翩然起舞。这咒语般的调子，回荡在地球隐蔽的洞窟与地窖，渗进牢房与地牢，飘过银色宫殿与金色塔尖，最终栖息在千家万户、人心深处。

星座之中有两道门户，以巨蟹与摩羯为象征，古人谓之「人类之门」与「神灵之门」。他们认为，巨蟹是灵魂踏入尘世血肉的起点，摩羯则是灵魂挣脱形骸桎梏、觅得自由的归处。

传说赫拉曾遣一只螃蟹，在海克力斯与九头蛇搏斗时紧咬他的脚跟。长蛇座北端紧邻巨蟹，据希腊神话所述，螃蟹与九头蛇同在勒纳湖栖身，亦称阿尔西尼亚湖。此湖被形容为无底深渊，是致命的渊薮，相传通往冥府。酒神狄俄尼索斯（或称巴克斯）试图自幽冥迎回母亲时，便由此入口潜入。

「这传说我也听过。湖面看似波澜不惊，实则任何试图渡湖之人，皆会被无形之力拽入深渊，卷进湖水暗流。湖周不大，约莫希腊赛道三分之一长。岸边蔓草丛生，芦苇摇曳。每年纪念狄俄尼索斯的秘仪于夜间举行，其细节恕我不能向世间透露。」

在诸多古老神话里，螃蟹与乌龟常与死亡相系，也象征穿越生死边界的使者。中国将此星座中一个星团（鬼宿星团）唤作「积尸气」。埃及制作木乃伊的匠人，则常以圣甲虫形石心取代逝者心脏，称其为「石心」，并将此仪式与巨蟹座相联。

然而，这些巨蟹座之歌仅有少数人能聆听；它们吟唱的不是同伴坠落凡尘，亦非死亡的毁灭，而是游子归家的故事。歌声诉说自我寻回正途的旅程；咏叹那些曾经迷失流离、终获平安的人，如何重回主的怀抱。妳是高低神秘的月，妳是名号众多的母亲，妳是滋养世间的子宫，妳是通往生与死的大门。请垂听我等祈祷，引领所有同伴安然归家。

他在小舟甲板上默然祷告，仿佛听见——或自以为听见——死亡与重生的吟咏渐次消逝，取而代之的，是一阙重生之歌缓缓升起。因为他与全体船员踏入此门时，皆是盲目受苦的孩童；但当他们从中走出，将如凤凰浴火，自一切光辉中重生翱翔。

噢！你是奥西里斯之鸟，你是再生复活之鸟。在你无尽的岁月里，双翼飘散乳香、没药与朱砂的气息；羽毛浸透圣火的芬芳。当燧石相击，火花自沉睡苏醒；火焰瞬起，焚尽一切尘世缠缚。天狼星之火在凤凰心中炽燃；凤凰腾空，栖息于黎明旭日点亮的金枝之上。

同伴们，且记住：在星罗棋布的夜空里，自某个转折之处，出现了一位高飞的翱翔者或航行者。他是一股宏伟之力，众星之王；他是羊群的牧人，牛群中的公牛，亦是平原上吼啸的狮王。

「向你致敬，噢！你是万王之王、万主之主、诸侯之君！你自努特子宫统御人世与冥界。你的身躯流转明亮的金属光泽，头顶苍穹，绿松石光芒环绕周身。噢，安神，你已存在千万年，贯穿万物，在神圣之地——即那冥界——拥有至美面容。请赐我天界光辉、尘世力量与冥界胜利。允我如活灵般下行至塔图，如凤凰般上行至阿布图，并许我自由出入冥界塔楼，无碍无阻。」

在古埃及人心中，神圣圣甲虫象征永恒与不朽。纵在死亡表象之下，那不断更新、循环往复的生命，仍是埃及创世神凯佩拉的颂歌。其名含义为存在、生成、创造与流溢。

在埃及人看来，创世并非独一无二、不再重现的事件，而是永无休止、循环调节的过程。黎明时是荷鲁斯-太阳，正午化作拉-太阳，威严赫赫；西沉时又变成图姆，遁入奥西里斯的冥界。在那里，太阳经历进一步的转化，终将以胜利者的姿态再度显现：他曾四散，而今复归完整；他战胜了黑夜，如同白昼般新生。

那曾被埋入土中的圣甲虫，同样重获生命；它展翅，迎著晨光振翅再起，紧紧攫住太阳。

古昔的同伴相信，这便是善人的命运。死亡于他并不可怕；那不过是通往新生的大门。即便被黑暗围困，看似毁灭，他的信念也毫不动摇。正如一位埃及学者所言，即便在那个时代：

「他期盼在永恒花园东隅的阿鲁之地，享用奥西里斯的筵席。」

哦，愿所有浪子都归返父亲的怀抱；愿迷途的母羊安然回到羊圈；愿离散的部族再度凝聚；让我们点亮灯火、清扫屋宇，直至寻回失落的珍宝。当所有隐藏的财宝都纳入救世主的箱箧；当遗失的银币重回寡妇的钱囊；当园门开启，羊群归栏：那时，所有同伴都将欢欣鼓舞，同声高唱归家的歌谣。

蟹与圣甲虫紧紧抓住所有，如同大地环抱种子，母亲怀抱婴孩，父亲拥抱回头的儿子。巨蟹座便如此吟唱著星辰之歌。

哦，你这珠宝的收集者，这些珠宝光耀夺目，宛如神秘夜空中纯然发光的星辰。请将它们珍藏于怀，令其复归本位。哦，你如伊什塔尔般沉降，又在你所爱之人的怀抱中再度升起；你寻觅并寻见，从物质经验的深井中，汲引天界之水。莫让种子在黑夜里无人照看而枯萎，让它们自黑暗土壤中迸发，如同永恒之树，在天空的永昼中闪耀。

希腊人称此星座为 Karkinos，拉丁人译作 Carcinus，即螃蟹，我们故称其为巨蟹座。在巴比伦，它唤作 Mul Al-lul，有学者将此名追溯至一则远古传说，关乎英雄尼努尔塔；这位英雄的历险，在在映照著海格力斯的功绩。海格力斯曾战多头蛇，尼努尔塔亦然。希腊版本中，多头蛇得螃蟹助阵；巴比伦版本里，则换作乌龟襄助。

此外，据怀特的观点，此星座的苏美尔名称在阿卡德语成为「螃蟹」（alluttu），其复数形式可指代夹钳。据说在希伯来语、阿拉伯语与叙利亚语中，此星座的名称象征著持守、拥有与保全之意。

在印度人的二十七宿体系中，此星座内许多星辰居于「花」宿或「吉祥」宿之中，其主宰神是祭主仙人，众神的祭司与导师。

鬼星团，这悬于巨蟹胸前的美丽星群，除了「腐烂尸体」的含义外，亦有「群众」的含义。因其光芒朦胧如雾，古代观星者称之为「小云」、「薄雾」、「多云者」或「旋转之云」。世上某些地方的神秘主义者相信，这小微云或薄雾能产生洗礼用的纯净露珠。鬼星团也曾被视作星群，因而得了「蜂巢」这个流传甚广的别名。

然而，希腊语中，「蜂巢」一词亦指「马槽」。而在鬼星团两侧，有两颗星被希腊人唤作「驴子」，又称北驴与南驴，阿拉伯语谓之「双驴」。因此，早在基督降生之前，马槽的形象便已存在于星空——位于两驴之间，处于关乎诞生与重生的天域。有人将这两头驴与希伯来的两位小天使相联系，或与埃及代表上下半球的两位狮神相联想，据说光明便诞生于这双狮之间。

尽管尘世居民无法直面，太阳的居民亦无由受益，但你们是我亲爱的儿女，你们将得见阿卡尔的奥秘；因我将牵著你们每个人的手，与你们同循椭圆之路，前往永生之地。

因我是时代之眼；我是大门；我是金蛙与绿松石蝶；我是变革之力；我是透入洞穴的光；我是水面上的风；我是珍珠的造主与贝的开启者；我是火的凤凰神；我是自生之我；我是生命之主，我的时刻将至，无可拒绝。
