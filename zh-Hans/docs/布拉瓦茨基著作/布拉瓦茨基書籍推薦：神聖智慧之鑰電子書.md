# 《神圣智慧之钥》电子书
## 评介布拉瓦茨基三部要著：《神圣智慧之钥》、《布拉瓦茨基夫人的神秘世界》及《神秘学研究》。

**一、《神圣智慧之钥》 — H. P. 布拉瓦茨基著**

![封面](/img/downloaded/0ca9629c0e4e96f378c30e029552286a.png)

中文版电子书：[点击阅读](/books/key-of-sacred-wisdom-blavatsky)

若觉得布拉瓦茨基两部鸿篇巨制——《揭开伊西斯的面纱》与《秘密教义》过于庞杂艰深，此书便是一把绝佳的钥匙，为西方读者初次开启古老智慧之门。全书以问答形式写就，直接回应学子诸多疑问，条理分明，娓娓道出神秘科学的基本原理。

探讨主题广博：人的七重结构、进化与幻象、灵魂与灵之区别、记忆本质、前世、轮回、死后与出生之际的意识、因果报应、通灵现象、死亡之后、祈祷、慈善、苦修、婚姻、智慧大师，乃至万物同源的根本一体性。

但须提醒：书中每则答案，未必全然无误。有别于《秘密教义》，此书写给背景各异的读者，其中许多人对布拉瓦茨欲阐明的真理所知甚少，甚或一无所知，且无意改变既有的思维与习惯，难以将真理吸收、践行于日常生活。因此，她采取了真正教师的策略：因材施教，调整答语，只为让学生更易领会。

书末附有二百四十七条词汇表，罗列宗教、哲学与神秘学术语，释义诸多晦涩梵文词汇，于不谙此道的读者大有裨益。

莫被「神智学」一词吓退。其意其实简单，无非「神圣智慧」，亦即神圣知识或科学。本书虽应神智学协会成员之请而写，至今仍作教材使用，却绝非该会宣传册子，目标也不限于会员。相反，它精彩阐释了贯穿历代的智慧，足以吸引所有寻求真理之人。故此，我们诚心荐予各位。

**二、《布拉瓦茨基夫人的神秘世界》 — 丹尼尔·H·考德威尔 编纂**

![插图](/img/downloaded/d7de6fdd875c388484897db81c3d90eb.png)

这是首次由真正了解布拉瓦茨基的人来述说她的故事。丹尼尔·考德威尔笔触鲜活，勾勒出这位旷世神秘学家的身影——她独力复兴古代智慧与各时代的神秘真理，其影响跨越古今。

作者成功汇集亲属、熟人、友人、敌手与同僚的诸多回忆，让读者得以窥见布拉瓦茨基的生活、工作，以及她那神秘难测的人格。用「窥见」一词，是经过斟酌的。她真实的身分，知者极少；即便生前最亲密的同事，亦不明了她为人类艰辛付出的真正目的，至今依然成谜。因此，在近年关于这位卓绝神秘学家的传记中，我们最钟爱考德威尔此书。

许多长期读者皆知，本专栏文章内容，多有承袭自这位神秘学先驱的教诲。过去数世纪中，这位古老秘密的伟大守护者所传授的神秘学真理，远超任何其他导师。若您尚未读过布拉瓦茨基的著作，对她的认识仅止于多年流传的诽谤谣言，那么这部出色的传记，将助您发现她的伟大，以及她所教导真理的崇高。

**三、《神秘学研究》 — 布拉瓦茨基著**

![插图](/img/downloaded/940d11d026ee3a05875fafbb315e7f9f.png)

此书汇集布拉瓦茨基晚年发表的数篇专文，深入探究神秘学的核心实践与理论。文字洗炼，见解犀利，直指修行与认知的本质。对于已略窥门径、欲更上层楼的读者，无疑是进阶的指引。书中论及意识转化、能量运作与宇宙法则等深奥课题，读来如暗夜明灯，廓清诸多迷思。若说《神圣智慧之钥》是启蒙之钥，此书便是深入堂奥的阶梯。

![插图](/img/downloaded/940d11d026ee3a05875fafbb315e7f9f.png)

《神秘学研究》所辑文章，原刊于布拉瓦茨基主编的杂志《路西法》。该杂志发行于一八八七年至一八九一年间，即她逝世当年。布拉瓦茨基清晰辨明：理论与实修本质有别，一如左道的邪术与右道的利他奉献，两者之间横著深渊。

书中探讨诸题：实修密学与秘术、催眠现象、十九世纪军火贸易之罪、科学中的黑魔法、宇宙意识与高等心智、智慧的双重面目、星光体，以及人在物质与灵性层面上的真实构成。

另收一篇长文，分三部分，题为《福音的奥秘性质》。此文深入剖解耶稣的神秘性——既为历史中人，亦是基督原则的体现。
