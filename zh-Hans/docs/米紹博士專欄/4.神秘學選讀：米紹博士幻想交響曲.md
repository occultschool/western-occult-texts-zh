# 《幻想交响曲》
## 这部作品以象征与寓言交织，铺陈魔鬼的阴谋，同时深掘身体、心智与灵魂层层交叠的奥秘。

![封面](/img/downloaded/9dd64495a58d253abe45c2b386e4db49.png)

电子书连结：[点击阅读](/books/fantasy-symphony-dr-michaud)

字典里将「魔鬼怪谈」定义为「描绘黑魔法或与魔鬼往来的文字图像」。这部「文字交响曲」虽处处描绘魔鬼，却不止于阴谋奇幻。与作者所有著作一样，《幻想交响曲》满载繁复多层的象征、寓言与隐喻，唯有深入研读冥想，方能窥见堂奥。并非说一般读者无法喜爱此书，而是它最深刻的真理与教诲，远远藏于表象之下。这个「魔鬼怪谈」的宏大主题，实是历经苦难而得的救赎与重生。它与弥尔顿史诗《失乐园》颇多相似，但弥尔顿意在「向人类证明神之道」，米绍博士却透过揭示若干伟大隐藏的奥秘，阐释神之道——至少在那可被揭示的范围内。

作者虽是技艺精湛的音乐家，叙事结构亦紧密遵循古典交响曲的四乐章形式，这却不是一本关于音乐的书。一九四二年，英国作曲家格兰维尔·班托克爵士（1868–1946）初见手稿，惊叹：「太精彩了，必将轰动！」此书主题崇高、语言优美、蕴含丰富神秘学知识，确实气象恢宏。然而出版后并未引起轰动，如今几乎无人知晓，也少人阅读。正如我们在多篇文章中指出：真理向来是最不受欢迎的商品，这结果并不意外。但这无关紧要。识货的读者自能在《幻想交响曲》中，发现与作者其他作品同等的美、启发与深奥隐秘，感受那种欣悦的「触动」，对每一页流泻的智慧、温暖和幽默，满怀深切的感激。这是一部以文字织就的灿烂交响。

书中重大事件发生在距今约四十二万七千年后，正值迦梨时代或「黑暗时代」终结之时——这段历史可在作者另一著作《克里希纳的真实福音》中读到。这出形而上学戏剧的主要角色包括：冥界王子长短格与其恶魔部属；两个高度进化的灵魂，马德隆与塞巴斯蒂亚诺；以及睿智的吉普赛老妇米泽拉——她将大自然隐藏的秘密传授给杜尔斯与法尼，那对孩童正是马德隆与塞巴斯蒂亚诺的小我。书末附有完整角色清单，以及那些完美交织于叙事中的芭蕾与舞蹈。

从未有哪本书，能如此清晰正确地解释身体、心智与灵魂的本质，以及「星光体」投射、「堕落天使」与下层星光界的情形。也从未有书如此深刻地刻画「魔鬼」的形象——那高傲的地狱王子长短格。第二乐章中，长短格揭示了黑暗势力用以在世间播撒邪恶种子的工具与诡计。这些玄秘真理从未如此公开陈述，但时机已然成熟，可将其更广泛传播：令善良之人从中受益，而那些背弃光明、拥抱黑暗者则将明白，等待他们的并非美好之事。

一个人即使读遍千本知名作家的「神秘」著作，所能发现的伟大真理与崇高智慧，恐怕还不及这本薄书（印刷版仅一百零一页）的一小部分。这部宏伟戏剧在第四乐章达到高潮，拟人化的行星及其统治者向长短格与其地狱军团发动战争。最终交战前，诸多事件接连发生——有些洋溢巨大喜悦、美德与美丽，有些浸透极度悲伤、罪恶与丑陋。每一事件都揭示若干真理或智慧宝石；这是其他作者任何书中都寻不著的天界珍藏。

在结束这篇简短而远非全面的介绍之前，必须再次声明：这不是一本关于音乐的书。然而，读者中若有古典音乐家，或能在第三乐章谐谑曲里，从那些涌现的古朴乐器、乐师与舞者间，窥见更繁复层次的含义。我们曾试图点算出现的人数，数至数百便放弃了——或许正因如此，《幻想交响曲》从未搬上舞台。要寻得一位非凡的制作人，与一群既勇敢又天赋超凡（甚至堪称鲁莽）的音乐家，几近不可能。

即便如此，这本书——或戏剧，或交响诗，任你称呼——因其包容上述一切，且不止于此，终将上演一出空前宏大的乐剧，实现七十余年前格兰维尔·班托克爵士初读手稿时所预言的「轰动」。谁知道呢？或许你读罢此书，为之倾心，进而将这部「文字交响曲」引荐予更多读者。那会是真正高尚之举，亦是对作者不朽才华最恰当的致意。
