#  白魔法师的典范— 德鲁伊祭司
## 真正开悟的白魔法师如何服务人类，非那些假托神秘者所能想像。

![封面图片](/img/downloaded/7951f137608defa64b3eb447051b82d7.jpg)

下文节录自一次真实魔法仪式的记载。仪式发生在遥远的过去，英国岛屿尚受德鲁伊智慧统治之时。内容由天赋异禀的神秘学者兼先知杰弗里·霍德森回忆并记录。

「我置身于一座完整的德鲁伊石圈，巨石高低参差，自三十公分至一百八十公分不等。群山环绕，独缺东面。以内在之眼，我看见许多奇异变幻的景致；其中一位气魄雄浑的人物，在此地留下了深刻的印记。他是一族的祭司、导师与医者，身形伟岸，犹如古老传说中走出的粗犷英雄，格外醒目。

「他比常人高大，仪态威严。长发与胡须乌黑，身著一袭垂至脚边的白衣，样式有别于今日的法袍。我看见他站在此处，亦即我此刻所坐之位，在岩石的内圈之中；身后是一众祭司，同样身披长袍。

「一根已不复存的巨岩顶端，悬著一面纯白旌旗，绣有金蛇图腾。大批人群聚在石圈外围远处，仿佛静候信号。这位大祭司显然深谙魔法，他举起双手朝向苍穹，仰首高呼。顷刻间，各阶天神于空中盘桓。应其召唤，其中六七位最为宏伟者在他头顶聚成圆环，直径约莫两三公里。天神们的手在中央的火焰中交会，那火焰凝实具现，以供肉眼得见。

「诸天神有一显著特征：每位的额际皆环著一道细窄金带，镶有璀璨宝石。金带于头部数处向上延展，前额上方尤有一组较大的光点。

「火焰降在大祭司面前的祭坛石上，无柴无薪，却燃烧得炽烈旺盛。随后其余祭司分作两列，行至坛前，低吟沉厚而带喉音的颂歌。火焰既降，人群便分成数行，面向北、南、西三个入口前进，直至汇聚于圆心，并在中央空出一片广阔之地。众人与祭司——除大祭司外——依次向火焰致敬；他们伸臂垂首，静止不动。大祭司则诵出一篇绵长的祷词。

「一道强大的磁性绝缘体随之显现，隐约笼罩整个石圈。从神秘学的角度来看，这将现场之人隔离并庇护，宛若一座完整而坚固的神殿。当人们垂首肃立时，天神降临，一股宛如闪电的力量，沿著天神脊背迸发为巨大的十字。较小的自然精灵占据了十字臂间的空隙。有些精灵沿著十字线条维系力量，有些精灵则如人类般前来参与这场集会。

「大祭司显然感知到某些强大的灵性存有，他向祂们祈祷并得回应，这些存有正是那股流动力量的源头。祷告的效果立竿见影，苍穹似被开启，一股雄浑之力倾注中央广场。广场三面人群密布，第四面便是祭坛石。人们再度挺身站立，诸天神则变得异常活跃。诸天神最关切之事，乃是这股力量能否尽数传达至人手中，并将溢散浪费降至最低。

「能量向下奔流，持续良久，形成一根名副其实的生命力巨柱。我无从描摹其样貌，任何言语都办不到。最接近的比喻，是说它如液态火焰，又如珍珠母贝，呈现乳白底色，内里透著玫瑰色的辉光。它向下贯入地底深处，向上延伸至目不可及的天际。大祭司清晰见之，若干祭司亦能得见。众人或可感知其存在，但真能目睹者，寥寥无几。

「众人神情肃穆，怀著虔敬之心，深知这场景的神圣。

「大祭司示意之下，几名病者与老者被抬进会众之间——即人群构成的十字两臂的空处；随后，他们或被引导，或被搀扶，来到中央广场。无法站立的，则安置于草地上，是在力量柱体之外。那日天寒，一些病弱之人似乎难以支撑。然而，他们此刻所处的位置，竟使状况大为好转。

「我看见一位面貌苍老的男子卧在广场北侧；他以右肘支起身子，左手朝柱子伸出，仿佛在炉火边取暖。一道温热而带著磁力的光流经他全身，双眼顿时明亮；不久，他竟起身跪下。尽管身形仍摇晃，精神与体力却明显焕然一新。接著，他掌心向上微捧，一名祭司将少许淡黄液体倾入其中，老人饮下；其余男女老幼，也皆得此液。即便有人带伤，大祭司只须以手轻划伤处，立时愈合。

「一股宏大的力量流贯大祭司全身，他仿佛整个人浸在金色辉光之中。他显然深谙人内在的潜能，指尖所触，于病者皆有奇效。这部分仪式历时约二十分钟至半小时，之后，众人的目光全聚向大祭司——他立在一方平石上训诲。我虽不懂其言语，大意似是呼唤人性本真，促人团结，相依共存。场中亦有些蛮野灵体；他们虽一时被大祭司与仪式之力所慑，本性却绝非温顺服从之辈。

「大祭司言毕，便领众致谢。众人齐身三鞠躬，至第三躬，大家仍垂首之际，大祭司忽又抬头望天，口中念诵。力量的灌注戛然而止，光柱亦倏然消逝。接著，人们转身与祭司一同诵经；末了，他们自入口处步出圈外，队形渐散，静候于原地。

「大祭司随即转向蛇形符号与面前的祭司们。他向他们说话、祝福，同时迅速以手在空中画出某些记号。那些记号圆转如环，或似螺旋，自他掌中流入祭司们的气场时，依然清晰可见。随后他双臂微张前伸，宛若欲拥抱这群祭司。他们低下头，一股力量从大祭司那儿涌向他们与旗帜；这情形持续了好一会儿，约莫十秒。其间，诸祭司神情升华，进入一种崇高的状态。接著旗帜卷起，祭司们走出内圈，与相识者交谈。人群渐散，他们也随之离去。」

## 后记
