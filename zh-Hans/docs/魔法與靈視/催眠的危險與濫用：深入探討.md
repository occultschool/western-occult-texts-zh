# 催眠的危险与滥用：深入探讨
## 催眠虽能提升意识，亦可能遭恶意利用，沦为不法手段。认清其中风险，方能防范滥用。

![cover](/img/downloaded/6c888aea89295b680ca8dd2903f74cf2.png)

本文将探讨催眠潜藏的危险与滥用问题。

正如诸篇所论，半真半假的言论，若披上善意外衣，往往比全然的谎言更为危险。催眠领域亦然。关于催眠，流传许多似是而非的说法，譬如声称它只是一种「自然」的放松与专注状态，可借由暗示提升意识。实情远非如此。根据[《催眠术的魔力与奥秘》](/zh-Hans/docs/the-magic-and-mystery-of-hypnosis)中的讨论，催眠实质是将一人意志加诸另一人。

这过程本身就潜藏危机。催眠常被描绘得安全无虞，毫无副作用。此说仅道出部分事实。在专家手中，催眠是极其强大的工具；但凡力量强大之物，既能造福，亦能为祸。正因如此，十九世纪神秘学家布拉瓦茨基曾写道：

「……不明白神秘法则者，所施行的『暗示』实验，是最危险的把戏……任何催眠师……无论多么善意、可敬，一旦不经意教人滥用这门神圣科学……在道义上，他便成了此手段所犯一切罪行的共谋。」

在深入剖析催眠过程与后续问题前，须先澄清几种常见误解。受试者常被告知：「你将完全清楚听到每一句话。」这仅适用于极浅的恍惚状态。当受试者陷入「深度催眠」时，往往无法忆起听闻或诉说的话语。二十世纪心灵感应者埃德加·凯西（Edgar Cayce，1877–1945）便是一例。在自我诱导的催眠中，他对旁人的言语毫无记忆，那些著名的预言，事后须经人复述方能知悉。

另一误解，是认为催眠师无法驱使人违背自身意愿行事。此说亦谬。催眠史上，个人受催眠而做出违心举止的案例，屡见不鲜。二十世纪俄罗斯心灵感应者沃尔夫·梅辛（1899–1974）的事迹可为佐证。苏联独裁者史达林为梅辛的神秘传闻所惑，命其催眠银行出纳人员，使之交出十万卢布（价值相当于今日数百万英镑）。梅辛仅递给出纳人员一张白纸，同时施加催眠暗示，令其视之为面额相符的支票。出纳人员竟真的照付现金！

现代社会亦不乏此类事件，部分甚至经电视台记录下来——如出纳人员遭催眠交出钱财，或行人在街头受催眠后遭劫。更广泛的暗示亦属可能，只需回想希特勒对数百万受蛊惑追随者的邪恶影响，便知催眠滥用之害可达何种境地。此等力量，有时如梅辛这般天赋异禀的术士有意施展，有时如希特勒这类独裁者无意识驱动。

常闻一种说法：「人在『无意识』状态下，会展现异常智慧。」这实际上源于催眠师与大众普遍对「低等心智」一无所知。于[神秘学课程文章](/zh-Hans/docs/twelve-lessons-in-esotericism)中，我们曾援引《死海古卷》经文，描述神所指定的「虚假之灵」——即低等心智——伴随每个男女的尘世之躯。据载，此「灵」包藏：「贪婪、懈怠正义、充满邪恶与谎言、傲慢与骄矜、虚伪与欺诈、残酷与诸般恶行、乖戾脾性，以及众多愚昧与无耻的狂妄……眼目昏盲，双耳沉聋。」

凡此种种，皆揭示了催眠的阴暗面貌。布拉瓦茨基早于其著作与无数文章中，向世人发出警告。

许多读者想必听闻一种催眠滥用现象：「撒旦虐待仪式」（SRA）引发的道德恐慌。此风潮源于一九八〇年代的美国，继而蔓延至英国乃至全球。这股令人不安的浪潮，起源于报导声称有人被迫参与所谓「撒旦仪式」，并在违背意愿下遭受身体与性虐待。恐慌达至顶峰时，更出现指控，指称全球存在一个由富贵权势精英组成的阴谋网络，绑架或养育儿童，用以献祭、制作色情内容及卖淫。

「撒旦虐待仪式」一事，几乎每个环节都充满争议——从定义、指控的来源与真伪、声称受害者的证词，到相关法庭案件及刑事调查。这场恐慌席卷了政客、名人、警察、司法机构、治疗师、社会工作者，最终连催眠师也牵涉其中。虽然催眠师绝非恐慌的主要推动者，但有证据显示，催眠在制造虚假记忆方面起了关键作用，催生出许多指控。

虚假记忆极易植入，此事已多次获证实，尤其透过美国认知心理学家伊丽莎白‧洛夫特斯的开创研究。1997年9月，她在《科学人》杂志撰文，记述一名叫纳迪安‧库尔的案例：库尔接受催眠后，揭露了自己童年受虐的压抑记忆。治疗过程中，她逐渐深信自己曾参与撒旦崇拜、食婴、遭强暴、与动物性交，并被迫目睹八岁友人遭谋杀。为她治疗的精神科医师甚至为她进行驱魔，一次长达五小时。

这位可怜受欺的女子最终意识到记忆是虚假的，便对医师提出诉讼。经过五周审讯，案件以240万美元达成庭外和解。纳迪安‧库尔并非唯一因催眠疗法而产生虚假记忆的患者。1992年，美国密苏里州的年轻护士贝思‧卢瑟福接受了所谓「记忆恢复疗法」。这类疗法通指透过催眠、镇静剂及引导想像等暗示手法进行的心理治疗。卢瑟福最初因工作压力求诊，治疗师（当地教堂辅导员）却为她「恢复」了记忆：父亲在她七岁至十四岁期间多次强暴她。

在治疗师持续「引导」下，卢瑟福更「忆起」父亲曾两次令她怀孕并逼她堕胎。这些指控迫使她担任牧师的父亲辞职。然而后来的医学检查显示，二十二岁的卢瑟福仍是处女，从未怀孕。她决定控告治疗师，最终于1996年获100万美元和解。值得注意的是，「记忆恢复疗法」至今仍是争议性疗法，未获任何专业医学认可。

早于1962年，阿尔弗雷德‧奥尔巴克博士在《美国医学会杂志》发表《精神病医生对于催眠使用的态度》一文。该研究对八百多位加州执业精神科医师进行问卷调查，结果显示催眠难以预测、效果短暂且不稳定，更会损害医患关系。

最常见的并发症是精神疾病骤然发作，部分案例甚至出现急性焦虑、恐慌、忧郁或自杀倾向。研究压倒性结论指出：精神不稳者，尤其是已有精神疾病、创伤或药物滥用者，最易受催眠影响。而恰恰是这些心境不安之人，最容易被催眠疗法吸引，冀望借此解决问题。

因催眠而植入虚假记忆、人生尽毁的案例不胜枚举，现有证据已充分揭示这种真实风险。若伤害出于故意滥用催眠，情况更为阴暗——正如布拉瓦茨基所言，称之为黑魔法亦不为过。

这并非否定催眠可能带来益处。只要治疗师深入理解催眠运作原理，并掌握生命能量本质对患者身心之影响，仍可有效施术。然而，何处寻得这般真正具天赋的从业者？更确切地说，多少自称心理治疗师、精神科医师与催眠师的人，同时亦是受过训练的神秘学家？恐怕寥寥无几。

在远古的埃及、印度等地，运用催眠治疗疾病、缓解疼痛者，皆自年幼便接受严谨训练，方获准为患者施术。那种训练不仅透彻理解肉体与生理，更涵盖星光体、小我与高我之认识，并辅以最高道德标准。今日何处还有这等培训？或者说，多少从事催眠者具备这些资格？答案同样是：几近于无。

正如阿尔弗雷德‧奥尔巴克1962年所指出，当今催眠术难以预测、不可靠，且充满严重危险。亦如布拉瓦茨基一百三十年前的警示：

> 「除非催眠者的『暗示』纯为了对方的利益，毫无自私动机，否则思想上的暗示，即属黑魔法，其后果比言语暗示更为险恶。剥夺他人自由意志始终是错误且非法的，除非为了对方或社会福祉；即便为此，亦须极度审慎。在神秘学看来，所有这类混乱尝试——无论有意或无意——皆是黑魔法与巫术。」

必须再次强调，催眠并非全无益处。然而当益处出现，往往更多出于偶然与幸运，而非源于对其设计与背后神秘法则的深刻理解。

有人曾问布拉瓦茨基：「若催眠不是为了治病，只为戒除酗酒、说谎等恶习，是否明智？」她的答复周详，值得全段引述：

「此举近乎慈悲与智慧。戒除恶习虽未必增添其善业——除非改变出于自愿，源自自由意志，且需耗费巨大身心努力，方能积累更多功德——但成功的『暗示』至少能阻止他不断累积过往罪愆。」

「想像力贯穿生活的每个角落。想像力与信心如同画师的双手，在意志的蓝图上勾勒轮廓，在布满荆棘的人生路上刻下深浅不一的印痕。帕拉塞尔苏斯有言：『必须以信心印证想像，因信心奠定意志的根基⋯⋯坚定的意志，乃一切魔法之始⋯⋯人往往无法全然的想像并相信结果，故法术时而飘忽不定；然法术是能绝对确凿的。此即一切疾病之奥秘。有一半或三分之二的病症，源于我们的想像与恐惧。一旦摧垮恐惧，为想像辟出新径，余下的便交给大自然。』这些话蕴藏著真切的实践智慧；可惜当今太多催眠师，显然欠缺这份智慧。」

前文提及一种普遍的误解，以为被催眠者不会违背本心或行恶。此说谬矣。治疗师滥用催眠之例，实在不胜枚举。一位美国心理学家曾公开承认，当催眠术再度「引进」美国时，「其重点几乎全在对受术者的操控。」他亦坦言，催眠师对病人施加「如神般」的权力，正是「吸引许多人投身此道之因」。布拉瓦茨基夫人必会谴责这般态度，视之为黑魔法，我们亦然。

2013年，英国催眠师蒂莫西·波特令一位受术女士相信自己是他的「性奴隶」。波特以协助减重、缓解焦虑为由诱使她接受催眠——这本是许多女性寻求此术的常见理由。然而，在她神识恍惚之际，他对她施以性侵。她报警，波特遭起诉。法庭上，波特承认未经同意发生性行为，最终判囚十八个月。此案最令人心寒之处，或许在于波特的催眠技艺，竟是透过观看YouTube影片自学而成。

如今世人多抱怨「暗网」之害。然「明网」中潜藏的伤人之物，未必少于暗网。就在数日前，我们见一则社交媒体贴文，标题写道：「谁愿与我体验线上催眠？」短短数小时，竟吸引数百留言，许多人跃跃欲试。

该文接著列出一系列所谓「服务」，其中声称：「我可为你催眠，探索前世、重获活力、动力、自信，或吸引爱情、财富、健康等。多么美好！我能为任何积极目的施行催眠。身为专业催眠师，这一切完全免费，只为带给你快乐与帮助。」文末以「爱与光」作结——但那真是它所提供的吗？

神秘科学证实，在记忆的迷宫里，存在幽秘的抽屉、晦暗的角落与隐蔽的缝隙，连精神科学家与心理治疗师至今也未尽明了。这些抽屉一生或许只开启一次，罕有第二次，且仅在特殊条件下，只为极特别的目的而启动。凡愿保有清醒神志的男女，切勿不经深思便刻意打开它们，或容他人代为开启，否则可能轻易释出恐怖柜中之物，如同轻易地打开荣耀宝箱。

我们此刻讨论的催眠术，若想于秘术中重获一席之地，那天似乎仍遥遥无期。我们这些已踏上光明之路、或愿引领他人前行之人，当以极致的谨慎与深切的敬畏看待催眠。此番探讨若能使你如此看待，并稍稍揭示这门古老科学的魔力与神秘，那么我们的努力，便算得到了圆满的回报。
