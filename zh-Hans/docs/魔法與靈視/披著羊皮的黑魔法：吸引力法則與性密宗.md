# 披著羊皮的黑魔法：「吸引力法则」与「性密宗」
![cover](/img/downloaded/f3025b01b09ba79ad3874ec3180adbdc.jpg)

Photo by [Elena Mozhvilo](https://unsplash.com/@miracleday?utm_source=medium&utm_medium=referral) on [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral)

>「无论是白魔法，还是黑魔法，都不是一般迷信所理解的术语。」——布拉瓦茨基

「黑魔法」这词，多数人立刻唤起对于女巫、魔鬼崇拜的联想，或是企图用咒语仪式伤害他人、操控命运。这些确属黑魔法；然而，许多同样黑暗的伎俩，我们未必察觉其本质。

甚至可以说，当今世上成千上万看似良善的人，实则都是不自知的黑魔法师；而许多流行的灵性教导与修行，几乎全沾染了黑魔法的色彩。

为何如此断言？

首先明确定义：**黑魔法，是怀抱自私或物质目的，刻意操弄灵性力量的行为**。白魔法同样运用灵性力量，但其目的纯粹是祝福与造福他人。

这并非作者独创的见解。自古以来，黑魔法一直以此定义为准。每当灵性团体发现追随者试图以灵性或心智力量，来汲取财富、物质或个人成就，便会将他逐出寺庙、寺院或修行社群，并引以为耻。

古人深知，灵性生活的真实进步与灵魂内在的升华，奠基于无私、利他、谦逊，以及全心造福人类的志向，并以最崇高的道德与个人纯洁为准绳。

佛陀、克里希纳、耶稣、赫尔墨斯、商羯罗、宗喀巴、老子、布拉瓦茨基，乃至历代所有伟大的灵性导师与世界改革者，其教诲无不清晰地阐明了这点。

几个世纪以来，「吸引力法则」与「欲望具象化」这类教导与实践，始终在可耻与黑暗中秘密进行。直至人类的自私与物欲执迷达到顶峰，它们才开始获得公众接纳与欢迎。从十九世纪到二十世纪，这些思想以惊人速度传播全球，并受到狂热追捧。

在西方人心中，自私自利与对权力财富的渴求早已根深蒂固；于是多数向往灵性之人，从未怀疑过一事:当他们透过观想、肯定语、「丰盛思维」与「梦想板」招引华宅、美伴或万贯家财时，这些手段究竟妥当与否？

我们不难从中看出谬误与可疑之处，因为上述行径，正符合黑魔法的定义。

渴望生命中的美好事物，本无可厚非；但若动用心智与灵性之力去攫取，便落入黑魔法的范畴。

这并非指责朗达・拜恩与《秘密》一书的其他作者蓄意害人、阴险邪恶。然而依据古老智慧传统的权威见解，他们所倡导的教义与实践，无异于黑魔法，无论他们如何粉饰，将其装扮成无害无辜的模样。

只因一切为己、为满足私欲的动念，哪怕再轻再微，都足以令任何灵性力量的行使由白转黑。布拉瓦茨基夫人曾引一句古老的西藏谚语：「右手之道（白魔法）与左手之道（黑魔法）之间，相隔仅如蛛丝。」

问题的关键，终究在于此人是否已臻至全然无私无求之境；除了助人与服务人类外，别无他念。这才是灵性生活的真义。

一切痛苦起于欲望，一切欲望源于无知。让「欲望」止息与舍弃，方是所有真诚灵性求道者的目标。这讯息，早由佛陀与其前的克里希纳，在其教诲中完整阐明。

还有一处迷障，令许多人不知不觉坠入黑魔法的深渊。那便是「性密宗」的修习。

性密宗如同其他黑魔法形式，被其倡导者巧妙伪装，致使许多天真的西方人误以为它属于印度教等古老东方宗教。因而，当这些人发现，绝大多数印度教徒对其厌恶甚至反感时，往往大为惊讶——性密宗在任何层面，皆不能代表这崇高宗教。

事实上，那些自称对性密宗感兴趣的新纪元信徒，一旦知悉其实践步骤，多半惊骇退却。印度教整体对性密宗亦极其冷淡，从未将其纳入六派哲学体系之中。

布拉瓦茨基承袭历代灵性大师的教导，指出：「若将肉体用于灵性目的，或以此作为工具达成形上目标，全然属于黑魔法。白魔法师——右途上的悟者——深知魔法无需假借肉体，因肉体不过是物质皮囊。」

真正的东方神秘主义者则认为，生殖系统所涵容的一切，与其宇宙原型有著直接灵性连结；故在物质层面上是黑魔法的有效对象，且被视为不洁。

同理，两人寻求性结合本无不当，但若试图借由性能量或体液以达致「灵性」目的，便沦为纯粹的黑魔法。

布拉瓦茨基断言，此类实践「将招致最可怕的业果」；任何形式的「性魔法」，皆自动归于黑魔法。

读至文末，或有人以为我们只是「灵性的破坏者」；实非如此。本文出于对人类真切而深切的关怀写就。欲望、野心、自私与感官之念，早已深植人心，我们并不奢望多数人听从、甚或稍加思量此文所言。但哪怕仅有一人，因读罢此文而从无知的黑魔法中脱身，我们便觉欣慰，一切努力也算值得。
