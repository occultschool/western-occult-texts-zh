#  星光界场景 — 迷失的孩子
## 任何爱，只要能助他们脱离那流浪的漩涡，都不算无私

![cover](/img/downloaded/25d253c7321d1f1ac03d0cc53d3ec4a4.jpg)

Photo by [Enrique Ortega Miranda](https://unsplash.com/@enriqueo?utm_source=medium&utm_medium=referral) on [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral)

这段摘自《神秘之谜》的短文，出自一位真正灵视者之眼，描绘许多人死后那骇人的真相。必须补充，米绍博士的描述毫无夸大；我们在星光界见过极其相似的景象，实在令人心碎……

「你若见过那漩涡中，无数低等进化、焦虑困惑的人类，随一道巨大洪流在星光界边缘漂流挣扎，你终将明白：此处付出的一切辛劳都谈不上艰难，献出的所有服务都称不上伟大，倾注的任何爱都算不上无私——只要能助他们挣脱流浪的涡旋。他们不知发生了何事，不知身处何方，一双双悲伤的眼望著你，无声恳求援助。只要见过那几百万张惊恐的脸一次，便永生难忘。

「他们有的飘荡在空中，显现生前那脆弱虚幻的形貌，一边诅咒，一边祈祷；一时爱，一时恨。全是鬼魂……神所遗落的孩子……非得等到再次被肉体束缚奴役，他们才仿佛得救；须忍受更深一层的禁锢，直到终于有资格获指引前路。

「道路固然狭窄，但依神意，人人皆能踏足。更甚者，已有大批先行者在前路等候。他们殷切盼你到来，对你这疲惫的旅人张臂相迎；欣慰你于阴影中的任务已然完结，更辉煌的工作正要开始。

「他们就在那里等著你：那些大师，天界的白光同胞，等候那些仍处尘世阴暗中的同胞。如今他们已成智慧、爱与慈悲的火焰，准备将自身光芒汇入更浩瀚的神圣之火；为尚未到来的人照亮前路。」
