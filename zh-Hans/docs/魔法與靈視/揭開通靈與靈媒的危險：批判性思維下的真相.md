# 揭开通灵与灵媒的危险：批判性思维下的真相
## 解析灵媒传讯之隐患，透视其对精神健康的潜在威胁

![cover](/img/downloaded/ae9ea76499d768a938ee9529050ce37d.png)

本文探讨通灵与灵媒的危险。

若以批判眼光审视灵媒传递的讯息，便知他们说出的内容并非新知识。近两百年来，无论冠以通灵传讯或通灵主义之名，皆未贡献新的科学、哲学或灵性知识，亦未引领人类走向更深觉醒。

人人皆可自行验证：观其言行，可识其人。通灵传讯的结果，一如通灵主义，往往带来混乱、争论与无知，而非清明、和谐与智慧。

某些通灵讯息或许夹杂零星智语，实则是对先贤教诲的曲解。现代以伪科学术语重新包装，许多人懒于深究便囫囵吞下，只因他们从不追溯这些教义的原始源头。

可悲的是，只要讯息披著庄严与权威的外衣，多数求道者就不质疑所闻所见。正因如此，那些教人如何通灵的指南，在新纪元大受欢迎；而载满通灵教诲的书籍，同样广为流传。

通灵主义未曾带来高尚的灵性教导、深刻哲思、可靠启示或准确事实，新纪元的通灵传讯亦然。它只是欺骗了数以百万计原本理性之人，让少数人获利，同时将更多人推入抑郁、精神失常与自杀的深渊——重蹈十九世纪通灵主义的覆辙。

值得注意的是，《一的法则》三位调查者之一的唐·埃尔金斯，于一九八四年饮弹自尽。离世前数月，他精神、情感与身体急剧恶化，无法成眠，难以进食。埃尔金斯只是众多不幸者之一，他们因涉足残酷的通灵游戏，最终亲手结束生命。

超心理学家汉斯·本德在《灵媒精神病》一书中，提及四例受灵体指使自杀的个案。其中一例，一名年轻女子透过自动书写传讯后，听到一个声音命令她结束生命。友人阻止她跳楼时，她说：「这是一股我必须顺从的力量。」

基督教心灵感应学会领袖约翰·d·皮尔斯-希金斯，提及两位女孩进行自动书写的例子：「起初一切顺利。一女孩自认与亡父连结，父亲似乎知晓她一切，并提供确凿证据。随后，父亲一位朋友出现，声称多年来一直是她的『守护天使』。灵体提供令人信服的证据，令她深受感动，接著告诉她：『亲爱的，生命于你已无太大意义，彼岸世界美好无比，你最好加入我们。』女孩闻言便冲向一辆公车。所幸她幸存。一年后出院，虽已康复，仍受接触过的灵体困扰。」

另一案例涉及一位知名美国通灵者，因她仍在世，姑隐其名，简称X。她开始接收愈发令人不安的讯息，包含关于人类未来的惊悚预言。X大为震惊，当未知灵体再次接触时，她选择忽略。当夜她便做了最可怕的噩梦。次日，该灵体的命令转为威胁，X于是告诉对方，她已改变主意，不愿再接触灵界。然而，灵体持续以心灵感应攻击，导致X精神严重失衡，试图从公寓阳台跳下，最终被送入精神病院。

这类悲剧并非新闻。上个世纪，无数人因接触通灵主义，本可拥有充实幸福的人生，却最终在精神病院潦草结束短暂生命；更多人陷入抑郁，沉迷酒精与药物。琼·格兰特在其自传体小说《身为卡罗拉》中，描写十六世纪遇见女巫索菲亚的情节，以令人战栗的准确笔触，刻画了通灵的诱惑与危险——这类警示，历来层出不穷。

「索菲亚猛地坐直，十指插进草丛；双目紧闭，身躯一阵抽搐。她深深吸了几口气，像渴极了似的。随后开口说话，语速快得我听不明白。我惊得呆住，只想逃开那从她唇间迸出的声音。接著，声音变了，变成我熟悉的声调。话语缓缓流出，仿佛一字一句都沁著心血：『卡罗拉，我的卡洛丽西玛，听我说！我已许久没同你说话了……』

「这真是母亲的声音吗？我为何要怀疑？只有她会唤我卡洛丽西玛。我知人可能被恶魔附身，但为何不信这也可能是天使的附体？为何我只感到恐惧，而非期盼？我多希望索菲亚只是在戏弄我，刻意改变了嗓音。我无法解释，却无比确信：那对我说话的，绝非索菲亚。我理应遵从这藉她之口传来的声音，不是么？

「之后几日，我忙得不可开交。黄昏穿过林子回家时，忽闻一声呼唤：『卡罗拉，卡洛丽西玛，你在哪儿？』那一瞬，我忘了自己已长大，也忘了母亲离世将近七年。我循声沿小径跑去，心中既怕，却又被莫名牵引。只见索菲亚迎面走来，见到我，她亦露讶色，说路上并未遇见旁人。

「我们在她的小屋里，我察觉索菲亚的身躯正被某个相似的灵体使用。方才我们还说著话，此刻她却静默下来，我以为她睡了。忽然，我听出她呼吸变了。她的手僵直，眼皮微张下的眼珠缓缓转动。接著，母亲的嗓音从她喉中响起：『卡罗拉，为何不回应？你忘了我不成？我塑造了你的身体，卡罗拉。如今我孤魂无依，能否借我一用？你曾经信我，卡罗拉。或许此刻你不信真是我？索菲亚可知道那只十五抽屉小匣里收著什么珠宝？还是你父亲送你、让你转交米梅塔的那条红项圈？让我用你的身躯吧，松开你对我紧闭的意志。没有你，我是何等孤单……』

「我失声喊道：『妈妈！妈妈！』但屋内空寂，只有索菲亚在炉火边轻鼾。我脑中没有鲜明的记忆指引，却有一股古老的直觉在心底回响：天使之音不会从人口中吐出，而魔鬼能操控地上仆役的言语。这警讯忽然清晰浮现，仿佛我很久以前便听过这永志不忘的声音。

「但若我的直觉错了？若母亲真从另一境地向我说话，我却充耳不闻？对她而言，藉血肉之躯向我倾诉，是否充满痛苦与折磨？这是她的声音，不是索菲亚的。是她在说话，不是别人。为何我心底深处对她升起抗拒，尽管我的心思与理智都相信她回来了？

「我渐渐觉得，我想让母亲的灵体远离我，并非出于明智，而是软弱。于是，我躺在那片空地上，祈求母亲的灵进入我的身体。等到我再度醒来，如果发现自己身在别处，却不记得如何抵达——那便是她曾借用过的痕迹。我想像一片宁静的黑暗；黑暗包围了我；愈来愈深；温暖、柔软、密不透光。

「然后，恐惧袭来了。比毒蛇更骇人，比烈火更噬魂。我陷在无尽阒暗里，四周是为死者纺织裹尸布的蜘蛛，以及回应死囚脚步声的蟾蜍。蝙蝠翅膀掠过身旁，教我作呕，我动弹不得。我想尖叫，嘴里却塞满腐肉鸟的羽毛。我使劲想掀开那沉重的眼皮……

「我终于挣脱黑暗。眼前是那片曾弃我的天空。我在恐惧中持续很久，直到完全清醒。我奔向河边，褪去衣衫。我逆流而上，试图洗净身体，冲刷那邪恶沾染的污秽。我懂了痲疯病人的感受——那种不洁，永不洁净的感觉。我不愿再见索菲亚。我避开我们曾共处的每个角落，也绕过通往她家的路，仿佛那是一条痲疯患者徘徊的小径。」

这段描述出自琼·格兰特令人悚然的笔记。并非所有灵媒皆会被邪恶实体占据，或者通灵必招此祸。然而危险始终存在，正如琼·格兰特在同书中告诫。她援引古埃及所受秘仪训练，提及：

「众神律法载明：一个灵只能有一个肉体。穿越他人躯壳者，必遭吾等定罪；背叛主人居所者，亦受惩罚。」

「主人居所」指的是身体，身体可能因精神疾病或其他残缺而「遭受惩罚」，而「主人」即高我，亦将承苦，因其载具之缺陷阻碍了此生使命。

或许你会问：为何许多自认「灵性」之人，声称追寻真理，却不肯学习古老智慧？为何众人热切聆听自称「拉」、天使指引或外星超人的虚妄讯息，却对阅读世界圣典毫无兴致——尽管此处才是寻觅生命真理的高尚所在？同样，为何那些原本聪慧、明理、饱学之士，却易受欺瞒、天真盲目，竟毫不质疑地接纳灵媒宣扬的无稽之谈，不愿费心探查其根源？

所有问题的答案，可归结为一字：思考。我们仿佛身陷一场全球性思维惰性的瘟疫，这种懈怠已令人类抗拒思考。若此态持续，全体人类，尤其是那些追寻真理者，将继续错把通灵感应当作灵性显现，视幻梦为事实。

通灵者或许乐于夸耀新纪元的荣耀奇迹，但清醒的现实是，他们日日滋生迷信、无知与妄想，自身反成人类迈向新纪元的最大阻碍。

关于通灵传讯的欺诈与危险，至此已论述清楚。正如本站在其他文章所提，确有可能从更高维度的星光界接收灵性教导，其途径不止一种。例如，在[《冥想的简单步骤，获得内在的知识和启迪》](/zh-Hans/docs/meditation-steps-inner-knowledge-enlightenment)结尾处，我们已作过阐释。

布拉瓦茨基在《揭开伊希斯的面纱》中，曾记述一则接收崇高灵性教导的实例：

「印度某大城住著一位高贵的婆罗门，其爵位与贫困的生活全然不相称。因贫穷所限，又须恪守本国礼仪，他只得将非凡启示，深深藏于其灵性丰饶的内心。这些启示来自不可见的作者，并透过他年仅约十二岁的小姪女作为媒介而传达。在这名小女孩的面前，一叠叠空白纸张会迅速被书写填满，却看不见任何手的动作，甚至完全不使用笔、铅笔或墨水等一般书写工具。

「只需将白纸置于三脚桌上，避开直射光线，但仍能让专注的观察者隐约看见。孩子坐在地板，头倚桌边，手臂环抱桌脚。如此姿势，她常会沉睡一小时，而此时三脚桌上的纸张便会浮现精致的古梵文。不及三年，已成就四卷以上著作。」

此例所示，乃心智间全然谐振的感通，而非通灵时心智被另一心智占据。正是孩子天真的纯粹，让她在睡梦中展现高我，暂时将自身意识与书写教诲者的意识融为一体。
