# 是魔法还是奇迹？ — 魔法十大重点整理，人人皆是魔法师
## 魔法运作的定律，过去曾为人知，只是今日科学尚未识得

![cover](/img/downloaded/18e6a1baff3308e54ba88d5b9587324c.jpg)

Photo by [Aziz Acharki](https://unsplash.com/@acharki95?utm_source=medium&utm_medium=referral) on [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral)

布拉瓦茨基《揭开伊西斯的面纱》十点精要

一、世上无所谓「奇迹」。

一切现象与结果，皆受永恒不变、持续运行的定律支配。卡彭特博士学识渊博却见识有限，认为「奇迹」违反「公认自然定律」，不该存在。然而，催生所谓奇迹的，正是那些公认定律之外的力量。许多人如同卡彭特博士，忽略了一个事实：奇迹运行的定律，过去曾为人知，只是今日科学尚未识得。

二、大自然实为三位一体。

存在一个**可见、客观的大自然**；亦存在一个**不可见、内在而活跃的大自然**。后者与前者全然对应，既是其蓝本，亦是其生命原则。两者之上，则是**「灵」**——一切力量之源，永恒不灭。前二者属较低层面，流变不息；**「灵」**属更高层面，永不更改。

三、人亦是三位一体。

人拥有**客观的肉体**与**活跃的星光体**，即「真实之人」；此二者由第三位「主宰者」所看顾照亮，亦即**不朽之灵**。当「真实之人」成功与灵融合，便成为不朽存在。

四、魔法作为一门科学，是对这些原理的认识。

灵魂寓于肉体时，能透过魔法获取灵之全知全能，并学会驾驭自然力量。魔法作为一门秘术，是对此知识的实践与运用。

五、滥用奥秘知识，即为巫术；善用其力，方是真魔法，亦即智慧。

六、灵媒与魔法师恰恰相反。

灵媒的通灵能力，是「被动」接收外来影响的工具；魔法师则主动掌控自身，掌控一切较低层次的法术。

七、一切过去、现在、未来之事，皆记录于「星光界」之中，或曰宇宙不可见的石板上。开悟的魔法师借由灵性视野，能知晓一切曾经发生、未来能知之事。

八、不同族群灵性天赋各异，犹如肤色、体态或其他外观。有些族裔多人天生具备灵视力，另一些则具通灵禀赋。其中不乏沉迷巫术者，将秘法代代相传，由此衍生形形色色的灵异现象。

九、施展魔法的一个关键阶段，是自愿且有意识地将内在之人（星光体）抽离外在肉身。

某些灵媒案例中，亦出现这种抽离，却是无意识且非自愿的。灵媒此时身体往往陷入不同程度的「僵直」；而魔法师的星光体离体时，旁人却难以察觉。因其肉身感官依然警醒，外表仅似心不在焉——有人称此状态为「深思出神」。

对于遨游的星光体，时间与空间皆非阻碍。深谙神秘科学的泰米尔人，能令自身（即肉体）看似消失，或化为任意形貌。亦能让星光体显形，呈现千变万化之相。然而，上述现象，实际上是透过催眠在场观者的感官而达成。此幻觉如此真实，以致目击者以为所见为真。但这只是他们脑海中的景象，被催眠者施以无法抗拒的意志，深深烙在他们的意识里。

星光体能无远弗届、穿墙越壁，相隔千里，并使形影清晰可见；但肉身的移动只能凭借寻常交通工具。肉体或许能在磁力作用下，悬浮片刻，但若要从一地迁徙至另一地，却终究得依循最平凡的方式。因此，我们不相信任何关于灵媒肉身御空飞行的传闻——那类「奇迹」，我们一概摒弃。无机之物尚可在某种情境下分解自身、穿透坚壁而后重组；至于鲜活的生灵躯体，则绝无此可能。

史威登堡学派深信，神秘科学亦昭示：**灵魂离弃肉身之事时常发生，我们日常生活中，常常遇见这般「活尸」。** 其成因纷杂：剧烈惊惧、深重哀恸、彻底绝望、急症骤发，或过度耽溺感官，皆可能引致此态。那具失神的躯壳，有时会被精于此道的巫者以其星光体侵入并占据；亦可能受**亡灵**所据，即失去肉体的人类亡魂；或在极稀有的情形下，为**精灵**所附。当然，白魔法师亦具同等能力，但除非为了某项非凡崇高的使命，他绝不愿玷污自身，入主一具不洁的皮囊。至于疯癫之徒，其星光体多半瘫痪、神识紊乱，易受任何过路灵体左右；又或者，其灵魂已永远消逝，身躯被灵体所占，濒临溃散，状若吸血鬼，拼命依附肉体，希望通过这种权宜之计，于短暂光阴里榨取更多感官欢愉。

十、 魔法的基石，在于深刻体认磁与电之属性、关联及效力。

尤须熟谙二者于动物界与人类身上的作用。而如矿物等，亦各怀玄秘特性，石膏即是一例。一切魔法师皆须通晓此等性质，而当今所谓精确的「科学」，对此却一无所知。植物亦具类似的玄妙特性，且异常优美。梦境与魔法药草之秘，在欧洲科学中早已遭忽视，不为人所解。鸦片、大麻等寥寥例外，纵使存在，它们对人类精神之影响，也只被视作暂时性的神智紊乱。塞萨利与伊庇鲁斯的妇女——即萨巴齐乌斯仪式的女祭司——并未因圣地倾颓而遗失其秘传。这些知识依然存续，凡识得苏摩植物本质者，亦通晓其他植物的特性。

## 总结

总而言之：魔法乃灵性智慧，大自然是魔法师的物质盟友、门徒与仆役。同一生命原则贯穿万有，而臻于完善的人类意志，得以驾驭此力。魔法师能在超常层面上，激发植物与动物内在的运行之能。此举非但无碍自然，反而助其加速，赋予更蓬勃的生机。

魔法师能操纵感官，改变他人的肉身与星光体状态；亦可随心所欲，统辖并差遣诸元素精灵。然而，他们无法控制任何人类的不朽之灵，无论活人或死人；盖因人之灵皆为神本质的火花，不受任何外力主宰。
