# 冥想，或称逃脱之道
## 一个充满奇迹与启示的故事，探讨冥想与成长的真正意义

![cover](/img/downloaded/4e7eab352faf6b31c07090ad66b73bba.png)

这故事看似荒唐，却暗藏玄机，埋著许多神秘学的真谛。

一切始于求道者常问的一句：「冥想，何为上策？」我们总答：「全心去做。」为说得明白，便借用了米绍博士《神秘之谜》里的一则轶事。故事的确凿，我们敢担保——所有细节皆有档案为证，收在个人卷宗里。

公元一二〇〇年四月一日，科隆著名大教堂的阶梯上，躺著一名新生男婴。这教堂历史悠远，超过六千年，尖塔、拱顶、钟楼巍峨入云，为纪念圣徒、殉道者与其他不甚明智的存在而建，是欧洲最慑人的建筑之一。

婴儿的出现，不只惊动科隆，更撼动远近。成千上万的人乘私人飞机、滑翔翼、游览车、蒸汽火车，甚至蹬著轮鞋赶来，只为亲眼一睹这桩奇迹，喂饱那无谓的好奇。电视与广播每小时追报，推文如潮水涌现。这古老小镇从未如此喧腾，人人争论、猜测婴儿的来历、身世，乃至父母——倘若真有父母。

据说前一夜，有人目睹一艘巨大的不明飞行物掠过神圣的教堂顶，轰声骇人，火花四溅。或许这孩子正是其中一星火花，悄然化为活生生、啼哭不休的男孩？还是与惊扰不列颠的尼斯湖水怪有关？或是几日前莱茵河里游过的海豚？无人知晓。好奇的群众围住市政厅，警察护送孩子前往，入场券已炒至天价。好莱坞最犀利的狗仔直飞而来，举起相机与手机快拍，捕捉孩子、人群、教堂石阶与市政厅的每一瞬。

市政当局由布尔戈梅斯特主事，他郑重赶来调查。在市议员、巫医、通灵者、寻常医生，以及其他自认聪明的人协助下，他们检视我们的小英雄，参详星象、羊肝、鹈鹕飞姿、企鹅、塔罗与种种征兆，最终达成一致：这孩子志向超凡，必将远行，攀上神圣高处，终成受人尊崇的圣人。

如您所知，德国人做事向来彻底，而这孩子似乎比谁都彻底——姑且当他是德国人吧，而非怀抱大任的外星来客。

占卜者的宣示留下了无与伦比的印象，敬畏与恐惧交织；朝圣者队伍络绎不绝，携来珍贵礼物，装饰华丽的骆驼低声嘟哝（难道牠们知道——又有谁知道——阿拉的第一百种尊名？），大象昂首吼叫，日日前来向未来圣人致敬，在他的摇篮前祈祷、呼吼或哀求。东方智者亦至，眸如煤晶闪烁，鼻似巨钩，须发苔绿，他们贩售孩子触碰过的护身符，或他衣衫的残片，所获钱财足以为千所孤儿院的孩童添置百年衣裳。每道符必带来鸿运与财富，每片布若藏入小囊、悬于颈链，将保你永健，赋你天使般的智慧。最奇的是，这些被虫蛀的救世主，竟未替自己留下半件圣物。

喧嚣渐息，古城沉回十二世纪的静谧里，孩子也如众人期望，开始全面成长。当所有陌生的骚动远去，老居民与孩童都舒了口气；那时代的人，本不惯我们现代这般喧嚷。往事，就随风吧。

不难想像，这孩子早年受尽细心呵护，监护人竭尽所能，为他铺垫光辉前程。他们首先留意到男孩一项特质：一种极致且奇特的懒。他懒洋洋起身，懒洋洋睡去，闲时懒得动，忙时懒得停；年岁渐长，这份非凡的懒——姑且称之为懒——愈发精纯。

对勤勉的科隆人而言，这般景况简直匪夷所思、闻所未闻。

局面日益不堪，终于他在格外怠惰的某一日，断然决定：再不工作了。众人皆惊！年满十八后，他开始写信给散居世界各地的亲戚——那些赫赫有名的探险家与旅人；他极力游说这些善心人资助他，好让他栽培那份稀有的懒惰天赋。他想，如此既能得些钱财，偶尔度个假、享享清福；毕竟只歇不玩，聪明孩子也会变傻——这句谚语当时人尽皆知。

然而怪的是，他并未如愿获得自认应得的资助。即便在十二世纪，许多人也同今日一般，不愿轻易掏钱。

眼看命运将临之际，一股异样的焦虑笼罩了他，旁人偶尔从他眼中瞥见梦幻般的远眺神情。

二十岁生日当天，他悄悄攀上巨大邮轮「太阳系之星」的舷梯。那船正准备展开每周从科隆至坎昆的航程。他成功藏身蒸汽机旁，不久便沉沉睡去。漫长两年多的航程中，一切如常。直至墨西哥湾，他们遇上一场骇人的飓风——这在向来以宁静温和著称的地区实属空前。

宁静夜色里，他不知不觉从船边滑落，伴著汽笛声与兴奋乘客船员的狂欢、道别与祝福。无人察觉他的离去。他展开了一段奇异旅程，历时逾四分之三个世纪。数月后，他筋疲力竭抵达秘鲁的幸福海岸。长途跋涉未伤他分毫，他穿越汹涌浪涛踏上陆地，精神竟已焕然一新。只是那一周异常的辛劳令他倦腻不堪，旅程就此划下句点。

他终究抵达了幸福与命运交会的目标，所有关于他光明未来的预言都将实现——彻底地实现！然而在达成这渴求的终点前，仍有试炼等候。才踏上陆地，他便遇上一大群生活在赤贫中的秘鲁印第安野蛮人，他们从这片完全平坦荒凉的沼泽地现身，坚定而有礼地要将他交给他们的名厨。他们最擅长以最难入口的肉料烹出佳肴，这回正打算将这份才智施展于这位英雄身上，为他们的丰盛胜利添上荣光。

众人都会同意，这确实是至高的荣耀。这位英雄在各方面皆令人向往。然而唉，人类的反常行为总难理喻。史上头一遭，这位英雄对野蛮人的善良意图展现全无兴趣，并以粗鲁自私的姿态向好客的主人高声抗议：他远道而来，不是为了当烧烤串、主菜，甚至牛排与肾脏布丁；因为他实乃一位伟大圣人，将为秘鲁带来光荣的新教义与祝福。身为圣人，他断然拒绝这些贪婪的计划。「若我说的不是事实，尽管打我屁股、掏我内脏！」（瞧，他有时很粗俗，但这些秽语从何学来却是个谜。）

可怜的印第安人对这般粗暴与忘恩负义深感困惑沮丧，伤心地走入战争议会，在和平烟斗与一瓶白兰地之间，商讨这前所未闻的行径。

月轮一次次横越夜空，秘鲁人吸食的烟斗愈多，空酒瓶愈积愈高，他们的困惑也愈发深重。我们的英雄在沙滩上徘徊，炎阳下气氛紧绷，他急不可待要开始那宏伟使命，为俘获他的人带来无数祝福与好处；同时也寻思各种借口，想躲开那口烹煮大锅，避开那些布丁、馅饼与甜点。

终于，当最年轻的战士无法再多抽一斗烟或斟一杯酒（他终究太过年幼，未有长者的历练），他站起身，激昂低语：「沼泽的弟兄们；我们长久争论，从各角度、三角与方块剖析问题，但无论重启多少次言语之旅，总似绕回原点。我建议将此事呈报大祭司与皇帝陛下，无论他们如何裁决，我们皆无异议。赞同者，请起身！」勇猛的战士们踉跄而起，关节咯吱作响，边起身边打嗝低喃，盛赞这绝妙提议堪比古圣先贤的智慧。

三声清亮的呼喊响起，他们踏上旅程，在英雄陪伴下展开了漫长无尽的探险。最终，他们抵达了隐于秘鲁山脉、峡谷与峭壁之间的黄金首都。

长途跋涉令人清醒，他们很快来到这座华丽的宝石之城——世上最富裕繁华的都城，日夜宴饮庆典不断——并将我们的英雄引见给微笑亲切的执法者、大祭司与仁慈的国王。

陛下听罢来龙去脉，忽有奇异灵感；他问这位英雄打算如何教导秘鲁的善良子民——他们早已精通掠夺之道、应对埋伏之法、最精细的折磨技术，以及最有价值且持久的牺牲朋友与敌人之术，还能学到什么呢？

难道他知道一种连他们都未听过的新型漫长苦刑？若是，那是什么？不妨在他身上试试，聊表善意，再交给厨子处置。自然，他明白在秘鲁，无人能指望比这更高的礼遇了。

国王与众人肃然静默，我们的英雄缓缓起身，以那骇人而凛然的圣洁目光直视国王，朗声宣告他的来意：他身负使命，要教导他们「逃脱之道」——方法便是与山巅的圣者一同冥想。

在山巅冥想？这对秘鲁人简直闻所未闻。若他真要离开（虽然看来不大可能），或者真要上山冥想，该冥想什么呢？这位英雄从未细想，一时答不上来，只得搬出「空」来搪塞。他本极懒，转念又想：倘若真得上山冥想（只要他们准许——横竖比煎成肉饼强），那么冥想「空」大概最省气力，至少能暂且摆脱那些烦心念头，像是沦为一盘猪肉炖豆子的惨状！

饥肠辘辘的群众强烈反对这计划；他们天生的机警，犹如油膏上聚拢的蝇群，一阵恐吓的低语掠过会场，像风吹过玉米田。然而国王以一记皇家手势——俗称「嗤之以鼻」——安抚了不安与猜疑；他与祭司，以及一群位高权重的智者展开深谈。此外还有钻研弗洛伊德心理学、哲学与形上学的秘鲁教授，炼金术士、魔法师、预言家，以及手持棍棒、石头、茶叶等法器的占卜者，皆为这新颖而全然现代的实验著迷，纷纷劝国王抱持「草草看过」的态度。于是，救世军的乐队、掷弹兵卫队，连同许多木琴、iPod等精致乐器，隆重护送他前往山间沼泽，将他悬吊在最高峰的尖顶，任他静心参究「空」。他宏伟的命运就此展开；在这世间，他终将显露非凡才华。

秘鲁印第安人在山脚跳著神圣的仪式舞，一面紧盯著他，防他悄悄溜走。他就坐在那纤细的尖顶上，深深专注，思索「空究竟是什么」。这专注持续了三十年，未曾离开一步。他也无法离开——焦急的村民早以印第安人特有的方式环山守候，连一只跳蚤也难逃他们的眼睛与议论！须知这群人是饥饿之族，绝不会为了一次冥想，放弃一顿圣人做成的丰盛晚餐。

三十年届满，印第安人再度攀上山巅，接他下来，询问他在漫长静观中领悟了什么。但当他们抵达尖顶，却发现三十年的冥想已令他消瘦、枯槁，终至消逝无踪，唯余一撮灰尘，留在他久坐的尖端。

他们虔敬地取下灰尘，安置在主庙金色神龛的水晶器皿中，依然尊崇他的回忆与遗灰。他被尊称为圣阿亚萨莫纳，意即「长久沉思空性之人」。最终，他蜕变为「空性」，彻底了悟其意旨。

这便是冥想能达到的境界，只要你全心投入；而逃脱之道，亦将为你敞开。

## 隐藏的真理

读罢这篇关于「冥想，或逃脱之道」的故事，或许你会怀疑作者是否真的冥想太久，以致脑髓销溶，终成尘埃。

然而，容我们坦言：这故事其实以最笨拙的方式，隐藏了许多真正的神秘学真理。正如我们在多篇文章中指出的，古代的哲学家、教师、启蒙者、炼金术士等，总是以类似手法掩盖他们的知识或发现；有时假借严肃议论，有时运用奇异词汇、符号或密码，但总带著一层神秘或隐晦的色彩。

依循我们的惯例，现在揭露这个故事中的若干神秘真相，以及那些刻意布下的「障眼法」、矛盾与混淆。这或许能给你提示，教你如何阅读并理解这类神秘著作。但我们必须强调：最好还是凭自己的努力探寻意义，因为这样你学得更多，更会珍惜所得的知识。

故事里，我们首先看见的第一个真相是无知。无知者认为，那些行事古怪却不触怒他人者，必是高人；而智者则看穿表象，得见真实。在此故事中，那孩子其实无足轻重，他唯一的名声，源于出生时的异状与那股奇特的懒散。

可以说，无知是诸「罪」中最重的一项，因为它阻碍人去探究最微小的真理，反而依赖他人的意见与臆测；正如我们所见，关于这孩子成长的种种荒诞幻想，不仅被无知者信以为真，更遭人添油加醋；他们丧失了独立思考之力。

对这孩子极度懒惰的描述，掩盖了第二个真相：「惰性」是所有神秘法则中最伟大的一条。它的反面是运动，这两道法则或力量之间，隐藏了一切物质情境与现象的运作法则。关于这些法则，你可在米绍博士的[《金黄星》](/books/golden-star-dr-michaud)中找到更多资讯。作者将此事实置于众目睽睽之下，正印证了：神秘主义与神秘学作家的文字，在明晰之处往往更欺人，在隐晦处往往更真实。我们已在文章中多次提及此点，但它至关重要，值得一再重申。

故事里的第三个真相是：一件事若「看似」愈神奇、愈不可思议，人们便愈热切相信，而它往往愈不真实。好比故事里围绕孩子发展出的「神圣护身符贩售」——今日基督教会或许不再贩售圣徒遗骸，以维持神职人员的贫俭生活，但现代异教徒、女巫与新纪元追随者之间，护身符交易依然活络。奇怪的是，这些易受哄骗的人，从不问问：为何兜售这些「护身符」的贩子，自己鲜少具备他们商品号称能赋予的特质？

这便引出第四个真相：多数宗教遗物或「魔法」护身符唯一的力量，源于我们对它的信心。之所以说「多数」，是因任何物件都可能被赋予或善或恶的神秘力量，无论佩戴者信或不信，它们确实可能生效。然而，如今懂得如何做到的人寥寥无几，可以说是没有。

第五个真相，是「吸引」与「排斥」的普遍法则。故事主角想从亲戚那里吸引一些金钱，却因自身的贫苦境遇而遭排斥，所求尽皆落空。成功或失败、健康或疾病、贫穷或富裕，皆系于这道神秘法则的运行。《圣经》里也有这话，耶稣说：「因为有的，还要给他；没有的，连他所有的也要夺去。」（马可福音4:25）倒想看看哪位神学家能解释这则看似极不公义的寓言。

有些政客热衷鼓吹「劫富济贫」，这不仅违背「蠢人留不住财」的古老箴言，也掩盖了众所周知的现象：不少出身寒微的彩票得主，在挥霍奖金、纵情奢靡之后，最终落得悲惨收场。事实是，富人仿佛处于吸引财富的状态，如同磁石吸铁；而穷人因其一无所有，反而排斥财富，甚至连手中仅有的也留不住。

这真相或许不讨喜，却适用于万事万物，无论物质或灵性层面：健康、力量、智慧、真理或知识。物以类聚，同类相吸，异类相斥——这与「异性相吸」的法则看似矛盾。实际上，两者皆成立，只是运作方式不同，端看所适用的原理。由此带出第六个真相：「矛盾」不过是哲学家用来遮蔽意义的障眼法。

故事里充斥看似矛盾的情节：主角身处的时代，竟有许多当时尚未存在的现代发明与技术；又如墨西哥湾从未受飓风侵袭，或是一个「荒芜」之国竟人烟稠密！这些矛盾迫使我们超越表象，以更高的理性去观照、调和。有些事物确实能同时虚空又充实，好比悲观者与乐观者眼中的半杯水，或是一个愚昧的头脑。

第七个，也是最后一个真相，将我们带回起点，圆满了整个循环。这真相藏在故事的标题里：「冥想，或逃脱之道」。标题本身隐藏了几件事实。其一，我们无法如故事主角那般冥想「空无」。科学告诉我们，自然「厌恶真空」。你或许要说，世上多的是脑袋空空之人，但我们认为，他们的脑中其实塞满了某种东西——至于是什么，就留给你去想。

其二，透过正确的冥想修习，我们能超越物质世界的幻象，深入灵性之境，亲见其中奥秘。冥想也能唤醒我们对生命伟大真理的觉知，这些真理难以言传，只能在内心静默中领悟。一旦我们掌握并透彻理解这些真理，便不再需要于尘世轮回中投生为人。

或许你会说，这些真理如此浅显，连孩子都懂（无论是不是在科隆大教堂的阶梯上发现的！）。这话也许有理，但若真如此，为何你从未实践？若你早已实践，便无需我们协助，也能读懂玄学与秘义之作。

然而，倘若你从这个故事与我们的讨论中学到新知，何不试著自己去发掘更多「浅显」的真理？不只这个故事，我们发表的许多文章里，都以隐微的方式蕴藏著大量神秘学真理。如果你愿意这么做，并将你的发现分享给我们，或许我们也能从中学到新事物。毕竟，我们还不至于狂妄到自以为已无新知可学。
