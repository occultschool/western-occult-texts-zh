# 精灵女王：米绍博士的神秘童话之旅

## 描绘神秘王国的华丽建筑与寻找伴侣的秘密使命，揭露精灵族情感与冒险之旅

![cover](/img/downloaded/elf-queen.png)

在一片神秘森林里，深藏一座精灵女王的城堡，矗立在阳光明媚的平原上。 此建筑美丽绝伦，如一块巨大宝石，优雅精致的花朵环绕周围：有芳香的玫瑰与茉莉花，有著金色的茎和闪闪发光的银叶；另外还有巨大蕨类植物，相互散发凉爽清新的空气。 而淡紫色的木莲花、森林中的白色苏木、和紫蓝色的多花野豌豆，都混合著快乐的颜色；成千上万的花朵大大小小，各种颜色和形状，在城堡周围的花坛和边界繁茂生长，由小地精们所种植，是为了女主人精灵女王服务。他们用尽全心来崇拜她，若能给她快乐和鲜花般的惊喜，任何事情都不嫌麻烦。此城堡及花园之美无法想像或描述； 只有内心和思想尚未失去仙子性质之人，才能在内心看到此荣耀。 

 城堡有十二座大门，各有两座高塔，塔顶上有块巨大蛋白石，能把各种颜色的火焰射向天空和四周。 整个塔楼的奇幻设计闪耀著钻石、红宝石、蓝宝石和翡翠，垂坠花饰是由玫瑰色、蓝色和绿色的珍珠组成，各个如鸡蛋般大；这将所有的塔楼连接在一起，柔和闪耀的光芒融合了可爱的温文尔雅。窗框是由珍稀的宝石珊瑚制成，只有伟大海神之仆才能找到。精灵女王在建造此奇妙宫殿时，海洋中强大的崔莱顿将这些送给精灵女王作为敬意。 城墙是用最好的大理石砌成的，城堡周围有一条长廊，由玫瑰石英、孔雀石、黄松石和青金石制成，柱头雕以各种奇幻的鸟、花和仙女图案。俏皮的晨光笼罩城堡，金光闪闪，仿佛一朵不可思议的美丽云朵升天。 接著，好奇的阳光从水晶竖窗外偷偷窥视，进入房间后，在墙上跳起了欢快宫廷舞，饰以花瓣和花朵；美丽的精灵女王在微笑中醒来。她从三色堇叶的沙发起身，穿上银色长袍，走进花园，用勿忘草蓝的眼睛静静地注视其管区。接著展开闪亮的翅膀，在花园里飞来飞去；骄傲的玫瑰和高大的鸢尾花低下了可爱的头，其他花朵也恭敬地向她致意。

精灵女王飞到城堡主门，进入地精们准备的交通工具，进行早晨巡访，用神圣的临在使臣民心生欢愉。此魔幻的交通工具是由巨大、玫瑰色的空心珍珠组成 : 内衬是纯正的丝绸，由特殊仙女蚕所纺织，再由华丽彩色蝴蝶织成；内部有个柔软芳香的座位，由精致的玫瑰花瓣制成，并用银色薄纱缝在一起，塞满小天鹅的细毛。在珍珠后方有个银色座位，坐著一只美丽而庄严的飞蛾；翅膀柔韧，闪闪发光，替精灵女王搧风。 而在前方的华丽金盒上，坐著一只神情庄重的青铜色甲虫；他持著山中百合花蕊织成的长鞭，维持八只小金甲虫的秩序，拉著此美妙交通工具前进。

轮子是无数细小的金色亮片，精灵们在正午时分从太阳的光线中摘取，而在午夜时分，地精们在地下工场将此銲接在一起，沈浸在铁砧和锤子的欢快歌声中。此交通工具四处都有小小灯笼，用生长在仙谷中的透明花朵制成，各自装著一只小萤火虫。 有时傍晚时分，女王会驾乘并环绕王国，这些灯笼散发出柔和、魔法般的绿光。整条道路的每一片挥舞草叶上，都有磷蓝色的萤火虫坐著，被巫术般的梦幻光芒所包围，如银月尘云降落乡间。或许你会认为，这位女王统治著如此美妙的王国，在如此美丽的环境中受人民爱戴，应该总是快乐的-- 然而不幸的，事实并非如此。

她常常花上几天在渴望的梦中度过，也不知道渴望的是什么；对于周围宁静的欢乐视而不见，即使地精们多么努力，也很难使她露出一丝微笑。 心灵的欢乐花园已成了绝望挣扎的荒野。忠臣徒劳的想让她重新振作，却无丝毫效果；只见她日复一日地憔悴，如裹著悲伤外衣，被欲望的麝香套索套住。

最后，地精首领召集开会，邀请最年长和最聪明的同伴，盼能找到解药，以终止心爱女王的悲伤情绪。在会议正要开始之时，远处响起了欢呼声和军乐声：邻国的代表团 (由花之国国王所统治) 赶来向精灵女王传达致意，送礼以表示友谊和赞赏。女王非常亲切地接待了这群光鲜亮丽的年轻贵族，有那么一瞬间，她似乎又高兴起来了；但很快，她又恢复了往日渴望的神情，变得非常安静。代表团受到丰盛的款待，满载而归。小地精们则继续开会，一名最年长的地精起身向众人说道 :

「很长一段时间以来，我们看到可怜的皇后日渐憔悴与苍白。我们尽一切努力照亮其生活，但都失败了，令每个人都深感痛心，不言而喻。我想了很久，似乎有个计划能恢复女王的幸福，使快乐与繁荣重返此地。刚才大家都见到了一群高贵的年轻骑士，向亲爱的女王表达了花王的敬意。国王的弟弟去年也在这支辉煌的队伍中，其高贵气质定是令女王留下深刻的印象，其灵魂正渴望著与她同年龄和同身份的伴侣。当王子和随行人员去年回国时，她才开始感到孤独，因为两人此前几乎都腻在一起；诚然，王子的青春、美丽以及善良的内心上，都与她不相上下。

「所以我很肯定，若能派一个秘密使团去找花王，告诉他女王这段日子有多么忧郁，他定能理解并提出唯一可能的解方；你们都知道是什么了。 此外，今年代表团的一位贵族告诉我，这位年轻王子也常常热切望向我们的国家。 国王作为兄长非常担心，看到王子也日益苍白，如亲爱的女王一样无精打采。由于此原因，才没允许王子陪同今年的代表团。而花之国著名且博学的医生，包括睿智的猫头鹰和仙鹤，尽管用了所有秘密神奇的药草与药剂，都无法恢复年轻王子的健康和活力⋯⋯因此可见，可爱的女王和高贵的王子都在为彼此的爱而憔悴，我们应尽己所能使佳偶重聚，解开其纠结的欲念之丝，剥除其不安的外衣。」

经过短暂的商议，大家同意由三名最年长且最聪明的地精，作为大使去面见国王，当然包括提出此想法的地精。三名地精任务开始，为了不被女王发现，他们前往秘密地下通道，这连接了仙境所有领域。他们充分准备，装备精良，但谁能保证说，最好的防护措施就不会被命运的可笑恶作剧捉弄呢? 我们暂且先不谈这个。他们在漆黑中走了几英里，地精们随身携带不灭魔法火炬，由浓缩星光制成，方得以照亮前路。首先抵达的是空气精灵的地下城，他们生活在大地的东部。其身体组成的物质与人类不同，仅由最纯净的空气原子组成。这些存在非常优美，却残忍、任性和野蛮；身体透明，并闪烁著柔和的乳白光色彩。在身体周围，有一些长长的、流动的薄纱长袍，在气流中飘动。这些空气精灵能快速移动，如思想般迅捷，甚至比光还快。他们不允许任何人入侵其领地；无论是地上还是地下，因为在地下也有巨大的空间呢！空气精灵能看得比老鹰远，监视著其他种类的精灵；每当他们抓到一个精灵，就会用空气吹它直到爆裂。地精可不希望遭遇这样的命运，因此他们小心翼翼地、悄悄地向前走，这也是地精最擅长 : 尽量躲在阴影里。然而就在要穿过城市时，一群漂浮的哨兵发现了他们，并以可怕的回旋冲下来，包围此三人组。幸运的是，地精知道如何使用正确的咒语来麻痹空气精灵，并向东、西、北和南大力吹气，将空气精灵们赶走，顺利逃脱。

尔后来到了西方的地下河与湖之国，居住著水精灵。一般而言，只要地精待在干燥的土地上，水精灵就不会伤害他们；但是，他们此时需穿过几条小溪和一个大湖，因此必须极其小心地避开水精灵。水精灵完全由微小的蒸汽粒子所组成。若地经被抓获，水精灵会把他们拖进水之王国中，作为奇珍异宝放在水晶博物馆里；又或者遇到脾气不好的，有可能被切成非常小的碎片，喂给其宠物鱼。总的来说，水精灵比空气精灵更加温柔，但还是有一些不那么善良；若地精被抓住的话，谁也说不准会发生什么事。在这些仙境中，往往有各种奇怪的和意想不到的惊喜，最好始终保持警觉心。起初一切都很顺利，渡过几条溪流后便到达了大湖。他们用干芦苇做成筏，到达湖中央，此时行踪还是被发现了；水精灵从四面八方的波浪下面升起，包围此脆弱的小船。有些抓住木筏，用尽全力摇晃，试图把这三人组翻倒，从而拖下水去。但小地精们也不甘示弱，打开随身携带的袋子，拿出一把把香灰，抛向四面八方，水精灵无法抵御此强大魔法而失明。于是地精以最快的速度划船，在水精灵恢复神智之前，到达了大湖另一岸。下一个危险任务是前往北方哥布林的国土。

地精虽住在地球内部，但大部分时间都在地表；而哥布林则是完全不同的种族，对地精怀有巨大的敌意。哥布林极为卑鄙、可恶、和嫉妒，由土的粒子、粘液和泥土所组成。他们有可怕、灰色的身体，闻起来如陈旧潮湿的地牢，凸起的眼睛如死鳕鱼般。地精们深知，若被哥布林抓到，将会失去一切，因而此时需比以往更加小心。他们不敢前往直达地心的地下长廊，因为那里到处都是哥布林，看守贵金属、矿物和各种稀有珠宝等巨大宝藏。他们唯一的机会是待在哥布林国家的表面（虽称为表面，但仍在地下深处）。尽管他们能设法躲过哥布林，却无法躲避警觉而凶猛的红龙和绿龙监视。这些怪物是由哥布林所饲养，如人类的看门狗般。不幸的是，其中一个地精踩到了一根干树枝，被一条野蛮的龙听到了，便立刻发出了警报，幸运的是牠被链条绑著。地精们很快拿出了魔笛，龙陶醉于音乐而安静下来。地精们轻轻地鸣笛离去，但为时已晚。 成群的哥布林从四面八方冲过来，不一会儿，他们就被包围，并结实的镣铐捆住，被胜利的哥布林抬到深山洞里。

地精们绝望地看向彼此，任务似乎已彻底失败。他们在阴暗的监狱里呆了一会儿，陷入凄凉的沉思；洞后突然传来一阵轻轻的敲击声。这是一种特殊的敲击代码，用于地精和盟友之间的沟通，如小精灵和棕精灵；他们能通过此代码在地下长廊里远距离交谈，而获得帮助的希望就在眼前。一名小精灵间谍在秘密处目睹了他们被抓的情况，于是透过代码告诉地精他已派人求援，棕精灵大军很快就会前来救援。过了一会儿，远处喊叫声越来越近，喧閙声也越来越大。小精灵和棕精灵正与哥布林作战，并杀死了大量的哥布林。尽管哥布林们点燃了大火，并焚烧由硫磺、旋花和阿魏草根制成的香，试图赶走并杀死入侵者；但小精灵和棕精灵们越来越近，在地精所处的洞穴外进行最后一战。 

与此同时，三位地精已设法挣脱束缚，并焦急地看著这场战斗。小精灵向哥布林投掷花环，是由柏木、梣树和铁筷子缠绕而成。由于这些花环拥有剧毒，哥布林意识到这场战斗已败；但仍不想放弃其俘虏，于是派了三个刽子手到山洞里杀地精。这些邪恶杀手并不知道地精已解开束缚，因此在进入山洞时，地精立即伸出死亡戒指，镶嵌著玛瑙（可以杀死任何哥布林），这些刽子手如同雷击般倒下。战斗结束了，小精灵和棕精灵满心欢喜地护送三名地精来到哥布林国家的边界。在致以谢意后，继续向南方走去，走向火精灵的国家。

从此刻开始，是旅程中最危险的部分：火精灵完全是由炽热的火焰原子组成，无物能承受；其飞行速度甚至比思想还要快，无人能逃过其炯炯有神的锐利目光。地精们希望能借由各种驱魔知识来通过这次试炼。将水洒在哥布林身上的话，就能使它们变成泥巴而消灭；若要对付火精灵，则可以使用盐、香、硫磺、樟脑和白树脂的混合物，浇在它们身上后便能杀死。此魔法配方是唯一能影响他们的东西，且只有地精知道正确咒语来搭配此混合物配方。尽管此前三位地精被哥布林们打个措手不及，但这一次他们做好了一切准备，把致命药物的袋子置于手边，以便立即使用。

有段时间一切都很顺利，抵达了火精灵国家的中部。在岩石峭壁之间有个狭窄峡谷，在他们迅速前行时，地面突然坍塌，跌入一个无法爬出的大深坑。 一切似乎完蛋了。 毒药袋中的珍贵物品散落一地，现已完全无力抵抗，也无法指望朋友或盟友的帮助，因为没人敢进入凶猛火精灵的国家，也没有人敢与之战斗。不久，这三名地精被发现，并被拖到了可怕的火精灵国王面前，其名为「德钦」。他坐在一个白炽宝座上，用巨大的火焰搧自己，气息发出可怕的嘶嘶声，吹出滚烫的蒸汽云；炽热的红眼瞪著此恐怖领域。他转向囚犯，气势汹汹地嘶吼著，大声咆啸 : 「你们这些可悲的蚯蚓在我的国家做什么？」可怜的地精颤抖告诉他，他们只是路过这里，且是执行花王的一项和平任务。地精中最年长的看到国王没有立即回答，于是便屈膝且伶俐的说到 : 『哦，玫瑰的尘埃，永恒火焰的神圣国王，火焰和火的主宰，火焰的主人，你像新鲜的年轻柏树；你那美丽树叶的沙沙声，如同一杯陈酿的乐音；当你开口说话时，甜美而高贵的语调如蜜般洒落，庇佑著穿行于你神圣领域的谦卑旅人。你的每一句话，都散发著智慧芬芳的吉祥气息。我们只是无害的旅行者，渴望得到您高贵的宽容，以向我们遥远的目标前进。』在这邪恶的厅堂内，地精试图下一场命运之棋；徒劳地希望至少能打成平手。

「哈，哈。」 国王如此喊道，像一只发疯的牛蛙，因母蛙躲离他而承受单恋痛苦。「你这个恶毒的骗子，圆滑虚伪的银舌蛇，我看透了你的谄媚哄骗，不会被你流利的口才所欺骗；你可告诉我：你通过我们国家的许可证在哪里？」
