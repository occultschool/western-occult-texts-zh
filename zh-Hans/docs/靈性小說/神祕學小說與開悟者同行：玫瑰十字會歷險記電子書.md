# 《与开悟者同行：玫瑰十字会历险记》电子书
## 任何对神秘学怀有兴味的读者，都不应错过这部入门经典。

![封面](/img/downloaded/e6d15ef3243c6cbd6035e77f4f914286.png)

电子书连结：[点击阅读](/books/awakening-rosicrucian-adventure)

《与开悟者同行：玫瑰十字会历险记》初版于一九一〇年，作者弗朗茨·哈特曼首次以小说体裁铺陈神秘学思想。笔调轻快，深浅皆宜，无论初探或深研的求道者，皆能从这本精要的入门书中找到乐趣。

故事始于阿尔卑斯山的一次远足。叙事者偶遇一位神秘矮人，引他前往山中隐修院，拜会一位玫瑰十字会的开悟者。书中探讨诸多玄奥主题：炼金术、前世、心灵感应、灵视力、星光体漫游、元素精灵，乃至驾驭心智的种种法门。

隐居处另有住民，包括两位神秘且极为美丽的女性「启蒙者」，她们娓娓道出鲜活的前世记忆。书中还演绎了一场奇妙的示范：如何以炼金术士闻名的「红色粉末」，将白银点化成金。

总归而言，这是一本奇书，真假相参，恰似哈特曼本人绚烂不羁的一生。即便如此，本书仍旧值得一读。其对自然的描摹令人神往，所揭示的神秘实修洞见，在其他「开悟者」小说中亦属罕见。

节选《第六章：炼金房》：

西奥多罗斯并未从那张大扶手椅起身，只吩咐我取来桌上的空坩埚，置于坛上燃著火的三脚架。我依言照办。接著他说：「现在，从你口袋里掏出几枚银币，扔进坩埚里。」我摸出随身的七枚硬币，全数投入。几分钟后，银币开始熔解；见那银液在锅中漾开，我便禀告开悟者，说它们已然熔化。他随即指向桌上一只盛著红粉的小瓶，要我往坩埚里洒一点。我取来一支小银匙，舀了一两匙红粉，正欲倾入，西奥多罗斯却拦住我，说这太多了，不该如此挥霍。他要我将粉末倒回瓶里，再用纸擦拭银匙，然后把纸扔进坩埚。粉末归瓶后，匙上残存的红粉已微不可辨；但我仍依他所嘱，将那小纸片掷入熔银之中。纸片霎时燃起，熔融的金属随即沸腾鼓胀，几乎要溢出埚缘；泡沫升到顶端便迸裂，绽出绚烂难言的缤纷色泽。

这色彩的幻变持续了约莫一刻钟，液体渐渐止沸，泡沫缓缓沉回埚底。西奥多罗斯听闻坩埚不再作响，便指示我将它从火上移开，把内容物倾倒在大理石板上。我照做，那团泡沫顷刻凝固，竟化作了成色极好的纯金。

「带著这金子去吧，」西奥多罗斯说，「让人检验一番，你才肯信这并非幻觉。」

我惊诧不已，心想世人或许愿意付出任何代价，只为窥探这红粉的秘密。我极想询问开悟者，这粉末该如何制得，却不敢贸然开口，唯恐西奥多罗斯以为我心存贪念。然而开悟者早已洞悉我的心思，他说：

「待人的灵性成熟，我们方能告知他如何制作这红粉。欲知此秘，必须亲身践行。我们又该如何教人运用他们既未拥有、亦不知其存在的力量？话说回来，这些力量的种籽，其实早已潜伏于每个人的生命之中。」
