# 亚特兰蒂斯的书单

## 本文简介七本探寻失落大陆亚特兰蒂斯的重要著作，内容涉及科学、考古、神话等证据。

## 介绍

一九三八年，布拉金上校曾写道：「巴黎有一座图书馆，专藏亚特兰蒂斯学文献，逾两万五千册。」齐罗夫所编书目，更列举八百二十五种资料。由此可见，以「科学方法」探讨此主题的著作何其浩繁，我们仅从中遴选七本，作为进阶阅读指南。

这几本书各有千秋，对钻研神秘学的学生皆有价值，也因而吸引不同的读者。若想通盘了解亚特兰蒂斯，伊格内修斯·唐纳利的《亚特兰蒂斯：太古的世界》不可或缺。布拉金的《亚特兰蒂斯的影子》则承其余绪，搜罗史实之广，罕有能及。

《安达卢西亚的亚特兰蒂斯》地位独特，它提供确凿的考古与人种学证据，证实西班牙曾有一处远古的亚特兰蒂斯殖民地。若是科学家读齐罗夫的《亚特兰蒂斯》，必觉兴味盎然；书中地理与地质证据之丰，远超前人。巴里·费尔的《公元前的美洲》虽非专论亚特兰蒂斯，却详述其文明遗迹及北美殖民地，故亦收入。

《奥拉琳达书》难以归类，既是史录，亦如圣典。凡渴慕亚特兰蒂斯古老智慧者，皆能为其吸引；这也是我们在文中屡加引述、力荐此书的主因。

最后推荐《秘密教义》。此书迄今仍是关于古代智慧最完备的资料之一，内藏许多亚特兰蒂斯及其居民的神秘过往。

## 1. 《亚特兰蒂斯：太古的世界》（伊格内修斯·唐纳利）

![封面](/img/downloaded/d2d17f5921be177c44db2e263f9d705f.png)

此书虽遭学院派全面否定，却是亚特兰蒂斯研究极关键的作品。唐纳利详考各国灾难传说，并藉语言、文化、宗教之比对，论证旧世界与新世界的同源。凡严肃探究此题者，皆应由此启程。

唐纳利之成就非凡。他从庞杂史料中梳理千丝万缕，将之织成经纬，把亚特兰蒂斯的完整轮廓呈现读者眼前。

他试图证实数个命题：柏拉图所述乃真实历史；亚特兰蒂斯是人类脱离蛮荒、迈向文明之摇篮；其殖民者曾遍布欧美；最古老的殖民地或是埃及；最终，一场骇人地震吞噬全岛，大陆沉海，生灵俱灭。

这些观点大致无误，也符合先前介绍的神秘科学事实。不过，最理想的还是直接阅读唐纳利原著，亲自思辨。

## 2. 布拉金上校《亚特兰蒂斯的影子》

![illustration](/img/downloaded/6ee8f287f27348d455cf59a99ab8ecb7.png)

《亚特兰蒂斯的影子》初版于一九三八年，至今仍是科学亚特兰蒂斯学的经典之作。布拉金上校广罗考古、人类学、历史与科学证据，佐证失落大陆的存在，探讨范围甚广：中美洲的埃及渊源、羽蛇神话、巴斯克语与亚特兰蒂斯的牵连，乃至墨西哥、埃及与亚特兰蒂斯金字塔的对照；此外亦论及猛犸灭绝之谜、巨人传说、各地洪水神话等多元议题。

与巴里·费尔（见下文《公元前的美国》）相同，布拉金亦是语言学家，书中详析古代语言的相似处，更考察了巴西鲜为人知的古铭文。其中一项精彩研究，甚至追溯香蕉的起源。

我们尤其赞赏作者的态度：他自陈其理论「仅奠基于事实，容人依此推敲合理的猜想」。此言至少堪称诚实，相信思想开阔的读者皆能认同。基于此，我们诚挚推荐此书。

## 3. E.M. 惠肖《安达卢西亚的亚特兰蒂斯——民间记忆研究》

![illustration](/img/downloaded/9845d9e95f2cd84a9a3fd80f2858ae6d.png)

这本独特的著作提出：西班牙安达卢西亚尼布拉一带发展出的史前文明，或多或少直接源自亚特兰蒂斯的殖民者。他们早在一万两千年前，甚或更早，便已建立塔尔特索斯城与港口。我们认同作者的观点——纵使当时学界极力贬抑她的研究，此书却处处流露她的谦逊、真挚与对真理的执著。

作者举证丰富，包括亚特兰蒂斯殖民者在水利工程上的巨大成就、采矿冶金与造船的精熟技艺，尤其关键的是他们以巨石结合混凝土筑造建筑，至今仍坚固如初。这项技术，或可追溯至三万年前。

## 4. N.F. 齐罗夫《亚特兰蒂斯》

![illustration](/img/downloaded/2ad7332e41e992a3b55f06167c349175.png)

这项研究对亚特兰蒂斯科学探索贡献卓著，聚焦于失落大陆的地理与地质证据，理应引起科学界广泛关注。然而，齐罗夫《亚特兰蒂斯》所处理的争议题材，与过去百年间其他严谨著作命运相仿，在俄罗斯以外几近湮没无闻。

齐罗夫身兼化学家与苏联知名科学家，书中汇整大量地震学、地质学、气候学、古生物学、生物学、板块构造与海底侵蚀数据，因而与当代其他揣测幻想之作迥异。

除列举诸多事实与科学数据佐证论点，本书亦参照历史与秘传文献，如《以诺书》、古埃及《死者之书》与赫尔墨斯典籍。

书中含大量照片、地图、表格、图解，并附三十五页书目，列举八百二十五种亚特兰蒂斯相关书籍与论文。

## 5. 巴里·费尔《公元前的美国——新世界中的古代定居者》

![illustration](/img/downloaded/d934a91be71854df5fd5fcffa5ea794d.png)

费尔教授在其研究中，探寻了亚特兰蒂斯殖民者与商人留在北美的语言遗迹，却刻意避开「亚特兰蒂斯」这个争议之名。他提出古埃及人、巴斯克人、凯尔特人与腓尼基人远在哥伦布之前便已抵达美洲，这番见解遭到学术正统的轻蔑与嘲讽，更是不在话下。

倘若他敢在那些出土的古文字书信中点出亚特兰蒂斯，其著作恐将遭逢无声的封杀。他的研究大多遭人遗忘，实属憾事——因其不仅照亮了亚特兰蒂斯学的诸多谜团，更印证并推进了早期书信学者与民族学家的成果，如奥古斯都·拉普隆贡（Augustus La Plongeon, 1825–1908）、布拉瑟尔·德·布尔（Brasseur de Bourbourg, 1814–1874）与詹姆斯·丘沃德（James Churchward, 1851–1936）。

## 6. 《奥拉琳达书》——威廉·R·桑德巴哈译自荷兰文

![illustration](/img/downloaded/695761320e7720278197f61446011b64.png)

一个多世纪前，《奥拉琳达书》英译本问世，随即被科学与考古的正统护卫斥为「骗局」与「厚颜无耻的赝品」；无论何时何地，此书总引来反感与责难。维基百科的介绍便是典型，其中写道：「本书真实性至少获部分新纳粹团体背书，或许因其主张某些中东文明源出北欧。」此言属实吗？试想，若有人明日发现希特勒床头总摆著《顽童历险记》，这本书必遭严厉谴责，视为助长那位奥地利独裁者种族理论的帮凶。

所幸在我们这时代，书籍不会因偶尔抵触或动摇主流正统而遭焚毁；否则《奥拉琳达书》早已被扔进教条式科学正统的烈火。事实上，任何人皆可阅读此书——它记载了亚特兰蒂斯殖民者在北欧最后残迹的独特叙事；读者自可判断，究竟该视之为「厚颜的骗局」，抑或全然不同之物……我们在文中引述它，因其蕴含某些极崇高的宗教概念与实用的生活法则，故而热切推荐给所有读者。由 sacred-texts.com 出版的电子版本，忠实复刻了 1876 年的原版。
