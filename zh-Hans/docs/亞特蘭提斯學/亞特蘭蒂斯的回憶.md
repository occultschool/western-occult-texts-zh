# 亚特兰蒂斯的回忆
## 雷德-哈格德笔下的亚特兰蒂斯，描绘出这个失落文明的科学、艺术与历史。文字生动，交织出那既辉煌又终归覆灭的共同故事。

![封面](/img/downloaded/54ae53087e97b7bc6310c57bc989978f.png)

先前提过，雷德-哈格德（Rider Haggard）这人，天生具有遥见远古的天赋，尽管他自己多半浑然不觉。他的几部历史小说，鲜活地勾画出亚特兰蒂斯的盛世，以及那漫长的倾颓。

本文将与各位分享其作品《当世界震动时》（When the World Shook）的几段摘录。此书虽被文学评论家讥为「高雅的通俗之作」，却真切呼应了我们对于数万年前亚特兰蒂斯艺术、科学与历史情境的记忆。

但开始前，容我先作几点说明。《当世界震动时》讲述三位伙伴——巴斯丁、比克利与阿布斯诺——漂流至一座未知南海孤岛的故事。岛上，他们遇见一个先进神秘种族的最后两名成员：欧罗勋爵（Lord Oro）与他那美丽的女儿伊娃（Yva）。两人已在玻璃棺中悬浮沉睡，历时二十五万年。

情节自然掺杂虚构。然而我们毫不怀疑，亚特兰蒂斯的科学家在保存人体方面，远比古埃及人更为长久完整。以下段落，透过伊娃女士的灵视能力，自她记忆中唤醒，向巴斯丁、比克利与阿布斯诺展现的，正是亚特兰蒂斯昔日那令人屏息的辉煌与力量。

「我先让你们看看，我们子民在最鼎盛时期的模样。请看前面。」

一幅壮丽图景倏然展开。一座白色大理石宫殿巍然矗立，前方广阔的庭院洒满阳光，生意盎然。宫殿台阶下搭著丝质遮阳篷，篷下坐著一位国王，头戴皇冠，身披华丽长袍，手执镶宝石的权杖。他神态庄严，身旁簇拥著一群服饰闪亮的廷臣。左右各有美丽女子为他打扇，稍远处还坐著几位珠光宝气的女子，料想是他的妻女。

伊娃说道：「这是智慧之子中的一位君王，新近加冕，赢得全世界的尊崇。」

话音刚落，其他国王陆续现身，一个接一个走向宝座，人人头戴王冠，手持权杖。抵达御前，他们屈膝跪下，亲吻宝座上那人的脚。行礼后，各人放下权杖，再依指示重新举起。这群君王约有五十余位，肤色相貌各异，有白人、黑人、黄种人、红种人。

接著，他们的臣子捧著贡品上前，显然是黄金与珠宝，堆放在宝座前的托盘上。我记得注意到一段插曲：一位跛脚的老者绊了一下，打翻托盘，盘中物品滚落一地。他慌忙追捡的模样颇为滑稽，连宝座上的君主也卸下威仪，展颜一笑。我特别提及此事，正为说明我们所见并非凭空臆想，而是往昔一段鲜活片段。因为若由我来设计这场景，断不会让一位衣饰华丽的老人在隆重典礼上失足跌倒——这荒谬一幕，定会被我删去。

不，这必定是过往的生活，真实的历史，已然发生的事。后续情景亦然。譬如，接下来是一场我们所谓的阅兵。步兵列队前行，有人手持剑矛，我猜想是仪仗卫队；有人吹奏著形似野蛮人用的管乐器，我全然不解其用途。现场虽无大砲，却出现几辆马车，车上载著一些带喷嘴的袋子。或许袋中装的是毒气。还有骑兵，骑的马与我们今日所见不同：体型更敦实，四肢较短，但颈项高昂，目光炯炯，想必极为强健。

我推想这些安排主要是为了观赏。接著，有人坐上一具长形装置，两两钻进一种护甲袋里，只露出头与手臂。那机器看来像一辆加长的自行车，却以惊人速度呼啸而过，我实在看不出它如何驱动。它能载二十对人，每人手持一种外观小巧、形如橘子却显然致命的武器。随后出现的其他同类机器，载运人数则在四十对至一百对之间。

然而，最令人惊叹的，莫过于那些飞行器。它们为数众多，时而如雁群齐飞，时而单独翱翔，时而列队前行；有时更编成严整的中队，前锋与指挥舰各居其位，间距精确如尺规所量。这些飞行器体型并不算大，至多容纳四五人，却迅捷如燕，灵巧非凡。它们似鸟类般振翅而飞，只是无人知晓，那驱动双翼的力量究竟从何而来。

阅兵式结束后，一座灯火通明的大厅凭空出现，庆典于此展开。伟大的国王高踞宝座，身后矗立著一尊命运女神的雕像——或是一件极其相仿的作品；我们则站在雕像的基座之下。大厅里，宾客们身著各国服饰，依长桌而坐。国王起身，转向女神像屈膝跪拜。他三次跪地祈祷，方才归座，举杯向宾客致意。宾客们举杯回敬，并仿效他先前之姿，在女神像前跪拜行礼。唯独我注意到，尚有数人始终站立未跪，他们身披的圣袍，样式竟与今日希腊教会的礼衣极为相似。

如今，这一切世俗荣华皆已消散。接下来的景象转为简朴，是这位国王临终卧榻的场景——我们从他枯槁的容貌认出了他。他躺在那里，年老体衰，生命垂危。医生、妇女、朝臣们围聚一旁，默默注视这最后一幕。画面忽又一转：欢呼的人群中，一位年轻的继任者赫然现身，他脸上故作哀戚的阴霾，终被喜悦冲散。这一幕亦随即隐去。

伊娃说道：「伟大的国王便是如此世代相传。他们前后共八十位，平均统治长达七百年。他们主宰大地，如同昔日的神明。他们积累知识，掌握权力，坐拥无尽财富。他们培育艺术，探寻隐秘，甚至与星辰对话。然而，正如诸神一般，他们的嫉妒之心也日益深重。」

以上所述，皆据雷德·哈格德的记载，描绘了亚特兰提斯最为辉煌、华美与强盛的时代。你或已察觉，这段描述与柏拉图的记述多有相似，足见两者皆对往昔做了真切忠实的呈现。

然而，正如伊娃在其故事中所言，亚特兰提斯那些强大的统治者渐渐「心生嫉妒」。他们本是神之子，却因贪婪、愚昧与堕落而步入邪途，化为布拉瓦茨基在《秘密教义》中所谓的「罪恶之黑」。战争降临诸神之地，曾经照耀亚特兰提斯大陆的「光明」，终被死亡与毁灭的「阴影」笼罩。伊娃接著讲述下去：

又一幕景象浮现。天空布满飞行器，如闪电般将火焰倾泻于城市之中。城内爆起更多火光，击落空中来敌；飞行器如遭火焚的飞虫，纷纷坠落。然而敌机不断涌来，城市终告崩塌，城中再无反击的闪光升起。智慧之子们被迫离开大地表面。

接著是一幕惊心动魄、变幻莫测的可怖景象：陆地沉陷，城市崩塌，火山怒啸；仿佛世界末日已然临近。只见人群如蝼蚁般仓皇奔逃。随后，数百公尺高的巨浪袭来，海洋为之沸腾。大部分陆地被海水吞没，仅余零星土地浮现浪涛之上，由大洪水后的幸存者居住其上。

正如我们先前所提，埃及人制作木乃伊的技艺，乃是传承自他们的亚特兰提斯先祖。这一点，在雷德·哈格德的另一部著名历史小说《她》中亦有提及。书中女主角阿耶莎曾向叙事者贺拉斯·霍利描述，这门科学在古代科尔城达至何等巅峰。雷德·哈格德选用此名，或许是为了避开当时颇具争议的「亚特兰提斯」一词——在他那个时代，多数出版商与读者对此词的接受程度，远不如今日。事实上，据我们所知，他在过往小说中从未使用「亚特兰提斯」一词，仅在其自传《我生命中的日子》中有所提及。

「科尔人也如埃及人一般为死者防腐，只是技艺更为精湛。埃及人剖腹取脑，科尔人却将防腐液注入血管，使之渗透全身每一处。但这还不止——让我为你细细展示。」她随意停在我们途经的走廊旁一扇小门前，示意哑仆引我们入内。那是一间狭小的石室，与我最初休息的房间相仿，只是空无一物，唯见两张石凳或石床。其中一张长椅上躺著以黄色亚麻布覆盖的人形，布面蒙著一层细沙，轻薄得几乎难以触觉，事实上也不如人们想像中那般积尘——因这些深掘地底的洞穴里，几乎没有什么能化为尘埃。墓室的石架与地板上，尸身周围摆放著许多彩绘陶瓶，然而在此类墓室中，我很少见到其他装饰或武器。

「哦，霍利，把布揭开吧。」阿耶莎说道。可当我伸手欲揭时，却又迟疑，将手缩了回来。这举动似有亵渎之意，何况老实说，此地的氛围与眼前人形那肃穆的威仪，著实令我生畏。她见我犹豫，只是微微一笑，便亲自上前揭开亚麻布，却发现石凳上的形体还裹著另一层更纤细的织物。她将其轻轻收起。于是数千年来第一次，那张冰冷的遗容，被一双充满生机的眼睛所凝视。那是一名女子；看来约莫三十五岁，或许更为年轻，确实是位美丽的女子。

她面容静雅，眉目清秀；长长的睫毛在象牙色脸颊上扫过微影，分外动人。身著白袍，墨蓝长发垂落袍边，安详长眠。臂弯里躺著婴儿，小脸贴著她胸口。这温馨一幕感人至深，我不禁真情流露，震撼得几乎落泪。

尸首都经精心处理，保存完好，恍如数千年前离世时模样。静寂石室中，无物能伤：不惧热、不惧冷、不惧湿气，身上弥漫的芳香气味，彷佛永存。

但也有一些例外。外表看来肉体完好，一触却陷，才知已化作尘灰。阿耶莎说，这些尸首可能葬得仓促，只以药剂浸泡，未能注入体内，故保存不全。

回到《当世界震动时》一书，继续探寻亚特兰蒂斯的艺术与科学。这失落文明精于天文学、磁力、重力与蒸汽技术，甚至微用电力。地下城的照明系统，正源于此。他们工于建筑艺术，所造楼宇雕像可见一斑；更能翺翔天际，技术远超我辈所知。

更奇者，其中有人掌握第四维度运用，能穿物越障，入出于无。他们拥有双重形体之力：在一处将身体解体，再到别处重组，或将所谓「双重体」——或称「星光体」——投射至远方。这星光体虽不可见，却能感知、能见、能听、能记，回到身体后经验犹存。

伊娃说，心灵感应于智慧之子亦是天赋；他们娴熟使用此能力，如同我们使用无线电。不过，其收发是由经验丰富者轮班值守，故信息传得又快又准。她还说，有此天赋者善解人心，不易受欺。

以伊娃对「爱」的见解作结，再恰当不过。「爱」在二十五万年中不曾改变，永不会变，因创造一切的造物主，不正是名为爱吗？

「汉弗莱，听我说。尘世、天界、地狱之间，无任何屏障能阻爱力通抵完满。爱必存在，以诸形貌显现，坚持前行，直到终点；这终点无关肉体。无论是忘我之爱、轻贱之爱、挫败之爱、虚妄之爱、背弃之爱、迷途之爱、漂泊之爱、沈睡之爱、活在梦中之爱，甚至醒中犹眠——但凡爱，皆蕴生命种子。爱的形貌并不紧要。若此是真爱，我告诉你，爱终将得胜；即便它似乎崇拜诸多对象，仍会找到那唯一的一个——纵或不在眼前。」

闻此言，一股寒意攫住我心。「不在眼前？那在何处？」我问。

她答：「去问你逝去的妻，汉弗莱。去问静默星辰。去问你所拜的神，我无法给你答案，只一词——某处！莫怕。你以为像你我这般人，会沈沦苦渊？我所知的不多，但我告诉你，我们是其主宰。告诉你，只要渴望不息、信念不灭，我们亦是神灵。对怀疑怯懦者，那里空无一物；对以灵眼观照、伸手探求者，那里便有一切。」
