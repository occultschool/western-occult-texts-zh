# 巨人的考古证据
## 本文检视过去「巨人」存在的科学证据。

![cover](/img/downloaded/b38786129c39190f2591856bb43c348e.jpg)

Photo by [Hulki Okan Tabak](https://unsplash.com/@hulkiokantabak?utm_source=medium&utm_medium=referral) on [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral)

许多阴谋论者虽极力奔走，盼能敦促学界正视大量考古化石，以证明远古曾有巨人族存在；但其善意却遭误导，反倒阻碍、甚至破坏了对此课题的严肃探讨。一则二零一五年于网路流传的「巨人新闻」，便是明证。它迅速引发关注，甚至若干主流报刊也予转载。

报导宣称，美国最高法院一纸裁决，迫使史密森学会公开二十世纪初的机密文件，揭露该会曾压制全美发现的巨人遗骸证据，唯恐主流人类演化年表因此动摇。表面看来，因多数学者皆否定史前「巨人」之说，这篇二零一四年十二月刊于《世界新闻日报》的报导，似乎言之成理。

史密森学会在巨人遗骸的挖掘上，固然遵循主流科学路线，却从未刻意隐藏；美国最高法院也从未对该机构下达此类裁决。这故事纯属虚构，而《世界新闻日报》本是一家讽刺刊物，其网站开宗明义声明：「本报对文章之讽刺性质及虚构内容，承担所有责任。」

事实上，若想扼杀诚实调查与科学事实，何须阴谋论？只需众人达成默契，让其他探求真相者无利可图，甚而危及事业、声誉与生计。此事本就不需、也不稀罕阴谋论插手。若有知名学者或科学家胆敢依据考古化石证据，公开证实远古巨人存在，运气好遭人讥嘲，运气差便饭碗不保。

正因如此，维基百科及所有相关资讯来源，对巨人存在的物证一无所知。倘若这类骨骸获准公诸于世，将掀起多大波澜！主流媒体与科学界于是选择缄默，并将持续如此，直到证据确凿、无可辩驳，迫使众人正视这头「房间里的大象」。但我们最好别抱太大期望。

一八七七年出版的《揭开伊希斯的面纱》中，布拉瓦茨基写道：『本书撰写之际，美国《堪萨斯城时报》刊出一则史前巨人族遗骸的重要发现，既印证卡巴拉学者的说法，也呼应《圣经》寓言。其文记载如下：「密苏里州西部森林调查中，韦斯特法官发现若干圆锥形土丘，结构类似俄亥俄与肯塔基所见者。」

「这些土丘位于俯瞰密苏里河的高崖上；更为庞大显著的土丘，则见于田纳西、密西西比与路易斯安那诸州。三周前，世人尚怀疑这些史前筑丘者曾以此地为家；如今证实，这支奇异而灭绝的种族确曾占据此域，并于克莱县悬崖上众多高丘间，留下一片辽阔墓地。」

报告续称：「迄今仅开启其中一丘。约两周前，韦斯特法官发现一具骨骸，并通报社群成员。众人随他重返土丘，于近地表处掘出另两具骷髅残骸。骨头异常硕大，现代常人骨架相形之下，仅及其一部份；头骨未朽部分亦大得骇人。其中一具下颚保存完好，较文明人之颚足足大上一倍。」

「颌骨中牙齿硕大，或因啃食肉类磨损严重，足见咬合肌力惊人。对比寻常现代骨架，其大腿骨长厚如马，肌肉异常发达。最奇特莫过额骨：低平至极，异于本地所见任何头骨。一道宽约二点五公分的厚骨脊横贯眼眶上方，不似今人额头向前隆起，而是自眉弓处向后斜退，形成扁平头型，显属低等人类。

「考察学者断定，此为史前人族遗骸。其形制迥异于现存印第安族群，所在土丘亦非美洲任何已知文明模式。丘中尸骸呈坐姿，骨骸间散落石制武器——燧石刀、刮削器形制特殊，有别于常见箭镞、战斧等物。而早在白人发现美洲原住民前，此类工具已然存在。

「负责发掘的学者将骨骸存于主街的福博士之处，预备对崖上土丘深入勘察。待科学院下回集会，将呈交完整报告，盼能得出确切结论。然目前已可断言：这些骨架属于不复存在的人类族类。」

一八九一年八月十日，《纽约时报》载，史密森学会学者在威斯康星州米尔斯湖畔发现数座巨型「金字塔形纪念碑」。报导续称：「古时麦迪逊一带人烟稠密，居民不下二十万。俄亥俄与印第安纳著名土丘，无论规模、设计或工艺，皆不及此等神秘巨碑。建者谁属、所为何用，今人唯能臆测。」

十九世纪末季，类似记载层出不穷。一八七〇年，《威斯康星州迪凯特共和国报》称，波托西附近掘出两具保存完好的未知巨人骨骸。一八九七年，《纽约时报》描述威斯康星州枫树溪发现的大型墓冢：「内有巨人骨架，自顶至踵逾二百七十公分，保存完好；颅骨硕大如半蒲式耳量器。骨骸旁散落精炼铜条等物。」此类发现遍及美国——明尼苏达、爱荷华、伊利诺伊、俄亥俄、肯塔基、纽约诸州，皆掘出巨型骸骨。

一九三一年，《圣安东尼奥报》报导德克萨斯大学联合考古队发掘出「当世所见最大人类头骨」。存世照片显示，该颅骨体积为现代常人两倍。标本运抵德克萨斯大学后，经史密森学会学者鉴定，推定距今约四万至四万五千年。

若将目光南移墨西哥，巨人遗迹愈见分明。首例见于纳亚里特：一九二六年五月十四日剪报载，佩奇上尉与德瓦尔达发现平均身高三公尺以上的巨人族骨骸。据当地传说，此族源于厄瓜多。此事耐人寻味——厄瓜多位处墨西哥西南方的南美太平洋沿岸。难道遗骸属雷姆利亚移民，而非亚特兰提斯人？此说颇有可能。

一九二五年六月廿一日《纽约先驱论坛报》（翌日获《华盛顿邮报》转载）记载，采矿队在奇瓦瓦市西索估切附近掘出身高三至三点六公尺的骷髅。一九二七年十月二日《洛杉矶时报》称，墨西哥探险家于西南部圣地亚哥-塔佩克特拉附近发现大型人骨，属「体型巨大」之族类。此后再无下文，实不足怪——如前所述，学者若调查此类遗迹，非但无益，反可能动摇「巨人从未存世」的主流定见，尽失学界立足之地。

一九七六年，考古学家在罗马尼亚特兰西瓦尼亚西部的罗西亚-蒙塔纳，发现一处地下长廊，其中躺著一具高逾九公尺的巨人骨骸。当时缺乏分析设备，骨骸遂被送往莫斯科。而罗马尼亚南部的久尔久，亦曾有类似发现：一九四〇至五〇年间，出土约八十具人形巨骸，身高皆达四至五米。可惜这些非凡发现，其后再无下文。

非洲亦存古代巨人的物证。仅举一例：约翰内斯堡WITS大学医学院地下库房里，藏著一段带髋关节的大腿骨；推算全副骨架约高三点六公尺。此遗骸自一九六〇年代初便存于该校，原由纳米比亚北部矿工掘得，估计至少已有四万年历史。

结束本文前，不能不提两项年代较近的惊人发现。其一在一九二六年，西班牙加泰罗尼亚地区，一座俯瞰普鲁兰斯镇的山上，考古学家于奥伦四千年石墓附近的古维丹尼洞穴，发现至少九具巨人骨骸。据说这些巨人身高三米，头骨与股骨皆异常硕大。

再往南约一百七十五英里，乔治·瓦切尔·德·拉普热于一八九〇年，在法国的卡斯泰尔诺-勒-莱斯，发现三块巨大的人骨碎片，当时覆盖于一处青铜时代墓穴之上。依骨块尺寸推算，德·拉普热断定所属个体身高必超过三点三米。其发现刊载于一八九〇年法国《自然》科学杂志第十八卷第八八八期，并获当时法国各大媒体广泛报导。蒙彼利埃大学的动物学、古生物学教授及其他学者检视该骨骸后，结论指出：这属于一个「体型极高大的族类」。

为免世人以为，历来从无重要学者或科学家敢于支持巨人存在之说——甚至不惜危及自身声誉——本文末略举数位勇者为例：剑桥大学唐纳德·A·卡佐博士在匹兹堡掘得一具二百二十公分骨骸；亚利桑那大学考古系主任拜伦·卡明斯博士发现数具逾二百四十公分的遗骸；史密森学会史前人类学馆长托马斯·威尔逊验证过二百四十公分骨骸与巨型颚骨；卡内基博物馆馆长W.J.霍兰德于宾夕法尼亚州出土超过二百四十公分的骨架，此事获《科学人》等多本科学期刊报导。
