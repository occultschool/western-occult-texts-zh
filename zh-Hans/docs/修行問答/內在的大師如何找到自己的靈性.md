# 内在的大师——如何找到自己的灵性？
## 唯经内在的大师，方能寻得生命的真谛

![cover](/img/downloaded/8e6c28c7031702438e051f32e6298574.png)

Jia Lu —— “Flight”

物质主义极盛之世，许多人总觉得日常生活琐碎，与任何灵性断了联系。另有些人，却感到内心有个「东西」，吸引他们探向灵性的更深处。不论男女，生命总有那么一段时光，会意识到这「内在的东西」；尽管称呼各异，各大宗教、灵性哲学与传统，都蕴藏了对它的描述。尤其在神秘传统中，常提及与内在神性接触甚至重逢的经验；有些传统视此为高我，是真正的个体性自我。我们也读过许多伟大的神秘主义者，「在光的云彩中，被带到神面前」；或受内在之光启发的先知，从此怀抱超凡的智慧与爱。这「内在的东西」，在神秘传统里，常被喻为与神的接合点，是神之火的火花；我们姑且称之为「内在的大师」。

许多求道者认为通往这种体验的道路陡峭，难免气馁。他们想：这路或许注定给另一些人，但绝非我，因为那目标看起来是多么遥不可及。于是他们放宽了对神秘主义生活的定义，降低眼界，将这件灵性衣裳裁剪得更合身些。至于少数不畏艰险的人，处境未必更好；眼前矛盾复杂的路线图交错，几乎尚未启程，已然迷失。两类人，都没能触及内在大师的皮毛。

或许，我们该略去细节，试著勾勒几条基本原则、几项指引，好让这条道路清晰些。

许多人怀疑自己能否感知内在自我，这很常见。当代多数人浸润于科学的物质主义思维，脑海里便堆砌了重重怀疑。但这是错的。科学距离揭开物质本质的奥秘还差得远；谈及灵性，更不该奉为圭臬。哲学家维根斯坦曾言：「人必须醒来思考——大众亦然；科学却是让他再度沉睡的方法。」科学虽在其领域成就卓著，所用工具终究是感官的延伸，探测不了那更微妙、更难以捉摸的灵性领域。物质科学能引我们到门槛，却无法再进一步。要越过门槛，唯有找到方法超越肉体与感官——方法即是令它们沉静。如今有些科学发现似乎佐证了这点：身体觉察的世界，原是一种幻觉；假使感官以不同方式运作，物质世界及其一切便会显现得截然不同，尽管物质本身毫无变化。再者，科学告诉我们，物质根本并非以往所想那般坚实，反而更像一片能量振动的海洋，状态与条件各异。有鉴于此，一些勇敢的科学家开始比较量子物理学与古代信仰的相似之处。然而主流社会依然固守反对立场。

话说回来，若要更了解内在大师，显然不该注目科学，而应转向宗教与神秘传统。教法学家鲁米说：「感官之光引我们向尘世，神之光却召唤我们到天界。」各派心理学家也试图解释：我们所思所行，皆不知不觉受本能天性影响。那么，除了从下方（本能天性）升起的影响，是否也有看不见的影响，从上方渗入我们的意识头脑？从神秘主义者的视角看，我们的任务，便是更轻易地接纳这些来自上方更高影响，并追溯其源头——那正是在我们存在的核心深处。

因此，从神秘教诲的宝库中，我们掘得的第一颗宝石是「毋疑」。无论古今，最真切的权威说道，怀疑恰似一片阴暗险恶的乌云，遮蔽内在的光，局限我们所能成就的一切。莎士比亚曾道：「疑虑是叛徒，它让我们畏于尝试，因而失去本可赢得的益处。」我们不该怀疑自己觅得内在之光的能力，但稍加思量便知，真我之山无法一步登顶。这并非一世之功，也难于数世之内完成；这是一段漫长而渐进的历程，因此起步愈早愈好。然而，在对眼前长路感到心灰意冷之前，我们或可先想：此刻我们之所以如此积极寻索古老的回归之路，必然是因为已积累了多世修得的经验，这些对成长至关重要。打个比方或许更清晰。伟大文明的兴起，往往通过一代又一代、一世纪又一世纪地铺展。例如，数千年前的埃及，其成就需从更早的时期就开始酝酿，延伸至更遥远的千年之前——纵使对史家而言，那些时代的纪录早已湮没。同样，我们当下的自我追寻，很可能只是「重拾」对生命更深意义的渴求；这份渴求，早在今生之前便已启程。相较于现代史观的有限视野，神秘传统坚信：我们过往积累的成果永不失落——只要知晓何处寻、如何寻。

过去数百年间，有不可胜数的词汇指出「内在大师」。古老传统亦以纷繁的比喻，描绘发现其居所的过程。他们以旅行、航行、朝圣、探险、嬗变等种种意象，象征这场神秘探索的进展。有人将其喻为攀爬长梯；所有伟大神秘见证皆显示：这梯虽向上延伸至神之国度，其底端却正立在我们此时此地所在之处。这点至为关键。若第一颗宝石是「勿疑自己发现内在大师的能力」，第二颗无疑是「他其实比我们所想的更亲近」。我们天性中无私、健康、充满活力、积极求索的一面，事实上便是通往内在大师的阶梯底端，亦间接即是内在大师本身。平常出现的内在警示、鼓舞的念头，或对真挚美感的欣赏，其源头皆可追溯至我们自我中更高的部分——只是我们通常未察罢了。有志的神秘者所需做的，正是辨识这些本就存于本性中的高尚元素，确认它们源自内在大师，进而凭借它们迈出下一步。听来简单，本质亦然；然而所有尝试过的人都同意：其理虽简，实行却绝非易事。

关键的障碍，似乎来自我们本性中另一面——那消极、自私、多疑、畏惧的一面——它屡屡扰乱此程。于是我们来到宝库中的第三颗宝石：在与内在大师建立完整而牢不可破的联系之前，我们必须学会驾驭小我最劣的面向。能否持守内心的宁静与平衡，决定了我们能否触及内在大师；这也是我们踏上征途的第一步，亦是最重要的一步，最终抵达他隐居的所在，存在于我们之中不可见的核心。从日常意识的角度看，内在大师多少仍处蛰伏，无法自如运作，也无法将其力量带入我们的物质生活。只要我们仍受制于低层的激情、恶习、恐惧与焦虑，他便无法提供助力。

其他传统中，人们将内在大师联结于克里希纳、荷鲁斯等神祇；基督教秘传文本则将其与基督相系。于他们而言，神秘主义即是内在基督的逐步开展：随著内心节制与内在平和的建立，基督对其生命的影响也日益加深。最终，当我们准备妥当，必要的工夫完成时，基督——或内在的大师——便会苏醒。坎伯曾言：「每个人内在都住著一位潜藏的基督。圣灵的触碰唤醒内在的基督，使其与上方的基督合一。」我们是否称之为基督，或冠以他名，其实并不重要。无论我们如何称呼，在这场生命伟大冒险里，一切遭遇的疑难，都能因他的光芒而得解。

从神秘主义者眼中望去，唯有寻见内在的大师，方能窥见各自的命运、本真面目，以及在宏大蓝图里所居的座标。唯透过他，我们才找得到生命真正的目的，超脱一切尘世的职分与责任。那些寻得内在大师的人，生活从此天翻地覆；那些接受基督之光的基督徒，皆觉宛若重生，脱胎换骨。奇迹降临了，神恩将他们拯救，自黑暗引向光明。

阿特金森说：「每人灵魂中皆存基督的原则，它不断努力提升个人，实现其真我——这便是『救赎』与『拯救』的真义。基督是救赎者，此事毫无疑问，由他们内在的转变足以证明。从那天起，他成了他们的太阳，是心底的光与生命，是救世主，是天主，是平和的君王，在他们神秘的心灵中央登基。他确实是内在的大师。」

米绍则道：「人心觉醒时，天界敞开，面纱撕裂，心灵扬起，乘狂热之翼张帆远航……或沉醉于狂喜之中。」
