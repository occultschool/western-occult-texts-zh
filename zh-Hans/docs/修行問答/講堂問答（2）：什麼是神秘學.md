# 讲堂问答（2）：什么是神秘学？
## 解释神秘学、神秘科学和魔法之间的区别。

![cover](/img/downloaded/8887462a138d0b066e77170aa47efbce.jpg)

Photo by [Michael Dziedzic](https://unsplash.com/@lazycreekimages?utm_source=medium&utm_medium=referral) on [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral)

关于神秘学的误解，如同对它的幻想一样繁多；许多人认定它等同于巫术、黑魔法与一切邪恶。但这假设是错的。过去与现在，许多虔诚的基督徒——例如牛顿爵士——都钻研过神秘科学。一般人或许不知，爱因斯坦不但是坚定的有神论者，其思想更汲取自布拉瓦茨基的神秘科学教义。圣雄甘地，另一位深具信仰的人，他对印度教的研读兴趣，正是在读了布拉瓦茨基夫人的[《神圣智慧之钥》](/books/key-of-sacred-wisdom-blavatsky)之后首次苏醒的。

可惜，许多自称「神秘主义者」或「魔法师」的人，对自身所涉领域懵懂不清，常将「魔法」与神秘科学混为一谈。他们幻想只要念些「魔咒」、背些咒语，便能如布尔沃-利顿笔下的札诺尼那般神通广大。

也有人以为，只要前往印度或西藏，在某位赤身「大师」脚下静坐数月，便能获得古老智慧；还有人迷信仪式魔法，照著古书或新出的魔法典籍行事，将神秘学视作满足私欲的工具。总之，多数探求者对神秘学的想像，斑斓一如幻梦。

世上很少人真正理解神秘学的本质，并能辨明神秘科学与法术的关联。首先须厘清「神秘学」与「神秘科学」的关系与分野；同时，一旦学会区分神秘学与魔法，便会发现其差异犹如灿阳之于烛火，犹如人类不灭不朽的灵魂之于易逝的个性与肉身。

西方语言为迎合现代空洞的科学唯物论，已丧失了表达伟大形而上的生命真理之能力。于是「魔法」沦为魔术的代称，「巫术」成了哈利波特的戏法，「神秘主义」则是被科学「揭穿」的迷信呓语。

上述词汇如今多带贬义，泛指黑暗时代、甚更早的异教迷信残余。然而，英语中竟寻不出一个词，足以界定那股超自然的神秘之力。至于「奇迹」二字——如今还剩下几分原意？

教条派的基督徒将「神迹」归于上帝或魔鬼这类超自然代理者。许多基督徒又将此代理者与神秘学挂钩，却忘了摩西的神迹，正是他「学习埃及人一切智慧」后的成果；即便在那时代，「智慧」也是「神秘科学」的同义词。那些嘲笑摩西、魔法与神秘主义者的物质科学家，只是不假思索地否定一切「迷信」。这一切混乱，正因英语缺乏词汇以标示差异；没有词语能辨明光与影，没有词语能在崇高真实与荒诞可笑之间划出界线。

难道我们因此被迫二选一：要么视「奇迹」为违反一切自然法则的科学荒谬，要么视其为超自然行为，不须解释、只能盲目接受的「信仰」？神秘科学却告诉我们，摩西的「神迹」只是自然法则的应用，是他在古埃及神庙中习得的。那些神庙正是当时的科学院，所授即为「神秘科学」——亦即「秘密知识」。然而是怎样的知识？古老的梵文或许能提供些许线索。

印度《往世书》所载的秘密知识，主要分为四种。

这四种分别是：一、通过特定宗教仪轨，在大自然中唤醒隐秘力量的学问（祭祷明）。二、「大术法」，近似西方魔法传统，常指最险恶的黑巫术。三、关于声音中隐秘力量的学问（秘音明）。四、「灵魂的识见」（Ātma-vidyā），或曰真智慧，其意蕴远不止于此。

若欲臻明智无私之境，「灵魂的识见」方为求道者当倾力追寻的惟一玄学；其余皆为「玄秘之术」的分支，诸如通晓万物（矿物、植物、动物）终极本质的秘术。炼金、占星、催眠暗示、灵视灵听、卜筮诸艺皆属此类；然「灵魂的识见」，即真智慧，对它们不屑一顾。「灵魂的识见」涵容前述所有，或偶有取用，且必须等待渣滓涤净后，选择有益者用之，并剔尽其中每一分私念。

且容细说。任何男女都可以学习上述任一种「秘术」，而无需事先做任何准备 ，甚至可以不考虑道德和礼节。如此，十名修习者中或有一人成为术士，却往往一头坠入黑魔法。可叹的是，醉心「玄学」者，罕有真正知道自己所渴求的究竟为何，其动机多为自私。

事实是：无论你读到或听到什么，灵性法则皆不能用于物质目的。灵性与物质各有其独立法则（科学界对此所知尚浅，寻常「神秘主义者」或所谓「魔法」实修者更如蒙昧），二者不可相混；譬如求取财富权柄、驱役「魔灵」、召唤星界「情人」等，尽是虚妄自私的痴想。

凡试行此类愚举者，无不发现：烦扰愈增，处境愈恶，其程度正比于在无用伪术中耗费的心力。只需浏览那些专论「玄学」的论坛网站，便知此言不虚，多少误入歧途的稚嫩新手陷于困局。那些自封法师的「玄学名流」，无人拥有丝毫「魔法」之力，只剩追逐虚妄幻想而败坏自身健康、人际关系与生活。

或许要问：若困于欲望矛盾该如何？一旦求道者心中对神秘学的渴望真正觉醒，其内在便永失安宁；茫茫人世，再无可憩息舒怀之地。此人将被一股无从平息、永不止息的不安驱策，闯入生命荒莽的边野。若其心仍浸淫情欲私念，无法踏上光明之途，就不可能在凡常生活中觅得休憩与平静。

那么，这些人注定落入黑魔法吗？别无他途吗？确有他途。这些人无须立下过高目标，亦莫负过重担荷；无须成为开悟者或术士，只要能研读[《金黄星》](/books/golden-star-dr-michaud)、《秘密教义》等书中的真正神秘科学，就已万幸；即使未获超常之力，亦可作人类谦卑的施惠者。

如此一来，他们至少能救赎自己的灵魂（以我们的术语称为「高我」），若执意为自私目的，滥用这人人与生俱来的力量，则必失灵魂。

若你真诚渴望理解自身、生命物质与灵性层面，研习神秘学于你有大益；这般研习不仅惠及你，亦泽被你亲近之人，更能助益整个人类，在多少有些盲昧的存在之路上，更顺利前行。
