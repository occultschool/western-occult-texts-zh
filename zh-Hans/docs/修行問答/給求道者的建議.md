# 给求道者的建议
## 最真诚的建议献给踏上这条道路的求道者

![cover](/img/downloaded/11a2c48c2973513b05aa236a33d603c9.png)

智慧是我们在一切困厄中最好的朋友，它甚至比「英勇」更重要——英勇最终会自取灭亡。神圣的洞察力引导圣人行止，令他前进时毫无迟疑。但真正的圣人不求名，也不图财；高我超越了所有人类的理解，即使最伟大的作家也写不尽它的荣耀。然而，一个仔细聆听的心，能在内在的寂静中听见它的声音。

就算有强大的智力，若无同样充沛的精力，也是徒然。正确运用智力时，所耗的精力比体力劳动更甚。精力不足，智者便无法正当发挥其灵性力量。另一种人，大脑僵固如石，永远将智慧的光芒拒于门外，没有一个念头能穿过物质构成的厚重大门。所有想法都受严密审查，阻挡任何高等思想进入耳中。就算此人空有力量与精力，却没有智力，与前面那种有智无力者同样无能。每个问题都有两面，有时甚至两面皆为负面。

黎明的太阳射出炽热的长矛，刺醒沉睡的世界；获得真智慧的人亦如太阳，散发开悟的火焰光芒，唤醒他人昏睡的心智。这就是诸神有时派遣光之子下凡的原因。他们建起真正的光明殿堂，凡符合美德者皆可进入。这些建筑立于不可动摇的磐石上，因为根基是神之意志所奠立。而那些非神所建的屋舍，根基则是错的。

智者不愿再经历死亡重生，因而努力求解脱，即自我救赎，脱离轮回转世。为此，严格的考验与试炼都是值得的。通过神圣的观照，不懈的求道者终能止息心思，连「想要止息」的念头也一并止息。因为任何尝试努力都是争斗，有争斗便无平和……平和只在停止努力之后才会到来。停止努力之后是寂灭，而后真正的灵性方能诞生。最终，智者发出清澈响亮的神圣话语，真正的神秘视野开启，他不再将短暂尘世放在心上。此时，开悟者真正受启，融入了自己的神性之中。

「服务他人」是通往开悟的道路，人与人之间的服务从来不徒劳。这会产生无数美好的后果，如一缕飘向天际的崇高精粹。它所带来的好处永不会抹灭，且将成倍增长、传播，最终迎来意想不到的幸福。服务形式多样，诗人或画家用他们的方式服务人类，真正的老师或圣人则以戒律的方式进行。

相反，一个恶行就像一首走调的歌，破坏了幸福的谐和。

人之所以屈服于仇恨与其他恶习，是因为小我仍滞留于过去生活的不利情境中，从而以破坏性的方式影响高我。高我必须学会自我控制，从而克服这些缺陷。在小我的秘密深处，潜藏著对物质事物的种族记忆，寄居于继承的本能里。这本能也是物质性的，只是形式略异。因此，只要不伤害众生，将来便能从尘世的痛苦与枷锁中解脱。这适用于念头与行为。陷于仇恨者终被仇恨征服。嫉妒者必将失去那令他嫉妒之物，即使那物从不曾属于他。生活在罪中的人终被罪毁灭。思想黑暗邪恶者将居于黑暗，处境比他自己想像的更可怕。

获得财富不需要智慧，只需足够精明加上命运的眷顾。但有智慧的人才知道如何正确掌管财富。财富唯有在用于减轻人类痛苦时才有价值，否则便是无用，是诅咒而非祝福。千万别追求财富，也别追求权力。心智一旦被权力蒙蔽，便会转向非法的目标。人的六大缺点是：欲望、愤怒、贪婪、迷惑、骄傲、嫉妒。巨额财富或繁盛的事业，若不用来造福他人，它会像闪电一样，瞬间闪烁后消失于未知之处，无人受益。自私的愿望同样徒劳。

当一个人追求灵性时，即使美德微小如种子，也会成长为大树，结出丰美的果实。但当一个人转向物质，沉沦于错误与黑暗，昔日的美德也会被黑暗领域的污水腐蚀，终成不幸。因为一个人的邪恶行为，也会在他内心结果，将他摧毁。

人对于「我」的认识也是如此，好比飞蛾的梦，没有实质，随著虚无缥缈的思绪飘散。然而人的心中，事实上有著永恒本体的胚种。这胚种唯有通过正确的行为与思维，通过永恒的信心，并坚信「人是众神之子，终能成为神」，方能发展。

但在这之前，求道者必须澈底辨明：物质世界的自我是幻梦，有别于灵性自我的实相。而获得这种觉知，极为困难。

除了「服务他人」，另有一条从不失效的康庄大道：真爱之路。这不是感官之爱，而是灵性之爱。尽管在极少数情况下，前者可以引发后者。

真正懂得「爱」的人不属于这尘世，因为他们的灵魂盛满美妙的甘露，从天界的容器流向他们干渴的心。爱在某些时刻能战胜最强大的人，就像苦行僧维萨米特拉（Vishvâmitra）看见梅纳卡（Menâka）时，放弃了他的苦修。或像亚亚提（Yayâti），因爱萨米什塔（Sârmishta）而变得比原本更苍老。尽管他们的爱起初由感官引起，但无人能断言这样的爱最终能否成为灵性的。唯有充满智慧的生命之主知晓。

通往荣耀的下一条道路是通过学习，通过真正的大师与高人的指导，他们的职责是将智慧传授给符合资格的人。因为若将教导传给不适格者，不会带来美德或荣耀，只会导致遗憾。

然而，无论你走哪条路，或是否遍历诸径，要成功还需要一个要件：「勇气」。沮丧毫无益处，灾难不会触及坚定的人，仿佛灾难也畏惧他们的坚定。因此，无论生活发生什么，都要保持决心。在这世上，对于有进取心的人，没有什么是不可能的。对于性格勇敢的人，勇气本身就是成功的原因。因为勇敢的人能凭自己缔造繁荣，无论其志向何指，或者有没有人支持——只需在生活的千般试炼中始终坚定，终将抵达成功。

永远切记，巨大的悲伤往往以更大的幸福作结。因为悲伤是众神的钥匙，用以叩开我们心智的门扉，使人望见灵性更高处的光景——那方是真正的福乐。而凡尘诸事中，时常蛰伏著不幸的虫蠹。
