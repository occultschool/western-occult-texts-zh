# 真正的大师在哪里？
## 当学生准备好了，大师就会出现。

![cover](/img/downloaded/8aeb68dc10df86bc2c2455193b88c669.jpeg)

许多追随「新纪元」教义的求道者相信，即使没有老师指引，也能在神秘学路上前行。

这话有几分道理，但并非全对，也非人人适用。确有可能独自修行，在神秘学领域取得长足进展甚至开悟。但这类学生总会遭遇瓶颈；若无真师引领，便难再进一步。

凡深入研究过释迦牟尼、毕达哥拉斯、柏拉图等古圣先贤教义的读者，便知我们所言不虚。细究这些伟大灵魂的生平，便知他们各有师承。追本溯源，每位真正的导师，其知识若非源于光明，便是得自黑暗。前者导向解放与启蒙，后者引向奴役与无知。唯独经此开悟之师，方能助学生在学习与冥想中明辨是非，识破那些夹杂虚实、最为危险的半真半假之说。

许多读者应听过这句古老箴言：「当学生准备好了，大师就会出现。」

此言与人类历史一样古老，可见于老子的教诲、《奥义书》乃至《圣经》中皆有回响。然其真意何在？何谓「大师」？如何辨认？「准备好」又意味什么？事实是，我们既要准备好「受教」，也要准备好「舍弃旧学」——这两者未必同步。太多神秘学爱好者声称渴求真理，一旦审视其内心，却无一例外发现，他们只想印证自己的幻想与揣测。犹如2012年，一批受惑的信徒，期盼世界立刻神奇地转变成灵性天堂。

正因如此，求道者若不愿放下成见、臆测与谬见，便无缘拥抱真理。他们尚未「准备好」，或者说，还不配遇见真师。心智闭锁，便无法领受真理与智慧那朴素的模样。爱德华·鲍沃尔-利顿在其19世纪名作《札诺尼》中，精准刻画了这类求道者的样态：书中人物怀抱不纯的欲望，又为神秘学里关于力量与知识的错误观念所困。

当学生开始关注内心状态，而非俗世性格时，才算是做好准备，无论是迎接真师，抑或真谛。[《金黄星》](/books/golden-star-dr-michaud)中有言：

「无知之后，是理性的曙光；理智之后，是真正觉悟的黎明，亦即开悟；最后臻于智慧。这五步将人从野蛮提升至灵性。然而其间尚须经历诸多阶段，如恐惧、怀疑、迷信、教条、狂热；未经奋斗与克服，无以见光。」

那光即是大师，大师即是那光，指的是永恒真理的真正导师。对未准备好的人，光明如同黑暗，大师到来亦无帮助，因其欠缺高等意识去认识与理解。

何谓「准备好」以迎接大师？布拉瓦茨基的话可作总结：「生活纯净，思想开放，心地澄明，理智热切，灵觉苏醒。胸怀同胞情谊，乐于接纳与给予指引，对师尊怀有忠诚的责任感。这些人深信师者掌握真理，并愿顺服真理的要求；能勇敢承受被加诸的不公，勇于申明道德准则，勇于捍卫受不义侵害之人，且心怀信念，确信人类能持续进步与完善。此即吾师所示范的黄金阶梯；真诚的求道者拾级而上，终能步入神圣的智慧殿堂。」
