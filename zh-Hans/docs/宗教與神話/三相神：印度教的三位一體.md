# 三相神：印度教的三位一体
## 拟人化的面具之下，蕴含著高度哲学性

![cover](/img/downloaded/5b22da2d11237dbb46ad7f8567b26a5b.jpeg)

三相神，梵语称 Trimûrti，字面意思是「三张脸」或「三重形式」，即三位一体。当代万神谱中，这三者是创造者梵天（Brahmâ）、保存者毗湿奴（Vishnu）、毁灭者湿婆（Shiva）。然而此观念成形较晚，《吠陀》里既无梵天也无湿婆。吠陀最初的三位一体，是火神阿耆尼（Agni）、风神伐由（Vâyu）与太阳神苏利耶（Sûrya）；如《尼禄多》（Nirukta）所解：大地之火、大气之火与天界之火。

《莲花往世书》（Padma Purâna）记载：「最初，伟大的毗湿奴意欲创造世界，遂化身为三重：创造者、保存者与破坏者。至高灵从右侧流溢出梵天，以司创造；从左侧生出毗湿奴，以司保存；从身体中央化现永恒的湿婆，以司毁灭。有人崇拜梵天，有人崇拜毗湿奴，有人崇拜湿婆；然毗湿奴虽一体却三重：创造、保存、毁灭。于真正虔信者，三者实无分别。」

三相神的本质，实则象征从「灵-物质」分化出的三种基本属性（Gunas）：自我形成、自我保存、自我毁灭，旨在促其再生与完善。

确切来说：梵天是「激性」（Rajoguna）的人格化体现，此属性为活动与繁衍的渴望，宇宙万物由此而生。毗湿奴拟人化了「平性」（Satvaguna），象征清明、平衡、和谐与觉知。湿婆则是「惰性」（Tamoguna）的拟人化，此属性为停滞与衰败，自然成为毁灭者。但这并非「终极毁灭」，而是回收、溶解与潜伏——让形相归于未分化的状态，为下一轮生成准备条件。

然而，拟人化的面具之下，蕴含著高度哲学性。这些原始概念若囿于字面，既不哲学，也显荒谬。
