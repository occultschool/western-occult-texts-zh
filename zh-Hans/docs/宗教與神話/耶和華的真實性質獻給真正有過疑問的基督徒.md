# 耶和华的真实性质 — 献给真正有过疑问的基督徒
## 为何耶稣在圣经里从未提及「耶和华」，只称神为「天上的父」？

![cover](/img/downloaded/491cb8e3108ce5be452fb60cd0858135.jpeg)

这是已知最古老的耶稣肖像之一，约有一千五百年历史。它提醒西方基督徒：耶稣原是中东人，非西方产物。西方从未诞生世界性宗教。

《圣经》明示，名为「耶和华」的存在拣选「以色列」作为选民。祂是专属以色列的「民族神」，与别国别族无涉。非以色列人想与祂建立关系，不免显得奇怪。祂与基督徒更无瓜葛：论及耶和华的律法诫命，耶稣的批评与反抗尤为激烈。福音书里这类例子比比皆是。

显然，耶稣口中的「慈爱天父」，与旧约里那「善妒、复仇、暴怒」的耶和华并非同一位。耶稣在《约翰福音》第八章与法利赛人辩论时，似乎便点明了这区别。（法利赛人是犹太教显赫派别，一心侍奉耶和华。）

v38 — 「我所说的是在我父那里看见的；你们所行的是在你们的父那里看见的。」

v40 — 「我将在神那里听见的真理告诉你们，如今你们却想杀我。亚伯拉罕未曾做过这事。」（指「亚伯拉罕未曾想杀我」，或指「亚伯拉罕未曾从父那里听见真理并告诉你们」。）

v41 — 你们是行你们父所行的事。」他们说：「我们不是从淫乱生的，我们有一位父，就是神。」

v42 — 「倘若神是你们的父，你们就必爱我；」

v44 — 「你们出于你们的魔鬼父，你们父的私欲偏要行。他从起初就是杀人的（翻开《旧约》前几页，触目皆是耶和华报复性、任性而费解的屠戮），不守真理，因他心里没有真理。他说谎是出于自己；因他本是说谎的，也是说谎之人的父。」

尽管仍有锲而不舍的虔诚信徒徒劳地否认，事实却是：绝大多数早期基督徒，不信耶稣是耶和华的代表。

试想，那位凶悍咆哮、下令「现在你要去击打亚玛力人，灭尽他们所有，不可怜惜，将男女、孩童、吃奶的，并牛、羊、骆驼和驴尽行杀死」（《撒母耳记上》15:3）的神，怎可能是启发耶稣在登山宝训中传递那美丽振奋教诲的神？

奥菲特派诺斯底基督徒坚称耶和华是「次级存在」，并全然拒斥《旧约》，理由是《旧约》乃此次级存在之作。马吉安主教（公元85–160年）主张，犹太教的耶和华神 **「与差遣耶稣来启示神圣真理、传扬喜讯、为万人带来和解与救赎的神，全然不同。」** 马吉安教导，耶稣的使命意在摆脱耶和华，他说耶和华 **「与耶稣基督的神与父对立，如物质之于灵、不洁之于纯净。」** 见布拉瓦茨基《揭开伊西丝的面纱》第二卷。

巴西里得（公元117–138年）持相似观点。他号称使徒马太及葛劳可斯门徒，后者曾是彼得弟子。

事实上，旧约时代也非所有犹太人都拥戴耶和华。他们中间存有两派——伊罗兴派与耶和华派。两派时常冲突，对立多由耶和华信徒挑起。思及耶和华「战神」之名，此事不足为奇。

新旧约经文，后来被早期教会某些神父有意完全混同，以遂其目的。

以色列那位善妒、暴怒、复仇的神，何处与耶稣所传、那不可知却仁慈的神——他在天上的父，也是全人类的父——相似？唯这位天父是灵性纯净之神，将祂与那次级而反复的西奈山神并论，实属谬误。耶稣可曾提过耶和华之名？……从未。

在那座山上宣讲教训的日子，耶稣划下了一道深刻的界线，将自己的神，与在西奈山上颁布律法的神区隔开来。耶稣的话语清晰而坚决，不仅带著反叛的意味，更是对摩西所称「主神」的正面挑战。他说：「你们听见有话说：「以眼还眼，以牙还牙。」只是我告诉你们，不要与恶人作对；有人打你的右脸，连左脸也转过来由他打。你们听见有话说：「当爱你的邻舍，恨你的仇敌。」只是我告诉你们，要爱你们的仇敌，为那咒诅你们的祝福，善待那恨你们的人，为那凌辱、逼迫你们的祷告。」（《马太福音》第五章）

> 如果摩西的「神」是独一的活神，耶稣又是其独生子，该如何解释这番叛逆的言语？他毫无犹豫、毫无保留地推翻了犹太教那报复的法则，代之以仁慈与克己的律令。倘若《旧约》真是神的启示，《新约》又是什么？难道我们必须信奉并崇拜一位每隔数百年就自相矛盾的神？究竟是摩西领受了启示，抑或耶稣并非神子？……马吉安主教曾质问：「**一位神怎能违背自己的诫命？他怎能一面禁止偶像与图像崇拜，一面却容许摩西竖起铜蛇？怎能既说「不可偷盗」，又命令以色列人掠夺埃及的金银？**」——摘自《揭开伊西丝的面纱》第二卷第三章

凡读过《旧约》者，想必记得上述事件中，耶和华如何命令其选民——以色列人——先杀埃及人，再从尸身上夺走一切金银珍宝。这究竟是怎样的一位「神」？今人常说，世间各教所信奉的神本为同一。神秘学则主张，一切宗教在其奥秘本质上并无二致，且绝不同意耶和华是神圣之爱与完美一体性的化身。

![illustration](/img/downloaded/2a30a4e502bb838979f13b75e59e0b0c.png)

耶和华不过是一支部落的私神，与他国之神争战，且时时自夸战胜他者……这位神祇不仅司掌战争、谋杀、暴力与掠夺，更充满傲慢、自诩与骄矜——实在难称神圣品质！《旧约》亦记载他如何不断索求血祭以得安抚。对血祭的渴求，正是衡量一个灵体实质与位阶的不变准绳；凡值得敬重或崇拜的存在，断不会提出如此残暴可怖的要求。显然，耶和华忘了自己曾藉摩西向以色列人颁布「不可杀人」的明确诫命；他更乐见无数牲畜，甚至活人，被杀献为血祭。

正如一位神学研究者曾写：

「基督徒若能摆脱这些原始的犹太观念，并在提及神时，遵循其创始者的教导——他总称神为天上的父——教会的许多困扰或可避免。」
