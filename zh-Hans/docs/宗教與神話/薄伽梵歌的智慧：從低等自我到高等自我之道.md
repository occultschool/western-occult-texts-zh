# 《薄伽梵歌》的智慧：从低等自我到高等自我之道
## 探讨克里希纳对自我、欲望与灵性成长的深刻教导

![cover](/img/downloaded/3cd7a1889cbd1a1d19f74cea4ec0ff8b.png)

《薄伽梵歌》中的克里希纳与阿周那

《薄伽梵歌》第六章里，克里希纳总结了当时哲学与思想流派的脉络。他为弟子阿周那剖析诸家观点，指明何者有价值、何者无益，并给予正确的见地。第五节如此说：

人当以高等自我提携低等自我，切莫令高等自我沦落；因高等自我是低等自我之友，而低等自我，恰是自己的敌人。

经文指出两个自我：一为高等，一为低等。我们应当从低等、复杂多元的自我，提升至高等、同质纯粹的自我——佛陀也曾如此教导。克里希纳亦道：勿轻鄙你的两个自我；它们互为友，亦互为敌；敌人非关他人，正在自身。人是三重存在：(一) 灵性个体；(二) 物质属性的聚合，即大自然的幻镜；(三) 个体投映于镜中的影像，那便是人格，是虚妄的「我」。这虚妄的人格之「我」，陷入幻觉之网，佛教称之为「渴爱」，正如《寂静之声》所说，这是蒙蔽我们灵性的尘埃。

心智与物质犹如镜子，「渴爱」这层幻觉的尘埃，令一切苦痛在人间蔓延。克里希纳在《薄伽梵歌》第三章第三十四节告诉阿周那：

每个感官对其对象，皆有固有的爱憎。智者不为这两种欲望所缚，因这是人类之敌。

克里希纳指出，爱与恨、喜与厌、吸引与排斥、好与恶，这些情感的滋生无可避免，源于人具感官与情绪。当人凝视所感知的对象，幻念便萌生，迷惑随之而来。然而，我们是否该避开这些感知对象？克里希纳认为不必。他说：莫受这些对立力量摆布，它们是人人之敌。喜爱与厌恶本是一体两面，犹如电能既可显为光，亦可显为热。因此，他劝诫世人勿陷于贪瞋、吸引排斥、爱恨等对立情绪。下一节，他指明如何超脱这些对立：履行己责、专注本分、遵守自身的法则；莫承担他人之责，莫干涉他人之事，亦莫盲从他人规范。每个人皆能从这段经文中寻得解答：尽心竭力，完成每一个属于自己的责任。

由此浮现一个最令人困惑的难题：不同职责间的冲突。我们该如何妥善处理自身事务？又该如何探寻自我存在的规律？这是根本之问。我们的文明制定了诸多发展计划，也提供了无数离苦免痛之法。因此，必须探究根本原因，并追问：若不履行职责、违背自身法则，为何会招致悲伤与痛苦？当然，若能确切知晓自己的职责与存在法则，生活便有依循。多数人渴望行正道，却总误入歧途。他们正如阿周那，提出了千万人共同的疑问：究竟是何种力量驱使人行错，即使此人决心向善？此问众所关切。然而答案何在？正统信徒说：「神旨。」神秘学徒则言：「业力。」但神旨何在？业力又如何生起？

正如我们所见，佛陀称此力量为「渴爱」，即渴望。克里希纳则称之为「激性」，一种驱使人追逐无数感官对象之力。《薄伽梵歌》指出，「激性」有两面。「欲‧激情‧贪爱」是吸引之力，牵引喜好、爱慕与诸多欲念；另一面「忿‧愤怒‧厌憎」是排斥之力，催生无数厌恶之情。「欲」与「忿」，吸引与排斥，实为同一「欲望」的两面。正如第三章第三十八节所言：

如同火焰被烟雾遮蔽，镜子为铁锈所覆，或如子宫包裹胎儿，宇宙亦被这股欲望笼罩。

此处三个比喻，极富启发，值得深究。

「激性」或「渴爱」如火焰外的烟雾。灵魂本是焰，烟却萦绕不去，非焰所愿。烟浓则焰弱，终至熄灭。此喻真切：若人欲望炽盛、激情淹心，便面临灵性毁灭，即灵性之死。当「渴爱」——那对感官之物的贪求，那向诸般尘缘奔去的冲动——吞噬投生的自我，压倒其心神，断绝与「个体性之星」的连系，人格便在灵性上死去。当今文明，许多人灵性上已死去。故而，我们首要踏出的一步是：不添烟雾，不纵容私好私恶，勿令其遮蔽灵魂的知觉。

上文也提到了镜子比喻。《寂静之声》云：「心智如镜，映照之际亦聚集尘埃。」贾吉先生注解《薄伽梵歌》此句时，将「尘埃」替换为「锈」，更为贴切。尘埃易拭，锈蚀难除；时日既久，愈难清洁。此时须通过化学方式，包括物质元素之科学特性、彼此组合的法则。高等化学即炼金术；其灵性的部分，能化贱金属为黄金。神秘学谓之「化学一分支，借此可得永生」。欲求永生，先须除尽镜上铁锈。

而后又举了一例。胎儿居母腹中；不出子宫，不得成人。我们的灵魂各如胎儿，困于自身物质本性之子宫内；无法看见白昼的光，若未经重生，便有灵性湮灭之危。恶人以烟雾自蔽，犹流产死胎，徒具形骸；灵魂中污秽不除者，亦必毁灭。唯经重生者，成灵性实体，方备永生。此即耶稣训尼哥底母之言：「人若不重生，就不能见神的国。」烟之笼火、尘锈蔽镜、胎裹于宫，皆如人格为欲望「激性」所困。真智慧亦如是，常被虚妄颠倒的知识所遮蔽。

三例各蕴深意，贯穿人类演化全程。烟与火属第一阶段，锈与镜属第二，胎儿及其诞生属第三，即第二度出生。若能掌握古智慧所授真钥，自可领悟玄奥。须深研布拉瓦茨基著作，学以致用，处理烟锈，蜕变崭新的自我。然首要者，须将《薄伽梵歌》教诲铭心，亦须警觉其智慧可能已遭曲解，故追索本义至关紧要。

激性、渴爱、欲望，皆如烟雾：筑屋设炉，须备妥烟囱，方得明亮暖焰。烟囱须导烟出户，首务即在排遣私欲之烟。激情之水当涸，以自律净心。净化为第一步。纯洁有四境：一、心智之洁；二、情感性情之洁，或曰道德之洁；三、言语之洁；四、行为作为之洁。此阶段发展，纯洁为关键。空怀愿望志向无用；每年年初立下誓约亦难成事，因为我们缺乏真知。须深究何谓纯洁，学习如何实践，除恶习，养美德。此阶段当学弃邪念，行善事。须如筑烟囱者，以科学之法掌握技艺，关键在「无私」。须学抑制、约束、泯灭私心。私心乃万恶之源，余孽皆由此生。私与不洁同根，无私与纯洁共生。烟雾污浊，令灵魂窒息；私心亦然，绞杀灵性。

无数微不足道的自私之行，终将积聚成巨力；日常琐碎私念，绵延时日，铸成恒久之私，如镜生锈。若不日日拭镜，尘埃积存；若不勤加磨亮，将在时间流逝下锈蚀。人性中低等本性亦复如是。有二种事物能用来拂尘与抛光：哲学与哲人。纯洁和无私能透过友好的同伴协助发展。学生若自满于无私，也可能在其极善之中生锈。此需灵性炼金术之智；须化无私为「无我」。第一阶段，人成无私与良善；第二阶段，理想升华，须达灵性境界。先是脱离了邪恶自私之雾，转为良善无私；今则须更进一层，践行真炼金术——灵性转化。须自恶趋善，再自善臻于灵性。从自私至无私，再至无我。良友不可缺。第一阶段乃灵魂「校园生活」；第二阶段乃灵魂「家庭生活」；真灵性家庭方带来真灵性情谊与伴侣。寻常生活中，我们区分亲属与友人。更高层面的生活里，友人即是亲属。师者如父母；同门弟子即为友朋兄弟。

当锈迹转化为光辉，变得无私而灵性之时，便是为第二次诞生作准备。要步入第二阶段的掌控，须约束低等本性，决不让杂质沾身，使自己成为纯粹的善；成长与展现，象征著「正义」与「仁慈」渐渐在生命中绽放。待我们以公正律己，以仁慈待人，便踏出了稳健的第一步。《寂静之声》因而要求：对待自身的小我，当如芒果核般坚定；对待他人的苦痛，则如果肉般甜柔。

如此，我们为第三阶段铺好了路。心中住著正义与仁慈的灵魂，那内在的自我宛如胎儿，被自身过去的善业恶业环绕。

持续学习并践行所学的善人，便是实践的哲学家。透过学与行，他遇见真正的同伴，找到灵性家庭的归属。随著心中正义与慈悲的灵魂日益茁壮，往昔业力也迅速消融。藉著越来越多的公正与仁慈之举，他偿还业债，在责任与实践中净化内在。这般善行吸引大师的目光，获得他们携来的正义与慈悲——那些被称为「夜间沉默观察者」的存在。他们在隐密与沉默中观察，不夸功，不扬名，只注视著根植智慧的灵魂，以及由慈悲滋养的怜悯。当伟大大师看见捍卫真理、对抗自私的战士身时，便会滋养那充满正义与仁慈的灵魂。正如太阳赋予莲花生命，从淤泥黏土与死水中，令其绽放光华，伟大人物也如此滋养灵魂。他们净化自己，勤学不倦，公正无私，怀抱仁慈，孕育奉献的潜能。灵魂犹如莲花；莲花在阳光下盛开，虔诚弟子的灵魂也在芬芳与圣洁中舒展。然而，在培养正义与仁慈、展现平和与圣洁的路上，我们切莫忽视伟大哲学与科学所带来的助益。
