# 基督的神秘本质：象征性的灵性原则，而非具体存在
## 深入探讨「基督」在神秘学中的真实含义，超越宗教教条的普世精神原则

![cover](/img/downloaded/fd639a6cd17a6b39fba71be2ba0a0b39.png)

在神秘学的原始真理中，根据智慧大师与布拉瓦茨基的教导，基督并非实体，亦无特定形相。这名字不是弥勒菩萨的别称，也非「大师中的大师」或所谓「世界导师」。更不存在基督的「再临」。事实上，基督从未到来，未曾显现，自然无从再现。

为何如此？「基督」一词源于希腊文「克里斯托斯」(Christos)，仅是一个象征性的名称，指向那遍在宇宙、非人格的灵性原则。

大师 KH 曾言：「但愿那些误入歧途的基督徒明白，真正的基督就在那『神秘之声』之中。耶稣与你我一样，不过是个凡人。他能开悟，凭借的不是拉比或已然堕落的埃及祭司所传的教诲，更多是源于内心的纯净。」

布拉瓦茨基写道：「且看保罗这位勇敢、真诚的人。读他留下的少数原著就会发现，他笔下的『基督』，只用来指『存于人心的神性抽象概念』，可曾赋予此词其他含义？对保罗而言，基督不是一个人，而是一个具体概念。『若有人在基督里，他就是新造的人。』这重生等同启蒙，因为基督即是灵，人的灵。保罗是唯一领会耶稣教诲中深藏玄机的使徒，尽管他从未亲见耶稣。」—《揭开伊西丝的面纱》第二卷，第574页

「克里斯托斯既非教会塑造的基督，也非福音书中的耶稣；它只是一项非人格的原则。」—《卡巴拉和卡巴拉主义者》

「需知，在诺斯替派眼中，克里斯托斯象征著超越人格的原则，是宇宙的阿特曼，也是每个人灵魂中的阿特曼，而非耶稣。」—《秘密教义》第一卷，第132页

「真正的神秘学者绝不接受基督化身为人… 更不接受神被人格化，或由教皇担任『牧师』…」—《论伪神秘学》

「……所谓『基督到来』，是指克里斯托斯在世间的转世中显现，而非耶稣『基督』肉身降临。若要寻觅这基督，不在荒野，不在『密室』，也不在人类建造的任何庙宇圣所；因为这基督——那真正神秘的救主——并非外在人物，而是存于每个人内在的神圣原则。灵魂被尘世欲望钉死，需竭力使其复苏，并将罪恶的肉身深埋于『墓穴』中；若有力量将内心圣地入口那块物质之石移开，复活的基督便将在他心中降临。」—《福音书的神秘性质》

「神秘科学里的基督，是灵性的克里斯托斯。这克里斯托斯原则非关人格，与任何化身为肉体的基督或耶稣全然不同。」—（回应罗卡神父《基督教教义的神秘主义》之脚注）

「我在每封信中都强调，神圣的基督（或克里斯托斯）从未以人身存在。那是亵渎者的幻想，将那普遍而完全非人格的原则具体化了。」—（回应罗卡神父《基督教教义的神秘主义》）

「当拉丁教会将《新约》核心人物具体化，并坚持道成肉身的教义，它便树立了一种学说，与佛教、印度教的神秘主义及希腊诺斯替信条截然相悖。只要教会不愿放弃这两项教义，东方与西方之间，就永远横亘一道深渊。近两千年来，教会对异端与异教徒的血腥迫害笼罩东方诸民族，只为强迫他们接受教会的哲学信仰，认同那些贬抑克里斯托斯原则的教导……

「随著诺斯替教派式微，真正的基督徒也渐渐消失；当今信徒仅在沿用一个他们已不明白的名号。只要此景持续，东西方人便难有共识；宗教思想的融合，终究无望达成。…」

据说卡迦乐季化身降临后（即《启示录》中那位骑白马的「所期待的来者」），黄金时代便会到来。届时人人皆为己师（灵性导师或「牧羊人」），因那神圣的「逻各斯」——无论名为克里希纳、佛陀、索西奥什、荷鲁斯或基督——本是普世法则，将在每个凡人心中统治。因此，所谓共同的「牧羊人」应视作譬喻。基督徒却将这伟大原则拘囿、私有，宣称唯拿撒勒的耶稣独有，再无他人；更将诺斯替的基督观实体化。这般行径，便断了他们与古老智慧追随者间的共通之根。

西方神秘学家所接纳的克里斯托斯，类似于基督教兴起前数世纪的诺斯替派，或吠檀多所理解的克里希纳：他们区分肉身与灵魂，在化身中，是神圣本源赋予了生命。历史上的英雄克里希纳本是凡人，其内在的神圣本源（毗湿奴）却亘古长存；克里希纳身殁后，仍留驻尘世；他并未全然化为毗湿奴，毗湿奴仅收回那属于自身、曾赋予此化身生命的部分，正如祂曾赋予无数其他化身生命。

罗马教会直到二世纪初叶甚至中期，仍笼罩著诺斯替与马吉安派的影响；知名的马吉安迟至一三六年方与教会决裂，塔提安离去更晚。他们为何离开？只因教会斥其为异端。然而，秘传文稿所载的宗派历史，却呈现截然不同的观点。这些文献告诉我们，著名的诺斯替教派之所以脱离，实因无法接受基督道成肉身——此举开启了基督原则的物化历程。与此同时，这也标志著形上寓言的首次蜕变；那寓言本是所有诺斯替兄弟会的核心信条之一……

「当自我与其『阿特曼-克里斯托斯』结合，那名为我执的巨大幻象便告消散，终得觉见圆满真理；此时自我方知，自己从未分离那伟大的『一切』，且与其本为一体。此即涅槃——不过是回返原初的状态罢了。当灵魂困于血肉与物质的牢笼，它甚至遗忘了此状态的样貌或记忆；然则，一旦灵性之光为其揭穿感官的幻戏，它便不再信赖尘世之物，因已学会轻蔑它们；子与父自此结合，灵魂与灵终归合一！人若臻至此般神圣智慧之境，又与任何教会的教条何干？」

——《基督教教义的神秘主义》注释
