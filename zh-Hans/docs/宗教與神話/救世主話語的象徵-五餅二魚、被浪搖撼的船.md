# 救世主话语的象征：「五饼二鱼」与「惊涛中的船」

## 引言

本文将更深入探掘《圣经》蕴藏的智慧，聚焦分析新约福音里的几则寓言。**偶然读到本文的虔诚基督徒，请放心，你们不会损失甚么。** 相反地，或能对自身信仰及其奥秘领会更深。

本篇援引杰拉尔德·梅西（Gerald Massey）的研究成果。他穷尽一生，深入钻研古埃及高等智慧，使之在黑暗中得以重见天日。其著作虽已问世一百二十余年，多年来饱受学者与求道者忽视，但在今人尝试调和基督真谛与教会教条之际，这些文献显得至关重要。

## 福音书的来源

无人确知福音书成书于何时、出自谁手。纵使普遍归名于马太、马可、路加、约翰四人，却不意味真为其亲撰，作者恐永难考实。目前可知，前三卷福音书背后共有四种主要书面来源，同属一类；而约翰福音则另属一类。

前三卷福音的四大来源如下：

1. 马可福音  
2. Q 来源：即一些言论与叙述，学界以德语 “Quelle”（意为「来源」）简称 Q  
3. 路加特有材料，学者标为 “L”  
4. 马太特有材料，标为 “M”

马太福音采用了马可、Q 与 M；路加福音则取马可、Q 与 L。因此马可福音堪称最原始的经文底本。马太与路加抄录后又改动它，只要细心比对者皆可察觉。然这些并非关键。重要的是存在这些福音书，内含教诲确实优美——尽管其中隐藏的深意，少有人觉。

现在，让我们检视几则著名比喻，运用先前文章讨论过的「钥匙」，探看能发现什么。就从「五饼二鱼」的比喻开始。表面看来，这是最不可思议的「神迹」，除了虚构的巫术外，从未有人复现！

## 五饼二鱼

> 「他便吩咐众人坐在草地上，拿著这五个饼、两条鱼，望天祝福，擘开饼，递给门徒，门徒又递给众人。众人都吃，并且吃饱了；他们把剩下的零碎收拾起来，装满了十二个篮子。吃的人，除了妇女孩子，约有五千。」（马太福音 14:19–21）

正如其他文所述，福音叙事在很大程度上源自古埃及的智慧教导，喂饱五千人的比喻亦不例外。埃及《死者之书》中，亡灵宣告：

> 「我是赫里奥波里斯的饼之主。天界的饼是拉之饼；地上的饼是盖布之饼。」

埃及文献明确区分：拉之饼象征灵性智慧，大地之神盖布之饼则指物质知识。圣经寓言同样以饼呈现「灵性智慧」，但相较埃及文本，缺少了「物质知识」之饼。饼的数目时而七、时而五，各有深意。五饼代表地上之粮，即物质知识；七饼则为天界之粮，即灵性智慧。因「五」属「属地之人」的数字，「七」属「属天之人」的数字，意指已解脱的高我（灵魂）。

五与七皆出现于埃及仪式，然马太所载神迹仅见五饼，与埃及文本未合。这或许因为耶稣将隐藏智慧（七）独留选定门徒，仅将表层知识（五）传予众人（五千人）。亦可能由于编纂者无知，对奥义认识未全。

此外，埃及《死者之书》记载，奥西里斯曾在哈索尔的梧桐树下进食。他说：

>「容他自尘世走出。你携来七饼，供我维生，此乃荷鲁斯所制之饼。你已安顿，你已进食。容他向众神呼告寻求寄托。」

福音书里，耶稣亦举目望天，以祝谢过的饼与鱼饱足众人。

此解方为正解，福音书中多处可证。如路加福音有名一节，纳入主祷文：

>「我们日用的饼，日复一日的赐给我们。」（路加福音11:3）

唯有拘泥字句的狂热者，方会将此言解作向上帝索求物资。此乃人之本分，除非我们愿将万物之父视为天上制饼师傅，日日专为人类烘制新饼，别无他务。路加记载：

>「在神国里吃饼的有福了。」（路加福音14:15）

而约翰福音则言：

>「我是从天降下生命的粮；人若吃这饼，必永远存活。我所赐的饼，即我的肉，为世人生命而舍。」（约翰福音6:51）

此言除指灵性食粮外，更可作何解？

约翰所言「肉」，并非指向教会圣餐变体之论，而是真正师者所为：舍己生命，以活他人。

饼尚有诸多他意。福音书载：

>「不宜拿儿女的饼丢给狗吃。」（马太福音15:26）
>「因他们吃饼时不洗手。」（马太福音15:2）

首句明令禁止，勿将圣洁之物付与不配之人。心智洁净者在经中常唤作「孩子」，唯其配得神赐隐藏的「吗哪」——那神秘的、或曰隐藏的智慧。将此智慧普授众人，实属不宜。次句则警诫：以不洁之思、不净之心接近真道，何等危险。

福音书亦屡论有酵与无酵之饼，即是否含发酵剂（如酵母）。正如本站作者探讨藏传佛教时所言，并非一切灵性教导皆善或健康。我们甚至可以说，藏传佛教乃佛陀之饼，掺入了原始喇嘛教的巫咒与迷信。这正是马太警醒我们须：

>「谨慎，防备法利赛人和撒都该人的酵。」（马太福音16:6）

「法利赛人」与「撒都该人」皆象征对真道与真师的歪曲。但「酵」亦可指真道之「内在精神」，正是它令教义饱满活泼。失却此酵，教义便成僵死文字。保罗在致加拉太人书信中提及此理：

>「一点面酵能使全团发起。」（加拉太书5:9）

分饼之后，所余「装满十二个篮子」又是何物？或许是黄道十二宫的数字隐喻，各宫按其特性，所需灵性养分不同，得之方式亦异。

略思饼之诸义后，且再看「鱼」中可觅得何物，以更明此寓言。基督最早的象征之一便是鱼，今人犹见，常贴于基督徒车尾。但我疑心这些贴标之人，是否知晓此象何等古远，又何等普遍。公元前三世纪，巴比伦祭司贝罗索斯曾记述欧涅斯——一位半人半鱼的神话人物，教授人类智慧与诸般技艺科学。

「日头西沉时，」贝罗索斯写道，「此人惯于再跃入海中，于深处度过整夜。」

此事在福音书中忠实再现：耶稣白昼以「耶稣鱼」之身教导众人，入夜则退至加利利湖。在那里，他藉夜间或黎明时分行走水面，彰显其太阳本性。

福音书记载，当门徒在船上时，

>「夜里四更天，耶稣走向他们，行于海上。」（马可福音6:48）

四更天是凌晨三点到六点，恰值黎明前后。此时正是太阳神现身行走水面，或作为鱼人、渔夫自水中步出。

据说欧涅斯与人共处时，并不进食。

>「白昼他常与人言谈，但在那时节不食。」

耶稣亦然。门徒曾劝他就食，耶稣答道：

>「我有食物吃，是你们不知道的。我的食物就是遵行差我来者的旨意，作完祂
的工。」（约翰福音4:34）

此乃对欧涅斯性情的完美复现：他不受食物，时间全用于「捕鱼」——亦即教导人。

如今明了，那五饼二鱼的比喻实是一则寓言，关乎真导师及其训诲。「两条鱼」所指便是导师，亦涵纳救世主的双重本性；其训诲即是「饼」，分与众人，倍增不绝。凡领受真知并了解其义者，皆能分享——或授予门徒，或间接化为善念善行、及对众生的悲悯。如前所见，寓言中的数字颇有深意。「五」是感官之数，亦代表人本身。毕达哥拉斯学派称此数为「平衡」，因其将完满之数十，一分为二。故可断言，「五饼」与食饼的「五千人」，正象征那滋养性灵的真知，广传于人间。另值一提，双鱼星座由两鱼构成，而非一尾；二鱼相背，各朝一方游去。其中象征，留予读者自行探寻。

## 船遭浪颠簸

喂饱五千人后，叙述陡然转至：

> 「那时船在海中，因风不顺，被浪摇撼。」（马太福音 14:24）

字面视之，此转折似显突兀，事实亦然。正如饼与鱼并非指凡俗食物，这「船」、「风」与「海」亦非寻常。我们在先前探讨中指出，经文中几无一词不藏隐义。有时昭然若揭，大多时候却幽微难辨。且这些词语常具双重、三重乃至多重涵义，益增理解的困难，却也平添发掘之趣。欲达此境，便须思量其背后所指之理念，而非拘泥于浅白字面。船为何物，人尽皆知，却少有人驻足细想它所承载的概念。它总含某种「载具」之意。同理，「海」代表一种流动的质体，未必湿润或咸涩。也可指太空中，星辰如舟，游弋其中，履行其玄奥使命，受种种电磁「流」之冲击——即便科学亦称之为「风」。

同样的对应关系，亦可应用于微观世界的人：他乘著一叶「舟」，浮沈于尘世之海，遭逢际遇之风、受自身与他人欲望之浪的颠簸。他也必须操舵，驶过险恶的暗流，避开令舟搁浅的浅滩，以及能使之覆没的致命礁岩。这般思维对于现代人已甚陌生，他们不识符号为何物，只知滥用「商标」，鼓吹那些无人需要、亦无价值之物；而多数人得手之后，便觉索然无味！我们的祖先则不然，他们视周遭世界是一系列符号与寓言，背后隐藏著关乎人世与宇宙的宏大奥秘。

当我们学著运用这崭新的（实则极其古老的）视角，也将重新领悟圣经与其他圣典中潜藏的义理。假以时日，只要耐心持守、勤于实践，外在世界与内心天地便再无不可解之谜——因为真理的语言是符号，而非智性的推演与臆测。持此知见便能带来洞察，「船」与「海」的比喻便清晰起来。若我们无法成为自身之舟的船长，便会继续被无知、疑虑、恐惧与信心匮乏之风「冲击」，直到耶稣——那存在于每个人内心的基督原则——打开门徒的双眼看到真相。
