# 数字「七」的神秘含义
## 为何「七」被视为至圣之数？我们将揭示其多重属性。

![cover](/img/downloaded/02d2b14b13888230d2f14b9a52a2946b.jpg)

Photo by [Waldemar Brandt](https://unsplash.com/@waldemarbrandt67w?utm_source=medium&utm_medium=referral) on [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral)

毕达哥拉斯学派称七为「未婚的处女」：它不像偶数（如二或四）由母亲所生，也不似奇数（如三或五）源于父亲；它直接来自万物之父的顶点——数字一，亦称单子。

印度的《往世书》、埃及的《死者之书》、波斯的《阿维斯陀》、玛雅的《波普尔武》，乃至最后的《圣经》，皆频繁提及数字七。《往世书》记载七位圣人，象征人类七大根种族、七个上下层世界、七座圣城、七个圣岛、七座圣山、七棵圣树，不一而足。

迦勒底-巴比伦神话有「苍穹七神」、「大地七神」与「七恶鬼」。《创世纪》几乎每页可见七的踪影，摩西诸书亦然；《约伯记》、《塔木德》与《卡巴拉》中，此数同样显著。

《启示录》载有七座金灯台、七颗星、七印、致七教会之七封信、七天使所吹七号角、七声雷鸣、七位国王，以及倾自七碗的七灾。《次经》则以隐晦笔触提及人的七种感官：「他们领受并运用主的五种运作；第六处，主赐予理解力；第七处，言语，乃其思想之诠释。」（传道书 17：5）此为圣经唯一论及七种感官之处，后文将再探讨。

希伯来人如此轻易采纳、频繁援引此数，显示他们借鉴了邻族教义，一如基督徒从犹太教汲取诸多信条。于是可见七项圣礼、小亚细亚七教会、七宗罪、七美德（四元德与三超德）等等。至于诺亚所见彩虹，世人多解为天上神迹，以志人神之约；然对于受过训练的神秘主义者眼中，其意更深——它与前述人类七原则、七根种族、七个上层世界，以及七个宇宙显现期紧密相系。毕达哥拉斯亦据此「七」数，构筑其关于七神圣行星的「和声与音乐」学说。

需补充说明：古人采纳七行星学说，并非因为不知其他星体，而是此七行星乃七逻各斯原始而初始之宫位。纵使未来天文学家发现九十颗新行星，亦不会改变此七行星神圣之事实。科学在此领域实属茫然，一如其他许多领域。有些天文学家主张实际上有九大行星（含冥王星），亦有学者认为若计入2005年发现之阋神星，则为十颗。此冰封天体大小近冥王星，轨道却在其外。另有观星者视冥王星过小，不足以称「适格」行星，如此一来，若不计阋神星，则复归九星，甚至八星之数。且让科学家自行解决此争议，这不属本讨论范畴。

毕达哥拉斯学派称「七」为希腊神祇「泰勒斯福罗斯」——因万物经此数方得圆满结局。关于七重神圣层面的秘传教义，源自数千年前的亚特兰蒂斯，毕达哥拉斯正是从其埃及导师处习得此道。这门秘传教义阐明：万物皆受制于七种力量——地上与月下的性质、宇宙的七大力量，并以七重音调演进。这既包含音阶的七个音符，也体现为光谱与棱镜折射出的七色虹彩。

有趣的是，双虹出现时，第二道光谱的颜色序列与第一道正好相反。这印证了一条秘传法则：尘世状态实为上下层世界的倒影，正如镜中映象必然颠倒。

毕达哥拉斯学派将人生划为十个七年阶段：
初七年，乳牙萌发。
次七年，男女具生殖之力。
第三个七年，躯体臻于成熟。
第四个七年，体能达至巅峰。
第五个七年，适婚之时。
第六个七年，心智渐次丰盈。
第七个七年，理性圆熟。
第八个七年，二者皆达完满。
第九个七年，激情转趋平和。
第十个七年，理想生涯终结。

人体亦处处见「七」：躯干显分七部——头、胸、腹、双腿、双臂；内藏七器——胃、肝、心、肺、脾与双肾。头部为主宰，开七窍通外——双目、双耳、双鼻、一口。人身天生七种排泄：泪、涕、涎、精、粪、尿、汗。

现代医学早已证实古说：胎儿七月便可存活。月经周期平均二十八日，恰合四个七天之数，与太阴历同步。人肠全长二十八英尺，亦为四乘七。

毕达哥拉斯学派更称「七」为生命载体，因其涵盖身体与灵魂。他们解释：人由四个低等元素与三个高等元素构成，以顶端三角叠加下方四方形为象征。关于这七重原理的正确序列，可参阅[《金黄星》](/books/golden-star-dr-michaud)第七章。

[《金黄星》](/books/golden-star-dr-michaud)第十一章中，更载有对应的埃及称谓。但须提醒：这些埃及名词包含了障眼法，意在遮蔽真相，防免遭误解滥用。印度圣典将人称为「七叶植物」（Saptaparna），足以证明印度和埃及圣人皆知人身七重构造；此知识皆承袭自亚特兰提斯先祖。

布拉瓦茨基夫人首度向西方引介吠檀多架构，将三个高等元素构成的三角命名为：阿特曼（Atma）、菩提（Buddhi，勿与释迦牟尼佛混淆）与心智（Manas），亦即神圣之灵、灵性之魂、与人类之魂。死亡之际，正是这高等三元组与低等四元组分离。低等四元组则包含：欲体（Kama rupa）、星光体（Linga sarira）、生命能量（Prana）与肉体（Sthula sarira）。

尽管我们现有感官为五——嗅、味、视、触、听，神秘科学指出，未来尚有二感官将会开发，终成七数。此二新增感官属更高层次的心理觉知与灵性洞见，然其普遍展现尚需历经亿万年岁月，此时臆测实属无益。

据说这七种感知对应七重秘传元素，而今仅五者显化：以太、火、风、水、土，分别呼应听、视、触、味、嗅。事实上，第五元素「以太」仍在显现地进程中；当代广播、电视、X光线、资讯科技等发展，不过是以太初步展现。有些神秘主义者推测，那第六与第七元素，或即时间与空间，对应于前述的高阶感知及灵性理解。

众作者中，唯一近乎揭开此谜的，唯米绍博士。欲深究者，可阅[《李文和的教诲》第四章](/books/chapter-four-fortune-teller)。但须警告：你很可能最终结论是，这些奥秘已超乎理解范畴——此判断极为正确。正如李文和所言：「若人类能以肉身之形、尘世之智，真正洞悉时间与空间之本质……则人将与『至一神』、『至一生命』、『万物神圣本源』平等……仅凭此知识，其智慧便已远超天界主宰——亦即诸造物主。」

「七」之数不仅存于人身，更贯穿整个自然。化学元素周期表中，「七」即居主导。若将元素按原子量分组排列（现代表格亦大抵如是），可见其构成以「七」为基的序列——故称「周期表」，意指排列之「周期」或群组。

周期表中，每组的第一、第二个成员与下一组相应成员，在属性上皆紧密呼应。由此可见，就化学变化及其内在本质而言，实系于数值关系；这体现于其周期律正受「七」之数所支配。

这个数字在声与色的世界里同样重要。自然的七个音符、光谱的七种色彩，都如此昭示著。

以上已足以说明，为何七被视为最卓越的神圣数字。正如前文所述，其重要性并非源于《圣经》。古希腊诗人赫西俄德早言：「第七日是神圣之日」，比摩西的安息日概念更早。七亦不囿于一地一国：罗马太阳神米特拉的祭坛前，长燃著七簇经年不灭的圣火；阿拉伯传说有七柄神圣扇子；印度典籍载有七座半岛、七个岛屿、七片海洋与山川河流；迦勒底人信奉七重世界与七灵；赫西俄德与荷马笔下常现七个星座；至于我们曾谈及无穷尽的「七」，更是不胜枚举。

倘若以同样方法剖析七之前的数字，乃至十以内诸数，你必将发现更多深邃意涵。
