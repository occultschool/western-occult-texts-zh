# 古埃及艺术-深意在象征中寻找
## 古埃及艺术充斥著象征、寓言和神话，揭示了埃及人对精神生活的理解。

古埃及艺术善用象征、寓言与神话，使其意蕴深远。因为高层面的灵性生活难以仅凭「逻辑」、「事实」或智性推敲去领会，唯有借助象征、寓言和神话的语言，才能揭示、认知并最终理解其隐藏的真理。简而言之，埃及艺术的锦绣织锦，其实由一连串多层次、且彼此关联的符号构成；或者说，那些精心设计的标志（复合符号），旨在以图像形式表达特定的自然法则与普世原则。要现代人欣赏埃及艺术的难处在于：若不理解（或不曾被教导）这些符号所指含义，于我们便毫无意义。一个现成的例子是美国的美元纸钞。我们或许自以为懂得金字塔与「全视之眼」的象征，但对于开国先贤、设计它的艺术家（以及背后指导者）而言，其含义可能截然不同。换句话说，除非握有解读这些符号的钥匙，否则隐藏的信息将永远是个秘密。

![cover](/img/downloaded/5ad13c6091c0177ccc644f191c372dd0.jpeg)

上图描绘一位埃及音乐教师与学生，画面蕴含多层次的寓言，能从不同角度解读。在古埃及，音乐被尊为至高无上的艺术，因为他们认为，造物主透过话语来创造宇宙。图中学生正为乐器「调音」，在古埃及密教义理中，这象征学习掌控自己的思想与情感。音乐在神庙与学院里亦举足轻重，不只用于宗教仪式与祈祷，也用以教学与治疗。康格里夫曾写：「音乐拥有安抚猛兽的魔力……」；莎士比亚则告诉我们：「在音乐甜美的力量下，野蛮的目光转为谦卑的凝视。」两句话皆点明音乐蕴含善恶之力，甚至可称为「魔力」。柏拉图描述古埃及音乐源远流长，他说这门艺术与诗歌在埃及已存在万年以上；而这些艺术特具启发灵感的性质，唯有神明或类神之人方能创造。当然，或许有人会说，柏拉图只是异教徒，他哪能真懂艺术！

象征主义在埃及秘仪学校里是一门精确科学，遵循严格规范。这些符号皆从大自然中精挑细选，用以表达或体现某种自然法则、力量、原则，抑或物质与灵性上的功能与概念。在古埃及艺术中反复出现，「眼睛」即是一例。无论是荷鲁斯之眼、拉之眼，或其他神祇的眼睛，基本含义总离不开灵视、觉悟与创造。因此，这些符号能同时代表太阳与月亮（这两只「眼睛」照亮世间，一者直射，一者反射），或指向灵性上的觉醒，乃至不同层面的创造活动。这些符号后来更演变为埃及语中的「阿里」，意即「做」、「制造」、「创造」。

古埃及人常以特定动物来象征概念，例如秃鹫（母性）、甲虫（蜕变与进化）、隼或鹰（智慧）。用秃鹫象征母性或许显得奇特，但动物学家深知，这种鸟对雏鸟的呵护往往令人动容。若曾见过猎鹰在高空自在盘旋，或如箭矢般俯冲捕猎，便不会怀疑古埃及人为何选其象征人类最高的灵性能力。但切莫像某些浅薄的埃及学家，误以为古埃及人相信他们的神长著鸟兽头颅。正如我们所见，这些都是复合符号或标志，旨在表现特定神祇的功能或活动。因此，相较于前述较单纯的符号，复合符号的解读更为复杂，是由数个不同符号组合而成的标志。众所周知，吉萨的人面狮身像可谓这类复合动物的典范，它将兽、人、神揉合成一个谜样形象，单凭智性永远难以「破解」。

![illustration](/img/downloaded/a628d1eab330c0f7221381b240819661.jpeg)

现代人最难跨越的障碍，是我们早已习惯直接论述，不再以象征思考；再加上古埃及艺术所承载的某些原则——如「灵魂」、「思想」、「生命」、「光」——其内涵今日已大不相同。每个符号都可能因时空背景与原始概念而有多重诠释，理解因而愈显艰涩。埃及历史漫长，符号标志历经修改变迁，难以为每个象征给出放诸四海皆准的定义。这正是当代观者面对古埃及艺术时的困境，也是为何鲜有埃及学者能真正深入作品核心。这些艺术，无论绘画、珠宝、雕塑或神圣建筑，皆蕴藏丰繁的象征。要读懂其中密码，非得经年累月钻研大自然与灵性的法则不可。尽管近几个世纪科学大步迈进，对此类法则的探索却才刚起步。如今，哪所大学、哪间学校传授这套丰厚却近乎失传的象征语言？坦白说，几近于无。然而，理解并非不可能，下文三个简例便足以说明。

## 古埃及神凯布利的象征意义

![illustration](/img/downloaded/235bb48939bb9f0184c8a05b01698d53.jpeg)

埃及神凯布利常被描绘为圣甲虫，托举太阳圆盘。他代表创造与转化的至高抽象原则。其名源于埃及词 Khepher（亦作 kheper、Khepri 等），意为「转变」、「转化」与「进化」。这符号以谦卑的圣甲虫滚动粪球为象征；而三个词义，正揭开符号的关键。

「粪球」在此喻指宇宙及其一切内涵。这物件看似不雅，却涵盖了最微小的细菌，乃至太空中孕育新星系的庞然气体云。一切物质形体，无不历经生、长、灭，而后以另一形态重生。其中更隐藏一层深意：透过转化与嬗变，宇宙终将进化为更高、更纯粹的存在。若读者涉猎古埃及乃至他国秘传教义，便知这转化过程无所不在。我们自己，从此刻到下一刻，已非原先那人。即便阅读此文当下，你也在变——身体、心智、灵，虽则细微难察。凯布利的法则正在运作：改造、转化、进化其造物，直至圆满。

## 古埃及神的象征意义——拉神

![illustration](/img/downloaded/71909dd8b2dd93fe6915dbd6a968e7fc.jpeg)

拉神或许是古埃及众神中最知名，却也最难参透的一位。这不意外：他受崇拜的历史悠久，数千年间埃及宗教历经多次变革。可以说，拉是最古老的神祇之一，自古在赫利奥波利斯受尊崇，以多样形象代表太阳的生命与光明。尽管后来他与「阿门—拉」及其他神祇融合，拉从未失去太阳神的至高地位，几乎每位法老皆自称太阳神后裔。而太阳神隐含的深意，成了埃及与希腊秘传教义的重要基石。

《圣经》中「喂饱五千人」的寓言，灵感很可能源自拉的神话。因为数千年前的埃及《死者之书》里，死者声言：「我是安努（即赫利奥波利斯）的饼之主。我天上的饼是拉的饼；我地上的饼是盖布的饼。」这寓言与《圣经》比喻略有不同——《圣经》将灵性真理化为现实事件，因而转移了寓言的重心；埃及原始教义则清楚区分物质知识（地神盖布的饼）与灵性智慧（拉神的饼）。这些饼有时七个，有时五个，两数字皆具重大象征：五指凡俗世界，七喻天界，或指解脱的高我与灵魂。此例足见拉神象征的丰厚层次。

## 古埃及女神伊西斯的象征意义

![illustration](/img/downloaded/dccb28a3af9a2c634fac07728ff09664.jpeg)

伊西斯（古埃及作「AST」）在漫长岁月中，呈现神格的多重形貌。自远古起，她便与母性相系——既是人间生母，亦是滋养万物的阴柔本源（自然之母），更可化为创世所必需的最高母性原则。她亦是奥西里斯（古埃及作「ASAR」）之妻与姊妹，二人与其子荷鲁斯，共为埃及人心中理想家庭的典范。

然而这位女神另有幽邃面目，唯经秘仪启蒙者能窥。普鲁塔克曾如此记述：「我，伊西斯，是往昔、此刻与未来的一切；凡夫不得揭我面纱。」秘仪确能化凡为圣，超凡者再无真理被隐蔽。希腊哲人所得智慧，部分正源于此。伊西斯那袭母性斗篷，是为面纱之外之人披上；面纱内所见，却是赤裸真相。正如柏拉图所言：「不可泄露于外。」尽管如此，伊西斯仍是古埃及最受爱戴的女神，那典范的母性形象，后来亦为基督教会所承纳。

最早一批圣母像自埃及携至罗马，其中可见端倪：玛利亚（或曰伊西斯）怀中所抱，非幼年基督，而是荷鲁斯。而基督生平与教诲，其间亦依稀存有荷鲁斯的身影与使命。
