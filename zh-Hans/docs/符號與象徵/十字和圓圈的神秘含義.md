# 十字和圆圈的神秘含义
## 安卡十字的圆圈象征阴性，意谓死者将在灵性的真实生命中重生。

![cover](/img/downloaded/ef4704ecf5dc0f90401a9a7bf1a49030.png)

本文探讨两个最古老普遍的符号：十字与圆圈。  
东西方智者皆指出，十字与那神圣无垠的圆圈共同存在，标志著宇宙本质的初次分化，即灵（竖条）与物质（横条）的结合。因此，十字常镌于圆圈之内。若将大写字母 T（Tau）之形，置于圆圈下方，如同占星符号中的金星，则表示 T 从宇宙蛋中显化的结果。

古人眼中，圆总带有神圣而神秘的性质。哲人觉察到，可见的圆周处处存在，不可见的圆心却不在任何地方，完美象征了神可见与不可见的无限性合而为一。柏拉图在《克拉底鲁篇》中也阐释这种合二为一，并指出「神」（Theos）之名同样适用于二者。此词源自动词「移动」（theein），他将天体运行与神相关联。

世界各民族流传的各种螺旋符号，实则皆衍伸自这个伟大符号，而其奥义从未有人说得比布拉瓦茨基在《秘密教义》中更清晰：

「眼前一份古老手稿，经由某种特殊未知的处理，使其棕榈叶材质不浸水、不燃火、不透气。首页是纯白圆盘，衬著漆黑背景；次页仍是同一圆盘，中心却多了一点。首页象征永恒中的太阳系（我们的太阳系），描绘其沉睡能量尚未苏醒的状态；圆心一点则预示分化的黎明。这点是古宇宙论中『世界蛋』的第一个点，这胚芽终将化生一切，成为那无垠而周期循环的太阳系。故此，圆圈象征神圣的统一，万物从中涌出，亦终将回归。」布拉瓦茨基接著告诉我们，圆周代表那抽象、永不可辨的「临在」，圆面则代表「宇宙灵魂」，二者实为一体。圆盘表面纯白而背景墨黑，更表明圆盘之外的一切，永远超出人类有限的理解。

同一概念在《约翰福音》中亦有精妙表述：「光照在黑暗里，黑暗却不理解。」（约翰福音1:5）这话有双重意涵：其一，无论我们短暂的物质之光多么明亮，都无法认识真正的「光」；因为真正的「光」对我们的感官与有限心智而言，恰是黑暗。其二，真正的光也不理解现象世界，现象世界是由显化之圆所包围或限制的世界，因而只是一场幻觉；尽管对我们来说，尘世是如此的真实。

![illustration](/img/downloaded/45ef3cffd62b8a75a252c6200f3c6a71.jpeg)

布拉瓦茨基所提及的手稿中，还收有几幅插图；每一幅皆精确描绘宇宙最初显现与成形的过程。首先见到的是先前所述，黑底上的素白圆盘。接著圆圈中出现一点，这是永恒无限的「临在」在空间中的初次分化。我们甚至可称此点为「生命」，因为它乃万物显化的胚种。各类细胞在分化初期，同样包含这样一个「点」或胚种，足证普遍的赫尔墨斯法则：「下如其上，上如其下。」此过程既适用于巨观世界（宇宙的显现），也适用于地球上的微观世界——人类、动物与植物。

第三幅插图中，点延伸为一条直径，将圆圈两侧平分：在涵容一切的绝对无限中，此刻象征著神圣无瑕的母性原则。这里再次对应微观世界里的细胞分裂。随后横线与垂直线相交，圆圈遂化为宇宙十字。当圆周彻底隐去，只余十字臂膀指向四方，我们便抵达现象世界——也就是我们所居的可见物质层面。

因此，第三幅插图中，被横线剖分的圆，可视为创造性大自然的初现——虽仍属被动，故而为阴。人类对生育最初的蒙胧感知亦偏阴性，因对母亲的认识早于父亲；是以远古神话中，女神往往比男神更显神圣。时至今日，我们仍惯称大自然为「她」，认可其孕育果实的阴柔本质，与阳性的创造原则相对。当这道横线上添一竖线，便成最古老的T形符号（Tau），亦是基督教十字架的源起；象征灵降入物质，即所谓「人类的堕落」。

正如《地球与宇宙的神秘演化周期》一文所述，万物在「白昼」与「黑夜」之间出现隐没，亦即静止与活动的交替之中。这永恒不息的动态，既不断生成，亦无所不在，正适合以圆来象征。圆同时能避免将神人格化——以圆喻神，便脱离所有具体形体。这是一场无尽的演化，历经千万年跋涉，终又回归绝对的至一。唯有次级神祇负责实际创造，古人方将其拟为人形，如埃及的拉神：祂代表我们在[《神秘太阳》](/zh-Hans/docs/sun-mystery-science-vs-mysticism)中探讨的太阳之力。

生命之符安卡（Ankh），是十字与圆最神圣的结合。法老惯持于右手，已抵达彼岸阿门提的亡者亦如是，即星光界高等层面。安卡顶端的圆环，进一步呼应象形文字「Ru」，意为门、闸、出口。这标示了太阳在天空北区的诞生之地，冬至时分，太阳由此重生。故安卡的圆环（Ru）实为阴性表征，寓意死者在灵性实相中复活。

布拉瓦茨基夫人在《秘密教义》中指出，安卡的圆环（Ru）在塞普勒斯语与科普特语中作Ro，在斯拉夫及俄文字母为R，拉丁文则为P；此形亦延续于希腊十字，仅多一横杠。

安卡十字不只属于埃及。印度教中它以「帕萨」（pasa）之名存在，表现为湿婆右后臂（共有四臂）所持的绳索。此神常示现苦行相，宛如大瑜珈士，而其第三只眼——即前述圆环（Ru）——以另一形态直立于T形十字上。持帕萨时，多以食指与拇指交叉相扣。

帕萨于此具双重意涵，一如埃及安卡。湿婆在《梨俱吠陀》中被称作咆哮者「楼陀罗」，兼具慈悯与凶暴，既是医者亦为毁灭之神。《毗湿奴往世书》记载，祂自梵天额间涌出，随后分化男女。祂是楼陀罗族与马鲁特族之父，其子半数聪慧温和，半数黝黑凶猛。在《吠陀》体系中，湿婆既是渴望回归纯净神性的高我，亦是被困于尘躯的灵性本体，受炽情所驱而成为「怒吼者」与「可怖之人」。

此处以寓言描绘人之高我与小我，我们多篇文章皆有论及。圆环或圆象征高我，物质的十字则代表小我。故而印度苦行者或瑜珈士手中的十字形绳索（帕萨），喻示通往天界的「窄门」，即须克服由十字所表征的小我。因此，埃及安卡与印度帕萨实藏同义。这才是真正的十字架：在启蒙者穿越「窄门」前，所有俗世欲念皆须钉于其上；一旦内在之人跨过门槛，这道狭窄的圆环便会扩展为无垠之圆。

十字四臂指向四方：上下左右。早在欧洲人识得此符之前，远古启蒙者已深谙其义，且此符号遍存全球。十字之谜亦载于赫尔墨斯《翠玉录》第七、第八节铭文，今人对其理解之浅，犹如六至八世纪该文本初现之时。

「分离土与火，区别精微与粗重，须审慎明辨。以绝顶智慧自地升天，再自天返地，汇集世界一切力量，所有晦暗必将远离。」

如前所述，十字竖线象征灵，横线代表物质；因而亦喻示高我与小我的演化，无论经左道或右途（横杠），升华或沉降（竖杠），分途或并进。正因如此，神秘主义者视十字为开启万能之钥，揭示一切科学，无论物质或灵性。

十字象征人类存在：生命之圆环绕著十字四端，依次代表出生、在世、死亡与永恒。哲学上，几何化的神在横竖两线的交点处，划分了高低宽广，其中蕴藏的深邃奥义，足以著述连篇。

有些人以为十字专属基督象征，然约翰．坦普尔在探讨复活节深意之文中阐明：直到耶稣身后数百年，十字架方为教会采用。实则十字渊源古远，遍布各处，其神秘性更显幽邃——出现在复活节岛巨像、古埃及遗迹、中亚岩刻、乃至前基督教时期北欧与中美洲石上所镌之「T」形与「卍」字，在在见证其踪。简言之，十字无所不在。

诚如坦普尔所言，唯有透过启蒙仪式取得钥匙，方能解开十字之谜。

候选者在通过诸般试炼后，将被束缚于十字或T形的卧榻，而非遭钉住。此状态持续三个昼夜，其间，候选者之灵魂（高我）据说会与「众神」交感，并降至冥府，或称冥界、阿门提、下界，端看此仪式举行于希腊、埃及或印度而定。候选者之灵魂会在冥府济渡无形存有，无论是人之灵魄或元素精灵。其身体则留于神殿地室或地下洞窟。在埃及，身体会躺在大金字塔王厅石棺，第三夜则抬至长廊入口。待晨光初绽，一缕朝阳恰照著候选者的脸，他恍恍惚惚的苏醒，继而祭司会代表奥西里斯与智慧神托特，施以启蒙。

若疑此说真伪，可参考古埃及庙墙与陵墓所绘神圣启蒙仪式。仅举一例：在阿斯旺水坝兴建时，有座迁往上埃及菲莱岛之伊希斯神殿，便存启蒙场景。有两位二祭司，一具鹰首（象征太阳神拉），一作鹮头（代表智慧与神秘学之神托特），立于初启蒙者躯体上方。二人自其顶倾注双流之水（生命之水与新生之泉）交错成十字形，并以小型安卡象征之。

晨光照射候选者头顶之际（其恍惚之身置于木制T架以承天光），喻示候选者觉醒（此时已成启蒙之人）。启蒙祭司随即现身，诵读圣礼言辞，以点化新生之人；话语表面上是称颂「太阳—奥西里斯」，实则是指内在灵性太阳。凡探查埃及法老陵墓，乃至众多大祭司、女祭司及其他显贵之墓者，皆可见各式十字，形貌各异，方位不同。

早期狂热基督徒见此十字竟随意取用，甚至以为是后世教徒暗藏其中！此中奥秘，只有曾入希腊秘仪者知悉，如奥利金及后的亚历山大克莱门，但他们明智的保持缄默。

早在十字采为基督象征之前，就已是启蒙者与入门者间的暗号：入门者被称作「苦伤之人」。十九世纪法国秘术家埃利法斯．莱维于《高等魔术教规》中，述及希伯来卡巴拉学者运用十字之法。

其文曰：「基督徒所用十字标志，非其独有。卡巴拉亦用之，表示元素相克与四元平衡。」翻阅《主祷文》的秘传诗句可知，画十字之法本有二途，其意涵至少有两种：一种为「祭司—启蒙者」所知，另一种是给入门者与凡俗。启蒙者以手触额，念「你」；接著念「属于」；继而按心口，念「王国」；再抚左肩，念「正义」；后触右肩，念「与仁慈」；终双手相合，补道「于万有循环之期……」

米绍博士于[《金黄星》](/books/golden-star-dr-michaud)中言：「十字的标志流贯巨力。」若以真诚专注、虔敬为之，确然如是。文末最后引用米绍博士所述，如何藉十字造福己人：「当我们依次触额、太阳神经丛、左肩，终至右肩，念及一切神圣使者皆自天父降临尘世。其后由尘世降至下层星光界（左径），复经右径归返天父，永驻荣光。」

「勤用此标记是好的，因为人易忘了居于神性理想世界的护佑，此举亦能驱散一切不健康之念与影响，尤当愠怒、自私或其他恶念诱袭之时。若心能恒驻于高等层面，便毋需此提醒，然此类人稀少。凡夫俗子从此小小的仪式动作中，得到许多帮助，可祛诸般不快之扰，存留一切美善。」

光是十字与圆圈这两个符号，便蕴含如此深意。若连这两个符号都难以穷尽，便不难明白，何以我们总在文章里一再申说：象征科学这门学问，需要人一生的光阴来学习。
