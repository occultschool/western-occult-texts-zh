# 宇宙意识
## 既忘不了，也否认不了；无论外在的记忆或内在的烙印，都抹不去它。

![cover](/img/downloaded/3daa8790f737eacbcfec6445e0ebb696.jpg)

Photo by [Greg Rakozy](https://unsplash.com/@grakozy?utm_source=medium&utm_medium=referral) on [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral)

许多哲学家与神秘学者，都曾著墨宇宙意识。

「宇宙意识」既是对宇宙真理的觉知，也是宇宙本身的意识，宇宙意识也即是神。人若拥有这种意识，便能同时觉察万事万物。他知晓过去、现在、未来浑然一体。因为一切皆涵容于神的意识中，也因此蕴藏在人的灵魂里。

只要懂得揭开面纱，或得到启引，便能使人的意识契合于神。一旦掀开那层面纱，望见内在神庙的堂奥，我们便在外在意识与内在意识之间，建立了永恒的联系——在小我与高我之间，在高我与神圣灵魂之间。那神圣的一刻，我们触及了真理。既忘不了，也否认不了；无论外在的记忆或内在的烙印，都抹不去它。

对神秘学学徒而言，进行这种神圣接触至关重要，因为他将在血肉之躯中体证灵性真理。即便死后暂居星光界，等待转世，这份体悟也将随身。正如我们所知，当脱离尘世躯体的枷锁时，唯有这意识能随身携带。其余一切皆如影逝散：不过是幻象，虚假而无实质。

这份对真理的认识，须透过冥想与观照达成，从外在世界抽离，转入内在世界。让灵性脱离低层区域，向上升华；背对阴影，将内在目光凝聚于光明。这种感知无法借由任何思辨过程获得。纵然纯粹思维的感知模式已超越单纯智性，仍只能对灵魂的高等能力产生模糊的理解。

宇宙意识唯独透过灵视的感知方能抵达，超越一切肉身限制，透过与灵魂契合，让高我的意识适应更高境界的状态。而灵魂又与高等境界的所有灵魂直接相通——它们本是一体！这便是宇宙意识。达到它，甚至仅仅追求它，便是人类对神最深的敬意。

但在触及宇宙意识之前，必须先充分觉知并认识我们作为「人类自我」的幻象。赫尔墨斯箴言早已告诫：「认识自己者，便知自身所蕴含的一切。」

然而，当注意力投向神圣事物时，仍须确切履行现实生活的职责。我们被安置于肉身的圣殿，本为执行某些尘世任务，并服务同胞。

人若只将思想欲望全然紧系物质世界、并在其中觅得昏暗乐趣，将难以理解此真理。他们只见到结果，却看不见生成万物的原因。他们不知道，从「至一实相」映现出的灵，再从灵映现出的物质——却误把这重重梦境视为现实。事实上，他们认定的现实并非真实，只是感官的体现，在世俗脑海中描摹出那实相的倒影，而客观存在却始终隐藏真理。

那么，如何从这些世俗幻象中抽身？如何熄灭欲望、止息渴求，从而抵达深邃的平和——这一切追求神圣奥秘者的目标？唯有放下贪婪与憎恨，转向爱与服务，方能揭开帷幕，显露真理。而心思粗鄙之人，在尘世的幽暗里活动，栖居于形体的世间，永远无法看见它。但那些曾见识光明的人，必须传递这讯息。因为有些人早已准备好，只待一句亲切的话语点醒。他们只需一只疗愈之手，轻触并冷却那疼痛的双眼，便能看见那敞开的永恒之门。

寻常的自然推理永远无法产出宇宙意识。神的本质属光的领域，任何从理性衍生的猜度都无法渗透那片天界之境。哲学家康德点破了此点，但其智慧的真义过于隐晦，能理解其崇高教诲者寥寥无几。主要缘故在于，多数学子忽视——或者说看不见——智性推理与灵感慧光之间的区别。

物质终将腐朽，因其为物；灵性却属神圣，因此永恒纯净。

此即康德所言的「纯粹」理性——与人灵魂中那近乎神性的禀赋相比，凡俗思维不过是超验真理的障碍。这真理只向信者显现，因唯有信心能开启内在的感知。

可以说，宇宙意识是一种超越粗钝物质的延展。我们或凭七种灵性感官之一触及它，或通过锻炼，或生来便是灵视者。宇宙意识，意味著更进一步与宇宙万物及众生「同在」。

实则只存在一种绝对意识，无论其名为宇宙意识或其他。它是至一的实相，绝对的存在，至一的生命。那是「本体」，与有制约的存在毫无瓜葛；而有意识的存在，仅是一个有制约的符号。有制约的存在属物质界，受变迁、谬误与幻象所困。意识与物质、主体与客体的二元，应视为「绝对者」的两个符号或面向，而非独立的现实。

灵魂愈接近「绝对者」，便愈贴近真正的宇宙意识，愈能远离物质幻象。在这通往绝对意识的途程中，将历经若干阶段。未进化的心智将在星光界中辗转千生万世，直到灵性知觉逐渐成熟。在投生下一世或更高层面的尘世生活前，它会在星光界居留一段时日，为来生预备。这过程需时百万年。于每一层面，它将显现为不同的身体样貌与形状。随著上升，其形体将愈发灵性化，如灵光缭绕，实则是人类形貌的延伸。

此后，意识持续扩展。及至最高层面，「灵魂—心智」便进入至福与智慧的境地：它散发著生命，舒展、充盈，臻于完美高妙的境界。那境界不可言说，只能亲证，因无一语能述其性质，或描摹其中所感。即便亲历宇宙意识者试图诉说，闻者也仅能听见字句，却无从领会——除非他们自身也曾经验。

由此我们明白：在肉身中证得宇宙意识的同时，也将对人世有通透的了悟。如此，我们方能为值得者引路，领他们回归至一的实相、至一的真理。愿读到这里的你，在神所赐的美好时光中，得享这份宏大的恩典。
