# 梦到底是什么？- 简易清醒梦指南，迈向开悟的第一步
## *人的初次开悟，始于昏睡。梦中，一切知识皆已开启。*

![封面图片](/img/downloaded/abdb6fe5d88c75ffa46367d7793e3a6b.jpg)

摄影：[R Mo](https://unsplash.com/@mooo3721?utm_source=medium&utm_medium=referral) 于 [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral)

梦有多种。除了常见的「消化梦」（譬如乳酪吃多了），简略划分，还有大脑梦、记忆梦、警告与预感之梦，以及当身体沉睡时，高我经历某些事情后，所产生的扭曲回忆梦。身为神秘学的学徒，我们关注的是接下来要介绍的梦。

> *人的初次开悟，是在昏睡状态。梦中，一切人类知识皆已开启；一道微弱的桥悬于无垠虚空，连结现世的灵性与更高世界的灵性。 — 布尔韦*‧*莱顿《札诺尼》*

布尔韦‧莱顿笔下的「桥」，在印度教中称为「安塔卡拉纳」。这是一座假想的桥，连接高我与小我；换言之，它横亘在我们不朽的灵性自我（高我）与物质的大脑感官（小我）之间。当我们的心思专注于灵性而非物质时，这座「微弱的桥」便得以加固。亦可将「安塔卡拉纳」比作两台无线电设备，彼此通讯。需将其中一台（小我）调至与另一台（高我）相同的高频，否则通讯杂乱无章，甚至完全断绝。

布拉瓦茨基将梦划为七类。

1.  **预知梦**：高我在记忆中留下的印象，通常简洁清晰。或是一句话语，或是一幕即将发生的事件。
2.  **寓言梦**：现实朦胧的片段被大脑捕捉，再经幻想扭曲而成。这类梦往往半真半假。
3.  **黑白魔法师植入之梦**：亦可能由催眠师藉强大精神力量发出，意在驱使我们遵从其意志。
4.  **前世梦**：梦见属于前尘往事的事件。
5.  **警告梦**：用以警示平日疏于戒备之人。
6.  **困惑之梦**：成因纷杂。
7.  **纯粹幻想与混乱画面之梦**：源于消化不良、精神困扰或外界干扰。

我们大致认同此一分类，尽管如布拉瓦茨基所言，这七类尚可细分，并对应意识的不同层面。但我们关心的仅是前五类，尤其第一、第二与第四类——预知梦、寓言梦与前世梦。它们对灵性成长确有裨益。

寓言梦时常予人深刻印象，无论是揭示自身处境，或激荡内心强烈情感，往往蕴含重要启示，有待我们自行发掘。当一个梦境及其变体在数日甚至数月内反复出现，便可确信其中必藏讯息。这是较高灵魂领域的导师，与尘世中心扉敞开的弟子沟通的方式之一。

关于第一与第四类梦（预知梦与前世梦）的运作，布拉瓦茨基给出一个精妙的譬喻。

> *「可将真正的自我（我们的高我）比作囚犯，而身体的人格（我们的小我）则是狱卒。狱卒若沉睡，囚犯便越狱，至少能脱离肉体牢笼。若狱卒半梦半醒，倚窗打盹，便只能偶尔瞥见囚犯掠过窗边的身影。这样的狱卒，岂能知晓囚犯的动静？更遑论知道其真正举止，尤其是内心所想？」*

答案是不能。对常人而言，很少人能清晰知晓身体沉睡时高我离体经历了什么，极为罕见。即便普通的神秘学学生，亦复如是。

置身这等梦境，我们对高我的体验往往只余残影，宛若帆布篷上映出的淡薄轮廓。醒来时只见光影绰绰，自以为作过梦、历过事，实则只是高我离体经验的浮光掠影。待到神智全然清明，回忆便每分每秒地扭曲变形，遭小我与肉体大脑投射的意象侵染、杂糅。这些意象又借联想之力，蔓生出种种念头，最终与高我短暂解脱时所获的真实体悟，几乎失却所有关联。这便是为何多数梦境在小我清醒时，总显得纷乱无序。

## 清醒梦及其实现之道

若要培植「清醒梦」这般能力，便需强化两个「我」之间的桥梁，即前文所提的「安塔卡拉纳」。收窄桥梁间隙的方法之一，是在床头备妥笔记本与笔。醒后立刻记下所有能忆起的梦——无论多么朦胧、零碎甚至荒诞。之所以强调「立刻」，是因为醒转刹那，诸多记忆便已逸散。若能持之以恒数星期，你会惊觉自己记住的梦竟多了许多。随著练习日渐规律，不仅梦境会愈发清晰，你更将辨识出：究竟是哪个意识正在「造梦」。或许还会接收到某些讯息、符号或知识，点亮你的灵性修习，照见其中隐藏的意义。这些讯息会以何种形式显现，无人能代你预言——正如前述，每个人皆活在自己的心智世界，所思所想从无雷同。

这项技术行之有年，成千上万的男女早已亲身验证其成效。
