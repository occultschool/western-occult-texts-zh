# 预言、命运与自由意志的真相揭秘

## 探讨预言如何运作、自由意志是否存在，以及高等自我与命运链的理性解释

![封面](/img/downloaded/free-will.jpg)

我们将在文章中探讨预言如何运作、自由意志与宿命论等问题。

你很可能会怀疑自由意志是否存在，若有人能预测未来，那么不就没有自由意志了？我们生活中的一切是否都是注定的，所以无论我们做什么，命运都如期发生？许多哲学家、神职人员和科学家，都曾与此难题搏斗一番，但没有得出满意的结论。

人类的理解力在大部分的时候是很薄弱的，尤其是试图用无助的智力来推理宇宙奥秘之时。若仅凭智力，无论多么发达，都不可能得出任何事情的真相。在讨论命运与自由意志表面上的矛盾之前，我们先探讨预言、占卜和算命实际上是如何运作。某人突然瞥见即将发生之事时，到底发生什么事 ? 其实道理非常简单，知识分子反看不出来，因为他们只依靠冷酷的推论来得出自认为的真相。

意识对于科学和宗教都是一个谜。如果你怀疑此说法，不妨试著感受一下大脑是何部分在「思考」。你会发现你做不到。原因是大脑不会思考，只处理从低等自我或高等自我收到的信息。这两个自我无法用目前科学所知的方法来检测、测量或分析。事实上，科学否认大脑之外有任何意识，尽管已有大量反例佐证，特别是关于「濒死体验」的研究。

濒死体验发生在临床上被认为「死亡」的病人，最常见的特征是在体外经历意识的扩展。这证明了大脑会限制意识，当这种阻碍被消除或瘫痪时，比如在深度冥想时，精神活动就会增强和扩大。日常清醒的思考可比作一部智能手机的正常计算能力，而更高的意识则比作一百万台超级计算机共同运作。若你有任何意识延伸的个人经验，你就会知道这是真的。

这适用于人的高等自我，而非低等自我；当低等自我不再受高等自我指导，就与会思考的动物没有区别。高等自我的心灵能力远超正常清醒的意识，能以极高速度和敏锐度理解物质事件与情况，远超单纯智力推理。因此，预见未来的原理是高等自我一系列极速计算（低等自我一般不知道此过程），计算出一个人的行为将如何不可避免地导致某些结果。

事实上，许多预测之所以没有实现，是因为很少人有能力将意识集中到很远的未来，或延长一段时间。 还有一个更复杂的情况是，当高等自我从身体中解放出来时，低等自我和大脑没有正确地接收高等自我的记忆，并忠实地保存。因此预测并没有这么简单，尽管其法则是直接明了的。

世界上存在著一个普遍的链条，一环扣一环，连接著人类的思想和行为。此链条的近端是「现在」；而「未来」是一系列更进一步的环节。大多数人只看到「现在」环节，无法看到更远。但有些时候，意识中会闪现出内在的光芒，此时会看到下一个环节，下下一个环节，甚至是沿著命运之链的遥远环节； 然后我们会惊讶地倒抽一口气。若将所见之事告诉他人，并且活得足够久，那么人们就会视此人为真正的先知。 你或许认为这是宿命论，但事实并非如此。

有一个任何人都可以尝试的简单实验能作为验证。 这并非关于未来或预测的实验，但能展示在地球上看到或经历的一切，都是星光界情形的反映。 

放松你的头脑和身体，尽量不要想任何事情。然后试著在不看钟的情况下说出现在的确切时间。要做到这一点，就是说出脑子里突然出现的第一个数字。你很快会发现，说出正确的次数比错误的次数多。随著进一步的练习，便能在任何地方说出时间，甚至是几秒，且不用看时钟或手表。这并非「魔法」，而只是上述的自然法则起作用。

也可以用一副扑克牌来做类似的实验。将牌面朝下放在桌子上，试著猜出这副牌中第一张牌的花色和数字。在满意自己的成功后，可以改猜第三张牌或第五张牌。通过练习，最终能够准确地猜出任何牌的数字。现在你知道一些舞台「魔术师」是如何表演他们的「戏法」的。

那关于诅咒呢？许多人都诅咒过别人，且诅咒确实生效，有时持续好几代。这也是宿命吗？如我们所说的，不是。 为什么某人要诅咒另一个人？ 因为被诅咒者定是在某方面伤害了诅咒者，不管是有意的还是无意的、自愿的还是非自愿的。如果被诅咒者相信此诅咒，那么诅咒就会生效。 若一个家庭一代代受诅咒，且他们认同这些诅咒，那就会延续数年甚至数世纪。 

首先发出诅咒的人，预见到了错误行为将带来不可避免的惩罚。 在念出诅咒的时候，心灵会在高度兴奋的状态下，瞬间被启发，能在一瞬间追溯错误行为导致的整个事件过程；一环扣一环地看到必将发生之事。咨询神谕也是同样原理；不管神谕的来源是男人还是女人，或几根棍子、一些茶叶、塔罗牌，或者你想要的任何东西。

命运之链的一段会在瞬间显现，而根据预言者洞察的深浅，便能看见并解读未来。不过，这并非宿命，而正如我们所说的，只是一种极快速的推演——从一个事件如何引发另一个事件，直到整个未来脉络在短暂一瞬间变得清晰可见。这才是此类现象的理性解释。

但请注意，虽然这是先前行为必然导致的结果，但绝非强加给任何人；这取决于自身是否相信诅咒或未来事件。若意志够坚强，这些不可避免的结果还是能被改变，尽管似乎与刚才所言矛盾。 但不尽然。假设有个人诅咒你，若你相信他，并按照此愿景行事，那么诅咒就会成真。

这同样适用于任何形式的算命，无论是通过塔罗牌、手相、占星术、拿破仑的《命运之书》，甚至是茶叶和咖啡渣。换句话说，若对于预测出的好运气或坏运气作出反应，就是与诅咒者或神谕者携手合作，帮助对方成了先知。若是在被诅咒时反过来诅咒对方，那么也是与之携手合作。但是，若能以祝福来迎接诅咒，或者以平静的心态和对神的信心来面对预言，那么就能避免厄运，诅咒也不会生效。那些想成为女巫的请多加留意 !

这就是「自由意志」。之所以许多诅咒能按计划发生，许多预言实现，是由于人的意志薄弱、无知或迷信，不知道有更好的方案。 当我们按照预言或诅咒行事时，就会成真；但这不是自由意志，而是胁迫。伟大的中国圣人老子说：「民不畏死，奈何以死惧之？」诅咒、预测或预言也是如此。若不惧怕这些，就不会以任何方式受到影响，我们便能获得自由，并在自己的内心得到平静。

这便是关于预测、命运和自由意志之谜的简单解释，未经任何花俏装饰，或许前所未闻。现在你知道，每个男人和女人都具有神赋予的自由意志，能以不同方式对情境做出反应。也更能理解「业力」的真正含义，对此误会导致懦弱的宿命论，这在许多东方国家和东方民族中非常普遍。

若宿命论为真，那么任何杀人犯或罪犯就能为自身行为辩解，说这一切都是神预先安排的，因此他是无罪，不应受到惩罚。甚至进一步说，若有人敢惩罚他们，便是反抗神本人，因为一切都是凭神的意志进行的。

相信宿命论者满足于在梦游状态中度过一生，表明大多数人都没有真正理解人类和宇宙中的真正法则。 希望此篇文章能让你开阔眼界，了解占卜、命运和自由意志到底是什么，如何运作。有耳朵的，让他们听；有头脑的，就去想 !
