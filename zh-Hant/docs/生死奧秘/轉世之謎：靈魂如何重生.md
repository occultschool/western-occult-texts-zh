# 轉世之謎：靈魂如何重生？
## 靈魂運作與業力法則的轉世剖析

![cover](/img/downloaded/40d80fcbf65c0a27f5152138d487d849.png)

若相信轉世（一旦看清其邏輯與哲理，便難以不信），又見證據越來越多，人難免想追問：轉世究竟如何發生？其方法、過程和階段為何？

本站[《對轉世的正確理解》](/docs/what-is-reincarnation-correct-understanding)、[《死亡與來世》](/docs/death-and-afterlife)、[《去世的人會到哪裡去？》](/docs/where-do-the-dead-go-after-death)諸文，已論轉世大要，探討人生盡時之過程，描述靈魂離體後、投生前的各段旅程。然則，重生具體如何發生的？

我們明白一切依業力法則而行，但這非本文焦點。我們意在探討：靈魂或自我如何自天界狀態顯現，於物質層面重獲生命，並從中窺見幾分真知。

神秘學提供若干有趣視角，亦涉及誕生過程的其他關鍵。對智慧大師而言，這些並非「奧秘」；對其「直接代理人」與「同志」布拉瓦茨基亦然。

然則，仍有諸多事理、真相、細節，是這偉大兄弟會尚不得公諸於世；故在循環法則與業力法則嚴密框限下，他們僅能在某些領域投以暗示、一瞥、或簡短指引。轉世的實際過程即是一例。

身為神秘學子，當務之急是深入、精準、且懷抱深意，去理解布拉瓦茨基留予我們的一切教誨與智識。若連此都不願深究，又何能企盼更多洞見？

無論如何，且讓我們把握機會，窺探此中玄機。…

於《秘密教義對談》中，學生問及此問題，布拉瓦茨基答曰：「生殖細胞僅含種子，亦稱星光體。父親將此種子植於物質土壤。… 單子到了第七月開始籠罩胎兒，待孩童具意識方完全融入。天界實體環繞新身體，點燃其火；然而兩者同化過程，要到意識初萌之後才開始，即七八月時。因此它未即刻進入。一開始單子籠罩新身體，業力引之趨近，卻無法立時合為一體。」（第559頁）

綜覽此言，並參照布拉瓦茨基及其他大師散見各處之論述，可歸結數項事實：

\* 父之種子僅含星光體之種，於母體內開始生長和形成。

\* 星光體之形成早於肉體，其生命亦較肉體綿長；此乃無形、精微之藍圖、模型與框架，肉體依此構築。

\* 受孕後，新星光體開始形成，迅即為肉體外殼所包覆。

\* 此形體雖未真正「具有靈魂」，然自受孕一刻，便以某種特定方式，與將降臨之「自我」相繫。

\* 此時之「自我」仍處天界狀態，故稱「天界實體」。

\* 其天界時光到了尾聲，便開始「環繞」、「點亮」、「籠罩」此微小形體；後者將於數月內，成為其新物質載具。

\* 此為神秘學反對墮胎諸多緣由之一，亦是布拉瓦茨基直指墮胎乃「謀殺」、「蓄意毀生」、「擾亂自然秩序」及「逆天之罪」的緣故。

\* 孕程約七月時，關鍵的「融合過程」啟動。新生兒的低等人格部分，此時與新身體結合。即便孩兒出生，靈魂較高部分（即高等心智），通常須待其七歲，方能與新人格建立穩固完整之連結。正因如此，七歲前夭折的孩童，其靈魂往往能迅即或幾乎立刻轉世重生。

\* 天界實體與新物質載體，日漸緊密聯繫，這與K.H.大師所述後半段相關：「自我在天界，猶如在塵世，歷經心智初動、臻於鼎盛、力漸消散、陷於半昧、遺忘昏沉、乃至全然遺忘……之後並非死亡，乃是誕生：誕生進入另一人格，重啟行動，每日種下新的因；此等積聚須待下一段天界時期處置，並再次投生肉體，成為新人。」

天界與塵世的生命，皆由業力注定。

同一位大師曾述及胎兒的發育：

「……母腹之中，胎兒歷經礦物、植物、動物諸態，終成人形。以人胎為例，自著床至七月，蜷縮於胞衣，如演化縮影，重複礦物、植物、動物的週期；直至末兩月，方始具備未來人形的雛型。其發育至七歲方成。人之本質亙古存在，不增不減；唯在自然之母的子宮裡——正如在俗世母親的懷中——才得以進一步開展。一位博學的哲人，篤信直覺勝於當代科學，主張：『人類在子宮的各階段，實乃地球歷史缺失片段的濃縮。』因此，你必須回溯動植礦物諸態。須將每一顯化初萌的實體，視為宇宙原初粒子，在生命初始的震盪中，分化而出。因此，圓滿行星神靈的潛能，正潛藏於這些原初粒子之中。」

此外，這位跨喜馬拉雅的開悟者，對費解的「二重身」現象，以及陌生人容貌極似的奇事，給出了一番精闢闡釋：

「有時，嬰孩的相貌會與另一人驚人相似，儘管兩人或許遠在千里，與嬰孩之母從無交集。然而，那人的形象早已烙印在母親的靈魂深處，或於夢中，或於清醒之時；這印象遂映照於她即將誕生的孩兒面容。」

據信中所指，「漂浮的圖象」實為星光界流質中的影像與畫面。星光界流質乃一種心靈感應的介質，環繞甚至滲透物質層面。其中，每一念、每一言、每一行皆被紀錄，永鐫於星光界流質。故如賈奇先生所言：「星光界流質充斥萬千圖像」，而我們與之接觸實屬頻繁，往往超乎我們的認知。

現在，且探輪迴如何發生。

受孕之際，即將到來的「自我」之星光體種子，蘊含在男性生殖細胞中。然而，此「自我」何以抵達？為父者的精子，又如何獲此神聖特性與力量？一切都在偉大的業力法則下運轉，無論雙親是否知曉、是否信服。

威廉·賈吉於《神智學的海洋》中《論輪迴》一章提及：「如何描述『自我』從非物質轉為物質狀態必經的實際過程」，以及「從不可見層面降臨可見層面的途徑、方式或方法」。

他繼而言道：「此題在神秘學著作中尚未明朗。一則因這需要極細膩處理，再則即便神秘學家自身，也未必全然信服這些細節——雖終有一日它們將被接受。既然此細節並非最關鍵，眼下暫不深究。但眾所周知，人體成形需兩性結合，胚種鎖於兩性之中，且必源自攝入體內的食物；故而，食物實與轉世之自我息息相關。」

於是，有人謹慎推想：「輪迴之路，或藉特定食物展開。」然此觀點，終未見進一步闡明。

賈吉先生另有一處，深刻描述了輪迴過程及其初期階段，令人驚異。晚年時，他構思一本關於前世經歷的書，且希望由同僚執筆，不署己名，不落任何個人痕跡。他寫此書，只為給細心而直覺敏銳的讀者，提供寶貴的靈性教誨與洞見。他從不渴求矚目，故堅持書中主角的身份必須隱去。

不巧，書未起筆，他便辭世，此作遂成未竟之章。然而，他曾交給朱莉婭·基特利（她在神學界以筆名賈斯帕·尼曼德聞名，並允諾協助完成此書）的若干筆記，後被收入《對我有幫助的信件》作為某種附錄出版。其中引人入勝的段落，見於第253至254頁：

「當他們抵達南印度，他見到了塔樓裡的老者。他向前致意，再次急切詢問羊皮紙之事。老人所言如舊，並附加一句：最好能在該處回到此世界。

「黑暗籠罩，萬籟俱寂。本是晴朗炎熱之日，卻無半點雨意。聽罷老者話語，他心中默許在該處生活，然倏忽之間，狂風暴雨襲來，雨水傾盆。他感覺自己被拽回塵世，陷入深沉黑暗。一道清晰的聲音環繞著他——那是植物生長之聲。眼前展開一片稻田，間雜幾簇芝麻。水滴落下，觸發萌動；他環顧四周，萬物皆在運轉，生機勃勃。他被困於茫茫稻禾之中，為自身命運感到悲哀。他出生在一個婆羅門家庭。」

註：輪迴之謎是否透過雲霧、雨水與種子，從人類的種子中投生？

應補充的是，即便此處所述經歷屬實，也並非所有輪迴的靈魂皆能意識到此過程，多數人恐渾然不覺。此特殊個例中，該「自我」已然在靈性上臻於成熟且進化完滿；在加上，他前世是因被謀殺而倉促終結——正因這雙重緣故，他能有意識地滯留於欲界，並得以遊蕩四方，「尋覓投生之機會」。

故而，他遇到啟蒙者指引他前往南印度，並勸言其當下便「重返人間」時，他能清醒地應允，隨即透過「雲、雨、種子」之過程，轉世進入人的種子，且對此中每一階段皆了然於心。

遠在三千五百至五千年前，古印度的聖者、大師與先知藉由神聖《奧義書》傳授教誨時，所思所慮或許正在於此。他們反覆申明：「雨生食物，食物生性別種子」（《蒙達卡奧義書》），「食物孕種子，種子育眾生」（《六問奧義書》），又言「一切身軀皆從食物而來」，並倡言「敬重食物，因其塑成肉身」（《鷓鴣氏奧義書》）。

在印度吠檀多不二論哲學中，肉體亦被稱作「食物製成的身鞘」。此說與前述或有關聯，或無關涉，端看讀者如何解會。

言盡於此，不宜再贅。或許對讀者而言，本文所陳資訊與細節，產生的疑惑反多於解答。然而我們希望，這些文字能激起幾分興味與啟發，至少為原本幽邃難明的謎題，投下一縷微光。

末了，且讓我們沉心默思《他氏奧義書》中的教示。此書視受孕為「初次誕生」，視脫離母體、踏入塵世為「二次誕生」，視捨離肉身為「三次誕生」。一切乃生命，而你，即是生命本身。
