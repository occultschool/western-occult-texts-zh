# 去世的人會到哪裡去？
## 所有人都將活在自己的行為裡，或美夢、或噩夢、或就此消亡

![cover](/img/downloaded/f3b25232ae7f28a823e62b149e168208.jpg)

Photo by [Zac Durant](https://unsplash.com/@zacdurant?utm_source=medium&utm_medium=referral) on [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral)

神祕學學生最常追問的問題之一，莫過於：「去世的人會到哪裡去？」

你若問尋常基督徒，他或可引用《聖經》經文答你，但你依舊茫然。新紀元的神祕主義者或各式「異教徒」，則可能依據各自信仰與通靈經歷，給你紛紜答案，卻同樣令人困惑。

答案其實極其簡單：**死者，住在自己的行為裡。** 請容我再說一遍：**死者，住在自己的行為裡。** 你或許要問：這如何可能？「行為」所指為何？

我們所謂的行為，即「契合」。我們與什麼契合，便成為什麼。殘忍之人將墮入下層星光界，那裡住滿折磨者——許多宗教稱之為「地獄」，也為現代瀕死經驗者如霍華德．斯托姆所印證。

斯托姆曾是公開的無神論者，敵視一切宗教與信徒。他以憤怒控制周遭所有人，在任何事物中都尋不著快樂。他不信物質世界之外有更高層面，認為一切宗教皆是無根幻夢，科學之外別無真實。然而一九八五年六月某日，三十八歲的他經歷一場瀕死體驗，從此徹底翻轉人生與視野。你可從他的敘述中讀到，手術中他「死亡」時的際遇。

那些徹底自私或怠惰者，將被抹去，或永遠沉睡；而善良、慈悲、仁慈之人，那些不計回報為同胞服務者，以及懷抱靈感的美麗夢想家、詩人、音樂家，將升至更高層的星光界。

在那裡，戀人們將在玫瑰與紫水晶色的雲霞下，於金色之境重逢。銀色溪流蜿蜒，永恆花朵綻放各色芬芳，芳香樹木結出甜美果實，青翠山丘環抱著晶瑩瀑布與閃爍湖泊。一切皆在夢幻與至福的領域中，此為俗世永不可知之境；二人遠離窺探的目光，獨自沉浸於完美結合的狂喜裡。

真正的神祕主義者與真理愛好者，將翱翔至遠超人類眼界之域，與同儕相遇，在智慧中生活，於不斷擴大的崇高循環裡擴展，永與眾神同在。

因此，所有人都將活在自己的行為裡，或美夢、或噩夢、或就此消亡——若「光明之主」如此定奪。但對某些人而言，這行為或夢境，將成為對更高意識、對宇宙意識的真正覺醒，非任何語言所能描摹；對這些人，再無重返沉睡的可能。
