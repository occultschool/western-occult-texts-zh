# 《神聖智慧之鑰》電子書
## 評介布拉瓦茨基三部要著：《神聖智慧之鑰》、《布拉瓦茨基夫人的神祕世界》及《神祕學研究》。

**一、《神聖智慧之鑰》 — H. P. 布拉瓦茨基著**

![封面](/img/downloaded/0ca9629c0e4e96f378c30e029552286a.png)

中文版電子書：[點擊閱讀](/books/key-of-sacred-wisdom-blavatsky)

若覺得布拉瓦茨基兩部鴻篇巨製——《揭開伊西斯的面紗》與《秘密教義》過於龐雜艱深，此書便是一把絕佳的鑰匙，為西方讀者初次開啟古老智慧之門。全書以問答形式寫就，直接回應學子諸多疑問，條理分明，娓娓道出神祕科學的基本原理。

探討主題廣博：人的七重結構、進化與幻象、靈魂與靈之區別、記憶本質、前世、輪迴、死後與出生之際的意識、因果報應、通靈現象、死亡之後、祈禱、慈善、苦修、婚姻、智慧大師，乃至萬物同源的根本一體性。

但須提醒：書中每則答案，未必全然無誤。有別於《秘密教義》，此書寫給背景各異的讀者，其中許多人對布拉瓦茨欲闡明的真理所知甚少，甚或一無所知，且無意改變既有的思維與習慣，難以將真理吸收、踐行於日常生活。因此，她採取了真正教師的策略：因材施教，調整答語，只為讓學生更易領會。

書末附有二百四十七條詞彙表，羅列宗教、哲學與神祕學術語，釋義諸多晦澀梵文詞彙，於不諳此道的讀者大有裨益。

莫被「神智學」一詞嚇退。其意其實簡單，無非「神聖智慧」，亦即神聖知識或科學。本書雖應神智學協會成員之請而寫，至今仍作教材使用，卻絕非該會宣傳冊子，目標也不限於會員。相反，它精彩闡釋了貫穿歷代的智慧，足以吸引所有尋求真理之人。故此，我們誠心薦予各位。

中文版電子書：[點擊閱讀](/books/key-of-sacred-wisdom-blavatsky)

**二、《布拉瓦茨基夫人的神祕世界》 — 丹尼爾·H·考德威爾 編纂**

![插圖](/img/downloaded/d7de6fdd875c388484897db81c3d90eb.png)

這是首次由真正了解布拉瓦茨基的人來述說她的故事。丹尼爾·考德威爾筆觸鮮活，勾勒出這位曠世神祕學家的身影——她獨力復興古代智慧與各時代的神祕真理，其影響跨越古今。

作者成功彙集親屬、熟人、友人、敵手與同僚的諸多回憶，讓讀者得以窺見布拉瓦茨基的生活、工作，以及她那神祕難測的人格。用「窺見」一詞，是經過斟酌的。她真實的身分，知者極少；即便生前最親密的同事，亦不明瞭她為人類艱辛付出的真正目的，至今依然成謎。因此，在近年關於這位卓絕神祕學家的傳記中，我們最鍾愛考德威爾此書。

許多長期讀者皆知，本專欄文章內容，多有承襲自這位神祕學先驅的教誨。過去數世紀中，這位古老祕密的偉大守護者所傳授的神祕學真理，遠超任何其他導師。若您尚未讀過布拉瓦茨基的著作，對她的認識僅止於多年流傳的誹謗謠言，那麼這部出色的傳記，將助您發現她的偉大，以及她所教導真理的崇高。

**三、《神祕學研究》 — 布拉瓦茨基著**

![插圖](/img/downloaded/940d11d026ee3a05875fafbb315e7f9f.png)

此書匯集布拉瓦茨基晚年發表的數篇專文，深入探究神祕學的核心實踐與理論。文字洗鍊，見解犀利，直指修行與認知的本質。對於已略窺門徑、欲更上層樓的讀者，無疑是進階的指引。書中論及意識轉化、能量運作與宇宙法則等深奧課題，讀來如暗夜明燈，廓清諸多迷思。若說《神聖智慧之鑰》是啟蒙之鑰，此書便是深入堂奧的階梯。

![插圖](/img/downloaded/940d11d026ee3a05875fafbb315e7f9f.png)

《神秘學研究》所輯文章，原刊於布拉瓦茨基主編的雜誌《路西法》。該雜誌發行於一八八七年至一八九一年間，即她逝世當年。布拉瓦茨基清晰辨明：理論與實修本質有別，一如左道的邪術與右道的利他奉獻，兩者之間橫著深淵。

書中探討諸題：實修密學與秘術、催眠現象、十九世紀軍火貿易之罪、科學中的黑魔法、宇宙意識與高等心智、智慧的雙重面目、星光體，以及人在物質與靈性層面上的真實構成。

另收一篇長文，分三部分，題為《福音的奧秘性質》。此文深入剖解耶穌的神秘性——既為歷史中人，亦是基督原則的體現。
