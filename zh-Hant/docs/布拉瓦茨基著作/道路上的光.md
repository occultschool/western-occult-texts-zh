# 《道路上的光》
## 成為光，照亮大師光輝未及之處

![封面](/img/downloaded/45d4f7ebd609ab95ed7b031e15f73156.jpeg)

電子書連結：[點擊閱讀](/books/light-on-the-path)

《道路上的光》這本小書，於一八八五年初版，至今仍在印行。

作者是英國神秘學者梅伯·柯林斯，但她自承靈感源於一位以三角符號「△」為簽名的人。布拉瓦茨基在《寂靜之聲》的註腳中曾作解釋：「△是高等弟子的標記，而另一種三角『∴』，則是高等啟蒙者的徽記。」（參見一八八九年原版第七十九頁）她獲准在自己的署名後附加「∴」符號，偶爾為之。

「高等弟子」與「高等啟蒙者」，皆為智慧大師門徒，二者實則均為啟蒙者，僅階次有別：前者較低，後者更高。高等啟蒙者雖已開悟，未必即是智慧大師——「開悟者」與「智慧大師」並非同義。智慧大師定然開悟，開悟者卻未必臻至智慧大師那等高深境地。然而，對常人而言，能冠以「高等弟子」或「△」者，內在進化與啟蒙修為已遙遙領先，本質上已是開悟之人。

布拉瓦茨基指明，《道路上的光》的靈感，來自「一位希臘人，雖非智慧大師，卻是開悟者……他於一八八六年方證開悟」，那已是該書出版隔年。這位開悟者，人稱伊拉里昂、伊拉里昂·斯默迪斯，或希拉莉昂大師，是希臘裔塞浦路斯人。

此書啟發之際，並非憑空創作，實為翻譯《金戒書》中的部分文句與箴言。布拉瓦茨基在一八八九年的一封信中曾提及此事。其後，她更以《寂靜之聲》為名，更完整、細緻地譯述了這部未公開的秘傳典籍。從某種意義上說，《寂靜之聲》與《道路上的光》堪稱姊妹篇，並讀相得益彰。

此書之美與力，曾震撼印度學者，乃至一八八八年便有了梵文譯本。布拉瓦茨基於同年十二月在《路西法》期刊宣佈此事，稱其為「真正的寶石，思想根源與《秘密教義》同出一脈，皆屬印度—雅利安學派與佛教教誨」。

全書分兩部，各有二十一條「規則」。但卷首第一頁，另列有四條未編號的規則。關於這四條，大師如是說：

「《道路上的光》首頁四條真理，直指神秘主義者必經的試煉性啟蒙。未能通過，便連知識殿堂的門閂也觸及不到。」（第三十一頁）

「《道路上的光》前四條規則，敘述看似奇詭，實是全書重中之重。」（第四十五頁）

「在某個持續活躍的兄弟會中，這四條規則鐫刻於每處聚會所的前廳。」（第四十六頁）

「我為有意探究的西方人寫下這四條，它們不只刻在每個活躍兄弟會的前廳；甚至，每個曾存或已逝、乃至尚未成形的教團前廳，皆有銘刻。」（第六十三頁）

規則為何？最好銘記於心，時時溫習。

其一：「雙眼若要看見，必先失卻流淚之能。」

其二：「雙耳若要聽聞，必先失卻敏銳。」

其三：「若要在大師前發言，聲音必先喪失傷人之力。」

其四：「靈魂若要立於大師面前，雙足必先浸浴心的血之中。」

這些警句，連同全書精義，並非寫給普羅大眾，甚至不是為每個對靈性感興趣者而作——儘管人人皆可從中獲得些許啟示。

《道路上的光》所載諸規定，並非為所有弟子而設，僅予那些「求道者」。是故，這對於他人既無用處，亦不會引其興致。《寂靜之聲》亦然——布拉瓦茨基明言，此書乃「供弟子日用」，且「僅奉少數人」。用書之前，未必需得大師認證；凡誠心謙卑、渴慕此道者，皆可依其所能取而習之。

解讀這四條準則，不可拘於字面與形質。第三條雖直白，我們皆需憑直覺方能領會。第三十三頁有言：『《道路上的光》通篇以星光界密語寫就，故唯有藉星光界之眼，方能破譯。』

前兩條準則亦見於《寂靜之聲》第二頁：

「靈魂欲得見，必先達至內在和諧，令肉眼對一切幻相視而不見。」

「靈魂欲能聽見，此身必須對咆哮與私語俱充耳不聞——無論巨象怒吼，抑或金螢銀嗡。」

大師於第三十五頁闡明：「這四句所指的感官，實為星光體之官能，亦即內在感知。」

「所謂『失卻流淚之能』，並非鐵石心腸，亦非漠然無感。而是直面並馴服凡人性情，臻至不為私情所撼的平衡。它不意味悲慟枯竭，不意味老年情感已死……凡此種種，皆非弟子應有之境。若有上述任一，必先克服，方可踏入此道。鐵石心腸屬私我之輩，此門永不對其敞開；冷漠則屬愚癡偽哲……」

「倘使悲苦、沮喪、失望或歡欣足以動搖靈魄，令其無法穩駐於啟明之靜定，生命之水便將潰決，將知性淹沒於感官之中——屆時萬象模糊，窗戶盡晦，光明亦失其用。」

《道路上的光》開篇四則箴言，全然指向星光界之修進。此等修進須達至某種程度，並完全的進行，書中餘下方能真悟而非僅理解；至此，本書方可作為實修指南而非形上玄談來讀。

「某秘修兄弟會中，年初必行四儀式，實則闡釋此四箴言。僅初學者參與，因其旨在助其跨閾入門。一旦悟得這些儀式關乎犧牲，便知成為弟子是何等莊嚴之事。首個儀式即我所言者——至烈的歡愉、至深的痛楚、失落與絕望，俱壓向顫栗的靈魂；靈魂尚未於黑暗中覓得光明，猶盲者般無助。唯當它能承受諸般沖擊而不失平衡，星光感官方得開啟。此乃慈悲之法……弟子必先成為自己的主宰，才堪涉此險途，嘗試與那些生活、運作於星光界之存有見面，乃至面謁我們所稱之大師——因其具大智慧，不僅能馭己，亦能御周遭之力。」（頁37–41）

或有人問：為何有人願受此等試煉，開啟此星光體感官？

神秘學之動機，非為自求解脫或力量；乃願盡己所能助人服務，成為人類真切有力的服務者，並有意識地、直接地、切實地襄助「聖者們」從事慈悲無私之工。此即菩薩理想，慈佛之道。

然此事非可一蹴而就。故有初階啟蒙與漸次步驟，始於凡人日常境遇，及其間必然湧現之「日常啟蒙」。

本文要旨之一，在提請留意下列訊息：

弟子之神聖使命，本質乃是「同化」之進程。布拉瓦茨基曾致函道：「除非汝等與大師或我同化；除非汝等與我及彼等同工」，否則功業難成。威廉·賈吉亦特以斜體強調：「凡能在某種程度上與大師同化者，於某種程度上即為其代表，並助其運作。」

「然則何從起始？如何涉入與大師同化之流？同化者，融合、滲入、與之合一，而成其永續部分。非僅憑希願或空想可致。必當實做一事——或內或外。弟子所能向大師與師長求索者，唯『同心同性』而已。」

《道路上的光》如是釋之：

「弟子欲將聲息傳抵眾神所居之高境，必先深入無光之深淵。此為鐵律所縛。若想成為初學者，頃刻須先成為僕役。然其服務之所以崇高，端因其品格如此——因大師亦為僕……他們的服務之一，是將此知識分享於尚未能立足其所立之地者。此非任何大師、師長或聖者任意所定；實為弟子所契入的生命法則。」（頁65）

一個人如何將從秘傳教義所得的知識傳予他人？此事唯待個人自行探求。非人人皆能撰文著書，非人人可翻譯神秘學教義於不同語言，亦非人人皆有餘力創建網站、攝製影像。既如此，除了上述途徑，必有他法既能收到成效，也能深重影響眾多生命與旅程——最要緊的是撼動其思維與心智。但凡能予同途者如此相助，便是襄助大師與其志業。

「若施以援手、或成其任務，則可得到實在報償；非關私己酬勞，而是獲得了共通的稟性。此為神聖的付出，在你成為其一員前需先付出。」（頁66）

雖為大師，他們仍須某類協助：

「有些任務唯大師能成，餘者則需同修共助。大師之職在保存真正哲學，然若要重新覓得並廣傳，則賴同道之力。我們兄長已再度指明真理可尋之處，而四海同修皆戮力播揚，以期其傳佈愈廣。」（威廉·賈吉）

《道路上的光》的真正作者繼而道：「世上存在著能授人知識、予人益助之力。」於今世眾人，這確屬新知——他們不知真理實存，不知有「識真理者」，更不知這些「識真理者」正竭力使我們亦成「識者」。

誠然，「弟子若欲跨過門檻，便須傳此訊息，並以某種方式錄存之。」

對於憂心自身力有未逮者，大師添了幾句鼓舞之言：「有些人驚覺自己做得倉促而不完滿；一股渴望便悄然滋生——要將此事做好。這渴望純粹如天降，不為功名，不求回報，只為成全他人。正因如此，他便有了實現的力量。」（頁67–68）

或有人言：「我誠願相助並盡力，然不欲為大師弟子；我知自己尚未備妥，知自己能力不足，至少此刻未能。」

實則，人若能這麼想，已較那些自認堪為弟子者更近資格。懷有真正謙卑的靈魂，且明見己身之闕漏與軟弱者，反更有望觸及大師；不似那些愚者，自以為已臻高境。

當然，無人被迫踏出成為弟子之步。然對於已舉步者，《道路上的光》中此言可作有益提點與導引。至於未舉此步卻願為大師志業效力者，上文當有助其認清：在行此無私而利他之工時，彼實已擔起若干『弟子之務』，從而使其未來弟子之路愈發坦順。」

《道路上的光》復云：

「於世界大城市之中，開悟者偶爾暫居，或只是途經；然而眾人皆曾蒙受他們真實力量與存在的援手。在倫敦、巴黎、聖彼得堡，有這樣一些人，其進境已達高處。但他們身為神秘主義者的身分，唯獨那些能夠辨識之人得以知曉——此種能力，必待戰勝自我之後方能獲得。若非如此，他們怎能身處都市的喧囂與失序之中，哪怕僅只是待一個時辰？城市的混亂滋生了心智與心感交雜的氛圍。若無護佑，其自身的成長便受干擾，所承之任務亦遭妨害。

初習者或曾邂逅一位開悟者，甚至與之同在一室，卻未能識別，亦無從使己聲傳達於其耳。縱使就在身邊、關係之近、日常之親，皆無法穿透那將開悟者隔絕的不可違逆之法則。任何聲音皆難滲入其內在聆聽，除非化為神聖之聲——一種不帶自我呼喊的聲音⋯⋯除非一人於內在與靈性上成為弟子，否則對那擁有多名弟子的師者而言，此人猶如不存在。唯有一途可臻此境：放下個人人格。欲令聲音失卻傷人之力（註：此指第三條法則：『聲音須先失去傷害之力，方能於大師面前言說。』），人必須能視己為廣大眾生之一員，不過是受到存在之海振動拂捲的沙粒而已。」（頁74–75）

閱此文的讀者裡，此生得於物質層面親見開悟者之人當屬寥寥；然於此生某刻，或能遇見開悟者之真弟子。若是這般，願到時我們內在已有足夠準備，以識其真面目（切記，真正開悟者絕不自稱開悟！），且能有意識地幫助他們，同時亦受其助。

不妨重溫：弟子若不傳播神聖智慧，並加以記錄，便無法越過門檻。此亙古原則不正闡明了《道路上的光》一書的性質嗎？此乃一位高階弟子贈予我們之禮，而他次年更進一步，成了開悟者（一八八六年，如前文布拉瓦茨基引語所述）。

由是觀之，此事其實至簡：

去成為光，照亮大師之光未及之處。
