# 巨人的考古證據
## 本文檢視過去「巨人」存在的科學證據。

![cover](/img/downloaded/b38786129c39190f2591856bb43c348e.jpg)

Photo by [Hulki Okan Tabak](https://unsplash.com/@hulkiokantabak?utm_source=medium&utm_medium=referral) on [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral)

許多陰謀論者雖極力奔走，盼能敦促學界正視大量考古化石，以證明遠古曾有巨人族存在；但其善意卻遭誤導，反倒阻礙、甚至破壞了對此課題的嚴肅探討。一則二零一五年於網路流傳的「巨人新聞」，便是明證。它迅速引發關注，甚至若干主流報刊也予轉載。

報導宣稱，美國最高法院一紙裁決，迫使史密森學會公開二十世紀初的機密文件，揭露該會曾壓制全美發現的巨人遺骸證據，唯恐主流人類演化年表因此動搖。表面看來，因多數學者皆否定史前「巨人」之說，這篇二零一四年十二月刊於《世界新聞日報》的報導，似乎言之成理。

史密森學會在巨人遺骸的挖掘上，固然遵循主流科學路線，卻從未刻意隱藏；美國最高法院也從未對該機構下達此類裁決。這故事純屬虛構，而《世界新聞日報》本是一家諷刺刊物，其網站開宗明義聲明：「本報對文章之諷刺性質及虛構內容，承擔所有責任。」

事實上，若想扼殺誠實調查與科學事實，何須陰謀論？只需眾人達成默契，讓其他探求真相者無利可圖，甚而危及事業、聲譽與生計。此事本就不需、也不稀罕陰謀論插手。若有知名學者或科學家膽敢依據考古化石證據，公開證實遠古巨人存在，運氣好遭人譏嘲，運氣差便飯碗不保。

正因如此，維基百科及所有相關資訊來源，對巨人存在的物證一無所知。倘若這類骨骸獲准公諸於世，將掀起多大波瀾！主流媒體與科學界於是選擇緘默，並將持續如此，直到證據確鑿、無可辯駁，迫使眾人正視這頭「房間裡的大象」。但我們最好別抱太大期望。

一八七七年出版的《揭開伊希斯的面紗》中，布拉瓦茨基寫道：『本書撰寫之際，美國《堪薩斯城時報》刊出一則史前巨人族遺骸的重要發現，既印證卡巴拉學者的說法，也呼應《聖經》寓言。其文記載如下：「密蘇里州西部森林調查中，韋斯特法官發現若干圓錐形土丘，結構類似俄亥俄與肯塔基所見者。」

「這些土丘位於俯瞰密蘇里河的高崖上；更為龐大顯著的土丘，則見於田納西、密西西比與路易斯安那諸州。三週前，世人尚懷疑這些史前築丘者曾以此地為家；如今證實，這支奇異而滅絕的種族確曾佔據此域，並於克萊縣懸崖上眾多高丘間，留下一片遼闊墓地。」

報告續稱：「迄今僅開啟其中一丘。約兩週前，韋斯特法官發現一具骨骸，並通報社群成員。眾人隨他重返土丘，於近地表處掘出另兩具骷髏殘骸。骨頭異常碩大，現代常人骨架相形之下，僅及其一部份；頭骨未朽部分亦大得駭人。其中一具下顎保存完好，較文明人之顎足足大上一倍。」

「頜骨中牙齒碩大，或因啃食肉類磨損嚴重，足見咬合肌力驚人。對比尋常現代骨架，其大腿骨長厚如馬，肌肉異常發達。最奇特莫過額骨：低平至極，異於本地所見任何頭骨。一道寬約二點五公分的厚骨脊橫貫眼眶上方，不似今人額頭向前隆起，而是自眉弓處向後斜退，形成扁平頭型，顯屬低等人類。

「考察學者斷定，此為史前人族遺骸。其形制迥異於現存印第安族群，所在土丘亦非美洲任何已知文明模式。丘中屍骸呈坐姿，骨骸間散落石製武器——燧石刀、刮削器形制特殊，有別於常見箭鏃、戰斧等物。而早在白人發現美洲原住民前，此類工具已然存在。

「負責發掘的學者將骨骸存於主街的福博士之處，預備對崖上土丘深入勘察。待科學院下回集會，將呈交完整報告，盼能得出確切結論。然目前已可斷言：這些骨架屬於不復存在的人類族類。」

一八九一年八月十日，《紐約時報》載，史密森學會學者在威斯康星州米爾斯湖畔發現數座巨型「金字塔形紀念碑」。報導續稱：「古時麥迪遜一帶人煙稠密，居民不下二十萬。俄亥俄與印第安納著名土丘，無論規模、設計或工藝，皆不及此等神秘巨碑。建者誰屬、所為何用，今人唯能臆測。」

十九世紀末季，類似記載層出不窮。一八七〇年，《威斯康星州迪凱特共和國報》稱，波托西附近掘出兩具保存完好的未知巨人骨骸。一八九七年，《紐約時報》描述威斯康星州楓樹溪發現的大型墓冢：「內有巨人骨架，自頂至踵逾二百七十公分，保存完好；顱骨碩大如半蒲式耳量器。骨骸旁散落精煉銅條等物。」此類發現遍及美國——明尼蘇達、愛荷華、伊利諾伊、俄亥俄、肯塔基、紐約諸州，皆掘出巨型骸骨。

一九三一年，《聖安東尼奧報》報導德克薩斯大學聯合考古隊發掘出「當世所見最大人類頭骨」。存世照片顯示，該顱骨體積為現代常人兩倍。標本運抵德克薩斯大學後，經史密森學會學者鑑定，推定距今約四萬至四萬五千年。

若將目光南移墨西哥，巨人遺跡愈見分明。首例見於納亞里特：一九二六年五月十四日剪報載，佩奇上尉與德瓦爾達發現平均身高三公尺以上的巨人族骨骸。據當地傳說，此族源於厄瓜多。此事耐人尋味——厄瓜多位處墨西哥西南方的南美太平洋沿岸。難道遺骸屬雷姆利亞移民，而非亞特蘭提斯人？此說頗有可能。

一九二五年六月廿一日《紐約先驅論壇報》（翌日獲《華盛頓郵報》轉載）記載，採礦隊在奇瓦瓦市西索估切附近掘出身高三至三點六公尺的骷髏。一九二七年十月二日《洛杉磯時報》稱，墨西哥探險家於西南部聖地亞哥-塔佩克特拉附近發現大型人骨，屬「體型巨大」之族類。此後再無下文，實不足怪——如前所述，學者若調查此類遺跡，非但無益，反可能動搖「巨人從未存世」的主流定見，盡失學界立足之地。

一九七六年，考古學家在羅馬尼亞特蘭西瓦尼亞西部的羅西亞-蒙塔納，發現一處地下長廊，其中躺著一具高逾九公尺的巨人骨骸。當時缺乏分析設備，骨骸遂被送往莫斯科。而羅馬尼亞南部的久爾久，亦曾有類似發現：一九四〇至五〇年間，出土約八十具人形巨骸，身高皆達四至五米。可惜這些非凡發現，其後再無下文。

非洲亦存古代巨人的物證。僅舉一例：約翰內斯堡WITS大學醫學院地下庫房裡，藏著一段帶髖關節的大腿骨；推算全副骨架約高三點六公尺。此遺骸自一九六〇年代初便存於該校，原由納米比亞北部礦工掘得，估計至少已有四萬年歷史。

結束本文前，不能不提兩項年代較近的驚人發現。其一在一九二六年，西班牙加泰羅尼亞地區，一座俯瞰普魯蘭斯鎮的山上，考古學家於奧倫四千年石墓附近的古維丹尼洞穴，發現至少九具巨人骨骸。據說這些巨人身高三米，頭骨與股骨皆異常碩大。

再往南約一百七十五英里，喬治·瓦切爾·德·拉普熱於一八九〇年，在法國的卡斯泰爾諾-勒-萊斯，發現三塊巨大的人骨碎片，當時覆蓋於一處青銅時代墓穴之上。依骨塊尺寸推算，德·拉普熱斷定所屬個體身高必超過三點三米。其發現刊載於一八九〇年法國《自然》科學雜誌第十八卷第八八八期，並獲當時法國各大媒體廣泛報導。蒙彼利埃大學的動物學、古生物學教授及其他學者檢視該骨骸後，結論指出：這屬於一個「體型極高大的族類」。

為免世人以為，歷來從無重要學者或科學家敢於支持巨人存在之說——甚至不惜危及自身聲譽——本文末略舉數位勇者為例：劍橋大學唐納德·A·卡佐博士在匹茲堡掘得一具二百二十公分骨骸；亞利桑那大學考古系主任拜倫·卡明斯博士發現數具逾二百四十公分的遺骸；史密森學會史前人類學館長托馬斯·威爾遜驗證過二百四十公分骨骸與巨型顎骨；卡內基博物館館長W.J.霍蘭德於賓夕法尼亞州出土超過二百四十公分的骨架，此事獲《科學人》等多本科學期刊報導。
