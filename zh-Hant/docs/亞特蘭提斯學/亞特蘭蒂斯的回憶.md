# 亞特蘭蒂斯的回憶
## 雷德-哈格德筆下的亞特蘭蒂斯，描繪出這個失落文明的科學、藝術與歷史。文字生動，交織出那既輝煌又終歸覆滅的共同故事。

![封面](/img/downloaded/54ae53087e97b7bc6310c57bc989978f.png)

先前提過，雷德-哈格德（Rider Haggard）這人，天生具有遙見遠古的天賦，儘管他自己多半渾然不覺。他的幾部歷史小說，鮮活地勾畫出亞特蘭蒂斯的盛世，以及那漫長的傾頹。

本文將與各位分享其作品《當世界震動時》（When the World Shook）的幾段摘錄。此書雖被文學評論家譏為「高雅的通俗之作」，卻真切呼應了我們對於數萬年前亞特蘭蒂斯藝術、科學與歷史情境的記憶。

但開始前，容我先作幾點說明。《當世界震動時》講述三位夥伴——巴斯丁、比克利與阿布斯諾——漂流至一座未知南海孤島的故事。島上，他們遇見一個先進神秘種族的最後兩名成員：歐羅勛爵（Lord Oro）與他那美麗的女兒伊娃（Yva）。兩人已在玻璃棺中懸浮沉睡，歷時二十五萬年。

情節自然摻雜虛構。然而我們毫不懷疑，亞特蘭蒂斯的科學家在保存人體方面，遠比古埃及人更為長久完整。以下段落，透過伊娃女士的靈視能力，自她記憶中喚醒，向巴斯丁、比克利與阿布斯諾展現的，正是亞特蘭蒂斯昔日那令人屏息的輝煌與力量。

「我先讓你們看看，我們子民在最鼎盛時期的模樣。請看前面。」

一幅壯麗圖景倏然展開。一座白色大理石宮殿巍然矗立，前方廣闊的庭院灑滿陽光，生意盎然。宮殿臺階下搭著絲質遮陽篷，篷下坐著一位國王，頭戴皇冠，身披華麗長袍，手執鑲寶石的權杖。他神態莊嚴，身旁簇擁著一群服飾閃亮的廷臣。左右各有美麗女子為他打扇，稍遠處還坐著幾位珠光寶氣的女子，料想是他的妻女。

伊娃說道：「這是智慧之子中的一位君王，新近加冕，贏得全世界的尊崇。」

話音剛落，其他國王陸續現身，一個接一個走向寶座，人人頭戴王冠，手持權杖。抵達御前，他們屈膝跪下，親吻寶座上那人的腳。行禮後，各人放下權杖，再依指示重新舉起。這群君王約有五十餘位，膚色相貌各異，有白人、黑人、黃種人、紅種人。

接著，他們的臣子捧著貢品上前，顯然是黃金與珠寶，堆放在寶座前的托盤上。我記得注意到一段插曲：一位跛腳的老者絆了一下，打翻托盤，盤中物品滾落一地。他慌忙追撿的模樣頗為滑稽，連寶座上的君主也卸下威儀，展顏一笑。我特別提及此事，正為說明我們所見並非憑空臆想，而是往昔一段鮮活片段。因為若由我來設計這場景，斷不會讓一位衣飾華麗的老人在隆重典禮上失足跌倒——這荒謬一幕，定會被我刪去。

不，這必定是過往的生活，真實的歷史，已然發生的事。後續情景亦然。譬如，接下來是一場我們所謂的閱兵。步兵列隊前行，有人手持劍矛，我猜想是儀仗衛隊；有人吹奏著形似野蠻人用的管樂器，我全然不解其用途。現場雖無大砲，卻出現幾輛馬車，車上載著一些帶噴嘴的袋子。或許袋中裝的是毒氣。還有騎兵，騎的馬與我們今日所見不同：體型更敦實，四肢較短，但頸項高昂，目光炯炯，想必極為強健。

我推想這些安排主要是為了觀賞。接著，有人坐上一具長形裝置，兩兩鑽進一種護甲袋裡，只露出頭與手臂。那機器看來像一輛加長的自行車，卻以驚人速度呼嘯而過，我實在看不出它如何驅動。它能載二十對人，每人手持一種外觀小巧、形如橘子卻顯然致命的武器。隨後出現的其他同類機器，載運人數則在四十對至一百對之間。

然而，最令人驚嘆的，莫過於那些飛行器。它們為數眾多，時而如雁群齊飛，時而單獨翱翔，時而列隊前行；有時更編成嚴整的中隊，前鋒與指揮艦各居其位，間距精確如尺規所量。這些飛行器體型並不算大，至多容納四五人，卻迅捷如燕，靈巧非凡。它們似鳥類般振翅而飛，只是無人知曉，那驅動雙翼的力量究竟從何而來。

閱兵式結束後，一座燈火通明的大廳憑空出現，慶典於此展開。偉大的國王高踞寶座，身後矗立著一尊命運女神的雕像——或是一件極其相仿的作品；我們則站在雕像的基座之下。大廳裡，賓客們身著各國服飾，依長桌而坐。國王起身，轉向女神像屈膝跪拜。他三次跪地祈禱，方才歸座，舉杯向賓客致意。賓客們舉杯回敬，並仿效他先前之姿，在女神像前跪拜行禮。唯獨我注意到，尚有數人始終站立未跪，他們身披的聖袍，樣式竟與今日希臘教會的禮衣極為相似。

如今，這一切世俗榮華皆已消散。接下來的景象轉為簡樸，是這位國王臨終臥榻的場景——我們從他枯槁的容貌認出了他。他躺在那裡，年老體衰，生命垂危。醫生、婦女、朝臣們圍聚一旁，默默注視這最後一幕。畫面忽又一轉：歡呼的人群中，一位年輕的繼任者赫然現身，他臉上故作哀戚的陰霾，終被喜悅沖散。這一幕亦隨即隱去。

伊娃說道：「偉大的國王便是如此世代相傳。他們前後共八十位，平均統治長達七百年。他們主宰大地，如同昔日的神明。他們積累知識，掌握權力，坐擁無盡財富。他們培育藝術，探尋隱秘，甚至與星辰對話。然而，正如諸神一般，他們的嫉妒之心也日益深重。」

以上所述，皆據雷德·哈格德的記載，描繪了亞特蘭提斯最為輝煌、華美與強盛的時代。你或已察覺，這段描述與柏拉圖的記述多有相似，足見兩者皆對往昔做了真切忠實的呈現。

然而，正如伊娃在其故事中所言，亞特蘭提斯那些強大的統治者漸漸「心生嫉妒」。他們本是神之子，卻因貪婪、愚昧與墮落而步入邪途，化為布拉瓦茨基在《秘密教義》中所謂的「罪惡之黑」。戰爭降臨諸神之地，曾經照耀亞特蘭提斯大陸的「光明」，終被死亡與毀滅的「陰影」籠罩。伊娃接著講述下去：

又一幕景象浮現。天空佈滿飛行器，如閃電般將火焰傾瀉於城市之中。城內爆起更多火光，擊落空中來敵；飛行器如遭火焚的飛蟲，紛紛墜落。然而敵機不斷湧來，城市終告崩塌，城中再無反擊的閃光升起。智慧之子們被迫離開大地表面。

接著是一幕驚心動魄、變幻莫測的可怖景象：陸地沉陷，城市崩塌，火山怒嘯；彷彿世界末日已然臨近。只見人群如螻蟻般倉皇奔逃。隨後，數百公尺高的巨浪襲來，海洋為之沸騰。大部分陸地被海水吞沒，僅餘零星土地浮現浪濤之上，由大洪水後的倖存者居住其上。

正如我們先前所提，埃及人製作木乃伊的技藝，乃是傳承自他們的亞特蘭提斯先祖。這一點，在雷德·哈格德的另一部著名歷史小說《她》中亦有提及。書中女主角阿耶莎曾向敘事者賀拉斯·霍利描述，這門科學在古代科爾城達至何等巔峰。雷德·哈格德選用此名，或許是為了避開當時頗具爭議的「亞特蘭提斯」一詞——在他那個時代，多數出版商與讀者對此詞的接受程度，遠不如今日。事實上，據我們所知，他在過往小說中從未使用「亞特蘭提斯」一詞，僅在其自傳《我生命中的日子》中有所提及。

「科爾人也如埃及人一般為死者防腐，只是技藝更為精湛。埃及人剖腹取腦，科爾人卻將防腐液注入血管，使之滲透全身每一處。但這還不止——讓我為你細細展示。」她隨意停在我們途經的走廊旁一扇小門前，示意啞僕引我們入內。那是一間狹小的石室，與我最初休息的房間相仿，只是空無一物，唯見兩張石凳或石床。其中一張長椅上躺著以黃色亞麻布覆蓋的人形，布面蒙著一層細沙，輕薄得幾乎難以觸覺，事實上也不如人們想像中那般積塵——因這些深掘地底的洞穴裡，幾乎沒有什麼能化為塵埃。墓室的石架與地板上，屍身周圍擺放著許多彩繪陶瓶，然而在此類墓室中，我很少見到其他裝飾或武器。

「哦，霍利，把布揭開吧。」阿耶莎說道。可當我伸手欲揭時，卻又遲疑，將手縮了回來。這舉動似有褻瀆之意，何況老實說，此地的氛圍與眼前人形那肅穆的威儀，著實令我生畏。她見我猶豫，只是微微一笑，便親自上前揭開亞麻布，卻發現石凳上的形體還裹著另一層更纖細的織物。她將其輕輕收起。於是數千年來第一次，那張冰冷的遺容，被一雙充滿生機的眼睛所凝視。那是一名女子；看來約莫三十五歲，或許更為年輕，確實是位美麗的女子。

她面容靜雅，眉目清秀；長長的睫毛在象牙色臉頰上掃過微影，分外動人。身著白袍，墨藍長髮垂落袍邊，安詳長眠。臂彎裡躺著嬰兒，小臉貼著她胸口。這溫馨一幕感人至深，我不禁真情流露，震撼得幾乎落淚。

屍首都經精心處理，保存完好，恍如數千年前離世時模樣。靜寂石室中，無物能傷：不懼熱、不懼冷、不懼濕氣，身上彌漫的芳香氣味，彷佛永存。

但也有一些例外。外表看來肉體完好，一觸卻陷，才知已化作塵灰。阿耶莎說，這些屍首可能葬得倉促，只以藥劑浸泡，未能注入體內，故保存不全。

回到《當世界震動時》一書，繼續探尋亞特蘭蒂斯的藝術與科學。這失落文明精於天文學、磁力、重力與蒸汽技術，甚至微用電力。地下城的照明系統，正源於此。他們工於建築藝術，所造樓宇雕像可見一斑；更能翺翔天際，技術遠超我輩所知。

更奇者，其中有人掌握第四維度運用，能穿物越障，入出於無。他們擁有雙重形體之力：在一處將身體解體，再到別處重組，或將所謂「雙重體」——或稱「星光體」——投射至遠方。這星光體雖不可見，卻能感知、能見、能聽、能記，回到身體後經驗猶存。

伊娃說，心靈感應於智慧之子亦是天賦；他們嫻熟使用此能力，如同我們使用無線電。不過，其收發是由經驗豐富者輪班值守，故信息傳得又快又準。她還說，有此天賦者善解人心，不易受欺。

以伊娃對「愛」的見解作結，再恰當不過。「愛」在二十五萬年中不曾改變，永不會變，因創造一切的造物主，不正是名為愛嗎？

「漢弗萊，聽我說。塵世、天界、地獄之間，無任何屏障能阻愛力通抵完滿。愛必存在，以諸形貌顯現，堅持前行，直到終點；這終點無關肉體。無論是忘我之愛、輕賤之愛、挫敗之愛、虛妄之愛、背棄之愛、迷途之愛、漂泊之愛、沈睡之愛、活在夢中之愛，甚至醒中猶眠——但凡愛，皆蘊生命種子。愛的形貌並不緊要。若此是真愛，我告訴你，愛終將得勝；即便它似乎崇拜諸多對象，仍會找到那唯一的一個——縱或不在眼前。」

聞此言，一股寒意攫住我心。「不在眼前？那在何處？」我問。

她答：「去問你逝去的妻，漢弗萊。去問靜默星辰。去問你所拜的神，我無法給你答案，只一詞——某處！莫怕。你以為像你我這般人，會沈淪苦淵？我所知的不多，但我告訴你，我們是其主宰。告訴你，只要渴望不息、信念不滅，我們亦是神靈。對懷疑怯懦者，那裡空無一物；對以靈眼觀照、伸手探求者，那裡便有一切。」
