# 《幻想交響曲》
## 這部作品以象徵與寓言交織，鋪陳魔鬼的陰謀，同時深掘身體、心智與靈魂層層交疊的奧秘。

![封面](/img/downloaded/9dd64495a58d253abe45c2b386e4db49.png)

電子書連結：[點擊閱讀](/books/fantasy-symphony-dr-michaud)

字典裡將「魔鬼怪談」定義為「描繪黑魔法或與魔鬼往來的文字圖像」。這部「文字交響曲」雖處處描繪魔鬼，卻不止於陰謀奇幻。與作者所有著作一樣，《幻想交響曲》滿載繁複多層的象徵、寓言與隱喻，唯有深入研讀冥想，方能窺見堂奧。並非說一般讀者無法喜愛此書，而是它最深刻的真理與教誨，遠遠藏於表象之下。這個「魔鬼怪談」的宏大主題，實是歷經苦難而得的救贖與重生。它與彌爾頓史詩《失樂園》頗多相似，但彌爾頓意在「向人類證明神之道」，米紹博士卻透過揭示若干偉大隱藏的奧秘，闡釋神之道——至少在那可被揭示的範圍內。

作者雖是技藝精湛的音樂家，敘事結構亦緊密遵循古典交響曲的四樂章形式，這卻不是一本關於音樂的書。一九四二年，英國作曲家格蘭維爾·班托克爵士（1868–1946）初見手稿，驚嘆：「太精彩了，必將轟動！」此書主題崇高、語言優美、蘊含豐富神秘學知識，確實氣象恢宏。然而出版後並未引起轟動，如今幾乎無人知曉，也少人閱讀。正如我們在多篇文章中指出：真理向來是最不受歡迎的商品，這結果並不意外。但這無關緊要。識貨的讀者自能在《幻想交響曲》中，發現與作者其他作品同等的美、啟發與深奧隱秘，感受那種欣悦的「觸動」，對每一頁流瀉的智慧、溫暖和幽默，滿懷深切的感激。這是一部以文字織就的燦爛交響。

書中重大事件發生在距今約四十二萬七千年後，正值迦梨時代或「黑暗時代」終結之時——這段歷史可在作者另一著作《克里希納的真實福音》中讀到。這齣形而上學戲劇的主要角色包括：冥界王子長短格與其惡魔部屬；兩個高度進化的靈魂，馬德隆與塞巴斯蒂亞諾；以及睿智的吉普賽老婦米澤拉——她將大自然隱藏的秘密傳授給杜爾斯與法尼，那對孩童正是馬德隆與塞巴斯蒂亞諾的小我。書末附有完整角色清單，以及那些完美交織於敘事中的芭蕾與舞蹈。

從未有哪本書，能如此清晰正確地解釋身體、心智與靈魂的本質，以及「星光體」投射、「墮落天使」與下層星光界的情形。也從未有書如此深刻地刻畫「魔鬼」的形象——那高傲的地獄王子長短格。第二樂章中，長短格揭示了黑暗勢力用以在世間播撒邪惡種子的工具與詭計。這些玄秘真理從未如此公開陳述，但時機已然成熟，可將其更廣泛傳播：令善良之人從中受益，而那些背棄光明、擁抱黑暗者則將明白，等待他們的並非美好之事。

一個人即使讀遍千本知名作家的「神秘」著作，所能發現的偉大真理與崇高智慧，恐怕還不及這本薄書（印刷版僅一百零一頁）的一小部分。這部宏偉戲劇在第四樂章達到高潮，擬人化的行星及其統治者向長短格與其地獄軍團發動戰爭。最終交戰前，諸多事件接連發生——有些洋溢巨大喜悦、美德與美麗，有些浸透極度悲傷、罪惡與醜陋。每一事件都揭示若干真理或智慧寶石；這是其他作者任何書中都尋不著的天界珍藏。

在結束這篇簡短而遠非全面的介紹之前，必須再次聲明：這不是一本關於音樂的書。然而，讀者中若有古典音樂家，或能在第三樂章諧謔曲裡，從那些湧現的古樸樂器、樂師與舞者間，窺見更繁複層次的含義。我們曾試圖點算出現的人數，數至數百便放棄了——或許正因如此，《幻想交響曲》從未搬上舞台。要尋得一位非凡的製作人，與一群既勇敢又天賦超凡（甚至堪稱魯莽）的音樂家，幾近不可能。

即便如此，這本書——或戲劇，或交響詩，任你稱呼——因其包容上述一切，且不止於此，終將上演一齣空前宏大的樂劇，實現七十餘年前格蘭維爾·班托克爵士初讀手稿時所預言的「轟動」。誰知道呢？或許你讀罷此書，為之傾心，進而將這部「文字交響曲」引薦予更多讀者。那會是真正高尚之舉，亦是對作者不朽才華最恰當的致意。
