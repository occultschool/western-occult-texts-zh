# 三相神：印度教的三位一體
## 擬人化的面具之下，蘊含著高度哲學性

![cover](/img/downloaded/5b22da2d11237dbb46ad7f8567b26a5b.jpeg)

三相神，梵語稱 Trimûrti，字面意思是「三張臉」或「三重形式」，即三位一體。當代萬神譜中，這三者是創造者梵天（Brahmâ）、保存者毗濕奴（Vishnu）、毀滅者濕婆（Shiva）。然而此觀念成形較晚，《吠陀》裡既無梵天也無濕婆。吠陀最初的三位一體，是火神阿耆尼（Agni）、風神伐由（Vâyu）與太陽神蘇利耶（Sûrya）；如《尼祿多》（Nirukta）所解：大地之火、大氣之火與天界之火。

《蓮花往世書》（Padma Purâna）記載：「最初，偉大的毗濕奴意欲創造世界，遂化身為三重：創造者、保存者與破壞者。至高靈從右側流溢出梵天，以司創造；從左側生出毗濕奴，以司保存；從身體中央化現永恆的濕婆，以司毀滅。有人崇拜梵天，有人崇拜毗濕奴，有人崇拜濕婆；然毗濕奴雖一體卻三重：創造、保存、毀滅。於真正虔信者，三者實無分別。」

三相神的本質，實則象徵從「靈-物質」分化出的三種基本屬性（Gunas）：自我形成、自我保存、自我毀滅，旨在促其再生與完善。

確切來說：梵天是「激性」（Rajoguna）的人格化體現，此屬性為活動與繁衍的渴望，宇宙萬物由此而生。毗濕奴擬人化了「平性」（Satvaguna），象徵清明、平衡、和諧與覺知。濕婆則是「惰性」（Tamoguna）的擬人化，此屬性為停滯與衰敗，自然成為毀滅者。但這並非「終極毀滅」，而是回收、溶解與潛伏——讓形相歸於未分化的狀態，為下一輪生成準備條件。

然而，擬人化的面具之下，蘊含著高度哲學性。這些原始概念若囿於字面，既不哲學，也顯荒謬。
