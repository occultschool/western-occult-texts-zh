# 揭開詭詐文本與真理迷思

## 從經文解讀的偏差與教義迷思，引導讀者辨識真偽與避免被權威所動。

![cover](/img/downloaded/tricky_text.png)

### 介紹

自上世紀起，東方哲學與神秘主義風行西方，至今未衰。我們在另一篇討論藏傳佛教的文章裡提過，此現象好壞參半。佛教最大的長處是易於調整——果真如此麼？倘若一種教導必須「調整」才能適應不同文化與時代，它本身便不可能是完整的。教導要麼真實，要麼虛假；要麼完整，要麼殘缺，道理顯而易見。今日人們所知所傳的佛教，與釋迦牟尼的教誨早已面目全非。此種流變在各種宗教、哲學與靈性教導中屢見不鮮。創立者深知自己所說，懷抱全部智慧，自高處領受啟示；其內在本體煥發巨光，那光包容並吸引真正的智慧。他是持炬者，手中火炬熊熊燃燒；待他重返塵世，看啊！火炬已然熄滅，黑暗吞沒了神聖的光。或許尚存餘燼，但那不是光。唯有真正的信使才擁有那光，旁人不過是一點微亮。

因此，神聖文本裡才充斥缺陷與謬誤。以下段落摘自《法華經》（亦稱《妙法蓮華經》）第五章，此乃大乘佛教最著名的經典之一。我們將剖析其中隱含的瑕疵與錯處。

### 《法華經》

「諸神與世人，我乃如來！我是阿羅漢，全然覺悟之人：我已抵達彼岸，亦要領人至此岸；我得自在，亦使人自在；我獲安慰，亦予人安慰；我已安息，亦引眾生得安息。憑我圓滿智慧，我知此世與彼世實相。我無所不知，無所不見。諸神與世人，到我面前來。聆聽此法。我是引路之人；我是示道之人，因我知曉道路，熟稔途徑。

「我將令形體枯槁的眾生重獲生機，他們困於三界不得出。我賜福於勞苦憔悴之人，予其喜樂與終極安息。諸神與世人，聽我之言；近前觀我：我乃如來，是主，無有更高者，我顯現此世只為救渡。我向千萬眾生宣講純粹簡樸之法，此法唯一目的，乃智慧、解脫與安息。我以恆常不變之聲說法，以覺悟為本。此法對眾生平等；無有偏私，亦無愛憎。我無情無緒，於人無愛亦無恨，唯對此眾生宣揚法則，一視同仁。

「我重造整個世界，如雲降雨不分差別；我待尊貴與卑賤者等同；視有德與無德者無異；墮落者與持守善行者齊平；抱持派系之見與信奉虛妄者，亦與見地正確者同等。我既向智慧低下者說法，亦向悟性高超、才能卓越者說法；我不覺疲憊，只在適當時機遍灑法雨。」

### 文本分析

這段話何等懾人，不是麼？正如前文所述，《法華經》是佛教極負盛名的經典。但此處摘引的內容果真正確麼？這位化身以全然無我的姿態說話，恰如他所宣稱：儘管帶來重生、喜樂與愉悅，他對誰都沒有恨，也沒有愛。他聲稱自己擁有圓滿智慧，無所不知，無所不見——然而這等屬性當歸於萬物之父。他說自己無私無偏，對有德者與無德者一視同仁。真有如此模樣的化身麼？然而，數百萬人曾深信這教導的每一字句，至今依舊，且無疑將繼續下去。因為這般咄咄逼人、自吹自擂、漠視人情的言語，能使人們愚蠢的低下頭，被「權威」震懾而順從；儘管現代「民主」的「平等」觀念也竭力說服我們：下屬並不比老闆差!

鮮少有人能抵擋這位自封「如來」者的誇飾之言。過去與現今的獨裁者皆運用同樣的技倆與手段，而能抗拒者寥寥。然而，這些話語與《克里希納的真實福音》中克里希納的高超教義何其不同，與耶穌《登山寶訓》裡的仁慈與耐心更是天差地別。

我們引用《法華經》經文，是將其視為錯誤信仰與教義的極端例證；它甚至能將最聰慧的人引入歧途。如今，這類「老師」依然充斥世間，不少讀者或曾邂逅，而我們自己亦遭遇多次。

當然，若對這些經文稍加編輯，令其顯得合情合理，甚至神聖非凡，那是絕對辦得到的。

然而這正是危險所在：學生若缺乏引導，未能識破那些似是而非的雄辯，便易陷入謬誤，甚至更糟。分辨二者並不容易。前文提及，此處所謂「如來」對男女之別、道德與否、賢愚善惡皆不在意——這實屬最惡劣的詭辯。或許你覺得這謬誤顯而易見，但請誠實自問：初讀《法華經》此段時，心中留下了什麼印記？

倘若心中自有光明映照，謬誤便無所遁形。但若初讀時心頭掠過一絲陰影——誰不曾有過這樣的瞬間呢？——這段似是而非的文字，或許就讓你相信那是賜予人類的美妙訊息，是救世主的真言，處處值得尊崇。從畢達哥拉斯到恩培多克勒與柏拉圖，從普羅提諾到希帕蒂亞，直至中世紀的神秘主義者，哲學長河裡這列莊嚴的大師，皆在探討「常識」以外的領域。那並非單憑理性推論所能觸及。這類課題只能用隱晦艱澀的方式，藉悖論、寓言與神話來表述；尤其在不理解神秘科學的真理與法則時，任何智識上的嘗試終屬徒勞，無法帶來真正或實際的光明。

多數深奧之事，其實能以淺白的話語說清；正如[《論神秘語言》](/docs/mysterious-language-unlocking-ancient-wisdom)的文章所言，我們可以從不同角度，用寓言與比喻探討此題。這正是耶穌、佛陀等偉大師者所用的方法。他們深諳宇宙法則的整體，從不採用浮誇傲慢的言辭，因那違背了神聖的法則、光明、真理與愛。細心的讀者立刻會明白，這正是我們剖析《法華經》等段落的緣由。我們無意貶損他人之作——那些試圖揭示生死大問與存在意義的著述。但仍須指出其中埋伏的陷阱、死胡同與扭曲，以警示走向光明之路的人避開險徑，方是對神聖真理盡了本分。

### 理性與直覺、靈感

傑出的東方思想家與前述西方哲人一樣，深知僅靠「推論」不足以把握並表述真理。思考，本質上都是一種無聲的內在交談。凡是無法用現有詞彙或符號來構築的，便不存在於當下的思考之中。若要突破那些未被言說或書寫之物的界限，需要一顆無畏且熱忱的心智，付出特定的創造性努力，從而洞見那被遮蔽的真理——隨後再付出一番努力，通過鑄造新詞，將真理帶回語言的疆域。真理無名、未知、似有若無，必須以充滿愛的心去追尋、發現，並透過心智帶回言語或文字之中——而在表述時，謬誤難免滲入。唯有極少數人的思維能受啟迪，並保有這些真理，從此不再失去。一般人常困於此難。最好的方法是即刻記下靈感與內在啟示的內容，竭力持守。如此方能尋得並保留內在的知見，讓心智得以進化、開展。這任務艱巨，卻在各方面皆值得。

前蘇格拉底時期的希臘思想家與詭辯士，幾乎摧毀了本土的神話傳統。他們未曾意識到，神話中埋藏著許多偉大神秘學義理的根基。神話常遭誤解，甚至破壞；但若配合前文討論過的神秘鑰匙，加上經驗與耐心，便能發現其中隱藏的奧秘。神話混雜真偽，須學著分辨。可嘆的是，亞里士多德等詭辯家及其追隨者，在探討宇宙與人類的本質及命運之謎時，寧願倚仗當時興起的自然科學邏輯——數學、物理學、天文學；卻未察覺這些學問已有數千年歷史，古人對此所知更深。古老的象徵神話在詭辯者的強勢影響下，淪為僅僅優雅有趣的故事，成了上流社會複雜的情愛與閒談的點綴！

然而印度情況大不相同：神話被用以促進哲學思想的表達。史詩傳說中豐饒的圖像文字、眾神各異的特質，其轉世與功業構成了無數神話，以及外傳與秘傳宗教的象徵符號；教師一再借取這些象徵作為傳達的載體。神話與哲學於是締結了美妙的友誼，最新與最古的、至高與至微的彼此交融，從而賦予印度文明靈性的深度。即便在現代，印度的民間傳說與通俗神話，仍將哲學家的真理與教誨帶到眾人眼前，不論那是事實或想像。在這象徵形式裡，觀念不必經過簡化便能深入人心，一些生動的圖畫手稿完好保存了教義，絲毫未損其神秘內涵。

這些教義在今人眼中或許為真、或許為誤，但無論如何詮釋，保留些許象徵性思想，總比陷入西方粗鄙的唯物主義要好——他們對於「靈」的概念，往往僅透過否定來認知。印度哲學對文字本存懷疑，這亦是明智的，因為無人能以完全一致的方式詮釋同一句話。印度哲學家疑慮語言是否足以承載哲思的重要主題；因此，他們極其謹慎地將宇宙與人類存在之謎轉化為純智性的形式。當我們閱讀《法華經》這類作品，思及那位自稱「如來」、「阿羅漢」與「救世主」的虛幻表象時，幻覺便如龐大難解的雲團壓頂——因沒有救世主會以此方式同時向神與人說話。這非但無法如他所宣稱的「使人自由」，反而奴役了那些相信其虛幻存在與錯誤教導的人。
