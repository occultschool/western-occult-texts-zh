#  行星鏈、輪次和根種族進化的神秘學教義
##  解密我們所屬的行星鏈與其隱藏的高等層面，探索宇宙進化背後的深奧奧秘

![illustration](/img/downloaded/6f3e4df06dd1cc3fd545e95e8455791d.png)

我們的地球是行星鏈的一員，此鏈包含七個星球，被稱為「地球鏈」。因此，我們所處的這顆行星有6個伴星球，但我們看不到它們，因為它們存在於不同於我們的三個層面之上。

正如我們在另一篇文章中所述，這是「基本的神秘學教義，每一個已知的星球，例如地球、火星、水星、金星、土星、木星等，事實上都構成了七個星球的七重系統，但其中只有一個存在於物質層面。換言之，每一個物質星球都與六個非物質星球密切相關，從而持續的進化。」

我們地球所在的層面，再加上其他六個星球所處的三個較高層面，一起被稱為「宇宙意識的四個低等層面」。整個宇宙由七個層面組成，包括四個低等層面和三個高等層面。這就像一個低等的四元組和一個高等的三元組，就像人類的七原則一樣，其中有四個凡人的低等原則和三個不朽的高等精神原則。

在這張圖表中，我們看到我們行星鏈的七個星球是如何對應於宇宙意識的四個低等層面。跨喜馬拉雅兄弟會的大師們認為這些關於行星鏈、星球等教導和知識，是極其重要和有用的，且布拉瓦茨基在早期仔細而準確地教授了這些知識。在那之前，這是一個完全神秘且深奧的教誨，僅保留給受選擇的門徒和啟蒙者，從未對外界公開。

然而，在布拉瓦茨基夫人離世後，阿迪亞爾神智學會的一些領袖，如C.W.李德比特和安妮·貝桑特，開始教授和推廣辛尼特早期的錯誤觀念，認為我們行星鏈的第三顆星球是火星，而第五顆星球是水星。

這種缺乏邏輯的教導帶來了許多混亂和誤解，直接違背了大師們和他們所傳授的「秘傳」。當然，在天文學和占星學上，這樣的說法是荒謬的。

此外，如果這兩顆行星是我們行星鏈的一部分，那麼我們將無法看見它們，也無法瞭解有關它們的任何事情；因為行星鏈的第3和第5個星球在「基質\-成形層面」，是完全超越物質層面的感知。然而，我們確實能夠觀察火星和水星並了解它們，因此可以確定火星和水星無疑是物質層面上的行星，並且必定屬於各自獨立的行星鏈。一條行星鏈上只能存在一顆物質行星，就像人類只有一個身體一樣。

類比和對應法則貫穿於所有真正深奧教導中，如同一條美麗的線索，但顯然在廉價的模仿品中缺乏或被扭曲。

在神秘學教義中所提及的行星鏈、星球、輪次和根種族等概念，看似錯綜複雜，令人難以理解。然而，神學運動的先知們認為我們有必要深入了解這些概念，因為這不僅揭示了我們地球的真實本質、起源和未來，還提供了重要的象徵性啟示，透過類比和對應的法則揭示更大的奧秘。

以下總結的信息希望能清楚闡釋這一教義的基本內容：

\* 地球的進化涵蓋無數相連的行星鏈，每個行星鏈都由七個星球組成，如圖中所示。

\*當前正在進化的地球鏈（我們將其稱為「地球鏈」）不是唯一的行星鏈。在它之前，已經有其他行星鏈完成了它們的進化過程並停止了發展。而未來將會有更多的行星鏈陸續出現，因為每一個行星鏈都是基於前一代的基礎上發展而來的。

\* 我們前一個行星鏈被稱為「月球鏈」，因為那個被稱為「月球」的死行星是該鏈的第四顆星球，現在成為我們地球的衛星。換言之，我們現在的月球曾經是一個活躍的「地球」，我們的生命曾在那個星球的原始環境中演化。當然，這些事件發生在數十億年前，對我們來說幾乎難以想像。

\* 進化的「生命之波」在各個鏈上通過7次，這七次被稱為七個輪次。在每個輪次中，我們可以看到人類達到明顯更高的進化水平，包括內在和外在的進化，但其中最重要的是內在的進化。每個輪次都始於星系中的第一顆行星，使其變得活躍並孕育生命。每顆行星都經歷了七個主要的文明時代，在神秘學中被稱為七個根種族，因為在每個時代都出現了新的主要「種族」或「類型」，並且逐漸主導。

\* 當七個根種族完成了他們的進化旅程後，生命波就會移至第二顆行星，依此類推，直到第七顆行星，每顆行星都會誕生七個根種族。當進化的「生命之波」離開行星鏈中的一個星球轉移到下一顆時，被遺棄的行星就進入一個靜止期，僅存最基本的生命形式（非人類）……直到生命波再次抵達該特定星球才會恢復，這將發生於下一個輪次。

\* 我們現在位於地球鏈的第四個星球，在這個鏈的第四輪次。我們當前的地球文明處於第五個主要時代，也就是第五根種族。根據教導，在這第四輪次的地球上，第一個根種族被稱為極地根種族，第二個是許珀耳玻瑞亞根種族，第三個是雷姆利亞根種族，第四個是亞特蘭蒂斯根種族，第五個則是雅利安根種族或「印度\-高加索」根種族。

\* 每個根種族都包含七個亞種族。在未來數百年裡，第五根種族的第六亞種族可能會逐漸浮現，但距離第六根種族還有很長的一段路要走。

\* 今天我們大多數人曾經是月球鏈上的動物，現在是動物的下一個鏈將成為人類。因為每個單子（Monad，也就是「神聖火花」，源自絕對者）都經過礦物界、植物界、動物界，進入人類界，然後逐漸進化至更高的存在。生命永遠不會停滯不前，而是不斷發展、前進、進化，永遠向前、向上、向神。當我們抵達下一個行星鏈時，現今的人類將進入更高、更靈性的存在狀態。

\*在偉大的進化計劃中，生命之波在我們地球鏈的每一輪次 ( 以及每一輪次的各個根種族)時不斷流轉， 旨在完善人類組成中的七原則之一。然而，若我們對七大原則一無所知，或不甚了解……若我們摒棄並拒絕大師們對七大原則的啟示（他們視之為至關重要且不可或缺的，是其秘傳教義的基石），轉而支持C.W. Leadbeater、Alice Bailey、Annie Besant等人提出的關於人類構成的虛偽神學教誨……我們將難以領悟事物演變的真實本質。

\*在這星球鏈的第五輪次中點，即將迎來「抉擇之時」，即所謂的「審判日」，屆時，將區分出那些準備持續內在進化、不斷向前邁進的人，以及那些不準備這樣做的人。那些不願如此者將被視為「不合格」，直到下一個星球鏈之前，他們將不會再轉世，他們的靈魂成長和進步也將被無限期地延後。這全因為他們在靈性和心智上的懶惰，以及拒絕付出努力。這樣的情況發生是為了確保人類進化的持續前行，不受那些拒絕進化的靈魂所拖累。

\*每一顆行星在太陽系中的演化都受限於這些「宇宙意識的四個低等層面」的顯化。循環演化的法則，涵蓋行星鏈、星球、輪次、根種族等，適用於我們太陽系的所有行星。正如布拉瓦茨基和其他大師所傳授的，我們太陽系中的幾乎所有行星，包括太陽系以外的行星，都有生命居住的。

然而，這並不代表所有在太空中看到的物質行星（如火星、金星、土星、木星等）都有生命存在其中…這僅當這些行星正在經歷生命與進化，並且位於行星鏈中的第四顆星球時才會如此，因為第四顆星球是唯一在物質層面上具有物質性且可感知性的。然而，在一個行星鏈中還有其他六顆星球，它們都是非物質的；因此，若生命的進化和「人類」的顯現發生在其他六顆星球上，我們將永遠無法感知、找到或理解它的存在。大師K.H.指出，火星作為物質星球目前處於完全不活動的狀態，因為 「生命之波」已經轉移到一個非物質星球上，而「生命之波」在水星處於 「矇昧狀態」後開始了一個新的輪次。

\* 金星系統與我們地球系統有著密切的聯繫。布拉瓦茨基在《秘密教義》中提及，金星和地球在神秘科學中被視為「孿生姐妹」，但「地球的神靈服從於金星的主宰」。她繼續說道：「金星是所有行星中最深奧、最強大和最神秘的；它對地球的影響和關係是最為顯著的。」

它也傳授說，金星是「地球和人類的守護靈」，並且「根據神秘教義，這顆行星（金星）是我們地球的根源，是其精神原型。」我們被告知金星上每一個變化「都被地球感知與反映。」布拉瓦茨基補充道，金星這個行星「無論是在物理還是在神秘意義上，都是我們地球的『光之承載者』。」金星是「明亮的晨星」，這是《聖經》對路西法和基督的稱呼。

## 金星和地球的象徵

有些人在閱讀或聽到這些相當技術性的神秘學教義時，例如行星鏈、星球、輪次、根種族、行星和星系進化等等，會說：「這一切聽起來很有趣也有點複雜，但似乎對於人類沒有任何真正或實際的好處和重要性。研究和瞭解這類事情到底有什麼好處與用處呢？」

首先，研究精神性的教導、形而上學和哲學本身就是一種寶貴的精神鍛鍊。它有助於提升我們的心智和意識，並超越普通日常事物和感官，讓我們對於更高更偉大的現實有靈感和直覺的一瞥。僅僅這一點就足以鼓勵人們努力研究和深入瞭解神秘學的更複雜方面。

最後還有一個非常重要的事情需要提及，這常常被忽視或被遺忘。用威廉·賈吉在《神智學的海洋》一書中所提到的《地球鏈》一章來說：

這七個「星球」組成了單一大球體，且它們相互滲透。它們彼此相互滲透。我們稱之為「星球」是因為它們的最終形狀是球形。如果一個人過分依賴辛尼特先生 (Mr. Sinnett)的解釋，那麼他可能會認為星球之間並沒有相互滲透，而是由電流或磁力線所連接。如果一個人過分專注於《秘密教義》裡解釋過程所用的圖表，而忽略了布拉瓦茨基提供的解釋與警示，那麼他也可能會犯下相同的錯誤。她和她的開悟老師都說，我們行星鏈上的七個星球是「相互結合的，但不是同質的。」

「它們作為星球與我們地球是相互結合的，但非同質的，因此屬於完全另一種意識狀態。」 — 《秘密教義》，［第1卷，第166頁]

「這進一步告誡學生們不要依賴統計數字或各層圖表，而是要關注該理論的形而上學和精神層面。因此，我們從辛尼特先生的書的資料來源（即K.H. 大師）得到這種說法，即這些星球雖然在本質上彼此不同，卻結合成單一球體；而這種本質的差異是由於意識中心的變化。」
