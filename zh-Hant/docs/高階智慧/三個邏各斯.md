# 三個邏各斯
## 真理只有一個，聖人卻用了許多名字稱呼它。

![封面](/img/downloaded/a42ae3844edbec1e5ae684e6f6535e13.jpeg)

**問：** 讀完[《邏各斯的哲學基礎》](/docs/logos-and-mysticism-philosophical-foundations)一文，我對「邏各斯」的實質更清晰了——那些內容源於布拉瓦茨基與大師的教誨，並非後世「教師」編造、嚴重曲解的產物。但布拉瓦茨基常提及「第一邏各斯」、「第二邏各斯」、「第三邏各斯」，這些術語究竟何指？我原以為邏各斯只有一個，即那「至一全靈之光與生命」。

**答：** 邏各斯的確只有一個，那便是宇宙邏各斯。每個大顯現期之初，它自「絕對者」輻射而出——無論你稱那絕對者為「梵」、「本初佛」、「無限」或其他名號。邏各斯是「絕對者」輻射的生命原則與進化能量，繼而化為活躍的宇宙本身。

正如另一篇文章所述，布拉瓦茨基與大師從未使用「行星邏各斯」一詞，僅有一次提及「太陽邏各斯」，用法也與偽神秘學家截然不同。真正的神秘學談論的是宇宙邏各斯，那是「第一因」；而「絕對者」是萬有的「無因之因」、「無根之根」。這個宇宙邏各斯，也稱作「核心靈性太陽」或「偉大核心太陽」。

若要弄清這三個邏各斯，不妨留意布拉瓦茨基在[《秘密教義闡釋》](/books/secret-doctrine-exegesis)中的定義。順帶一提，該書是不可或缺的優秀資源，釐清了《秘密教義》中諸多內容，包括邏各斯、七光線、行星神靈、絕對神性原則、《多基安集》、時間與永恆等，還有一章啟發人心的論夢篇章。

書中，她將三個邏各斯稱為「進化的三個靈性階段的擬人化象徵」。

因此，它們實際上只是符號，旨在呈現關鍵概念供人領會。現在，讓我們特別關注這三個邏各斯的幾個要點：

第一邏各斯總被稱為「未顯之邏各斯」。其象徵是圓心一點。圓圈代表「梵」那無邊無際的絕對性，圓心一點則標誌著宇宙分化的起點（或者說重新開始），發生於大休止期之後。宇宙大休止期中空無一物——「既無聲響，亦無靜默」——只有未分化、無制約、無限而絕對的梵本身。

未顯的第一邏各斯也稱「最高邏各斯」。據說它超越時間與空間，僅是「潛在的潛能」。第一邏各斯是宇宙的「潛能」，第二邏各斯則是宇宙的「效力」。第二由第一發出。由於第一邏各斯只是潛伏、潛在而未實現的狀態，它極其接近「絕對者」，是後者的直接輻射。除此之外，無法多言。但在神秘學教義中，我們注意到，作為人類最高構成原則的「阿特曼」，有時對應於未顯之邏各斯；儘管多數時候，阿特曼被等同於「絕對者」。這並不矛盾，反而是一種明確的暗示。

第二邏各斯是「顯現的邏各斯」。它本身其實只是「半顯現」，因為仍需透過第三邏各斯才真正顯現。在[《秘密教義闡釋》](/books/secret-doctrine-exegesis)中，布拉瓦茨基說它是「普遍而智性的靈魂，是神之意念，蘊含萬事萬物的理型藍圖與原型，不管是客觀或主觀的顯現世界。」

第一邏各斯被描述為「潛在的潛能」，第二邏各斯則被描述為「分化的意識」。我們得知，當第二邏各斯自第一邏各斯流溢後，「時間」與「空間」便開始了。未顯現的是潛能；顯現的是效力。

第三邏各斯便是「宇宙心智」，印度教稱之為「偉大」。正如印度寓言所示，神之心智產生了「七位心智所生之子」，這些兒子在宇宙中扮演主要而重要的角色。這七者的總和（實際是七道光線、七種力量或七種作用力），便是神秘學所稱的「第三邏各斯」。

第三邏各斯又稱七光線、七創造力或七邏各斯。如前所述，第二邏各斯本身便涵容第三邏各斯，並借由它而顯現。若將第一邏各斯視為潛存的能量，第二邏各斯為分化的意識，則第三邏各斯可謂第二邏各斯最終的展開——它以宇宙力量個體化之姿，從中「發展出無窮的階序序列」。

這「七者」實為宇宙七種奧秘之力，將宇宙心智中潛伏的主觀原型化為現實，使之客觀顯現。換言之，宇宙建構的藍圖，由它們實現。

問：我想我掌握了這番解釋的要義。但這說法是否太過抽象，太形而上學？

答：確實如此。然而，當我們談論宇宙誕生與演化的課題時，是如此形上且宏偉，又怎能使用其他語匯？任何簡化、擬人或人格化的嘗試，只會遮蔽它、褻瀆它。我們必須開始以形上、抽象、哲學的方式思考，這終將提升我們的意識，抵達更高、更真實的靈性層面。大師們深知這是我們所缺，因此《秘密教義》主要以此風格寫成。脫離哲學的靈性，往往只是蒙昧。大師們在信中提到，此時「正是一個更具靈性智慧的周期的黎明」。我們該成長了，該真正運用天賦的思考能力，而非停滯於現今的尋常狀態。

問：你提到布拉瓦茨基曾說，三個邏各斯其實是「演化的三個靈性階段的擬人化象徵」；照你所說，是否實際上並無三個不同的邏各斯，而是一個邏各斯經由三個階段推動宇宙演化？

答：我相信這樣理解正確——只要我們始終記得：那「推動宇宙演化的至一邏各斯」並非某種存在、實體或神靈，而是宇宙生命本身的能量；它依循永恆法則的內在脈動而回應、行動。此外，這「三個階段」劃分，也不似我們有限心智所期望的那般獨立分明。我們總習慣將事物切分清楚、放入整齊的格子；但稍加思索便知，大自然萬物多非如此。

問：這三個邏各斯是否有各自的名稱？我確信邏各斯本可無名；但若為說明方便，用什麼名稱描述它最恰當？

答：你說得對，邏各斯可以無名，正如「絕對者」必然無名。實際上也無須使用具體名稱，「第一邏各斯」、「第二邏各斯」、「第三邏各斯」已足夠。但我們仍能從各宗教與哲學中見到邏各斯的種種稱呼。正如你先前提到的文章所說：

「不同靈性傳統所用的諸多名稱，或會令人困惑；除非我們牢記，這些名稱指涉的都是同一邏各斯。《奧義書》哲學中，梵天即邏各斯，源自絕對之『梵』。卡巴拉中，天人亞當卡蒙是邏各斯，源自『無限』。密傳佛教裡，觀世音菩薩是邏各斯，源自本初佛。有印度教徒將『絕對者』及其邏各斯對應於濕婆與莎克蒂，亦有人以毗濕奴、那羅延或自在主稱呼邏各斯；更有人稱邏各斯為宇宙神聖母親。實際上，神秘學指出，邏各斯與所謂『邏各斯之光』不可分離，二者在印度思想中對應於自在主與莎克蒂——宇宙的男性能量與女性能量，後者傳遞前者的光、生命、能量與力量。」

須謹記，這些名稱並非指不同存在或實體，只是描述那賦予宇宙靈魂與生機的「邏各斯原則」。布拉瓦茨基寫道：「在秘傳哲學中，邏各斯僅是一個抽象術語。」

在秘傳學說中，未顯之邏各斯常稱那羅延。那羅延是毗濕奴的別名，可解釋為「神之意念的靈在水面移動」。還記得《聖經·創世記》開篇說，世界未成之時，「神之靈」運行在「深淵之水」上嗎？更古的印度經文則載，那羅延在抽象空間的無垠水面移動，這正是「最初顯化的顫動」，令宇宙在大休止期後循環重現。

不過，布拉瓦茨基有時稱梵天為第一邏各斯，有時又說是第二邏各斯。因此我們最好以她為榜樣——不必過於執著名稱，而應關注名稱背後的思想與概念。

七光線的總和，是為第三邏各斯。這概念在印度教中對應七鳩摩羅，即七位「心智所生、梵天之子」，或曰「宇宙心智之子」；在佛教是七禪那佛，在基督教是七大天使，在猶太教是七埃洛希姆，在卡巴拉則為七個下層質點。名相雖異，所指實同。正如《梨俱吠陀》這部人類最古老的經文所揭示：「真理唯一，聖人卻以諸多名號呼之。」

簡言之，三邏各斯作為「進化三階段的靈性擬象」，精確對應於人類物質演化的三階段：先是受孕，這不可見的神秘過程，是胚胎形成的初階，尚未顯化；繼而妊娠，胎兒在子宮內成形生長，始終隱於內裡，半顯半藏；歷經前兩階段後，人——這宏觀世界的縮影——終於誕生，客體全然顯現。

宇宙邏各斯之說源遠流長，自然非自布拉瓦茨基開始。但研習比較古代哲學後便會察覺，她關於邏各斯的教義，清晰而準確地重述了這永恆真理。若欲更深入、更全面地理解與探索，則必須研讀《秘密教義》，尤其是其中論及神秘象徵語言與「邏各斯」的章節。
