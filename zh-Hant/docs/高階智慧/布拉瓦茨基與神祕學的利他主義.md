# 布拉瓦茨基與神祕學的利他主義

## 布拉瓦茨基的觀點指出，自我與周遭生物界間的分離感只是幻覺，真正的神祕主義在於消解這種錯誤的觀念。

![封面](/img/downloaded/ffccf6cda2e1fdab4c4d384feaf43643.png)布拉瓦茨基 著

神祕學的根本教義是：我們在自我與萬物之間劃出的那道「界線」，實為幻影。真相是，萬物本然一體，這並非情感的喧嘩，而是冷靜的覺知。東方哲學恆言，宇宙之中唯有那「至一本體」真實存在；我們自認的「自我」，不過是這本體投射於塵世的、一段稍縱即逝的虛幻映像。

真正的神祕主義，便在於拆穿「自我」這道幻幕。心智圓滿與智慧的表徵，即是將有限的「自我」等同於那宏偉的「一切」。除非人類作為一個整體取得進步，否則想要在靈性層面提升幾乎是不可能的。個體永恆的幸福繫於社會整體的福祉——個體本就鑲嵌於整體之中，不可分割。

因此，神祕學的利他主義與「斷除物慾，追尋靈性圓滿」的教導，並無衝突。因為靈性的圓滿與真知，唯在靈性層面方能抵達；唯有在此境界下，所有隔閡、自私、個人慾念，才會消融於人類共通的、更廣大的意識之海。

這也意味著，要求人盲目服從他人指令，毫無意義。每個人都必須親身跋涉，經歷考驗，學會辨認何者有益於人類。隨著他靈性成長，逐一擊潰自私的壁壘，他的心智便會敞開，接納其內在那神聖單子的指引，即他真實「本體」。於他而言，過去與未來俱皆消隱，唯剩永恆的此刻……

與真實靈性生活的喜悅相比，物質生活的幸福便顯得輕淺。人類在塵世逗留數十寒暑，遠不及在主體層面度過的百萬年之久。

至於能力與天賦的培養，答案極為明澈：應當培育它們，用以服務人類。畢竟我們同屬人類社會，應當無條件地為其福祉盡心。
