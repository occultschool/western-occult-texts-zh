# 人類靈魂的永恆進化：從礦物到神性
## 靈魂轉世的真相：人類靈魂不會退回到較低界別的進化之謎

![cover](/img/downloaded/e4f201e410760fd9f75a6aefd95bff34.png)

[《對轉世的正確理解》](/docs/what-is-reincarnation-correct-understanding)列有十點說明，其第四點闡明：「人類總是轉世為人類。」文中指出：「自然界劃分不同領域或部門，各自代表演化中清晰獨特的階段。靈魂一旦從某領域進化至下一階段，便無法退回已完成的週期與階段。演化雖緩慢漸進，進程卻清晰有序。」

「從礦物界進入植物界後，便不能再成為礦物；自植物界跨入動物界後，也無法重返植物界。一旦我們以具自我意識的個體步入人類領域，歷經漫長系列生命，就絕無可能退回低層次演化——那已是全然通過的階段，不再對我們產生影響或作用。

「人類靈魂永不會轉世為動物、樹木、花草、石塊，或任何非人的存在。」

神秘學固然承認，人類靈魂在意識上可能墮落至近乎動物的境地，並在連續人生中逐漸趨近獸性，但它堅信：縱然遙遠的過去我們或曾隸屬動物界，轉世之後，卻絕不可能重返動物界或任何更低層的領域。

然而，諸多印度教、佛教或其他哲學體系卻主張：因惡業之故，靈魂可能輪迴轉世為動物乃至更低等的存在。對此，我們應視之為迷信或無知嗎？抑或這些教誨背後潛藏著某種真理？

布拉瓦茨基在《生命原子的轉世》中寫道：「此說立基於一項真理——更準確地說，是一項公理般的真理，但僅牽涉人類的原子及其流溢；這不僅發生在人死後，更貫穿其一生。《摩奴法典》（第十二章第三、五十四及五十五節）記載：『每一行為，無論屬思想、言語或身體，皆結善果或惡果（業）；人（非指靈魂）的各種轉世，所經歷的最高、中間與最低階段，皆由其行為所生。』又言：『殺婆羅門者將投身為狗、熊、驢、駱駝、山羊、綿羊、鳥禽等。』此處的秘傳真義，並非指向人類的自我，而是指其身體的原子、低等三元組，及其流體發散物。」

神秘學指出，每個人在投生時，皆由七個「原則」或組成部分構成。其中三者——阿特曼、菩提與心智——具備靈性且持久不滅；其餘四個低等原則，包括欲望、生命能量、星光體與肉體，則屬暫時、必死，僅存續於特定生命期內。它們共同構成小我，即當下的人格。此結構在[《人的七重性質》](/docs/blavatsky-seven-principles)中有更深入的闡述，剖析了這七項原則的本質與功能。

布拉瓦茨基進而解釋：「婆羅門為了自身利益，曲解了法則中的真義。前述引文的原意，絕非後世所解讀的那般。原本『婆羅門』在寓意上代表人第七原則，即其不朽的單子與自我的本質；婆羅門卻自私地指他們自己。凡是在自身之內扼殺或熄滅婆羅門之光者——即將自我與阿特曼切斷、從而扼殺未來『天界者』——便成了『殺婆羅門者』。此人不但沒有過上德行生活與追求靈性，心智無法與菩提相融，反而以其惡行，任憑慾望創造出磁性親和力，導致自身每一低等原則的原子，都被吸入低等動物或畜生的形體中。

「這正是『靈魂轉生』學說的真義。」她總結道：「這並非懲罰某人，使其粒子與動植物原子混雜。絕非如此。此種混合反映一種因果關係，其影響會顯現於未來轉世之中，除非當事人的人格已被抹除。從起因到結果，每一個果都將成為新因，沿著輪迴循環運行；最初的衝動唯有在休止期的門檻上，方得自行消散。但那終將降臨的時刻，必會到來。」

業力的重擔，始終壓在每個人的肩上。要到何時，我們才肯認真看待自己的每一念、每一言、每一行？終有一日，那樸素而巨大的真理終將顯現：世間沒有無果之因，也不存在毫無回響的行動。此時此刻，我們必須學著成為自己的主人——《入菩薩行論》中，寂天早已說得明白：「從今而後，我當成為自己的主宰，世界的僕人。」

威廉・賈吉在其短篇《波斯學生的教義》裡亦有觸及此主題。書中人物如此說道：「那些細微的原子，時時刻刻在我們周身飄移。它們尋覓著相應的中心，投向性質相似的事物……我們從同伴身上，吸納與自己相近的成分。人在這粗重的世界輪轉生滅，他既是自然的統治者，也是關鍵、是焦點，是天地實驗室中最精密的聚合體。他曾施加於禽獸身上的原子，終會在未來的某個生命階段，重返其身，帶來傷痛或哀愁。但他──那永恆的部分──不會墮落；墮落的，是低劣的、個體的、原子般的殘渣。他是所有低於他者的兄弟與導師。當心，莫因你道德的殘缺，阻滯並拖累整個大自然。」

因此，「每時每刻自我們身上飄散的那些原子」，正憑著磁性的親和與吸引，尋覓它們應有的歸處。布拉瓦茨基曾說：「這不需開悟者的智慧，只需一點清明的靈視，便能看見它們在人與物之間往返穿梭，恍若一簇幽藍閃爍的火焰。」
