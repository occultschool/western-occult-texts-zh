#  老子與道：神秘學的洞見
## 「千里之行，始於足下。」- 老子

![cover](/img/downloaded/74a8e69acecec197f04438be8684b318.jpeg)

老子，歷久不衰的靈性導師與聖人。

他代表「中國智者」的典型形象。其著作《道德經》譯本之多，世上罕有書籍能及；僅次於《聖經》、《小王子》與《木偶奇遇記》，位居第四。

布拉瓦茨基亦曾提及老子，次數固然不及佛陀或克里希納等偉大導師，卻也足以引人注目。

在《神秘學詞彙》第186頁「老子」條目下，她簡潔道：「孔子之前的聖人、聖徒與哲學家。」世人相傳老子為孔子之師，兩人皆生活於約兩千五百至兩千六百年前。

第320頁「道德經」條目則闡述更詳：

「字面意為『自然完美之書』，乃哲人老子所著。其宇宙開創論，蘊含萬物誕生之根本原理。老子謂：太初惟有無限空間；其間一切存在，皆生於『自有自化之原則』，即自性。因其名不可知、其質不可測，哲家遂稱之為『道』——宇宙靈魂，乃大自然未創造、未生且永恆之能量，週期顯現。待大自然與人類復歸純淨，便臻休止，萬物終與『道』合一。『道』為一切幸福喜樂之源。一如印度教與佛教哲學所言，此純淨、幸福與不朽之境，唯憑踐行美德、達至靈性圓滿平靜方可得。須駕馭人的心智，終至制伏，乃至粉碎人性中動蕩的物質本性；道德淨化愈早達成，幸福愈早降臨。誠如漢學家波蒂爾所言：『世間語言，無一能比《道德經》更聖潔、更深邃地傳遞人類智慧。』」

有趣的是，此處暗示了老子所傳授的，實含秘傳教義——有時稱秘密教義、神秘哲學、古老或永恆智慧，亦即廣義上希臘語所謂的「神聖智慧」。如同歷史上許多偉大導師，老子顯然亦蒙此神聖知識啟迪，並依據自身時代、地域與環境，盡可能透露其奧秘。

由此，我們在布拉瓦茨基《秘密教義》第二卷第37頁，見到「秘傳老子」一說。她於《編輯附錄》一文中寫道：「自老子至玄奘，其（中文）著述皆影射並提及那島嶼（即香巴拉）與喜馬拉雅開悟者的智慧。」

《秘密教義》第一卷《導言》裡，她論及許多失傳神秘典籍的軼事，其中亦包括老子。布拉瓦茨基言：

「據東方學家集體研究，尤以近世比較語言學與宗教學學者努力所得，可以確定：曾有無數抄本，乃至印刷品，今已蕩然無存。它們消失得無影無蹤。若其內容無足輕重，自可在時間長河中湮滅，名號亦從人類記憶抹去。然事實不然；今已確知，其中多數實為理解現存神秘著作之關鍵鎖鑰；缺此額外註疏闡釋，於大多讀者而言，那些作品全然無法參透。老子之作，即屬此類。」

據稱老子撰有倫理與宗教著作九百三十部，魔法典籍七十部，總計千卷。然其學說核心之偉大經典《道德經》，如斯坦尼斯拉·朱利安所示，僅「約五千字」，不過十數頁篇幅。馬克斯·穆勒教授卻指出：「此文本若無註解便難以理解；朱利安先生為此不得不參閱六十餘家註疏，方能翻譯。」最早的註釋可溯至公元前一六三年，更早者已不可考；因在此前四個世紀，老子真義除受啟之僧侶外，幾乎向世人掩蓋。日本人在研究老子方面，被視為最博學之僧徒與追隨者，他們常譏歐洲漢學家謬誤層出、臆測連篇。蓋其自身傳統認為，西方漢學所能觸及之註解非真秘錄，乃刻意遮掩之帷幔；真正的註疏連同幾乎全部經文，早已自俗世眼中隱沒。（第一卷，第xxv頁）

此處所言老子在日本之追隨者，大抵暗指「山伏」——布拉瓦茨基屢次提及此群人。維基百科載有專頁，稱其為「日本山野苦行之隱士」。於《秘密教義》中，布拉瓦茨基闡明月球實為地球之母，而地球乃月球轉世，繼而寫道：「此乃『月球七秘』之一，今得揭示。日本山伏——即老子教派之神秘修士，與京都苦行僧——將此七秘稱之為『七寶石』。然而，日本與中國之佛教苦行僧及開悟者，在傳佈『知識』方面，比印度教徒更為緘默。」（第一卷，第173–174頁）

神秘學說告知，人與太陽系皆由七種根本面向或「原則」構成。於《神聖智慧之鑰》第一一七頁，布拉瓦茨基言：「老子於《道德經》中僅提及五則原則，與吠檀多相同，略去了靈（阿特曼）與肉身二者，且稱後者為『屍骸』。」

《現代醫藥箱》中有一文，題作《神秘主義或魔法》。該文首刊於一八七五年七月《靈性科學家》雜誌，標題《致希拉弗數問》，比神智學會成立早四月。此文值得通篇細讀；《玫瑰十字會隱秘起源》一文中亦詳加引述，此乃布拉瓦茨基首度公開提及自身與開悟者兄弟會之關聯。其後她以更具體、更詳盡之措辭，書寫東方開悟者之事。該文中，她分享對孔子與老子之若干見解：

「……比羅夫對於基督、佛陀、孔子三者之比擬中，亦犯同類謬誤。前二位乃睿智之靈性開悟者，末者中國哲學家實難與之並論。基督崇高之抱負與視野，與後者冷峻務實之哲學毫無干係；孔子乃一聰慧之異類，生活於天生沉悶、物質主義之民族——此族自歷史濫觴，便愛好和平、躬耕隴畝。然孔子終無法與前二位偉大改革者相提。基督與佛陀之原則與教義旨在包容全人類，孔子卻僅注目己國，欲以其深湛智慧與哲學應合同胞之需，鮮少為他族勞神。其哲學教義於愛國精神與觀點上純然中國本色，卻欠缺詩性之純粹元素——此正為基督與佛陀此類神聖典型教義之特質。孔子人民的崇高精神裡缺乏了宗教傾向，例如印度教所見那般。孔子甚至不及同時代老子之深邃情思與靈性追索。博學之思莫瑟曾言：『基督與佛陀之精神於世間留下不可磨滅、永恆之印跡。孔子學說僅可謂人類冷靜理性最輝煌之歷程。』哈維於《世界歷史》中以數語精準勾畫中華民族：『其沉重、稚拙、冷峻、感性之天性，恰可解釋其歷史之獨特面貌。』故試圖於論玫瑰十字會主義之文中，將前二位改革者與孔子任意比附，實不可取。」（第43–44頁）

然孔子在其他地方所獲的評價更為正面，甚至在《秘密教義》第一卷第一六二頁中，被稱為「第五輪次者」。此為一個寓言式術語，意謂在當前第四輪次演化時期（[參閱《行星鏈、星球、輪次與根種族》](/docs/planetary-chains-rounds-root-races-mysticism)），孔子內在已達至更為進展之階段。普通人類大眾欲普遍臻至此境，須待下一大演化周期，即數百萬年後。

然則「第五輪次者」未必即靈性高度進化者，蓋第五輪次猶如第五根種族，關乎人的七大原則中之第五原則「心智」——即思想與智性之發展。《秘密教義》以孔子與柏拉圖為「第五輪次者」之例，而以佛陀與商羯羅為「第六輪次者」。

然而，若孔子已臻第五輪次，而老子境界更為超然、更具靈性，或可推論老子同屬第五輪次，甚或更為深遠。

《道德經》譯本繁多，以下譯文不刻意直譯，寧取「演繹」之徑——與其拘守字句，毋寧勾勒其思想本貌。

首章論「道」，溯其宇宙本源與形上之思：

「能用語言表達的道，不是永恆的道；能被說出的名字，不是它永恆的名字。它在沒有名字時，是天與地的開端；在有名字時，是萬物之母。只有永遠擺脫塵世激情的人，才能領會其靈性本質；被激情阻礙的人，只能看到其外部形體。這兩種東西，靈性和物質性，雖然我們用不同的名字稱呼它們，但它們的起源是一樣的。這種同一性是一個謎，謎中之謎；它是所有靈性的大門。」

「哦，它是如此靜止、如此無形、獨一而不改變、無處不在而不受傷害！它必須被視為宇宙的母親。我不知道它的名字；為了指名它，我稱它為『道』。」

「天之下的萬物皆源於『存在形式的道』，『存在形式的道』源於『非存在形式的道』。」

「道產生了一體；一體產生二元；二元產生三元組；三元組產生了萬物。 這些無數物體把黑暗拋在身後，擁抱著光明，並被『虛空』的氣息所調和。」

「道產生了萬物，其美德滋養它們，其本性賦予它們形體，其力量完善它們。」

寥寥數行，隱含玄機：（一）永恆之「道」，即「非存在形式」之形態，似絕對之黑暗，深不可測和不可表達，是「永恆之零」，乃一切顯化之源，由之而生「一」；（二）非永恆之「道」，乃顯化萬物之直接根源，即「存在」之形態，為萬物統一體，尚可勉強以言語形容。此二者關係，恰似「絕對者」與「邏各斯」之別。（三）靈與物質，究其根本同出一源，猶如《秘密教義》所言：梵即原初質，至聖之靈與至聖物質本為一體。（四）所指是「大氣息」之和諧本性。（五）唯內心澄明，褪盡俗情慾念，方能照見無所不在之神性本質。

書末則散落智慧箴言，看似矛盾而耐人尋味：

「了解別人，不失為聰明；認識自己，才是真覺悟。戰勝別人，固然強大；超越自己，方為至強。知道自己何時足夠，就是富足；精力充沛的人有決心；不離其適當位置者，能保持長久；死而不亡的人，才是真正的長壽。」

「知道的人不說，說的人不知道。」

「最真實的話語是矛盾的。」

「對善人，我願為善；對不善的人，我也願為善，以使他們變為善人。」

「面對忠信之人我要保持信實，面對不忠信的人我還是保持信實，以使他們成為忠信之人。」

「即使一個人很壞，拋棄他怎麼可能是正確的呢？」

「以德報怨。」

「總是認為事情很容易的人，最終會發現它其實很困難；因此，聖人總是預見到困難點，因而他從未遇到過困難。」

「千里之行，始於足下。」

「當一個人確實知道，卻好像不知道般，這是智慧的高度；然而一個人若不知道，卻假裝知道，是一種惡習。如果我們正視這種惡習，就能擺脫它。」

「利用你體內的光，恢復你天生的清晰視野。那麼，失去身體時就不會受到災難影響；這被稱為雙重持久。」

「真實的話不好聽，好聽的話不真實。」

末段老子自述，同樣的話也適用於布拉瓦茨基：

「我的話語有線索，我的行動有基本原則。正是因為人們不知道這條線索，所以他們不理解我。」

「認識我的人很少，因此我有更大的榮耀。」

「因此，聖人穿的是粗布衣裳，但懷中揣著一顆珠寶。」
