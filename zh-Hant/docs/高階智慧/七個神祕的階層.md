#  七個神祕的階序
## 這套秘傳教義，所揭示的尚不及全貌十分之一。

![cover](/img/downloaded/60efdbb188d2d0cc4e7ff68e2bcd6b37.jpeg)

研讀《秘密教義》的學生會記得，書中大師與布拉瓦茨基提及十二種偉大的「天界存在」秩序，其中包含七個階序、群體或秩序。它們有時被稱為「天界階序」、「禪那主階序」或「天人階序」。

這些內容大多人難以領會，除了熟讀《秘密教義》的學生外。他們對這部書鑽研至深，方能看懂看似無意義的章節。《秘密教義》必須視作整體研究，才能得其真義；也應置於布拉瓦茨基思想的完整圖景之中——連同《揭開伊西斯的面紗》、《神聖智慧之鑰》、《寂靜之聲》等核心著作，一併考量。

《秘密教義》第一卷《宇宙生成論》第213至233頁，稍稍揭開這個深奧主題的面紗。首先，「創造力」劃為七個（或說四加三個）神祕階序，存於十二大秩序內，並記錄於黃道十二星座；此外，七個顯現階序又與七顆行星相連。這一切再細分成無數神靈、半神與空靈存在。

因此，共有十二種偉大的靈性存在秩序，某種程度上對應黃道十二星座：白羊、金牛、雙子、巨蟹、獅子、處女、天秤、天蠍、射手、摩羯、水瓶、雙魚。

十二大秩序中，存在七個「創造力的階序」，與七顆神聖行星相連：金星、水星、火星、木星、土星，以及另外兩顆據說是太陽與月亮——實則是神祕的祝融星，與另一顆神秘學未具名的行星。但布拉瓦茨基在[《秘密教義闡釋》](/books/secret-doctrine-exegesis)中描述後者為「一顆逆行的行星，夜半時分偶爾可見，似在月球附近。其神祕影響經由月亮傳遞。」

以下是關於七個階序的記述……

\#1 — 「最高等的群體由神聖『火焰』構成，亦稱『火獅』與『生命之獅』，其神祕面向深藏於獅子座中。它是高等神聖世界的核心。它們是無形的『火熱氣息』，某種意義上等同於卡巴拉『原型世界』的上層三質點。」

\#2 — 「第二階序的『天界存在』屬火與以太，對應靈與靈魂，或稱『阿特曼-菩提』。其名目繁多，雖無形體，卻更具『實質』……如其名所示，他們是投生單子的原型，由生命的火熱之靈構成。光線穿過他們，如純淨陽光，他們輸送光線與其未來載體，即神聖靈魂。這群體與我們太陽系的高等世界直接相連。從這個雙重單元，流溢出三重單元。」

\#3 — 「第三階序對應『阿特曼-菩提-心智』，即靈、靈魂與心智，被稱為『三元組』。」

\#4 — 「第四階序是實質性的存在體，為原子形體中最高等的一組。它是人類靈魂之溫床，具有意識與靈性。他們被稱為『不朽的靈魂』，並透過其下的階序，構成最初的七重群體——此乃人類意識與智性存在的偉大奧祕。」

由於第四階序居於七階序中間，或許正是「禪那主身體的『心』，據說曾投生於人類第三種族，使其臻於完美。」（《秘密教義》第二卷，第91頁）

#5 — 第五階序極為神秘，它與微觀世界的五角星相連，而五角星象徵人。在印度與埃及，這些天界存在與鱷魚相關，居所位於摩羯座。印度占星中，這兩個詞可以互換，因為黃道第十二宮（摩羯座）直譯即為「鱷魚」。……這第五組天界存在被認為具備雙重屬性：宇宙的靈性面與物質面，可說是「宇宙智性」的兩極，也對應人類靈性與肉身的雙重本質。因此，其數字五乘以二成為十，正與黃道第十宮「摩羯」相繫。……那些執掌摩羯座——在印度與埃及稱為「鱷魚」——的神秘存有，其使命是教導空靈縹緲的動物形體（即地球第四輪次早期的人類），使其進化為理性之人。這是公眾鮮少談論的主題。它確實是個奧秘。有些人只將意識完全局限於人類、視意識僅為「大腦功能」；他們拒絕承認宇宙中存在著具備智性與意識的靈性生命。自人類誕生之初，便有無數靈體投生於肉身，同時，且如以往般獨立存續於無限的空間。……更明確地說：這些不可見的實體以肉身居於塵世，同時也不失去其在超感知領域的地位與職能。

# 第六與第七階序佔據四元組的下層性質。他們是有意識、空靈的實體，如以太般不可見……在靈性下行的尺度上，第六組是最後第七組精靈的前一階，他們逐步形塑、構築並凝聚自己的軀體。這第六組的演化形成了人類朦朧的形體，如一層薄膜、近乎透明的複製體。……此外，第六組與人幾乎不可分割，人汲取了此組全部原則，如除了至高（靈性）與至低（肉體）外；五個居中的人類本質，正是「天界之人」的真髓。神聖之光（阿特曼）直接源自「至一」。

必須牢記這關鍵一點：「正如教義所示，宇宙中並無享有特權的存在，如西方宗教與猶太教天使那般；無論此存在處於太陽系或他系，在外顯世界或內在之境。一位禪那主須經修持方能成為禪那主；他無法在誕生之際、或顯現於生命層面時，便立刻成為完全發展的天使。當前顯現期中的『天界存在階序』，將在下一生命周期中遷移至更高、更優越的世界，並為由人類菁英組成的新階序騰出空間。『存在』是絕對永恆中無盡的循環，其內運行著無數受限且有制約的內在週期。」

《秘密教義》第一卷第292–293頁引述舒巴羅《黃道十二星座》，其中列舉六種宇宙力量的名稱與簡述，並由第七種綜合代表——即「邏各斯之光」。

「宇宙的七種主要力量整體亦稱七道光線，包括：(1) 至上之力 (2) 智識之力 (3) 意志之力 (4) 思維之力 (5) 火焰之力 (6) 咒語之力。」布拉瓦茨基隨後指出，這些「皆是秘傳教義，所述尚不及可言之十分之一。此處所提六力之名，即那六階序的禪那主，統合於至高的第七力。」它們是宇宙大自然（神秘意義中的「母親」）第五原則的化身。僅列舉瑜伽力量便需十卷篇幅。每一力量頂端皆有一活躍的意識實體，乃是該力之流溢。

布拉瓦茨基與大師們屢次強調，對應法則與類比法則統御萬有；欲清晰準確領悟秘密教義，此二法則是不可或缺的關鍵。
