# 太陽力的神秘力量和特性
## 這段文字取材自《秘密教義》，內容描述了太陽中神秘力量的性質和特性

![cover](/img/downloaded/9e7a9f03e977bc171bd42a8372696d72.jpg)

Photo by [Rajiv Bajaj](https://unsplash.com/@rajiv63?utm_source=medium&utm_medium=referral) on [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral)

本文取材自布拉瓦茨基的《秘密教義》，闡述太陽中那股神秘力量的性質。欲深入探究，可參閱該書第一卷第五百二十四頁起。迄今無人能如她這般詳盡精確地描述此力——這在神秘科學中，向來是極度隱秘的一環。

布拉瓦茨基在書中詳引了一位幾近被遺忘的思想家兼科學家：塞繆爾·萊特勒·梅特卡夫（1798–1856）。這位美國化學家兼內科醫生，著有《熱質：其在自然現象中的機械、化學與生命作用》一書。維基百科自詡政治正確，卻帶著幾分虛偽地宣稱：「熱質……是一種過時的科學理論，視熱為一種自我排斥的流體，從熱物體流向冷物體。」我們將證明，梅特卡夫絕非陳舊理論的作者；他精準描述了布拉瓦茨基所說的「太陽力」之性質，而這知識，僅為最高層次的啟蒙者所洞悉。

我們討論的起點是一個十九世紀的物理學家大多認同的前提（這一點直到21世紀的物理學家們仍未有新的發現）。自然界存在兩種媒介：一是可量、可見、可觸的「物質」；另一種則不可量、不可見，我們只能透過它對物質的作用感知其存在。梅特卡夫假設，有種活躍而不可量度的媒介，他稱之為「熱質」，並非單純的運動形式，亦非可量物質粒子的振動。它是一種源自太陽、流布太空的物質基質，充斥整個太陽系，並透過感官傳遞一種名為「熱」的特性。梅特卡夫列舉此力的特性如下：

「它既能注入物體，亦能從中抽出，可用精確數學方法測度其量。

它使物體體積膨脹，一旦移除，體積便相應收縮。

它能改變一切物體的形狀、性質與狀態。

即便在完美真空中，它亦能輻射傳播，且對溫度計的影響，在真空或大氣中並無二致。

它能產生無可抗拒的機械力與化學力，其強堪比火山爆發、火藥爆炸或其他劇烈化合反應。

它以可感的方式作用於神經系統，引發劇痛；若過量，則導致肌體組織紊亂。」

梅特卡夫進而補充：「若太陽力僅是屬性，便無法增加他物體積；故其自身必具體積，必佔據空間。因此，它必是一種物質媒介。」此說更近神秘學法則，而非物質科學觀點。他的假設基於一事實：太陽力或曰熱質，是一種自我活躍的原則。它排斥自身粒子，卻對一切可量物質粒子懷有親和力。此力依距離平方反比作用，吸引任何可量物質粒子——這也印證了現代物理的平方反比定律。

梅特卡夫還提出一條他視為絕對的定律：「熱質藉由吸引可量物質，將萬物凝聚；同時，憑其自我排斥之力，它又分離並擴張萬物。」這，便是純粹的神秘科學。

《大眾科學評論》中，理查森博士評述梅特卡夫假說時寫道：「梅特卡夫關於太陽力與地球力的假說，不僅簡明，更別具魅力。宇宙存有二元素，一為可量的物質，另一為無所不在的以太——太陽之火，無重無體，無形無色。它是無限可分的質體，微粒相斥；除『以太』一詞形容其稀薄，別無他語可述。它彌漫空間，充滿萬有，獨存時靜止，是死的。」

梅特卡夫對太陽能之描述，以及理查森對該研究的評論，大致符合已知的神秘科學原則。然而，正如布拉瓦茨基在腳註所言，他們的推論仍有謬誤。「太陽力並非僅在太空流動，而是浸透太陽系每一角落，因它可視為以太的物質殘餘，是以太在我們層面上的內襯。以太不單是光的傳播『媒介』，更肩負其他宇宙與物質世界的功能。它是卡巴拉學者所稱的星光界流質，亦是『太陽–毗濕奴』的『七道光線』。」從神秘科學觀之，理查森將太陽力稱作「以太」並不妥當，因太陽能僅屬以太眾多原則之一；事實上，以太本身也只是阿卡莎或原初質諸原則的其中一項。理查森另有一誤，即視可量物質為「死物」。物質靜止時或顯惰性，卻永不死亡。

理查森繼續寫道：「以太滲入可量物質的微粒，與其結合，使之凝聚；物質亦溶解於以太之中。依梅特卡夫之說，這堅實可量之物質透過以太分布，延伸至當下一切存在。以太無所不在。人體滿載以太；身軀的微渺粒子由它聚攏；植物亦然；就連最堅硬的土地、巖石、磁石、水晶與金屬，也不例外。」若在理查森對梅特卡夫理論的總結中，將「以太」替為「星光界流質」，便與神秘科學學說無異。

神秘科學主張，不同物質對太陽力的吸收各異，主要取決於物質自身的多變狀態；如固體、液體、氣體。固體較流體更易吸收太陽力，因而凝聚力更強。若將部分熔化的鋅傾倒於固態鋅板，熔鋅即轉為固體。這是因為太陽力自液態流向固態，在平衡過程中，原本散逸的顆粒或液體變得更緊密結合。梅特卡夫對此現象詳加探討，並將其歸因於作用原則的統一性，如前文所述。在一篇關於物質密度的評論中，他清晰總結了自己的觀點。

他說：「硬與軟，固體與液體，皆非物體本然狀態，而取決於其所含以太物質與可量物質的相對比例。最具彈性的氣體一旦被抽去熱質，或將復為液體，再轉為堅實固體。微粒間的黏合力，與其對熱能吸收的親和程度成正比。反之，若向最密實的金屬注入足夠的同成分，令其膨脹為氣態，則其吸引力減弱，內聚力亦隨之瓦解。」

梅特卡夫以如下話語總結其卓越理論：「我不擬深究此理論所隱含的太陽力與地球之同一性。但仍須補充，基於此假設——即將純粹運動視作一種無形質、僅有特性而無實體之力——我們能從所有這些繁複深奧的問題中，推導出最貼近真相的結論：

(a) 宇宙、星際空間、行星之間、物質之間、有機體之間，並非虛空，而是充滿微妙流體或氣體。雖無更恰當之名，我們仍可如古人般稱其為「太陽之火」或「以太」。此種液體成分恆定、不可毀、不可見，卻無所不在，滲透所有可測物質。溪中卵石、搖曳樹木、凝視之人，皆受以太影響；卵石少於樹木，樹木少於人。此行星之萬物皆以同樣方式帶有電荷！世界築於以太流體中，航行於其海洋。

(b) 以太，無論本質為何，皆源自太陽。太陽不僅產生以太，更是其儲藏之所與傳播媒介。

(c) 若無以太，運動不存；若無以太，物質粒子無法彼此滑動；若無以太，激發粒子運動之力亦將消逝。

(d) 以太是決定物體結構之關鍵。若無以太，物質的組成變化無從發生；例如，若無以太，水僅能以一種緊密不可溶的形態存在，超乎我們一切認知。它將無法成冰、化液，甚至轉為蒸氣。

(e) 以太連結太陽與行星、行星之間、人類與行星、乃至人與人。在沒有以太的宇宙中，信息無從傳遞；無光、無熱、亦無運動現象。」

布拉瓦茨基夫人所言極是：「以太僅在顯現階段時不可改變，之後便永遠復歸『不可見』之中。」尚可補充：太陽雖是太陽系中以太的儲庫，卻非此力之源。此力源自核心太陽，即宇宙中那不可見、不可窺的心臟，我們的太陽不過是其映象。

若塞繆爾·梅特卡夫的太陽力量理論獲得認同，且他關於太陽內存有生命的大膽推論被接受，這將證明神秘科學的啟示正確無誤，遠早現代科學理論數百萬年。
