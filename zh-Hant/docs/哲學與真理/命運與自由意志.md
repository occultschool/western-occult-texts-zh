# 預言、命運與自由意志的真相揭秘

## 探討預言如何運作、自由意志是否存在，以及高等自我與命運鏈的理性解釋

![封面](/img/downloaded/free-will.jpg)

我們將在文章中探討預言如何運作、自由意志與宿命論等問題。

你很可能會懷疑自由意志是否存在，若有人能預測未來，那麼不就沒有自由意志了？我們生活中的一切是否都是注定的，所以無論我們做什麼，命運都如期發生？許多哲學家、神職人員和科學家，都曾與此難題搏鬥一番，但沒有得出滿意的結論。

人類的理解力在大部分的時候是很薄弱的，尤其是試圖用無助的智力來推理宇宙奧秘之時。若僅憑智力，無論多麼發達，都不可能得出任何事情的真相。在討論命運與自由意志表面上的矛盾之前，我們先探討預言、占卜和算命實際上是如何運作。某人突然瞥見即將發生之事時，到底發生什麼事 ? 其實道理非常簡單，知識分子反看不出來，因為他們只依靠冷酷的推論來得出自認為的真相。

意識對於科學和宗教都是一個謎。如果你懷疑此說法，不妨試著感受一下大腦是何部分在「思考」。你會發現你做不到。原因是大腦不會思考，只處理從低等自我或高等自我收到的信息。這兩個自我無法用目前科學所知的方法來檢測、測量或分析。事實上，科學否認大腦之外有任何意識，儘管已有大量反例佐證，特別是關於「瀕死體驗」的研究。

瀕死體驗發生在臨床上被認為「死亡」的病人，最常見的特徵是在體外經歷意識的擴展。這證明了大腦會限制意識，當這種阻礙被消除或癱瘓時，比如在深度冥想時，精神活動就會增強和擴大。日常清醒的思考可比作一部智能手機的正常計算能力，而更高的意識則比作一百萬台超級計算機共同運作。若你有任何意識延伸的個人經驗，你就會知道這是真的。

這適用於人的高等自我，而非低等自我；當低等自我不再受高等自我指導，就與會思考的動物沒有區別。高等自我的心靈能力遠超正常清醒的意識，能以極高速度和敏銳度理解物質事件與情況，遠超單純智力推理。因此，預見未來的原理是高等自我一系列極速計算（低等自我一般不知道此過程），計算出一個人的行為將如何不可避免地導致某些結果。

事實上，許多預測之所以沒有實現，是因為很少人有能力將意識集中到很遠的未來，或延長一段時間。 還有一個更複雜的情況是，當高等自我從身體中解放出來時，低等自我和大腦沒有正確地接收高等自我的記憶，並忠實地保存。因此預測並沒有這麼簡單，儘管其法則是直接明瞭的。

世界上存在著一個普遍的鏈條，一環扣一環，連接著人類的思想和行為。此鏈條的近端是「現在」；而「未來」是一系列更進一步的環節。大多數人只看到「現在」環節，無法看到更遠。但有些時候，意識中會閃現出內在的光芒，此時會看到下一個環節，下下一個環節，甚至是沿著命運之鏈的遙遠環節； 然後我們會驚訝地倒抽一口氣。若將所見之事告訴他人，並且活得足夠久，那麼人們就會視此人為真正的先知。 你或許認為這是宿命論，但事實並非如此。

有一個任何人都可以嘗試的簡單實驗能作為驗證。 這並非關於未來或預測的實驗，但能展示在地球上看到或經歷的一切，都是星光界情形的反映。 

放鬆你的頭腦和身體，儘量不要想任何事情。然後試著在不看鐘的情況下說出現在的確切時間。要做到這一點，就是說出腦子裡突然出現的第一個數字。你很快會發現，說出正確的次數比錯誤的次數多。隨著進一步的練習，便能在任何地方說出時間，甚至是幾秒，且不用看時鐘或手錶。這並非「魔法」，而只是上述的自然法則起作用。

也可以用一副撲克牌來做類似的實驗。將牌面朝下放在桌子上，試著猜出這副牌中第一張牌的花色和數字。在滿意自己的成功後，可以改猜第三張牌或第五張牌。通過練習，最終能夠準確地猜出任何牌的數字。現在你知道一些舞台「魔術師」是如何表演他們的「戲法」的。

那關於詛咒呢？許多人都詛咒過別人，且詛咒確實生效，有時持續好幾代。這也是宿命嗎？如我們所說的，不是。 為什麼某人要詛咒另一個人？ 因為被詛咒者定是在某方面傷害了詛咒者，不管是有意的還是無意的、自願的還是非自願的。如果被詛咒者相信此詛咒，那麼詛咒就會生效。 若一個家庭一代代受詛咒，且他們認同這些詛咒，那就會延續數年甚至數世紀。 

首先發出詛咒的人，預見到了錯誤行為將帶來不可避免的懲罰。 在唸出詛咒的時候，心靈會在高度興奮的狀態下，瞬間被啟發，能在一瞬間追溯錯誤行為導致的整個事件過程；一環扣一環地看到必將發生之事。諮詢神諭也是同樣原理；不管神諭的來源是男人還是女人，或幾根棍子、一些茶葉、塔羅牌，或者你想要的任何東西。

命運之鏈的一段會在瞬間顯現，而根據預言者洞察的深淺，便能看見並解讀未來。不過，這並非宿命，而正如我們所說的，只是一種極快速的推演——從一個事件如何引發另一個事件，直到整個未來脈絡在短暫一瞬間變得清晰可見。這才是此類現象的理性解釋。

但請注意，雖然這是先前行為必然導致的結果，但絕非強加給任何人；這取決於自身是否相信詛咒或未來事件。若意志夠堅強，這些不可避免的結果還是能被改變，儘管似乎與剛才所言矛盾。 但不盡然。假設有個人詛咒你，若你相信他，並按照此願景行事，那麼詛咒就會成真。

這同樣適用於任何形式的算命，無論是通過塔羅牌、手相、占星術、拿破崙的《命運之書》，甚至是茶葉和咖啡渣。換句話說，若對於預測出的好運氣或壞運氣作出反應，就是與詛咒者或神諭者攜手合作，幫助對方成了先知。若是在被詛咒時反過來詛咒對方，那麼也是與之攜手合作。但是，若能以祝福來迎接詛咒，或者以平靜的心態和對神的信心來面對預言，那麼就能避免厄運，詛咒也不會生效。那些想成為女巫的請多加留意 !

這就是「自由意志」。之所以許多詛咒能按計劃發生，許多預言實現，是由於人的意志薄弱、無知或迷信，不知道有更好的方案。 當我們按照預言或詛咒行事時，就會成真；但這不是自由意志，而是脅迫。偉大的中國聖人老子說：「民不畏死，奈何以死懼之？」詛咒、預測或預言也是如此。若不懼怕這些，就不會以任何方式受到影響，我們便能獲得自由，並在自己的內心得到平靜。

這便是關於預測、命運和自由意志之謎的簡單解釋，未經任何花俏裝飾，或許前所未聞。現在你知道，每個男人和女人都具有神賦予的自由意志，能以不同方式對情境做出反應。也更能理解「業力」的真正含義，對此誤會導致懦弱的宿命論，這在許多東方國家和東方民族中非常普遍。

若宿命論為真，那麼任何殺人犯或罪犯就能為自身行為辯解，說這一切都是神預先安排的，因此他是無罪，不應受到懲罰。甚至進一步說，若有人敢懲罰他們，便是反抗神本人，因為一切都是憑神的意志進行的。

相信宿命論者滿足於在夢遊狀態中度過一生，表明大多數人都沒有真正理解人類和宇宙中的真正法則。 希望此篇文章能讓你開闊眼界，了解占卜、命運和自由意志到底是什麼，如何運作。有耳朵的，讓他們聽；有頭腦的，就去想 !
