# 先有雞還是先有蛋 ?

## 本文討論柏拉圖與亞里士多德對此的見解，並揭示科學難以詮釋的形而上真諦。同時藉神祕學的視角，探討生命與意識的源起，延伸對二者不朽本質的思辨。

![cover](/img/downloaded/7e88c1b993f60f56b8e85ec9fc6ce3e2.jpeg)

## 常年的悖論

「先有雞還是先有蛋」這道古老的謎題，或許和人類的歷史一樣久遠。公元前四世紀，柏拉圖在《巴門尼德》中便觸及了這個問題。他最有名的學生亞里士多德則認為，雞與蛋都沒有先後之分：每一隻雞來自一個蛋，每一個蛋又來自一隻雞，於是得出結論，鳥類與蛋亙古並存。這個見解雖然樸素，卻勝在簡潔，也比後世那些科學論證更深邃。亞里士多德接著說道：「若說最初有一人，此人卻無父無母，這是違反自然法則的。正如不可能有第一顆蛋孵出鳥，也不會有第一隻鳥生出蛋。」

柏拉圖的看法不同。他主張，「雞」的理型先於實體的雞而存在。我們提過，柏拉圖被視為啟蒙者，他洞悉萬物本源，深諳因果的實質。他的學生亞里士多德卻缺乏這份覺悟，處理問題時過於倚重理性，無法認同老師的觀點。他認為「雞」的理型僅是人類的概念，源自我們對雞的認知——好比辯士認為神的「理型」（不管意義為何）僅屬人為的虛構。柏拉圖的真實含義，並非弟子所能領會；這種誤解屢見不鮮，尤其當學生試圖剖析師長的言語，卻沒有師長的智慧！柏拉圖所說的「理型」，正是神祕學中事物或存在的「靈性原型」，他自己稱之為「形體之缺如」。

柏拉圖與亞里士多德之後四個世紀，希臘史家普魯塔克在《道德小品》中提到：「蛋與雞孰先孰後，這個問題始終困擾著追問的人。」這種「困擾」已持續兩千四百年，或許更久。現代科學將問題局限在物質層面，將其瑣碎化、放大千倍，反而無法觸及其中蘊含的深刻形而上真理。後文將揭示，柏拉圖與亞里士多德其實既對也錯；正如我們在多篇論述中指出的，在神祕學的迷宮裡，許多看似矛盾之處，唯有憑藉更高層次的理性之光才能調和，非單靠有限的智力所能釐清。

## 科學詭辯

那些身穿白袍、才智出眾的人，常在這裡誤入歧途，只因他們誤信單憑智性分析就能解開一切謎團。一位科學家曾巧言迴避，說：「蛋與雞孰先，端看蛋的本質而定。」話聽來，不過是兩邊押注的托辭罷了！達爾文提出的自然選擇理論，深刻影響了多數科學家的思維。若信奉此說，答案便是蛋先出現——科學告訴我們，基因只能在生殖時傳遞，因此某種非雞的生物必先使其DNA突變為雞的DNA，經受精孵化後，第一隻雞才誕生。然而，這解答並無太大幫助，因為它立刻引出另一問題：那變異的DNA最初從何而來？無論科學界如何推測雞由何種鳥類進化而來，也不論此事發生在多久以前，它們總不可能憑空顯現；儘管許多科學家似乎相信，宇宙本身正是從一場所謂的「大爆炸」中無中生有。如此看來，或許不該全然責怪他們，使用了同樣盲目的邏輯（如果這也能算邏輯）來處理雞與蛋的悖論。

一群科學家卻主張雞先於蛋。2010年，謝菲爾德與華威大學的研究團隊動用超級計算機，以「元動力學」模擬並放大觀察蛋的形成過程。他們發現，蛋殼的生成關鍵在於一種存在於雞卵巢中的蛋白質——ovocledidin-17——它能催化蛋殼發育。然而科學家也同意，在雞出現之前，其他鳥類產蛋未必倚賴相同蛋白質以相同方式形成蛋殼。這論點雖較以往周密，卻仍未解答悖論的核心：若有第一隻鳥，牠從何而來？結果在無休的爭辯後，多數科學家避開了這棘手的「起源」問題，轉而共識蛋為最先。科學所描繪的，是一個不斷演化的蛋，卻只以盲目的機遇與「自然選擇」為演化理由——未提任何理據。無論這說法多麼動聽，它終究漠視了靈之存在與第一因。科學深陷唯物泥沼，徒勞地追問萬物起源，無論是變形蟲、雞，抑或宇宙。

科學解釋不了DNA從何而來，又為何驟然「變異」成雞。暫且擱置這些尷尬難題，科學家得出的機械論結論，實則封閉了通往柏拉圖與亞里士多德哲學之門。現代教育體制中，直覺、冥想這類高等感知遭禁用。在早期社會，智慧被矮化為知識；理性主義時代，知識再被壓縮為經驗事實；如今科學唯物論的短視，更將事實降格為個人意見與智性臆測。柏拉圖所教導的高尚概念——一個由嚴格正義所支配的有序宇宙，由各類創造性存在構成層級體系——已被科學取代；科學的信條是「大爆炸」，主張無心智的物質在無計劃、無目的、無緣由的狀態下，盲目前往未知之境。這或許令某些科學家滿足，但對我們、對任何具備正常思辨與常識者而言，遠遠不夠。且看神祕科學能否更進一步。

我們如今明白，雞與蛋的悖論，真正關鍵在於生命本身的起源。一切物質與靈性存有皆有一共通處：生命。先前文章曾論及此，尤其在《神祕學學生與健康》一文中寫道：「地球上所有生命體唯一共享的，正是生命本身。」若問我們生命最根本的本質為何，我們只能說不知——那是宇宙至高神的奧秘：一切生命皆源於祂，我們不敢妄論與造物主相關的詞彙。但這不代表我們無法對第一因形成某種概念，或對雞與人的起源獲得有限理解。

## 神祕學真理

實情是，雞與蛋的悖論並不難解，只是科學家糾結的思維難以領會。他們忙於將所謂「事實」分門別類，塞進標籤箱裡，再如滾雪球般擴充理論（一旦遭同行質疑，這些理論便消融無蹤），卻忽略了「第一因」這類哲學抽象概念。若他們稍具柏拉圖的智慧，便會意識到：只要存在第一因——無論稱之為「神」、「力量」、「神之心智」或其他——那麼雞必先於蛋，正如神祕科學所驗證。換言之，依柏拉圖啟示，「理型」即雞的理想原型必先存在。產蛋是母雞天性，要孵蛋，就得有雞來生蛋；無論這雞是物質創造的實體，或僅存於造物者心中的「理型」。為何如此簡明的事實仍令科學家困惑，這問題就留給各位思索了。

再者，在這世上（或在那稱為「星光界」的更高或更低的非物質層面），我們感官所見的一切，早已存在於某位創造者心中。無論你搬入的「理想房屋」是郊區半成屋、尋常公寓，還是豪華別墅，它都先存於創造者腦海——無論創造者是建築師、建商或其他具創造力的存有。你駕駛的車、腳上的鞋，甚至眼前這篇文章，無一例外。

或許你會問：這些理型或原型，在進入房屋、汽車、鞋子或雞隻製造者的腦海之前，最初從何而來？這便將我們帶回原初問題：孰先孰後，雞或蛋？讓我們換個角度，向自己提幾個問題：腦袋或想法，孰先孰後？是腦袋創造想法，還是想法塑造腦袋？首先須記住，物質本身無思考之力，亦不能產生思想；因物質乃創造者想像力的結晶，正如我們在[《論物質重要性》](/docs/why-matter-is-important)一文中所言。然而，一旦物質受造物主思維啟動，它便成為生命體，擁有或多或少的自我意識。但即便最基礎的思想或意識，需要有一位具思想、有意識的存在將其注入。就像舞台魔術師從帽中變出兔子，我們都明白：必是有人先將其放入，儘管某些科學家似乎自認懂得更多！

也許你會問：意識從何而來，又如何注入？神秘學指出，它源於渺遠的往昔——某位先人將自身部分意識移入他人軀體，正如《秘密教統》、[《金黃星》](/books/golden-star-dr-michaud)與《聖經》所載：「神用地上的塵土造人，將生氣吹在他鼻孔裡，他就成了有魂的活人。」（創世記 2:7）然而這仍未能解答：誰是那最初擁有意識與生命者？我們又落入了「先有雞或先有蛋」的古老謎題，無數智者為之困頓。可是答案竟如此簡單，幾乎無人察覺。你可曾看出端倪？容我道來。

若無生命，意識便無從依附。有些人看似意識空缺，那只是表象，或許只是意識暫離此處，遊移他方。意識是生命的一部分，生命亦是意識的一部分，兩者皆不「死亡」，僅會轉化形態。倘若生命與意識當真消逝，宇宙便隨之終結。關鍵在於：生命、意識、心智或思想皆不可能「誕生」——一旦它們徹底湮滅，則萬物無從顯現，無論是物質還是靈。

若說生命與意識能憑空出現，那是違逆一切理性的。我們不能將生命、意識、思想與心智的完全寂滅稱作「死亡」，因為死亡隨生命而來；但須記住，人類所謂的死亡，只是一場蛻變。而蛻變必以存在為基，存在卻無法從虛無中萌生，因此恆有「某物」始終存在，那便是萬物的根源。此物即永恆之生命，縱然歷經無數變化，它始終常在，亦將永續。倘若你願稱之為神，亦無不可，因為神即是生命，生命即是神。

下一次聽見人問：「是因為我不快樂，所以生活一團糟；還是生活一團糟，才使我不快樂？」你已了然於心：兩者皆是，又兩者皆非。若仍困惑，不妨細想：無心智之處，便無思想，正如無水之處，便無濕潤。你也許會反問：究竟是水先於濕，還是濕先於水？火先於熱，還是熱先於火？濕潤是水的本性，灼熱是火的特性，思想是心智的彰顯，而蛋則是雞的延續——因為它們本就不能分離。
