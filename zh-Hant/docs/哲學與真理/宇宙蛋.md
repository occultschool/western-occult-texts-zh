# 宇宙蛋
## 中國盤古神話中，宇宙始於一顆蛋。神人盤古自蛋中誕生，將蛋殼分作兩半，上半化為天，下半化為地。

![cover](/img/downloaded/0cba28ba2f63dcd9d83e7c4148ebe8d9.png)

Sheila Rose — “Cosmic Egg”

「蛋」作為神聖象徵，遍及世上各個民族的宇宙觀。人們敬重它，既因其圓潤外形，也因那殼內包裹的神秘。自人類靈性概念初萌，蛋便是一個標誌，完美代表了存在的起源與奧秘。正如[《先有雞還是先有蛋?》](/docs/plato-aristotle-chicken-or-egg-paradox)這篇文章所述。

那難以察覺的胚種，在最初封閉的殼內悄然生長；它不受外力左右，從一片混沌的「無」中，自行生出鮮活的「有」，只需熱量。待到漸漸長成具體、能動的生命，它便破殼而出。在外頭的觀察者眼中，這存在彷彿無中生有，自我創造。自有記載以來，哲學家們無不視此為奇妙而深奧的謎。

諸多遠古種族與文明皆敬重蛋的神聖象徵，主因在於，他們尚未為「第一因」找到一個恰當的名字。這是因果鏈最初的起點，亦稱終極因。事實上，試圖為那超越人智、不可名狀之物強加稱呼，被視為一種不敬；至今仍有許多神秘主義者持此見解。

於是，人們心中便勾勒出這樣一幅圖景：一隻永遠隱於無形的神鳥，在混沌中產下一枚蛋，那蛋即是宇宙。古印度人稱梵天為卡拉漢薩（Kalahansa），意為「時空之天鵝」。後來，他化作「永恆的天鵝」，並在每一宇宙顯現期——即大循環周期的起始——產下一枚「金蛋」，象徵宇宙及其間無數日月星辰的球體之形。

選擇蛋作為「第一因」、「宇宙」乃至「地球」的象徵，其二在於其外形——一個圓，一個球體。我們這顆星球的卵形，想必自符號系統初創之時便為人知曉，否則不會被天涯海角的民族如此普遍地採用。宇宙初現時形如蛋，是古代流傳最廣的信仰。希臘人、敘利亞人、波斯人、中國人乃至古埃及人，皆採此象徵。埃及《死者之書》描述，時間與大地之神產下一蛋，即是宇宙：「那是擁有雙重力量的偉大者，在彼時孕育的蛋。」所謂雙重力量，自然是指電與磁。

盤古神話裡，宇宙始於一顆蛋。神人盤古自蛋中誕生，將蛋殼分作兩半，上半化為天，下半化為地。《創世紀》中也能找到同樣的象徵：「神造出穹蒼，將穹蒼以下的水和穹蒼以上的水分開。事就這樣成了。」這分開的過程，意味著「可見的宇宙」從「未顯現的空間」中浮現，二者皆以「水」為其象徵。

古埃及神話中，神祇拉如同梵天，亦從宇宙之卵孕育而生。《死者之書》描述死者「在神秘之地的蛋中閃閃發光。」書中也明言，這是「賦予眾神生命的蛋，偉大母雞的蛋，塞布的蛋，他如鷹一般從中破殼而出。」

古希臘喜劇作家阿里斯托芬筆下，描述了奧菲斯蛋，從中孵化出原始的雌雄同體神，此神又創造了希臘萬神殿的諸神。哲學家波菲利指出，奧菲斯蛋乃是世界的象徵，這奧秘只傳授給有幸參與厄琉息斯秘儀的人。

一些《聖經》學者試圖將蛋解作諾亞方舟。然而，這是虔誠教士的一廂情願，因為方舟並非只是純粹的寓言象徵。唯有當我們將方舟等同於月亮時，蛋才能被視為方舟。月亮司掌孕育，而方舟則好比承載生命種子的容器。但這種聯想，與《聖經》中的方舟並無實質關聯。總之，正如我們所揭示的，宇宙肇始時形如一蛋的信念，廣為流傳，根基深厚。

印度聖典《往世書》中，多次描摹元素初聚成卵的意象，常以「輝光」、「燦耀」來形容，非如某些學者誤解的「金黃」。翻開《毗濕奴往世書》，便讀到這般記述：「宇宙心智……隱而未顯的粗質元素俱蘊其中，凝成一卵……宇宙之主以梵天之身棲居其內。婆羅門啊，那卵中涵藏大陸、海洋和山脈、星體諸界、神魔人眾。」

須知此處「海洋」、「大陸」、「山脈」並非塵世所見之物，而是指宇宙萬有的分野——物質與靈、可見與無形——皆納其中。若只作字面解，這段經文便失了意義：諸天星辰，怎可能囿於我們這渺小地球？當代學者往往在此失足。他們未經秘傳訓練，總執意將這類寓言經文讀得僵直。我們在先前的[象徵論述](/docs/mysterious-language-unlocking-ancient-wisdom)中已點明此弊，而今值得再提——許多人讀諸般聖典，尤其是《聖經・創世記》時，正墮入這般謬誤。

希臘與印度神話裡，最初可見的男性生命，自身兼融兩性之質，居於卵中，又破卵而出。部分希臘人視此「世界初生子」為戴奧尼修斯。這位神祇自宇宙卵誕生，凡人眾神皆源於祂。埃及《死者之書》則描述拉神在其卵（太陽）中灼灼生輝，當舒神（太陽能量或宇宙電）甦醒，卵便搏動起來，以光與生命照亮其疆域（我們的太陽系）。故而對埃及人、印度人與華夏先民而言，拉象徵著宇宙蛋中卓絕的太陽神。祂曾宣告：「我是太虛深淵的創生之靈。無人得見我巢，無人能破我卵——我即是主！」

由此可見，雞與蛋之問，並非僅是令人困惑的謎語。若本文能引你更探此題，我們將所思「陳於」你眼前的這番工夫，便算得了圓滿回報。
