# 披著羊皮的黑魔法：「吸引力法則」與「性密宗」
![cover](/img/downloaded/f3025b01b09ba79ad3874ec3180adbdc.jpg)

Photo by [Elena Mozhvilo](https://unsplash.com/@miracleday?utm_source=medium&utm_medium=referral) on [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral)

>「無論是白魔法，還是黑魔法，都不是一般迷信所理解的術語。」——布拉瓦茨基

「黑魔法」這詞，多數人立刻喚起對於女巫、魔鬼崇拜的聯想，或是企圖用咒語儀式傷害他人、操控命運。這些確屬黑魔法；然而，許多同樣黑暗的伎倆，我們未必察覺其本質。

甚至可以說，當今世上成千上萬看似良善的人，實則都是不自知的黑魔法師；而許多流行的靈性教導與修行，幾乎全沾染了黑魔法的色彩。

為何如此斷言？

首先明確定義：**黑魔法，是懷抱自私或物質目的，刻意操弄靈性力量的行為**。白魔法同樣運用靈性力量，但其目的純粹是祝福與造福他人。

這並非作者獨創的見解。自古以來，黑魔法一直以此定義為準。每當靈性團體發現追隨者試圖以靈性或心智力量，來汲取財富、物質或個人成就，便會將他逐出寺廟、寺院或修行社群，並引以為恥。

古人深知，靈性生活的真實進步與靈魂內在的昇華，奠基於無私、利他、謙遜，以及全心造福人類的志向，並以最崇高的道德與個人純潔為準繩。

佛陀、克里希納、耶穌、赫爾墨斯、商羯羅、宗喀巴、老子、布拉瓦茨基，乃至歷代所有偉大的靈性導師與世界改革者，其教誨無不清晰地闡明了這點。

幾個世紀以來，「吸引力法則」與「欲望具象化」這類教導與實踐，始終在可恥與黑暗中秘密進行。直至人類的自私與物欲執迷達到頂峰，它們才開始獲得公眾接納與歡迎。從十九世紀到二十世紀，這些思想以驚人速度傳播全球，並受到狂熱追捧。

在西方人心中，自私自利與對權力財富的渴求早已根深蒂固；於是多數嚮往靈性之人，從未懷疑過一事:當他們透過觀想、肯定語、「豐盛思維」與「夢想板」招引華宅、美伴或萬貫家財時，這些手段究竟妥當與否？

我們不難從中看出謬誤與可疑之處，因為上述行徑，正符合黑魔法的定義。

渴望生命中的美好事物，本無可厚非；但若動用心智與靈性之力去攫取，便落入黑魔法的範疇。

這並非指責朗達・拜恩與《秘密》一書的其他作者蓄意害人、陰險邪惡。然而依據古老智慧傳統的權威見解，他們所倡導的教義與實踐，無異於黑魔法，無論他們如何粉飾，將其裝扮成無害無辜的模樣。

只因一切為己、為滿足私欲的動念，哪怕再輕再微，都足以令任何靈性力量的行使由白轉黑。布拉瓦茨基夫人曾引一句古老的西藏諺語：「右手之道（白魔法）與左手之道（黑魔法）之間，相隔僅如蛛絲。」

問題的關鍵，終究在於此人是否已臻至全然無私無求之境；除了助人與服務人類外，別無他念。這才是靈性生活的真義。

一切痛苦起於欲望，一切欲望源於無知。讓「欲望」止息與捨棄，方是所有真誠靈性求道者的目標。這訊息，早由佛陀與其前的克里希納，在其教誨中完整闡明。

還有一處迷障，令許多人不知不覺墜入黑魔法的深淵。那便是「性密宗」的修習。

性密宗如同其他黑魔法形式，被其倡導者巧妙偽裝，致使許多天真的西方人誤以為它屬於印度教等古老東方宗教。因而，當這些人發現，絕大多數印度教徒對其厭惡甚至反感時，往往大為驚訝——性密宗在任何層面，皆不能代表這崇高宗教。

事實上，那些自稱對性密宗感興趣的新紀元信徒，一旦知悉其實踐步驟，多半驚駭退卻。印度教整體對性密宗亦極其冷淡，從未將其納入六派哲學體系之中。

布拉瓦茨基承襲歷代靈性大師的教導，指出：「若將肉體用於靈性目的，或以此作為工具達成形上目標，全然屬於黑魔法。白魔法師——右途上的悟者——深知魔法無需假借肉體，因肉體不過是物質皮囊。」

真正的東方神秘主義者則認為，生殖系統所涵容的一切，與其宇宙原型有著直接靈性連結；故在物質層面上是黑魔法的有效對象，且被視為不潔。

同理，兩人尋求性結合本無不當，但若試圖藉由性能量或體液以達致「靈性」目的，便淪為純粹的黑魔法。

布拉瓦茨基斷言，此類實踐「將招致最可怕的業果」；任何形式的「性魔法」，皆自動歸於黑魔法。

讀至文末，或有人以為我們只是「靈性的破壞者」；實非如此。本文出於對人類真切而深切的關懷寫就。欲望、野心、自私與感官之念，早已深植人心，我們並不奢望多數人聽從、甚或稍加思量此文所言。但哪怕僅有一人，因讀罷此文而從無知的黑魔法中脫身，我們便覺欣慰，一切努力也算值得。
