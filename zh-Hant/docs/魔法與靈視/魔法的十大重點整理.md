# 是魔法還是奇蹟？ — 魔法十大重點整理，人人皆是魔法師
## 魔法運作的定律，過去曾為人知，只是今日科學尚未識得

![cover](/img/downloaded/18e6a1baff3308e54ba88d5b9587324c.jpg)

Photo by [Aziz Acharki](https://unsplash.com/@acharki95?utm_source=medium&utm_medium=referral) on [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral)

布拉瓦茨基《揭開伊西斯的面紗》十點精要

一、世上無所謂「奇蹟」。

一切現象與結果，皆受永恆不變、持續運行的定律支配。卡彭特博士學識淵博卻見識有限，認為「奇蹟」違反「公認自然定律」，不該存在。然而，催生所謂奇蹟的，正是那些公認定律之外的力量。許多人如同卡彭特博士，忽略了一個事實：奇蹟運行的定律，過去曾為人知，只是今日科學尚未識得。

二、大自然實為三位一體。

存在一個**可見、客觀的大自然**；亦存在一個**不可見、內在而活躍的大自然**。後者與前者全然對應，既是其藍本，亦是其生命原則。兩者之上，則是**「靈」**——一切力量之源，永恆不滅。前二者屬較低層面，流變不息；**「靈」**屬更高層面，永不更改。

三、人亦是三位一體。

人擁有**客觀的肉體**與**活躍的星光體**，即「真實之人」；此二者由第三位「主宰者」所看顧照亮，亦即**不朽之靈**。當「真實之人」成功與靈融合，便成為不朽存在。

四、魔法作為一門科學，是對這些原理的認識。

靈魂寓於肉體時，能透過魔法獲取靈之全知全能，並學會駕馭自然力量。魔法作為一門秘術，是對此知識的實踐與運用。

五、濫用奧秘知識，即為巫術；善用其力，方是真魔法，亦即智慧。

六、靈媒與魔法師恰恰相反。

靈媒的通靈能力，是「被動」接收外來影響的工具；魔法師則主動掌控自身，掌控一切較低層次的法術。

七、一切過去、現在、未來之事，皆記錄於「星光界」之中，或曰宇宙不可見的石板上。開悟的魔法師藉由靈性視野，能知曉一切曾經發生、未來能知之事。

八、不同族群靈性天賦各異，猶如膚色、體態或其他外觀。有些族裔多人天生具備靈視力，另一些則具通靈稟賦。其中不乏沉迷巫術者，將秘法代代相傳，由此衍生形形色色的靈異現象。

九、施展魔法的一個關鍵階段，是自願且有意識地將內在之人（星光體）抽離外在肉身。

某些靈媒案例中，亦出現這種抽離，卻是無意識且非自願的。靈媒此時身體往往陷入不同程度的「僵直」；而魔法師的星光體離體時，旁人卻難以察覺。因其肉身感官依然警醒，外表僅似心不在焉——有人稱此狀態為「深思出神」。

對於遨遊的星光體，時間與空間皆非阻礙。深諳神秘科學的泰米爾人，能令自身（即肉體）看似消失，或化為任意形貌。亦能讓星光體顯形，呈現千變萬化之相。然而，上述現象，實際上是透過催眠在場觀者的感官而達成。此幻覺如此真實，以致目擊者以為所見為真。但這只是他們腦海中的景象，被催眠者施以無法抗拒的意志，深深烙在他們的意識裡。

星光體能無遠弗屆、穿牆越壁，相隔千里，並使形影清晰可見；但肉身的移動只能憑藉尋常交通工具。肉體或許能在磁力作用下，懸浮片刻，但若要從一地遷徙至另一地，卻終究得依循最平凡的方式。因此，我們不相信任何關於靈媒肉身御空飛行的傳聞——那類「奇蹟」，我們一概摒棄。無機之物尚可在某種情境下分解自身、穿透堅壁而後重組；至於鮮活的生靈軀體，則絕無此可能。

史威登堡學派深信，神秘科學亦昭示：**靈魂離棄肉身之事時常發生，我們日常生活中，常常遇見這般「活屍」。** 其成因紛雜：劇烈驚懼、深重哀慟、徹底絕望、急症驟發，或過度耽溺感官，皆可能引致此態。那具失神的軀殼，有時會被精於此道的巫者以其星光體侵入並佔據；亦可能受**亡靈**所據，即失去肉體的人類亡魂；或在極稀有的情形下，為**精靈**所附。當然，白魔法師亦具同等能力，但除非為了某項非凡崇高的使命，他絕不願玷污自身，入主一具不潔的皮囊。至於瘋癲之徒，其星光體多半癱瘓、神識紊亂，易受任何過路靈體左右；又或者，其靈魂已永遠消逝，身軀被靈體所佔，瀕臨潰散，狀若吸血鬼，拼命依附肉體，希望通過這種權宜之計，於短暫光陰裡榨取更多感官歡愉。

十、 魔法的基石，在於深刻體認磁與電之屬性、關聯及效力。

尤須熟諳二者於動物界與人類身上的作用。而如礦物等，亦各懷玄秘特性，石膏即是一例。一切魔法師皆須通曉此等性質，而當今所謂精確的「科學」，對此卻一無所知。植物亦具類似的玄妙特性，且異常優美。夢境與魔法藥草之秘，在歐洲科學中早已遭忽視，不為人所解。鴉片、大麻等寥寥例外，縱使存在，它們對人類精神之影響，也只被視作暫時性的神智紊亂。塞薩利與伊庇魯斯的婦女——即薩巴齊烏斯儀式的女祭司——並未因聖地傾頹而遺失其秘傳。這些知識依然存續，凡識得蘇摩植物本質者，亦通曉其他植物的特性。

## 總結

總而言之：魔法乃靈性智慧，大自然是魔法師的物質盟友、門徒與僕役。同一生命原則貫穿萬有，而臻於完善的人類意志，得以駕馭此力。魔法師能在超常層面上，激發植物與動物內在的運行之能。此舉非但無礙自然，反而助其加速，賦予更蓬勃的生機。

魔法師能操縱感官，改變他人的肉身與星光體狀態；亦可隨心所欲，統轄並差遣諸元素精靈。然而，他們無法控制任何人類的不朽之靈，無論活人或死人；蓋因人之靈皆為神本質的火花，不受任何外力主宰。
