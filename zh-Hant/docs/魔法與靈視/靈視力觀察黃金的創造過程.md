# 靈視力觀察 — 黃金的創造過程
## 作者描述在南非威特沃特斯蘭德含金礦脈表面，所見與黃金相關的眾神與自然神靈

![cover](/img/downloaded/88c0d5363d25f99d25a7b46a04fc9667.png)

摘自杰弗里·霍德森《眾神的國度》

## 礦物界

科學至今無從確知礦物基質的最初創造，或其實體如何成形——無論結晶、待結晶，抑或膠狀無定形。

我嘗試觀察此過程，記下一些片段，卻不敢自認已解開謎題。以下記述若有些許價值，或可視為旁觀者的筆記；或許未來將證實其有效。

物質媒介的存在，只能部分解釋礦物成因。眾所周知，特定比例的混合氣體，是溫度與壓力的函數，然而物質何以結晶？又為何總遵循特有的幾何形狀或分子排列？仍是未知。溶液蒸發後，規律的晶體總會顯現；同一物質的晶形往往固定，譬如鹽呈立方，明礬為八面體，硝酸成棱柱。礦物確會「生長」，但那並非新物質加入，而是外部媒介活動改變了內涵；多數晶體早已預先存在，隨水滲入，或趁地球冷卻時，自熔融狀態緩緩沉積岩中。

## 創造力的階序

孕育萬形的創造能量，最初藉由「話語」發聲。「話語」可視為源於一個靈性中心，在物質界由太陽代表。這股能量在其源頭威力無匹。從太陽大天使到行星天使，整個眾神族裔宛如層層變壓器。他們承接原始創造力，並以某種抵抗其流動的方式，逐步降低其「電壓」。創造力自太陽諸神起始，經由較小的同胞們逐級傳遞，終抵物質界。在物質界藉助自然精靈，將物質塑成「創造力心智」所構想的形貌。

聲音塑造形體的能力，可見於物質振動形成的有聲圖形。譬如，以小提琴弓，拉過玻璃或金屬板邊緣，板上的沙子便排列成幾何圖案。德國物理學家克拉尼便以此法製作幾何聲學圖形，這些圖案產生自振動板上的節點線，而沙子總匯聚於阻力最小之處。

![illustration](/img/downloaded/0848a3c4e7bdcb3ebba4da58737f9015.png)

法國科學家A·利莎朱爾斯創製出的曲線圖形，由兩種簡諧運動疊加而成。展示方式多元：或藉由兩支音叉對光束連續反射，或透過機械追蹤兩個擺錘的組合運動（如諧波儀），又或利用惠斯通桿。利莎朱爾斯亦曾使用一支水平與一支垂直音叉，同時振動產生圖形。當兩音叉相位、頻率相同或相異時，圖形隨之變幻。倘若物理聲音的造形之力，可歸因於超物質層面上的創造能量，或是「話語」力量所發出的聲波，那麼「邏各斯學說」便尋得了若干科學佐證。

因此，「話語」催生不斷演化形式，而協助邏各斯的諸神秩序被稱為「建造者」。此秩序的高階成員中，有一族在印度教中名為「乾闥婆」或「音樂之神」。他們覺知創造意圖，感知並理解原型或神之意念；在自身結合下降的「話語力」後，將之放大，尤其是與自身振動頻率相契之流，從而增強其塑造形式的力量。在建構者秩序裡，某些階序正是創造性「話語」中和弦的顯現，而產生的原型與形式，便是這些和弦的表現。

這種振動的親合力，將特定階序吸引至相應的工作領域，成為大自然四個界的形體塑造者。

譬如，黃金可視為創造性能量振動的物質結晶，依據使黃金顯現的力量頻率。黃金與一切物質相同，在創造的「話語」中呈現為和弦；黃金的神之意念以聲音的形式表達。這「話語」之力從靈性源頭發出，經由前述過程，觸及原初物質，使其在物質層面呈現黃金特有的分子排列與結晶形態。

## 黃金的靈體

此過程並非全然自發。存在一個精靈階序，其和弦特質與黃金相同。他們可謂黃金的神之理型，表現為活躍存在的秩序。該階序的成員，受振動親合力牽引，投身於黃金力量的洪流之中——這股力量持續從創造源頭降注物質世界。他們的存在與協助，強化了構成黃金的頻率，從而增強了「話語力量」的賦形能力。因此，黃金的精靈——乃至所有「建造者」神靈秩序——其職能在於協助物質與形態的生產進程。

在南非威特沃特斯蘭德等金礦脈地表，我目睹諸多與黃金相關的天神與自然精靈，涵蓋創造之力、賦予靈魂的生命，以及棲居其中的意識。在它們之上的空間或演化層面，皆存在更高階的神祇；而在這些神祇之上，隱約可感知一個黃金的行星智性體。這偉大的存在似乎已與黃金的力量、生命及意識交融，成為協調者、指引者與喚醒者。

在情感與具體思想的層面，則存在著一種黃金的集體意識。

黃金區別於其他礦物族群之處，在於其外膜、不同的創造力頻率，以及賦予黃金的靈魂更優越。若試圖描繪這股下降的力量，以圖示從三維視角觀之（非實際狀況），大抵類似一道發光的陽光錐形軸，自頂點（即行星的創造源泉）射入地底。這生命之流比該地區其他礦物的色澤都更鮮明奪目。當我契合至它的頻率或節奏，便感到其力量在我體內運作，能喚醒並激發人類體質中與黃金相應的一切。

黃金精靈的群體在黃金力量的下降流中移動；如文首圖片所示，這些存在的外貌略帶陰柔。臉龐予人印象極蒼白，肌膚幾無血色。而「髮絲」（實為流動的力量）呈亞麻色，泛著金光。氣場以流動的曲線向下掃掠，帶著葉綠、玫瑰、黃與淡藍等柔和色調，並隨其下降逐漸拓寬。氣場下三分之一布滿無數金色光點，疾速運動；越近這美妙氣場的袍裾「下擺」，光點越是稠密。這位天神或女神的整個形態與氣場，皆閃耀著黃金光澤。

奇的是，其他較低階的黃金精靈態度漠然。他們偶爾靠近荒地地表，以鏈狀交織的方式緩移，宛如某種莊嚴舞步。同時保持著手臂的優雅擺動，似在手工播種。他們顯然對外界視而不見，面對「下降」的黃金力量、生命與意識之流，則以自身思想，為這三股流注入額外的力量與個性。縱使我引得一位的注意，它也只能如隔金霧般朦朧顯現，無意費力回應。

各類自然精靈深藏地底，往往在一英里之下。有些精靈形貌怪異，類似半人半獸，那是早期演化周期中，大自然努力構建形體的遺跡——長而瘦削的尖臉、裸露多疣的人形身軀，以及由地下巖石構成的腿足。他們似乎操縱著土的力量；在此過程中，各自運用巨大意志，彷彿在錘鍛與焊接下降的能量，使其趨於同質與統一。然此工作並非手動操作，而是本能發揮意志的結果。

他們似從這活動中獲得極大滿足，體驗一種掌控強力的感覺，這賦予他們持續專注的沖動。

黃金出現在岩石中，宛如細珠；最微小的黃金精靈在其周緣與內部遊移，狀如螺旋的金色微小菌體。在空靈層面，可見無數精靈於沈積的下降流中「游動」。整體予人印象如同一座巨型實驗室，內有無數做工者；各種元素持續形成，並有一策劃者主持其間。

黃金的天神與自然精靈，似乎對採礦並無反感。他們全無個體性，黃金被帶往何處，他們便與其中的生命接觸。類似地，巖石侏儒亦不介意巖石被擊碎。反之，爆破與鑽探刺激了他們；他們陶醉於力量的展示，不在乎巖碎。事實上，他們甚少感知巖石的堅硬，更多感知壓縮其中的力量。他們能看見枯燥的工具，卻不知人的存在；因其頻率與人相去甚遠，對人類幾乎視而不見。他們視自身的工作為偉大的遊戲；他們喜愛這遊戲，因其能刺激他們，增長意識與活動。
