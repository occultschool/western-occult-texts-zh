#  星光界場景 — 迷失的孩子
## 任何愛，只要能助他們脫離那流浪的漩渦，都不算無私

![cover](/img/downloaded/25d253c7321d1f1ac03d0cc53d3ec4a4.jpg)

Photo by [Enrique Ortega Miranda](https://unsplash.com/@enriqueo?utm_source=medium&utm_medium=referral) on [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral)

這段摘自《神秘之謎》的短文，出自一位真正靈視者之眼，描繪許多人死後那駭人的真相。必須補充，米紹博士的描述毫無誇大；我們在星光界見過極其相似的景象，實在令人心碎……

「你若見過那漩渦中，無數低等進化、焦慮困惑的人類，隨一道巨大洪流在星光界邊緣漂流掙扎，你終將明白：此處付出的一切辛勞都談不上艱難，獻出的所有服務都稱不上偉大，傾注的任何愛都算不上無私——只要能助他們掙脫流浪的渦旋。他們不知發生了何事，不知身處何方，一雙雙悲傷的眼望著你，無聲懇求援助。只要見過那幾百萬張驚恐的臉一次，便永生難忘。

「他們有的飄蕩在空中，顯現生前那脆弱虛幻的形貌，一邊詛咒，一邊祈禱；一時愛，一時恨。全是鬼魂……神所遺落的孩子……非得等到再次被肉體束縛奴役，他們才彷彿得救；須忍受更深一層的禁錮，直到終於有資格獲指引前路。

「道路固然狹窄，但依神意，人人皆能踏足。更甚者，已有大批先行者在前路等候。他們殷切盼你到來，對你這疲憊的旅人張臂相迎；欣慰你於陰影中的任務已然完結，更輝煌的工作正要開始。

「他們就在那裡等著你：那些大師，天界的白光同胞，等候那些仍處塵世陰暗中的同胞。如今他們已成智慧、愛與慈悲的火焰，準備將自身光芒匯入更浩瀚的神聖之火；為尚未到來的人照亮前路。」
