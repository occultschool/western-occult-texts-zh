#  古埃及魔法的神秘與啟示：白魔法與黑魔法的深度探討
##  探究布拉瓦茨基夫人的作品及古埃及魔法體系的真實性與影響力

![cover](/img/downloaded/551aa1ac8ad802b05561a1d7034251f1.jpeg)

探討古埃及秘術的著作不少，弗洛倫斯·法爾的書是其一，可見於本站[外文書籍推薦](/docs/general-hermetic-book-recommendations)。然而論及此道，鮮有作者比布拉瓦茨基夫人更為透徹真切。她最知名的兩部作品——《揭開伊希斯的面紗》與《秘密教義》——正是本篇文章的底本。此外，我們也直接引述了記載魔法的紙莎草文獻，其中最著名者，當屬藏於倫敦大英博物館的哈里斯紙莎草紙。此卷年代久遠，可溯至公元前一千兩百年左右的第二十王朝。

這些殘卷雖零碎，卻證實埃及自古並存兩套涇渭分明的魔法體系：一為白魔法，亦稱神聖魔法；另一為黑魔法，即所謂巫術。二者皆傳承久遠。《死者之書》的年代早於《創世記》或《舊約》諸卷，字裡行間處處透出這種分野。書中滿是連綿的祈禱與驅逐黑魔的咒語。奧西里斯被尊為「空中惡魔的征服者」，信徒向他懇求援助，以對抗馬塔——切莫與象徵正義真理的瑪亞特混淆——因那馬塔「自眼中射出無形之箭」。

這「眼」與「箭」究竟何指？所指正是「邪惡之眼」。無論施術者是生是死，皆能以此「邪惡之眼」施加催眠般的暗示，其影響「遍及寰宇」。此概念源於前文討論過的[以太奧祕特性](/docs/mysterious-aether)，以太雖根植於太陽系的本質，對星光界與物質層面的作用卻屬塵世。甚至基督教裡也存在相同信念，認為某些禱詞便是針對「黑暗中飛馳的箭」。

在埃及，神聖魔法稱作「白魔法」(Netra)，巫術則名為「黑魔法」(Heka)。後世女神赫卡特之名，便由此詞演變而來。埃及人也精於決疑占星術，這門技藝兼具科學，透過精密計算行星恆星之位，及其與地球的角度關係，以預示吉凶。

前述哈里斯紙莎草紙中，我們發現一份詳列吉凶日子的清單，足見此信仰古老且綿長。「五月二十日使牛耕作者，必死；同月二十四日高聲誦念塞特之名者，其家自當日起便歸他統治。生於包菲月第五日的孩童，將斃於牛角；生於二十七日者，亡於蛇吻；若生於阿瑟爾月第四日，則難逃重擊而殞。」

哈里斯紙莎草紙將死者靈魂分為兩種。其一通過審判，在奧西里斯座前稱量心臟，滿盛正義真理，因而得以在第二生命中重生。換言之，在下次轉世塵世之前，此魂可暫居更高層的星光界。其二則為罪魂，「死於第二次死亡」，即遭受懲罰。第二次死亡並未毀滅這些幽暗靈魂，他們注定在低層星光界徘徊，直至償清罪孽。

那些研習黑魔法而被捕者，下場極其嚴酷。弗朗索瓦–查巴斯曾援引哈里斯文獻中的一例：一名喚作海伊的牧羊人，專司看守（或飼養）羊群。查巴斯寫道：「他獲得一卷載有拉美西斯–梅里–阿蒙祕法的書，這位偉大神祇亦是他的皇室主人。他成功汲取力量，得以蠱惑他人。更造出『魔法侏儒』，施行情愛咒術，並將心中所思一切恐怖憎惡之形俱都顯現，令眾神與女神俱感驚懼。」於是，審閱黑魔法師案卷的法官宣判：「依法老之命處決其人，並按神聖語錄所載，將其徹底消滅。」

換言之，這牧羊人所受的極刑，正與他藉以行惡的黑魔手段如出一轍。

或許有讀者知道，赫爾墨斯號稱人類的「牧羊人」，正如後來耶穌為師時也沿用此喻，稱信徒為「羊」。由此推想，「牧羊人」大抵指某座神廟的祭司，負責照護信眾，卻濫用神職，墮入黑魔法，終致毀滅。這教訓意味深長，堪為沉迷邪術者借鑑。

埃及亦早有惡魔附身之說，孔蘇神廟牆上銘文可證——此神本有伏魔之力。此事溯至第二十王朝拉美西斯十二世年間。銘文記載：一位拉美塞德法老戀上附庸部族巴克頓酋長之女，迎至埃及立為王后，賜名拉-涅飛魯。未幾，巴克頓首領遣使求見法老，因其妹本特-雷什四肢染怪疾，盼望埃及學術施救。

使者明確要求法老派遣一位「智者」，即啟蒙者。法老遂命召齊宮中術士與秘典守護者，最終擇定皇家文士塔胡提-姆-赫比前往診視。塔胡提-埃姆-赫比抵巴克頓，斷定女子為惡靈所附，卻自承無力驅除。十一年過去，少女未見起色。巴克頓首領再度遣使懇請，孔蘇神乃化身，親赴彼邦。須知孔蘇原屬底比斯三聯神——「阿蒙-拉」為父，「穆特」為母，「孔蘇」為子。

孔蘇神驅散了附在王后妹妹身上的惡靈，王后含淚道：「神靈顯聖，驅走了邪祟，我今歸於神座下，願為汝僕！」學者多以為，能醫治她的，是神像或某種護符；這說法我們不敢苟同。布拉瓦茨基論及魔法時亦持此見:「顯而易見，此乃真正的大祭司，名列『神之子』，被描述為孔蘇化身，是名圓滿的啟蒙者。」

我們認同這位知名神祕學者之見。在同一銘文也記載，此啟蒙者所屬的廟宇亦設魔法學校。內有聖殿，唯最高階祭司可入，其中藏置神聖經卷，由專職祭司研習保管。這些典籍無一傳世，或許反是好事。

如今僅存托勒密時期（約三世紀）紙莎草殘卷，滿載召靈之術、驅魔之法，對象兼及亡魂與低階魔神。可見埃及人能清晰分辨真正的諸神（即行星統治者與天使）、下界邪魔，以及人類死後滯留星光界之空殼——亦即神祕學家所稱「亡靈」。

上文已述古埃及人對白魔法與黑魔法之別。白魔法，或曰神聖魔法，被尊為一門聖科學，使人參贊神性。希羅多德、泰勒斯、巴門尼德、恩培多克勒、奧菲斯、畢達哥拉斯、柏拉圖、阿波羅尼烏斯，皆曾於各自時代赴埃及求教大祭司，以解宇宙之謎。斐洛有言：「秘儀揭自然運行之理，啟人思悟天上之力。」

然若擁有過大的權力、知識妄用、野心橫流，常誘使啟蒙者墮入自私不義的黑魔法，埃及亦未能免此弊。瓊-格蘭特在其小說《荷魯斯之眼》與《地平線之主》中寫道：第十一王朝時（約公元前三千年），埃及歷代法老或專制或昏庸，實權操於一群沉迷黑魔法、惡習與腐敗的祭司之手。最終，神聖秘儀之廢，實因黑魔法猖獗，而非如常人所誤解乃基督教會所為。

早在公元前五六零年，羅馬人已察覺一派極其穢惡的黑魔法學派。此派承襲伊特魯里亞秘儀，其道德敗壞之風迅即蔓延全意大利。據提圖斯-利維烏斯載，一年之內，竟有三千啟蒙者因涉投毒罪被判刑。所幸今日自稱鍾情黑魔法、循所謂「左道」者，鮮具真知實能。

且擱下這不快話題，轉論神聖魔法——此學曾是、至今仍是埃及最偉大的科學。於古埃及人而言，至卓越者乃歷經艱苦修練，制伏自己的動物本性，自低等慾望灰燼中如鳳凰重生，擁有一顆不為物質幻象所蔽之心；他們涵容過去、現在、未來，且知三者本是一體。

在埃及，白魔法師是在「生命之宮」經歷二十一載深修嚴訓的菁英。此「生命之宮」實為一處授授玄秘之學的「神廟—學院」。他們構成哲學上的貴族，向世人敞開宗教之門，引領眾人穿行生命的荒原，直至輪迴多次後，靈性與德性臻於成熟，終得匯入啟蒙者的行列。

埃及神聖魔法的根本，奠基於一套精密的對應法則：物質的開展，必遵循靈性所顯化的原型與象徵；靈與物質，實為同一奧秘的兩面。埃及的悟道者深知，大腦所思、靈魂所向、口中所述、身體所行，彼此呼應，足以構築完整的宇宙法則與實用生活準則。他們更相信，凡有形的軀體或現象，皆以物質為根基；而其內在的非物質本體——我們稱為「高我」，他們稱作「巴」或靈魂——自至高光明之境降臨，寄寓肉身，復經重重輪迴，終將回歸萬物本源。

普羅克洛注釋柏拉圖《蒂邁歐篇》第五卷第330頁時，論及靈魂：「靈魂的種子落入生成之域，她須得從流轉的塵世中抽身，滌淨自己。因她懷抱雙重力量——一者，導向生成；一者，自生成回歸真實存在。前者領她周旋於塵寰輪迴，後者則護她繞行智性的軌跡。」此處點出高等與低等心智，及其各自的進化歷程，與我們過往諸多文章所述呼應。

埃及聖賢歷經數代，錘煉出一套精妙的象徵體系。諸神，或曰大自然的普遍力量，皆以獨特鮮明的形象示現。正如我們曾論及的獅身人面像，融合了動物、凡人與神性於一體；每位神祇亦透過象徵形貌，展露其獨特的神性、力量與特質。這與俗常誤解相悖：埃及人從不膜拜貓、狗、牛、鷹、獅、蛇或鱷魚！他們僅是洞察這些動物——以及其他眾多生靈——蘊藏某些獨特稟性，遂藉象徵比擬，以這些形象傳遞關於自然之力（即諸神）的奧義。

部分女神繪作獸首，如賽克邁特與哈托爾，藉由母獅的勇猛、母牛的哺育，象徵其特質。伊希斯與瑪特則始終以人首示現，分別喻示母性與靈性重生、智慧、正義與真理。諸神的冠冕、衣飾與手中權杖，更承載無數屬性。這些元素，共構埃及神話中神聖魔法的肌理。

流光推移，偉大的啟蒙者不復存在，神聖真理亦大半湮沒。神聖科學漸次蛻變為今日流俗的偽魔法信仰與操演，憑藉二、三世紀的世俗文獻與希臘紙莎草流傳於後世，而非源於古老大祭司的神聖魔法。萊頓紙莎草約於公元二五〇年以希臘文書寫，其內容綱要，正呈現這類魔法文獻的典型旨趣：

1.  招引幸福與好運的符咒。
2.  實現夢想的咒語。
3.  諮詢神靈的儀軌。
4.  止息怒火的咒文。
5.  使戒指能賦予事業成功、好運之能的咒法。
6.  其他功效相類的戒指。
7.  使夫妻或他人離異之法。
8.  令人至死難眠之術。
9.  使自身得寵、友誼長存的符籙。

由此可見，近兩千年來，世態並無多大變化。我們在現代神秘學的交流圈子裡，目睹全然相同的情狀；而那些易受矇蔽或經驗未足之人，正如古希臘與後世的埃及民眾一般，對這些事物傾注熱切興味！因而，我們實有必要對埃及魔法作一回顧，願此番探勘能矯正若干謬見，並為你積澱豐厚素材，以供日後深思與研尋之用。
