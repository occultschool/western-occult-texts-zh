# 巫術的危險
## 巫術與黑魔法皆由實踐者的意志駕馭。他們從內在投射出一種磁性流體，學會以全然自覺且科學的方式塑造、導引此流體。

本文將深入探討當代人對巫術的迷戀，同時揭露此舉為涉入者帶來的巨大風險。

首先須釐清，現代人常對相關名詞感到困惑：巫術、魔法與神秘科學之間有何區別？在古埃及，智者明確將占卜、咒語、通靈、連結「靈體」等行徑，歸入巫術（Heka）範疇——儘管此舉可能令某些所謂「白女巫」不悅。而神秘科學被尊為「神的科學」（Netra），代表至高智慧。因此可以說，至高智慧涵蓋了當代所謂神秘科學的實知與應用；巫術則是其陰影，多為私利服務。

魔力是人人與生俱來的天賦，超乎尋常感官。這不違背自然，而是一門科學——建立於對神秘之力、顯隱世界運行規律的深透領悟。但這奧義從不載於典籍，未嘗落為文字；唯有真正的覺者或明師能授之以鑰，開啟那道幽玄之門。這知識如雙鋒之刃，非愚昧自私者所能持。

正如十六、十七世紀的記錄所示，昔日神祕學門徒曾遭慘烈迫害，而置身於額外險境。一百五十年間，歐洲近九萬人因所謂巫術被焚。他們如野獸般遭「基督教」教士獵捕；教士視神秘知識的傳播嚴重威脅其信仰生活，遂決心殘酷滅絕。僅在大不列顛，一六四〇至一六六〇短短二十年間，便有三千人因與「魔鬼」立約而被處死。時至今日，仍有牧師樂於鑽研法律漏洞，以延續這類迫害。

當代一般女巫對於「咒語」所引發的現象、背後的隱密法則與原理，幾乎一無所知（儘管她們總假定咒語有效，此事頗值懷疑）；因而只能在黑暗中盲目摸索，至多僅能觸及模糊輪廓。她們亦不斷面臨接觸低等元素精靈的風險；這些實體可能利用她們達成自身目的。反之，真正的白魔法師通曉靈性與物質法則，能有意識地運用它們行善，不受此類威脅；因為他們時刻掌控全局，迫使一切低等靈體服從命令。

另一重危險，在於許多巫師與異教徒濫用精神藥物。某些藥物的確能令心智掙脫肉體束縛，但這不牽涉靈魂。若欲釋放心智，亦可透過所謂的「星光體投射」。此為意識延伸之法，使心智得以探訪任何地方或存在，無論其距離身體多遙遠。但若未得明師正導，隨意神遊星光界者，易受其中充盈的元素存在侵擾，淪為它們邪惡圖謀的犧牲品，身心俱遭可怖之後果。

這便解釋了通靈者所見的「動物靈體」為何，他們時而目睹貓狗豬畜，並與之對話。這些實為元素精靈，顯露出人性墮落中最不堪的特質，卻非人類本身。亦有靈視者會見駿馬與所謂的「天上騎士」；或有黃鳥棲肩，在耳畔低語駭人之事。此類元素精靈，有的自行其是，有的聽命於熟知操控之術的黑魔法師與女巫。對未受啟蒙者而言，它們無一不是邪惡且危險的，但描述這些星光界來客時，人們卻常流露狂喜之情。

他們忘了，或根本不知，這類異象源於外在，而非內在。凡來自外者，永不可輕信，因真理唯在己身之中。外在的星光界可藉由特定法術踏入，而棲居其中的元素精靈能襲擊人類，將其徹底摧垮，猶如颶風掃蕩森林。這般精靈可同時進犯千處而不顯形；亦可聽從巫師指令，擇定男女加以侵害。即便啟蒙者一時鬆懈，亦可能遭襲；然此時同道必將速來相救。

你可能會問：一位智慧淵深、具足神聖之力的開悟者，怎會為區區元素力量所傷？只因開悟者往往心繫他人，時而忘了以護盾環繞自身，使負面之力無法穿透屏障。他們的使命是服務眾生，竭盡所能護導尚未開明之人，卻常疏忽了抵禦邪惡，這是他們屢需避免的疏失。在塵世不可見的氛圍裡，棲居著無數靈體，你可於布爾沃·萊頓的神祕學小說《扎諾尼》中窺見一斑。

星光界潛伏著無形卻真實的磁性吸血鬼，與小說家及好萊塢B級片的繽紛想像大不相同。那裡也存在秘典記載的諸般惡魔，即古人所謂的夢魘與魅魔。偉大的埃及啟蒙者摩西深諳此類存在，故而頒布嚴酷律法，以懲治遭附身的女巫；然而耶穌懷著神聖的正義與對世人之愛，選擇驅逐元素精靈以治癒患者，而非將其毀滅。可惜當今多數教會人士的見解，卻與摩西和耶穌的智慧相悖：他們僅僅抱持無知的否定，將巫術視為純然迷信。

女巫、巫師、狂人乃至罪犯，體內皆可能棲宿邪惡靈體。若能成功驅逐此靈，解除附身，便可達成療癒；但前提是，受害者須由專業之士持續觀察一段時日。通常受害者心神已然脆弱，受的缺損需以細緻耐性的引導緩緩修復。聖保羅知曉如何喝令邪惡靈體離開受擾者之身。《卡巴拉》亦提及：當對附魔者闔上門扉，惡魔便如遭驅趕般逃竄。

這意味著，你絕不可讓這些靈體主宰你、進而附身於你。避開它們最好的方法，正是遠離惡行——因為惡行會形成罪孽的氛圍，吸引它們前來。

元素精靈畏懼劍鋒刀芒。在《魯魯的尋求》裡，候選人魯魯揮動聖劍，便嚇退試煉途中襲來的萬千精靈。

或問：若不憑藉典籍，真正的巫師如何獲取術法？其秘在於師徒啟蒙，而最強的武器，乃是某一句「話語」。真正的巫師必須將此「話語」傳予弟子，才能離世。這是眾所周知的事實；巫師若瀕死卻無徒可傳，便會困於痛苦，徘徊不去。知曉此事者皆不敢靠近他，因他只需將話語傳給任何人，便能解脫。據聞曾有位醫師，臨終巫師對他耳語了可怖的話語，巫師遂得解脫，醫師卻無法承受知悉秘密的後果，一個時辰後舉槍自盡。

不久以前，連皇室也曾借助巫術，譬如那位背棄教派的雅各賓牧師。他法術無雙，深得凱瑟琳·德·美第奇王后的讚賞與庇護，他能為虔誠的女主人遠距殺傷，以咒語折磨敵人的蠟像。當王子查爾斯重病時，她命隨扈帶來一名幼童，餵食聖潔白餅後斬首，將駭人的頭顱置於祭壇黑餅之上，以此頌讚魔眾。他們要求魔鬼宣讀神諭，卻毫無效用，查爾斯最終還是病故；然而凱瑟琳仍堅信自己是羅馬教會的忠貞女兒。

人類各種族的靈性天賦各異，猶如膚色與其他特徵。有人生為靈視者，有人是天賦靈媒，還有人深陷巫術的誘惑——如尼日利亞、孟加拉乃至昔日西藏等地，皆將此種知識代代相傳，伴隨着或多或可怕的通靈現象。可悲在於，教會已不信魔法巫術，現代世俗權柄亦嘲弄女巫之說，任憑這些墮落的男女持續邪行，無從辨識，亦無阻礙。事實上，許多歸咎於所謂「精神疾病」的罪行，例如犯罪者聲稱受到某「聲音」驅使，實為各類精靈附身佔據的結果。

巫術與黑魔法皆由施術者的意志駕馭。他們從內在投射出一股磁性流體，並學會以全然自覺、近乎科學的方式塑造引導。這在某種程度上，類似無線電台投射特定波長。但不同之處在於：無線電波須接收設備完全配合，方能捕捉波長，使其可視可聞；巫師卻能以意志之力操控對象，使其服從命令、任其影響。此法與催眠術相似，卻更為強大。例如印度尼爾吉里斯地區的穆拉·庫龍巴巫醫，或錫金、不丹的杜格帕部族巫師，當他們意欲剷除受害者時，便以訓練有素的意志達成目的。

他們的意志絕對堅定，並不受制於受術者神經的感受性強弱，一旦巫師選定受害者，並建立起感應相通，其「流體」必會尋得路徑，直抵目標。若此人被注定要死，他便會死；若被注定要去完成某個行為，他就一定會去做；若被注定要看見某種異象、景象或人物，他便必然會看見。巫師的意志之力無可抵擋，除非有更強大的意志前來拯救。這迥異於歐洲「實驗者」那般隨意的手法，他們對所操弄的力量一無所知。

這些巫術與術士的傳承，源頭可追溯至百萬年前的亞特蘭提斯與雷姆利亞。巫術的施展，往往源自於強勢的意志對弱者施加的惡意。魔法本具雙重性——一旦染上邪念，便輕易滑向巫術深淵。對神秘學的學生來說，分辨左道與正道極其艱難；其中的奧義符號，唯有開悟者能以口耳相傳。

坊間流傳不少高階魔法典籍，聲稱無需導師指引即可修習，但對未受啟蒙者，這實是險途。稍有不慎，這些書冊便淪為黑巫術的媒介，為學生與其家族招來無盡災厄。

古埃及人清楚區分亡者的星光體外殼、亡靈惡魔、行星神祇與真神；若有巫師濫用這類知識行黑暗之事，一經發覺，立處極刑。希羅多德、泰勒斯、巴門尼德斯、俄耳甫斯與畢達哥拉斯，皆曾遠赴埃及，向大祭司求取破解宇宙之謎的智慧。十一世紀的希臘哲人邁克爾・普塞洛斯曾言：「魔法乃神聖學問的終章。它探究月下萬物的本質、力量與屬性——元素與其構成、動物、草木果實、金石藥草，無所不包。」這是門神秘「科學」，與巫術及女巫的差別，正如燦爛晴日之於地窖中搖曳的燭火，而窖中盡是惶惑盤旋的蝙蝠。

倘若本文能點燃你對神秘學的探求之心，助你看清它與其物質暗影——巫術——之間的深淵，那便是我們最大的期許。
