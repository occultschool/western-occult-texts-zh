# 因果報應的調查
## 揭開因果報應學說的面紗

![cover](/img/downloaded/3a8ceb7e43c32e30d38d44f55975cdd1.jpg)


在神秘學研究課程的第三部分，我們直面「因果報應」這令人困惑的概念，探究其本質與謬誤。

關於因果報應的闡釋，眾說紛紜，光怪陸離。倘若因果報應是頭驢子，恐怕得生就一副寬闊的脊背，才馱得動人們強加於它的種種指責。一位神祕主義者曾說，某人之所以整骨手術失敗，乃因前世效力於西班牙宗教法庭，專司酷刑。我們不信這套，深知許多良善之人，亦因醫者疏失而受苦；你必然也不會相信。儘管如此，仍有人將此邏輯推演得更遠，聲稱手術失手的醫生，前世曾被這些病人傷害。

更有些怪誕的東方教派相信，殺雞斬牛者，來世必投生為雞為牛，身受同樣命運。但願這並非事實！我們雖未親自宰殺牲畜，卻可能做過同樣殘忍的事：上週才噴灑殺蟲劑除蛾。我們可不願來世化作飛蛾，經歷同樣遭遇。不過，也許飛蛾蛀蝕衣裳在先，因果報應由此介入，驅使我們噴霧懲治——如此說來，一切倒也合理。

我們認為，靈性追求者往往過分憂慮因果報應。因果報應對他們而言已成心病，滲透思緒，干擾行動，滋生恐懼與罪惡感。另一些人則認為因果報應過於複雜難解、晦澀幽微，索性放棄理解。

事實上，正統的因果報應學說，核心不過是「因果」。

多數神秘主義者都能理解並接受此正統學說。權威典籍皆言，因果報應即是「因」與「果」的法則。你若作惡，惡果遲早降臨，好教你學會日後行事更善。但此說前提是，你至少不能比驢子笨！驢子不會被同一塊石頭絆倒兩次，牠第一次就學乖了。這是否意味著驢子能在進化階梯上攀升，來世投胎為騾，甚至成為一匹壯碩的拉車馬？我們認為，這得看牠成功避開石頭時，那份新獲的智慧在心智中刻下多深的印記。

這是理解「因果報應」真義的第一個線索：行為對心智的影響。我們來深入探討。古印度教義中，「因果報應」或「業力」（Karma, Karman）源於梵文，意為「行為」。此外，在印度哲學與日常語境中，指的是「一個人的行為決定其未來命運」。這不只是模糊的善惡有報，而是指每個行為都必將引發其無可避免的後果，承受其報應，無論過程需要多少年才到來。物質宇宙的每一部分——男女、蟲蟻、樹木、石頭，乃至令我們食壞肚子的小蟲——皆是一個永恆之靈的寓所。萬物正決定自己的命運：領受過去行為贏得的獎賞或懲罰，同時為未來播下獎懲的種子。

東方視存在為無盡的播種與收穫。博學者與無知者皆接受此觀點，這是人世不平等的緣由；若非如此，人們或將質疑神之公正。然而，這並非意味印度教徒不相信天堂與地獄。印度教中，二者以各種形式存在，或多或少有些怪誕。據說死後，高我會前往其一，安於應得之位；但這取決於欲望與行為所積累的功德或罪過。若高我仍困於業力之網，便須再度投生塵世，繼續修行。印度教的救贖概念，在於從因果報應之力中解脫，各派哲學自有其解脫法門。此學說源頭雖不可考，但無疑屬後吠陀時代產物。公元前六世紀，佛陀便欣然接納（儘管作了相當修改），至今仍存在於婆羅門教、佛教及諸多「新紀元」信仰體系。

此教義進一步闡明，所行之事皆有永恆影響，簡言之：「種瓜得瓜，種豆得豆。」行動並非單一，而是包含三要素：構思行動的「思想」、尋求實現方法的「意志」，以及二者結合以「付諸行動」。由此可見，思想力量強大，無論善惡。思想如何，行動便如何：滿心貪婪的守財奴慾壑難填，念頭污穢的浪蕩子惡行不斷。反之，心懷善念者，行為自然展現美德。然而，因果報應學說不能僅限於一世，必須考量轉世。初生的男女帶著前世的種子；他們的性格承襲過往，也將延續下去，除非他們決心改變——人人皆有此能力。在每一次接續轉世中，性格會朝某個方向愈發固化。若是邪惡，改變便日益艱難。邪惡的誘惑更加強烈，而難以抵抗；然而，真正明瞭業力之人，縱使最終屈服於命運，也必先經歷最頑強的掙扎。請記住，這並非我們的信仰；到目前為止所述，不過是東西方普遍接受的因果報應學說，而這是多麼陰鬱且令人不安。

## 因果報應的誕生與傳播

然而，僅憑上述就能解開因果報應之謎、闡明其運作嗎？我們一直強調的是思想的力量，或曰心智之力。只要意志足夠強烈，心智所造之物便能化為實體。心理學家稱這些心智圖像為幻覺，但在創造者眼中，這與可見之物同樣真實。意志愈強，專注愈深，便能將這些形體在眾人眼前變得客觀可見。思想若受強力引導與維持，便能成就這些壯舉。數千年前，「因果報應」之說尚不為人知，人們自由快樂地生活，如空中飛鳥、田間百合，不思明日之憂，不惱昨日之逝。然後，有人想出一個主意；我們猜想他是祭司，且必是射手座——這類人總想改變一切，美其名曰「進步」。他將此念傳與同僚，告訴他們，只要齊心協力，向世人宣揚這個新思想，祭司的權力便將更加強大；最終藉由恐懼，他們將統治世界與一切財富。

於是，因果報應學說（或許稱之為教條更貼切）公諸於世，祭司掌權的時代就此開啟。人作為神之子女本應享有的自由真知，被抹殺遺忘。強力的儀軌與勸誡、因果、罪罰、對死亡與來世的恐懼，這套新學說被無情灌入人類心智，自此，人便成了它的奴隸。我們因而目睹這可悲景象：牧師們在講壇上無休止地宣揚毀滅，將對神的恐懼注入人心，而非以神之愛充盈會眾的胸懷。

任何人皆可於《西藏度亡經》（Bardo Thodol）中讀到，狡黠的西藏喇嘛將地獄描繪最為精緻狂喜的殘酷刑罰；或者中世紀及後期「神聖」宗教裁判所，對異端、女巫與術士的可怕迫害、火刑與拷打。今日則懷著二十一世紀的沮喪，聽見狂熱毛拉吹響仇恨的號角，他們要殘害、切割、謀殺伊斯蘭的敵人，甚或穆斯林同胞——只要對方恰巧屬於不同教派；而這一切，皆以真主——那慈悲仁愛之神——的名義進行。最後，我們也見證《和平王子福音》中偽善的女佈道者，她詛咒同性戀者與在美國墮胎之人，祈禱他們死亡。

## 何以沒有「因果報應」？

若你還未看清因果報應的本質，我們願直言相告：因果報應是某種意識狀態的認知，與現實無關。當我們說「沒有因果報應」，但願這不會令你過分震驚。我們再說一次：因果報應並不存在！我們不信此說，正如我們不信罪與罰的教條。

我們的思想與行為，若是消極與破壞性，便會連結宇宙萬物的黑暗面；若是積極建設，則根植於真理與善的光中，不像那些懦夫蜷縮於黑暗角落，與地獄眾魔為伍。此間並無中間道路，除非將第三種狀態稱為漫無目的的漂流，連嘗試轉向光明或黑暗都不願。但任何理智之人不會視此為「道路」：我們要麼是黑暗的居民，終日憎恨、呻吟、毀滅；便是光明的孩子。中間地帶並不存在。

然而，每天都可見焦慮的人們，悲傷地與我們擦肩而過。我們痛苦地意識到，確有太多人於生活中漫無目的地漂泊，內心充斥恐懼與虛妄想像。我們撰寫此文，正是為了幫助這些可憐的迷途者，讓他們認識到，有個更偉大生活等待著他們，認識那不斷召喚遊子歸家的光明燈塔，以及那份浩瀚的愛、耐心與理解——如此深切渴望再次將遊子們擁入庇護的懷抱。一旦你認識那偉大的光，便會覺知它對你完全的護佑，以及閃耀著永恆真理；至此，黑暗、恐懼、疾病、迷信或因果報應這些念頭，便再無從想起。

但請勿就此來信，引用各派「神秘學」與其他玄學「權威」，試圖證明我們的論點錯誤，因為我們不會回應。神賦予人人自由意志，接受或拒絕此教義皆由己擇。但我們希望，你若已讀懂並理解我們對此議題的闡述，便會將「因果報應」一詞徹底從你的詞彙中抹去。如此，你將不再是「命運」或「宿命」的受害者，而是扮演積極角色，塑造自身未來「轉世」，這正是本神秘學課程系列第四篇文章的主題。
