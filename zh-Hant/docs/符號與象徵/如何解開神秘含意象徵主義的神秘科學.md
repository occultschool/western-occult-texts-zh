# 如何解開神秘含意 ?— 象徵主義的神秘科學
## 每個符號、寓言、神話、古代文字、聖書或神秘書籍，皆蘊含至少三層意義。

![封面圖片](/img/downloaded/ec2922a07545c671ecb61faea160fbab.jpeg)

本文旨在闡明，為何象徵主義是神秘科學的重要分支，並揭示其運用之法。

## 導言

神聖典籍，例如《聖經》與埃及《死者之書》，皆可從三種層次閱讀：

1.  **字面層**：對應物質界與小我，訴諸理智。
2.  **象徵層**：對應星光界與高我，訴諸直覺。
3.  **靈性層**：對應靈性領域與靈魂，直指神聖。

正因如此，單憑智性無從理解任何聖典、神話或傳說。必須借助鑰匙，方能解開其隱藏的意義。這些鑰匙包括：象徵、寓言、數字學、占星術，以及語素文字（亦稱表意文字）。

語素文字是最早的編碼方式之一，用以封存隱秘的訊息。古埃及的象形文字，幾乎每一枚都承載著一個完整的詞彙、概念或思緒。即便今日，中文仍保留這古老的溝通模式。《聖經》中的許多詞語，乃至完整句子，亦是語素文字，其真義往往迥異於世俗的解釋。

這表明，每一個古老神話、傳說或民間故事，都封存著某些隱蔽的歷史真相、科學原理、自然法則或哲學真諦。它們正是古人用以形成、保存與表述宇宙知識的最初形式。

試舉一例：古埃及人將月亮描繪為貓，並非出於無知，以為月亮真是貓；也非幻想，硬要找出兩者的相似；更非創造口頭隱喻，如我們用「貓鬚」形容優越。他們絕非刻意設計謎題，以困擾後世的埃及學家。

事實簡單得多：他們觀察到貓能在黑暗中視物，其瞳孔在夜間擴張、煥發幽光。天上的月亮是黑夜的先知，他們便以地上的貓作為對應的象徵。再者，古埃及「貓」字寫作「Mau」，此詞亦指「先知」，源自動詞「mau」，意即「看見」。由此推演，太陽在夜間沉入地底亦可稱為貓，因其仍能在黑暗中洞察。而貓所象徵的月亮，本身又可象徵為太陽之眼，因為月亮確如一面明鏡般的眼眸，反射著太陽光輝。

若不借助象徵主義這門科學，任何神話、古文字、聖書或秘典，皆無法被正確而完整地理解。尤須注意，每個符號都須從多重面向審視。不僅每個民族有其獨特的表述方式，隨著哲學與宗教體系自然演變，描述它們的符號亦會隨之嬗變。

簡言之，每個符號、寓言、神話或語素文字，皆如一顆多稜面的寶石。每一面不僅能衍生多種詮釋，更與多門學科相連。換言之，任何特定符號或文字，皆可從天文、生理、道德、數字、歷史等角度解讀，這僅是眾多可能組合的一部分；而這些組合本身，亦能以不同方式審視。

由此可見，象徵主義科學絕非易事。有人耽於幻想，以為存在一種「密碼」——如丹·布朗等作家虛構的情節——竟妄想幾週或數月便能「學會」此道，實屬謬誤。正因如此，對任何符號、神話或文本的含義，若妄下定論，斷言已窮盡其全部真理，是危險的。至多，我們只能說窺見了真理的某些側面。那些聲稱獨攬全部真理、並指摘他人皆謬誤者，不過落入了迂腐的陷阱。此言謹記，恕不重述。

## 數字學

這亦能解釋，為何埃及與希臘秘儀學派的學徒，耗費多年光陰，僅為掌握數字學的基本原理。然而今日，數字學卻常被一知半解者貶為廉價的「算命」把戲。畢達哥拉斯的門徒，曾費時七年，方精通此神秘科學的分支！因此，下次你若遇見某書或某網站，聲稱能在數週甚至更短時間內教會你命理學，便可斷定其作者對此一無所知。

數字一至九與零本身蘊含的深意，便足以寫滿百部書冊。若將這些數字透過加法、除法、乘法相互組合，衍生的可能含義更會呈指數增長。我們尚未論及畢達哥拉斯學派所謂「偶數」、「奇數」、「偶奇數」、「完全數」、「不完全數」與「超完全數」的概念，以及它們各自的特性與排列方式。

儘管在其他文章裡曾提及三、四、六、七等數字，亦僅是淺談輒止──因其詮釋向來紛繁。若斷言某數字在所有情境下皆指向相同事實、法則或原理，便是嚴重謬誤。

## 符號與標誌

符號與標誌存在根本區別。符號描繪的是單一而具體的觀念、事實、法則或原理。譬如圓形象徵「無限」，因其圓心無所定，圓周無所不在；因而它既是神凝練的象徵，亦同時指涉空間與時間。在此我們看見，單一符號至少能引發三種正確的解讀。這正印證先前所言：任何符號皆可從多方審視。每一種觀點本身俱為真實，卻都不完整。故而片面詮釋甚是危險，易將誠心的求道者引入歧途。下次若聽聞或讀到對某特定符號的解讀，而闡釋者僅提供單一說法時，你大可置之不理：因其並不明白自己所言之物。

標誌所涵蓋的概念較符號更為廣闊，正如那著名卻遭誤解的神秘圖紋「卐」字所展現。此標誌曾被希特勒及其狂熱追隨者為政治目的而篡用，此事與我們無涉。對能讀懂其隱喻者而言，其中蘊藏的信息，幾乎超越一切其他神秘主義標誌。

標誌乃是某些原理，或一系列原理的綜合圖像或記號。必須以寓言方式觀之、解之，或以全景視野逐層展露其意念序列，如上所述。此刻你當明白，《聖經》與其他聖典不過是以文字書寫的標誌；若不解其象徵何物，便無從窺見其中隱藏的深意。
