# 數字「七」的神秘含義
## 為何「七」被視為至聖之數？我們將揭示其多重屬性。

![cover](/img/downloaded/02d2b14b13888230d2f14b9a52a2946b.jpg)

Photo by [Waldemar Brandt](https://unsplash.com/@waldemarbrandt67w?utm_source=medium&utm_medium=referral) on [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral)

畢達哥拉斯學派稱七為「未婚的處女」：它不像偶數（如二或四）由母親所生，也不似奇數（如三或五）源於父親；它直接來自萬物之父的頂點——數字一，亦稱單子。

印度的《往世書》、埃及的《死者之書》、波斯的《阿維斯陀》、瑪雅的《波普爾武》，乃至最後的《聖經》，皆頻繁提及數字七。《往世書》記載七位聖人，象徵人類七大根種族、七個上下層世界、七座聖城、七個聖島、七座聖山、七棵聖樹，不一而足。

迦勒底-巴比倫神話有「蒼穹七神」、「大地七神」與「七惡鬼」。《創世紀》幾乎每頁可見七的蹤影，摩西諸書亦然；《約伯記》、《塔木德》與《卡巴拉》中，此數同樣顯著。

《啟示錄》載有七座金燈台、七顆星、七印、致七教會之七封信、七天使所吹七號角、七聲雷鳴、七位國王，以及傾自七碗的七災。《次經》則以隱晦筆觸提及人的七種感官：「他們領受並運用主的五種運作；第六處，主賜予理解力；第七處，言語，乃其思想之詮釋。」（傳道書 17：5）此為聖經唯一論及七種感官之處，後文將再探討。

希伯來人如此輕易採納、頻繁援引此數，顯示他們借鑒了鄰族教義，一如基督徒從猶太教汲取諸多信條。於是可見七項聖禮、小亞細亞七教會、七宗罪、七美德（四元德與三超德）等等。至於諾亞所見彩虹，世人多解為天上神蹟，以誌人神之約；然對於受過訓練的神秘主義者眼中，其意更深——它與前述人類七原則、七根種族、七個上層世界，以及七個宇宙顯現期緊密相繫。畢達哥拉斯亦據此「七」數，構築其關於七神聖行星的「和聲與音樂」學說。

需補充說明：古人採納七行星學說，並非因為不知其他星體，而是此七行星乃七邏各斯原始而初始之宮位。縱使未來天文學家發現九十顆新行星，亦不會改變此七行星神聖之事實。科學在此領域實屬茫然，一如其他許多領域。有些天文學家主張實際上有九大行星（含冥王星），亦有學者認為若計入2005年發現之鬩神星，則為十顆。此冰封天體大小近冥王星，軌道卻在其外。另有觀星者視冥王星過小，不足以稱「適格」行星，如此一來，若不計鬩神星，則復歸九星，甚至八星之數。且讓科學家自行解決此爭議，這不屬本討論範疇。

畢達哥拉斯學派稱「七」為希臘神祇「泰勒斯福羅斯」——因萬物經此數方得圓滿結局。關於七重神聖層面的秘傳教義，源自數千年前的亞特蘭蒂斯，畢達哥拉斯正是從其埃及導師處習得此道。這門秘傳教義闡明：萬物皆受制於七種力量——地上與月下的性質、宇宙的七大力量，並以七重音調演進。這既包含音階的七個音符，也體現為光譜與稜鏡折射出的七色虹彩。

有趣的是，雙虹出現時，第二道光譜的顏色序列與第一道正好相反。這印證了一條秘傳法則：塵世狀態實為上下層世界的倒影，正如鏡中映象必然顛倒。

畢達哥拉斯學派將人生劃為十個七年階段：
初七年，乳牙萌發。
次七年，男女具生殖之力。
第三個七年，軀體臻於成熟。
第四個七年，體能達至巔峰。
第五個七年，適婚之時。
第六個七年，心智漸次豐盈。
第七個七年，理性圓熟。
第八個七年，二者皆達完滿。
第九個七年，激情轉趨平和。
第十個七年，理想生涯終結。

人體亦處處見「七」：軀幹顯分七部——頭、胸、腹、雙腿、雙臂；內藏七器——胃、肝、心、肺、脾與雙腎。頭部為主宰，開七竅通外——雙目、雙耳、雙鼻、一口。人身天生七種排泄：淚、涕、涎、精、糞、尿、汗。

現代醫學早已證實古說：胎兒七月便可存活。月經週期平均二十八日，恰合四個七天之數，與太陰曆同步。人腸全長二十八英尺，亦為四乘七。

畢達哥拉斯學派更稱「七」為生命載體，因其涵蓋身體與靈魂。他們解釋：人由四個低等元素與三個高等元素構成，以頂端三角疊加下方四方形為象徵。關於這七重原理的正確序列，可參閱[《金黃星》](/books/golden-star-dr-michaud)第七章。

[《金黃星》](/books/golden-star-dr-michaud)第十一章中，更載有對應的埃及稱謂。但須提醒：這些埃及名詞包含了障眼法，意在遮蔽真相，防免遭誤解濫用。印度聖典將人稱為「七葉植物」（Saptaparna），足以證明印度和埃及聖人皆知人身七重構造；此知識皆承襲自亞特蘭提斯先祖。

布拉瓦茨基夫人首度向西方引介吠檀多架構，將三個高等元素構成的三角命名為：阿特曼（Atma）、菩提（Buddhi，勿與釋迦牟尼佛混淆）與心智（Manas），亦即神聖之靈、靈性之魂、與人類之魂。死亡之際，正是這高等三元組與低等四元組分離。低等四元組則包含：慾體（Kama rupa）、星光體（Linga sarira）、生命能量（Prana）與肉體（Sthula sarira）。

儘管我們現有感官為五——嗅、味、視、觸、聽，神秘科學指出，未來尚有二感官將會開發，終成七數。此二新增感官屬更高層次的心理覺知與靈性洞見，然其普遍展現尚需歷經億萬年歲月，此時臆測實屬無益。

據說這七種感知對應七重秘傳元素，而今僅五者顯化：以太、火、風、水、土，分別呼應聽、視、觸、味、嗅。事實上，第五元素「以太」仍在顯現地進程中；當代廣播、電視、X光線、資訊科技等發展，不過是以太初步展現。有些神秘主義者推測，那第六與第七元素，或即時間與空間，對應於前述的高階感知及靈性理解。

眾作者中，唯一近乎揭開此謎的，唯米紹博士。欲深究者，可閱[《李文和的教誨》第四章](/books/chapter-four-fortune-teller)。但須警告：你很可能最終結論是，這些奧秘已超乎理解範疇——此判斷極為正確。正如李文和所言：「若人類能以肉身之形、塵世之智，真正洞悉時間與空間之本質……則人將與『至一神』、『至一生命』、『萬物神聖本源』平等……僅憑此知識，其智慧便已遠超天界主宰——亦即諸造物主。」

「七」之數不僅存於人身，更貫穿整個自然。化學元素週期表中，「七」即居主導。若將元素按原子量分組排列（現代表格亦大抵如是），可見其構成以「七」為基的序列——故稱「週期表」，意指排列之「週期」或群組。

週期表中，每組的第一、第二個成員與下一組相應成員，在屬性上皆緊密呼應。由此可見，就化學變化及其內在本質而言，實繫於數值關係；這體現於其週期律正受「七」之數所支配。

這個數字在聲與色的世界裡同樣重要。自然的七個音符、光譜的七種色彩，都如此昭示著。

以上已足以說明，為何七被視為最卓越的神聖數字。正如前文所述，其重要性並非源於《聖經》。古希臘詩人赫西俄德早言：「第七日是神聖之日」，比摩西的安息日概念更早。七亦不囿於一地一國：羅馬太陽神米特拉的祭壇前，長燃著七簇經年不滅的聖火；阿拉伯傳說有七柄神聖扇子；印度典籍載有七座半島、七個島嶼、七片海洋與山川河流；迦勒底人信奉七重世界與七靈；赫西俄德與荷馬筆下常現七個星座；至於我們曾談及無窮盡的「七」，更是不勝枚舉。

倘若以同樣方法剖析七之前的數字，乃至十以內諸數，你必將發現更多深邃意涵。
