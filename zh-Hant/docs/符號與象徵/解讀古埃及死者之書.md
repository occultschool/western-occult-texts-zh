# 解讀古埃及《死者之書》
## 心臟代表死者一生的行為總和，將置於「神聖正義」的天平上稱量。

![封面](/img/downloaded/0de99e48427d7a9302931b6a464f1467.jpeg)

阿尼紙莎草紙 — *心臟稱量* — 紙莎草紙蛋彩，約西元前1500年。

## 介紹

古埃及人與許多民族一樣，相信人死後，靈魂（或稱「高我」）將受審，並依判決歸於三種命運之一：升入天堂、墜落地獄，或在塵世重生。此教義的變體流傳至今，仍為基督教會保留，只是信者寥寥。埃及原文中，審判過程猶如一場冷酷的稱量，以「神聖正義」為天平，衡量死者一生的善惡。這些善惡由他的心臟象徵，天平另一端的砝碼，則是瑪特女神（Maat）的羽毛——代表正義與真理。主持稱量儀式的是托特神（Thoth，亦稱塔胡提），一旁協助的則是烏帕圖（Up-Uatu，亡者嚮導）及其犬首猿身的化身哈皮（Hapi），皆見於上方主圖。此圖摘自阿尼紙莎草紙；阿尼是埃及文士，活躍於第十八王朝後期。

紙莎草紙畫面中，除了阿尼本人與高踞其上的「四十二位評判者」（即死者法官），還有一位關鍵角色：阿米特（Ammit），那「吞食死者之人」——Am-mit 一詞本意即「食死者」。這是世間最奇詭的怪物，鱷魚頭、獅子前肢、河馬後肢。這般組合絕非隨意拼湊，每一部分皆是重要符號，整體構成一道玄奧的徽記。對受過神聖象徵學訓練的人而言，處處藏著隱晦的深意。阿米特將於後文詳述。同時值得指出，上文所述的人物、審判過程及一切埃及儀式，實為一系列相互勾連的符號，共同構成一個龐大的標誌體系；最初，只向那些在古埃及秘儀學校受啟蒙者揭示。

## 死者的審判

據維基百科所述，心臟稱量的儀式在「雙真理之殿」舉行，卻未解釋何以有「雙」真理，或真理所指為何。這怪不得網路百科的編輯或埃及學家，因為他們皆不知人有雙重心智、雙重自我，且死後兩者皆須受「審判」。因此，人有兩份「真理」或兩本「帳簿」待查，亦即兩個自我——高我與小我的思想與行為。若你尚未猜出，事實上「法官」正是神聖靈魂本身，在埃及儀式中由托特或塔胡提代表。我們在此稍作停頓，細看上方圖片（下方已放大局部），便能認出審判的主要參與者。下圖僅有兩位不在視野之內：托特與阿米特；他們立於上圖天平的右側。現在，且將目光聚焦畫面中央，看它訴說些什麼。

## 心臟

最左側可見兩尊女性神像，依據上方的象形文字，可辨識為女神梅斯赫內特（Meskhenet，亦作 Mesket 或 Meshkent）與雷內特（Renenet）。實際上，梅斯赫內特在此場景中出現了兩次——天平左側那個奇異的、頂著女人頭顱的方塊亦象徵著她，正位於文士阿尼的上方。這兩種形貌有何深意？梅斯赫內特是埃及神話中的神聖助產士、生育女神，因此她現身於審判現場合情合理，因這裡關乎靈魂（高我）在來世的重生或復活；但她的意義不止於此。身為「卡」（Ka，靈魂）的創造者，她與孩童的未來命運相連，因她在嬰兒誕生一刻，將靈魂吹入其軀體。在此圖中，她化為古埃及婦女生產時所蹲的分娩磚，象徵阿尼將於下一個人世獲得新投生，或在埃及死後世界以靈體復活。由此可見，確有兩次「誕生」須考量：一次在塵世，一次在天界。

千年後，聖保羅重述此理，寫道：「死人復活也是如此：所種的是朽壞的，復活的是不朽的；所種的是羞辱的，復活的是榮耀的；所種的是軟弱的，復活的是強壯的；所種的是血氣的身體，復活的是靈性的身體。既有血氣的身體，也必有靈性的身體。」（《哥林多前書》15：42–44）他在《致哥林多書信》中又言：「頭一人出於地，屬土；第二人出於天，屬天。」（哥林多前書15：47）此處顯然指兩重出生、兩重身體、兩重自我——高我誕於靈性之體，小我誕於血肉之軀，雖以隱晦之言述之。梅斯赫內特不單是助產婦，更能定人命途。因此她常與司掌壽數的薩伊神相繫，二者時而與雷內特同現；正如雷內特亦在「雙真理之殿」的儀戲中扮演要角。

雷內特形象多變，時為女子，時為眼鏡蛇，或作頭戴雙羽冠、頂著日盤的蛇首女身。此圖中，她身分不僅由草寫象形文字標明，更見於額上蜿蜒的蛇形曲線。其名涵義透露關鍵線索：「雷內」意為哺育、照看，「雷內特」卻指運氣乃至財富。故兩位女神分掌「命運」與「財富」——前者可視為前世寫定的命途，後者則是隨之而來的福報；前提是，我們的行止總能招引祥瑞。

有趣的是，雷內特亦被視為大地之神蓋布之妻。對常人而言，「好運」無非屬地的恩賜，而猶太人稱此大地之神為耶和華，希臘人與諾斯替者則謂之造物主。這非太陽神拉所賜，因拉給予追隨者的，是屬靈的祝福，非屬世的厚利。雷內特亦傳為蛇神奈赫布卡之母。此巨蛇鎮守冥界入口，每夜護佑拉神穿越幽域。這位守護者的象徵及其在埃及末世論中的角色，深意綿延。若細論之，恐離題太遠，唯有留予讀者自行探尋。然由此可知，埃及眾神皆有千面；若將拉、奧西里斯或伊希斯簡化為單一意象，便犯了大錯。何況數千年間，神祇觀念流轉不息。若不牢記此點，莫說複雜如「雙真理之殿」的審判場景，即便最單純的埃及符號，其深意也難掌握。

梅斯赫內特與雷內特上方，也可見阿尼的靈魂以人首鷹身之姿棲立——此即埃及所謂的「巴」。這昭示阿尼乃高階靈魂，攜塵世歷練之精粹進入死後世界。他立足一間小屋，象徵「平和之域」的居所，位於埃及人稱為「奉獻平原」的上層天界。阿尼的巴與梅斯赫內特之間，是先前提及的分娩磚，而阿尼的心臟正懸於秤桿一端。如前所述，心臟凝聚死者一生行止之總和，將在「神聖正義」的天平上稱量。

若心臟輕於或等於瑪特之羽，死者生前德行具足，便可前往埃及天堂。反之，若心臟沉重，便遭怪物阿米特吞食，永困埃及冥府。凡人所知僅止於此；縱是當今學者，所知亦未多於此。唯少數如E·A·沃利斯·巴奇這般啟蒙的埃及學者，幾近觸及寓言隱義。而且只有進入神聖秘儀的人，才知道其中隱藏的偉大真理。

## 阿米特

前文曾提及，我們將進一步闡述「食死者」阿米特的象徵。諸位或仍記得這頭怪物：鱷首、獅前肢、河馬後足。那麼，誰會成為被牠「吞食」的「死者」呢？乃是那些主動切斷與高我連結、甘願沉溺於黑暗與邪惡之人。此類靈魂恰是阿米特最合宜的食糧，亦是依其形象所塑造的造物。如阿米特一般，他們殘暴如鱷、耽溺權力如獅、深陷物慾泥淖如河馬。由此觀之，阿米特無非是我們的老相識——「小我」的象徵。阿米特吞噬自身，意味著死者生前的人格將隨肉體湮滅，因其無任何值得在死後世界存續之價值。這亦解答了人格於死後是否延續之謎：端視塵世生活中，此人性格有多少傾向高我。正如《星光對話》中，神秘的M告知兩位對話者：「靈性的『我』不朽；而你們現今的性格，僅有值得不朽的部分得以存留。換言之，死亡鐮刀割下的花朵，只餘香氣縈繞。」

由此可斷言，智慧的埃及啟蒙者——至少在阿尼所處時代——早已透徹掌握我們於[神秘學課程](/docs/twelve-lessons-in-esotericism)中探討的輪迴與演化法則。於他們而言，阿米特並不可怖，因牠僅是法則的具體象徵：將正直者自重生之輪解脫，將不義者綑縛於另一番轉世。我們見證獲釋的阿尼立於烏帕圖神面前——此神後世常與阿努比斯混淆。烏帕圖，其名直譯為「開路者」或「引導者」，專司引領正直者的靈魂（高我）步入彼世應得之位。

值得注意的是，阿尼紙莎草繪卷中的諸般場景，並非見於每一部流傳的《死者之書》。有些抄本略去梅斯赫內特與雷內特；另些則以瑪特女神取代天平上的托特猿。此類差異，不應如「專家」所言，悉數歸咎於抄寫員「訛誤」。我們堅信，如同阿尼卷軸所示，某些細節是刻意納入，旨在向持具特定鑰匙之人闡明真理。

此主題既無暇深論，亦無篇幅贅述。暫不探討《死者之書》本身，惟望日後系列文章專述之。茲可一提：該書基本文本可溯至早期王朝，自第五王朝始，見於薩卡拉金字塔地下墓牆與石棺刻文。此等最古文字斷代已逾五千載，然我們確信其淵源必更久遠。最初，它們為啟蒙預備者而設，使其熟稔神秘科學的法則與原理，為將來探索星光界高低維度鋪路。故而，這些文本亦提供一幅「地圖」，描繪探險者將遭遇的種種境況與靈體，無論生前透過所謂「星光體投射」，抑或死後進入星光界。

之後，這些獨立篇章編纂成連續敘事，即今稱《死者之書》者；然埃及人自稱其為「通往白晝之章節」，白晝即「光明」。啟蒙者亡故時，其專屬「書卷」隨葬墓中。此俗後來擴及貴族階層，不論是否啟蒙。至第十九王朝以降晚期，凡於國中居要職者，縱使抄本殘缺訛誤，亦多以全書或部分篇章陪葬。然此已無關緊要，因時人視此書為聖物，具「神異」之力，可護佑死者——無論善惡——安然抵達天堂。迷信終究壓倒智慧，自此，《死者之書》之用途，幾與天主教會神聖遺物無異，目的與手段皆相仿。但瓊·格蘭特在《羽翼法老》中，鮮活重現了埃及第一王朝的風貌：彼時智慧仍駐於統治者心中，其廟堂仍庇護真理微光。此事容後篇再續。

## 無罪宣告

下文將論及死者於「四十二判官」面前的「無罪宣告」。內容取自瓊·格蘭特之著作，她憶述前世於古埃及的生活——當時身為第一王朝法老梅里奈特。此法老確有其人，載於巴勒莫石碑之法老王表，名為傑特，埃及學界亦稱瓦吉、澤特或瓦吉。須指明，此處四十二判官所詰問與死者之答辭，與阿尼紙莎草紀錄有所出入；蓋因第一王朝至阿尼所屬第十八王朝，其間相距已四千餘載，難免嬗變。格蘭特於書首有段簡介，蘊含重要真理，我等認為值得全文引錄。其文曰：

![插圖](/img/downloaded/d78efc2078cf258baa5cb47339881c57.jpeg)

「旅人走至終點，見一條河。河邊泊著一舟，是那必渡的「時間之舟」。然獲准登船前，他須先喚出船槳之名，否則槳不為他划動；亦須喚出船頭之名，否則船首不為他引航。登舟之後，便在黑暗水面上行去，直至河水墜入「大洞穴」。惡魔環伺，可怖形影襲來；若能鎮住恐懼，形影便瑟縮退回陰影之中。隨後他於一處碼頭登岸，七級石階通向一扇巨門。他必須喚出門閂與鉸鏈之名；更須知曉門板如何製成的秘辛。說出名號後，門便在他眼前敞開。穿過廳堂，四十二位亡者判官端坐於寶座之上。

「他們飛越他頂上，沒入陰影，其貌不可得見——眾神如群山環峙，他如在深谷之中。

「每位判官皆將詰問。若他不能以真理回應，並道：『憑真理之羽，我已勝你』，腳下地磚便會裂開；他將永困黑暗，直至自另一母腹重生。而在應對詰問之時，美德固然存在於心，邪惡亦以沛然之力湧入。

「判官列坐廳堂四圍。

「第一位詰問：你可曾明智體貼地照料己身，一如造物主在你年少時那般愛護你？

「第二位說：神所分派的年歲，你可曾活盡？

「第三位說：你是否守身如玉，未染污穢之河？

「第四位說：你只與靈犀相通之女同房嗎？

「第五位說：你對母親、女兒、姊妹、姑姨的身體，可曾毫無認識？

「第六位說：可有男子在你面前如女子般行事？

「第七位說：有何動物曾喚你為夫？

「第八位說：你的手，可曾取那不當取之物？

「第九位說：你可曾食至腹疼哀鳴，或飲至神智淪為肉體的奴隸？

「第十位說：你可曾動用暴力，割斷任何人的銀線？

「第十一位說：你的怒氣可屬公義？你手中的連枷，可如『法老之連枷』？

「第十二位說：見富足能幹之人，你可曾不生羨慕？

「第十三位說：你的心可曾被妒忌的利爪撕裂？

「第十四位說：除了譴責『惡』本身外，你可曾口出惡言？

「第十五位說：當種子待播，你可曾讓犁閒置於壟溝？

「第十六位說：你可曾貪求那不合你耳目的知識？

「第十七位說：你可曾望著牆上巨影，便自認形貌魁偉？

「第十八位說：途中遇險，你可曾偏離正道？

「第十九位說：你可曾以黃金鐐銬，將自己鎖在塵世？

「第二十位說：你可曾凝望塵事，直至雙目失明？

「第二十一位說：市集交易，你可正直無欺？

「第二十二位說：旅途之中，你可曾感謝所有相遇——無論同伴，或是為你解渴的石榴？

「第二十三位說：你可曾將麵包分予窮人，或將葡萄園的果實給予疲乏者？

「第二十四位說：你可曾絕不說謊？

「第二十五位說：你可曾自恃聰明，反而蒙蔽了智慧？

「第二十六位說：你的友誼，可如流沙荒漠中的磐石？

「第二十七位說：你可曾以仇恨之鎖，將自己與他人相繫？

「第二十八位說：你是否不識巫術、不玷污己身，只視此身為唯一居所？

「第二十九位說：你是否令母親心滿意足，令父親引以為榮？

「第三十位說：你可尊崇所有真正的祭司？

「第三十一位說：旅程之中，你可曾時刻銘記眾神，並求問其指引？

「第三十二位說：智慧高聲言語時，你可曾充耳不聞？

「第三十三位說：你可曾以智慧，止息求道者的饑渴？

「第三十四位說：你的力量，是否只用於『光明』之事？

「第三十五位說：你曾是『荷魯斯大軍』中的一柄利劍嗎？

「第三十六位說：你可曾引人走向不自由之路？

「第三十七位說：你心中可有自我榮耀的幻象？

「第三十八位說：你是否識得己心，是自身一切所為的真實記錄者？

「第三十九位說：你可知曉，一段旅程的終結，只是另一段的開始？

「第四十位說：你可記得那些曾為兄弟的植物，為它們解渴、照料，使之繁茂？

「第四十一位說：對待一切曾是兄弟的動物，你可曾如大師待你那般，以智慧、仁慈與憐憫相待？

「第四十二位說：你是否能誠實言道：『我從未使人或牲畜超負荷勞作；我知世上眾人皆是我的旅伴，並在途中施以援手』？

「到了那時，他將聽不見眾神洪亮的聲音——只有他自己的聲音在寂靜中響起，說：『我已征服你，因我此生所行，未曾造成罪惡、悲傷與苦痛。』

「隨即，真理之殿便如正午般，被他靈魂的純淨火焰照亮。即使世間所有的風都聚向這焰，它仍靜靜燃燒，不為所動。眾神不再居於高處，因他的靈已與祂們齊平；對他說來，眾神的臉容猶如鏡中映見的自己，祂們與他已是兄弟。

「眼前出現了塔胡提的大天秤。一端承著他的心，另一端置著『瑪特之羽』。二者平衡無差，只因兩邊皆是真理。

「城牆如門敞開。他邁步向前，走進天界的光明之中，那裡有七肘高的玉米，等著他前去收割。」

## 總結

一些讀者對這些「無罪宣告」中關於同性戀、通姦與貪食的譴責深感不適。更多人則對其中嚴格的道德要求無法接受，更遑論達成。然而，確實有少數人——包括本文作者——能夠對每一個問題回答「不曾」。這些都非重點。我們研究、撰寫、發表此文，並非為了冒犯、羞辱或審判誰，而是為了闡明：這些問答將在每個人死後的審判中浮現。即使我們自認此生何等邪惡、犯過多少過錯，總有新的一生可以補償。更何況，從未失足、未犯一罪的人，其實不是人，而是機器——沒有自由的意志，血管裡也流不動鮮紅的血。

正如開篇所言，我們才是自己的判官。離世之時，甚至在某些瀕死之際，我們不得不回顧剛結束的一生；這經驗與世間任何評判皆不可比擬。因為此時無法偽裝，更無法說謊。我們——那剛剛逝去的過往人格——被徹底剝除所有遮掩，赤裸裸地站在判官面前。而這判官，正是我們自己的靈魂，儘管鮮少有人意識到這一點。基督徒會視之為耶穌，甚或全能上帝本身；穆斯林看作穆罕默德或偉大的伊斯蘭天使；印度教徒可認為是任何一位神祇、女神或聖人。簡言之，所有人都將依據自身的宗教框架或無信仰，看見屬於他們的判官。他們做過的善行會與惡行一同衡量；同樣地，他們未盡的善事、與雖受誘惑卻抵抗了的惡念，也將被放在天秤上。

「神聖正義之光」將尋見並照亮一個人過往生活中的每一行為、每一念頭。無一遺漏，無論曾經如何隱藏或壓抑。但天秤終究傾向慈悲——因為「神聖正義」本是慈悲的，它深知人多麼容易誤入歧途，多麼容易陷入錯誤與幻覺。睿智的埃及聖賢明瞭這一切，將其記於《死者之書》等篇章中，使後人得以從中獲益，知曉肉體死後等候著的是什麼。對多數人而言，那不是死亡，而是覺醒；他將獲得比人世更宏偉、更豐盛的生命。這是一個被遺忘的真理，而我們有幸能在此提醒讀者。
