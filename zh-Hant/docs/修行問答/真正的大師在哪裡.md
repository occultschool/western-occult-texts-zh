# 真正的大師在哪裡？
## 當學生準備好了，大師就會出現。

![cover](/img/downloaded/8aeb68dc10df86bc2c2455193b88c669.jpeg)

許多追隨「新紀元」教義的求道者相信，即使沒有老師指引，也能在神秘學路上前行。

這話有幾分道理，但並非全對，也非人人適用。確有可能獨自修行，在神秘學領域取得長足進展甚至開悟。但這類學生總會遭遇瓶頸；若無真師引領，便難再進一步。

凡深入研究過釋迦牟尼、畢達哥拉斯、柏拉圖等古聖先賢教義的讀者，便知我們所言不虛。細究這些偉大靈魂的生平，便知他們各有師承。追本溯源，每位真正的導師，其知識若非源於光明，便是得自黑暗。前者導向解放與啟蒙，後者引向奴役與無知。唯獨經此開悟之師，方能助學生在學習與冥想中明辨是非，識破那些夾雜虛實、最為危險的半真半假之說。

許多讀者應聽過這句古老箴言：「當學生準備好了，大師就會出現。」

此言與人類歷史一樣古老，可見於老子的教誨、《奧義書》乃至《聖經》中皆有迴響。然其真意何在？何謂「大師」？如何辨認？「準備好」又意味什麼？事實是，我們既要準備好「受教」，也要準備好「捨棄舊學」——這兩者未必同步。太多神秘學愛好者聲稱渴求真理，一旦審視其內心，卻無一例外發現，他們只想印證自己的幻想與揣測。猶如2012年，一批受惑的信徒，期盼世界立刻神奇地轉變成靈性天堂。

正因如此，求道者若不願放下成見、臆測與謬見，便無緣擁抱真理。他們尚未「準備好」，或者說，還不配遇見真師。心智閉鎖，便無法領受真理與智慧那樸素的模樣。愛德華·鮑沃爾-利頓在其19世紀名作《札諾尼》中，精準刻畫了這類求道者的樣態：書中人物懷抱不純的慾望，又為神秘學裡關於力量與知識的錯誤觀念所困。

當學生開始關注內心狀態，而非俗世性格時，才算是做好準備，無論是迎接真師，抑或真諦。[《金黃星》](/books/golden-star-dr-michaud)中有言：

「無知之後，是理性的曙光；理智之後，是真正覺悟的黎明，亦即開悟；最後臻於智慧。這五步將人從野蠻提升至靈性。然而其間尚須經歷諸多階段，如恐懼、懷疑、迷信、教條、狂熱；未經奮鬥與克服，無以見光。」

那光即是大師，大師即是那光，指的是永恆真理的真正導師。對未準備好的人，光明如同黑暗，大師到來亦無幫助，因其欠缺高等意識去認識與理解。

何謂「準備好」以迎接大師？布拉瓦茨基的話可作總結：「生活純淨，思想開放，心地澄明，理智熱切，靈覺甦醒。胸懷同胞情誼，樂於接納與給予指引，對師尊懷有忠誠的責任感。這些人深信師者掌握真理，並願順服真理的要求；能勇敢承受被加諸的不公，勇於申明道德準則，勇於捍衛受不義侵害之人，且心懷信念，確信人類能持續進步與完善。此即吾師所示範的黃金階梯；真誠的求道者拾級而上，終能步入神聖的智慧殿堂。」
