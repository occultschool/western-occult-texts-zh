# 講堂問答（3）：黑魔法、知識的保密
## 探討巫術、威卡與神秘學的分野，揭穿黑白魔法及左道的真相與風險，警示儀式與道具之謹慎。

## 一、威卡、巫術和神秘學有何區別？

簡而言之，區別極微——三者皆屬操弄大自然隱藏的力量、以達善惡目的之秘術。然這與神秘「科學」之別，猶如地窖燭光與太陽的距離。涉獵各個秘術以廣見聞，本無大害，但須謹記：切勿輕易嘗試書本或網路所見之咒語儀式，否則身心俱損，其害真實不虛。

巫術不過是魔法的別稱，詳見神祕學課程[《魔法》](/docs/magic-occultism-mysticism-religion-history)一文。若行咒作法意在害人，無論如何粉飾，皆是黑魔法。換言之，魔法之善惡，全繫於施術者意圖。此理甚明，稍具思辨者皆能了然，與是否為巫師無關。關於黑魔法，下文將進一步闡述。

尚需補充：咒語、沙鹽、蠟燭、火焰、魔杖等現代巫術用具，大抵只是戲劇道具，用以聚焦心神。器物本身並無魔力——雖有極少數例外：任何物件，若經賦予神秘力量，就能用於善惡。若知道如何使用，確能生效。

然而西方真通此道者，屈指可數。故所謂具「神效」之魔杖、儀式刀、護身符等物，盡可視為虛張聲勢或浮誇廣告。真正的開悟者無需、亦不屑使用這些道具。他們僅憑鍛鍊有素的意志、專注與想像，便足以實現無知者口中的「奇跡」，無待外物。此理見於《新約》，耶穌即為神秘科學之開悟者。

或有巫師自稱能與「祖先」及靈體溝通，此法危機四伏。以此途徑接觸的靈體，很少比你我更睿智或高尚。多半只是誤導人，而有的是全然邪惡。因此，《聖經》與所有真正的靈性導師皆譴責此類交流。欲接觸高等存在，唯一正道是將思想提升至其境界，而非強拽他們墮落至我們的層次。

## 二、何謂左道與黑魔法？

即為私欲濫用任何知識與力量。  
黑魔法遍行於世，那些追逐權勢、財富、名望、復仇的無能愚者，總妄想憑藉幾道「符咒」或儀式便得償所願。如今，黑魔法常假「左道」之名，宣稱為通往「神格」的有效道路。此說荒謬至極。

左道絕非求道者或欲脫輪迴者所應涉足，它只會導致更深重的塵世束縛，導致靈魂死滅（我們所謂的高我）——耶穌、佛陀及一切真師皆如是說。踏上歧路者，大多只收獲失望、迷惑與抑鬱，終致身心俱病，詳見[《金黃星》](/books/golden-star-dr-michaud)所述。

魔法本是雙刃，一念之惡、一己之私，便足以令「白」轉「黑」。

追求物質生活的改善，本無過錯；財富若能澤被他人，便是福祉；聲望亦能化作善的影響力，惠及值得之人。但這一切，無需倚仗「魔法」，只需決心、勤勉、主動與耐性便能達成；而醉心黑魔法者，恰恰欠缺這些。

時人對左道、黑魔法與巫術的沉迷，終無善果。即便真有術士能與邪靈交易，換得名利，代價也必隨之而來。宇宙間沒有免費的餐食；世上一切，皆有價碼。

那些曾助黑魔法師的「靈體」，待術士身亡，便來索取全數報酬——這正是《浮士德》傳說背後的秘義；故事雖有虛構，根基卻是真實的法則。

若欠債尤重（我們親身所知確有其事），或需歷經多世方能償還。其間苦難，恐難想像。這類黑魔法師的終局，記載於[《魯魯的尋求》](/books/chapter-six-wizard)與[《金黃星》](/books/golden-star-dr-michaud)異象一之中。

科學亦可能成為黑魔法。科學家發明毀生傷世之法，縱然對所作之惡渾然不覺，也終須為自己的念頭與行為付出代價。如前所述，萬物皆有價。今生未償，來世必還——這法則對科學家與黑魔法師一體適用。

目前所知最卑劣的黑魔法師，能夠奪人命軀，但無法毀滅生命本身。他所帶來的苦痛與死亡，與頂尖科學家無異。至此，科學家與黑魔法師竟無分別：二者皆能毀壞人身，但都無法創造生命……這或許值得你沉思！但這並非俗謂之因果報應，只是種種因緣聚集的結果。

你也許辯稱，你無意專職為惡，不過想略知黑魔法以廣見聞，或藉此消閒。若僅止於此，我們無意阻止你沉溺；但若你想學黑魔法，我們的文章裡並無門徑。

或者，你只求自保，免受黑魔法與施術者所害？若是如此，最好與之斷絕一切牽連。我們還須說：你若充滿神性之愛與光，世上世外皆無力量能傷你我分毫。

若你只為擴充知識，了解黑魔法並無害處；但我們力勸你：無論多麼心動，切勿嘗試書中網上所載任何咒語、儀式或召喚。一旦施行，你可能召來黑暗之力，身陷險境；或是引來心術不正之徒，以你最不願見的方式，侵擾你的生活與高我。

此刻你或許自認強悍自信，深信能把持得住；但那些存在的陷阱與詭計危險至極，且充滿惡意的狡詐——往往在你察覺之前，早已落入網中。

## 三、為何如此多秘傳知識皆被隱藏？

簡言之，是為防止濫用。  
深一層看，知識即力量；而力量常遭誤用與濫用。物質力量尚且如此，何況催眠、暗示這類秘術背後的隱知？落在錯的人手中，將遺禍無窮。

正因如此，從畢達哥拉斯到耶穌、釋迦牟尼，每位秘傳科學家都只向公眾揭示被允許揭示的部分，真諦僅傳予選定的弟子。這般審慎，遠勝當代科學家的作風；科學家毫無顧忌地將知識交給任何想藉此牟利之人，並以科學「中立」為由，對其後果不負責任。

顯然，現代科學洩密的報酬是金錢、榮譽與自滿；若非如此，機械化戰爭、原子彈、生化製劑，以及那足以令人無聲癲狂的無形電磁武器——種種悲劇，或可倖免。

將秘傳知識隱藏，尚有其他緣由。最重要的是，真理若只揭露片段，易使求學者心生困惑。部分永非整體；未見全貌，便難明部分與整體之關聯。例如許多秘術修行者、異教徒與「新紀元」追隨者所普信的「業報」觀念；今日世人對「業力」的理解，早已淪為不分皂白的臆測與誤解，弊多於利。

另一緣由在於：二手知識無論多麼崇高、來源多麼神聖，終不能取代親身體證。

歷來最偉大的靈性導師，也無法令弟子悟出超過自身意識水平的內容。若將每一處奧秘都向求道者點破，非但無益，反令其無從領會——縱使導師耐性以言語闡釋，試圖拆解玄理的真義，終是徒然；未曾迷失過的人，怎能懂得真理的珍貴？

話雖如此，有些神秘教導確實被無端隱匿。但此種情形正在轉變，我們刊載的文章便是這轉變的一例。[《金黃星》](/books/golden-star-dr-michaud)中寫道：「縱使核心真諦須在塵世中深藏，裹以隱喻，埋入幽微之境；那些簡明的法則卻不該隱蔽，否則將對人類造成無可估量的傷害——眾人在黑暗中摸索，懷著至善之心，卻犯下最駭人的謬誤。」

誠心的求道者，必會贊同這番睿智之言。而文中所指那些「簡明法則」，正於本系列文章中逐步揭露。

這些玄秘之學實不該對世人隱瞞。它們被掩蓋數百年，恰證明古人同樣易陷謬誤，亦難免為私利所困。或許他們當年確有苦衷：在那個時代，宗教教條仍主宰多數人心，敢於宣講此類「異端」學說者，往往面臨迫害與死亡。

幸而，我們活在一個較為「開明」的歲月。但願這份《講堂問答》與本站其他文字，能助人睜開眼睛。
