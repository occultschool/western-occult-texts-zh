# 給求道者的建議
## 最真誠的建議獻給踏上這條道路的求道者

![cover](/img/downloaded/11a2c48c2973513b05aa236a33d603c9.png)

智慧是我們在一切困厄中最好的朋友，它甚至比「英勇」更重要——英勇最終會自取滅亡。神聖的洞察力引導聖人行止，令他前進時毫無遲疑。但真正的聖人不求名，也不圖財；高我超越了所有人類的理解，即使最偉大的作家也寫不盡它的榮耀。然而，一個仔細聆聽的心，能在內在的寂靜中聽見它的聲音。

就算有強大的智力，若無同樣充沛的精力，也是徒然。正確運用智力時，所耗的精力比體力勞動更甚。精力不足，智者便無法正當發揮其靈性力量。另一種人，大腦僵固如石，永遠將智慧的光芒拒於門外，沒有一個念頭能穿過物質構成的厚重大門。所有想法都受嚴密審查，阻擋任何高等思想進入耳中。就算此人空有力量與精力，卻沒有智力，與前面那種有智無力者同樣無能。每個問題都有兩面，有時甚至兩面皆為負面。

黎明的太陽射出熾熱的長矛，刺醒沉睡的世界；獲得真智慧的人亦如太陽，散發開悟的火焰光芒，喚醒他人昏睡的心智。這就是諸神有時派遣光之子下凡的原因。他們建起真正的光明殿堂，凡符合美德者皆可進入。這些建築立於不可動搖的磐石上，因為根基是神之意志所奠立。而那些非神所建的屋舍，根基則是錯的。

智者不願再經歷死亡重生，因而努力求解脫，即自我救贖，脫離輪迴轉世。為此，嚴格的考驗與試煉都是值得的。通過神聖的觀照，不懈的求道者終能止息心思，連「想要止息」的念頭也一併止息。因為任何嘗試努力都是爭鬥，有爭鬥便無平和……平和只在停止努力之後才會到來。停止努力之後是寂滅，而後真正的靈性方能誕生。最終，智者發出清澈響亮的神聖話語，真正的神秘視野開啟，他不再將短暫塵世放在心上。此時，開悟者真正受啟，融入了自己的神性之中。

「服務他人」是通往開悟的道路，人與人之間的服務從來不徒勞。這會產生無數美好的後果，如一縷飄向天際的崇高精粹。它所帶來的好處永不會抹滅，且將成倍增長、傳播，最終迎來意想不到的幸福。服務形式多樣，詩人或畫家用他們的方式服務人類，真正的老師或聖人則以戒律的方式進行。

相反，一個惡行就像一首走調的歌，破壞了幸福的諧和。

人之所以屈服於仇恨與其他惡習，是因為小我仍滯留於過去生活的不利情境中，從而以破壞性的方式影響高我。高我必須學會自我控制，從而克服這些缺陷。在小我的祕密深處，潛藏著對物質事物的種族記憶，寄居於繼承的本能裡。這本能也是物質性的，只是形式略異。因此，只要不傷害眾生，將來便能從塵世的痛苦與枷鎖中解脫。這適用於念頭與行為。陷於仇恨者終被仇恨征服。嫉妒者必將失去那令他嫉妒之物，即使那物從不曾屬於他。生活在罪中的人終被罪毀滅。思想黑暗邪惡者將居於黑暗，處境比他自己想像的更可怕。

獲得財富不需要智慧，只需足夠精明加上命運的眷顧。但有智慧的人才知道如何正確掌管財富。財富唯有在用於減輕人類痛苦時才有價值，否則便是無用，是詛咒而非祝福。千萬別追求財富，也別追求權力。心智一旦被權力蒙蔽，便會轉向非法的目標。人的六大缺點是：欲望、憤怒、貪婪、迷惑、驕傲、嫉妒。巨額財富或繁盛的事業，若不用來造福他人，它會像閃電一樣，瞬間閃爍後消失於未知之處，無人受益。自私的願望同樣徒勞。

當一個人追求靈性時，即使美德微小如種子，也會成長為大樹，結出豐美的果實。但當一個人轉向物質，沉淪於錯誤與黑暗，昔日的美德也會被黑暗領域的汙水腐蝕，終成不幸。因為一個人的邪惡行為，也會在他內心結果，將他摧毀。

人對於「我」的認識也是如此，好比飛蛾的夢，沒有實質，隨著虛無縹緲的思緒飄散。然而人的心中，事實上有著永恆本體的胚種。這胚種唯有通過正確的行為與思維，通過永恆的信心，並堅信「人是眾神之子，終能成為神」，方能發展。

但在這之前，求道者必須澈底辨明：物質世界的自我是幻夢，有別於靈性自我的實相。而獲得這種覺知，極為困難。

除了「服務他人」，另有一條從不失效的康莊大道：真愛之路。這不是感官之愛，而是靈性之愛。儘管在極少數情況下，前者可以引發後者。

真正懂得「愛」的人不屬於這塵世，因為他們的靈魂盛滿美妙的甘露，從天界的容器流向他們乾渴的心。愛在某些時刻能戰勝最強大的人，就像苦行僧維薩米特拉（Vishvâmitra）看見梅納卡（Menâka）時，放棄了他的苦修。或像亞亞提（Yayâti），因愛薩米什塔（Sârmishta）而變得比原本更蒼老。儘管他們的愛起初由感官引起，但無人能斷言這樣的愛最終能否成為靈性的。唯有充滿智慧的生命之主知曉。

通往榮耀的下一條道路是通過學習，通過真正的大師與高人的指導，他們的職責是將智慧傳授給符合資格的人。因為若將教導傳給不適格者，不會帶來美德或榮耀，只會導致遺憾。

然而，無論你走哪條路，或是否遍歷諸徑，要成功還需要一個要件：「勇氣」。沮喪毫無益處，災難不會觸及堅定的人，彷彿災難也畏懼他們的堅定。因此，無論生活發生什麼，都要保持決心。在這世上，對於有進取心的人，沒有什麼是不可能的。對於性格勇敢的人，勇氣本身就是成功的原因。因為勇敢的人能憑自己締造繁榮，無論其志向何指，或者有沒有人支持——只需在生活的千般試煉中始終堅定，終將抵達成功。

永遠切記，巨大的悲傷往往以更大的幸福作結。因為悲傷是眾神的鑰匙，用以叩開我們心智的門扉，使人望見靈性更高處的光景——那方是真正的福樂。而凡塵諸事中，時常蟄伏著不幸的蟲蠹。
