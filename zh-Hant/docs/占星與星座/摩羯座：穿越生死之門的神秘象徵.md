# 摩羯座：穿越生死之門的神秘象徵
## 從古代神話到星象學，摩羯座如何象徵靈魂的死亡與重生

![cover](/img/downloaded/7c6397929b51770d7f421f3457460801.png)

摩羯座在古人的想像中，是一頭半山羊半魚的複合生物。這形象源於太陽每日的循環：每到黃昏，太陽便沉入西方，猶如剪斷光芒，沒入黑暗深淵或無垠夜海；待到黎明，又從東方重生，帶來嶄新的一天。山羊與魚的結合，正暗喻了太陽旅程中的黑夜與白晝。因此，某些神秘主義者將摩羯座詮釋為「穿越並超越死亡的生命」。

在古代哲學裡，巨蟹座與摩羯座被視為「人類之門」與「神靈之門」。巨蟹座象徵靈魂進入肉身的入口；其對面的摩羯座，則是靈魂離開塵世的出口。古人認為，世人所謂的死亡實為覺醒，彷彿從夢中醒來，回歸熟悉的原初狀態。相反，靈魂投入凡胎肉體，反倒成了一種禁錮、遺忘或沉睡，甚至有人視之為另一種死亡。如此，肉體的消亡並非終結，而是新生的開端，恰如太陽夜行之後，必然迎來破曉。

埃及人相信，塵世生涯結束後，靈魂將前往阿門提（Amenta）居留，之後更可進入天堂（若其靈魂歸屬那裡）。他們有時稱人類為蓋布之子女、奧西里斯之子女、或拉之子女，正因這三位神祇分別掌管塵世、冥界與天堂。

當那一刻降臨，願這葉小舟的旅者能找到穿越深淵的路徑。願他們在愛與智慧的活水中重獲新生，於真理的清流間航行，遠離謬誤的暗礁。願他們復歸完整，安然渡過上下兩界間的狹窄海峽；願他們航向安穩的故鄉，避開慾望的穢水——據說那裡有吞噬者潛伏等待。

因為摩羯座吟唱著一首父親之子歌，超越小我，服務其主。他們歌詠超脫塵世的境界，歌詠光明與火焰的重生。他們歌詠辛勤耕耘後的公正收穫，歌詠嶄新而更美好的生活，那生活已看破死亡的幻相。他們與昨日、明日、今日的同伴攜手前行，唱出與自然諧和的旋律。這歌聲告訴我們：尋找自己的位置，明晰生命的目的。它唱盡舊時代的終結，也唱響新紀元的開端。

「我們追尋的已非個人成就，而是靈性層面的意志。生活的目的從此改變。我們不再為一己之私奮鬥，而是為自然的整體福祉努力……」

摩羯座與印度教的摩伽羅（Makara）淵源深厚。摩伽羅被稱為深淵之龍，受土星統治——土星別號「懲戒者」。對某些人而言，它象徵肉體的死亡與解體，同時也代表靈性的誕生。摩伽羅常與鳩摩羅相聯，據信他們參與諸多神秘的啟蒙儀式。它亦被喚作「有鱗者」，其形象常被雕鑄在門檻與大門上，作為建築的守護神。

在巴比倫，神祇埃亞（Ea）或恩基（Enki）被尊為「海洋的羚羊」，身披魚皮斗篷，自深水中現身教導人類。其標誌正是山羊與魚，許多人因此將摩羯座的「山羊—魚」意象溯源於他。傳說他居於清水深淵，與七位聖者為伴；其追隨者相信，這深淵可延伸至塵世之外。他為荒蕪之地帶來生機，令井泉湧出甘冽淨水。數世紀以來，所謂的「魚-神」的龍與奧涅斯常被視為同一神話存在，儘管近代學者對此存疑。

山羊頭的埃及神祇努姆（Num），據說是「深淵的主宰」，曾在某一時期掌管大地的洪水。努姆的形象承襲自更古老的神祇努（Nnu）。在某些層面上，他被視為聖經中諾亞的原型。

「努是洪水與方舟之神，在尼羅河洪水的宰控中，其角色延續為努姆。他的方舟（teba）被視為底比斯城的象徵，即哈特謝普蘇特女王所稱的『塵世上的天堂』。」

願所有夥伴在俗世風暴之外，覓得通往方舟的安全路徑，安然登入其中。讓舟身承載他們，一如宮城裡的王子、聖殿中的祭司。願神聖寧靜的光，在方舟高牆內永恆照耀；願擋路者，自覺轉身離去。

希臘人將這星座稱為「山羊角」，並與牧神潘 (Pan) 相連——潘是自然之神，也是潘神管的發明者。傳說大地曾派遣巨怪提豐追殺眾神。眾神逃至埃及，紛紛化為異獸藏身。潘潛入尼羅河，將下半身變為魚尾。另有一說：眾神初敗於提豐之後，潘與赫爾墨斯助宙斯療傷。宙斯為表感激，將潘送上星穹，成為今日所見的摩羯座。

此星座中亮星包括：摩羯座α (Algedi)，源於阿拉伯語 al‑jady，意為「山羊」或「孩童」，標示右角。另有數星，象徵阿拉伯民族祭祀山羊的儀式——每當太陽自摩羯座升起，他們便舉行此祭。其中包括摩羯座β (Al Sa’d al Dhabih)，名為「屠宰者中的幸運者」。而摩羯座θ (Dorsum)、γ (Nashira) 與δ (Deneb Algedi) 三星，則標記著魚尾。

中國天文學家以十二行省之名，為摩羯座中十二星命名。另有一組星宿，構成他們所謂的「牛宿」，描繪了仲冬獻祭之景。又有三星，勾出一件玉飾輪廓。此天區另有一星座，稱作「羅堰」——在中國人眼中，這是灌溉天田的渠堰網絡，屬於天帝的星野。

晨光自星空深處浮現，白晝輝煌隨之降臨。空氣澄澈如水晶，鳥鳴綿延似聖詠，安恬如嬰孩偎於母懷。草甸上的花，將芬芳投進晨風；萬物齊鳴，融成一片美聲。金色山脈之下，朝聖者已獲新生，此時踏上一條活躍之路——以愛與大自然為藝術，前往他的應許之地。正在那刻、在那歡欣之處，他聽見孩童笑著歌吟。

塵世之人，將生命的遺產虛擲於物欲的賭局。低劣的渴望如朦朧魅影，纏繞這些背離者，無形卻可感，在他們耳畔細語甜言，引他們遠離光明。唉，他們在淚與悔恨中熬過幽暗長夜，只為領受應得的報應。

你們是我深愛的孩子，須知我的天使必護佑你們；且將金銀寶石，懷擁於心。正如王子每日自寢榻起身，醒於宮殿燦華；正如僕人甦醒，分享主人成功的喜悅——你們也將從這世界被提升，去往更美的境地。

因我是超越墳墓的永恆生命；我是神秘太陽的靈；我是邪咒的破除者，解開死亡束縛之人；我是領人歸家的愛，上升路途的光；我是長夜的主宰，帶來白晝的晨曦。
