# 天鵝的神話與象徵：從古代神話到宇宙的啟示
## 探索天鵝在神話、宗教與占星學中的象徵意義 —— 從阿波羅的優美歌聲到宇宙的起源傳說

![cover](/img/downloaded/fcac2ccab1a7bf9415253835d019c761.jpeg)

「……天鵝平日也唱歌，但預知死期將至，反會以最甜美的歌聲，歡欣地唱給神聽。天鵝本是神的僕役。凡人懼死，這才誤解了天鵝，以為牠臨終悲鳴，是為自己哀悼……天鵝既然屬於阿波羅，便有預見異象的本事；牠們預知彼岸有福，所以死前歡唱。」

卡里馬科斯在《德洛斯讚歌》中記載，阿波羅誕生時，鳴唱的天鵝曾繞德洛斯島盤旋七匝：

「……雪白的天鵝身為阿波羅的美妙歌者，自帕克托盧斯的金色海岸啟程，繞島飛了七圈，歌聲溫煦如流轉的日光；柔美的旋律深得九位繆斯鍾愛，以絕妙音色迎接新生。阿波羅是和聲之神：七弦琴之主。隨後他賜下金豎琴。在第八場溫柔合奏開始之前，他降臨人世…」

天鵝是候鳥，在秋天如太陽般南飛，春日又如備受期盼的太陽歸來。在《帕西法爾》故事裡，羅恩格林被一些人視為太陽的象徵，乘坐著白色的月天鵝之舟。凱爾特神話中，天鵝少女著名的變形能力，對神秘主義者而言，隱約暗指德魯伊祭司學院裡的神秘轉化。德魯伊祭司儀式所穿的斗篷，便以天鵝皮羽織成。

北半球的民間傳說與神話，常將天鵝與轉變、蛻變相連。此觀念流傳甚廣，歷久不衰，就連醜小鴨化為美天鵝的故事，亦可追溯至這古老的思想寶庫。

中世紀煉金術士曾隱晦提及黑烏鴉變為白天鵝，同時暗示這天鵝體現了對立面的結合或連結。宙斯與萊達的結合亦復如是；宙斯化身天鵝誘引了少女。此番結合誕下了雙子座的孿生子，以及與之相對應的女性。有人留意到，天鵝的形貌融合了男女特質，既似產卵的鳥，亦如使之受孕的蛇。

在埃及，鵝被描繪為產下太陽金卵的巨禽；牠也是偉大母親或偉大雌鵝的典型，被視為人類的起源。聖母哈索爾是慈愛的典範，純潔無瑕，曾以神聖母牛之形，以乳汁哺育榮耀者。然而數世紀後，其形象竟演變為「維納斯—阿佛洛狄忒」的淫祀，連同其聖鳥白天鵝，也一併成了情慾與感官愉悅的象徵。

印度與埃及共有相似的觀念，認為宇宙源於絕對者光線所生的「天鵝」，或稱萬古之鵝。據說此鵝產下宇宙蛋，萬物自此誕生。另有傳說提到，梵天乘坐七鵝牽引的戰車；天鵝亦是薩拉斯瓦蒂的坐騎——薩拉斯瓦蒂乃梵天之伴，是智慧、詩歌與音樂女神。在這片夜空區域的音樂象徵裡，她與流水息息相關。

「願水為吾母，滌淨吾身；願那以酥油淨化者，以酥油淨化吾。因眾女神承擔污穢，我自她們之中誕生，純潔而光明。」

古老體系中的天鵝，與純潔、靈感之水、歌聲及音樂緊密相繫。阿波羅身為音樂與詩歌的守護神，其戰車由白天鵝牽引，傳說他有時也會化為天鵝。其他傳統中，天鵝與繆斯女神關係密切；許多著名詩人與歌者據說死後皆化為天鵝。

奧菲斯死後化為天鵝，與他珍愛的里拉琴一同安息於天堂。名琴師阿里翁死後亦成天鵝，他曾為德爾菲努斯的海豚所救。荷馬是米蘭德的天鵝，維吉爾是曼圖亞的天鵝，而離我們時代最近的莎士比亞，有時被追隨者喚作艾芬的天鵝。

至於天鵝座，阿拉圖斯與托勒密皆未稱之為「天鵝」星座。對他們而言，這星座僅是一隻「鳥」。有些現代學者認為，牠對應遠古巴比倫的安祖鳥星座，儘管據說此星座早在約三千年前便已自天際隱去。

「天鵝座展翅翱翔，雙足輕曳，與安祖鳥的描繪極為相似，而安祖鳥座原先應當位於這片天域。」

埃拉托色尼與希吉努斯則將此星座與天鵝相連，並講述類似的故事：宙斯化身天鵝，追求能產卵的涅墨西斯。埃拉托色尼稱之為「天鵝」，拉丁文「天鵝座」便由此而來。

一如往常，天鵝座的淵源向來眾說紛紜。有人說是宙斯為求萊達，化身天鵝；亦傳法厄同駕日車墜落埃里達努斯河，這星座便是為他哀泣的故友或手足。阿拉伯觀星者視其為飛鷹，亦有人稱作母雞。因群星勾勒出十字輪廓，它亦被喚作南十字。

座中最亮者當屬天鵝座α，其名取自阿拉伯語「尾羽」，正是天鵝尾端那點銀芒。此星與天鷹座α、織女一相映，織就夏日大三角。天鵝座β美如珠玉，標記鳥喙；天鵝座δ與ε則素樸無名，悄然點綴雙翼。

在中國星圖中，天鵝座α偕眾星構成「天津」，恰是銀河一處津渡。天鵝、天琴兩座之間，星光迤邐形成「輦道」，彷彿直抵天宮的御路。另有「羲仲」之星，相傳乃發明戰車的古人。

哦，讓我們如曠野白鵠，航向龍口，靜候命定之時——屆時此岸與彼岸的帷幔將薄如晨霧。讓我們懷著內心的寧靜，吟唱重生的天鵝之歌，航行於銀河澄澈的水面，駛向藍天與青梧的麗土；在那兒與靈魂知己共享恩典與欣悅，慶賀安然歸返，遠離一切宵小之擾。

且乘這葉輕舟，憑純粹本心引領，載著聖潔之貨——聖盃、生命之水與光明之源——穿越長夜，迎向新創造的黎明。當旭日初昇，光芒遍灑，我們將聽見歡笑孩童的歌聲。

塵世之人總溺於物欲洪流；叛逆之徒則被月光下幽暗的激情吞噬。唯追隨我者，當辭世之刻來臨，將如天鵝般優雅振翅，飛往更美的彼岸。
