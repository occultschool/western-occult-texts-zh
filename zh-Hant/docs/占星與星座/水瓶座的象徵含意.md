# 水瓶座的象徵含意
## 巴比倫視水瓶座為「偉大者」，其形象關聯冬季的洪流與雨水。

![cover](/img/downloaded/71215c0ca3042d7c5aebcb80e1cd1514.png)

水瓶座在巴比倫稱為「偉大者」。傳說他手持花瓶，傾瀉出冬日的洪水與雨。這「偉大者」常塑作男子形象，雙股水流自瓶口湧出，緊密連結水神與智慧之神，亦即蘇美爾的恩基或阿卡德的「泉水之主」埃亞。然而，另有一說指「偉大者」曾名為古拉，是位女神，其名意謂「深淵之水」。

「古拉，即宇宙初始混沌的深淵水域；故與水瓶座恰有關聯。」

更泛論之，黎明時分，太陽據說正從黑夜的深水中浮現。

每日晨光，太陽自水中重生，許多人以此比喻人之誕生。

「有人將此描繪為逃離冥界的孩子，或棲於水中的新生太陽；若以人間術語解讀，猶如胎兒安居子宮的創造之水。」

埃及尼羅河每年氾濫，河水漫過密布的灌溉渠道，使土地重獲豐饒。尼羅河神哈皮的形象，正是手持雙罐，傾倒流水——這或許是水瓶座形象極為古老的源頭。

每年水的再現與重生，亦關聯一位尋找孩子的母親，其子迷失於黑暗深淵。埃及人相信，水的源頭是神聖奶媽梅納特(Menat)，位於水瓶座；而摩羯座的山羊努姆 (Num) 或卡努姆 (Khnum)，則被尊為「水之主」。

這水縱有不同名號與頭銜，埃及人卻比誰都清楚：那最初的深淵之水，蘊藏著宇宙創生的永遠隱藏起因。

荷魯斯在另一層面亦是水之子，他前來拯救瀕臨乾涸死亡的埃及。這救贖主題，在世界諸多古老傳統中，皆可尋得各式版本。

若是赤誠者，救贖便是眾神的恩賜。然而，這間接也是一種神聖介入，開闢通往更高、更善存在層面的道路。

這靈知的救贖根基於超越推理的洞見，亦即啟示與啟蒙；它揭示了一條「歸家」之路，通向「太陽之門」與「彼岸」。

在印度教的黃道十二宮中，水瓶座稱為水罐 (kumbha)。大壺節 (Kumbha Mela) 是印度教的朝聖盛事，信眾齊聚聖河沐浴，相信能滌淨罪愆。大壺節大抵十二年一輪，擇地舉行；每隔一百四十四年，則有摩訶壺節 (Maha Kumbha Mela)。

而今西方世界，水瓶座的代表人物則是希臘神話的伽尼美德 (Ganymede)。這美少年據說為宙斯所愛，攜至天界成為持杯者，為眾神斟上美酒與甘露。有說他原是特洛伊王子，亦有指他是牧羊人；關於他如何擔任此職，一如往常，眾說紛紜。

此星座最亮的星是水瓶座α (Sadalmelik)，名稱源於阿拉伯語「國王的幸運兒」(Al Sa’d al Malik)，標示右肩；水瓶座β 則標示左肩。「女宿一」光芒自左手發出，「泣二」標示臀部，「羽林軍二十六」則位於小腿。水瓶座多顆星名皆含幸運吉祥之意。

在中世紀伊斯蘭傳統中，基德爾能在人需要時施予超凡助力，為遇到他的人帶來好運。他在傳說中的性格和特徵，與《古蘭經》記載不盡相同：他有時是靈性導師，傳授神秘奧義；在《古蘭經》裡，他是神的僕人，摩西曾受其指引。其他記載中，他時而為撫慰者、安慰者與幫助者，時而療治人類過錯，時而是魔法師、拯救英雄之人，是邪靈之敵、詩人與戀人的靈感之源，是聖地的聖化者與守護者，更是神之友、先知等等。然而對於蘇菲派而言，他最鮮明的標誌，始終關乎那生命之水的追尋；在這條路上遇見他，便是至要且至幸之事。傳說他本是無形的臨在，卻不時顯現於「神之友」眼前，與他們直接交流。

不少蘇菲詩人將他與聖水緊密相繫，有時，他自身便似那美妙泉眼的延伸。也難怪他總與綠意、春日、海洋相連；諸多傳統、故事與傳說，皆隱約指向他在宇宙學上的深意。不過，既然基德爾是生命之水的發現者，他最重要的使命，終是指引他人尋獲此泉。關於他如何覓得，說法各異，但那水具有驚人神效，世人倒無異議。

「並非每顆寶石皆有那般光輝；人既可稱之為水，亦可稱之為日。
當基德爾認出這泉源時，他雙眼頓時被照亮。
他下車，迅速褪去衣衫，將頭與身軀浸入那純淨泉中。他盡情啜飲，以求配得永生。」

基德爾是否確為某一具體人物，抑或是一長鏈靈性導師的原型，甚或是求道者自我的投射擬現，歷來爭論不休。真正的持杯者是誰？那盈滿聖杯的閃耀液體又是何物？這奧秘始終縈繞不散。無人能向他人展現其奇蹟，唯有尋者能向自己揭示；但曾有一位作者，在目睹天界及其聖水時，描摹得比誰都更為接近：

「……世間之水不過影子，而愛我、信我之人，將與我同享天界活水。
須知，一切「如是」者，皆源自天上之井，是我父——人稱毗濕奴聖主——的流溢。
一切亦源自我母，即毗濕奴的天后德瓦基，在世間被稱為摩耶（少女），她乃至高神的配偶。
正如那水是萬物之始，不論存於塵世、諸天界，抑或諸神世界之間的空間；
我亦是那活水，是那神聖雙親最初的流溢。
因我是萬物的起始、中程與終結，無論生死；我是那活水，凡渴慕我者皆可自由取飲，無阻無礙。
我即永生之水，而水即是靈；於一切生命，我是其永恆本質；於一切似死非死之物，我亦是其本質。
因它們同樣充滿生命之水的靈，唯對俗眼隱蔽，故凡人若欲尋覓這初始之水的靈性本質，終將徒然。
它超越一切知識與概念，全然神聖，承繼父母所創之性。
它是一種本質、一種水、一種靈、一道金焰，至聖無比，見我者即見本質。
故我告你真相：水乃萬物中最先被創造，包含毗濕奴之火與摩耶之濕；凡以靈眼尋覓觀照者，必於其本質中見我，即克里希納——父與母之子。
凡居於其輝光中者，必得永福，因他們知曉，是我將火與濕融合，成為遍在之本質，在我人身與靈性之中。
我是宇宙之水的主，行於其波瀾上，吐露溫婉言語，如撫慰靈魂的油膏，使軟弱者得安慰——皆奉我父我母之名。」

啊，但願神所立的祭司能向尋求者親身顯現。願他們能同行，紮營於高原，傍依永生的清溪。願不落的星在頭頂柔柔發亮；願澄澈的水在心底注入甜潤樂音，喚起歡笑少年的歌：

世人在荒蕪沙漠度其一生；任性的叛徒，被無羈慾念的詛咒擊成聾啞。

但跟從我的人，必在平和園中、天堂泉邊安歇；他們將於天使歌吟裡，得享喜樂與滿足。

因我是那獻上言語之杯者；是注滿杯盞者；是泉畔取水者；是療治諸病的藥；是使一切圓滿者；是善人的命運；是分賜靈感活水者；是圓頂廳堂的導師；是灑水之人；是翠色極光；是紅顏黎明；是內在異象的賜予者；是永葆青春的老者；是盛納「如是」的壺；我是隱藏之神的兒子，我的時刻將至，無可拒卻。
