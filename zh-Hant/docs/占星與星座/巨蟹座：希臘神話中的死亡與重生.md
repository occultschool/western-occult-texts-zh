# 巨蟹座：希臘神話中的死亡與重生
## 探尋巨蟹神話與文化意涵，揭開生死纏繞與重生微光

![封面圖片](/img/downloaded/f70816a4bd66f406800e147899e042de.png)

那夜，一輪神秘的月亮將它王國的祕語傾瀉大地。月色圓滿、靜謐、高懸，幽邃難測。星光灑落的松林間，旋律流淌，仙女與林間精靈翩然起舞。這咒語般的調子，迴盪在地球隱蔽的洞窟與地窖，滲進牢房與地牢，飄過銀色宮殿與金色塔尖，最終棲息在千家萬戶、人心深處。

星座之中有兩道門戶，以巨蟹與摩羯為象徵，古人謂之「人類之門」與「神靈之門」。他們認為，巨蟹是靈魂踏入塵世血肉的起點，摩羯則是靈魂掙脫形骸桎梏、覓得自由的歸處。

傳說赫拉曾遣一隻螃蟹，在海克力斯與九頭蛇搏鬥時緊咬他的腳跟。長蛇座北端緊鄰巨蟹，據希臘神話所述，螃蟹與九頭蛇同在勒納湖棲身，亦稱阿爾西尼亞湖。此湖被形容為無底深淵，是致命的淵藪，相傳通往冥府。酒神狄俄尼索斯（或稱巴克斯）試圖自幽冥迎回母親時，便由此入口潛入。

「這傳說我也聽過。湖面看似波瀾不驚，實則任何試圖渡湖之人，皆會被無形之力拽入深淵，捲進湖水暗流。湖周不大，約莫希臘賽道三分之一長。岸邊蔓草叢生，蘆葦搖曳。每年紀念狄俄尼索斯的祕儀於夜間舉行，其細節恕我不能向世間透露。」

在諸多古老神話裡，螃蟹與烏龜常與死亡相繫，也象徵穿越生死邊界的使者。中國將此星座中一個星團（鬼宿星團）喚作「積屍氣」。埃及製作木乃伊的匠人，則常以聖甲蟲形石心取代逝者心臟，稱其為「石心」，並將此儀式與巨蟹座相聯。

然而，這些巨蟹座之歌僅有少數人能聆聽；它們吟唱的不是同伴墜落凡塵，亦非死亡的毀滅，而是遊子歸家的故事。歌聲訴說自我尋回正途的旅程；詠嘆那些曾經迷失流離、終獲平安的人，如何重回主的懷抱。妳是高低神秘的月，妳是名號眾多的母親，妳是滋養世間的子宮，妳是通往生與死的大門。請垂聽我等祈禱，引領所有同伴安然歸家。

他在小舟甲板上默然禱告，彷彿聽見——或自以為聽見——死亡與重生的吟詠漸次消逝，取而代之的，是一闕重生之歌緩緩升起。因為他與全體船員踏入此門時，皆是盲目受苦的孩童；但當他們從中走出，將如鳳凰浴火，自一切光輝中重生翱翔。

噢！你是奧西里斯之鳥，你是再生復活之鳥。在你無盡的歲月裡，雙翼飄散乳香、沒藥與硃砂的氣息；羽毛浸透聖火的芬芳。當燧石相擊，火花自沉睡甦醒；火焰瞬起，焚盡一切塵世纏縛。天狼星之火在鳳凰心中熾燃；鳳凰騰空，棲息於黎明旭日點亮的金枝之上。

同伴們，且記住：在星羅棋布的夜空裡，自某個轉折之處，出現了一位高飛的翱翔者或航行者。他是一股宏偉之力，眾星之王；他是羊群的牧人，牛群中的公牛，亦是平原上吼嘯的獅王。

「向你致敬，噢！你是萬王之王、萬主之主、諸侯之君！你自努特子宮統御人世與冥界。你的身軀流轉明亮的金屬光澤，頭頂蒼穹，綠松石光芒環繞周身。噢，安神，你已存在千萬年，貫穿萬物，在神聖之地——即那冥界——擁有至美面容。請賜我天界光輝、塵世力量與冥界勝利。允我如活靈般下行至塔圖，如鳳凰般上行至阿布圖，並許我自由出入冥界塔樓，無礙無阻。」

在古埃及人心中，神聖聖甲蟲象徵永恆與不朽。縱在死亡表象之下，那不斷更新、循環往復的生命，仍是埃及創世神凱佩拉的頌歌。其名含義為存在、生成、創造與流溢。

在埃及人看來，創世並非獨一無二、不再重現的事件，而是永無休止、循環調節的過程。黎明時是荷魯斯-太陽，正午化作拉-太陽，威嚴赫赫；西沉時又變成圖姆，遁入奧西里斯的冥界。在那裡，太陽經歷進一步的轉化，終將以勝利者的姿態再度顯現：他曾四散，而今復歸完整；他戰勝了黑夜，如同白晝般新生。

那曾被埋入土中的聖甲蟲，同樣重獲生命；它展翅，迎著晨光振翅再起，緊緊攫住太陽。

古昔的同伴相信，這便是善人的命運。死亡於他並不可怕；那不過是通往新生的大門。即便被黑暗圍困，看似毀滅，他的信念也毫不動搖。正如一位埃及學者所言，即便在那個時代：

「他期盼在永恆花園東隅的阿魯之地，享用奧西里斯的筵席。」

哦，願所有浪子都歸返父親的懷抱；願迷途的母羊安然回到羊圈；願離散的部族再度凝聚；讓我們點亮燈火、清掃屋宇，直至尋回失落的珍寶。當所有隱藏的財寶都納入救世主的箱篋；當遺失的銀幣重回寡婦的錢囊；當園門開啟，羊群歸欄：那時，所有同伴都將歡欣鼓舞，同聲高唱歸家的歌謠。

蟹與聖甲蟲緊緊抓住所有，如同大地環抱種子，母親懷抱嬰孩，父親擁抱回頭的兒子。巨蟹座便如此吟唱著星辰之歌。

哦，你這珠寶的收集者，這些珠寶光耀奪目，宛如神秘夜空中純然發光的星辰。請將它們珍藏於懷，令其復歸本位。哦，你如伊什塔爾般沉降，又在你所愛之人的懷抱中再度升起；你尋覓並尋見，從物質經驗的深井中，汲引天界之水。莫讓種子在黑夜裡無人照看而枯萎，讓它們自黑暗土壤中迸發，如同永恆之樹，在天空的永晝中閃耀。

希臘人稱此星座為 Karkinos，拉丁人譯作 Carcinus，即螃蟹，我們故稱其為巨蟹座。在巴比倫，它喚作 Mul Al-lul，有學者將此名追溯至一則遠古傳說，關乎英雄尼努爾塔；這位英雄的歷險，在在映照著海格力斯的功績。海格力斯曾戰多頭蛇，尼努爾塔亦然。希臘版本中，多頭蛇得螃蟹助陣；巴比倫版本裡，則換作烏龜襄助。

此外，據懷特的觀點，此星座的蘇美爾名稱在阿卡德語成為「螃蟹」（alluttu），其複數形式可指代夾鉗。據說在希伯來語、阿拉伯語與敘利亞語中，此星座的名稱象徵著持守、擁有與保全之意。

在印度人的二十七宿體系中，此星座內許多星辰居於「花」宿或「吉祥」宿之中，其主宰神是祭主仙人，眾神的祭司與導師。

鬼星團，這懸於巨蟹胸前的美麗星群，除了「腐爛屍體」的含義外，亦有「群眾」的含義。因其光芒朦朧如霧，古代觀星者稱之為「小雲」、「薄霧」、「多雲者」或「旋轉之雲」。世上某些地方的神秘主義者相信，這小微雲或薄霧能產生洗禮用的純淨露珠。鬼星團也曾被視作星群，因而得了「蜂巢」這個流傳甚廣的別名。

然而，希臘語中，「蜂巢」一詞亦指「馬槽」。而在鬼星團兩側，有兩顆星被希臘人喚作「驢子」，又稱北驢與南驢，阿拉伯語謂之「雙驢」。因此，早在基督降生之前，馬槽的形象便已存在於星空——位於兩驢之間，處於關乎誕生與重生的天域。有人將這兩頭驢與希伯來的兩位小天使相聯繫，或與埃及代表上下半球的兩位獅神相聯想，據說光明便誕生於這雙獅之間。

啊，願基督在每個人心中誕生，如同巨蟹座的天使們，歡唱那孩童的笑語。

儘管塵世居民無法直面，太陽的居民亦無由受益，但你們是我親愛的兒女，你們將得見阿卡爾的奧秘；因我將牽著你們每個人的手，與你們同循橢圓之路，前往永生之地。

因我是時代之眼；我是大門；我是金蛙與綠松石蝶；我是變革之力；我是透入洞穴的光；我是水面上的風；我是珍珠的造主與貝的開啟者；我是火的鳳凰神；我是自生之我；我是生命之主，我的時刻將至，無可拒絕。
