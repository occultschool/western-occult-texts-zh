# 雙魚座的神秘含意
## 我是偉大的魚，我的無數後代是閃爍的繁星

![cover](/img/downloaded/81e33d17192d33bd6619bcbc5f83c472.jpeg)

現今，雙魚座被視為春分的星座。我們總以為春分點仍落在白羊座，然而歲差推移了天體軌跡，過去兩千年間，春分時刻太陽所穿越的，其實是雙魚座。更早的兩千多年裡，春分點則駐守金牛座，沿著黃道十二宮的序列緩緩遷徙。

古人習慣將太陽與春分當值的星座相連，賦予象徵。於是，太陽的形象依序化為公牛、公羊、羔羊、魚，視哪個星座宣告春天來臨而定。微妙之處在於，即便歲差為太陽帶來「嶄新」的象徵，舊有的象徵卻遲遲不肯退場。正因為如此，基督教的「太陽—神」基督，其形象同時匯聚了白羊座的羔羊與雙魚座的魚。

真相是否盡然如此，不得而知。但希伯來人確曾將期盼的導師與救世主喚作「魚」（dag）。後有學者將這「魚」與美索不達米亞等地淵遠流長的「魚神」（Dagon）相繫。然而，如今更傾向認為，「魚神」之名或許並非源於游魚，而是出自意為「穀物」或「種子」的古詞根。

縱有疑竇，耶穌基督的形象與魚之間的千絲萬縷聯繫，大多可溯源至埃及。

「春分點已退入雙魚座；先前在白羊座時，荷魯斯或耶穌尚是『神的羔羊』，或在金牛座為小牛，後來卻被『埃及—諾斯替』的藝術家繪成了魚。」

多年以來，人們總愛將多個星座並置思索，提供更遼闊的視角，或自成體系。例如，黃道上遙遙相對的星座常被一同解讀。雙魚座的對宮是處女座，二者時而被視為一對。但雙魚座位居黃道之末，有時也與起點的白羊座相配。由此衍生一種信念：雙魚座既圓滿了始於處女座的歷程，也終結了發端於白羊座的循環。

類似的思路，也解釋了雙魚座自身顯而易見的雙重性。對某些人而言，兩尾魚可喻指人的兩部分；或指靈性與物質；或象徵航行的起點與歸宿。

雙魚座的拉丁名為Pisces，希臘人則稱之Ichthyes。阿拉圖斯為這組星宿，描繪了這般形象：

「這兩尾魚，一尾比一尾優雅，  
當它們初次浮現，北風也變得溫柔。  
它們的尾鰭向兩側舒展，如一條鎖鏈將彼此牽繫，在行進中交纏。  
其間鑲著一顆星，既美且亮，人們稱它為『繫尾之結』。」

星圖上，一尾魚游向天球北極，另一尾則西行。二者總以一縷繩索、絲帶或鎖鏈相連。從托勒密為雙魚座群星編目的方式來看，他認定兩魚由兩根繩索在中間打結繫縛。

古典神話流傳諸多故事，皆自稱為雙魚座形象的起源。其一講述阿佛洛狄忒與厄洛斯為躲避百首巨怪提豐的攻擊，倉皇逃逸。另一方面，埃拉托色尼則敘說：南魚座——兩尾小魚（即雙魚座）的母親——從湖中救起因跌落的女神德賽托；女神為表感激，將三魚一同送上星空。還有一則關於維納斯（或稱阿佛洛狄忒）誕生的版本，部分解釋了她與魚、鴿子的神聖關聯：

「據說一枚巨蛋落入幼發拉底河，魚將其推至岸邊。鴿子伏於蛋上溫孵，最終誕出了維納斯（或稱阿斯塔蒂，敘利亞的阿佛洛狄忒），後世尊為敘利亞女神。因她在正義與正直上超越眾人，宙斯賜予殊榮，將魚群置於星宿之間；故而敘利亞人既不食魚，也不食鴿，奉二者為神靈。」

敘利亞女神阿斯塔特，常與伊什塔爾等同。此神話形貌源起中東，其特質或映現了更古遠的星座方位。

雖起源未明，然有現代學者推斷，今日雙魚座，實由古巴比倫三座拼合而成：大燕子、魚繩、天女。後者涵括雙魚北魚及仙女座部分星辰，名為阿努尼特姆，乃阿卡德女神。她與阿斯塔特一般，亦被視作伊什塔爾或伊納娜的化身。燕子尾端星宿與飛馬座眾星，則共構西魚。因此之故，某些星圖所繪雙魚天域，並非兩魚相連，而是一魚以帶索繫於燕身。

「『阿努尼特姆』、『燕子』與『田野』諸星圖，共敘『敘利亞女神』之神話，亦可謂此季節的主要神話。」

「田野」可能是古代飛馬座四邊形的起源，丹達臘黃道圖示其位處雙魚之間。《死者之書》載有二魚引領拉神乘舟渡越冥界，恐亦此星座遙遠餘音。

魚於印度教洪水神話中，同樣擔當引導之責，祖先摩奴在此取代了諾亞。某些版本稱，魚（摩蹉）乃梵天化身之一。

「洪水滔天之際，波瀾之上空無一物，唯餘摩奴與七聖，及牽舟之魚。

歲月悠悠，魚不厭其煩引導船乘風破浪，終抵喜馬拉雅山巔。

魚輕輕的微笑，對聖人說：『趕緊將船繫於高崖。當知我乃萬物主、大造者梵天，力勝一切，無所不能。我化魚形救你於危厄。一切造物、諸神、阿修羅、世人，皆由魚摩奴所創；世間動植萬象，亦由他出。』」

另有一更古老的說法，視其為印度三相神之第二身——毗濕奴。

主星「雙魚座α」名出阿拉伯語「繩索」，標示繩結所在。有研究者認為這很可能即所謂伯利恆之星。它過去比較明亮，雖然在現代星表仍居α位，繩索中另有一星現已比它更亮。此星乃「雙魚座η」，其名來源雖不詳，有人稱之為「水之傾注點」。另有一星「雙魚座β」，意即「魚口」。

噢，願眾同伴皆抵達水之傾注點，於全然覺知中，在亞麻線上繫上光輝的結。願他們於智慧之水中調和這兩條魚。願他們舉手向天，俯身向地，事奉我主。在平和安穩中，願他們列隊迎候歡笑孩童之歌聲。

塵世之人浸溺於物質濁流，夢見隊伍的盡頭；叛逆者則漸次沒入深淵暗影。然而，我的孩子，你與我將共浴光中昇華，登臨天界之水的水晶表面；我們的水之旅程，將從深淵直抵天界高處。

因我乃偉大的魚，我無數後裔即閃爍群星；我是水域引導者；我引舟至安穩之地；我是懸於枝頭的魚；我是伊希斯之魚，我的時刻將臨，無可置疑。
