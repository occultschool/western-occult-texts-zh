# 天龍座：龍的象徵與神話
## 龍或蛇是最古老的象徵之一，世界上幾乎到處都能見到不同種類的龍

![cover](/img/downloaded/3ea3786c9020342b8ae7864cf6a6d7c4.jpeg)

你，是夜空裡那條繁星點綴的巨龍，盤踞在創造的頂點；永恆靜止，永不西沉，而少女們圍著你欣然起舞。我們該稱你為慾望之龍，是一切求道者心智的敵手嗎？還是稱你為生命之龍，萬物缺你不得？抑或你是智慧之龍，帶來白晝的甘泉？我們該把你畫得如夜一般墨黑，吞噬光芒，雙眼如炭火般冷酷而赤紅嗎？還是在萬物向你鞠躬的時刻，為你染上正午烈陽的輝黃？或者你是電光般的湛藍，像一道懾人的霹靂，疾速劃過紫羅蘭色的天穹，迅疾如思緒？

龍或蛇是最古老的象徵之一，古代神話裡，牠們的形態變幻萬千，遠比其他生物更富特色；至少在某些方面，龍與蛇常被視為等同。古代習俗也慣於將各類複合生物喚作龍。多數情況下，這些神話造物純粹只是象徵或標誌。

世界各地幾乎都見得到各類龍的身影：伊甸園裡的老蛇，《聖經》中飛天的火蛇；希臘與他處的守護龍，波斯的食人龍，威爾斯的紅龍，還有吠陀神話裡的「覆蓋者」巨龍弗慄多。奧菲克蛋周圍盤著龍；在巴比倫，一頭巨龍是眾怪之母；甚至如埃及所描繪，龍即是夜空本身，鱗片便是繁星。牠是中國的國徽，儘管中國的觀星者將牠畫在天秤座的天空。整個亞洲，複合生物摩伽羅有時也被視為龍。也有無數傳說講述龍被眾神、聖人、天使與英雄所殺。這樣的例子難以窮盡，各自皆有深意；此處只需留意，星座之中亦有幾處龍或蛇的蹤跡。

與此同時，舞動的星空下，船長往煙斗裡填滿菸草；填菸時，他想起了家，山谷外那片草地——清晨，那隻眼神溫柔的母鹿曾領著幼鹿前去。在他心眼中，牠們再次浮現，那優美身姿令人驚嘆；周遭，大地的銀色露水清澈純淨，正被朝陽從羈絆中釋放。

他就這樣在甲板上沉思，船飄浮於汪洋，憶起故鄉美景，愛意自心底湧起。但那湧起不像以往——不像杯中斟滿酒，不像聖酒灑上潔淨桌布，也不像愛的眼淚盈眶而出。不，它是猝然的，猶如巨龍剎那衝出海面，帶著燃燒的吐息，將混沌殘酷的黑夜一口吞盡。他想：你們尋求善的人啊，去尋求愛吧；你們追尋力量的人啊，去尋求愛吧；你們渴求智慧的人啊，去尋求愛吧！

太陽系最初的創造者，常被塑造為龍或蛇的模樣。牠象徵永恆與完整，自成一體，別無所缺。

「神的靈在混沌上盤旋，每個民族皆將其化作火蛇之形；祂在原初的水面吐納火與光，直至蛇身盤蜷成形，銜住自己的尾巴。」

在中國，禁錮於物質之中的火，有時亦被稱為龍。據說那是一道神聖的白熾，既為火亦為水，既陽剛亦陰柔：是日與月的擁抱，是陰與陽的交合，顯現於存在的每一層階。

古老的故事裡，有的龍長著七顆頭，有的三顆或十二顆，還有一百顆頭的。從某個角度看，太陽系確實「七呎深」，但龍能去往任何地方——上下、高低、左右、前後；牠環抱著整個世界與其中的一切。傳說牠守護山洞、聖園與極樂天堂。

另一方面，傳說中那些吞噬光明的暗黑之龍，會在每個黎明被強大的太陽殺死；正如乾旱之龍總在雨季來臨時殞命。然而，基督教神學家將牠塑造成叛逆、邪惡的天使，一個具象化的魔鬼，是神的對手。

無論何時何地，英雄試圖制服並戰勝惡龍的歷程，總被視為艱險重重，但成功總帶來豐厚回報。傳說巴哈姆國王獨自出獵時，曾被一頭野驢領到山洞，洞裡住著一頭吞食幼驢的惡龍。國王屠了龍，驢為表謝意，將他引向洞中埋藏的一批巨大寶藏。

「一位為野驢伸張正義的君王；一條被囚於墓穴的龍，最終，君王因他的義行得救，且獲珍寶。」

據說，若在龍的食物中摻入精心調製的蜜水混合物，或能稍減其凶性。極少數情形下，這種龍甚至可被騎乘。大英博物館藏有一枚古巴比倫的綠蛇紋石印，其上刻有神靈騎乘馴化之龍的景象。

「可見神靈跨坐怪獸背脊，一名崇拜者立於其後。前方為一公牛，吉爾伽美什正以匕首刺之。上方女神舒展雙臂，水流自臂膊瀉落地面。吉爾伽美什身後天際，亦有一瓶傾水，淌入塵土。」

自古龍分善惡，有上界之龍，亦有下界之龍。善龍有時象徵太陽，太陽又代表神聖之光與生命之火。例如，波菲利曾述及密特拉秘儀：

「……你們稱太陽（及其相關本性）為公牛、獅子、龍與鷹。」

白晝時，太陽代表此龍；夜晚則由月亮代表，或深居北極星空的天龍座星群代表。天龍座環繞極星，永不沉落（至少在北半球），恰似那象徵光明與生命的善龍，永遠守護神話中寶樹，植根於天界永恆之水、結滿金果。

此即所謂「崇高山巔之樹」。

「……乃天極一隅，審判之地，由天蛇——即天龍座——守衛。」

或有一些解釋者稱樹上金蘋果喻指星辰。然更具玄思之人，則視金果為智慧寶藏，由永恆警醒的龍所看守。

「於北溫帶的緯度地區，此星座永不沒入地平，故希臘人視之為永恆警惕之象徵，喻為園中守護珍果的龍。」

這座園子屬於赫斯帕里得斯三姊妹。關於極樂園、龍與果樹的希臘神話版本紛紜，以下精要亦不遜於任何一說：

赫斯帕里得斯三姊妹乃黃昏的「女神－仙女」，亦是落日金光。她們是黑夜妮克絲之女，或謂天界泰坦阿特拉斯之女。三姊妹受託照看金蘋果樹——此樹為大地蓋亞於婚禮當日贈予天后希拉。她們得到百首的守護龍德拉貢幫助。赫拉克勒斯受命摘取金蘋果，作為其十二功業之一；他屠龍後竊得珍果，雅典娜其後將蘋果歸還三姊妹。

赫斯帕里得斯亦保管眾神其他寶物。珀爾修斯誅殺美杜莎所需之神器，便是從她們獲得。

三仙女與其輝光流轉的金蘋果，被認為是日落金霞的來源，此景慶祝著天王宙斯與天后希拉之婚禮。

天龍座起源尚有他說：包括守護金羊毛之龍；或被卡德摩斯所殺之龍；或被密涅瓦（羅馬之雅典娜）所誅、擲於天際之龍。

早於希臘人，埃及人已將「邏各斯」表現為火龍。其大祭司有時被稱為「龍之子」，龍象徵神聖智慧之光與火；而古埃及一切偉大的開悟者與啟蒙者，皆與神聖智慧的守藏、隱蔽、有條件啟示緊密相繫。其最早形態之一，乃以鱷魚神索貝克為代表，該神於埃及人的日出日落觀中佔有要位。

在中國，天龍座部分星宿組成與天帝相關之星象，尤構成了紫宮——即紫禁城——的兩道垣牆。餘星代表諸般皇家侍從與皇帝內寢部分。而同觀沙漠蒼穹的阿拉伯人，所見非龍：他們將我們視為龍首的星群，看作一群駱駝，正護衛幼駝免遭鬣狗侵襲。

天龍座中值得注目的單星乃右樞（天龍座α）。此星今雖非星座最亮，卻被視為至關重要。公元前四千年至二千年間，右樞本是北極星。因歲差之故，極星今為勾陳一（北極星）。有研究者與埃及學者認為，大金字塔之方位特意對準右樞。此星名稱源於此星座的阿拉伯語稱謂。亦有人主張，巴比倫星表中極古老、被稱為「王冠」與「天庭法官」之星，所指正是右樞。

此星另有「拉斯塔班」之名，意為「龍之睛」。順帶一提，天龍座的其他星星如天棓四等，或者龍首龍尾相關之名，不宜與指稱月交點的術語混淆。

在平和的天界，於智慧之水中的聖丘之巔，生長著神聖知識的常青樹，具有永恆生命。太陽一旦西沉，終會再度登上神的山巔，為這濁世重沐光輝。星夜璀璨，同伴們繼續航行，尋覓那珍貴寶石。晨風將至，男孩的笑聲便不遠了。

凡耽溺低俗慾望者，必遭死亡之龍吞噬，永墜墳塚幽暗，又聾又啞；悖逆之徒行至半途，只得折返，因無法穿過烈焰氣息。

但那些追隨我、心口如一的人，必在深處聽見智慧迴響——那是我鮮活的話語。我將賜予他們智慧之愛，一路相隨；而我，作為愛的智慧，將安居在他心中。

因我是活水的靈，是光輝之言，是寶藏的守護者；  
我是神聖光明的龍，是神聖生命的龍，是神聖的愛之龍。  
我的時刻迫近，絕不落空。
