# 星空神秘學：天蠍座
## 天蠍座的起源：多種神話解釋

![cover](/img/downloaded/23c3f7dffa3a7c3836b3110542553a3e.jpeg)

在希臘神話裡，天蠍座殺死了偉大的英雄獵戶座。

一說獵戶座過於誇口，自認能戰勝地上一切野獸，才引來天蠍。有人指牠是大地的使者，另一些人則歸因於月神阿耳特彌斯。此後，獵戶座沉入地平，天蠍座隨之升起。

蘇美爾人稱其為「蠍子」(GIR-TAB)。於巴比倫，此星座與西方神域相繫：

「……冥界的蛇身守護者，引渡亡魂前行，直至他們在星群中的最終安息處。」

蠍人，巴比倫混沌怪獸半蠍人的一種形體，守護著太陽之門。在《吉爾伽美什史詩》中，蠍男與蠍女鎮守聖山，那是英雄追索永生必經之路。守衛放此英雄進入地下通道，只因認出他是「太陽-神」的寵兒。

埃及人相傳，太陽進入天蠍座之時，便是奧西里斯遭背叛、蘇特開始統治之刻。

「基爾切爾的某些希臘-埃及星圖裡，也將蘇特視為殺死奧西里斯的那隻蠍子。」

然而，當光明之神與黑暗巨蛇阿帕普搏鬥時，蠍子女神塞爾凱特卻站在光明一方。因而她據說能開闔喉嚨，給予或奪取生息。

波斯人同埃及人相似，視此時節為閹割公牛之時。中國人將天蠍座部分星宿劃入青龍；希伯來-基督教傳統則視蠍子為邪惡、背叛與衝突的象徵。

瑪雅祭司也將天秤與天蠍視為同一星座。他們似乎還把這片天域的星辰與死亡、重生相連。一份古老的瑪雅黃道手稿中，蠍子伏於祭壇之上，其下暗室內躺著一具人形。

「左側，一株綠植於夜空下開花生長，暗指瑪雅人的『綠樹太陽』(Yaxkin)。」

整幅圖景明示蠍子、墳墓、死亡與來世之間的關聯。此外，他們慶祝羽蛇神庫庫爾坎之死與升天，便在與天蠍座相關的「終結」(Xul)時期舉行。庫庫爾坎等同於《波波爾·烏》中的古庫馬茨，也相當於阿茲特克的羽蛇神奎薩爾科特爾，時而被描繪為鷹或美洲虎。

《吠陀經》中，死亡與重生同樣聯繫著天蠍座。此星座主要由兩大象徵代表：蠍子與老鷹。吠陀傳統有時描繪蠍子居於暗坑，坑中藏匿珍寶。據推算，印度教的迦梨時代約始於五千年前，當時天蠍座標記秋分點；每個與天蠍相關的月宮，皆象徵如此劇烈的變遷。

另一種與天蠍座相關的生物是巴斯里斯克蛇。傳說此物單憑氣息與目光便能致人死命，其毒劇烈，

「縱使騎士以長矛將其刺殺，毒素亦會沿矛桿上溯，不僅殺死騎士，亦奪其馬性命。」

一些中世紀煉金術士，依據三重偉大赫爾墨斯之權威宣稱，巴斯里斯克蛇的灰燼能點銀成金。

至於天蠍座的個別星體，主星是那顆超巨型紅星「心宿二」。其希臘名有時譯作「火星的對手」，儘管現代權威更傾向直譯為「火星」。無論何解，名字總歸指向它那抹紅色。

在古代美索不達米亞，與此星相關的名號有數個：烏爾巴特、種子之主、繁榮創造者、國王、英雄與國王，以及朱雀之星。在波斯，它被稱作薩特維斯（Satevis），乃四顆皇家星之一，被視為天界的守望者。其中文名之一為「火」，意為火之星。於印度，它與另兩顆恆星共組二十七宿中的「心宿」(Jyeshthā nakshara)。巴比倫人稱它「蠍子的乳房」(GABA GIR.TAB)；拉丁語中則名為「蠍子的心」(Cor Scorpii)。古埃及將心宿二奉為蠍子女神塞爾凱特的象徵；據巴吉所述，蠍子亦是伊希斯聖物。

天蠍座其餘星辰之名：天蠍座β (Acrab)，阿拉伯語意為蠍子，亦稱拉丁語之「爪子」(Graffias)。天蠍座δ (Dschubba)，源自阿拉伯語「額頭」；天蠍座λ (Shaula)，意為「刺」，懸於蠍尾末端。
