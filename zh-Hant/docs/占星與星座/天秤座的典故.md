# 天秤座的典故
## 平衡、法則、正義、判斷與辨識的星空歌詠

![cover](/img/downloaded/f8b7426b03f761654480e19d9b692d57.png)

在某些古文明裡，天秤座與太陽有關，奉為真理與正義之神。美索不達米亞人視其為沙瑪什（Shamash）的神聖符號，這位神祇掌管度量衡的準繩。埃及則將天秤形象簡化為軛或橫梁，呼應尼羅計——一種測量洪峰水位的器具。

如今許多人相信，構成天秤座的星域，古時原屬天蠍座的螯爪。更有人說，在更早的年代，處女座與天蠍座並非獨立星座，而是一體橫跨今日處女、天秤、天蠍三星座的大星座。據傳，待處女與天蠍分離，才在其間置入天秤座，將星座總數從十調整為十二。無論此說真假，天秤座的引入似乎意在標記秋分——這象徵再貼切不過，此時晝夜等長，不偏不倚。

考量天秤座的位置與歷史，它與兩側處女、天蠍的淵源仍未斷絕。一側是純潔的阿斯特拉亞，那位「星之少女」，希臘的純潔女神，身影常現於處女座。她與執掌正義、手持天秤的女神狄刻關係密切。另一側則是帶來死亡的蠍子，沙漠的住民。構成天秤座的星辰，古典天文學家時而稱之為天秤，時而喚作螯爪。至今，天秤座中若干星辰仍以蠍螯為名。居於兩星座之間，天秤座的位置自有深意。

「為免這場可怖爭鬥，永恆者在空中懸起他的黃金天秤，高掛於阿斯特拉亞與天蠍座之間；他先以此稱量萬物，令懸浮的塵世與蒼天保持平衡……」

阿拉伯人的宮位 Al Jubana 囊括了天秤座相關星辰，其名卻指蠍螯或其他爪具。印度教中，天秤座四星組成星宿維薩卡，意為分叉、具分支。其象徵被描繪為一座華麗的拱門或凱旋門。

在中國星野，這四星亦代表「氐」，此為第三宿之名，與青龍有關。氐字本義為根柢、基礎，猶如樹根。然而，氐雖常譯作「地」，與其互補原則「天」相對，詮釋這些術語時卻須格外謹慎，因其涵義往往隨語境流轉，別有層次。

其中一例便是所謂「崆峒的天界」。雖名為天界，實則深藏地下：

「文字流變中，崆峒一詞既指龐大地穴，亦指遠西高峰。這處神話之地，象徵重生、啟蒙與頓悟。相傳黃帝曾在此地領悟終極實在的啟示。」

縱然這些典故如今看來有些幽深難測，但追尋奧秘者當知：陰陽調和，於中國人而言何其要緊。不論顯現為月與日、塵世與天穹、底與巔、根與冠、濕潤與乾涸——創造之力，正源自二者的交會。恰似鳳凰棲於樹梢，喚醒沉眠的龍；看啊，那位自發的金童伸手採摘熟透的桃實。

然則天秤座亦預示歲時的「臨界」將至。那時晨星的光縷被裁斷，墜入灼熱的西天，踏上漫長孤獨的旅途，重新尋覓屬於它的高處。因為天秤之歌吟詠平衡、法則、正義、裁斷與明辨。他們歌頌真理與正義；歌頌言語、意志、行止的權衡；歌頌從內裡光芒升起的良知之曲。

人啊，當你的時刻到來，世界的重擔會壓垮你嗎？抑或你將如飛鳥掙脫羅網，騰入芬芳的空氣？

埃及的末日觀裡，心臟的稱量在審判廳進行——有時稱為奧西里斯之廳，或「雙重真理」之殿。奧西里斯既是天界之主，亦為瑪特之主，故被視為亡者面前的公正判官。

「瑪特的核心意義是『直』，正如我們所見，與希臘語『正典』（κανών）同源——原指直竿、石匠的矩尺，後引申為規範人與其行為的律則、法典、準繩。埃及人在物質與道德層面皆用此詞，意為『正確的、真實的、真理的、真正的、正直的、正義的、公正的』。因而《普塔霍特普箴言》有云：『真理（瑪特）偉大而強韌，永不動搖，自奧西里斯時代以來從未被破毀。』普塔霍特普告誡聽者：『令真理永存。』公正正直之人即為瑪特，是『被斷為正義的神』。」

瑪特作為阿門提的女主，手中執一根羽毛。她面前，一束金光從綠松石色的天穹垂落，輕觸下方緋紅的大地。啊，當我們在眾神審判前被稱量之際，願我們誠實述說真相。當我們凱旋踏入白晝，願先祖為我們歡呼。在那裡，陽光將在水中起舞，歡笑孩童的歌聲會在我們心底迴盪。

仇恨終將吞噬仇恨，受卑劣欲望驅使之人，必被命運的暗影吞沒。他們將在道上踉蹌而行。但親愛的孩子，你們不會失敗。因為善者如舟，雖行於驚濤之上，終將抵達靈魂的港灣——屆時，我平和的庇護必將降臨。

我乃天平，遊移於擴展的光線；我是金色天秤的光縷；我是公正的權杖；我是真理與正義的太陽；我是地平線的主宰；我是交握雙手者——我的時刻即將來臨，無可抗拒。
