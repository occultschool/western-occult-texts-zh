#  探索金牛座：神話、歷史與天文的交織
##  從古代文明到天文觀測，揭開金牛座的神秘面紗，探索它與昴宿、畢宿星團的豐富傳說。

![cover](/img/downloaded/b20c3b0d315db434cdea1b9cde6f9ef8.png)

金牛座曾標誌著西元前四千年至一千七百年間的春分點。阿拉伯人稱其為Al Thaur，敘利亞人喚作Taura，波斯語裡則有Tora、Ghav或Gau等名。希伯來人稱之為Shor或Re’em，意皆為「野牛」，聖經學者多認為那指的是已絕種的「原牛」。希臘語稱Tauros，時常聯想到朱庇特化身公牛、拐走歐羅巴公主的故事；也有人將朱庇特的偷情對象描繪為伊俄——那被喚作「流浪者」、視為月亮化身的女子。印度教寓言裡，則有梵天與其言語女神瓦查，被喻為吠陀經的吟唱之牛。

巴比倫人將金牛座稱作「天上之牛」（Gud.Anna或Gu-an-na）。值得一提的是，當英雄吉爾伽美什與恩基杜戰勝這頭公牛後，曾將牠的心臟攜往高天，獻祭給太陽神沙馬什。

胡神是德魯伊教裡守護神聖牛欄的神祇，尊號「至高擁有者」，以公牛之形顯現，「其聖所坐落在潮水環抱的島嶼上。」

金牛座肩上簇著七顆昴宿星，其中六顆可見，一顆據說隱而不顯。在印度，這七顆星象徵七位聖人的妻子。傳說戰神卡蒂克雅由她們撫養長大，最終斬殺暴虐的阿修羅，完成使命。印度二十七宿中，昴宿星構成第一宮「克里蒂卡」。

西方有時視昴宿星為鴿群，夜裡遭獵戶座追趕；在荷馬筆下，這些鴿子啣來甘露，哺育嬰兒宙斯，因而被賦予不朽的象徵。基督徒心中，這關聯到聖靈，並不意外。

有人認為「昴宿星團」（Pleiades）之名源於普勒俄涅——七姊妹的母親、永恆阿特拉斯的妻子。但這名字本身來自希臘語的「航行」（Pleine）。昔時水手相信，昴宿星團隱沒的月份不宜出航，故稱其為「水手之星」。亦有人追溯至阿拉伯語「豐收」（Al Thurayya）的譯名，推測指的是與星團同時出現的雨水，因而聯想其對農作的益處。

這些聯想，讓神秘主義者將昴宿星團詮釋為光的高等靈性力量，一種「甜蜜的影響」；有人說，這需要神之諸子與物質生活的情境結合、契合、調和。更有人認為，它們不僅象徵求道者為完成修行必須歷經的階段，也代表物質束縛之後的解脫，以及採取這些步驟所需的神聖手段。

「來吧，持杯人！這杯如皓月在我手中，為紀念國王親吻大地。當我將搖籃曳至昴宿星團之巔，便飲下金杯，為國王祝禱。」

據說賈姆希德國王的珠寶杯曾埋藏城基之下。另有一說未經證實：昴宿星團標誌著我們宇宙的中心。

金牛座另有一片美麗星群，同樣常與滋潤大地的雨水相繫。畢宿星團標誌著公牛的臉龐，是昴宿星團的姊妹星群，數量說法不一，常見為五或七顆。兩團星雖皆與雨水之賜有關，古希臘人卻賦予畢宿星團狂暴的聲名。

神話裡，許阿得斯（畢宿星團）是獵人許阿斯的姊妹。許阿斯之死眾說紛紜，或獅或野豬或蛇，端看採信哪個版本。姊妹們悲痛欲絕，因而升天化為星群。

在天上，她們依然徒勞地傾瀉液態的光輝，為墜落的兄弟哀悼，吟唱悲歌。

金牛座群星之中，畢宿五最為明亮，位列四大「皇家星」之一。其光輝如炬，點亮巨牛的眼眸。在牡羊座尚未取代金牛成為春分首星座之前，此星曾是眾星之主，標誌著春分降臨。更早的世代，它被喚作「天穹之溝」（阿卡德語 Gisda），或「光之信使」（Dilgan）。亦有人稱它為「耀眼的隨行者」（Na’ir al Dabaran），因它總似追隨著昴宿星團的蹤跡。

昴宿六被尊為「中央之星」，是昴宿星團中最醒目、最恢弘的主星。阿拉伯人稱之為「核桃」（Al Jauz）、「居中者」（Al Jauzah 或 Al Wasat）與「明耀者」（Al Na’ir）。在巴比倫，它被定為黃道第四星的基石（Temennu）。

故而，讓我們掙脫一切卑下形體的桎梏。應如昴宿六那般穩立中心，根基深固。讓我們點燃純淨火炬，宛若昴宿星團中的畢宿五。開啟通往昴宿豐饒之門；願那美德之光，如溫潤雨露，輕灑於正舒展的太陽系。讓昴宿星歌喚醒我們，使人自夢中甦醒；願星光化為甘霖，潤澤大地豐收。讓遙遠星辰的歌吟，與孩童歡笑的歌聲交融——唯有他，能將最深沉的長夜，轉為蒙福的白晝：

「當時日流轉，我將再臨，餵養你們的並非苦澀之物，而是善、真與美。我的甘言將觸動你靈魂深處，彷彿五月向晚，花香隨風拂過繁枝。我的光將成為你的光，如同遠星為夜空加冕。我將擁抱你們每一個人，親愛的孩子，一如我們光輝之母升起，以她璀璨的護佑之光，將人心提升至神前。

縱使塵世中人困於小我牢籠，你們必得自由；這自由，正是我們神聖耕耘的果實。

因我是生死與重生之主腳下的堅石；我是歷久不摧的強者；我是田野間耐心的耕者；我是金棚之主；我是「我是」之公牛；我將天界的祝福傾灑予同伴；我是塵世牢獄中熊熊燃燒的火炬；我的時刻即將來臨，無人可擋。」
