# 專注本體：與宇宙心智合一的修行之道

## 深入解析專注本體的實踐要點與心智合一的高階意識，說明以空白開始到內在大師形象的實踐路徑。

![cover](/img/downloaded/focus.png)

K.H.大師論專注於本體並與之結合

去專注在「本體」的概念上，  一開始可以先花個半小時的時間，不允許其它想法進入。漸漸地，便能把意識與本體結合起來。本體總是要在內在尋找的。向外看是一個致命的錯誤。你要努力達到所能達到的最高境界，並將自己保持在那裡。在返回肉體的最後一刻，才會覺察到此更高層次意識，而此時星光體也通常是活躍的，因此經常產生雙重意識，這正是錯誤的來源。欲獲得專注力，第一步是保持空白。 然後，逐漸地產生意識，最後，這兩種狀態之間的轉換變得迅速且容易，以至於幾乎不會注意到。 在專注的時刻，身體處於出神狀態中，並保留了一種夢境的感知（當然這只適用於弟子）。所要克服的巨大困難，是如何在物質層面上記錄對於本體的認知。要達成這一點，必須使肉體大腦對於一切完全空白，而只意識到「高層意識」；而星光體必須被麻痹，否則會導致錯誤和混亂。首先，試著讓自己處於某種狀態，使你對身體經歷的一切都不再有所感覺；事實上是把自己與身體分開。若在此嘗試過程中，你感覺到任何東西、任何外來的影響從外面進入，應立即中斷專注力。最好的方法是專注在大師上，也就是你內在活躍之人。在你內心形成其形象，成為你專注的焦點，以便在一心一意中失去對身體存在的感知。在專注的過程中，必須儘可能積極對抗幽靈和所有星光界低等影響，而儘可能接收大師的影響。心懷大師最能對抗幽靈。 對於通靈者而言，面對的危險比其他人要多，因為其身體更敏感，吸引更多其他自然力量。幽靈和星光界的影響會通過星光體對肉體起作用。 星光體是肉體的本質；但若無肉體， 星光體也無智能和意識。肉體感官的本質是星光體感官，肉體感覺的本質是星光體感覺。自由感是高等意識的顯著特徵之一，而使身體安靜下來所需的意志努力，與忘記痛苦所需的意志努力基本相同。人人在結合本體的經驗上都不相同。真正的本體是《道路上的光》中提到的「勇士」； 本體從不在此層面上行動，「心智」才是此層面活躍的代理人，是真正的參與者。這種與本體的結合是消除「分離感」的最佳手段，因此一個人必須成為本體的奴隷。本體是無形狀的、無性別的、無形體的，是一種意識狀態，一種氣息，而不是一個身體或形式。其最高等的形體是幻身，指的是整個人減去肉體和生命。其形體的樣貌如同肉體，除非被意志改變。本體是完美的□，「心智」(幻身)加上「菩提」和「阿特曼」，共同構成一面鏡子，反映「梵」，也就是「至一本體」。本體是宇宙靈的一個火花-「阿特曼-菩提」-是普遍的、永恆的、在此層面上無感知的。高等心智是心智，當上升到「阿特曼-菩提」時，就完成了三位一體，這就是至一。心智是自我意識，受限于一個大顯現期 。菩提則會從大顯現期延續至下一個大顯現期。幻身是中間的自我，是開悟者所使用的身體，並以欲體作為載體。這是人類之魂，是情緒、情感以及思想的所在地。當在身體裡時，便是心智。這七項原則是同一實體的不同方面，而其形體和特徵取決於意志。在19世紀，人類的本能（慾望）在很大程度上被心智或智能的發展所壓制。心智是墮落的天使，是「心智之子」所吸入的本質，而「心智之子」共同構成宇宙心智或顯現的邏各斯。單體，即「阿特曼 - 菩提」，其實根本不屬於這個層面，可以說是在梵層面。因此，心智是人在此層面上最高原則，正是這個心智使人成為神靈或魔鬼，因為神聖的單體通過心智在此層面上發揮作用，或者說，當「心智」在此層面上產生作用時，乃是藉由影響「菩提」較低部分中的「神-生命」之力而實現的。大師是那些生來就有應身的人。你們每個人都能為自己創造一個大師，在星光界中賦予他客觀存在。 若他是真正的大師，他將發出自身的聲音。若他並非真正的大師，那麼聲音將是本體的聲音。每個人都會根據自己的內在功德和發展來接受。
