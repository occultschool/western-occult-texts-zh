# 巨石陣之謎
## 關於英國巨石陣的起源與建造者，歷來眾說紛紜。本文梳理從外星造訪、天文觀測到宗教儀式的各類假設，結合考古發現與社會脈絡，剖析現代學者對其建造動機的不同詮釋。

這座矗立在英國的史前紀念碑，究竟何人所建？緣何而建？我們將撥開迷霧，檢視種種理論與事實。

## 緣起

巨石陣的起源與用途，數百年來始終成謎。自十七世紀中葉以來的考古發掘，未能解開這道難題，一如獅身人面像。猜測四起，有合理的，有荒誕的，也有天馬行空的。有人聲稱此處乃古代外星人的登陸地。實話說，若真需一處降落場，選這巨石林立之地，可謂最糟不過；況且我們對飛碟之說，本就將信將疑！

較可信的理論大抵分作兩派：一派視之為聖地，一派當它是觀象臺。兩說皆援引天體為證，指其結構對準日月軌跡，儀式則繫於季節流轉、冬至夏至。這些觀點實則延伸自諾曼·洛克耶爵士的假設——早在一世紀前，他便在其開創著作《巨石陣與其他英倫石碑的天文學探討》中首倡此說。

此說倒不排斥遺址兼備神聖與宗教功能。畢竟，瑪雅與其他中美洲文明，早將此二者熔鑄於金字塔之中。近來另有一所謂「激進」新論，指巨石陣乃史前「盧爾德」，即療癒之地。我們不解此論何以稱「激進」，基督教堂也用於療愈目的，早已尋常。看來，這些所謂的高明「專家」，似乎總難做到自己所提倡的「通盤考量」。

最奇特的現代理論，當屬考古學家邁克·帕克-皮爾遜所提。他認為巨石陣本是威爾士「石器時代」精英的葬地，後因某些費解的理由，「遷移」至索爾茲伯里平原。然則，何苦在紀念碑內精心佈置複雜的天文排列？帕克未給答案。這不禁令人想起另一樁蠢論：指大金字塔乃法老胡夫之陵，儘管尷尬的是，其中未見遺骸。

且讓我們暫離科學幻想的疆域，回到事實層面，探尋巨石陣的建造者與緣由。

## 建造者

普遍認為巨石陣建於公元前2000至3000年間，亦有理論指它由三個不同史前部族，歷經數百年合力而成。首批建造者稱「風車山人」，考古學家描述為「新石器時代農民」。第二批是「貝克人」，約公元前2000年遷至索爾茲伯里平原。第三批則是「威塞克斯人」，約公元前1500年抵達此地。

為何跨越一千五百年，三個不同族群要接續營造這座既非最宏大、亦非最複雜的建築？這問題留待所謂「專家」解答，儘管我們私下認為，此說難以成立。

事實上，無人確知巨石陣出自誰手。我們掌握的僅是各種理論臆測，可信度參差不齊。隨著紀念碑內外的新發現，許多既有觀點正受挑戰。2016年，一座據信比巨石陣早千年的龐大宗教中心出土，考古學者認為此舉將改寫人們對此聖地早期發展的認知。坦白說，我們對此持疑。畢竟，那些持保守正統觀點的人，需要跨出一大步，才能領會遺址的靈性意涵。更早之前，二〇〇四至〇六年間發現的杜靈頓城牆——位於巨石陣東北兩英里處的新石器時代大型聚落——據信原有千戶房屋、四千居民。這些發現的重要性，容後再述。

據說，德魯伊使用巨石陣已數世紀，多數考古學家卻嗤之以鼻，不認為是他們所建。主要論據是：德魯伊屬凱爾特族群，而碳十四定年顯示，巨石陣建成時，凱爾特人尚未在此定居，其間相隔近千年。然而碳十四定年法如今愈顯可疑——它僅測定遺址出土的有機物，而大氣中的碳十四含量歷來並不穩定，實則波動劇烈。這種波動緣於自然過程與人類活動；宇宙射線與地球氣候的變化，亦會導致大氣碳十四含量不規則起伏，連帶影響有機物的吸收。

年代鑑定還有個少被提及的難題：無人能斷定同一遺址出土的每件器物皆屬同一時期。這可能顛覆現行的考古方法。舉個淺例：假使你不慎將智能手機落入深井，數世紀後，地層運動使它貼近中世紀建築。千年之後，未來考古學家發掘此址，或許會推論手機發明於十五世紀中葉！

關於凱爾特起源的史料本就稀少，即便有，也常零散矛盾。無論德魯伊是否凱爾特人，若非他們所建，巨石陣又出自誰手？回答之前，須細察巨石本身。首先應注意——正如諾曼·洛克耶爵士所指——埃及神廟與英國石圈雖結構布局各異，天文用途卻基本一致。倫德爾·哈里斯認為，太陽崇拜約公元前三千年源於埃及，至公元前一千七百年左右達其「黃金時代」。其實起源更早：正如另文所述，埃及記載了最初亞特蘭蒂斯殖民者帶來的藝術、科學與信仰。哈里斯進一步推測，埃及商人與殖民者將此崇拜引入不列顛，約公元前兩千年傳至巨石陣。這些時間點，與我們先前討論埃及人抵英的時期、以及巨石陣建成的普遍看法相當吻合。此為建造者身分的第一條線索。然而線索遠不止此。

## 考古證據

可有考古證據支持這些主張？確實有。索爾茲伯里博物館（位於威爾特郡德維茲）展出了數件巨石陣周邊出土的黃金月牙飾品。這些飾物，與古埃及舊王國至中王國時期（約公元前三至四千年）的胸飾極為相似。在遙遠的愛爾蘭亦發現類似月牙飾，其中最醒目的一件由韋斯特米思郡農民覓得，年代約公元前兩千年。若埃及殖民者能抵愛爾蘭，抵達威爾特郡自非難事。

或許存在更多證據，指向埃及人與巨石陣的關聯。然而多數學者從未認真考慮古埃及人殖民不列顛群島的可能性，致使支持此假說的發現往往遭誤解，甚或完全忽視。我們在先前的文章中提過，十九世紀初巨石陣出土了琥珀首飾與黃金工藝品。威廉·坎寧頓於一八〇八年發現部分珠寶，至二〇一六年首度於巨石陣遊客中心展出。這些物品出自一位女性的葬禮，媒體迅即稱她為「巨石陣女王」。她的遺骸出土於諾曼頓唐塚，位於石圈以南約半英里。這位女性會是埃及女祭司嗎？難以斷言，但其珠寶工藝之精，絕不遜於埃及古墓所見的相似物件。

這是否意味古埃及人參與了巨石陣的建造？極有可能，甚至可能由他們擴建並掌控。由於埃及人與德魯伊最初同源亞特蘭蒂斯，共享諸多信仰習俗——尤其是太陽崇拜——故可斷言，巨石陣的建造目的，與埃及宏偉神廟並無二致。此間聯繫，容後深入探討。先前文章曾提及一部描寫古埃及生活的記憶小說《羽翼法老》，作者對威爾特郡阿韋伯裡石圈有詳盡描述。這段文字值得重引：「眾人靜聽，他們的老師來自一處名為a-vey-Baru之地，祭司們在那裡依阿努比斯之道受訓。其神廟外環深溝，牆以未雕整石砌成，以木與黏土相銜，表面覆著白石膏。」

蘭德爾–哈里斯認爲，英格蘭若干聖地實爲埃及殖民者所建。這些殖民者遠渡而來，爲的是開採銅、錫與其他珍貴原料，如羊毛、木材與琥珀。正如前文所述，埃及人極重視琥珀的宗教與巫術意義。哈里斯所列地點中，便有康沃爾郡的聖內克安基夫，它坐落在北康沃爾海岸的廷塔格爾附近。埃及語中，「內克」意指財產，「安」則是奧西里斯之名。當地傳說與此相印證：善良的聖人「內克安」離世時，兩位女子來到峽谷，爲他舉行葬禮，將他安葬在瀑布下溪畔的石棺中。此地至今仍是遊人如織的勝景。

所謂「兩位女子」，自然是埃及女神伊希斯與奈芙蒂斯——她們是奧西里斯的姊妹，曾立於棺旁哀泣。耐人尋味的是，聖內克安的節日在六月十七日，而奧西里斯遇害之日，恰是埃及曆法哈托爾月第十七天，對應今曆十一月二十六日。兩者月份雖不一致，但六月之名源於羅馬女神朱諾，其神格與埃及的哈托爾相似。由此可見，在相隔遙遠的兩個國度裡，都有供奉埃及神祇的聖所——阿努比斯與奧西里斯。如前所述，不列顛也曾敬奉女神伊希斯。那麼，對太陽神拉的崇拜在此出現，亦是順理成章之事。而這，正是我們在巨石陣所見的真相。

## 太陽大教堂

巨石陣的布局，能讓夏至陽光筆直射入中央聖殿。然而，單憑此點並不足以解釋這座建築的宏偉構造或真實用途。若它僅是一座天文觀測台，一座簡小建物便已足夠。公元前一世紀的迪奧多魯斯·西庫魯斯，曾將巨石陣稱為「大教堂」與「大合唱團」；倘若它真如今日權威所稱，只是史前「菁英」的埋骨之地，他斷不會如此描述。坦白說，迪奧多魯斯、希羅多德，乃至六世紀威爾士的行吟詩人塔利辛與阿內林，他們的記述仍值得今之學者與考古學家審視。畢竟，他們所處的年代，正是巨石陣作為太陽神廟、不列顛太陽崇拜最熾盛的歲月。且讓我們繼續深談此題。

蘭德爾–哈里斯堅信，埃及人建造巨石陣，是為了同時崇拜拉與奧西里斯，亦即融合太陽崇拜與亡者崇拜。我們認同這位誠實而著名的學者的觀點，但如前所述，巨石陣可能源於更早的太陽神廟，此說亦不能排除。此外，它曾被一座完整的城市所環繞，只是那些城區早已被流光歲月與人類掠奪摧毀，其中包含德魯伊的居所、與神廟相關的建築，以及傳授藝術、科學與神聖奧秘的學堂。2016年發現的巨大「宗教中心」，會不會是這座偉大城市的前期部分？極有可能，尤其是考慮到達靈頓牆與該地區目前出土的眾多遺址。歷史記載支持此一看法：據傳公元五世紀時，巨石陣附近曾發生一場大屠殺，此事與七世紀成詩的《阿內林的Y戈多丁》以及蒙茅斯的傑弗裡在《不列顛諸王史》中的描述相符。野蠻的撒克遜人在其背信國王亨吉斯的率領下，如同許多古往今來的部族與邦國，來到巨石陣與不列顛人舉行所謂的和平會談。然而他們的真實意圖，卻是掠奪德魯伊之城的神聖寶藏、黃金與珠寶。

阿內林告訴我們，大屠殺過後，城中傳來可怖的呼喊。不列顛人與撒克遜人之間爆發激戰，不列顛王子艾迪奧奮勇殺敵無數；但隨後，那些日耳曼蠻族冷酷地屠殺了三百多名手無寸鐵的德魯伊啟蒙師及其弟子。威爾士的三行詩句記下了對這場暴行的哀嘆。且讓我們聆聽詩中所言，進一步理解德魯伊人如何看待巨石陣這座偉大的教堂之城。六世紀中葉的威爾士吟遊詩人庫赫林如此悲嘆：

「黑色象徵奸詐之狼的憤怒，他只信奉鋼鐵的法則，劍是他統治的工具。公正的艾迪奧以勇智聞名，正主持這場集會。然而那邪惡之主（亨吉斯）佯裝與不列顛人交好，如今雙方結成虛假同盟，他更公開宣告，邀集同等數目之人前赴節慶般的會談。指定之地在伊歐地區，那片美麗的四角地域，正是該地大神廟所在。

一位優秀的詩人竟讚美這偽善者，還縱容他。此人唯一的美德，便是持矛衝向毫無防備之人。這冷酷戰士在狂怒中顯得至高無上。於是，領袖向這叛徒唱起頌歌，但他回報歌者的唯有死亡。他是位宏偉而深沉的智者。此刻，毫無預警的突襲帶來浩劫，死亡的尖叫劃破長空。一切源於虛假的攻擊與深謀的謀殺之計，真正的目的，只是欺騙。」

德魯伊大祭司就此殞命，遭怯懦狡詐的撒克遜人所害，全因他們貪嗜血腥、覬覦財寶。阿內林證實了每個細節——那場盛宴設在巨石陣大教堂附近一處臨時建築，血腥儀式便在那裡完成；這建築坐落於伊斯特雷或庫爾斯之上，一條大道自教堂直通該地。據古代史家所述，庫爾斯距巨石陣北側八百公尺，長三點二公里，兩道溝渠相夾，間隔百公尺。此處正是伊額爾區，即庫赫林所稱「大神廟所在的秀麗四角地帶」。伊歐之名有時意指「至高存在」，但這源於英國神話，喻示太陽循其軌跡或圓環運行。伊歐與荷魯斯淵源甚深，亦如奧西里斯，常被描繪為太陽本體。這便坐實了巨石陣的太陽屬性，以及為祭拜那賜予生機的偉大存在所行的儀式。

巨石陣太陽神廟的方位在天文上已然固定——正如崇拜阿努比斯的阿夫伯裡神廟——它坐鎮正東北，與一切古太陽廟宇無異。外圈六十石，象徵六十週期；內殿小圈十九石，恰如印度週期所載。數字十九可化為一加九，其和為十，這最圓滿之數一旦抵達便無法超越。畢達哥拉斯學派稱其為「神」、「天」、「永恆」與「太陽」。內殿之外，矗立五座宏偉的三石碑，皆由雙柱與楣石構成。由此衍生五、十、十五三數，十五在數學上可化為六。讀過我們(《論數字學》)[/docs/mystic-numerology-intro-1-to-10-meanings]之作的讀者，當不忘這些數字的神秘意涵。

巨石陣以圓形紀念太陽。其次，如上圖所示，中央聖殿呈橢圓。此形象徵世界蛋，或任何行星、太陽系乃至生命的輪廓，亦可喻指環繞眾生的能量場或磁場。壇上聖火永不熄滅。我們曾在[《白魔法師的典範— 德魯伊祭司》](/docs/white-mage-paragon-druid-priest)述及這靈性之「火」，並描繪昔日英國石圈內舉行的神聖儀軌。若你未曾讀過，此刻正是時機——如此你方能明瞭數千年前巨石陣上發生之事：那時，德魯伊大祭司身披純白長袍，召天火降臨塵世，以躍動之光療癒眾人。

此廟無遮無蓋，因德魯伊與許多古民族皆認為，若將神禁錮於密閉龕室，實為不敬。至於石材本身？或許你已知道，築造巨石陣時主要採用兩種石料。其一為質地較軟的石灰岩，當地方言稱「撒森岩」，此名被視為「撒拉森人」之變體，帶有非基督教、異教之寓意。正是此石構築了三石碑，形成環繞神廟的內圈。

巨石陣所用的第二種石材名為「藍石」，取自彭布羅克郡的普雷塞利山脈。此石較花崗岩更堅硬。我們刻意強調「軟」「硬」二字，因發現築大金字塔時亦採此軟硬混用之法：石灰岩築主體，花崗岩則用於所謂「國王墓室」及通往該室的「大走廊」牆頂。最初，整座金字塔外殼皆覆紅色花崗岩，而非石灰岩——儘管今日幾乎無跡可循。此說證據可見於吉薩高原上另兩座金字塔，即卡夫拉與門卡烏拉之塔，其表仍殘存些許花崗岩外殼石。我們不知大金字塔的花崗岩外殼何時被移除，但合理推測，這與古代神秘學派的消亡時序吻合，我們在[《啟蒙者與啟蒙》](/docs/enlighteners-and-enlightenment-myths)一文中曾論及。米紹博士在其短篇[《銅鏡》](/docs/copper-mirror-telepathy-ancient-egypt)中提及，在他以心靈感應追溯的前世記憶裡，這些外殼石僅覆蓋大金字塔上半部。由於我們確知這段記憶正處第十八王朝中期，約三千五百年前，由此可窺這座宏大啟蒙之殿何其古老。

無論在埃及或英國，採用這兩種迥異石材皆非偶然或巧合。礦物具多重特性，然非所有特性皆為材料科學所熟知。這些奧秘特質常遭誤解誤用，成為當代諸多新紀元信徒沉迷「水晶」療法的根基。但石頭確擁有些屬性，被無知者視為「魔法」，而古人深諳此道，並依循某套玄妙法則謹慎運用；這正為何埃及人與德魯伊皆在其宗教建築中擇用兩種石材。或可說，花崗岩親和高我，石灰岩親和小我，而古太陽神廟的宗旨之一，正是透過啟蒙過程使高我解脫。此間，高我潛藏的某些能力被喚醒，轉為明晰的意識，使蟄伏的心靈感應力得以全然舒展。

由此觀之，巨石陣具多重目的。其一，乃是作為禮敬靈性太陽及其一切所涵的中心。其次，這裡是一所教學學院，讓德魯伊精心揀選的學生秘密研習神秘科學，一如他們埃及同胞在神廟學校中那樣。第三，這裡是進入秘儀啟蒙的中心。第四，這裡是心靈與身體療癒之地，每當大祭司及其幕僚舉行公開儀式時，眾人皆可進入。靈視者傑弗裡・霍德森曾記錄斯諾多尼亞一處石圈中的儀式，可供參閱[《白魔法師的典範— 德魯伊祭司》](/docs/white-mage-paragon-druid-priest)。第五，德魯伊在此為國君與百姓伸張正義、提供諮詢，以謀福祉。

導言所提諸問——巨石陣由誰建造、何時建成、為何而建——至此已得解答。此建築由埃及人所建，或至少由他們擴建，年代約在公元前一千五百年左右，雖有主張可能更早，與埃及金字塔相仿。建造緣由，前文已述。若再添筆墨，便淪為臆測；我們從未妄加猜測，此刻亦無理由破例，儘管深知或有讀者期待這般。

## 總結

先前論及英國埃及淵源之文，曾提及巨石陣與艾夫伯裡。若將該文所述英國諸多聖地，與本文內容相參，便知數千年前此地無疑是歐洲神秘儀式的重要據點。那些秘儀殿堂與學園規模多大、範圍多廣，今已難考，但相形之下，梵蒂岡城不過一處微末聖所。我們另文探討過德魯伊的起源、信仰與智慧，亦刊載過亞特蘭蒂斯人四散寰宇的記述。對於欲深究亞特蘭蒂斯豐厚遺產的讀者，這份三部調查必引興味:[《亞特蘭蒂斯的事實和幻想》](/docs/atlantis-facts-and-fiction)。同時，但願此番探究能予您豐沛思緒，並匡正關於這些不列顛群島上最古老、最矚目遺跡的若干誤解。
