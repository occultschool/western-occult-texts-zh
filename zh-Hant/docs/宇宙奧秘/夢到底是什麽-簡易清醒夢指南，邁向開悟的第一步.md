# 夢到底是什麼？- 簡易清醒夢指南，邁向開悟的第一步
## *人的初次開悟，始於昏睡。夢中，一切知識皆已開啟。*

![封面圖片](/img/downloaded/abdb6fe5d88c75ffa46367d7793e3a6b.jpg)

攝影：[R Mo](https://unsplash.com/@mooo3721?utm_source=medium&utm_medium=referral) 於 [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral)

夢有多種。除了常見的「消化夢」（譬如乳酪吃多了），簡略劃分，還有大腦夢、記憶夢、警告與預感之夢，以及當身體沉睡時，高我經歷某些事情後，所產生的扭曲回憶夢。身為神秘學的學徒，我們關注的是接下來要介紹的夢。

> *人的初次開悟，是在昏睡狀態。夢中，一切人類知識皆已開啟；一道微弱的橋懸於無垠虛空，連結現世的靈性與更高世界的靈性。 — 布爾韋*‧*萊頓《札諾尼》*

布爾韋‧萊頓筆下的「橋」，在印度教中稱為「安塔卡拉納」。這是一座假想的橋，連接高我與小我；換言之，它橫亙在我們不朽的靈性自我（高我）與物質的大腦感官（小我）之間。當我們的心思專注於靈性而非物質時，這座「微弱的橋」便得以加固。亦可將「安塔卡拉納」比作兩臺無線電設備，彼此通訊。需將其中一臺（小我）調至與另一臺（高我）相同的高頻，否則通訊雜亂無章，甚至完全斷絕。

布拉瓦茨基將夢劃為七類。

1.  **預知夢**：高我在記憶中留下的印象，通常簡潔清晰。或是一句話語，或是一幕即將發生的事件。
2.  **寓言夢**：現實朦朧的片段被大腦捕捉，再經幻想扭曲而成。這類夢往往半真半假。
3.  **黑白魔法師植入之夢**：亦可能由催眠師藉強大精神力量發出，意在驅使我們遵從其意志。
4.  **前世夢**：夢見屬於前塵往事的事件。
5.  **警告夢**：用以警示平日疏於戒備之人。
6.  **困惑之夢**：成因紛雜。
7.  **純粹幻想與混亂畫面之夢**：源於消化不良、精神困擾或外界干擾。

我們大致認同此一分類，儘管如布拉瓦茨基所言，這七類尚可細分，並對應意識的不同層面。但我們關心的僅是前五類，尤其第一、第二與第四類——預知夢、寓言夢與前世夢。它們對靈性成長確有裨益。

寓言夢時常予人深刻印象，無論是揭示自身處境，或激盪內心強烈情感，往往蘊含重要啟示，有待我們自行發掘。當一個夢境及其變體在數日甚至數月內反覆出現，便可確信其中必藏訊息。這是較高靈魂領域的導師，與塵世中心扉敞開的弟子溝通的方式之一。

關於第一與第四類夢（預知夢與前世夢）的運作，布拉瓦茨基給出一個精妙的譬喻。

> *「可將真正的自我（我們的高我）比作囚犯，而身體的人格（我們的小我）則是獄卒。獄卒若沉睡，囚犯便越獄，至少能脫離肉體牢籠。若獄卒半夢半醒，倚窗打盹，便只能偶爾瞥見囚犯掠過窗邊的身影。這樣的獄卒，豈能知曉囚犯的動靜？更遑論知道其真正舉止，尤其是內心所想？」*

答案是不能。對常人而言，很少人能清晰知曉身體沉睡時高我離體經歷了什麼，極為罕見。即便普通的神秘學學生，亦復如是。

置身這等夢境，我們對高我的體驗往往只餘殘影，宛若帆布篷上映出的淡薄輪廓。醒來時只見光影綽綽，自以為作過夢、歷過事，實則只是高我離體經驗的浮光掠影。待到神智全然清明，回憶便每分每秒地扭曲變形，遭小我與肉體大腦投射的意象侵染、雜糅。這些意象又借聯想之力，蔓生出種種念頭，最終與高我短暫解脫時所獲的真實體悟，幾乎失卻所有關聯。這便是為何多數夢境在小我清醒時，總顯得紛亂無序。

## 清醒夢及其實現之道

若要培植「清醒夢」這般能力，便需強化兩個「我」之間的橋樑，即前文所提的「安塔卡拉納」。收窄橋梁間隙的方法之一，是在床頭備妥筆記本與筆。醒後立刻記下所有能憶起的夢——無論多麼朦朧、零碎甚至荒誕。之所以強調「立刻」，是因為醒轉剎那，諸多記憶便已逸散。若能持之以恆數星期，你會驚覺自己記住的夢竟多了許多。隨著練習日漸規律，不僅夢境會愈發清晰，你更將辨識出：究竟是哪個意識正在「造夢」。或許還會接收到某些訊息、符號或知識，點亮你的靈性修習，照見其中隱藏的意義。這些訊息會以何種形式顯現，無人能代你預言——正如前述，每個人皆活在自己的心智世界，所思所想從無雷同。

這項技術行之有年，成千上萬的男女早已親身驗證其成效。
