# 宇宙意識
## 既忘不了，也否認不了；無論外在的記憶或內在的烙印，都抹不去它。

![cover](/img/downloaded/3daa8790f737eacbcfec6445e0ebb696.jpg)

Photo by [Greg Rakozy](https://unsplash.com/@grakozy?utm_source=medium&utm_medium=referral) on [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral)

許多哲學家與神秘學者，都曾著墨宇宙意識。

「宇宙意識」既是對宇宙真理的覺知，也是宇宙本身的意識，宇宙意識也即是神。人若擁有這種意識，便能同時覺察萬事萬物。他知曉過去、現在、未來渾然一體。因為一切皆涵容於神的意識中，也因此蘊藏在人的靈魂裏。

只要懂得揭開面紗，或得到啟引，便能使人的意識契合於神。一旦掀開那層面紗，望見內在神廟的堂奧，我們便在外在意識與內在意識之間，建立了永恆的聯繫——在小我與高我之間，在高我與神聖靈魂之間。那神聖的一刻，我們觸及了真理。既忘不了，也否認不了；無論外在的記憶或內在的烙印，都抹不去它。

對神秘學學徒而言，進行這種神聖接觸至關重要，因為他將在血肉之軀中體證靈性真理。即便死後暫居星光界，等待轉世，這份體悟也將隨身。正如我們所知，當脫離塵世軀體的枷鎖時，唯有這意識能隨身攜帶。其餘一切皆如影逝散：不過是幻象，虛假而無實質。

這份對真理的認識，須透過冥想與觀照達成，從外在世界抽離，轉入內在世界。讓靈性脫離低層區域，向上升華；背對陰影，將內在目光凝聚於光明。這種感知無法藉由任何思辨過程獲得。縱然純粹思維的感知模式已超越單純智性，仍只能對靈魂的高等能力產生模糊的理解。

宇宙意識唯獨透過靈視的感知方能抵達，超越一切肉身限制，透過與靈魂契合，讓高我的意識適應更高境界的狀態。而靈魂又與高等境界的所有靈魂直接相通——它們本是一體！這便是宇宙意識。達到它，甚至僅僅追求它，便是人類對神最深的敬意。

但在觸及宇宙意識之前，必須先充分覺知並認識我們作為「人類自我」的幻象。赫爾墨斯箴言早已告誡：「認識自己者，便知自身所蘊含的一切。」

然而，當注意力投向神聖事物時，仍須確切履行現實生活的職責。我們被安置於肉身的聖殿，本為執行某些塵世任務，並服務同胞。

人若只將思想欲望全然緊繫物質世界、並在其中覓得昏暗樂趣，將難以理解此真理。他們只見到結果，卻看不見生成萬物的原因。他們不知道，從「至一實相」映現出的靈，再從靈映現出的物質——卻誤把這重重夢境視為現實。事實上，他們認定的現實並非真實，只是感官的體現，在世俗腦海中描摹出那實相的倒影，而客觀存在卻始終隱藏真理。

那麼，如何從這些世俗幻象中抽身？如何熄滅欲望、止息渴求，從而抵達深邃的平和——這一切追求神聖奧秘者的目標？唯有放下貪婪與憎恨，轉向愛與服務，方能揭開帷幕，顯露真理。而心思粗鄙之人，在塵世的幽暗裡活動，棲居於形體的世間，永遠無法看見它。但那些曾見識光明的人，必須傳遞這訊息。因為有些人早已準備好，只待一句親切的話語點醒。他們只需一隻療癒之手，輕觸並冷卻那疼痛的雙眼，便能看見那敞開的永恆之門。

尋常的自然推理永遠無法產出宇宙意識。神的本質屬光的領域，任何從理性衍生的猜度都無法滲透那片天界之境。哲學家康德點破了此點，但其智慧的真義過於隱晦，能理解其崇高教誨者寥寥無幾。主要緣故在於，多數學子忽視——或者說看不見——智性推理與靈感慧光之間的區別。

物質終將腐朽，因其為物；靈性卻屬神聖，因此永恆純淨。

此即康德所言的「純粹」理性——與人靈魂中那近乎神性的稟賦相比，凡俗思維不過是超驗真理的障礙。這真理只向信者顯現，因唯有信心能開啟內在的感知。

可以說，宇宙意識是一種超越粗鈍物質的延展。我們或憑七種靈性感官之一觸及它，或通過鍛煉，或生來便是靈視者。宇宙意識，意味著更進一步與宇宙萬物及眾生「同在」。

實則只存在一種絕對意識，無論其名為宇宙意識或其他。它是至一的實相，絕對的存在，至一的生命。那是「本體」，與有制約的存在毫無瓜葛；而有意識的存在，僅是一個有制約的符號。有制約的存在屬物質界，受變遷、謬誤與幻象所困。意識與物質、主體與客體的二元，應視為「絕對者」的兩個符號或面向，而非獨立的現實。

靈魂愈接近「絕對者」，便愈貼近真正的宇宙意識，愈能遠離物質幻象。在這通往絕對意識的途程中，將歷經若干階段。未進化的心智將在星光界中輾轉千生萬世，直到靈性知覺逐漸成熟。在投生下一世或更高層面的塵世生活前，它會在星光界居留一段時日，為來生預備。這過程需時百萬年。於每一層面，它將顯現為不同的身體樣貌與形狀。隨著上升，其形體將愈發靈性化，如靈光繚繞，實則是人類形貌的延伸。

此後，意識持續擴展。及至最高層面，「靈魂—心智」便進入至福與智慧的境地：它散發著生命，舒展、充盈，臻於完美高妙的境界。那境界不可言說，只能親證，因無一語能述其性質，或描摹其中所感。即便親歷宇宙意識者試圖訴說，聞者也僅能聽見字句，卻無從領會——除非他們自身也曾經驗。

由此我們明白：在肉身中證得宇宙意識的同時，也將對人世有通透的了悟。如此，我們方能為值得者引路，領他們回歸至一的實相、至一的真理。願讀到這裡的你，在神所賜的美好時光中，得享這份宏大的恩典。
