# 魔戒人物的象徵意義
## 弗羅多與咕嚕：高我與小我

![封面圖](/img/downloaded/d3ebdeece9b0d051831407693cff30d5.jpeg)

《魔戒中的善與惡》（/docs/tolkien-good-evil-in-lotr）提過，甘道夫是「秘火之僕」，乃眾神維拉直接的使者。托爾金曾在信中寫道：「甘道夫並非人類，也非哈比人。現代詞彙無法定義他。他是降世的「天使」，或更精準地說，是信使。」希臘文「angelos」即是信使之意。托爾金藉此詞印證其神聖信使之身分，正如[《金黃星》](/books/golden-star-dr-michaud)裡的涅特魯-赫姆。書中的甘道夫，不似彼得·傑克遜電影所描繪，沉溺於拳腳紛爭，更非揮舞魔杖的卡通英雄。衝突當前，甘道夫給予啟發、鼓勵與睿智建言。即便偶爾對抗敵對勢力，他也僅是領袖與發起者，絕非衝鋒陷陣的戰士。

巫師薩魯曼同為信使，故事一開始，他力量更在甘道夫之上。然而他墮入「黑暗」，為膨脹私慾，不惜傾其所能摧毀遠征隊。他善用催眠術，或稱「暗示」的魔力，在故事中屢屢奏效。但他終究敗了。甘道夫對他說：「薩魯曼，我太瞭解你。你的言詞作為，我記得一清二楚。你看，我不再是遭你背叛的灰袍甘道夫，我已是死而復生的白袍甘道夫。如今你一身灰濛，我要將你逐出教團與議會。」

書末，弗羅多與潦倒失勢的薩魯曼有過一番對話。薩魯曼的僕人稱為「巧舌」，名副其實，曾暗藏匕首意圖刺殺弗羅多，但未得逞。「十幾個哈比人在山姆帶領下，吶喊撲來，將這惡棍壓倒在地。山姆抽出劍來。『別，山姆！』弗羅多阻止：『現在也別殺他。他並未傷我。何況，我不願他在這邪惡之境喪命。他昔日何等偉大，曾是我們不敢忤逆的尊者。他雖墮落，治癒之法已超出我們所能；但我仍想饒恕他，盼他能尋得救贖。』

薩魯曼站起身，直直望向弗羅多，眼神複雜交織著驚詫、敬意與憎恨。『你長大了，半身人，』他說：『你確實長大許多。你既睿智，又殘忍。你奪走我甜美的復仇，如今我只能含恨離去，皆因我欠你這份憐憫。我痛恨這憐憫，也痛恨你！』」
驕傲與我執令最強者沉淪；智慧與胸懷卻讓最卑微者崛起。然而，正如弗羅多所言，縱是最邪惡的生靈或神祇，只要仍渴望「治癒」，救贖猶有可能。這確是耐人尋味。

書中人物，弗羅多背負著的魔戒重擔，也歷經最劇烈的轉變——最終，他更似精靈，而非哈比人。他穿過[《中土世界的生與死》](/docs/tolkien-mysticism-middle-earth-life-death)中提到的「都林之門」，是他啟蒙之路關鍵一步。這趟覺醒之旅，始於他慨然承接摧毀魔戒的重任，最終將它投入鑄造它的烈焰之中。任務始於夏爾，當時甘道夫向他闡明原委。「我恨不得毀了它！」弗羅多喊道，隨即補充：「或是讓它被毀。但如此險途，我並不合適。」
若將「魔戒」視為小我的某種象徵，那誘惑欺瞞人心的力量，弗羅多既想擺脫它、又懼怕涉險的心境，聽來何等熟悉且人性化？掌握小我是一回事，真正扛起責任又是另一回事。因此，當弗羅多對甘道夫吐露：「但我自覺渺小，滿心絕望。敵人如此強大可怖。」我們毫不意外。在學會駕馭小我之前，誰不是如此？待論及咕嚕時，我們將重返這雙重自我的主題。

後來在愛隆會議上，佛羅多才真正體會到任務的艱鉅。「眾人皆垂首默坐，彷彿沉入深思。一股巨大的恐懼攫住了他，像在等候早已預知的判決，又暗暗盼望那判決永不被宣出口。他心底湧起強烈的渴望，只想回到瑞文戴爾，留在比爾博身邊，求一份平靜。最後，他掙扎著開口，聽見自己的聲音時吃了一驚——彷彿有別的意志藉著他微弱的嗓音在說話。他說：『我會帶上魔戒，雖然我不識路。』」這話多麼有力而動人，且蘊藏著深意！教人想起從前聽過的另一些話：『那沒有看見就信的有福了。』（約翰福音20:29）『若有人要跟從我，就當捨己，背起他的十字架來跟從我。』（馬太福音16:24）「十字架」或「戒指」所指原是同一物：即人的肉身與小我。只要活著，這副重擔就得背上，是走向榮耀與解脫，或是奴役與黑暗，選擇在於我們。

書中有個極神秘的人物，湯姆・龐巴迪。托爾金在一封信裡寫道：「即便在神話時代，也須留有謎團，因為謎團本是常在的。湯姆・龐巴迪便是一個（有意為之的）謎。」這話何解？回答前，先得分清這角色兩個不同的來歷：其一所謂「湯姆・龐巴迪」，原只是個憨傻的荷蘭玩偶，屬托爾金次子麥可所有，還被兄長約翰沖進了馬桶（！）；其二才是現於《魔戒》中的神秘人物。幸而玩偶最終獲救，未因那不愉快的洗禮而損毀。此事促托爾金寫下《湯姆・龐巴迪歷險記》，1934年刊於《牛津雜誌》。他後曾向出版社提議，將龐巴迪的故事擴寫為《哈比人》續篇，出版社卻反應冷淡。於是湯姆二度復活，出現在《魔戒》第七章裡。托爾金自承，一個孩童玩偶竟在他筆下演變成《魔戒》裡深具神秘色彩的角色，連他自己也覺困惑。早先一封信中，他稱龐巴迪為「（正在消失的）牛津與柏克郡鄉野之靈」，同時坦言無法將他從《魔戒》中抽離，正因他代表了某種更偉大的存在——儘管他不願細說那是什麼。

托爾金這番話，我們並不意外。或許這也是導演彼得・傑克遜將湯姆・龐巴迪從電影中刪去的原因之一。湯姆既非精靈、矮人、人類，亦非哈比人，也不是甘道夫或薩魯曼那類巫師；他到底是什麼？《魔戒》第七章裏，佛羅多曾向湯姆展示魔戒。「湯姆將戒指套上自己小指末端……片刻間，哈比人未覺異樣。接着他們齊抽一口氣：湯姆絲毫沒有隱形！」佛羅多困惑極了，疑心湯姆戲弄他，便自行戴上戒指，頃刻消失。但湯姆與哈比人不同，他依然清楚看見隱形的佛羅多從身旁悄悄走過。事實上，書中無人能完全不受魔戒影響。連甘道夫亦難豁免其力：本書稍早，佛羅多想將戒指交給他時，他厲聲道：「別引誘我！我不願變得如黑暗魔君一般。」後來到羅斯洛立安，佛羅多想將戒指交給凱蘭崔爾，她也拒而不受。然而魔戒到了湯姆・龐巴迪手中，卻失卻誘惑與腐敗之力。這該如何看待？湯姆究竟是誰？

托爾金自己在書中留下了謎題的線索，他寫湯姆是「森林、水與山巒的主人」。第七章裡「主人」一詞反覆出現，且多以大寫標示，足見其分量。當佛羅多問龐巴迪「你是誰」，他的回答深富啟示：「你還不知我的名嗎？這便是唯一的答案。告訴我，你又是誰，孤身至此而無名無姓？你年輕，我年老。我是最古老者。在河流與樹木出現之前，湯姆已在此地……他知曉星光下的黑暗，那時黑暗尚不可懼；那是在黑暗魔君自外而來之前。」因此可以說，湯姆是一種力量的化身，這力量既在中土之內，亦在中土之外。若他再多說一句，便等於要點明那托爾金不願「仔細探討」的「更偉大的存在」為何了。

我們亦知湯姆是個偉大的歌者。他正是通過歌聲來發現並解救了困在老森林惡柳中的霍比特人。後來，古塚屍妖囚禁他們時，仍是湯姆的歌聲破除了禁錮。英國古時的德魯伊自稱「歌者」，湯姆的歌聲令人想起他們如何以話語施法，因而贏得凱撒等人的敬重。據《英國德魯伊的神話與儀式》記載，大酋長辛維林曾言：「凡我譜曲，若由我歌之，咒語便生。」換言之，倘若他如書中的湯姆‧邦巴迪爾那般吟唱，魔咒便會湧現，令囚禁霍比特人的荒塚石崩岩落。屍妖亦歌，然其聲蘊黑暗魔力，能將哈比人牢牢縛住。由此可見，托爾金不自覺間重現了諸多神秘科學真理，並以靈感充盈之筆加以呈現。

## 佛羅多與咕嚕：高我與小我

至於咕嚕，他許是書中最複雜的人物，在善惡慾望的撕扯間煎熬。在《霍比特人》中，比爾博懷著憐憫與同情取得魔戒；咕嚕——原名史麥戈——卻是殺害友人狄戈奪得，視之為生日禮物。同類相感，道德層面上，他一旦戴上戒指，邪惡便佔了上風，這點在比爾博與佛羅多身上並未發生。於此可見：小我對高我猶有優勢；這優勢因人而異，或大或小。然而，儘管惡行累累，咕嚕卻拯救了中土：若非他在霍比特人即將戴戒挑戰索倫之際，從佛羅多手中搶走戒指，使命必將失敗，中土亦將陷入新的黑暗紀元。

若視佛羅多為求解脫、求啟悟的高我，咕嚕便是他的小我。但莫忘，每個自我皆有多重面貌。高我非盡善，因受物質沾染；小我非盡惡，因受另一半——即高我——影響。隨著故事展開，我們在佛羅多與咕嚕的糾葛中窺見此理。托爾金告訴我們，得戒之前的咕嚕類似於霍比特人，與佛羅多並無二致。到了故事後段，當咕嚕在他較善的部分（史麥戈）與最惡的本能（咕嚕）之間自我辯論時，其雙重性展露無遺。我們由此更進一步窺見兩個自我或兩種心智間的複雜關係與相互依存。不確知托爾金是否察覺此中深意，但凡稍解雙我及其糾葛者，讀《雙城奇謀》第四卷第二章中咕嚕的自相辯難，皆可深味。

「史麥戈答應了，」第一個念頭說。  
「是，是，我的珍寶，」應聲道：「我們答應了：保住咱們的珍寶，絕不讓他到手。可珍寶會往他那兒去呀，一步近一步。霍比特人拿它想做啥，咱們倒要瞧瞧……」  
「我不知道。我沒法子了。主人得了它。史麥戈答應要幫主人的。」  
「是，是，要幫……珍寶的主人。可要是咱們成了主人，那就能幫自己啦，可不是？諾言也守了。」  
「但史麥戈說他會好好待人，非常非常好……」  
「非常非常好，嗯，我的珍寶？那咱們就好好的——只對自己好。」  
「可珍寶會守信哪，」史麥戈抗議。  
「那就拿了它，」另一個聲音說：「咱們自己拿了！那咱們就是主人啦……」  
「說不準我能變得好強，比戒靈還強。史麥戈大人？咕嚕大帝？咕嚕！天天有魚吃……要新鮮，從海裡來的。非得吃不可。咱們要，咱們要，咱們非要不可！」

我們身上的兩個自我，便是這般爭奪主導。有趣的是，咕嚕的慾望不過是天天有魚吃。現實中，多少人的志向亦不過如此，由此可見他們的命運受哪個自我主宰！多數人身上，時而此我佔上風，時而彼我得勢，而爭鬥的最終勝負，或許取決於爭鬥者自身。德國有句諺語，恰切道出此中艱難：「與己鬥最難，勝己者至偉。」智慧的埃及人早知這場內在戰爭，便以荷魯斯與賽特之爭為喻。神話中，托特拯救了二者。他令「狂暴者」（小我）靜定，洗淨「赤紅者」（高我）。「赤紅」一詞源自與太陽神拉的關聯；拉為荷魯斯之父，而荷魯斯在神話的某一階段正代表高我。托特在此象徵神聖靈魂——我們在[神秘學課程](/docs/soul-essence-origin-and-mysticism)末篇曾加探討。那便是我們內在深處的「微小聲音」，即「寂靜之音」；唯傾聽它，我們方學會在兩個交戰的自我間締造和平，使二者皆得解脫，各獲應有之報。
