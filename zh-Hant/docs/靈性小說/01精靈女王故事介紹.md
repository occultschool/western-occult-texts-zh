# 精靈魔法 ：《精靈女王》故事介紹

## 揭示地精穿越四元素尋回女王健康的啟蒙故事，融合神秘主義與心智成長寓言

![cover](/img/downloaded/elf-queen-1.png)

西奧多·鮑姆評論

當我得知《神秘奧秘》雜誌準備出版一篇米紹博士不為人知的童話故事時，我在驚訝中帶有懷疑。這是真的嗎？ 或者更糟，只是粗製濫造的靈感寫作？然而，在迅速瀏覽他們發給我的草稿後，便毫不猶豫地消除顧慮了。而且，隨著繼續閲讀，懷疑變成了驚訝、驚奇和越來越多的敬畏。這就是精靈魔法，我高興地叫了起來。我毫不懷疑，每一位熱愛米紹博士作品的人，在閲讀此非凡故事時都會有同樣的想法。

《精靈女王》是場如煙火般精采紛呈的文學盛宴，不可能是由米紹博士以外的人來寫。任何其他作家都無法如此精采的、令人陶醉的描述大自然。三位勇敢的地精穿越空氣、水、土和火這四種神秘元素，期望能恢復美麗女王的健康和幸福。其中有感傷、諷刺和幽默的元素，交織成一個最不尋常的啟蒙故事。

在故事的主題中，巧妙地隱藏著一個潛在含意，象徵著高等心智和靈魂的煉金術結合，這是人類的兩個高等原則。可以將女王視為高等心智，追求與所愛的王子(靈魂)結合，或者是相反過來也可以；因為這都傳達了同樣的真理：只要對自身靈魂陌生，都是不完整的。

這些地精和元素精靈使用的魔法武器，也不是隨意選擇的；每一件都有一些潛在的神秘意義，勤奮的讀者能自行發現；都與必須穿越的神秘元素有著微妙關聯。簡而言之，米紹博士後來神秘學書中的所有要素，都可以在這令人愉快的故事中看到，而此故事寫於1903年，當時他才19歲。 

是的，19歲！在很久以後的1937年，他才將《精靈女王》從荷蘭語原文翻譯成英語，翻譯得如此之好，根本看不出來原本是用荷蘭語創作。我不想破壞此故事，更不想洩露情節，而是讓讀者注意到，此作者年輕時的作品與其成熟時期有許多相似之處。

熟悉米紹博士的讀者會注意到，《精靈女王》中的主題預示了後來作品，特別是寫作風格方面。讀到此故事的某些段落時，會覺得已進入了《幻想交響曲》或《魯魯的尋求》中一些未寫的段落；某些敘述也預示了《李文和的教誨》中火爆人物盧尚與對話者之間的幽默交談。

這並非米紹博士的第一件創作作品。八年前，他為一部不尋常的歌劇創作了音樂和劇本，其中所有人物都被相繼謀殺，後來作為幽靈重新出現。雖然創作仍然是米紹博士生命中永恆的愛好，但他從小就熱衷於研究民間傳說、神話，特別是童話故事。據說，他在12歲時就能背誦安徒生的所有童話故事了，難怪《精靈女王》能與安徒生的精采故事相媲美。

1908年，在完成《精靈女王》的五年後，米紹博士移民到了英國，很快就確立了自己的地位，成為一名具有相當天賦和聲譽的作曲家和音樂家。在大戰結束後不久，他創辦了音樂出版公司，後來被併入環球音樂代理公司（UMA）。這後來成為一間出版社，於1939年至1950年間出版此作者的六本神秘學書籍。

1929 年，米紹博士加入了一個獨特社團 : 精靈調查協會，是兩年前由昆汀克 • 勞福德上尉創立。入社的候選人必須能夠看到自然精靈（仙女）並證明自身的特異功能，才能被接受為成員。米紹博士的會員證書轉載於下。

![cover](/img/downloaded/elf-queen-2.png)


此協會一名成員曾吹噓說，其祖父聲稱與美人魚有染。你可能認為這種說法，與那些盲目發誓自己前世是埃及艷后的人，有異曲同工之妙！但讀過《加巴利斯伯爵》 這本書的人都知道，人類和自然神靈之間發生關係不僅是可能的， 而且根據這位神秘伯爵的說法，還是非常可取的。 

 神秘主義編年史上有大量關於這種奇異關係的記載，所以我不太確定是否只是童話！ 但若仙女和人類之間的歷史是可信的，總對一方或雙方都有不好結果，如濟慈著名詩歌《無情的妖女》所述。

該協會對於其存在和成員是嚴格保密，這主要是因為其成員中有幾個高級別的人，據稱包括英國皇室成員，熱衷於與那些「能看到不存在之物 」的人來往。直到後來，該協會的目標和活動細節才洩露出來。

1940年協會解散後，克萊爾-坎特隆講述了協會成員目擊一個典型「小人」的過程，而克萊爾本人從米紹博士加入的那年起一直擔任榮譽秘書，直到1937年。她寫道：『我住在格洛斯特的一座老房子裡，後面的花園盡頭是博得裡森林，覆蓋科茨沃爾德山一部分。我已經洗了頭髮，正在森林裡曬陽光，離房子有段距離。突然，我感到有什麼東西在扯我的頭髮，我轉過頭去看。

「我看到了一個非常特別的景象。一個人形生物大約20公分高，是我見過最醜陋、最畸形、最多皺紋和最微小的種類。其顏色如枯萎的楊樹葉子，有點黃褐色；聲音很高，吱吱作響。 他被我的頭髮纏住了，掙扎著想要逃跑，並一直在埋怨和抱怨，說我沒有權利出現那裡，打擾到樸實的居民，並說我的頭髮會勒死他。 最後，他自己脫困並消失了。」

她描述的可能是米紹博士童話故事中的可怕哥布林 !傑弗裡-霍德森在《工作和遊戲的仙女》中記錄了靈視力眼觀察，也證實了這兩段描述。米紹博士在19歲時，便能看到大自然隱秘王國中難以捉摸的居民。因此，《精靈女王》描述的事件可能真的發生過，儘管不一定完全一樣；總是得給作家一些詩意修飾的空間！

至於為什麼米紹博士在其一生中從未將此發表？當我向其中一位編輯提出此問題時，他以一種非常狡諧的方式敲了敲鼻子，露出了神秘的笑容。我想米紹博士會同意的！米紹博士一生致力於驅散無知烏雲，無知遮蔽了真理之光；並努力恢復古老智慧，指引通往自由之路。 

《精靈女王》為飢者賜食，為渴者解渴；這是真正的天賜甘露，獻給那些遠離家園、渴望美好事物的流離之人。當光明、愛與希望似乎永遠離開這個世界之際，這裡仍滿溢著安慰與扶助。願它照亮你的聖誕，溫暖你的心靈，並在你一生乃至永恆的歲月中帶來喜悅。
