# 《與開悟者同行：玫瑰十字會歷險記》電子書
## 任何對神祕學懷有興味的讀者，都不應錯過這部入門經典。

![封面](/img/downloaded/e6d15ef3243c6cbd6035e77f4f914286.png)

電子書連結：[點擊閱讀](/books/awakening-rosicrucian-adventure)

《與開悟者同行：玫瑰十字會歷險記》初版於一九一〇年，作者弗朗茨·哈特曼首次以小說體裁鋪陳神祕學思想。筆調輕快，深淺皆宜，無論初探或深研的求道者，皆能從這本精要的入門書中找到樂趣。

故事始於阿爾卑斯山的一次遠足。敘事者偶遇一位神祕矮人，引他前往山中隱修院，拜會一位玫瑰十字會的開悟者。書中探討諸多玄奧主題：煉金術、前世、心靈感應、靈視力、星光體漫遊、元素精靈，乃至駕馭心智的種種法門。

隱居處另有住民，包括兩位神祕且極為美麗的女性「啟蒙者」，她們娓娓道出鮮活的前世記憶。書中還演繹了一場奇妙的示範：如何以煉金術士聞名的「紅色粉末」，將白銀點化成金。

總歸而言，這是一本奇書，真假相參，恰似哈特曼本人絢爛不羈的一生。即便如此，本書仍舊值得一讀。其對自然的描摹令人神往，所揭示的神祕實修洞見，在其他「開悟者」小說中亦屬罕見。

節選《第六章：鍊金房》：

西奧多羅斯並未從那張大扶手椅起身，只吩咐我取來桌上的空坩堝，置於壇上燃著火的三腳架。我依言照辦。接著他說：「現在，從你口袋裡掏出幾枚銀幣，扔進坩堝裡。」我摸出隨身的七枚硬幣，全數投入。幾分鐘後，銀幣開始熔解；見那銀液在鍋中漾開，我便稟告開悟者，說它們已然熔化。他隨即指向桌上一只盛著紅粉的小瓶，要我往坩堝裡灑一點。我取來一支小銀匙，舀了一兩匙紅粉，正欲傾入，西奧多羅斯卻攔住我，說這太多了，不該如此揮霍。他要我將粉末倒回瓶裡，再用紙擦拭銀匙，然後把紙扔進坩堝。粉末歸瓶後，匙上殘存的紅粉已微不可辨；但我仍依他所囑，將那小紙片擲入熔銀之中。紙片霎時燃起，熔融的金屬隨即沸騰鼓脹，幾乎要溢出堝緣；泡沫升到頂端便迸裂，綻出絢爛難言的繽紛色澤。

這色彩的幻變持續了約莫一刻鐘，液體漸漸止沸，泡沫緩緩沉回堝底。西奧多羅斯聽聞坩堝不再作響，便指示我將它從火上移開，把內容物傾倒在大理石板上。我照做，那團泡沫頃刻凝固，竟化作了成色極好的純金。

「帶著這金子去吧，」西奧多羅斯說，「讓人檢驗一番，你才肯信這並非幻覺。」

我驚詫不已，心想世人或許願意付出任何代價，只為窺探這紅粉的祕密。我極想詢問開悟者，這粉末該如何製得，卻不敢貿然開口，唯恐西奧多羅斯以為我心存貪念。然而開悟者早已洞悉我的心思，他說：

「待人的靈性成熟，我們方能告知他如何製作這紅粉。欲知此祕，必須親身踐行。我們又該如何教人運用他們既未擁有、亦不知其存在的力量？話說回來，這些力量的種籽，其實早已潛伏於每個人的生命之中。」
