# 《星光界對話》終章：揭示心智與身體的真相

### 系列第十二部，以幽默神秘學對談探討恐懼、求道與星光體真相，以及意識階段的淨化過程

![封面](/img/downloaded/astral-con-1.png)

## 形而上學思索

隆隆和撲通在最後一次對談中，再次與那位僅知名為「M」之人重逢。這位神秘人物為混亂帶來秩序，在兩人的思索中投下一道澄明的光。

## 介紹

這篇文章是《星光界對話》系列第十二部分，此系列是帶有幽默感、不尋常的神秘學研究。

自從隆隆和撲通第一次為「大師」爭論不休以來，已取得很大的進展。雖然這兩位性格鮮明的神秘學學生名字是虛構的，但確實是我們認識的真實人物；在經歷了重重階段後，才得以在通往智慧的狹窄道路上，獲得最小一線光明。

這些階段包括懷疑、迷信、質疑、輕信、教條以及狂熱；在達到光明之前，每個階段都需奮鬥並加以克服。這是個漫長淨化過程，逐漸掃除頭腦中錯誤的想法和誤解，包括隆隆和撲通在先前11次對談中討論過的。

在此最後一篇文章中，此過程達到了頂峰。但這並不表示這兩位朋友已無事可學，也不意味著他們已徹底探究了追尋真理之路上所有的幻象、錯誤與障礙。這麼做需要寫上好幾本厚重的書，仍難以窮盡其全。但只要多少消除求道者的妄想，這些努力也不會白費了。

隆隆：「自從上次與黑魔法師交手後，你的心神恢復了嗎？」

撲通：「還沒有完全恢復呢。我這輩子從沒那麼害怕過。」

隆隆：「這就是為什麼你會陷於危險，而我不會。」

撲通：「恐懼是一種惡嗎？」

隆隆：「最大的惡之一。」

撲通：「為什麼？」

隆隆：「因為這會麻痺人心中更高的志向。人若向恐懼屈服，就像如同囚犯被關在自己製造的鐵柵欄裡，任何信心、希望和勇氣的光芒都無法通過。不僅完全無助，且一旦繼續住在自造的囹圄裡，任何幫助都無能為力。」

撲通：「難道你一點都不怕嗎?」

隆隆：「當然會，但這還不足以破壞我對光明力量的信心，且我們都受到保護。」

撲通：「我們都受到保護嗎？」

隆隆：(微笑著)：「當然。我知道M一直在保護著我們，才敢暴露於此番危險中。」

撲通：「要是他沒有來救我們呢？」

隆隆：「別人會的。」

撲通：「若他們沒有呢？我們會死嗎？我們會去哪裡呢？」

隆隆（帶著嘲諷的不耐煩）：「你是個頑固的害蟲，不是嗎？」

撲通（狡猾地）：「這是我為數不多的美德之一。」

隆隆：「好吧，若你能走進廚房去燒個水，也許能滿足你的好奇心。」

(撲通好奇的從椅子上起身，走進廚房。片刻後，他壓抑著一聲驚呼，又有些不好意思地回來，後面跟著 M 微笑的身影，手裡端著一個托盤，上面放著三個杯子、一個茶壺、牛奶壺和一碗糖。）

M：「恐怕你得用茶包了，因為散裝茶葉莫名其妙地消失了。」

隆隆 （調皮地）：「我不知道原來屋子裡有星光界老鼠。」

M：「我懷疑是一隻大『老鼠』拿走了散裝茶葉，是不是這樣啊，撲通？」

撲通（羞愧）：「呃...嗯...好吧，我只是做了一些茶葉算命的實驗，純粹是為了研究，你明白的...」

M（坐在兩位朋友的對面）：「那你的結論是什麼？"

撲通: 「我認為這是在浪費時間！」

隆隆（啜飲著茶，神情比平常少了幾分滿意）：「更別提浪費相當好的茶了。」

撲通：「如果你真想知道，我正試圖用茶葉占卜當星光體被摧毀時會發生什麼。」

隆隆：「向茶葉詢問？你沒有受過教育嗎？」

M：「我親愛的隆隆，自視甚高比愚蠢還要更嚴重。因為即使是傻瓜，也可能在過一段時間後變得有智慧，但傲慢則無盡頭。驕傲展現出對於智慧的全然無知，以及對眾神慈恩的毫無感念。因爲是眾神賜予那份智慧——而人類卻愚昧地以為是 自己的。」

隆隆（懺悔）：「我不經思索就脫口而出；我很抱歉。」

M：「這是我們每個人偶爾都會犯的錯誤。我們就不再多說了，不如來看看撲通的問題，如這樣大家或許都能從中受益。若你回想一下之前的談話，你會記得，在身體死亡後，高等自我回到了家，無論這個家在哪裡，或許是在更高或更低的層面中，繼續其生活。因此，若有黑魔法師成功地摧毀了你們的星光體，你們就會在另一個身體中回到了『家』；因為宇宙任何地方都沒有『死亡』，只有變化。

「隆隆本可以將這些告訴你，但沒這麼做的原因，是因為各種『身體』極其錯綜複雜，關於此主題的謬論幾乎比任何其他神秘學主題都多。有人提到『乙太雙體』、『星光體』、『心智體』、『菩提體』，以及『能量體』甚至『情感體』。我們可以再添加一個，叫做『多管閒事體』。之所以有這麼多關於『身體』的胡言亂語，是因為一代代通靈者看到了無法完全理解的現象，接著被無知的好事之徒盲目模仿、加油添醋所致。應該好好清除這些錯誤觀念，這些身體根本不存在。對於真正追求真理的人而言，擺脫這些謬論只會更有益處。」

撲通：「親愛的M，關於星光體的真相是什麼？」

M：「所有層面的顯現只涉及兩種基本原則，即心智與身體。也可稱為靈和物質，重點是心智需要某種身體，才能在物質世界中顯現。我用『星光體』指我們此刻所居住的身體，只是因為沒有更合適的詞。物質身體也是『星光體』；這個詞的意思是『如星光照耀的身體』。各種不同的身體只是由同一原初 「星光」物質的不同狀態組成，彼此之間的區別只是程度上的，而非本質上的不同。簡而言之，無論心智在死後住在哪裡，或在「星光體投射」過程中前往哪裡，都會擁有與所處環境相適應的身體。心智在存在的層面上提升得越高，身體就越纖細，反之，若下降到低等領域則越粗顯。 為這些身體取不同的名字，就好比每換一套衣服自己就換個新名字，而且根本不存在所謂的『能量體』或『情感體』，那些以為自己會被『阻塞』的人，只是自欺欺人；而那些靠清理所謂『阻塞』來謀生的人，不管他們是否自知，都是十足的騙子。」

隆隆：「正如某些教義所說，是否真的有一天，心靈會完全脫離一切形體的存在？」

M：「是的，這種變化會發生在最高的靈性層面。 但在那之前，還有無數百萬年要在星光界和物質界中度過，包括我自己在內；所以我們現在完全不必為此操心。」

撲通：「說到改變，我注意到星光界有些變化是不可能在塵世上發生。 例如有一天，我成功地改變了花園裡的花的顏色，使它們變成兩倍大。為什麼在星光界可以這麼做，而在塵世上則否？

M：「在塵世上也是可以的，但只有最偉大的開悟者才具備此能力。之所以在星光界更容易實現，是因為星光界的物質與塵世上的物質密度不同；星光界裡的物質更容易對魔法師的心智力量做出反應。」

撲通：「這是否意味著星光界的事物比塵世上的更不真實？」

隆隆（皺著臉放下杯子）：「這難喝的茶對我來說可是真實得很。可惜你沒換茶葉！」

撲通：「你怎麼知道我沒有？」

隆隆：「因為你太正派了，不會做這種事。」

撲通（笑）：「好啦，說真的，有什麼區別呢M？」

M：「非常少。星光界此部分與物質層面非常接近。這就是為什麼此處情況與塵世上情況幾乎完全相同。正如我剛才所解釋的，所有物質都是由同一基質的不同狀態組成。當此基質的狀態越精細，心智所見的物質形體就越不堅實，也就越容易被改變。 塵世上的物質密度較大，因而變化是漸進的。然而在星光界，事物的外觀可能在轉眼間發生變化--看啊！」

( M揮了揮手，驚喜地倒吸了一口氣，只見他們房間的墻壁瞬間消失，取而代之的是一片起伏的草地。他們發現自己正坐在草坡上，身旁是微笑著的老師。不遠處，金色花莖上是碧藍睡蓮，在波光粼粼的池水中亭亭玉立，彷彿無聲地向湛藍天穹中的燦爛太陽祈禱。）

撲通：「這可真是個好變化！不過，我們現在坐的這片河岸，怎麼感覺和剛才房間坐的扶手椅一樣結實。這怎麼可能呢？」

M：「我是說同樣堅實而非堅硬。對星光界的居民來說，此處的事物和塵世上的一樣堅實，但卻沒有粗糙物質那樣的堅硬。(由於撲通 繼續露出不解的神情，M補充說）：就像黏土的堅硬比不上燒製成磚後，更精細層面的物質因為較少阻力，而更易被塑造成不同的形體。」

隆隆：「你的意思是說，物質形體不是真實的，還是說我們沒有看到其真實面貌？」

M：「兩者都對。所有的形體都是幻象，因為它們不斷變化，而凡是會變化的，就不能稱之為真實，正如赫爾墨斯在《神聖皮曼德》中所說。」

撲通：「那麼什麼是真實的？」

M：「唯有不變的才是真實的。凡是會變化的，不僅自身是虛幻的，展現給感官的表象也是虛幻的。」

隆隆：「根據這個道理，我們也不是真的，威脅我們的黑魔法師也不是真的。」

M：「沒錯，我的朋友。真實只存在於自身，並始終如一。人由多種本質構成，其中唯有『神聖靈魂』始終不變。這才是人真正的實在，其余皆為幻象。若你真正明白這一點，就能像我一樣輕易驅散那巫師及其追隨者製造的幻象。」

隆隆：「那麼，什麼是根本的實在？」

M: 「至一源頭，至一生命，或稱隱藏而不可知的最高神；萬物皆由它而生，卻絲毫不損；萬物最終歸於它，卻也未曾增添分毫；它擺脫了存在和出現的幻覺。這就是至一真理，別無他物。然而，正是在那不可思議的非存在之中，蘊藏著難以言喻的榮耀。人的心智若能與那神聖境界相契合，並完全領悟其中奧義，那麽人便真正與那至一神、至一生命無二無別。」

撲通：「這太深奧了，難以理解。」

M（眼睛閃爍著逗趣的光芒）：「我的朋友，你會期望神的奧秘淺顯易懂嗎？」

隆隆（笑著）：「科學家們似乎這麼認為。」

M：「這也正是他們始終在原地打轉，不斷翻攪著物質主義的泥淖，卻始終未曾發現哪怕一絲真理的原因。每個人都在心中構建了一個他們所認知的宇宙圖景。但這不過是一幅圖像，並不存在於現實之中，就像他們用塵世的頭腦或肉眼所想像或看到的其他事物一樣，並沒有真正的存在。當他們為宇宙的各種理論賦予種種心智或物質屬性時，這些屬性注定會逐漸消散、歸於虛無；因為這本就只存在於他們的想像之中。聖者的偉大想像也是如此，只不過是通過大自然這面偉大的鏡子，映現在人類心智之中。」


「這些都是同一種想像力，但所在層面無限高，其影響似乎永恆不滅；然而，此乃源自隱密邏各斯心智的想像，或稱創造性成像；或者源自光之諸子的想像。隱密邏各斯是不可知至高神的最初散發物，而光之諸子是隱密邏各斯的自生後代，是其代理者，被稱為諸創造者；這些生命之主們的創造性觀想，基於更高世界或層面的光中原型來創造。這些創造物是透過人類的想像力而存在，能夠將大自然中觀察之物，從自己的頭腦中反射回到大自然。正如你們二位都相信我們此刻正安坐於這美妙的地方交談，而在此之前，我們事實上還在別處品茶。」

撲通：「我已經忘記了茶的事！它到哪去了？」

M：「我的年輕朋友，只要你不再去想它，它就變成了幻象。你賦予它想像中的存在，雖然延長其壽命，但也注定了最終會像海市蜃樓一樣消散，就像你也注定如此。還請記住，人類或任何其他存在，所能反映的眾神思想，需內在進化狀態允許、且與之契合的部分。 進化程度最低的人，只能像在夢中一樣，反映自身的形象和想像。而進化程度稍高之人，能反映大自然一些物質方面，吸引了自身動物本能和需求；以此類推......直到進化程度最高的人，幾乎能反映眾神所有崇高思想。

「人能以此方式反映創造者們的思想；不論階層多麼崇高，這些思想仍然只是反映；儘管看似存在，事實上並非如此！如果你能透過冥想，將這些概念套用於實在、空間、時間、永恆、存在和神；也許你就能明白我想要表達的意思。」

撲通（嘆氣）：「至少我們都看起來非常真實！」

M（笑著說）：「如果你強調的是『看起來』而不是『非常』的話，你會更接近真理，我的朋友。」

(隨著他們交談的進行，周圍的氛圍漸漸多了一層光彩與生氣。那柔和的光暈從覆滿青草坡的花草間散發出來，映照著他們坐著的地方。頭頂的天空，也悄然起了些微的變化。夏日深藍的天幕此刻已化作淡淡的金色，無數微小明亮的光點在其中漂浮，靈動地穿梭往來，彷彿各自肩負著神秘的使命。隆隆最先察覺到這一變化， 當他輕撫身旁一朵雛菊柔軟的花瓣時，，散發出令人愉快的香味，是世間任何一株普通雛菊都未曾擁有過的。）

正當他想問 M 這神秘的變化預示著什麼，一隻美麗的蜻蜓突然出現眼前，在他入迷的目光前盤旋。 這比先前見過的任何昆蟲都要大得多，薄紗般的翅膀閃爍著彩虹般的光輝，彷彿粗心的珠寶商撒滿珍貴的寶石。 撲通剛伸出手時，蜻蜓便落在攤開的手掌上，令他無比欣喜。它靜靜的停在那裡，紅寶石般的眼睛凝視著他，高興地顫抖翅膀。言語無法形容他們之間傳遞的瑰麗心念，彼此分享著對造物主的共同敬愛與感激。撲通的思緒與蜻蜓交融，突然意識到神之愛，心中溢出喜悅。在靜默的感激中，眼淚慢慢地順著臉頰滾了下來。） 

 撲通：「哦，太美了！我們剛才聊天的時候，肯定已經進入天堂了！」

M：「我的朋友，這不是天堂，只是天堂的外院，只有生前積德行善之人，死後才會被引領至此。」

隆隆（驚訝地環顧四周）：「我們到底是怎麼到這兒來的？這到底是哪裡？」

M：「你真的確定自己在這裡嗎？」

撲通（驚訝地）：「難道我們不是嗎？」

M：「請保持耐心，我親愛的朋友，關鍵藏在那位如神般的詩人彌爾頓話語中，他寫道：『心智決定了自身所在，能將地獄化作天堂，或把天堂變成地獄。』請放心，隨著時間的推移，這其中的含義將變得清晰。」

 撲通：「可自從你和我們在一起，就一直沒離開過這個地方，怎麼可能不穿過任何東西就到此處呢？」

M（輕聲笑著）：「你是指某種屏障嗎？比如維吉爾筆下的冥河，但丁的『黑暗森林』，或者許多『出體』經歷的人提到的『黑暗隧道』嗎？恐怕這些屏障都只是感官的幻覺。對於普通人來說，在從一種意識狀態到另一種時，若沒這些幻覺的輔助，可能會更令人困惑和不安。 星光界的各個層面之間有許多劃分，代表不同的意識狀態，但展現樣貌取決於觀察者心智的進化狀態。有些人會看到一條河，有些人看到一堵牆，如穆斯林所謂的『艾爾·阿拉夫』在《古蘭經》中將天堂和地獄分開。對於遇到這種屏障的靈魂而言，它們確實非常真實。但對於啟蒙者而言，這些障礙並非阻礙，因為他已獲得自由，能依其內在修為的程度，進入星光界中的任何層面。」

撲通：「那麼，阻止我們領略眼前奇跡的，其實只是我們自己嗎？」

M：「沒錯。但是只要心智仍停留在肉體中，執著於物質和欲望，就不容易實現這一點。因為除非此人聖潔善良、純潔仁慈，否則無法見到更高的層面，其心智對光明是封閉的，聽不見靈魂的聲音。然而，一旦心智能聽到，靈魂就會以種種奇妙方式顯現。開悟者捨棄肉身之前，都已明白感官只是幻覺。有許多方式能達到此境界，比如行善積德、將思想契合於與美、真理和智慧，或者通過在真正大師指導下靈修。這種境界不是靠空談、幻想或放縱自我就能獲得的。只有寬容、耐心和無私努力的人，才能逐漸領悟其中的真諦。如此，心智最終能抵達所向往的任何境界。」

隆隆 （若有所思地）：「我想我現在明白米爾頓的意思了。」

M（眼中閃爍著愉快的光芒）：「我不是早說過，一切都會更加明朗嗎？」

撲通：「在我看來，人們在追求覺悟的過程不打算花時間；他們都希望在五分鐘內就能達成。」

M：「那是因為時間並非人們所想的那樣存在，就像空間一樣。若認為時間會向前或向後移動，或可以『浪費』的話，那就好比認為空間有維度一樣，是個巨大的錯誤。所有物質都在空間和時間內運動，但凡人的眼睛永遠看不到這兩者。事實上，空間沒有維度，時間也不會移動。甚至將『永恆』一詞與時間關聯的想法也是錯誤的。因為這暗示了永恆與流動的時間有關，意味著有開始和結束。按照此想法，『永恆』作為一種原則，必然有個開始的時刻；但是，由於人類所能想像的一切都無法持久，此原則最終必須消失，不再存在。時間和空間之所以能存在，並非以人類想像的方式存在，正如光明和黑暗並非如人類想像的那樣存在。正如我先前所說，人所看到的一切都只是幻象。如果人類能夠在塵世型態下，憑藉塵世智力，真正瞭解時間和空間的本質，那麼人類將與最高神、至一生命、一切存在與非存在（但看似存在）之神聖源頭平起平坐；如此一來，人類的智慧將遠遠超越天界的主宰——那些造物主。」

（就在M說話的時候，那隻一直在認真聽他睿智話語的蜻蜓，忽然展開雙翅，飛離而去，也許是為了想自己獨自沉思時間與空間的奧秘。 撲通深深地嘆了口氣，目送那位神奇的使者直沖雲霄，消失在頭頂無垠的蒼穹中。 四周盛開著杜鵑與映山紅，粉紅與深紅的花朵交織成一片花幕，垂落至翠綠的草地上。含笑的毛茛和雛菊羞澀地向外窺視，微微側首，彷彿也在傾聽。空氣中彌漫著花草交融的芬芳，是在黑暗塵世上從未聞過，因為此處展示出了真實形態和本質；是投影到塵世的天界原型。遠處可見林間空地和莊嚴的橡樹林—象徵正義太陽—以及高聳的松樹，為疲憊的人間子女指引歸途。偶爾還有銀色的小溪流入湛藍的池塘，池面上優雅的天鵝悠然漂浮，四周環繞著睡蓮和其他美麗的水生植物，花葉交錯，令人心生喜悅。遙遠處，傳來大象歡快的鳴叫聲，正盡情享受新獲得的自由，擺脫塵世上無情天敵，其中最殘酷的是下流的人類。）

隆隆: 「我們該如何才能到達這天界境地？」

M：「只要人類真誠地希望得到這種神聖幸福，就有許多通往天界的途徑，現在和將來都是如此。人所能做的最好的事情，就是感恩上天賜予的一切美好，在逆境中保持耐心；當自己犯錯、以某種方式違背自然法則時，懂得請求寬恕，並且記住，無論迷失得多遠，總會有援手將他重新扶起；因為眾神極其寬容忍耐。只要盡力遠離邪惡，就能保持內心的明鏡清澈，其餘的將由永遠慈愛的光之諸子完成；因為他們也曾經是脆弱的人類，因而都能明白。」

撲通（若有所思地）：「這個想法真棒。」

M：「但這卻是千真萬確的，我的年輕朋友。如果人類能夠真正領悟這個道理，就會竭盡全力追求善良、仁愛與慈悲。那時，世界將立刻變成我們眼前所見的樂園，而如今橫行的『人間豺狼虎豹』，也會與溫順謙卑者並肩而臥，將其兇猛之心轉而用來幫助同胞，而非毀滅他們。」

隆隆：「這真的會發生嗎？」

M：「偉大的信息已經多次傳達，凡是聆聽的人都得到了拯救。那些剛從影子中進化而來的新來者，則必須被重新教導。真正的教義不應中斷；若所有人都時刻準備好接受，這些教義也絕不會消失。然而，每個人都必須在物質世界這所嚴酷的學校中學會自己的功課，才能最終升華到靈性層面。

這便是所有真誠追尋求光明者的獎賞，我親愛的朋友們，在此一瞥結束後，你們必須回到自己的肉身之中，因為在塵世間，還有許多工作等待著你們去完成。許多人只需要一只溫柔的手撫慰他們的額頭，或一句善意的話語，就能睜開雙眼，看見光明。」

撲通（眼含淚水）：「我們一定會努力的。」

隆隆：「是的，我們會的。儘管恐怕這會像用耙子把水推上山坡一樣艱難。儘管我們盡了最大的努力，但那些自封為薩滿的小團體、『揚升』的先知，以及在YouTube上揭示驚天『秘密』的視頻製作者，人數卻越來越多。」

M：「這是意料之中的事，我的朋友。自古以來皆是如此。真理之光只為少數人所見，夢幻般的影子則籠罩著多數人。然而，這些影子也有一個神聖的目的，只是那些棲身於黑暗中的人並不自知；若沒有影子，光名就不能被看見，也無法被認知。」

撲通：「這個道理實在難以理解。」

M：「但這是至關重要的。我有最後幾句話要對你們倆說。 無論對自己或他人，都要追求最真、最善、最美的境界；在思想與行動上，都要力求做到最好。別被那些自稱有智慧的人所迷惑——他們拐彎抹角的言辭，只是掩飾空洞心智的外衣。最重要的是，要警惕世俗知識與感官的誘惑，提防對權勢與財富的貪戀，以及那種自以為靈性的虛榮——因為這一切終將消逝，如幻影在上帝永恆之光中消融。再見了，我的朋友們，不要感到沮喪，因為我們終將再次相見。」
