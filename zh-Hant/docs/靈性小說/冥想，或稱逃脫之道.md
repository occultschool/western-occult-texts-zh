# 冥想，或稱逃脫之道
## 一個充滿奇蹟與啟示的故事，探討冥想與成長的真正意義

![cover](/img/downloaded/4e7eab352faf6b31c07090ad66b73bba.png)

這故事看似荒唐，卻暗藏玄機，埋著許多神祕學的真諦。

一切始於求道者常問的一句：「冥想，何為上策？」我們總答：「全心去做。」為說得明白，便借用了米紹博士《神秘之謎》裡的一則軼事。故事的確鑿，我們敢擔保——所有細節皆有檔案為證，收在個人卷宗裡。

公元一二〇〇年四月一日，科隆著名大教堂的階梯上，躺著一名新生男嬰。這教堂歷史悠遠，超過六千年，尖塔、拱頂、鐘樓巍峨入雲，為紀念聖徒、殉道者與其他不甚明智的存在而建，是歐洲最懾人的建築之一。

嬰兒的出現，不只驚動科隆，更撼動遠近。成千上萬的人乘私人飛機、滑翔翼、遊覽車、蒸汽火車，甚至蹬著輪鞋趕來，只為親眼一睹這樁奇蹟，餵飽那無謂的好奇。電視與廣播每小時追報，推文如潮水湧現。這古老小鎮從未如此喧騰，人人爭論、猜測嬰兒的來歷、身世，乃至父母——倘若真有父母。

據說前一夜，有人目睹一艘巨大的不明飛行物掠過神聖的教堂頂，轟聲駭人，火花四濺。或許這孩子正是其中一星火花，悄然化為活生生、啼哭不休的男孩？還是與驚擾不列顛的尼斯湖水怪有關？或是幾日前萊茵河裡游過的海豚？無人知曉。好奇的群眾圍住市政廳，警察護送孩子前往，入場券已炒至天價。好萊塢最犀利的狗仔直飛而來，舉起相機與手機快拍，捕捉孩子、人群、教堂石階與市政廳的每一瞬。

市政當局由布爾戈梅斯特主事，他鄭重趕來調查。在市議員、巫醫、通靈者、尋常醫生，以及其他自認聰明的人協助下，他們檢視我們的小英雄，參詳星象、羊肝、鵜鶘飛姿、企鵝、塔羅與種種徵兆，最終達成一致：這孩子志向超凡，必將遠行，攀上神聖高處，終成受人尊崇的聖人。

如您所知，德國人做事向來徹底，而這孩子似乎比誰都徹底——姑且當他是德國人吧，而非懷抱大任的外星來客。

占卜者的宣示留下了無與倫比的印象，敬畏與恐懼交織；朝聖者隊伍絡繹不絕，攜來珍貴禮物，裝飾華麗的駱駝低聲嘟噥（難道牠們知道——又有誰知道——阿拉的第一百種尊名？），大象昂首吼叫，日日前來向未來聖人致敬，在他的搖籃前祈禱、呼吼或哀求。東方智者亦至，眸如煤晶閃爍，鼻似巨鉤，鬚髮苔綠，他們販售孩子觸碰過的護身符，或他衣衫的殘片，所獲錢財足以為千所孤兒院的孩童添置百年衣裳。每道符必帶來鴻運與財富，每片布若藏入小囊、懸於頸鍊，將保你永健，賦你天使般的智慧。最奇的是，這些被蟲蛀的救世主，竟未替自己留下半件聖物。

喧囂漸息，古城沉回十二世紀的靜謐裡，孩子也如眾人期望，開始全面成長。當所有陌生的騷動遠去，老居民與孩童都舒了口氣；那時代的人，本不慣我們現代這般喧嚷。往事，就隨風吧。

不難想像，這孩子早年受盡細心呵護，監護人竭盡所能，為他鋪墊光輝前程。他們首先留意到男孩一項特質：一種極致且奇特的懶。他懶洋洋起身，懶洋洋睡去，閒時懶得動，忙時懶得停；年歲漸長，這份非凡的懶——姑且稱之為懶——愈發精純。

對勤勉的科隆人而言，這般景況簡直匪夷所思、聞所未聞。

局面日益不堪，終於他在格外怠惰的某一日，斷然決定：再不工作了。眾人皆驚！年滿十八後，他開始寫信給散居世界各地的親戚——那些赫赫有名的探險家與旅人；他極力遊說這些善心人資助他，好讓他栽培那份稀有的懶惰天賦。他想，如此既能得些錢財，偶爾度個假、享享清福；畢竟只歇不玩，聰明孩子也會變傻——這句諺語當時人盡皆知。

然而怪的是，他並未如願獲得自認應得的資助。即便在十二世紀，許多人也同今日一般，不願輕易掏錢。

眼看命運將臨之際，一股異樣的焦慮籠罩了他，旁人偶爾從他眼中瞥見夢幻般的遠眺神情。

二十歲生日當天，他悄悄攀上巨大郵輪「太陽系之星」的舷梯。那船正準備展開每週從科隆至坎昆的航程。他成功藏身蒸汽機旁，不久便沉沉睡去。漫長兩年多的航程中，一切如常。直至墨西哥灣，他們遇上一場駭人的颶風——這在向來以寧靜溫和著稱的地區實屬空前。

寧靜夜色裡，他不知不覺從船邊滑落，伴著汽笛聲與興奮乘客船員的狂歡、道別與祝福。無人察覺他的離去。他展開了一段奇異旅程，歷時逾四分之三個世紀。數月後，他筋疲力竭抵達秘魯的幸福海岸。長途跋涉未傷他分毫，他穿越洶湧浪濤踏上陸地，精神竟已煥然一新。只是那一週異常的辛勞令他倦膩不堪，旅程就此劃下句點。

他終究抵達了幸福與命運交會的目標，所有關於他光明未來的預言都將實現——徹底地實現！然而在達成這渴求的終點前，仍有試煉等候。才踏上陸地，他便遇上一大群生活在赤貧中的秘魯印第安野蠻人，他們從這片完全平坦荒涼的沼澤地現身，堅定而有禮地要將他交給他們的名廚。他們最擅長以最難入口的肉料烹出佳餚，這回正打算將這份才智施展於這位英雄身上，為他們的豐盛勝利添上榮光。

眾人都會同意，這確實是至高的榮耀。這位英雄在各方面皆令人嚮往。然而唉，人類的反常行為總難理喻。史上頭一遭，這位英雄對野蠻人的善良意圖展現全無興趣，並以粗魯自私的姿態向好客的主人高聲抗議：他遠道而來，不是為了當燒烤串、主菜，甚至牛排與腎臟布丁；因為他實乃一位偉大聖人，將為秘魯帶來光榮的新教義與祝福。身為聖人，他斷然拒絕這些貪婪的計畫。「若我說的不是事實，儘管打我屁股、掏我內臟！」（瞧，他有時很粗俗，但這些穢語從何學來卻是個謎。）

可憐的印第安人對這般粗暴與忘恩負義深感困惑沮喪，傷心地走入戰爭議會，在和平煙斗與一瓶白蘭地之間，商討這前所未聞的行徑。

月輪一次次橫越夜空，秘魯人吸食的煙斗愈多，空酒瓶愈積愈高，他們的困惑也愈發深重。我們的英雄在沙灘上徘徊，炎陽下氣氛緊繃，他急不可待要開始那宏偉使命，為俘獲他的人帶來無數祝福與好處；同時也尋思各種藉口，想躲開那口烹煮大鍋，避開那些布丁、餡餅與甜點。

終於，當最年輕的戰士無法再多抽一斗煙或斟一杯酒（他終究太過年幼，未有長者的歷練），他站起身，激昂低語：「沼澤的弟兄們；我們長久爭論，從各角度、三角與方塊剖析問題，但無論重啟多少次言語之旅，總似繞回原點。我建議將此事呈報大祭司與皇帝陛下，無論他們如何裁決，我們皆無異議。贊同者，請起身！」勇猛的戰士們踉蹌而起，關節咯吱作響，邊起身邊打嗝低喃，盛讚這絕妙提議堪比古聖先賢的智慧。

三聲清亮的呼喊響起，他們踏上旅程，在英雄陪伴下展開了漫長無盡的探險。最終，他們抵達了隱於秘魯山脈、峽谷與峭壁之間的黃金首都。

長途跋涉令人清醒，他們很快來到這座華麗的寶石之城——世上最富裕繁華的都城，日夜宴飲慶典不斷——並將我們的英雄引見給微笑親切的執法者、大祭司與仁慈的國王。

陛下聽罷來龍去脈，忽有奇異靈感；他問這位英雄打算如何教導秘魯的善良子民——他們早已精通掠奪之道、應對埋伏之法、最精細的折磨技術，以及最有價值且持久的犧牲朋友與敵人之術，還能學到什麼呢？

難道他知道一種連他們都未聽過的新型漫長苦刑？若是，那是什麼？不妨在他身上試試，聊表善意，再交給廚子處置。自然，他明白在秘魯，無人能指望比這更高的禮遇了。

國王與眾人肅然靜默，我們的英雄緩緩起身，以那駭人而凜然的聖潔目光直視國王，朗聲宣告他的來意：他身負使命，要教導他們「逃脫之道」——方法便是與山巔的聖者一同冥想。

在山巔冥想？這對秘魯人簡直聞所未聞。若他真要離開（雖然看來不大可能），或者真要上山冥想，該冥想什麼呢？這位英雄從未細想，一時答不上來，只得搬出「空」來搪塞。他本極懶，轉念又想：倘若真得上山冥想（只要他們准許——橫豎比煎成肉餅強），那麼冥想「空」大概最省氣力，至少能暫且擺脫那些煩心念頭，像是淪為一盤豬肉燉豆子的慘狀！

飢腸轆轆的群眾強烈反對這計劃；他們天生的機警，猶如油膏上聚攏的蠅群，一陣恐嚇的低語掠過會場，像風吹過玉米田。然而國王以一記皇家手勢——俗稱「嗤之以鼻」——安撫了不安與猜疑；他與祭司，以及一群位高權重的智者展開深談。此外還有鑽研弗洛伊德心理學、哲學與形上學的秘魯教授，煉金術士、魔法師、預言家，以及手持棍棒、石頭、茶葉等法器的占卜者，皆為這新穎而全然現代的實驗著迷，紛紛勸國王抱持「草草看過」的態度。於是，救世軍的樂隊、擲彈兵衛隊，連同許多木琴、iPod等精緻樂器，隆重護送他前往山間沼澤，將他懸吊在最高峰的尖頂，任他靜心參究「空」。他宏偉的命運就此展開；在這世間，他終將顯露非凡才華。

秘魯印第安人在山腳跳著神聖的儀式舞，一面緊盯著他，防他悄悄溜走。他就坐在那纖細的尖頂上，深深專注，思索「空究竟是什麼」。這專注持續了三十年，未曾離開一步。他也無法離開——焦急的村民早以印第安人特有的方式環山守候，連一隻跳蚤也難逃他們的眼睛與議論！須知這群人是飢餓之族，絕不會為了一次冥想，放棄一頓聖人做成的豐盛晚餐。

三十年屆滿，印第安人再度攀上山巔，接他下來，詢問他在漫長靜觀中領悟了什麼。但當他們抵達尖頂，卻發現三十年的冥想已令他消瘦、枯槁，終至消逝無蹤，唯餘一撮灰塵，留在他久坐的尖端。

他們虔敬地取下灰塵，安置在主廟金色神龕的水晶器皿中，依然尊崇他的回憶與遺灰。他被尊稱為聖阿亞薩莫納，意即「長久沉思空性之人」。最終，他蜕變為「空性」，徹底了悟其意旨。

這便是冥想能達到的境界，只要你全心投入；而逃脫之道，亦將為你敞開。

## 隱藏的真理

讀罷這篇關於「冥想，或逃脫之道」的故事，或許你會懷疑作者是否真的冥想太久，以致腦髓銷溶，終成塵埃。

然而，容我們坦言：這故事其實以最笨拙的方式，隱藏了許多真正的神秘學真理。正如我們在多篇文章中指出的，古代的哲學家、教師、啟蒙者、煉金術士等，總是以類似手法掩蓋他們的知識或發現；有時假借嚴肅議論，有時運用奇異詞彙、符號或密碼，但總帶著一層神秘或隱晦的色彩。

依循我們的慣例，現在揭露這個故事中的若干神秘真相，以及那些刻意佈下的「障眼法」、矛盾與混淆。這或許能給你提示，教你如何閱讀並理解這類神秘著作。但我們必須強調：最好還是憑自己的努力探尋意義，因為這樣你學得更多，更會珍惜所得的知識。

故事裡，我們首先看見的第一個真相是無知。無知者認為，那些行事古怪卻不觸怒他人者，必是高人；而智者則看穿表象，得見真實。在此故事中，那孩子其實無足輕重，他唯一的名聲，源於出生時的異狀與那股奇特的懶散。

可以說，無知是諸「罪」中最重的一項，因為它阻礙人去探究最微小的真理，反而依賴他人的意見與臆測；正如我們所見，關於這孩子成長的種種荒誕幻想，不僅被無知者信以為真，更遭人添油加醋；他們喪失了獨立思考之力。

對這孩子極度懶惰的描述，掩蓋了第二個真相：「惰性」是所有神秘法則中最偉大的一條。它的反面是運動，這兩道法則或力量之間，隱藏了一切物質情境與現象的運作法則。關於這些法則，你可在米紹博士的[《金黃星》](/books/golden-star-dr-michaud)中找到更多資訊。作者將此事實置於眾目睽睽之下，正印證了：神秘主義與神秘學作家的文字，在明晰之處往往更欺人，在隱晦處往往更真實。我們已在文章中多次提及此點，但它至關重要，值得一再重申。

故事裡的第三個真相是：一件事若「看似」愈神奇、愈不可思議，人們便愈熱切相信，而它往往愈不真實。好比故事裡圍繞孩子發展出的「神聖護身符販售」——今日基督教會或許不再販售聖徒遺骸，以維持神職人員的貧儉生活，但現代異教徒、女巫與新紀元追隨者之間，護身符交易依然活絡。奇怪的是，這些易受哄騙的人，從不問問：為何兜售這些「護身符」的販子，自己鮮少具備他們商品號稱能賦予的特質？

這便引出第四個真相：多數宗教遺物或「魔法」護身符唯一的力量，源於我們對它的信心。之所以說「多數」，是因任何物件都可能被賦予或善或惡的神秘力量，無論佩戴者信或不信，它們確實可能生效。然而，如今懂得如何做到的人寥寥無幾，可以說是沒有。

第五個真相，是「吸引」與「排斥」的普遍法則。故事主角想從親戚那裡吸引一些金錢，卻因自身的貧苦境遇而遭排斥，所求盡皆落空。成功或失敗、健康或疾病、貧窮或富裕，皆繫於這道神秘法則的運行。《聖經》裡也有這話，耶穌說：「因為有的，還要給他；沒有的，連他所有的也要奪去。」（馬可福音4:25）倒想看看哪位神學家能解釋這則看似極不公義的寓言。

有些政客熱衷鼓吹「劫富濟貧」，這不僅違背「蠢人留不住財」的古老箴言，也掩蓋了眾所周知的現象：不少出身寒微的彩票得主，在揮霍獎金、縱情奢靡之後，最終落得悲慘收場。事實是，富人彷彿處於吸引財富的狀態，如同磁石吸鐵；而窮人因其一無所有，反而排斥財富，甚至連手中僅有的也留不住。

這真相或許不討喜，卻適用於萬事萬物，無論物質或靈性層面：健康、力量、智慧、真理或知識。物以類聚，同類相吸，異類相斥——這與「異性相吸」的法則看似矛盾。實際上，兩者皆成立，只是運作方式不同，端看所適用的原理。由此帶出第六個真相：「矛盾」不過是哲學家用來遮蔽意義的障眼法。

故事裡充斥看似矛盾的情節：主角身處的時代，竟有許多當時尚未存在的現代發明與技術；又如墨西哥灣從未受颶風侵襲，或是一個「荒蕪」之國竟人煙稠密！這些矛盾迫使我們超越表象，以更高的理性去觀照、調和。有些事物確實能同時虛空又充實，好比悲觀者與樂觀者眼中的半杯水，或是一個愚昧的頭腦。

第七個，也是最後一個真相，將我們帶回起點，圓滿了整個循環。這真相藏在故事的標題裡：「冥想，或逃脫之道」。標題本身隱藏了幾件事實。其一，我們無法如故事主角那般冥想「空無」。科學告訴我們，自然「厭惡真空」。你或許要說，世上多的是腦袋空空之人，但我們認為，他們的腦中其實塞滿了某種東西——至於是什麼，就留給你去想。

其二，透過正確的冥想修習，我們能超越物質世界的幻象，深入靈性之境，親見其中奧秘。冥想也能喚醒我們對生命偉大真理的覺知，這些真理難以言傳，只能在內心靜默中領悟。一旦我們掌握並透徹理解這些真理，便不再需要於塵世輪迴中投生為人。

或許你會說，這些真理如此淺顯，連孩子都懂（無論是不是在科隆大教堂的階梯上發現的！）。這話也許有理，但若真如此，為何你從未實踐？若你早已實踐，便無需我們協助，也能讀懂玄學與秘義之作。

然而，倘若你從這個故事與我們的討論中學到新知，何不試著自己去發掘更多「淺顯」的真理？不只這個故事，我們發表的許多文章裡，都以隱微的方式蘊藏著大量神秘學真理。如果你願意這麼做，並將你的發現分享給我們，或許我們也能從中學到新事物。畢竟，我們還不至於狂妄到自以為已無新知可學。
