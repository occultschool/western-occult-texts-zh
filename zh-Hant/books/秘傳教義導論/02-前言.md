#  前言

對於學習神秘學的學生而言，要彌合基礎知識和進階知識之間的差距，是非常困難的。這兩種極端的神秘學文獻很多，但很少能引導學生跨越兩者之間的鴻溝。本書試圖彌合此鴻溝。我們假定讀者已經閱讀或學習過一些既有的神秘學文獻，並且嘗試閱讀了一些更為艱深的著作，也許自認成效不彰。此學習指南旨在提供基礎知識，使學生準備好接觸更進階的內容，勇敢面對其中令人困惑的問題，並能有條理地形成自己的見解。書中除了探討棘手的哲學難題和教義中的難點外，也提到一些較為淺顯的內容。

我坦率誠實地指出，晚期神智學作者在某些觀點上開始存在分歧，並且已偏離布拉瓦茨基及早期「智慧大師」所傳授的教義。別人可能指責我將布拉瓦茨基奉為權威，我只能坦然承認，並請求諒解。

我也請求讀者原諒，因自身理解有限，某些內容難免與原教義有所出入。任何人只要自以為可以論述神智學，卻非擔任大師的書記員  ( 如布拉瓦茨基 ) ，必然承受此「詛咒」。

每當這本學習手冊再版時，我都想根據自身多年來不斷改進的理解，對內容做出各種修改。但大幅度的修訂將會極大地改變和擴展原本內容，導致無法實現最初的寫作目的。因此，我盡量只做最少量的修改，希望本指南對某些學生群體仍然有益。這次我特別感謝大衛布魯斯和尼古拉維基審閱手稿並提出改進建議。
