# 第二十一章 狄奧

每當對於未來的憂慮將要籠罩我的時日，我便擱下不想。此刻繁花正好，腳下芳甜。清晨醒來，心裡盈滿歡欣——陽光照耀這片土地，**狄奧**與我同在。我不再央求**托特．特拉．達斯**講古老智慧或光明勇士的故事；他倒說起那些戀人，情意堅貞，強大如神。

**狄奧**與我從不言愛，卻知愛正滋長。他明白我須待啟蒙儀式後方能離開神廟，卻以為那時我便自由，能與他共度人生。我彷彿三人困於一身：渴望與**尼亞**並肩，成為法老；然而在與**狄奧**相伴時，但求自由，身為他妻，豐足安寧。我們共處時光清澈似夢，未染塵埃，不見時日催迫的陰影；我守護思緒，不讓恐懼玷污——深怕若在啟蒙中死去，這幸福便戛然而止。

啟蒙之時將近，母親的智慧看透我心，便遣**狄奧**南行六個月，為神廟雕像挑選新石。雖然感傷他將遠行，卻慶幸啟蒙之時他不在神廟。

臨行前夕，他告訴我：從今起第六個滿月之夜，他將歸來，我們在沼澤邊相會——那往昔常漫步之地。他說，那時他終於自由，能訴盡心中一切。

我思忖：歸來時，他是否將擁我入懷？抑或我的身軀已躺入石棺，脈搏不再為愛之旋律躍動，生命之息已絕，鼻孔冰冷，瀝青與沒藥的氣味不復聞得？

**狄奧**南行不久，寄來一首詩，以彩墨工整抄於小卷紙莎草上：

> 我是失卻雙手的雕匠， <br/>
> 是無水流淌的果園； <br/>
> 我是無風裡的帆船， <br/>
> 是無法振翅的鳥； <br/>
> 我是枯池中的蓮， <br/>
> 是弦斷的弓； <br/>
> 我是無神駐守的聖所， <br/>
> 是無星點綴的夜空—— <br/>
> 只因我須遠行，長途跋涉， <br/>
> 而你未將心交給我攜帶。 <br/>

後來他又寄來：

> 種子已播下， <br/>
> 穀物自犁溝萌發。 <br/>
> 漁夫撒網， <br/>
> 魚群躍動其中。 <br/>
> 採葡萄者壓榨果實， <br/>
> 酒甕滿盈。 <br/>
> 投棍劃過空氣， <br/>
> 飛鳥墜落獵人眼前。 <br/>
> 夜雖漫長， <br/>
> 黎明終重燃白晝。 <br/>
> 正午烈日炎炎， <br/>
> 涼夜裡影子漸長。 <br/>
> 我已將心獻你， <br/>
> 你可願將心予我？ <br/>
