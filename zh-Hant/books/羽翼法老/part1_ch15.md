# 第十五章 札．阿泰特

我問尼亞長大後最想做什麼，他說：「噢，我想治理人民，制定明智的法律，以蛇的智慧與獅的勇氣率領軍隊。」

我說：「我想做許多奇妙的事……連我自己也不太明白的事。有時日常瑣事顯得無比重要，卡姆彷彿遼闊無邊，少有人願跨出疆界。但有時，深夜躺在床上，我望向窗外星辰，想著地球多麼渺小，我們的國家像一粒沙，而我微小如塵，螞蟻爬過也不會覺得路不平。明白這份渺小，我渴望看見遠方……就像你的野貓隔著籠欄，聽見夜晚胡狼嚎叫，知道牠們正目睹自己只能猜測的景象……」

「別動，塞凱塔！有頭瞪羚來喝水，你看陰影深處。」牠啜飲時，水面泛起漣漪。接著牠抬頭聆聽，彷彿受驚，隨即躍入蘆葦叢中。

天色漸暗，尼亞說該回家了。途中，我們聽見農夫唱著耕犁歌，牽牛返回牧場。這是我最喜歡的耕犁歌：

> 套上軛，我的牛，套上軛。 <br/>
> 筆直前進，犁啊！讓我的田畦平整如梳分秀髮。 <br/>
> 大地，敞開子宮接納我撒播的穀粒，用溫暖庇護，在陽光下孕育。 <br/>
> 聽啊，種子！聽鳥兒歌唱，向上生長，聽得更清。 <br/>
> 水，快流過溝渠，為我的作物注入汁液。 <br/>
> 溫暖它們，太陽！用生命之光溫暖它們。 <br/>
> 溫柔些，風，對待我成熟的穀穗，別讓沉甸甸的穗壓彎細莖。 <br/>
> 割斷莖稈，我的鐮刀，割斷莖稈如新月劃破黑暗，讓我的打穀場堆滿金黃。 <br/>
> 搗吧，杵臼，磨我的麵粉，讓我家不缺生命之塵。 <br/>
> 燒旺些，火，讓烤爐熾熱，麵包烘透，我吃了有力氣為牛套軛。 <br/>

歌聲漸遠，我們緩緩歸家。抵達宮殿前，道路已沉浸在月光的銀色水波中。
