# 揭開面紗

以至聖者之名。<br/>
我的靈立於上帝寶座之前，<br/>
在至高者前的光中移動；<br/>
祂的眉宇隱沒於無垠之中，<br/>
祂的雙足安息於太陽之上。 <br/>
諸天在祂面前如地毯鋪展； <br/>
祂身披閃電之焰； <br/>
黑暗與神秘的寂靜 <br/>
在主面前環繞、籠罩著。 <br/>
祂獨自處於光中—— <br/>
獨自於遙遠的天域。 <br/>
獨自——獨一，<br/>
無受造物與至高者共享黑暗的奧秘。 <br/>
所有的力量，所有的王子，<br/>
宇宙所有蒙福的靈， <br/>
所有偉大的能量與靈質， <br/>
皆立於上帝寶座之前。 <br/>
他們以群星的節奏向祂頌歌， <br/>
在元素之球永動的光輝中， <br/>
歡欣於祂本質的榮光， <br/>
在主流溢的美麗中喜樂。 <br/>
此時聲音自寶座傳來， <br/>
深沈如雷、如恒久旋律； <br/>
明亮如閃電，柔美如長笛輕吹。 <br/>
那是上帝的聲音； <br/>
我認得它——我靈在靈中認得它： <br/>
那是父與王的永恆聲音， <br/>
祂一開口，宇宙便靜止。 <br/>
聲音說道：「人子啊！」 <br/>
我回答道：「看哪，我在這裡。」 <br/>
聲音又道：「我是主，你的上帝，是我創造了你， <br/>
並將我的火焰吹入你的胸膛。 <br/>
我差遣你到世上，向世人宣講 <br/>
真理、光明與智慧的話語； <br/>
去復興古老的聖潔； <br/>
去揭示我所顯明之事：<br/>
在晨曦榮耀行走之處，<br/>
我以太陽的閃電托舉你； <br/>
在黃昏時分群星沈落之地， <br/>
我以智天使的翅膀載你； <br/>
我揭開奧秘的帷幕， <br/>
向你的眼睛開啟諸多秘密； <br/>
將我的雷霆置於你火焰之手， <br/>
去擊碎迷信的神殿。 <br/>
如今，回答我， <br/>
你是否已完成莊嚴的使命？ <br/>
你是否已從宇宙陰暗的結構中 <br/>
投下一塊石頭？」 <br/>
聲音止息，寂靜降臨； <br/>
那沈默刺透我的靈魂； <br/>
一種可怖的靜默籠罩我的靈； <br/>
我惶恐、羞愧、低頭不敢仰視。 <br/>
不久，聲音再次響起； <br/>
對我說：「人子啊！」 <br/>
我回答說：「看哪，我在這裡。」 <br/>
聲音說：「回答我。」 <br/>
我俯伏在寶座之前； <br/>
死亡的寒意貫穿我全身； <br/>
那神秘寂靜的威嚴 <br/>
凍結了我靈魂深處的靈。 <br/>
「至聖者啊，我比蟲還卑微； <br/>
我是滿載罪孽的卑賤罪人； <br/>
我曾迷失於黑暗的雲霧中； <br/>
我曾行走在虛妄與死亡之路； <br/>
我偏離了應與祢同行的正道； <br/>
我由塵土、雲霧與灰燼所造， <br/>
追逐著塵土、灰燼與腐朽。 <br/>
主啊，塵世與塵世之物， <br/>
使我的靈遠離了祢； <br/>
上帝的聲音在我不安的靈魂中低語，<br/>
我卻充耳不聞。 <br/>
我將心思繫於必朽之物，<br/> 
繫於終將消逝的事物； <br/>
卻鮮少將漂泊的思緒安放於<br/>
永恒不滅的王。 <br/>
然而如今，主啊，如今我將覺醒， <br/>
如今我將訴說； <br/>
愚昧啊，離開我吧！ <br/>
從今以後，我是至高者的僕人。」<br/> 
我說話時仍俯伏在地， <br/>
因為上帝的奧秘如山壓在我身上；<br/> 
一種深沈、無可抗拒的敬畏； <br/>
如死亡般昏厥之影。 <br/>
此時諸天界傳來樂音， <br/>
柔和而神聖的和聲； <br/>
遠處的力量 <br/>
以金色樂器甜美頌唱： <br/>
一道如紫紅寶石的光芒， <br/>
緩緩降臨在我頭頂； <br/>
如同榮耀的冠冕將我覆蓋， <br/>
宛如彩虹橫跨大海。 <br/>
當我伏地時，聽見號角響起， <br/>
孤獨而狂野的爆響； <br/>
使我的靈震顫， <br/>
彷彿被雷霆劈為兩半。 <br/>
一道光的漩渦將我向上卷起； <br/>
是一道閃耀、顫動的火焰旋渦； <br/>
宇宙間所有的光輝 <br/>
以斑斕色彩將我環繞。 <br/>
上帝是無限、全能者， <br/>
將我升至群星的火焰基質之中； <br/>
我見到了太陽的壯麗； <br/>
行星合唱團環繞著舞蹈。 <br/>
多彩美麗的晨曦 <br/>
照亮金色的拱門； <br/>
天界七條純凈的河流 <br/>
在陽光下流淌閃耀。 <br/>
自然的創造者映入眼簾， <br/>
那神聖輝煌的天界之神； <br/>
其臨在如春夏般溫暖； <br/>
是樂園中生長的生命之樹。 <br/>
如同月亮著伴隨旅人， <br/>
在午夜孤行之時； <br/>
她以柔光安慰他， <br/>
彷彿在他身旁美麗地滑行： <br/>
然而她依舊靜止遙遠， <br/>
高坐於蒼穹深處； <br/>
上帝也如此引領我， <br/>
向上，進入廣闊的天界。 <br/>
我未見祂，卻知祂在； <br/>
我未見祂威嚴的形象： <br/>
祂遠在自己的領域； <br/>
是神聖臨在的中心榮光。 <br/>
其面容不斷展現新輝， <br/>
如同天際變幻的美麗； <br/>
紫色、白色、玫瑰色、銀色； <br/>
以及夜空星火的深邃濃烈； <br/>
上帝永恆閃耀的面容也是如此， <br/>
在我眼前一掠而過； <br/>
卻如轉瞬即逝的夢境， <br/>
祂在其中閃耀，卻又自隱其身。 <br/>
我被火焰托舉前行， <br/>
見到一位太陽的輝煌天使； <br/>
他高聲宣告， <br/>
主旨如下： <br/>
「向你致敬，你是持斧羅摩， <br/>
在美麗的金色原野中； <br/>
是閃電與真理所揀選的孩子；<br/> 
我久已盼望你的到來。 <br/>
向你致敬，你是伊瑪目·馬赫迪， <br/>
被天言烈焰所環繞： <br/>
我久自群山遙望， <br/>
只為見證你降臨的陽光。<br/> 
如今你升至上帝支撐宇宙的柱石，<br/> 
包裹在黑暗閃電中， <br/>
在烈焰迅疾翻滾的雷霆中。<br/> 
這些高聳柱石<br/>
從寂靜之水的深淵中升起 ；<br/> 
黑暗籠罩其可怖的胸膛； <br/>
無數偉大的靈引導其廣闊的力量；<br/>
進入山岳， <br/>
深處湧出這些海洋之水； <br/>
進入風的懷抱中安息，<br/>
比陽光中的塵埃還要渺小。 <br/>
進入那靜止而莊嚴的旋風，<br/>
其微弱的氣息誕生這些風； <br/>
旋風遠在彼方， <br/>
安居於全能上帝的懷抱。 <br/>
這些柱石被稱為真實； <br/>
這些水被稱為古老； <br/>
這些山被稱為堅強； <br/>
這些風被稱為永恒； <br/>
在其中，你將成為真理的見證，<br/> 
見證神聖之光——上帝的氣息； <br/>
在你狂喜的目光前<br/>
將揭開宇宙的奧秘 。」 <br/>
隨即閃電劃破長空， <br/>
雷聲回應了閃電； <br/>
太陽的天使離去； <br/>
我繼續前行飛翔。<br/>
看到世上無人能見的群星， <br/>
在金色光域中閃耀； <br/>
美麗且迅疾舞動； <br/>
不可思議，唯有思想可比擬。<br/> 
黑暗環行的廣闊， <br/>
超越人類一切想象； <br/>
太陽光的七芒星球亦在其中，<br/> 
唯有上帝能丈量其偉大軌跡。 <br/>
蒼穹在遠方延展， <br/>
如同金與綠柱石築成的墻；<br/> 
在最深的夜、火焰與星辰的掩蓋中，<br/> 
我見到上帝榮耀的永恒寶座。 <br/>
天界的諸靈被光芒包裹， <br/>
美如陽光下安歇的花朵； <br/>
那裡沒有晝夜的更替， <br/>
唯有上帝恒久的榮耀照耀一切。<br/> 
「聽我說吧，美麗的星球， <br/>
傾聽我，天堂的孩子們； <br/>
還有你，時間，永恒者的象徵， <br/>
你，地球，側耳傾聽； <br/>
因為無垠的威嚴向我開啟， <br/>
展現宇宙浩瀚的本質； <br/>
諸天與恒轉太陽的熾烈光輝，<br/> 
廣闊的大地與無數的海洋星球。 <br/>
在光中——在火中——在霧中——在焰中， <br/>
我如乘日光漂浮， <br/>
屏息於熾天使的狂喜中；<br/> 
晨曦的駿馬載我前行。 <br/>
那是什麼光芒——什麼榮耀在遠方綻放？<br/>
是何輝煌在太陽中閃耀？<br/>
<br/>
她起身——她微笑——她站立無遮——我看著——我看到了一切。
