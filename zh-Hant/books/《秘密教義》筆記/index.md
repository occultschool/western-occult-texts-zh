## 📘 關於這本書

許多讀者在面對神智學鉅著《秘密教義》時，常因其浩瀚深奧而感到無所適從。這份簡短卻極具份量的筆記，記錄了作者親自傳授的「解鎖」祕訣，教導我們如何跳出線性思維，用直覺去領悟宇宙的奧秘。

### 這本書的獨特之處

- 🗝️ **作者親授的研讀法** 兩位秘傳學生親自傳授的解讀鑰匙  
🧠 **打破思維定勢** - 訓練直覺與抽象思維的獨特方法  
📝 **精煉的學習綱要** - 提煉出最核心的形而上學概念  
🧭 **避免誤區的指引** - 幫助初學者避開字面理解的陷阱

### 你將學到什麼

- 掌握研讀《秘密教義》的正確心態與方法
- 理解「三項基本原理」在學習中的應用
- 培養直覺與宏觀的思考模式
- 建立對神智學宇宙觀的整體架構

### 適合誰讀?

- ✓ 覺得《秘密教義》艱深難懂的讀者  
✓ 希望提升直覺思維能力的學習者  
✓ 所有神智學的認真研習者  

## 👤 關於作者

**記錄者：兩位秘傳學生**

這部作品的作者自稱為「兩位學員」，他們是早期神智學運動中極具熱忱與洞察力的研究者。當布拉瓦茨基夫人於1888年出版其鉅著《秘密教義》時，由於書中內容深奧且結構龐雜，許多讀者難以領悟。這兩位作者成為了最早嘗試將其內容系統化的先驅。

他們以極其嚴謹的態度對《秘密教義》進行了索引編制，將內容分類為「顯教」與「密教」兩部分，並梳理出關於宇宙演化與單體旅程的根本原則。這項成果最初於1890年發表在布拉瓦茨基夫人創辦的《路西法》（Lucifer）期刊上，並獲得了布拉瓦茨基夫人本人的高度讚賞與認可。她稱讚這兩位學員的努力是其他神智學徒的榜樣，而他們的作品也被視為後世學生解讀神智學哲學的重要指南與地圖。
